I'm doing some research on downsizing applications from mainframe
CICS/ESA to unix (AIX) using Microfocus COBOL and Microfocus Transaction
System and I've hit with a snag -

In the mainframe system, there are some files loaded into 'assembler'
programs - basically programs with data space and assembled. The
data is loaded using EXEC CICS LOAD.

Does any one know how I can implement something like that in MTS ?
I thought of using C load modules, but not sure how to define it so
that MTS can recognise it correctly.

Any suggestions are appreciated. Thanks.
```

MCOv3/MTS does support EXEC CICS LOAD (as it did in MCOv2.5 on
DOS/OS2).
The table data must be assembled prior to porting to the UNIX
machine, and the table data must have an extension of .mod.

Regards, Alan.
```
