Yes I have done it with NetExpress. You can use the Win32 apis: CreateFileA,
GetCommState, SetCommState, ReadFile, WriteFile, GetCommTimeOuts,
SetCommTimeouts, and CloseHandle...

Beware of slight differences in the way these apis function under 95 vs NT...
