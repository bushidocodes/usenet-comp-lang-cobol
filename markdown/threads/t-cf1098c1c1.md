Welp, the only I've ever did a SET ADDRESS was in a CICS program many moons ago so it sounds like to me what y'all want to do is:

MOVE item-in-linkage TO item-in-working-storage.

Doesn't matter if the WS item is an 01 or not.  All you have to worry about is if the types are the same (PIC X to PIC X - PIC 9 to PIC 9 etc) and the lengths are the same.  Not the same lengths you'll have to worry about high or low order elephantnosation.
