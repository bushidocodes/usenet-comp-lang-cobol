The official website is at www.thebignumber.co.uk

Basically they're changing the numbers (again!!) to accomodate, more
numbers and prefixes which indicate the type of number to be called.
e.g. 07 is a mobile. 09 is a premium rate number. 08 is free or low
rate.

The problem this time is that most of the country isn't changing
numbers for fixed line phones but some are.

It difficult for me to gauge the business case for automating change
over for those numbers affected. If you have a business that trades in
an area with a new code you will quickly learn the new code. If you
trade nationally you will not know the new codes, how many of your
customers are in the areas that are changing etc.

The official website has an applet that will tell you what the new
number is when given the old number. It would be nice if they shared
the code with us so that we could reuse in our applications...
