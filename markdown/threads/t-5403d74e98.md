Which COBOL compiler is the student using?  There are several "flavors" of
screen section, for example, AcuCOBOL provides a fully GUI-ized screen
section.  Micro Focus would probably allow you to do this via the CONTROL
phrase.  A purely "X/Open conforming" Screen Section might well NOT support
this easily.
