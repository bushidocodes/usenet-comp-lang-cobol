MicroFocus Cobol manual indicates that you can skip locked records on
a network by using the SET COBSW=+B0 environment variable. I tried
this with both Ver 3.2 and 3.4 and the file pointer would not
advance beyond the locked record. Every "Read NEXT" returned the
locked record. I tried it using both GNT and INT programs and also
"read next with no lock". The record had been locked using Auto lock
at another station. Even tried it as part of the RUN statement:
RUN +B PROGNAME. Also put the +B in RUN.CFG. same results.
Any suggestions ?
