IBM mainframe environment? (that's the only place I know the term 'abend' is
used)....

Well, yes you would get an abend.

But bear in mind that simply adding a FILE STATUS to the SELECT is
considered 'error handling' code in this environment, and if you do not
check your file status right after the OPEN and 'do something' on failure,
you will not get the abend until later, when you try to access the file you
thought you had opened.

MCM
