DIR is not a program, it is a command for COMMAND.com or, on W2000,
CMD.exe.
Try (I have not tested this!):
CMD.exe DIR ... 
as parameter of the X"91" F35.

for call "system" on a 16 bit COBOL compiler you need to link (use
litlink directive) with a C library from a 16 bit SDK or from a 16 bit
C compiler. I do not know, if these libraries are available with the
WB.
