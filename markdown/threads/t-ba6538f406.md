There are those that say COBOL will never die.
There are those that say COBOL is already dead.

COBOL, which is an acronym for COmmon Business Oriented Language, has been
around for many years.
There are literally millions, if not billions or even trillions, of lines of
code written in COBOL. I don't know if I would recommend COBOL as a
beginners language though. BASIC (like Microsoft's QuickBASIC) and PASCAL
(like Borland's TurboPASCAL) have traditionally been noted as good languages
for beginners. They teach logic and structure but are not as verbose COBOL.
