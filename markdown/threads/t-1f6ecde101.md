What COBOL are you using - that you want restructured?

If you are working on an IBM mainframe, most of the
"restructuring" products (including IBM's COBOL/SF) include a
reformatting option. This products, however, are fairly
expensive.

If you happen to have source code that will compile with IBM's
old (about to go out of extended support) OS/VS COBOL compiler,
you might want to look at the LISTER compiler option - which will
do a lot of reformatting for you (for free).
