I have no idea what the problem is, but I did a Google search for 
"crore," and some things became clear:

An Indian crore is ten million according to Wikipedia and one other 
source.  (The word is not commonly used outside the Indian subcontinent 
and is probably not good to use when communicating with English speakers 
elsewhere.)

It sounds like a field with the value (in American terms)

1,210,000,000

is being truncated to

210,000,000

when it is stored in a database record.

The database record description (schema, or whatever Ingres calls it) is 
probably worth checking.
