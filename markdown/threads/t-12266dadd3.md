› hi, does any body have an example of how to make bar menus in cobol ?

Shai:

Download an evaluation version of COBOL sp2 for the specific compiler
that you currently use. It allows you to create menu bars using ANSI
Standard CALL USING statements in one of 2 different ways:

1. Paint the menu bar with the screen painter and make a single CALL
to sp2 to invoke the menu screen.
2. Dynamically create the menu bar, pulldown menus and cascading
pulldown menus using COBOL sp2's panel maintenance functions invoked
from COBOL CALL USING statements.

Please also feel free to call us or send e-mail during the evaluation
period. We encourage it.
