```
In answer to the general question of whether VS COBOL II and PL/I V2.3 ILC
is supported with the LE run-time library, the following is the reply that I
got from my "usually reliable sources".

"  They sure are!  They should be link-edited with the PL/I Migration
aid (OS PL/I V2) first, then they can run under LE or pre-LE.  If the
OS PL/I V2 library is up to date (latest service) and the application
has been re-linkedited since the Migration Aid was added (it first
shipped in 1993), then no relink is necessary.  Oh, I just remembered,
if the PL/I is using multitasking, then there are some restrictions,
but PL/I multitasking use is not too common."

This doesn't answer the question of who has had experience with this, but it
at least give you the "official" answer.
```
