That fine.

But a word of advice.

1- You need to be carefull in the type of numerics you use in the
file.
As an example 
COMP-5 (Fuji) does not (yet) have a corresponding in RM.
and COMP-6 (RM) does not have the Fuji counterpart.

2- Do not try to use the file concurrently. File lock mechanism
will most probably be implemented differently over a network.

3- One thing I have not tried is to have a big file (> 2GB) being
create by Fuji and then accessed by RM. I do not know if this is
implemented in the same way.

Apart from the above you should not have any problem.
