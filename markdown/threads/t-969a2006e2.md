Redefine your variable as a table of one-byte elements (or move it to a
work field defined as a table of one-byte elements).

zero out your space counter
PERFORM VARYING x from last-byte by -1
until byte (x) not equal space
or x is less than 1.

Add 1 to a counter for every space you find.

If your compiler supports in-line performs and reference modification,
you can do this without branching away to another paragraph and without
redefining the field as a table.

01 test-variable pic x(20).
01 cntr pic 9(2).
if test-variable = spaces
move 20 to cntr
else
Perform with test after varying cntr from 20 by -1 until cntr = 1 or

= space
End-perform

compute cntr = 20 - cntr
end-if
.
Cntr is the number of spaces AFTER the last non blank character.

OK, here's an "improvement" on Thane's code:

drop the IF THEN LOGIC and just use the in-line PERFORM; change the "cntr = 1"
to "cntr < 1". This is simplier code, but may be less "efficient" than
Thane's if the field is blank often.

An alternative, if your compiler has the intrinsic functions, is:

MOVE zeroes to cntr
INSPECT FUNCTION REVERSE(test-variable) TALLYING cntr for LEADING ' '

Here is a routine that will return the length of a alphanumeric variable less
trailing blanks. With a small change to the code you can make it return the
number of trailing blanks.

This routine will work for any length string. Since you pass the trailing
character as an argument to TRUNCLEN you can look for any trailing character.


Example of usage:

77 RETURN-LENGTH PIC S9(8) COMP SYNC.
77 ALPHANUMERIC-STRING PIC X(100) VALUE 'ABCDEFGHIJK'.

MOVE LENGTH OF ALPHANUMERIC-STRING TO RETURN-LENGTH
CALL 'TRUNCLEN' USING ALPHANUMERIC-STRING, RETRUN-LENGTH,
BY CONTENT ' '


TITLE 'NESTED PROCEDURE - TRUNCLEN'
ID DIVISION.
PROGRAM-ID. TRUNCLEN COMMON.
*
* Return the length of a string less trailing character.
*
* 77 OUT-L PIC S9(8) COMP SYNC.
* 77 IN-CHAR PIC X(1).
*
* MOVE LENGTH OF IN-STRING TO OUT-L
* CALL 'TRUNCLEN' USING IN-STRING, OUT-L,
* BY CONTENT ' '
*
* On return OUT-L will contain the length of IN-STRING less
* trailing characters. In this example less trailing spaces.
*
ENVIRONMENT DIVISION.
DATA DIVISION.
WORKING-STORAGE SECTION.
01 WS-WORK-AREAS.
04 WS-STRING-PTR POINTER.
04 WS-STRING-9 REDEFINES WS-STRING-PTR
PIC S9(8) COMP.
04 WS-STRING-START-PTR POINTER.

LINKAGE SECTION.
77 OUT-L PIC S9(8) COMP SYNC.
77 IN-CHAR PIC X(1).
77 IN-STRING PIC X(1).

PROCEDURE DIVISION USING IN-STRING, OUT-L, IN-CHAR.

IF OUT-L < 1
MOVE +16 TO RETURN-CODE
GOBACK
END-IF

SET WS-STRING-PTR TO ADDRESS OF IN-STRING
SET WS-STRING-START-PTR TO WS-STRING-PTR

* Advance pointer to end of string.
COMPUTE WS-STRING-9 = WS-STRING-9 + OUT-L - 1

* Point to end of string.
SET ADDRESS OF IN-STRING TO WS-STRING-PTR

PERFORM
WITH TEST AFTER
UNTIL WS-STRING-PTR = WS-STRING-START-PTR
OR IN-STRING NOT = IN-CHAR
SUBTRACT 1 FROM WS-STRING-9
SET ADDRESS OF IN-STRING TO WS-STRING-PTR
SUBTRACT +1 FROM OUT-L
END-PERFORM

MOVE ZERO TO RETURN-CODE

GOBACK.

END PROGRAM TRUNCLEN.
