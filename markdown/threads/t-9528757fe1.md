Is WS-COL-LIKE-NAME defined as a VARCHAR?

Is the length correct?

Do you have a trailing %?

I found that using _ without a trailing % did not match. For example, 

 LIKE 'ABCD' (note - no masking characters) matched 'ABCD       '
 LIKE 'A_CD' (with a masking character) did not
 LIKE 'A_CD %' did match  

What mask are you using?
