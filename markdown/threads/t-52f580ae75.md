I wouldnt go to a university, I would find out what applications in COBOL on the
net are written in and I would go to a certification program that teaches that.
At Many Universities there are certified programs that are taught not for credit
in DATABASE applications that are designed to be used on the internet.  I think
there are some Microsoft Certification programs that use Oracle and SQL Server
to connect to a web server.  A big poblem is having a web server and a mainframe
to practice on.  A well rounded Oracle program might come in handy.  Oracle can
be implemented on the PC and on the Mainframe.  Oracle seems to be pretty
popular.  Microsoft SQL Server is also popular.  

I don't think a University can teach you what you want to konw.  It might be
easier to just work for some company for a while that does what you want to
learn, like an internship or something.  We had an oracle teacher that went and
worked at a local company so he could teach oracle.

There are many different ways to connect a mainframe to a web server.  Sometimes
we use a simple IP server with NT.
