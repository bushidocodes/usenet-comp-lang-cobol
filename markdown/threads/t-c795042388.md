```
It depends on WHICH COBOL compiler you are using.

1) For *MOST* compilers (that conform to the ANSI/ISO COBOL Standard), the
answer is
     NONE
because the OCCURS clause is not allowed for an 01-level item.

2) For those compilers that do allow an OCCURS with an 01-level, you will
need to check out their "limits" documentation.  There is no Standard (or
standard) for this.

3)  If you really meant to say what is allowed for
    05  a-table                occurs ????? times
then this is allowed by all Standard compilers - but again, WHAT value is
allowed will vary from compiler to compiler (often with a single compiler
depending on the release, compiler options/directives chosen, and/or
platform)
```
