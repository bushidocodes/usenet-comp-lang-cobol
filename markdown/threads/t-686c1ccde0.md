Hello,

there is one COBOL compiler for the HP3000 sold by HP. It has an ANSI 74 and
ANSI 85 entrypoint. Both versions of the compiler are very reliable.

On the HP3000 there is no exact equivalent of VSAM.
The basic file system knows sequential, direct, message, circular, etc.
filetypes.
The index sequential filetyp is called KSAM.

Most of the applications on the HP3000 use the build in database TurboIMAGE.
If you have further questions you can email me privatly.

Regards,
Guenter Kuhn
