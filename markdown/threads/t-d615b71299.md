```
Scope terminators are NEVER for phrases (such as ON SIZE, ON EXCEPTION, AT
END, etc).  They are for verbs (such as COMPUTE, READ, UNSTRING, CALL, etc)

Therefore, if you want to use a "scope terminator" for a STATEMENT with an
ON SIZE ERROR phrase it is always possible - but will depend on the verb
being used, e.g.

  Compute a = B * C
     On size Error
           Add D to E
                On Size Error
                    Divide F by G
                          On Size Error
                                 Call "ABC"
                                       On Exception
                                              Display "You have GOT to be
kidding"
                                 End-Call
                    End-Divide
            End-Add
   End-Compute
      .
```

    ON SIZE ERROR  DISPLAY "Too BIG"
