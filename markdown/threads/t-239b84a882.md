Hi Ken,

Currently, NetExpress from www.microfocus.com can work with external XML
parsers. There's a white paper on the Micro Focus we site that discusses
XML. Some time in the not too distant future, Net Express will be able to
deal with XML directly.

www.datadirect-technologies.com have an ODBC driver which can read and write
XML directly by mapping XML documents as if they are SQL database tables -
clever stuff.

The XML ODBC driver will work with Net Express and one for development is
supplied with Net Express I believe.

A white paper on the COBOL Portal here discusses, among other issues, XML
http://www.cobolportal.com/resources/articles/20010910_001.asp

Hope this helps,
Paul
