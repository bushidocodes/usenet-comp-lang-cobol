What it is telling you that *a* file name is an "Illegal file
name" on the OPEN statement. Because two files are
OPENed in the same statement, one of the two has an
"Illegal file name". In this case, it is likely the file-name
for FINAL-REPORT, device-choice, which contains
spaces.

The file-name for FINAL-REPORT is filled in
700-DISPLAY-RTN.

Try PERFORMing 700-DISPLAY-RTN before you
OPEN the files.
