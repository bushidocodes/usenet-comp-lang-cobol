```
Check your compiler, If it includes:

   USAGE NATIONAL

and the

    NATIONAL-OF and DISPLAY-OF intrinsic functions,

then the chances are good (but not 100%) that it does already support SOME level 
of Unicode.

Now, when you say "displaying" - what environment are you using to "display" 
data?  PC? Unix terminal? Mainframe attached terminal? printed report? other? 
The answer to this (and what software you use to display non-Unicode data may 
impact how well the compiler interacts with the display capabilities)
```
