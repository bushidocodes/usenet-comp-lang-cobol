Peter,

In NetExpress (which replaces Visual Object COBOL) you can place a call
to CBL_DEBUGBREAK in your COBOL. When this is executed a dialog
box is displayed asking you if you want to debug the COBOL.

I am not sure if this was in VISOC. Please check the documentation (I
no longer have access to VISOC).

Hope this helps.
