CICS
Customer Information Control System from an already acronym-happy IBM.

It is for supporting telecommunications, typically across dumb terminals
(327x).

It provides support to run one instance of an application to support
multiple terminal sessions, screen mapping support, i/o, and a whole bunch
of other stuff.

CICS also handles task switching between the different terminal sessions,
rolling in and out the different context.

It's called "kicks"
