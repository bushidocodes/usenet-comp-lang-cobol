You cannot make executable files with QBasic. This is because
QBasic is not a compiler -- it's an interpreter. The /run option is
the only way to run a QBasic program from the command line.
There is no way to force QBasic to exit without displaying the
program; QBasic's Help says that the function of the /run option
is "to load and run a program file before displaying it." Sorry.
