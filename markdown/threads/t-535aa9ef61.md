```
You need to use some kind of logic to read the key presses. How you 
retrieve the key press depends on the facilities provided by the 
compiler runtime libraries. If the run time libraries do not provide 
keyboard control routines you will have to write your own using your 
operating system API calls.

To do it properly you should a have the "keyboard scanning" routine 
running on its own thread (if your compiler supports multi-threading) 
and it starts another thread that handles the key press event if you 
want special treatment for that key. This is one of the areas that OO 
COBOL can do a good job if the compiler supports running different 
classes on different threads.
```
