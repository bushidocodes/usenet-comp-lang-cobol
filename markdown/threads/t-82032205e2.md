This should get you started. I didn't take the time to test it so there
may be a few bugs or inconsistencies. For more info look at the Win32
SDK documentation for the referenced calls.


SPECIAL-NAMES.
CALL-CONVENTION APIENTRY IS 74.

WORKING-STORAGE SECTION.
01 REG-Rc pic s9(9) comp-5.
01 REG-KeyHandle pic 9(9) comp-5.
01 REG-szData pic x(256).
01 REG-iDataLen pic s9(9) comp-5.

78 KEY-READ 1 OR 8 OR 10 OR
h"00020000" OR h"FFEFFFFF".
78 HKEY-CURRENT-USER h"80000001".

READ-KEY.
Call APIENTRY "RegOpenKeyExA" using
by value HKEY-CURRENT-USER
by reference z"MyKey"
by value 0 size 4
by value KEY-READ
by reference REG-KeyHandle
returning REG-Rc
End-Call
Call APIENTRY "RegQueryValueExA" using
by value REG-KeyHandle
by reference z"MyDataName"
by value 0 size 4
by value 0 size 4
by reference REG-szData
by reference REG-iDataLen
returning REG-Rc
End-Call
Call APIENTRY "RegCloseKey" using
by value REG-KeyHandle
returning REG-Rc
End-Call
