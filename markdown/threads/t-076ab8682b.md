Specifically what documentation are you looking for Ron. If *only* 
Language Reference Manuals then you *could* look at the current Net 
Express V 4.0 on-line manuals at :-

http://supportline.microfocus.com/supportline/documentation/books/nx40/nx40indx.htm

Bear in mind the above now contains COBOL 2002 features, which you wont 
have. Other manuals in the set like File Handling or Screen Handling 
should be fairly similar to what you are used to. If it's things like 
compiling/animating then I'm afraid you are screwed - you will have to 
go from memory because N/E V 4.0 is based on a Windows-oriented IDE.

I would doubt you would have any luck locating the DOS 3.2 on-line. If 
you go to the Micro Focus home page and check out local sales/tech 
support - if lucky you might be able to get an old set of the manuals 
gathering dust in a warehouse :-)
