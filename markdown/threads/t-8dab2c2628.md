The NO87 env variable turns off the use of the i87 floating point math
coprocessor and uses s/w emulation instead. Looks like your TCP/IP
interferes with the coprocessor in some way. Shouldn't make any
difference to your COBOl app using NO87 as long as the performance is
still acceptable.
