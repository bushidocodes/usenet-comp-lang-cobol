Here is a snippet of ECI code from a Micro Focus COBOL prog:-

      ***** Populate the CICS ECI Parameter Block

           move low-values to eci-parms
      *
           set  eci-sync            to true
           move "MERANTSR"          to eci-program-name
           display "MERANT"     upon environment-name
           accept eci-system-name   from environment-value
           set  eci-commarea        to address of wl-datahand-commarea
           move length of wl-datahand-commarea
                                    to eci-commarea-length
           move 0                   to eci-timeout
           set  eci-no-extend       to true
           set  eci-version-1       to true

      ***** Perform the synchronous call

           call  cics 'cicsexternalcall'
                 using by reference eci-parms
                 returning ECI-ERROR-ID
           end-call
           perform eci-check-status

You also need a special names

       special-names.
           call-convention 8 is CICS.   *> Used to force static linking

Hope this helps.
David.
