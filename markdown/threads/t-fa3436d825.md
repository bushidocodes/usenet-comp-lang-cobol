someone had asked for a routine to convert dates from Gregorian
YYYYMMDD format to Scalar Day, and back, and a method was posted
involving the use of, I believe, Function Integer-of-Date.  I had
asked if anyone had a method using 'older' cobol, and one poster had
responded with a sample, which I could not get working, and felt that
his base assumption was incorrect.  His base date date was 1901/01/01,
and he called this scalar day 0.  All other scalar functions I had
seen, including Microsoft's, use 1900/01/01 as scalar day 1.  

So I wrote one (had a couple of very slow days at work) in Fujitsu v4,
but entirely with Cobol74 constructs.  I'm sure it could be optimized.

Here is the subroutine:

 IDENTIFICATION DIVISION.
 PROGRAM-ID. DATECVTS.
 ENVIRONMENT DIVISION.
 CONFIGURATION SECTION.
 INPUT-OUTPUT SECTION.
 DATA DIVISION.
 WORKING-STORAGE SECTION.

 01  CD-CALCULATION-FIELDS.
     05  CD-Y-CALC               PIC 9(4)      COMP.
     05  CD-Y-QUO                PIC 9(4)      COMP.
     05  CD-Y-REM                PIC 9(4)      COMP.
     05  CD-Y-CALC-2             PIC 9(4)      COMP.
     05  CD-Y-QUO-2              PIC 9(4)      COMP.
     05  CD-Y-REM-2              PIC 9(4)      COMP.

 01  CD-DAYS-STRING              PIC X(36)
                 VALUE '000031059090120151181212243273304334'.
 01  CD-DAYS-TABLE               REDEFINES  CD-DAYS-STRING.
     05  CD-NUM-DAYS             PIC 999
                                 OCCURS  12  TIMES
                                 INDEXED BY CD-DAYS-INDEX.
                                 
 01  CD-DAYS-LEAP-STRING         PIC X(36)
                 VALUE '000031060091121152182213244274305335'.
 01  CD-DAYS-LEAP-TABLE          REDEFINES CD-DAYS-LEAP-STRING.
     05  CD-NUM-LEAP-DAYS        PIC 999
                                 OCCURS  12  TIMES
                                 INDEXED BY CD-DAYS-LEAP-INDEX.

 LINKAGE SECTION.
*  TO INVOKE TO CONVERT DATE TO SCALAR,
*      MOVE SPACES OR ZEROS TO CD-SCALAR-NUM-A
*      MOVE DATE (YYYYMMDD) TO CD-DATE-8.
*  TO INVOKE TO CONVERT SCALAR TO DATE,
*      MOVE SPACES OR ZEROS TO CD-DATE-8
*      MOVE SCALAR NUMBER (9999999) TO CD-SCALAR-NUM.
 01  CD-SCALAR-NUM               PIC 9(7).
 01  CD-SCALAR-NUM-A  REDEFINES  CD-SCALAR-NUM
                                 PIC X(7).

 01  CD-DATE-8.
     05  CD-YEAR                 PIC 9(4).
     05  CD-MONTH                PIC 99.
     05  CD-DAY                  PIC 99.
 
 PROCEDURE DIVISION  USING  CD-SCALAR-NUM  CD-DATE-8.
 
 PROGRAM-BEGIN. 
     IF (CD-SCALAR-NUM-A  =  SPACES  AND  CD-DATE-8  =  SPACES)
       OR
        (CD-SCALAR-NUM-A  =  ZEROS   AND  CD-DATE-8  =  ZEROS)
*  NO FUNCTION REQUESTED, SO RETURN        
         MOVE ZEROS TO CD-SCALAR-NUM-A  CD-DATE-8
         GOBACK.
     IF CD-SCALAR-NUM-A  =  SPACES  OR  CD-SCALAR-NUM-A  =  ZEROS
         PERFORM DATE-TO-SCALAR THRU DATE-TO-SCALAR-EXIT
         GOBACK.
     IF CD-DATE-8  =  SPACES  OR  CD-DATE-8  =  ZEROS
         PERFORM SCALAR-TO-DATE THRU SCALAR-TO-DATE-EXIT
         GOBACK.
 
*--------------------------------------------------------------

 DATE-TO-SCALAR.
*  BASIC DATE EDITS
     IF CD-DATE-8 NOT NUMERIC
         MOVE  0  TO CD-SCALAR-NUM
         GO TO DATE-TO-SCALAR-EXIT.
     IF CD-YEAR  <  1900  or  cd-year  >  2099
         MOVE  0  TO CD-SCALAR-NUM
         GO TO DATE-TO-SCALAR-EXIT.
     IF CD-MONTH  <  1  OR  CD-MONTH  >  12
         MOVE  0  TO CD-SCALAR-NUM
         GO TO DATE-TO-SCALAR-EXIT.
     IF CD-DAY  <  1
         MOVE  0  TO CD-SCALAR-NUM
         GO TO DATE-TO-SCALAR-EXIT.
     IF CD-MONTH  =  1 OR 3 OR 5 OR 7 OR 8 OR 10 OR 12
         IF CD-DAY  >  31
             MOVE  0  TO CD-SCALAR-NUM
             GO TO DATE-TO-SCALAR-EXIT
         ELSE
             NEXT SENTENCE
     ELSE IF CD-MONTH  =  4 OR 6 OR 9 OR 11
         IF CD-DAY  >  30
             MOVE  0  TO CD-SCALAR-NUM
             GO TO DATE-TO-SCALAR-EXIT
         ELSE
             NEXT SENTENCE
     ELSE
         IF CD-DAY  >  29
             MOVE  0  TO CD-SCALAR-NUM
             GO TO DATE-TO-SCALAR-EXIT.
              
     MOVE 0 TO  CD-SCALAR-NUM  CD-Y-CALC  CD-Y-QUO  CD-Y-REM.
     SUBTRACT 1900 FROM CD-YEAR  GIVING CD-Y-CALC.
     DIVIDE 4 INTO CD-Y-CALC     GIVING CD-Y-QUO
                                 REMAINDER CD-Y-REM.
*  IF NOT DIVISIBLE BY 4, NOT A LEAP YEAR - COMPUTE & GET OUT    
     IF CD-Y-REM > 0
         IF CD-MONTH = 2  AND  CD-DAY > 28
             MOVE  0  TO  CD-SCALAR-NUM
             GO TO DATE-TO-SCALAR-EXIT
         ELSE
             COMPUTE CD-SCALAR-NUM = (CD-Y-CALC * 365) +
                                 CD-Y-QUO + CD-DAY +
                                 CD-NUM-DAYS (CD-MONTH)
             GO TO DATE-TO-SCALAR-EXIT.
     
     MOVE  0  TO CD-Y-QUO-2  CD-Y-REM-2.     
     DIVIDE 100 INTO CD-YEAR     GIVING CD-Y-QUO-2
                                 REMAINDER CD-Y-REM-2.
*  DIVISIBLE BY 4, BUT NOT BY 100 - IT IS A LEAP YEAR       
     IF CD-Y-REM-2 > 0
         COMPUTE CD-SCALAR-NUM = (CD-Y-CALC * 365) +
                                 CD-Y-QUO + CD-DAY +
                                 CD-NUM-LEAP-DAYS (CD-MONTH) - 1
         GO TO DATE-TO-SCALAR-EXIT.
     
     MOVE  0  TO CD-Y-QUO-2  CD-Y-REM-2.    
     DIVIDE 400 INTO CD-YEAR     GIVING CD-Y-QUO-2
                                 REMAINDER CD-Y-REM-2.
*  DIVISIBLE BY 4, AND BY 100, BUT NOT 400 - IT IS NOT A LEAP-YEAR 
     IF CD-Y-REM-2 > 0
         IF CD-MONTH = 2  AND  CD-DAY > 28
             MOVE  0  TO  CD-SCALAR-NUM
             GO TO DATE-TO-SCALAR-EXIT
         ELSE
             COMPUTE CD-SCALAR-NUM = (CD-Y-CALC * 365) +
                                 CD-Y-QUO + CD-DAY +
                                 CD-NUM-DAYS (CD-MONTH)
             GO TO DATE-TO-SCALAR-EXIT.
*  DIVISIBLE BY ALL - IT IS A LEAP YEAR        
     COMPUTE CD-SCALAR-NUM = (CD-Y-CALC * 365) +
                             CD-Y-QUO + CD-DAY +
                             CD-NUM-LEAP-DAYS (CD-MONTH) - 1.

 DATE-TO-SCALAR-EXIT.
     EXIT.

*-------------------------------------------------------------- 

 SCALAR-TO-DATE.
     IF CD-SCALAR-NUM-A NOT NUMERIC
         MOVE ALL  '0'  TO  CD-DATE-8
         GO TO SCALAR-TO-DATE-EXIT.
*  SUBROUTINE CAN CONVERT YYYYMMDD RANGE 19000101 TO 20991231
     IF CD-SCALAR-NUM  <  1  OR  CD-SCALAR-NUM  >  73049
         MOVE ALL  '0'  TO  CD-DATE-8
         GO TO SCALAR-TO-DATE-EXIT.
         
     MOVE 0 TO  CD-DATE-8  CD-Y-CALC  CD-Y-QUO  CD-Y-REM.
*  THERE ARE 1461 DAYS IN EACH 4 YEAR PERIOD, THIS GIVES US THE
*    LAST LEAP YEAR AS A BASE YEAR     
     DIVIDE  1461  INTO CD-SCALAR-NUM 
                                 GIVING CD-Y-QUO
                                 REMAINDER CD-Y-REM.
     MULTIPLY  CD-Y-QUO  BY  4   GIVING CD-Y-CALC.
*  NOW FIND HOW MANY YEARS PAST THE BASE YEAR     
     MOVE  0  TO  CD-Y-QUO-2  CD-Y-REM-2.
     DIVIDE  365  INTO  CD-Y-REM GIVING CD-Y-QUO-2
                                 REMAINDER CD-Y-REM-2.
     IF CD-Y-REM-2  =  0
         IF CD-Y-QUO-2  NOT  =  0
             SUBTRACT  1  FROM  CD-Y-QUO-2
             ADD  365  TO  CD-Y-REM-2.
                                         
     ADD  CD-Y-QUO-2  TO  CD-Y-CALC.
     ADD  1900  CD-Y-CALC        GIVING CD-YEAR.
*  NOW WE HAVE THE YEAR, SO FIND THE MONTH & DAY     
     MOVE  0  TO  CD-Y-QUO  CD-Y-REM.
     DIVIDE  4  INTO  CD-YEAR    GIVING  CD-Y-QUO
                                 REMAINDER  CD-Y-REM.
*  IF YEAR NOT DIVISIBLE BY 4 - NOT A LEAP YEAR  
     IF CD-Y-REM  >  0
         PERFORM SEARCH-REG-YEAR THRU SEARCH-REG-YEAR-EXIT
         GO TO  SCALAR-TO-DATE-EXIT.
         
     MOVE  0  TO  CD-Y-QUO  CD-Y-REM.
     DIVIDE  100 INTO  CD-YEAR   GIVING  CD-Y-QUO
                                 REMAINDER  CD-Y-REM.
*  IF DIVISIBLE BY 4, BUT NOT BY 100 - IT'S A LEAP YEAR
     IF CD-Y-REM  >  0
         PERFORM SEARCH-LEAP-YEAR THRU SEARCH-LEAP-YEAR-EXIT
         GO TO SCALAR-TO-DATE-EXIT.                             

     MOVE  0  TO CD-Y-QUO  CD-Y-REM.
     DIVIDE  400 INTO  CD-YEAR   GIVING  CD-Y-QUO
                                 REMAINDER  CD-Y-REM.
*  IF DIVISIBLE BY 4 AND 100, BUT NOT 400 - NOT A LEAP YEAR
     IF CD-Y-REM  >  0
         PERFORM SEARCH-REG-YEAR THRU SEARCH-REG-YEAR-EXIT
     ELSE
         PERFORM SEARCH-LEAP-YEAR THRU SEARCH-LEAP-YEAR-EXIT.
     
 SCALAR-TO-DATE-EXIT.
     EXIT.


 SEARCH-REG-YEAR.
     SET CD-DAYS-INDEX  TO  1
     SEARCH CD-NUM-DAYS VARYING CD-DAYS-INDEX
             AT END
                 SET CD-DAYS-INDEX TO 12
         WHEN CD-NUM-DAYS (CD-DAYS-INDEX)  >  CD-Y-REM-2
                 SET CD-DAYS-INDEX  DOWN BY  1.
                 
     IF CD-NUM-DAYS (CD-DAYS-INDEX) = CD-Y-REM-2
         SET CD-DAYS-INDEX  DOWN BY  1.
                     
     SET CD-MONTH TO CD-DAYS-INDEX.

     SUBTRACT CD-NUM-DAYS (CD-DAYS-INDEX) FROM CD-Y-REM-2.
     MOVE CD-Y-REM-2 TO CD-DAY.

 SEARCH-REG-YEAR-EXIT.
     EXIT.
     

 SEARCH-LEAP-YEAR.
     SET CD-DAYS-LEAP-INDEX  TO  1
     SEARCH CD-NUM-LEAP-DAYS VARYING CD-DAYS-LEAP-INDEX
             AT END
                 SET CD-DAYS-LEAP-INDEX TO 12
         WHEN CD-NUM-LEAP-DAYS (CD-DAYS-LEAP-INDEX)  >  CD-Y-REM-2
                 SET CD-DAYS-LEAP-INDEX  DOWN BY  1.
                 
     SET CD-MONTH TO CD-DAYS-LEAP-INDEX.
     
     SUBTRACT CD-NUM-LEAP-DAYS (CD-DAYS-LEAP-INDEX) FROM CD-Y-REM-2.
     ADD 1 TO CD-Y-REM-2.
     MOVE CD-Y-REM-2 TO CD-DAY.

 SEARCH-LEAP-YEAR-EXIT.
     EXIT.

                                 PIC X(7).

     05  CD-YEAR                 PIC 9(4).
     05  CD-MONTH                PIC 99.
     05  CD-DAY                  PIC 99.

 procedure division.
 begin.
     display 'enter date as YYYYMMDD' upon console.
     accept Input-Date-A from console.
     
     if Input-Date-A = '99999999'
         go to pgm-end.
         
     move Input-Date-A  to  CD-DATE-8.
     MOVE SPACES  TO  CD-SCALAR-NUM-A.
     CALL 'DATECVTS'  USING  CD-SCALAR-NUM  CD-DATE-8.
 
     display cd-scalar-num upon console.
     if cd-scalar-num  =  0
         display 'INVALID DATE YYYYMMDD ENTERED...TRY AGAIN...'
         go to begin.
     
*    MOVE ....(input-field)  TO  CD-SCALAR-NUM-A.
     MOVE SPACES  TO  CD-DATE-8.    
     CALL 'DATECVTS'  USING  CD-SCALAR-NUM  CD-DATE-8.
     
     display cd-date-8 upon console.
     if cd-date-8  =  zeros
         display 'INVALID SCALAR DAY ENTERED...CANNOT CONVERT...'.
         
     go to begin.
 pgm-end.
     
     goback.
