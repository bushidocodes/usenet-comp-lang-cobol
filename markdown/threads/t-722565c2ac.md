Deleting a file does not delete the file.  It deletes the entry in the file
index, then returns the block numbers to the available space manager for the
OS.  If you want to scan through unused disk space, you can still see the
data that was in the files, though finding it and putting it back into a
semblance of order could be a major undertaking.  A "secure" erase writes
over top of the pages before erasing them.

Be that as it may, you have to be a terminal paranoia case (or really
stupid) to buy that software.  It is a scam, feeding on people's ignorance.
