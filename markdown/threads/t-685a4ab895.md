```
Hello,
I'm a student and i trying to figure out working with files by myself since
i think the class is going too slow, so i tried with my cursus book and a
buggy example on this NG to create a program that writes names you enter
with the keyboard to a file called namelist, the program work as it is now,
but i want to improve it.
So i have two questions : 1) i'd like to be able to give an extension to the
file, now it only creates a file called NameList but with no ".whatever" 2)
i'd like to write only a name by line, as now it writes them one following
each other.
Thanx for any help
Thomas

PS : code follows and please bear with my poor english

       Identification Division.
        Program-Id. essai.
        author. Thomas Singleton.
       environment division.
        input-output section.
        file-control.
         select NameList assign disk
          organization sequential
          access sequential.
       Data Division.
         file section.
          fd NameList.
          01 Name-Record pic x(20).
         Working-Storage Section.
          01 Nom pic x(20).
       Procedure Division.
        Main.
           open output NameList.
           perform until Nom = "xyz"
            display "Enter a name, end with xyz"
            accept Nom
       * "Nom" means name in french...
            move Nom to Name-Record
            write Name-Record
           end-perform.
           close NameList.
           stop run.
```

```
[snippage]

1.  For the file name question:  You don't specify what COBOL
    compiler you are using, but see if, in the FD for the NameList
    file, you can specify the filename using something like VALUE OF
    FILE-ID IS "NameList.txt".  Depending on the compiler, the key
    word represented by FILE-ID might be ID or FILENAME or some such.

2.  For separating the file name entries onto separate lines, try
    specifying ORGANIZATION IS LINE SEQUENTIAL for your NameList file.
```
