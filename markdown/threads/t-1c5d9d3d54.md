```
I suspect that the reference to 'Micro Focus 4.5' is actually
Microsoft 4.5, which is a rebadged version of Micro Focus 2.5.

Once the compile has been done the result is a .obj, this needs
to be linked with the libraries to create an executable (.exe).

Use:

LINK name;

However, if you are using ISAM files and/or extended accept or
display (eg DISPLAY ... AT nnnn, or SCREEN SECTION), then you
will need to link support for these too:

LINK name+extfh+adis+adisinit+adiskey;

where extfh is indexed file support and adis~ is accept display
support.
```
