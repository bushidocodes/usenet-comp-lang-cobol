You should realize that a WRITE statement without any ADVANCING phrase
acts as if you say WRITE ... AFTER ADVANCING 1 LINES. You probably
want to use WRITE ... BEFORE ADVANCING 1 LINES. There is not enough
of your program to be sure, but this is probably the cause.
