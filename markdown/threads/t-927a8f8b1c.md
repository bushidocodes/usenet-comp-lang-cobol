Greg,

For this particular purpose a much faster way would be to call a Win32 API,
MoveFile.  Here is a sample (VisualAge COBOL for Windows):

CBL pgmname(lm)
       Identification Division.
       Program-id "fubar".
       Data Division.
       Local-storage Section.
       01 fu  pic x(12) value z"c:\list.dat".
       01 bar pic x(12) value z"c:\list.old".
       01 rc  pic 9(9)  comp-5.

       Procedure Division.
           Call "MoveFileA" Using fu bar Returning rc.
           If rc Is Not Zero Then
               Display "success"
           Else
               Display "failed"
           End-If.
       End Program "fubar".

Hope this helps.

Gary Mazo
