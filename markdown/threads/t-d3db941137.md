```
Surely you just need to have an SD such as:

     SD   User-Sort
     01   Sort-Record.
          03  User-Keys           PIC X(32).
          03  Original-Key        PIC X(20).


STRING (or reference modify) the selected sort criteria into
'User-Keys', put the data records' key into 'Original-Key' (assuming it
is an index file) and then SORT ON ASCENDING KEY User-Keys.  In the
output procedure reread the data record using the key carried in the
sort.
```
