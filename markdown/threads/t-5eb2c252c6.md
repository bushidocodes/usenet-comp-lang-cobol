```
I use the following when running under Micro Focus Cobol Workbench,
NetExpress, or Mainframe Express:

       environment division.
       special-names.
           call-convention 66 is win32nol.

       working-storage section.
       01  c3comreg-data-length            pic s9(9) comp-5.
       01  c3comreg-data                      pic x(8192).

Procedure Division code:
      * Get the currently logged-on UserName:
                   move spaces                 to c3comreg-data
                   move length of c3comreg-data to c3comreg-data-length
                   call win32nol 'GetUserNameA' using
                       by reference c3comreg-data
                       by reference c3comreg-data-length
                   end-call

      * Get the computer name:
                   move spaces                 to c3comreg-data
                   move length of c3comreg-data to c3comreg-data-length
                   call win32nol 'GetComputerNameA' using
                       by reference c3comreg-data
                       by reference c3comreg-data-length
                   end-call


Parameters
c3comreg-data = Points to a buffer to receive the null-terminated
character string containing the computer name.
c3comreg-data-length = Points to a variable that specifies the maximum
size, in characters, of the buffer. This value should be large enough to
contain MAX_COMPUTERNAME_LENGTH + 1 characters.

Return Values
If the function succeeds, the return value is nonzero and the variable
represented by the length parameter contains the number of characters
copied to the destination buffer, not including the terminating null
character.
If the function fails, the return value is zero. To get extended error
information, call GetLastError.
```
