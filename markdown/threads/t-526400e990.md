I am not really a programer and I'm doing a part time degree in
computing and have to finish of a cobol program by thursday and have
very little time for it as I work as well and have just had an
emergency order for 9 - high quality machines which I have to order
the parts for and build straight away!!
The program I have to write is done on VAX 4000. It's rather
simple, it's basically an accounts program. Required to enter are:
Account number, amount of money and DB or CR (debt or credit). Then it
is required to add records to current file, edit records and read the
current file.
I have done the above but now I have to make the prog sort the
file using the account number as the key field, (some account number
can be the same, but then these are added together later on).
Finally I have to call a report module (seperate program)
which creates a report of the transaction file which includes the date
and the time when the report file was created, then a series of lines
each of which relates to single account, it should report the account
number and the net transaction total derived from all the transactions
on the account (hence above more than one of the same account number).
At the bottom of the report there should be a trailer record
containing the total numer of transactions processed, the total number
of accounts and the net transaction value of the whole file.
I have enclosed a copy of the source code to date.

CONFIGURATION SECTION.

01 Lower PIC A(26) VALUE "abcdefghijklmnopqrstuvwxyz".

IF Choice = 0 OR >7 THEN PERFORM Error-para
END-IF

EVALUATE Choice
WHEN 1 PERFORM Choice1-para
WHEN 2 PERFORM Choice2-para
WHEN 3 PERFORM Choice3-para
WHEN 4 PERFORM Choice4-para
WHEN 5 PERFORM Choice5-para
WHEN 6 PERFORM Choice6-para
END-EVALUATE.

BLINKING BOLD
