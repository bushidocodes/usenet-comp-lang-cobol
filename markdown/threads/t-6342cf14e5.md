For all intents and purposes, Cobol is Cobol--at least the compliant
compilers. So, there really is no such thing as translating from Cobol
to Cobol. If you are talking about mainframe Cobol, Micro Focus and
CA-Realia Cobol can compile that code on a PC-based compiler. If you
want to do PC-based development, there are several compilers that can be
used for that purpose. Did you have something specific in mind?
