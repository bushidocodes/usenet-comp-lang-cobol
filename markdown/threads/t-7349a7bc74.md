I don't quite understand what you have done.

Is the program still a CICS program on the NT?. Or you've ported it to NT,
and converted it to a "batch, non-CICS" program.

In any case, if the program is still a CICS/DB2 program, you cannot compile
it to an exe. It must be a DLL. You use the CICSTCL command to transalte,
compile and link it.

ie... CICSTCL prgname.CCP (-q"SQL('DATABASE dbname')",TEST,DYNAM

if the program is now batch program then the correct command should be

COB2 -g -q"SQL('DATABASE dbname')" prgname.CBL

If you did any of the above, it should work fine


Good luck and keep on COBOLing
