Here is sample source to calculate log to any base in
Micro Focus Cobol:

  identification division.
       program-id. TESTLOG1.
       environment division.
       input-output section.
       file-control.
       data division.
       file section.
       working-storage section.
       77  c1-x       comp-1 value 1000.  *> number
       77  c1-b       comp-1 value 10.    *> base of log
       77  c1-loge-x  comp-1.  *> log base e of x
       77  c1-loge-b  comp-1.  *> log base e of b
       77  c1-logb-x  comp-1.  *> log base b of x (= 3)
       procedure division.
           display "TESTLOG1 starting"
           display "c1-x number   = " c1-x
           display "c1-b base     = " c1-b
           compute  c1-loge-x = function log(c1-x)
           display "c1-loge of x  = " c1-loge-x
           compute  c1-loge-b = function log(c1-b)
           display "c1-loge of b  = " c1-loge-b
           compute  c1-logb-x     =   c1-loge-x / c1-loge-b
           display "c1-logb of x  = " c1-logb-x
           display "TESTLOG1 ending"
           goback
           .

Microfocus Cobol



The LOG10 function returns a numeric value that approximates the
logarithm to the base 10 of argument-1. The type of this function
is numeric.

General Format

FUNCTION LOG10 (argument-1)

Arguments

1.      Argument-1 must be class numeric.
2.      The value of argument-1 must be greater than zero.

Returned Values

1.      The returned value is the approximation of the logarithm to the
        base 10 of argument-1.
2.      Floating-point format is used for numeric non-integer results.


-------------------------------------------------------------

The LOG function returns a numeric value that approximates the
logarithm to the base e (natural log) of argument-1. The type of this
function is numeric.

General Format

FUNCTION LOG (argument-1)

Arguments

1.      Argument-1 must be class numeric.
2.      The value of argument-1 must be greater than zero.

Returned Values

1.      The returned value is the approximation of the logarithm to the
        base e of argument-1.
2.      Floating-point format is used for numeric non-integer results.
