### New CICS removes Translator support for VS COBOL II (and OS/VS COBOL)

- **From:** "William M. Klein" <wmklein@ix.netcom.com>
- **Date:** 2001-12-04T19:11:13+00:00

```
As expected (and stated as the "direction" in the last CICS release), the
latest release of CICS - as announced at:
  http://www.ibmlink.ibm.com/usalets&parms=H_201-354

removes translator support for OS/VS COBOL and VS COBOL II.  The exact
statement (including what to expect NEXT) is:

"Translator support for the earlier (pre-Language Environment) compilers
OS/VS COBOL (5740-CB1, 5740-LM1, and 5734-CB4), VS COBOL II (5668-958, also
5688-023), OS PL/I Version 1 (5734-PL1), OS PL/I Version 2 (5668-910, also
5668-909), and SAA� AD/Cycle� C/370T (5688-216) is withdrawn and is not
available in this release. It is planned that run-time support for OS/VS
COBOL programs (regardless of run-time library used) and for any other
programs running under pre-Language Environment run-time libraries will be
withdrawn from the next release."
```
