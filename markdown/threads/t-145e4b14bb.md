dump the data to text files, transferring the text files to the Oracle
system, and using Oracles SQL*Loader program to load the data into
Oracle. Not a direct conversion, but not too bad & it uses the tools you
already have. (Or should already have unless you got rid of your Wang
before converting the data.)
