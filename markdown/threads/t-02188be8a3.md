Microfocus offers a product (from memory) called co-writer. This is a
report generator. All that your required to do is initially set it up
by describing the file(s), record(s), and field(s) and then you can
query the files for information. Never learned enough about the
extension to know if you could define file relationships such as
one-to-one, one-to-many, so that if you could perform expanded queries
even when the co-writer would have to pull that information from
several files. This if from Microfocus.

A product that I have used extensively that will do everything that I
mentioned (and more) is called Intelligent Query (IQ). This product
runs on most any platform and is simple enough that anyone can create
reports with headers, footers, subtotals, grand totals, etc. just by
selecting fields and building a picture of what you would like the
output page to look like.

Hope this helps,

Tony Heffner
