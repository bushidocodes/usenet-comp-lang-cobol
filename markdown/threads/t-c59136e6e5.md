```
I have created the "following" sample program to show how to use the COBOL
"RANDOM" Intrinsic Function (as this topic comes up from time to time).  I
plan on adding it to the FAQ (or at least a portion of it).  This sample
shows both pseudo-random and "real" random series; how to get integers from 1
thru n; and how to make certain that you don't return "duplicated" values.
Are there any "enhancements" that people suggest that I add to help readers
with this?

NOTE:
  The use of the "multi-digit" variable rather than just sticking with
"two-digit" is due to a "bug" in earlier versions of the compiler that I am
using (name withheld - as I think they have fixed the problem in more recent
versions).  It should be possible to just use two-digit (without the
intermediate MOVE statements) - but this does seem to work.  ALSO, I think I
have some lines past column 72 - if your compiler doesn't accept this, just
change the appropriate variables.

   ****

 Identification Division.
  Program-ID. SampRand.
 Data Division.
  Working-Storage Section.
 01  Misc.
     05  Pseudo      Pic X.
     05  Multi-Digit  Pic 99.9(15).
     05  Two-Digit   Pic 99.
     05  Vary        Pic 99.
 01  Tabl.
     05  Lvl Occurs 52 times.
         10  Elem   Pic X   Value "N".
 Procedure Division.
  Mainline.
     Initialize Misc
     Display "This program will return 10 unique 'random' values"
        with no advancing
     Display " between 1 and 52"
     Display " If you use the ""pseudo-random"" option," with no advancing
     Display " the series will always be the same -"
     Display "     i.e. reproducible"
     Display "Do you want a pseudo or real random series?"
     Display "   Enter 'P' or 'R':" with no advancing
     Accept Pseudo from Console
     Evaluate Function Upper-Case (pseudo)
       When "P"
         Perform Return-Pseudo
       When "R"
         Perform Return-Real
       When Other
         Display " "
         Display "Incorrect reply, program terminated"
     End-Evaluate
    GoBack
      .
 Return-Pseudo.
     Compute Multi-Digit = (Function Random (1) * 51) + 1
     Move Multi-Digit  To Two-Digit
     Move "Y"    To Elem (Two-Digit)
     Display Two-Digit
     Perform
       Varying Vary from 1 by 1
       Until Vary > 9
        Compute Multi-Digit = (Function Random * 51) + 1
        Move Multi-Digit to Two-Digit
        If Elem (Two-Digit) = "N"
           Move "Y" to Elem (Two-Digit)
            Display Two-Digit
        Else
            Subtract 1 from Vary
        End-If
     End-Perform
      .
 Return-Real.
     Compute Multi-Digit = (Function Random
         (Function Integer
           (Function NumVal (Function Current-Date (13:4))))
              * 51) + 1
     Move Multi-Digit  to Two-Digit
     Move "Y"     To Elem (Two-Digit)
     Display Two-Digit
     Perform
       Varying VAry from 1 by 1
       Until Vary > 9
        Compute Multi-Digit = (Function Random * 51) + 1
        Move Multi-Digit to Two-Digit
        If Elem (Two-Digit) = "N"
            Move "Y" to Elem (Two-Digit)
            Display Two-Digit
        Else
            Subtract 1 from Vary
        End-If
     End-Perform
      .
```
