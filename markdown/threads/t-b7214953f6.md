It would be far easier to take your machine to the demonstration.

In sum, you have to create an installation distribution for your
program. Fujitsu provides an installation 'wrapper' which you
incorporate with your application. This 'wrapper' is an OEM version of
InstallShield with much stuff predefined (like required directories).
In addition to your code, the installation process installs maybe 200
DLLs and updates the target machine's registry.

You have no hope of simply copying a few (or many) files to the target
machine - well, there's one way: Install the full PC package on the
demo machine and copy in your PPJ file. Compile and execute. Of course
with Fujitsu's new license scheme this trick won't work.

Sorry.
