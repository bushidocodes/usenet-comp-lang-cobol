› How do I generate a random number with cobol code?

First try:

05 WS-VARIATE COMP-2.
05 WS-SEED PIC S9(9) COMP VALUE 12345.

* initial call
COMPUTE WS-VARIATE = FUNCTION RANDOM(WS-SEED).

* succeeding calls
COMPUTE WS-VARIATE = FUNCTION RANDOM.


If your compiler accepts this, you are home free.

If your compiler doesn't accept this, look up linear congruential
generators in Knuth, or switch compilers.

gen··.@a··.com (Gentlmn10) wrote:

› How do I generate a random number with cobol code?

If you have COBOL with intrinsic functions, the process is easy. I
wrote a routine which assigned a random record number to a given range
of records. I started with the current date and time and turned it
into a seed number by computing the total number of hundreths of
seconds since the beginning of the year (you could prevent any chance
of a repeated seed number by creating a seed number starting with the
total number of days from where IBM assigned its start date -
somewhere in the 1600's I think). Then I generate a random number.
It worked pretty good:

01 WS-WORK-AREAS.
05 WS-GREGORIAN-DATETIME.
10 WS-GREGORIAN-DATE PIC 9(08).
10 WS-GREGORIAN-HR PIC 9(02).
10 WS-GREGORIAN-MN PIC 9(02).
10 WS-GREGORIAN-SS PIC 9(02).
10 WS-GREGORIAN-HS PIC 9(02).
05 WS-INTEGER-DATE PIC 9(09).
05 WS-JULIIAN-DATE PIC 9(07).
05 FILLER REDEFINES WS-JULIIAN-DATE.
10 FILLER PIC X(04).
10 WS-JULIAN-DAY PIC 9(03).
05 WS-SEED-DATE PIC 9(11).
05 WS-TOTAL-NUMBER-OF-RECORDS PIC 9(04).
05 WS-RECORD-NBR PIC 9(04).

MOVE ZEROS TO WS-SEED-DATE
MOVE ZEROS TO WS-INTEGER-DATE
MOVE ZEROS TO WS-JULIAN-DATE
MOVE FUNCTION CURRENT-DATE(1:16) TO WS-GREGORIAN-DATETIME
COMPUTE WS-INTEGER-DATE = FUNCTION INTEGER-OF-DATE(WS-GREGORIAN-DATE)
COMPUTE WS-JULIAN-DATE = FUNCTION DAY-OF-INTEGER(WS-INTEGER-DATE)
COMPUTE WS-SEED-DATE
= WS-JULIAN-DAY * 15120000
+ WS-GREGORIAN-HR * 360000
+ WS-GREGORIAN-MN * 6000
+ WS-GREGORIAN-SS * 100
+ WS-GREGORIAN-HS
COMPUTE WS-RECORD-NBR
= FUNCTION INTEGER(FUNCTION RANDOM(WS-SEED-DATE)
* WS-TOTAL-NUMBER-OF-RECORDS + 1)
.

Hope this helps,
Boyce G. Williams, Jr.
