i had the same prob, those nice folks <g> at fujitsu walked me thru it.
assuming you have your source code in, i'll call the source file
source.cob
ok, in the programming staff screen, click project, click open
lets name the project
project.prj  (remember the .prj)
it asks you "does not exist, create it?" yes
then you are asked for target files, it suggests the name project.exe
click add, then ok
as for dependent files, click browse and find source.cob
click it, click add, click on the box right next to the name source.cob
and click the button that says "main program"
click ok, then when the little project box comes up, click build.
