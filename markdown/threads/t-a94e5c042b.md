```
I was interested to see if anyone has converted from Micro Focus Cobol to
Acucobol? We are in a Unix environment and would like to have a graphical
software package. The current version is character only. It seem to be
very complicated to do this in Micro Focus. Acucobol claims to be able to
have one source for both character and graphical. Acucobol also has the
ability to compile on any platform and the the object to another platform
and run it. This give to greater portibility. Micro Focus can only do
graphical using a package called Dialog, which seems to be very difficult
to use. There also appears to be alot going back and forth between Cobol
and Dialog. Any information would be greatly appreciated.
```
