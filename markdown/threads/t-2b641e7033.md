with an apology, along with the rate information, but most of them have
responded with the typical "BS" line:  The rate depends on experience.

just like there is a salary range for permanent jobs.  Still, these
"narrow minded" recruiters refuse to post the rate range.  Up here in
Seattle, there is a jobs newsgroup, and the same problem exists.
