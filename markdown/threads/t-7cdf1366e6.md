OpenCOBOL uses C as an intermediate step in producing an executable
program. This is not much different from other compilers in that
source code does result in a program that will run.

There is also Tiny-Cobol and Kobol for Linux and/or Windows. A demo
version of Kobol can be found at www.TheKompany.com, it is cheap to
buy.
