Included in the "attached" note is an important ASSUMPTION that you need to
think about (depending on exactly what you intend to do).  Load Modules in
PDS (or PDSE's?) and Program Objects (only in PDSEs) have VERY different
formats.  If you are creating an application that "assumes" Load Modules, at
least be aware that at some time they may change to Program Objects.
