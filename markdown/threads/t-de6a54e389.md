```
* Illustration of sequential file processing in COBOL by Chris
* Mason.
* copyright 1995 by Chris Mason.
* you may copy this freely, but if you included it in a published
* work for money, please ask my permission.
* HCM··.@ix.··m.com
IDENTIFICATION DIVISION.
PROGRAM-ID. PHONESEQ.
AUTHOR. CHRIS MASON.
DATE-WRITTEN. MARCH 10, 1995.
DATE-COMPILED.
INSTALLATION. INTERNET.
ENVIRONMENT DIVISION.
CONFIGURATION SECTION.
SOURCE-COMPUTER. IBMPS2.
OBJECT-COMPUTER. IBMPS2.
INPUT-OUTPUT SECTION.
FILE-CONTROL.
SELECT INPUT-SEQ-FILE
ASSIGN TO "PHONE.TXT"
ORGANIZATION IS LINE SEQUENTIAL
FILE STATUS IS WS-FILE-STATUS.
SELECT REPORT-FILE
ASSIGN TO "REPORT.TXT"
ORGANIZATION IS LINE SEQUENTIAL
FILE STATUS IS WS-FILE-STATUS.
DATA DIVISION.
FILE SECTION.
FD INPUT-SEQ-FILE
DATA RECORD IS INPUT-SEQ-REC.
01 INPUT-SEQ-REC PIC X(80).
FD REPORT-FILE
DATA RECORD IS REPORT-REC.
01 REPORT-REC PIC X(80).
WORKING-STORAGE SECTION.
01 WS-FILE-STATUS PIC X(02) VALUE ZEROES.

01 WS-INPUT-SEQ-REC.
05 WS-LAST-NAME PIC X(15) VALUE SPACES.
05 WS-FIRST-NAME PIC X(15) VALUE SPACES.
05 WS-PHONE-NUMBER.
10 WS-AREA-CODE PIC X(03) VALUE SPACES.
10 WS-PHONE-EXCHANGE PIC X(03) VALUE SPACES.
10 WS-PHONE-EXTENSION PIC X(04) VALUE SPACES.
05 FILLER PIC X(40) VALUE SPACES.

01 WS-LITERALS-CONSTANTS.
05 LI-LINES-PER-PAGE PIC 9(02) VALUE 55.
05 LI-LINES-IN-HEADER PIC 9(02) VALUE 4.

01 WS-COUNTS.
05 WS-INPUT-REC-COUNT PIC 9(06) VALUE ZEROES.
05 WS-REPORT-LINE-COUNT PIC 9(02) VALUE 56.
05 WS-PAGE-COUNT PIC 9(04) VALUE ZEROES.

01 WS-FLAGS.
05 INPUT-EOF-SW PIC X(01) VALUE SPACES.
88 INPUT-EOF VALUE "Y".

01 RPT-HDR-1.
05 FILLER PIC X(01) VALUE SPACES.
05 FILLER PIC X(20) VALUE
"PHONE NUMBER REPORT".
05 FILLER PIC X(05) VALUE SPACES.
05 FILLER PIC X(04) VALUE "DATE".
05 FILLER PIC X(01) VALUE SPACES.
05 RPT-DATE.
10 MM PIC X(02) VALUE ZEROES.
10 FILLER PIC X(01) VALUE "/".
10 DD PIC X(02) VALUE ZEROES.
10 FILLER PIC X(01) VALUE "/".
10 YY PIC X(02) VALUE ZEROES.
05 FILLER PIC X(04) VALUE SPACES.
05 FILLER PIC X(04) VALUE "PAGE".
05 FILLER PIC X(01) VALUE SPACES.
05 RPT-HDR-1-PAGE PIC ZZZ9.

01 RPT-HDR-2.
05 FILLER PIC X(01) VALUE SPACES.
05 FILLER PIC X(15) VALUE
"FIRST NAME".
05 FILLER PIC X(01) VALUE SPACES.
05 FILLER PIC X(15) VALUE
"LAST NAME".
05 FILLER PIC X(01) VALUE SPACES.
05 FILLER PIC X(12) VALUE
"PHONE NUMBER".

01 RPT-DET-1.
05 FILLER PIC X(01) VALUE SPACES.
05 RPT-FIRST-NAME PIC X(15) VALUE SPACES.
05 FILLER PIC X(01) VALUE SPACES.
05 RPT-LAST-NAME PIC X(15) VALUE SPACES.
05 FILLER PIC X(01) VALUE SPACES.
05 RPT-PHONE-NUMBER.
10 FILLER PIC X(01) VALUE "(".
10 RPT-AREA-CODE PIC X(03) VALUE SPACES.
10 FILLER PIC X(01) VALUE ")".
10 FILLER PIC X(01) VALUE SPACE.
10 RPT-PHONE-EXCHANGE PIC X(03) VALUE SPACES.
10 FILLER PIC X(01) VALUE "-".
10 RPT-PHONE-EXTENSION PIC X(04) VALUE SPACES.

01 TIME-AND-DATE-AREA.
05 CUR-TIME.
10 CUR-HOUR PIC 9(02).
10 CUR-MINUTES PIC 9(02).
10 CUR-SECONDS PIC 9(02).
10 CUR-HUNDREDS PIC 9(02).
05 CUR-DATE.
10 CUR-YEAR PIC 9(02).
10 CUR-MONTH PIC 9(02).
10 CUR-DAY PIC 9(02).
05 DSPLY-TIME.
10 DSPLY-HOUR PIC X(02).
10 FILLER PIC X(01) VALUE ":".
10 DSPLY-MINUTES PIC X(02).
10 FILLER PIC X(01) VALUE ":".
10 DSPLY-SECONDS PIC X(02).
05 DSPLY-DATE.
10 DSPLY-MONTH PIC X(02).
10 FILLER PIC X(01) VALUE "/".
10 DSPLY-DAY PIC X(02).
10 FILLER PIC X(01) VALUE "/".
10 DSPLY-YEAR PIC X(02).

PROCEDURE DIVISION.
A0000-MAIN.
PERFORM A1000-BEGIN-HOUSEKEEPING.
PERFORM B0000-PROCESSING-LOOP
UNTIL INPUT-EOF OR
WS-FILE-STATUS IS GREATER THAN ZEROES.
PERFORM A2000-WRAP-UP.
STOP RUN.

A1000-BEGIN-HOUSEKEEPING.
DISPLAY "PROGRAM PHONESEQ BEGINNING EXECUTION"
ACCEPT CUR-DATE FROM DATE.
ACCEPT CUR-TIME FROM TIME.
MOVE CUR-YEAR TO YY
DSPLY-YEAR.
MOVE CUR-DAY TO DD
DSPLY-DAY.
MOVE CUR-MONTH TO MM
DSPLY-MONTH.
MOVE CUR-HOUR TO DSPLY-HOUR.
MOVE CUR-MINUTES TO DSPLY-MINUTES.
MOVE CUR-SECONDS TO DSPLY-SECONDS.
DISPLAY "PHONESEQ RUN DATE = "
DSPLY-DATE.
DISPLAY "PHONESEQ RUN TIME = "
DSPLY-TIME.
OPEN INPUT INPUT-SEQ-FILE.
* ALWAYS CHECK YOUR FILE STATUS, COULD BE A FILEIO ERROR...
IF WS-FILE-STATUS EQUAL ZEROES
CONTINUE
ELSE
DISPLAY "ERROR, INPUT FILE PHONE.TXT = "
WS-FILE-STATUS
END-IF.
OPEN OUTPUT REPORT-FILE.
IF WS-FILE-STATUS EQUAL ZEROES
CONTINUE
ELSE
DISPLAY "ERROR, OUTPUT FILE REPORT.TXT = "
WS-FILE-STATUS
END-IF.
PERFORM X0000-READ-INPUT-SEQ-FILE.

A2000-WRAP-UP.
CLOSE INPUT-SEQ-FILE.
CLOSE REPORT-FILE.
DISPLAY "NUMBER OF INPUT RECORDS = " WS-INPUT-REC-COUNT.
DISPLAY "END OF PROGRAM PHONESEQ".

B0000-PROCESSING-LOOP.
PERFORM B1000-REPORT-PROCESS.
PERFORM X0000-READ-INPUT-SEQ-FILE.

B1000-REPORT-PROCESS.
IF WS-REPORT-LINE-COUNT GREATER LI-LINES-PER-PAGE
PERFORM B2000-RPT-HEADERS
END-IF.
PERFORM B3000-MOVE-RPT-DET-FIELDS.
MOVE SPACES TO REPORT-REC.
PERFORM X1000-WRITE-REPORT-LINE.
MOVE RPT-DET-1 TO REPORT-REC.
PERFORM X1000-WRITE-REPORT-LINE.



B2000-RPT-HEADERS.
ADD 1 TO WS-PAGE-COUNT.
MOVE WS-PAGE-COUNT TO RPT-HDR-1-PAGE.
MOVE RPT-HDR-1 TO REPORT-REC.
PERFORM X2000-WRITE-REPORT-TOP-OF-PAGE.
MOVE SPACES TO REPORT-REC.
PERFORM X1000-WRITE-REPORT-LINE.
MOVE RPT-HDR-2 TO REPORT-REC.
PERFORM X1000-WRITE-REPORT-LINE.
MOVE SPACES TO REPORT-REC.
PERFORM X1000-WRITE-REPORT-LINE.
MOVE LI-LINES-IN-HEADER TO WS-REPORT-LINE-COUNT.

B3000-MOVE-RPT-DET-FIELDS.
MOVE WS-LAST-NAME TO RPT-LAST-NAME.
MOVE WS-FIRST-NAME TO RPT-FIRST-NAME.
MOVE WS-AREA-CODE TO RPT-AREA-CODE.
MOVE WS-PHONE-EXCHANGE TO RPT-PHONE-EXCHANGE.
MOVE WS-PHONE-EXTENSION TO RPT-PHONE-EXTENSION.

X0000-READ-INPUT-SEQ-FILE.
READ INPUT-SEQ-FILE INTO WS-INPUT-SEQ-REC
AT END
SET INPUT-EOF TO TRUE
NOT AT END
ADD 1 TO WS-INPUT-REC-COUNT
END-READ.
IF WS-FILE-STATUS GREATER ZEROES
AND
NOT INPUT-EOF
DISPLAY "SOME PROBLEM ON READ ON INPUT = "
WS-FILE-STATUS
END-IF.

X1000-WRITE-REPORT-LINE.
WRITE REPORT-REC.
IF WS-FILE-STATUS GREATER ZEROES
DISPLAY "SOME PROBLEM ON WRITE TO REPORT = "
WS-FILE-STATUS
END-IF.
ADD 1 TO WS-REPORT-LINE-COUNT.

X2000-WRITE-REPORT-TOP-OF-PAGE.
WRITE REPORT-REC
AFTER ADVANCING PAGE.
IF WS-FILE-STATUS GREATER ZEROES
DISPLAY "SOME PROBLEM ON WRITE TO REPORT = "
WS-FILE-STATUS
END-IF.

```
