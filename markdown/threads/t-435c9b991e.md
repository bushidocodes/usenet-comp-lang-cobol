As a somewhat experinced developer in AcuCobol, I can say that indexed
files is exactly the same on Unix & Dos/windows (we deveplop for both
Unix on Digital Alpha & Windows). To create a sequential file you
could use vutil - unload , but this is only
useful if the indexed file does not have any binary fields. If so you
have to do it the hard by coding a program that reads the indexed file
and writes a sequential file.
