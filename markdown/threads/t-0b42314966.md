```
To send an email from COBOL you could use the Messaging API (MAPI) on Win32
platforms.

The relevant API calls could be coded as follows:-

....

           call WinApi MAPILogon   using by value 0 size 4
                                         by reference wl-profile-name
                                         by reference MapilogonPass
                                         by value MAPI-NEW-SESSION
                                         by value 0 size 4
                                         by reference Mapihandle
                                         returning ws-retcode
           end-call

....

           call WINAPI MAPISendMail   using by value     Mapihandle
                                            by value     0 size 4
                                            by reference mapimsgstruct
                                            by value     0 size 4
                                            by value     0 size 4
                                            returning ws-retcode
           end-call

....

           call WINAPI MAPILogoff   using by value Mapihandle
                                          by value 0 size 4
                                          by value 0 size 4
                                          by value 0 size 4
                                  returning ws-retcode
           end-call

These and the other MAPI calls (together with the parameters) are documented
in the Win32 Software Development Kit.

Hope this helps.
David.
```
