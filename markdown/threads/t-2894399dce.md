Sorry - but you really will have to give more details...

I certainly have seen things like this - usually some run-time error
but in an environment where there is nowhere for the application to
display the message. The application is probably encountering a fatal
error, the program is not trapping the error and thus not reporting it
to the user itself, the COBOL environment would send some sort of
message (often an information window) but in the particular
circumstances this fails, and then the whole thing terminates.

1) is this your program?
2) where (i.e. the developer's or the user's machine) is it failing?
3) is it reproducible?
3) can you (or have you at any time in the past) use(d) Animator to
investigate it?
