Hi Alexander,

I think you need to put a Variant in the first safearray so you need to wrap
the second safearray as a variant. Then put the variant in the safearray.

Your code could be something like:-

ws:-

01 anOLEVAR object reference.
01 ptr-Variant pointer.
01 ptr-array pointer.

01 intIndex1 PIC 9(9) COMP-5.

01 lngresult PIC 9(9) COMP-5.

linkage:-

01 aSafeArray object reference.
01 lnk-Variant variant.

proc div:-
invoke sasecond "getsafearray" returning ptr-array

MOVE VT-BSTR TO vType
ADD VT-ARRAY TO VType
invoke OLEVariant "newwithdata" using vType
returning anOLEVAR
invoke anOLEVAR "getVariantAddress" returning ptr-variant
set address of lnk-variant to ptr-variant
set VARIANT-VT-SafeArray of lnk-variant to ptr-array

MOVE 1 TO intIndex1
INVOKE saFirst "putVariant" USING intIndex1
lnk-variant
RETURNING lngResult
---
Regards
David
