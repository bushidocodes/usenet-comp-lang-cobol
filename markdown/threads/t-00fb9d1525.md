```
Paul wrote:
> 
> a Newbie question
> 
> How can I read a file input and store it into one variable instead of
> store only one line/record.

A READ statement will only read one record from a file.  To read the
whole file you have to have a loop that will READ each record until the
AT END is reached:

       OPEN INPUT MyFile
       MOVE "N"    TO EoF
       MOVE ZERO   TO Store-Index
       PERFORM
           UNTIL EoF = "Y"
           READ MyFile
               AT END
                   MOVE "Y" TO EoF
               NOT AT END
                   PERFORM store-record
           END-READ
       END-PERFORM
       CLOSE MyFile

To store the records create a table in working-storage (use OCCURS) and
each READ increment a subscript and MOVE the record area to the table
item using the subscript.
```
