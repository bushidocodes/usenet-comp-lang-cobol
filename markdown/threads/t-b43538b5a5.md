Don't know when, but you could try IBM's CICS site
http://www.hursley.ibm.com/ The theory concerning the format (this was
discussed at some length not too long ago in comp.software.year-2000) is
to keep it consistent with the mods to the TIME macro, which returns the
date with the same century indicator. If you're writing (or fixing)
code, remember to add 1900 to IBM's CYY to get the real year.
