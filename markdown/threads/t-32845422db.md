According to the manual, a 98,nn error indicates invalid file
structure.  For indexed file it may indicate an inconsistency in the
index structure. For sequential or releative files, it usually
indicates that  the file desciption does not match the organization of
the file. In either case, it may also indicate a hardware read/write
error.
