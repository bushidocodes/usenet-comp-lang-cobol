In case anyone else has this problem I figured id post the answer here. When
compiling the source code with the Fujitsu compiler, you need to add a MAIN
clause to the compiler. This tells the compiler this is a stand-alone
program and isn't called by another one, and so it is compiled accordingly.
Once it is compiled right, there isn't any problem
linking it.
