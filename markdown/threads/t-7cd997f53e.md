Izhar,

If you use the standard rts, you  add C programs to the Cobol runtime
and call them from COBOL.  The microfocus web site has some good info
on calling C programs, and also how to create your rts.

I use Real World Accounting software, and needed to write some C
programs to do Credit Card authorization.  Real World comes with a
custom pre-built rts, so I could not call the C programs directly from
C.  Instead I did a system call to run my C program.  I passed data
between the two using an ASCII file.  Something like:

	Write data into ASCII file
	Close ascii file
	Call "SYSTEM" USING "myprogram"
	Open ASCII file

Paul
