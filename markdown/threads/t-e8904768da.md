Hi Frank,
The example you quote from uses vocabularies, which is a way of defining 
your own syntax in COBOL. OLEREP.CBL doesn't uses the vocabularies - the 
example that does is OLEREPV.CBL (which is just the same program 
demonstrating their use) in the same project.

OLEREP.CBL, as someone else mentions, uses the Word95 Word interface 
("Word.Basic"). While this is perfectly acceptable to Word97, Word97 now 
only documents how to call the old methods using its newer interface, and 
not the old interface. You might want to look at using the new 
"Word.Application" interface.
