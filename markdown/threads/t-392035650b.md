Have you looked at Linux? It is definately a multitasking OS (UNIX like) which
is compatable with Novell. I guess by compatable with Novell, you mean that
it has a client which can use a Novell server. Linux can act as a client to
a Novell server (file & print both). Linux also comes with a function which
is called DOSEMU. That this does is emulate an MS-DOS environment under Linux.
This is supposed to allow you to run DOS (not Windows!) programs without
any modification. However, I don't know if it supposed the "DOS Extenders"
which would run program in protected mode (Phar Lap is a name I remember
which did this).

Now for the usual disclaimer. Although I am a Linux user and lover,
I haven't used this function myself, so I am not familiar with how well it
runs. I would suggest asking that sort of on
comp.os.linux.development.system or comp.os.linux.development.apps or
comp.os.linux.misc or well, you get the idea <grin>.

I hope this was of at least some help,
