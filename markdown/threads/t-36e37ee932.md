You're in luck. I just did this.

Ok, first of all you need to call cobinit() in your C mainline:

/* mycprog.c */
extern  void cobinit();

int main()
{

   cobinit();

   cblprog1();

   return 0;
}

Then, compile with the following flags:

cc -c mycprog.c
cob -x -e "" mycprog.o cblprog1.cbl

(you can use the -l and -L flags like in C to add additional libraries
to compile)

If my memory serves me right, that should make it work for you.

If not, talk to MF customer support. They were able to send me sample
code that worked.

libraries... (But that's another thread)
