Looks like you need to get the JCL for this program off of the
mainframe. Mainframes generally require JCLs to run them....

What it looks like that your mainframe program is defining a card
reader....

SELECT CRD-FILE assign to sys016-ur-2501-s.

Equivalent to a regular text file in logic......

So, for MFCOBOL, just go ahead and make it a text file path with your
data in card form...one card per line of text...
