Try this:  (assumes your isam file is FILE.DAT and FILE.IDX and REBUILD.EXE
located in same directory)

from a DOS prompt do this.  Change to the directory where your files are
located.  Then:

REN FILE.* OLD.*
REBUILD OLD.DAT,FILE.DAT /V/I/E/D

Ok....  try that FILE.DAT and FILE.IDX.    (that'll probably work)

If on the odd chance the problem still persists then try this.   Delete the
FILE.IDX and replace it with a FILE.IDX from a empty FILE.DAT (maybe you can
run your prog and it'll create a new FILE.DAT and FILE.IDX for you with no
records.... if so you can use that FILE.IDX along with the original FILE.DAT
and then repeat the above instructions (the ren and rebuild).

Hopefully that does the trick for ya,
Mike Kinasz
