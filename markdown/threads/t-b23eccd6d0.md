```
Hey, no sense messing around with hints.  I'll tell you flat out, there
ain't no such thing.  In COBOL the record format is entirely defined in
the program.  There is no concept of 'COBOL standardized data formats',
in a global sense.  There are record layouts for specific files in a
given system, and perhaps some shops have standards for storing dates
or address fields.  But no industry-wide COBOL data formats.
```
