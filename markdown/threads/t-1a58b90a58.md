***** IBM BROADENS ARSENAL OF TOOLS FOR YEAR 2000 CHALLENGE *****

http://www.software.ibm.com/ad/news/pr980407.html

As the millennium approaches,and as companies wrap up or gear
up their Year 2000 efforts, IBM continues to offer a broad range
of application development tools for each phase of the Year 2000
conversion process. We've expanded the VisualAge 2000 portfolio
to include unique Millennium Language Extensions (MLE) and the
new Application Testing Collection (ATC). We've also partnered
with our customers to offer Allstate's Millennium Date Compression
Tool (MDCT), and IBM and Bellcore's C and C++ Maintenance and
Test Toolsuite.

In this newsletter, we'll introduce you to these four new tools
and tell you where you can find out more on the Web. And, of
course, we recommend visiting the VisualAge 2000 home page for
the latest information on how IBM Application Development can
help you get ready for the Year 2000:
http://www.software.ibm.com/ad/va2000/.


***** MAKE A DATE WITH MILLENNIUM LANGUAGE EXTENSIONS *****

http://www.software.ibm.com/ad/features/va2kapr.htm

IBM's Year 2000-ready COBOL and PL/I compilers can now be equipped
to help you automatically process windowed dates using IBM VisualAge
Millennium Language Extensions. MLE relies on system resources
not human resources, producing efficient, compiler-generated
windowing code and eliminating the need to manually add logic
for each occurrence of date variables within each program.


***** TESTING, TESTING, Y-2-K *****
***** THE VISUALAGE 2000 APPLICATION TESTING COLLECTION *****

http://www.software.ibm.com/year2000/test0.html.

The enhanced testing capabilities of the new Application Testing
Collection offer an invaluable resource to help reduce and streamline
the testing efforts for MVS/ESA*, COBOL, PL/I Assembler applications
on S/390*. The ATC is part of the VisualAge 2000 Test Solution,
a set of productivity tools that provides a systematic approach
for efficient, high quality Year 2000 testing.

You'll find a factsheet, technology demonstration, user's guide,
and other information about the ATC and the VisualAge 2000 Test
Solution at the Testing Page of the Year 2000 Technical Support
Center.


***** ALLSTATE'S MILLENNIUM DATE COMPRESSION TOOL *****

http://www.software.ibm.com/news/327e.html

Allstate developed the Millennium Date Compression Tool (MDCT)
to find, fix, and report date problems within their primary
application for tracking customer policies and expiration dates.
Using the MDCT, Allstate addressed the Year 2000 issue in their
Alert System Application with a fraction of the allocated budget,
with only a few people, and in only three months. And now Allstate
has teamed up with IBM to make this tool available to customers
who secure consulting and installation services from selected
IBM VisualAge 2000 business partners.


***** IBM & BELLCORE'S C/C++ MAINTENANCE AND TEST TOOLSUITE *****

http://www.software.ibm.com/ad/va2000/xsudpdf.pdf

The IBM C and C++ Maintenance and Test Toolsuite is a a set of
tools designed to help developers and testers of C and C+
applications address software maintenance and testing issues,
including the year 2000 challenge. Packaged in a common, integrated
graphical user interface these tools incorporate xSuds technology
from Bellcore.

A downloadable demonstration version of this software for Unix
(AIX and Solaris) and Windows (Windows NT and Windows 95) will
soon be available at the VisualAge 2000 home page:
http://www.software.ibm.com/ad/va2000/.
