```
Specify that the ORGANIZATION of the output file (which can have a .txt
extension) is LINE SEQUENTIAL (not RECORD SEQUENTIAL - or just SEQUENTIAL).
This should create a "file" that any old Windows text program will let you
read.

There are several different ways to specify "line sequential" output files
in both Fujitsu and Micro Focus.  Therefore, check your documentation for
which method works best for you. (Let us know if you have trouble finding
them.  We would need to know which compiler, which release, and where you
have already looked in your documentation.)
```
