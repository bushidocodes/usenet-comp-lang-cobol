```
Hello,
i found how to access a webpage from powercobol using IE without dde or API calls.
1) Add this code to the REPOSITORY of the form
     CLASS COM AS "*OLE".
2) Then add this code to the event you want
 WORKING-STORAGE SECTION.
 01 IE-APP		PIC X(28) 	VALUE "InternetExplorer.APPLICATION".
 01 IE    		OBJECT REFERENCE COM.
 01 OLE-TRUE        	PIC 1 BIT  	VALUE B"1".
 01 url			PIC X(34)	VALUE "http://ar.geocities.com/pcobolar/".
 PROCEDURE       DIVISION.
     INVOKE COM	"CREATE-OBJECT"	USING	IE-APP	RETURNING	IE
     INVOKE IE 	"SET-Visible" 	USING 	OLE-TRUE
     INVOKE IE 	"NAVIGATE" 	USING 	BY REFERENCE url
     SET    IE     		TO    	NULL

and that's all.
Hope be usefull

Regards from Argentina

Jose Holzmann
http://ar.geocities.com/pcobolar/
```
