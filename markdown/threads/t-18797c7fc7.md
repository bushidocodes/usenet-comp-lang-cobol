> are they Merant or not

> and the introduction of a licence fee (it wasn't always there), 

Actually runtime licences were 'there' from the earliest days of CIS
Cobol and Level II Cobol.  These products could not create executables
that did not rely on a runtime which had to be purchased. 

It was only later, with Cobol/2 on DOS and Windows, that they added the
facilty of building a complete .EXE that incorporated the runtime
modules and thus didn't require separate licences for the stand-alone
runtime.  This also required that the program only used facilities from
the basic compiler package, if any 'Toolkit' or 'OSX' components were
used, or the program was built as .INT or .GNT, then a run-time licence
was required.

So MF runtime licences _were_ always there, but were not required under
specific circumstances for some products.

*NOTE the free Fujitsu version 3 download prohibits distribution
