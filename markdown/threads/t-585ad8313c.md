Warren,

ACCEPT WS-FILE-PREFIX FROM ENVIRONMENT "FILE-PREFIX".

Should get you the current setting of the FILE-PREFIX environment
variable / configuration file setting. 

Check out the callable routine C$FULLNAME, which you can use
to get the full pathname of a file, using the runtime's filename
translation and search logic.  The routine returns the full name of
the corresponding file. Use SET ENVIRONMENT to set the output
file's name and you should be there!


Best regards

Nigel
