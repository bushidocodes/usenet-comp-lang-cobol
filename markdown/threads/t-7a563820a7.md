The usual problem with NetExpress V3.1 in Vista is the installation
portion. Before installing the MF NetExpress v3.1 you have to select
'compatibility mode' already to be able it to setup properly (using
the IE browser).

Compatibility mode means you have to run the setup (via browser) by
allowing legacy applications to run... it is then that it will install
the MF NetExpress. Once installed then you're ready to go.

Here is the help site;
http://windowshelp.microsoft.com/Windows/en-US/help/bf416877-c83f-4476-a3da-8ec98dcf5f101033.mspx
