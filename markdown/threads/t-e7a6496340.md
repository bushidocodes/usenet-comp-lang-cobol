Take a look at the EnumWindows function. It enumerates all top-level
windows returning a window handle. You have to write a callback function
to do something with the window handle, like convert it to a file name,
but there's an article in the knowledge base on how to do that. The
sample program pview.c has all the code needed. Good luck wading through
it. I gave up after the first few hundred lines.
