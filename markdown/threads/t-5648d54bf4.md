       IDENTIFICATION DIVISION.
       PROGRAM-ID. Y2KCONV.
      *AUTHOR. SHINE.

      *-----------------------------------------------------------------
      * CONVERT A SIX DIGIT DATE WITH A FORMAT OF 'YYMMDD' TO AN
      * EIGHT DIGIT DATE FORMATED 'CCYYMMDD' USING A SLIDING WINDOW
      * TECHNIQUE AND A COMPUTED PIVOT.
      *-----------------------------------------------------------------

       ENVIRONMENT DIVISION.
       INPUT-OUTPUT SECTION.
       FILE-CONTROL.
       DATA DIVISION.
       FILE SECTION.
       WORKING-STORAGE SECTION.

      *-----------------------------------------------------------------
      * THE FIELD PIVOT-NUM-YEARS CAN BE POSITIVE (+) OR NEGATIVE (-).
      * IF THE PIVOT IS NEGATIVE, THE BEGINNING CENTURY OF THE WINDOW
      * WILL BE THE PREVIOUS CENTURY AND THE END CENTURY WILL BE THE
      * CURRENT CENTURY. IF THE PIVOT IS POSITIVE, THE BEGINNING
      * CENTURY OF THE WINDOW WILL BE THE CURRENT CENTURY AND THE END
      * CENTURY WILL BE THE NEXT CENTURY.
      *
      * FOR EXAMPLE USE -18 TO DERIVE THE CENTURY FOR A CURRENT OR
      * PREVIOUS EMPLOYEE WHERE AGE REQUIRE FOR EMPLOYMENT IS 18.
      *
      * MOST USES WILL SPLIT THE CENTURY AND USE +50. IF THE CURRENT
      * DATE IS 1998 THEN YEARS 48 THRU 99 WILL BE 19TH CENTURY, YEARS
      * 00 THRU 47 WILL BE 20TH CENTURY.
      *-----------------------------------------------------------------

       01  PIVOT-NUM-YEARS    PIC S9(02) VALUE +50.

       01  PIVOT-DATE VALUE ZEROES.
           05 PIVOT-CCYY-GRP.
              10  PIVOT-CC    PIC 9(02).
              10  PIVOT-YY    PIC 9(02).
           05  PIVOT-CCYY REDEFINES PIVOT-CCYY-GRP PIC 9(4).
           05  PIVOT-MM       PIC 9(02).
           05  PIVOT-DD       PIC 9(02).

       01  WRK-DATE           PIC 9(08).
       01  WRK-DATEX REDEFINES WRK-DATE.
           05  WRK-CC         PIC 9(02).
           05  WRK-YY         PIC 9(02).
           05  WRK-MM         PIC 9(02).
           05  WRK-DD         PIC 9(02).
       01  FILLER REDEFINES WRK-DATE.
           05  WRK-CCYY       PIC 9(04).
           05  FILLER         PIC 9(04).
       01  FILLER REDEFINES WRK-DATE.

           05  CUR-YYMMDD     PIC 9(06).

           05  DOB-DD         PIC 9(02).

       01  AGE                PIC S9(03) VALUE 0.
       01  E-AGE              PIC ZZ9-.

       PROCEDURE DIVISION.

      *-----------------------------------------------------------------
      * SET CURRENT DATE USING A FIXED WINDOW
      * This is demo code only. I use MVS using COBOL II and quite
      * frankly don't know how to get a date from the OS that has a
      * century. We use a date from a date card in all our aps anyway.
      *-----------------------------------------------------------------
           ACCEPT CUR-YYMMDD FROM DATE.
           IF CUR-YY > 50
               MOVE 19 TO CUR-CC
           ELSE
               MOVE 20 TO CUR-CC
           END-IF.

           DISPLAY 'CUR DATE= ' CUR-DATE

      *-----------------------------------------------------------------
      * DERIVE CENTURY USING A VARIABLE WINDOW
      *-----------------------------------------------------------------

           MOVE -18 TO PIVOT-NUM-YEARS.
           PERFORM SET-PIVOT-DATE.

           MOVE '990717' TO DOB-DATE.
           MOVE DOB-DATE TO WRK-YYMMDD.
           PERFORM DERIVE-CENTURY.
                                     PERFORM COMPUTE-AGE.
           DISPLAY
               '     DOB = ' DOB-MM '/' DOB-DD '/' WRK-CC DOB-YY
               '  AGE=' E-AGE.
           GOBACK.

       SET-PIVOT-DATE.

           MOVE CUR-DATE TO PIVOT-DATE.
           ADD PIVOT-NUM-YEARS TO PIVOT-CCYY.

       DERIVE-CENTURY.

           IF WRK-YY > PIVOT-YY
               COMPUTE WRK-CC = PIVOT-CC - 1
           ELSE
               IF WRK-YY = PIVOT-YY
                   IF (WRK-MM > PIVOT-MM)
                   OR (WRK-MM = PIVOT-MM AND WRK-DD > PIVOT-DD)
                       COMPUTE WRK-CC = PIVOT-CC - 1
                   ELSE
                       MOVE PIVOT-CC TO WRK-CC
                   END-IF
               ELSE
                   MOVE PIVOT-CC TO WRK-CC
               END-IF
           END-IF.

       COMPUTE-AGE.

           COMPUTE AGE = CUR-CCYY - WRK-CCYY.
           IF (DOB-MM > CUR-MM)
           OR (DOB-MM = CUR-MM AND DOB-DD > CUR-DD)
               COMPUTE AGE = AGE - 1
           END-IF.
           MOVE AGE TO E-AGE.

Hope this helps.
