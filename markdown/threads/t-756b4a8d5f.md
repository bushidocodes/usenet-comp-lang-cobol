ODBC driver to access Vision files) runs only under a Windows environment
(NT or 9x).  In addition, since the data files are one a UNIX system you
would need the product Acuserver which allows the AcuODBC driver to access
Vision files on a UNIX server acting as a file server.  If you have any
additional questions on this feel free to contact Acucorp Systems
