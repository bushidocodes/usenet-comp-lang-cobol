I don't use Dialog System - so wouldn't know where to start. Hence my suggestion
about Answer Exchange - where some developer *must* have already done it, or the
Micro Focus tech people can give you some ideas.

As background, have you taken a look at Type Library Assistant from the IDE ?
You can select MS Excel and it will generate a HUGE copy file with all data
types necessary, plus methods applicable to Excel. (It's a fair bet many of the
methods will be applicable to other MS products like Word - e.g., "OpenFile",
"CloseFile", "SaveAs"  etc.).

I did a quick search and found references to  :-

X1CopyPictureFormat
XlChart.....etc....

Watch out when you are searching -  those "X's" above are in fact  :-

"XOne" and "XL"  - and you will find accompanying methods using the data types.
