Hi,

AcuCOBOL comes with a utility called "vutil" (vutil32.exe on Windows).
Use this utility to dump the data into text files using 

vutil -extract {filename} > {textfilename}

If you don't have a copy of vutil you should be able to run it on your clients 
computer who should have it.

Regards,

Ray Smith
