Hi,

Could anyone please tell me if I could find a subprogram to display
images in the file formats of *.TIF, *.JPG, *.BMP. I need to call
this subprogram from within my COBOL application that I have developed
for a client who would like to see the photograph, related to each
data record. The ideal routine would be one, which can be statically
linked to the COBOL program, accepting the image file name, the image
file type, window size of the image, etc. as parameters. It may also
be a *.DLL file that I can dynamically call from within my program.

I am using the Microfocus Cobol Compiler Version 3.2.37 with MS
Windows 95 and my application runs on Novell network.

The subprogram may have been written in any language (C, C++,
Assembler, etc.). I would appreciate it very much if you could give me
a code fragment, completeley illustrating how the subprogram is being
called from within the COBOL program as well as the declarations of
the data names and/or constants, used as call parameters. Further, I
would, of course, be happy if that subprogram happens to be a
freeware one.

Thanks hell of a lot in advance for the reply.

Enver
