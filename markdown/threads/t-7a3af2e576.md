I am using NetExpress 3.0 and DIALOG 3.1 to create an application.  Part of
the application generates a report and saves the file to disk and allows the
user to view it.  I am using the fontdlg class to popup the font box so that
the user can change the font, but what I really want to do is limit the user
to only selecting monospaced fonts (things don't line up correctly if they
use proportional fonts).  Thus far I get all of the available fonts in the
font dialog box, but I know there must be some way to limit this to just the
monospaced fonts because in NetExpress itself they do this (in the menu
choose Options/Customize/Fonts/Change).  If anyone knows how to do this I
would really appreciate it.  Thanks in advance.

Bill

The fontdlg class doesn't provide quite enough methods you need, but there 
is a solution - copy the class source (fontdlg.cbl) and create your own, 
near-identical class (called, say, fontdlg2; don't forget to change the 
class-control section entry), then in the "createWindow" method add 
cf-fixedpitchonly to flags of chFont.
I think this should do it, anyway.

Steve.
