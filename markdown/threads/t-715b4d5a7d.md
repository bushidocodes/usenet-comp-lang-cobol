16 bit Realia WINDOWS programs work under Windows 2000 - but the file
locking has problems because of netbios.

You have to have a special Identification division and linkage section
to compile them as Windows programs, but you might try that.

Here's the key:

 IDENTIFICATION DIVISION.
 PROGRAM-ID. 'P_Winmain'.


*****************
 LINKAGE SECTION.
*****************

 01  INSTANCE			   PIC 9(4) BINARY.
 01  PREV-INSTANCE		   PIC 9(4) BINARY.
 01  CMD-LINE			   PIC X(120).
 01  CMD-SHOW			   PIC 9(4) BINARY.

********************
 PROCEDURE DIVISION
********************

	  USING BY CONTENT INSTANCE PREV-INSTANCE
	  BY REFERENCE CMD-LINE
	  BY CONTENT CMD-SHOW.


If you need a link bat - try LINKWINM.BAT.

Still keep me posted.  This is not good.
