```
You can't do this in one run, but it is easy in two.
In the first one use

RECORD TYPE=V,LENGTH=xxx
OUTREC FIELDS=(1,16447)

Then run the following on the output

SORT FIELDS=COPY
RECORD TYPE=F,LENGTH=16447
OUTREC FIELDS=(5,16442)

As long as there are no records shorter than 16442 in your file, this will
work
```

```
Alternatively (according to one of my "usually reliable sources" <G>),

   ***

  To get this to work with DFSORT (and probably Syncsort), change the OUTREC
statement to:

     OUTFIL OUTREC=(5,16442),CONVERT

CONVERT tells DFSORT to change from VB to FB.
```
