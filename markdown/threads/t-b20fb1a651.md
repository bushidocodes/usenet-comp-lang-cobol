As soon as the above condition is met the loop exits
The following move will not place a value where
you expect as it will not execute.

>         MOVE W4-Port-Num TO W5-Port-Num(W-User-Count, W-Log-Count)

Default for PERFORM is a "TEST BEFORE" you would have to change
the PERFORM to

	PERFORM TEST AFTER VARYING W-Log-Count FROM 1 BY 1
	  UNTIL W5-Port0Num(W-User-Count W-Log-Count = 'X'

And even then it might not work

The code 

perform varying W-Log-Count from 1 by 1 
  until W-Log-Count > 'table size limit'
    if W5-Port-Num(W-User-Count W-Log-Count) = 'X'
      move W4-Port-Num to W5-Port-Num(W-User-Count W-Log-Count)
     exit perform
   end-if
