just a guess: you recently installed a new version of the compiler
and the new compiler is somehow reading a file from the old
compiler's installation. check your COBDIR environment variable.

remember, the mf rts is doing a lot of work in the background
that includes reading it's own datafiles, one of these is probably
the culprit.

Assuming it is in your code:
you can begin to debug it by using the divide and conquer method.
put a goto in the program that jumps to the end of the program,
keep moveing the goto up in the program until the error goes away.
