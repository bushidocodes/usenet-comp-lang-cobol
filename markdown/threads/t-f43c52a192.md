I would suggest a different approach. Try hooking up a PC with a
terminal emulator program to the TI system. Set the terminal emulator
program to log everything to a file. Then enter a command to copy the
data files to the screen. Emulator logs to disk. Edit the file on the
PC using a hex editor to eliminate any echoed commands. Then transfer PC
file to your new system.

Some things that may complicate the situation might be cr-lf treatment
(UNIX might be different enough to matter) and binary characters in the
data if it has any COMP field data in it. You may have to play with
this, but it should work. If it's all display format data it is almost
guaranteed to work.

JR
