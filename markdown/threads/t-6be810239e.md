### BIT Manipulation

```
To set a bit add a power of 2, eg to set bit 3 (counting from the right):
ADD 8 TO BIT-PATTERN
This only works if it is not set already, so to test if bit 3 is set:
DIVIDE BIT-PATTERN BY 16 GIVING JUNK REMAINDER LOW-BITS
(where 16 is just the next higher bit value)
IF LOW-BITS > 7
bit 3 is set
ELSE
bit 3 is off
```
