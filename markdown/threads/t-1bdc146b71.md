You can walk the registry to find this, but any one component
is likely a collection of class ids.  For instance, if you want to
know if BestOfWare's SmartUI is installed, look in

HKEY_CLASSES_ROOT for BOWSmartUI.SmartUI.  That means
that this name is registered.  If COM is asked to instance an
object of this type, it will first come here.  It will grab the
CLSID (from the CLSID subkey) and use that GUID to lookup
details of the object's implementation.
