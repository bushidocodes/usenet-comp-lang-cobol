Long time has passed since the promised date.
If you are not afraid to tweak the registry, the solution is very simple:

REGEDIT4

[HKEY_USERS\.DEFAULT\Software\Micro Focus\NetExpress\3.1\IDE\Ide]
"Recent File List"=dword:00000005
"Recent App List"=dword:00000006


The surest method is to run "regedit" from the Run menu
open the above mentioned place
right click on the right pane
select new DWORD
rename  into Recent App List
and modify into the value you wish.

Otherwise, for Win98 users, copy the above code into a text document
you must have a blank line at the end.
give the text .reg extension
dblclick it
I am not sure about other version of WINDOWS registry location, but if you are 
familiar with registry manipulation, you can adapt the reg file accordingly.

Luck

Ilan
