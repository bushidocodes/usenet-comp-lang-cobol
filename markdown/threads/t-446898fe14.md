```
Per the Fujitsu Website:

Fujitsu COBOL V4 will not install on the Microsoft Windows 2000 operating
system. If you attempt to do this you will receive the following message:
"NT Service Pack 3 must be installed before Fujitsu can be installed"


Go to http://msdn.microsoft.com/compatibility/ where you should be able to
find the "Application Compatibility Toolkit".


Download the Toolkit and go from there. The file at time of writing is
named act.exe. After installing it, it will likely enable you to run several
programs that may not otherwise be compatible with Win 2000.

If this does not work, you will have to purchase Version 6.1. If you are a
student, there are academic licenses available.
```
