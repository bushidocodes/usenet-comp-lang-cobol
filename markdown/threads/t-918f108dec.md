If you are using the Server Express product, you can use the DB2 directive
to invoke the MF DB2 precompiler with COLLECTION option.  The Database
Access manual should have details.

If you are using an older MF product, try the SQL directive with COLLECTION
option.  There should be a DB2 readme that details syntax for new directives
in the mf install directory.

IBM also supplies a precompiler that is documented in the Command reference
under PRECOMPILE.

In general MF maps the IBM precompiler directives using the similar
syntax/options.
