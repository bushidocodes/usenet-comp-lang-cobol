The error means that the file you named does not exist. Since you are
opening as an input file, it must be there.  Make sure that it is in the
same area as your EXE program when you run it, or change your select clause
to reflect a fully qualified file name.

If this is a master file, and you are writing a program to update it, then
you must have an error handling routine for the first time that you run the
program. In effect, when you encounter a "file not there" error, then you
must open the file in output mode to originally created the file.  Once the
file exists, then you can open it in Input or in I/O mode.
