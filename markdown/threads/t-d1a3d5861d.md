```
Someone asked for a "Calendar generator" relatively recently in this NG.  I
remembered that the Report Writer "product" that is available for several
compilers (and which includes both extensions to Standard RW - and features
from the 2002 Standard) had such a program.  I have now received permission
to post this code to the NG.  I hope that whoever asked the original
question is still watching.  (I can't find the old post).  Please feel free
to use or pass this code on.  For information on the RW product that uses
this as their IVP (Installation Verification Program) see:

   http://www.spc-systems.com/

        ***

       IDENTIFICATION DIVISION.
       PROGRAM-ID.  CXRIVP01.
       ENVIRONMENT DIVISION.
       CONFIGURATION SECTION.
       SOURCE-COMPUTER.  IBM-370.
       OBJECT-COMPUTER.  IBM-370.
       INPUT-OUTPUT SECTION.
       FILE-CONTROL.
           SELECT PRINT-FILE ASSIGN TO UT-S-CALENDAR.
       DATA DIVISION.
       FILE SECTION.
       FD  PRINT-FILE
           LABEL RECORDS OMITTED
           RECORDING MODE IS F
           REPORT IS CALENDAR.
       WORKING-STORAGE SECTION.
       01  WORKING-LOCATIONS.
           05  TODAYS-DATE.
             07  YEAR-NO            PIC 99.
             07  FILLER             PIC X(4).
           05  START-DAY            PIC 99 COMP.
           05  MONTH-NO             PIC 99 COMP.
           05  DAY-COUNT            PIC 99 COMP.
           05  LEAP                 PIC 9  COMP.
           05  TEMP                 PIC S9(9) COMP.
           05  MONTH-NAME-TABLE.
             07  FILLER             PIC X(54)  VALUE
           " JANUARY  FEBRUARY   MARCH    APRIL      MAY     JUNE".
             07  FILLER             PIC X(54)  VALUE
           "   JULY    AUGUST SEPTEMBER OCTOBER  NOVEMBER DECEMBER".
           05  FILLER REDEFINES MONTH-NAME-TABLE.
             07  MONTH-NAME         PIC X(9)  OCCURS 12.
           05  MONTH-DAYS-TABLE     VALUE "312831303130313130313031".
             07  DAYS-IN-MONTH      PIC 99  OCCURS 12.
           05  DAY-TABLE.
             07  DAY-NUM            PIC 99 COMP   OCCURS 42.
       REPORT SECTION.
       RD  CALENDAR
           PAGE LIMIT 60    FIRST DE 7.
       01  TYPE PH  LINE 2.
           05  COL 50           VALUE "CALENDAR FOR".
           05  COL + 2          VALUE "19"  WHEN YEAR-NO > 50
                                      "20"  WHEN OTHER.
           05  COL 65           PIC 99    SOURCE YEAR-NO.
       01  MONTH-BLOCK TYPE DE  REPEATED 4 TIMES EVERY 30 COLUMNS.
         03  LINE + 4  COL 6    PIC X(10)   MONTH-NAME (MONTH-NO).
         03  LINE + 2  COL 1    VALUE "  S  M  T  W  TH F  S ".
         03  LINE OCCURS 6      VARYING SUNDAY FROM 1 BY 7.
           05  COL 2  OCCURS 7  STEP 3  VARYING DAYNO FROM SUNDAY
                      PIC Z9    DAY-NUM (DAYNO)   BLANK WHEN ZERO.
       01  TYPE PF.
         03  LINE.
           05  COL 35           "PRODUCED USING COBOL REPORT WRITER ON".
           05  COL + 2          PIC <X(9)B<99B9999   FUNCTION MDATE.
       PROCEDURE DIVISION.
       CALENDAR-START.
           ACCEPT TODAYS-DATE FROM DATE
           DIVIDE 4 INTO YEAR-NO GIVING TEMP REMAINDER LEAP
           COMPUTE TEMP = (1461 * YEAR-NO) / 4
           IF   LEAP = 0 MOVE 29 TO DAYS-IN-MONTH (2)
           ELSE ADD 1 TO TEMP.
           DIVIDE 7 INTO TEMP GIVING TEMP REMAINDER START-DAY
           OPEN OUTPUT PRINT-FILE
           INITIATE CALENDAR
           PERFORM PRINT-MONTH VARYING MONTH-NO FROM 1 BY 1
           UNTIL MONTH-NO > 12
           TERMINATE CALENDAR
           CLOSE PRINT-FILE
           STOP RUN.
       PRINT-MONTH.
           MOVE LOW-VALUES TO DAY-TABLE.
           PERFORM STORE-DAY VARYING DAY-COUNT FROM 1 BY 1
           UNTIL DAY-COUNT > DAYS-IN-MONTH (MONTH-NO)
           GENERATE MONTH-BLOCK
           DIVIDE 7 INTO START-DAY GIVING TEMP REMAINDER START-DAY.
       STORE-DAY.
           ADD 1 TO START-DAY
           MOVE DAY-COUNT TO DAY-NUM (START-DAY).
```
