MS COBOL was "MicroSoft COBOL" which was actually Micro Focus COBOL with
a different development/testing platform. The last version of MS COBOL
was v5.0 - after that, MicroSoft paid-off Micro Focus to pickup all
support of the product with the exception of the Microsoft "only" tools
which were bundled with the software.

A quick trip to http://www.microfocus.com/cgi-bin/Samples will yield a
plethora of sample code for your perusal.

In a word MS COBOL is "dead", but COBOL is far from dead...
