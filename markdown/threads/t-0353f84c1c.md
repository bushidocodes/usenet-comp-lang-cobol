depends on compiler;
some have a CLEAR SCREEN option on the DISPLAY verb

if you have ANSI.SYS loaded you can use an Escape "[..." clear or
scroll etc. check you ansi code lists Escape "[2J" comes to mind.
ESCape is decimal 27...drop it into a hex field.
some compilers allow you to DISPLAY x"27" "[2J"
the variations are almost endless...

that IBM introduced late last year, then it is NOT a "Micro
