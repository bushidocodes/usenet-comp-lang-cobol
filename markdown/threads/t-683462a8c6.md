I evaluated Realia CICS for my company back in 1992, when the current
release of Realia COBOL was 4.2, I believe.  We weren't able to
emulate the way our CICS production application works, because it had
conversational tasks and used the EXEC CICS START command.  But for
more typical CICS applications involving BMS maps,
pseudo-conversational programs and VSAM files it was quite good at
emulating CICS in a single user mode.  I also like the screen painter
for BMS maps.  It was better than what we were using on the mainframe.

I would imagine that the current release of Realia CICS would have
more features.  Back in 1992, I was running it in an OS/2 command line
window.  I think it would just fit in a DOS box under Windows.

My impression at the time was that Realia CICS was a good option for
off-loading CICS development and unit-testing to a PC.  

I really liked the Realia COBOL compiler, and wish I could afford one
for home use.
