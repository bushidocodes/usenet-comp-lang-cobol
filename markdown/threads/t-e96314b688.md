Oracle themselves have only certified Oracle 9i with Server Express 2.0.11
SP1 (or later) on UNIX, and NX 3.1 on Win32. That's not to say that earlier
versions of Server Express will or won't work, but there's no guarantees ...

Which flavours of UNIX are you working on? You may want to consider
upgrading to the latest release of Server Express. The Micro Focus UNIX
product matrix at
http://supportline.microfocus.com/productreleaselevels/unix.asp shows the
latest available versions for each platform.

Simon.
