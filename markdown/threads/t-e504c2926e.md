The FAA has many systems which use mainframes. The system
that has proven difficult to upgrade is the Air Traffic Control system.
The new system was/is called Advanced Automation System (AAS).

AAS was initiated in 1983. IBM FSC won the bid and started in
1988. The system was to be implemented on distributed computers
using both 390 and RS/6000 computers. The SLOC estimate was
2.4 million, of which 1.8 million was Ada and much of the rest C++.
(FSC was later sold to Loral.)

In 1994, the AAS project was restructured. More than half the
original project was cancelled. Most of what remained was the
workstation software for the air traffic controllers. (For $7.6B,
they should have pretty displays to show that all was not wasted!)

Some of the references, on the web, mention AAS (along with
many other government projects) as: What not to do!

Search using the keywords FAA and AAS. There are several
"hits" for both keywords where AAS does not refer to the project.

It is likely the source code is proprietary; hence not available under
the Freedom of Information Act.
