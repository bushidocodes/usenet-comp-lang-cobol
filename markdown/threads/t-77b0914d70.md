I think you are running Realia cobol.
There was indeed such a bug, when the compiler tries to
compute how many lines per minute it compiles. The
bug happens if that number exceeds 65,535. It is easy
to fix by hacking the binary compiler. You can download the
patched compiler module (REALCOB.P) from
http://www.etk.com/products/etkr16/index.htm
then click on fixes.zip, unzip the file into your Realia directory.
If you like, rename the old REALSCOB.P first to
something else.
