Like this.

01 TEMP-SWITCH PIC 9.
	88 FIRST-VALID	VALUE 0.
	88 SECOND-VALID VALUE 1.

then

01 TEMP-SWITCH-TABLE.
	02	FIRST-ONE 	PIC X(12) VALUE "FIRST-VALID".
	02	SECOND-ONE	PIC X(12) VALUE "SECOND-VALID".

then

01 OTHER-TS-TABLE	REDEFINES TEMP-SWITCH-TABLE.
	02	VALID-STRING	PIC X(12) OCCURS 2 TIMES.

then in the procedure div

	DISPLAY VALID-STRING (TEMP-SWITCH).

Presumably if you don't want to be doing loads of if statements, you have
a lot more than two valid values, otherwise this would be a lot of
unnecessary work.
