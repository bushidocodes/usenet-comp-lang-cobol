The following code run fine only under

 Identification Division.
 Program-Id. COBMAIN1.
******************************************************************
* 06/98 First version                                            *
******************************************************************
 Environment Division.
 Configuration Section.
 Data Division.
 Working-Storage Section.
 01                      pic x(12)       value "**Working**".
 01 CUPRO-Dynamic        pic x(8)        value "CUPRO1".
 01 Result               pic x(20)       value Spaces.
 Linkage Section.
 Procedure Division.
     Display "CobMain1:This is the Cobol-Main-Program and we test"
     Display "CobMain1:the inter language communication"
     Display "CobMain1:=========================================="
     Display "CobMain1:1. Dynamic call of C-Program CUpro1"
     Call  Cupro-Dynamic using Result
     Display "CobMain1:   back from C-Program CUpro1"
     Display "CobMain1:   Feedback:" Result
     Move Zero to Return-code
     Goback
     .
#include <stdio.h>
#include <string.h>
/* the entry point (Binder) have to set to CUpro1 */
void CUpro1(char *pBuffer)
{                                                                   
                                                                    
   strcpy( pBuffer, "CUpro1 RC=0");                                 
   fprintf(stderr,"Cupro1  :   --> Dynamic call of CUpro1 done\n"); 
}                                                                   

On older C-compiler you need a special #pragma
