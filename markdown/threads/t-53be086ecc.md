Several people have responded to my posting a few routines
on the net. This prompts me to post the following:

The ETK tool set contains several general routines written
in portable cobol (will run anywhere with any compiler).
I've prepare a complete package of these routines with
documentation. The package is too long to post on the
net, but can be had by request (e-mail to le··.@i··.net or
lsv··.@i··.net (that's me too)). After Easter, you can
download the stuff (zip file) from our web home page.

Here is a short description of the routines:


Package: Purpose:

ARITPK Provides the four standard arithmetic
operations on large numbers with 18-
digit accuracy for multiplication and
division and 36-digit accuracy for
addition and subtraction, as well as
conversion to/from a portable floating
point format.

CALCPK Computes the value of an expression.
The expression can contain decimal
numbers, arithmetic operators,
mathematical functions, parentheses,
and user-defined variables. This
package is developed as a test case in
chapter 5.

CALRPK Converts dates between various formats
(Gregorian, Julian, Lillian), and
computes the day of the week, and week
of the year.

CERTPK Converts a number or date into full
text in a number of European languages,
including English, French, German,
Spanish, Italian, and Roman numerals.

COMPPK Compressed/expands arbitrary binary
data. Will compress/expand piece-meal
if needed.

DATEPK Converts dates between internal
(YYYYMMDD) and external (displayable)
formats. Will format dates using the
desired national conventions.

MATHPK Computes various mathematical functions
such as square root, PI, and tangent.
All results have 18 digits of
precision; 9 before and 9 after the
decimal point.

STRNPK Performs various string-manipulation
functions, such as concatenate,
substring, and get next word.



Leif Svalgaard
