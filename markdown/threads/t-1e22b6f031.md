```
It looks to me as a study in perform varying. Redefine the name as a table
and define an output table. Go through inputtable and move input element to
output element if the value suits you. Remember to have a separate index to
the output table (Start with 1 and increase only when you move to output).

the logic.
```

```
If you want to replace non alphabetic characters with spaces:

  01  TXT     PIC  X(30).
  01  X       PIC  9(02) COMP.

  PERFORM VARYING X FROM 1 BY 1 UNTIL X > FUNCTION LENGTH(TXT)
      IF TXT(X:1) ALPHABETIC
          MOVE FUNCTION LOWER-CASE(TXT(X:1)) TO TXT(X:1)
      ELSE
          MOVE SPACE TO TXT(X:1)
      END-IF
  END-PERFORM

If you want to shift following text to the left when non alphabetic
characters are found:

  01  TXT     PIC  X(30).
  01  X1      PIC  9(02) COMP.
  01  X2      PIC  9(02) COMP.

  MOVE 1 TO X2
  PERFORM VARYING X1 FROM 1 BY 1 UNTIL X1 > FUNCTION LENGTH(TXT)
      IF TXT(X1:1) ALPHABETIC
          MOVE FUNCTION LOWER-CASE(TXT(X1:1)) TO TXT(X2:1)
          ADD 1 TO X2
      END-IF
      IF X1 > X2
          MOVE SPACE TO TXT(X1:1)
      END-IF
  END-PERFORM
```
