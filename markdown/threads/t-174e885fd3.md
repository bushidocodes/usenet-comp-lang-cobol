move function upper-case(my-input) to my-input
evaluate my-input
    when '0' thru '9'
        compute my-index = function ord (my-input) - function ord ('0') + 1
   when 'A' thru 'I'
        compute my-index = function ord (my-input) - function ord ('A') + 11
  when 'J' thru 'R'
        compute my-index = function ord (my-input) - function ord ('J') + 20
   when 'S' thru 'Z'
        compute my-index = function ord (my-input) - function ord ('S') + 29
   when other
        display 'Bad input ' my-input
end-evaluate

This should be insensitive to enviroment and, therefore, portable.

-----
      $set ans85 flag"ans85" flagas"s"
       identification division.
       program-id. ltr2num.
       environment division.
       configuration section.
       object-computer. cobol-85
           sequence alphanumeric-subscript.
       special-names.
           alphabet alphanumeric-subscript "123456789"
                "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
                "abcdefghijklmnopqrstuvwxyz"
           class valid-ltr "123456789"
                "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
                "abcdefghijklmnopqrstuvwxyz".
       data division.
       linkage section.
       01  ltr pic x.
       01  num pic 99.
       procedure division using ltr num.
       begin.
           if ltr not valid-ltr
               move 0 to num
           else
               compute num = function ord (ltr)
           end-if.
-----

----- Test program
      $set ans85 flag"ans85" flagas"s"
       identification division.
       program-id. testl2n.
       data division.
       working-storage section.
       01  ltr pic x.
       01  num pic 99.
       01  x binary pic 9(4) value 0.
       procedure division.
       begin.
           perform
           varying x from 1 by 1
           until x > 256
               move function char (x) to ltr
               call "ltr2num" using ltr num
               if num not = 0
                   display space quote ltr "=" num quote space
                   no advancing
               end-if
           end-perform
           display space.
-----
