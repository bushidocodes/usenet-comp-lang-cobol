Derek,
    You have to have the FTP software installed on MVS.  Then you just
execute the FTP PROC from your calling JCL.  An example of the FTP PROC
follows:

//PJAMFTP1     PROC SAVRS=9,
//             CARDLIB='PFIZ.CARDLIB',
//             FTPMEM='',
//             FTPPARM='',
//             BADNEWS='*** SEVERE ERROR - CONTACT SYSTEMS REP ***'
//*
//*---------------------------------------------------------------*
//*                                                               *
//*    FTP BATCH JCL FOR EXECUTING THE FTP SUBCOMMAND             *
//*                                                               *
//*---------------------------------------------------------------*
//FTPPROC      EXEC    PGM=FTP,PARM='&FTPPARM',REGION=6M
//SYSUDUMP     DD      SYSOUT=D
//SYSPRINT     DD      SYSOUT=&SAVRS
//OUTPUT       DD      SYSOUT=&SAVRS
//INPUT        DD      DSN=&CARDLIB(&FTPMEM),DISP=SHR
//*
