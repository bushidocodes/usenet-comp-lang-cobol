Some (many, most) compilers freak when attempting to open a file as I-O when
the file does not exist.

Try:

   OPEN OUTPUT PROPERTY-FILE.
   CLOSE PROPERTY-FILE.
   OPEN I-O PROPERTY-FILE.
   etc.

Or, better:

   OPEN I-O PROPERTY-FILE.
   IF PROPERTY-FILE-STATUS NOT = '00'
      PERFORM FILE-NOT-FOUND.

  FILE-NOT-FOUND.
      (in addition to error messages, might include)
      OPEN OUTPUT PROPERTY-FILE
      CLOSE PROPERTY-FILE
      OPEN I-O PROPERTY-FILE.


P.S.
Lose the LABEL RECORDS... clause. It's ignored.
