```
› I can print correctly to my printer, but when I put the printer off line,
› the rest of the file get lost.

That is not a problem with the program but is because the printer and/or
computer setup aren't implementing handshaking and/or the printer
cable is wrong.

The printer should signal back to the OS that it is 'busy' by using
DTR, CTS or XOFF. It is either not sending the signal, or the
cable isn't carrying it or the computer is ignoring it (because it
is looking for a different one).
```
