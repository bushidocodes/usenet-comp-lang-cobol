Save it as ASCII source.txt file and then upload it to that main frame.
Take care of the usual stuff of deleting CR/LF delimiters etc. and
perhaps you should explicitly specify the file to be converted to EBCDIC
i.s.o. taking the bitpatterns unmodified. Most tools are able to do this.
I think you have the CR/LF=variable-to-fixed-format conversion wrong, and
/ or no ASCII->EBCDIC conversion?

The Frog
