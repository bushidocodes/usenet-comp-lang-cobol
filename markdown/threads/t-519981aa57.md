Component names and functions of the [Customize Editor] dialog

� [Command Line] edit box:

Specify the name of the executable file for the editor you wish to use.

�  [Browse] button:

Opens the [File Browse] dialog box.  The file specified in the dialog box is 
displayed in the [Command Line] edit box.

�  [Use the Editor to edit the Module-Definition file] checkbox:

Specify whether to use the editor displayed in the [Command Line] edit box when 
editing the module definition file.

Caution: [The Also Module Definition File] checkbox is valid only when an edit 
class is specified for a DEF file under the tool linkage operating environment 
of the PowerGEM Plus editor.

�  [OK] button:

Confirms your selections and closes the dialog box.

�  [Cancel] button:

Ignores your settings and closes the dialog box.

�  [Help] button:

Displays help text for the [Customize Editor] dialog box."
