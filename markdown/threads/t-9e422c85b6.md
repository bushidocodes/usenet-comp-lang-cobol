```
You don't tell us which compiler (or operating system) you are using.  If
you have a compiler that supports Intrinsic Functions, the EASIEST way to
convert a one-byte binary field to a "decimal" value is by using the
intrinsic function "ORD", i.e.,

  Compute Decimal-Field = (Function Ord  (Hex-one-byte-field)) - 1

The "minus 1" is because FUNCTION ORD goes from 1 to 256 instead of the
"usually" desirable 0 to 255.

If you do NOT have intrinsic functions available to you, then there are a
variety of other ways - but tell us about your compiler and OS and we can
answer appropriately.
```
