```
Support for ISO 10646 (which *basically* is the same as Unicode) is in the
draft of the next Standard.  I do NOT know of any compiler that has "full"
support for it yet.  I do know that Micro Focus (at least some variations)
has some support - as does IBM's COBOL/400 (but which actually has more
support than OS/390 COBOL - I think).

If you check out the COBOL FAQ and go to the various vendor's web-sites, you
should be able to ask each of them for their current level of support.
```
