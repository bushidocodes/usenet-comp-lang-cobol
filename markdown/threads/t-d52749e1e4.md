INITIALIZE GR 5.b.1 makes D99 (15491:4500)
a possible receiving item; but GR 5.c.1.b requires
that D99 (15491:4500) have a VALUE clause for
it to be a receiving-operand. Reference modification
creates a unique data item; but that data item does
not have a VALUE clause. Thus, I see this statement
as being non-executable because the VALUE phrase
requires a VALUE clause.
