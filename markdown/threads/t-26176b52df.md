The object of this game is to add letters to the string (in front or
appended), WITHOUT making a word. If you make a word, you lose.

BUT, before you go adding z and/or x, it must be POSSIBLE to make a word...
and it must be an ENGLISH word, no Proper Names, although plurals are
acceptable. If the string resolves to something which doesn't look like it
is possible to make a word from, any player can "challenge" and the person
who made the last move MUST state the word he believes is possible. If he
can't (he was bluffing...<G>) then he LOSES and another game starts (or
doesn't...whatever...)

Any connection to COBOL or IT is purely coincidental and unintentional and
side bets on how long this thread stays on topic can be placed with William
Hill...just like so many threads in this forum <G>.

Enjoy!
