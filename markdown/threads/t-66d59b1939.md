The options above will only work with the REBUILD utility.

With fhrebuild, specify the alternate keys in a parameter file

eg

PK (0:10:CHARTYPE) // Primary Key
AK D(11:10:CHARTYPE) // Alternate Key, dups allowed
AK (21:10:INTTYPE) // Alternate Key

It is possible that the parameter file can be created by using
fhvalidate first.
