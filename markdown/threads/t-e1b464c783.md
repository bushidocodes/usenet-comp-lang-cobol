Andrew,
Do you have a PC-based COBOL compiler already, or are you preparing code
to upload to a host computer? If you're looking for a PC-based package,
IBM's VisualAge for COBOL for OS/2 has a so-called "COBOL-intelligent"
editor. There is color coding of reserved words, limited syntax
checking, etc. (Don't be put off by the name, I believe the product now
has Windoze versions on the same CD as the OS/2 version. There is an
AIX version, too, COBOL Set for AIX - I haven't tried that.)

I don't know how one could get the editor separately. It is called
LPEX, I think, and is part of the other VisualAge products (or at least
C++, for sure).
Colin Campbell


Micro Focus COBOL has more bells and whistles, but the IBM product has
superb compatibility with their mainframe compilers, since it is
actually a code port.
