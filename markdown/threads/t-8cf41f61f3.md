I assume file manager...

View / options / filetypes

New type

Fill in the blanks and click New

Type in a descriptive name under action and the put in you command
name under application.  For example to edit a file my command is:

"C:\Program Files\vim-5.3\gvim.exe" "%1"

If you want options then put them in.  If you want multiple
selections then keep adding new actions and define one as the
default.
