```
A while ago I was asked about the status of X/Open COBOL. The following is
the reply that I received from a "usually reliable" source.

"The current X/Open COBOL is officially email only I believe, and, in
practice, this has become inactive. I cannot foresee anything that might
raise it from this lethargic state. So I guess 'faded away in favour of
ANSI 95' is a fair summary."

P.S. and we all know that ANSI '95 became '97 which became 2000 which is now
(unofficially) 200x.
```
