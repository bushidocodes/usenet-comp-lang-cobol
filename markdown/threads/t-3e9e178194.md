```
Hi ...

Finally managed to suss this out for myself experimentation winning
out over RTFM, for future reference the following works:

        IDENTIFICATION DIVISION.
        PROGRAM-ID. DYNALLOC.
        ENVIRONMENT DIVISION.
        INPUT-OUTPUT SECTION.
        FILE-CONTROL.
            SELECT SYSOUT
                   ASSIGN TO SUB.

        DATA DIVISION.
        FILE SECTION.
        FD SYSOUT
           RECORDING MODE IS F
           BLOCK CONTAINS 0 RECORDS
           LABEL RECORDS ARE STANDARD.
        01 OUTREC          PIC X(133).

        WORKING-STORAGE SECTION.
        01 BPXWDYN         PIC X(8) VALUE 'BPXWDYN'.
        01 ALLOC-STRING.
           05 ALLOC-LENGTH PIC S9(4) BINARY VALUE 100.
           05 ALLOC-TEXT   PIC X(100) VALUE
           'ALLOC DD(SUB) SYSOUT(6) FORMS(1234)'.
                                                                      
        01 PRINT-LINE      PIC X(133)       VALUE
           ' TEST PRINT - CLASS 6 FORMS 1234'.
                                                                      
        PROCEDURE DIVISION.
                                                                      
               CALL BPXWDYN USING ALLOC-STRING

               IF RETURN-CODE NOT = ZERO
                                                                      
                DISPLAY 'ALLOC FAILED, RETURN-CODE WAS ' RETURN-CODE

                ELSE DISPLAY 'ALLOC OK'

                    PERFORM B-PRINT

                END-IF

                GOBACK.

         B-PRINT SECTION.

                OPEN OUTPUT SYSOUT

                MOVE PRINT-LINE TO OUTREC

                WRITE OUTREC
        
                CLOSE SYSOUT

                EXIT.

Cheers,
Reg.
```
