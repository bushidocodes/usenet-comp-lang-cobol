MAGNA X RELEASE 2.1 INTEGRATES THE MAINFRAME FOR SECOND GENERATION CLIENT/SERVER
APPLICATION DEVELOPMENT

Latest Magna Software Release Features Mainframe Connectivity, Relational Database
Access and Significant Development Enhancements

Application Development Conference, New York, NY, April 3, 1995 P Magna Software
Corporation today announced the immediate availability of MAGNA X Release 2.1,
featuring powerful new mainframe connectivity, automatic creation of Microsoft
Visual Basic client services, seamless access to Informix, Sybase and Oracle
databases, and an improved development graphical user interface.

The goal of all these new features is to provide a straight-line path for
integrating current business-critical data sources with new client/server
application technology,S explained Kent Lawson, president of Magna Software.
You get the security and continued return on investment of using traditional
data sources combined with the flexibility and power of advanced partitioned
application design, According to a recent MAGNA X Release 2.1 adopter, the
product is a success. RMystaff began producing production-level applications
with MAGNA X 2.1 withinhours of installation,S said Deiter Zink, director of
AustriaUs largestinsurance provider, Wiener Gebietskrankenkasse of Vienna.
MAGNA X 2.1 is alreadysecond nature for my staff and the results are powerful
client/serverapplications that use our mainframe as a datasource.S

Bridges the gap between desktop and mainframe
MAGNA X 2.1 addresses the realities of todays client/server deployment,
explainedLawson. RCompanies must include existing data sources in their
application mix,including mainframes and relational databases. They also need
an easy way todevelop powerful client/server applications while relying on the
current skillset oftheir staff. This has always been the mission for Magna,
and MAGNA X 2.1 is theonly tool delivering these requirements.

MAGNA X 2.1 automatically bridges the gap between the desktop client and
mainframe data. Developers no longer need to code calls to invoke mainframe
services because MAGNA X generates these application services automatically. No
other client/server application development tool provides this seamless
integration making the mainframe a functioning part of a client/server
deployment.

Seamless integration with Informix, Sybase and Oracle databases
Recognizing the widespread popularity of the relational databases from Informix,
Sybase and Oracle, MAGNA X has been extended to completely integrate these data
sources into the partitioned application structure. MAGNA X generated server
programs can now support Sybase, Informix and Oracle interchangeably without additional
coding.

Automatically generates Visual Basic client services
With the new RMAGNA X Wizard for Visual Basic,S front end Visual Basic services
can be generated automatically. This feature generates fully functional Visual
Basic clients that can be used for quickly running tests of MAGNA X applications
during development or as production-level, deployable Visual Basic clients.

Improved GUI development environment
To facilitate development of MAGNA X applications, version 2.1 contains several
enhancements to the GUI development environment. In response to customer
requests, MAGNA X 2.1 supports both graphical and outline views of the
application structure. As a result, developers now have improved flexibility
for viewing the relationships between different components and the varying
application tiers. MAGNA X 2.1 also features improved navigation tools and
application diagraming functions.

About Magna Software Corporation
Magna Software Corporation, of New York City, has 14 yearsU experience as a
supplier of
application development tools to MIS developers. Magna has an additional office
in Washington D.C. and a European affiliate company in Paris. MAGNA X is the
third major application development product produced by Magna Software. Its
first product was a fourth generation language for Control Data mainframes that
dominated its market in the early 1980s. The second product was an application
generator for the Groupe Bull mainframe series that has been licensed by 45% of
Groupe Bull mainframe sites in the U.S. and 25% of the Groupe Bull sites worldwide.
