December 21, 2000

About twelve months months ago, I asked here if anyone could suggest a 'VSAM
for Dummies'- type tool to enable a calculation of the space allocations
which should be used during an IDCAMS DEFINE of a VSAM dataset. Thanks to
several people who helped me with this project, that software is now
available.

Version 1.1 of "VSAMSpace" is a single 55K Windows 32-bit (NT/9x/2K)
executable. It does not require any support DLLs, does not read or write to
the system registry or do any other annoying stuff. To run it, you click on
it twice in Windows Explorer. To un-install, you delete the file.

When you click the "Doc" pushbutton, it extracts the doc file to a text file
in the same directory where the executable is and launches NOTEPAD to view
it. (All the math details are in the documentation).

Operation of the software is so simple even a mainframer can handle it.

And, the best thing about this software - at least insofar as I can tell
about this crowd - is that the price is right: free.

If you'd like a copy of this software, just send an e-mail request to
michael.mattias@gte.net with a return e-mail address. I can attach the file
to the email as an EXE or as a ZIP file (about 25K) if you need to get past
a firewall or just feel better about it that way; so let me know if you you
want the software attached as "EXE" or "ZIP."

Regards,
Michael Mattias
Racine WI
michael.mattias@gte.net

PS:I will NOT send the software in response to postings here, nor do I edit
out so-called "spam-traps" from e-mail addresses. If I cannot click 'reply'
or on an address in the text of the mail, it won't happen.
