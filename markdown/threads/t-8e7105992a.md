```
Here's one way to get the COBOL compiler :
Go to http://www.acs.oakland.edu/cgi-bin/vsl-front/CatForm
In the category box, select DOS
In the directory/filename, enter COBOL650.*
Click on Start Search and you'll be able to download
the file COBOL650.ZIP just after.
But you'll need to find an old linker in order to build
executable files from the object files generated by the
compiler. The linkers of the Borland products don't seem to
work.
I use the Visual C++ 1.0 linker, but any old linker should work.
```
