but it depends upon which version of RM you are running
RM74 used to use 2 Physical files for each indexed file = one for the data
and
one for the index, with the index held as a linked list
RM85 puts both the index and data in the same .dat file with the index at
the
start of the file (minimum size 8k) in the form of a Binary+ Tree and the
data
records to follow - most Computer Science text books describe Binary Trees
quite well
