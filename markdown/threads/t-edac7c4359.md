[I am no Linux expert.]  Our SCO runtime works with the IBCS II library.
This library can be downloaded from any of the Linux vendors, I am told.  If
you want to customize the runtime (e.g. adding subroutines written in C), it
must be linked on a SCO system, then moved to Linux.  I hope this helps.
