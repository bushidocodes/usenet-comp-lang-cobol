Hi all, the project I am working on is going ahead. I moved the
guicobol tool to an opensource license and then you can try it.

The guicobol is a preprocessor that wants to give a gui interface to
opencobol 1.1 by using the gtk opensource libraries. It realizes this
using an object like syntax.

The project and documents are available at www.tp-srl.it

here a sample demo...
