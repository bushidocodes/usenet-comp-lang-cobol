It helps to remember that, in general, the last two digits of the ABEND
code are the SVC number.

x13 is an OPEN problem
x37 is end of extent, volume switch, etc.
x22 is operator (manual or automatic) action
x06 is LINK
x07 is XCTL
x08 is LOAD
x09 is DELETE

Just as some examples.

Here is my annotated list of MVS SVC's. It's certainly not complete, but
it is all that I need to do GTF trace analysis of application
performance. Not that the numbers are decimal and are off by one. EXCP
is SVC 0, OPEN is SVC 19 (x13). I keep this list taped to my wall.
Sometimes, it is all I need to understand what went wrong.
