You can download a 3270 terminal emulator for MAC from: 
http://www.versiontracker.com/dyn/moreinfo/macosx/15091

You will need the TCP/IP address of the mainframe (or an alias for it) and 
the port it is receiving on. You should be able to get this information from 
PCCOMM when you are at work. You will also need a TSO user account and 
password to logon with once the TSO logon screen appears.

It will connect over TCP/IP and works just as if it was a standard 3270, 
directly connected to the mainframe.

(It is very similar to using PCCOMM but you will need to configure the 
emulator yourself. It isn't hard; get the information you need from PCCOMM 
or talk to a systems programmer about it.)

Pete.
