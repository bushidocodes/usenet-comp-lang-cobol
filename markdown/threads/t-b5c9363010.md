good news -
   What you are looking for (I think) is now a "built-in" (free) feature of 
IBM's current CICS offering.  Look for information on "CICS Explorer".  See (for 
a starting point)
   http://www-01.ibm.com/software/htp/cics/explorer/

bad news:
  This is a CICS TS V3 (not V2) product. So if you are really running on the 
(soon to be unsupported) V2 of CICS TS, this probably won't help you.  I think 
you can still download the free version (from the site listed above) but I don't 
know how much will and won't work if you only have V2.

For "end of service" for CICS TS V2.3, see:
   http://www-01.ibm.com/support/docview.wss?uid=swg21154953
