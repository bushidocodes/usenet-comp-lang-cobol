```
Sounds as if the upload mechanism treats tab characters differently than you PC
editor.  I would recommend using a utility to expand the tabs to spaces before
uploading.  Bringing the code into the PC editior is probably where some smart
wonder thought to save you space and automatically converted strings of 4
spaces to tabs while the upload software probably uses something silly like 8
characters per tab char.  This is one of the things I ran into and resorted to
never using the tab key (my editor does not do the automatic compression).

Hope that this points you in the right direction.
```
