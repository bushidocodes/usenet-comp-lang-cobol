to change the default behavior, you have to modify ADIS Control. You do

Possible options are :

1 : Normal mode. No pre-clear or auto-clear action.

2 : Pre-clear mode. On moving forwards into a field for the first time
it is
cleared to spaces or zeros. The original contents of the field can
be
recovered by pressing the UNDO key.

3 : Auto-clear mode. If the first keystroke after moving forwards into
a field
for the first time is a valid data character, the field is cleared
to
spaces or zeros before processing the character. Any other type of
key
turns off auto-clear mode. The original contents of the field can
be
recovered by pressing the UNDO key.

4 : Pre-clear mode. As for option 2, except that the UNDO key cannot
be used
to recover the original contents of the field.

Alberto. What you want to do is make a call to the ADIS before your
accept.

First place the desired default value in the field you are accepting. Then
make a call to X"AF", with an 86 in the bit-pair-number and a 2 in the
bit-pair setting of the standard adis parameter block. That will make the
first character that they enter, erase the entire field.

Hope that helps you!
