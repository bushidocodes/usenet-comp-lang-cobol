Summing up the number of records read is simple:

WORKING-STORAGE SECTION.

01 WS-TOTAL-ACCUMULATORS.
05 WS-TOTAL-RECORDS-READ-ACCUM PIC S9(05) VALUE ZEROS.

PROCEDURE DIVISION.

800-READ-A-RECORD.
READ INPUT-RECORD
AT END SET WS-END-OF-FILE TO TRUE
END-READ
IF NOT WS-END-OF-FILE
ADD 1 TO WS-TOTAL-RECORDS-READ-ACCUM
END-IF
.

The details are left as an excersise.
