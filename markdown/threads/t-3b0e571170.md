If you try the shareware sites, there are a number of cheap products
for generating RTF files using any text editor. You can also download
the Windows 3.1 help compiler (HC.EXE), and instructions for using it.


I looked at several recently. There's one called RTFGEN, another
called HWA (Help Writer's Assistant), and I'm sure there are many
more. Some are freeware.

Microsoft also has a book out, I believe it's called the Help
Authoring Guide, with the Windows 95 Help compiler on CD-Rom in the
book. The help compiler is supposed to work with Windows 3.1, 95, and
NT. They really want you to use MS-Word to create the RTF files,
though. The book is about $50.

I hope this helps!
