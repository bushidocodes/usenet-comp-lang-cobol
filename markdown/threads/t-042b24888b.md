Azrex Inc is announcing the availability of the 4GL / Report Writer :
AZ7 for Windows 3.1/95/NT.

Some of AZ7 Features:

Supports Acucobol, Realia, Microfocus data and file types.

4GL for Computer Professionals or
AZEZ GUI interface for End Users

Dictionary driven system (easy conversion from Cobol 01 record layouts

All typical report writing functionality you would expect (sorting ,
printing , summarizing etc..)

Also extraction of data into Excel , CSV or DIF file formats for import
into various applications

Conversion of output reports into HTML Documents
