> 
> And after installing it I tried to install Net Express 5.0 I faced
> error with You should install Visual Stduio .Net.

If you don't need Visual Studio integration or the ability to create 
managed code, then during the Net Express installation, when you get 
to the page of the installation wizard for selecting components to 
install, deselect "Net Express support for .NET" (the first 
component). It requires Visual Studio .NET 2005.

(Personally, I don't think that component should be selected by 
default, but I didn't make the packaging decisions.)

If you *are* looking to create managed code, your Net Express media 
should include a copy of Visual Studio 2005 Standard Edition. Install 
that first.
