Guenther,
        The solution is quite simple. Define a CALL-CONVENTION as
follows :

          special-names.
                call-convention 74 is win32.

and then have

procedure division win32 using a b c.

where a, b and c are linkage section items.

That will declare your program as __stdcall. If you need more information on
using these extensions there are sample programs supplied with 4.0.32 that
demonstrate these features.
