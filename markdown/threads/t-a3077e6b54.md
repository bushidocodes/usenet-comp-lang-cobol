```
Your problem is in assigning everything to sysin.  Every platform
handles
just how an actual file is linked to the Cobol program a bit
differently.
SYSIN and SYSOUT can be changed to other things.  For example, on a PC,
you
would probably code something like

SELECT FILENAME ASSIGN TO "C:\MYDATA\MYFILE.DAT"

You are probably going to have to hunt down an actual example for your
particular OS to write your code.
```
