Depends on the compiler.  In the ANSI '85 Standard (but not the draft of the
next one), 05-levels (or any level greater than 01) needs to start in the
"B-margin" which begins with column 12.

Almost all the PC compilers have an extension that does not enforce this
rule - but other compilers do (or do with certain options set).
