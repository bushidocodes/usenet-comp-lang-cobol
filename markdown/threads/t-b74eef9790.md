```
Changing a 'GO TO F4-TERM' which does an EXIT PROGRAM into
a 'PERFORM F4-TERM' which does an EXIT PROGRAM doesn't make
the program 'more structured', it simply disguises the
GO TO as a PERFORM-that-will-never-return.  This is a _bad_
thing because the next programmer along will expect
PERFORMs to return.
```
