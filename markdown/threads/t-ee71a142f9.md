```
Tomba wrote:

> I have a question again about screens in miscosoft cobol 4.5.
> 
> How can I change the color of some tekst on the screen by clicking an arrow
> button?

You can change the display colour using CALL x"A7" see Operating Guide
page D-13.

I use this for all colouration in MF/MS as I allow the users to set
their colour codes for each system/company/terminal combination which I
store away and reload when they log into my systems.  I then just need
to perform routines such as Screen-HL (highlight) or Screen-BG (normal
background text) and it retrieves and sets the colour combination that
the user has chosen.  

In your case you would need to detect the arrow keys in CRT Status field
and then set the colour for each selection and redisplay them.
        
        ...
        CRT STATUS IS CRT-Status
        ...
        01   Selection-Colours.
             03  Select-Attr         PIC X OCCURS 3.
        ...
        MOVE ALL Colour-Normal   TO Selection-Colours
        MOVE 1                   TO Selection-Index
        MOVE Colour-Highlight    TO Selection-Attr(1)
        MOVE ZERO                TO Selection-Made
        MOVE 0416                TO Selection-Cusor
        PERFORM
            UNTIL Selection-Made > ZERO

            CALL x"A7" USING Set-Colour Select-Attr(1)
            DISPLAY Select-Text-1       AT 0416
            CALL x"A7" USING Set-Colour Select-Attr(2)
            DISPLAY Select-Text-2       AT 0431
            CALL x"A7" USING Set-Colour Select-Attr(3)
            DISPLAY Select-Text-3       AT 0449

            ACCEPT Selection-Field      AT Selection-Cursor
            IF ( Key-Stroke = RIGHT-ARROW 
             AND Selection-Index < 3 )
                MOVE Colour-Normal      TO
Selection-Attr(Selection-Index)
                ADD 1                   TO Selection-Index
                MOVE Colour-Highlight   TO
Selection-Attr(Selection-Index)
                ADD 15                  TO Selection-Cursor
            ELSE
            IF ( Key-Stroke = LEFT-ARROW 
             AND Selection-Index > 1 )
                MOVE Colour-Normal      TO
Selection-Attr(Selection-Index)
                SUBTRACT 1            FROM Selection-Index
                MOVE Colour-Highlight   TO
Selection-Attr(Selection-Index)
                SUBTRACT 15           FROM Selection-Cursor
            ELSE
            IF ( Key-Stroke = ENTER-KEY )
                MOVE Selection-index    TO Selection-Made
            END-IF
            END-IF
            END-IF
        END-PERFORM
                
Of course you would want to made this somewhat more generalised.
```
