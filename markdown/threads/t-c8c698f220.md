Manually will do it.

IF YY > 80
MOVE 19 TO CC
ELSDE
MOVE 20 TO CC
.

If you think the programs may be running in 80 years time then
make the 80 some variable in a COPY file so it can be changed
in a few decades time.

IF YY > Some-Chosen-Year
MOVE First-Century TO CC
ELSE
MOVE Second-Century TO CC
.

Or have these values in a control file that can be amended by
a simple maintenance procedure.

IF the version of MS Cobol is 5.0 then there is a function to
get this called CURRENT-DATE, it is in 4.5 as well.
