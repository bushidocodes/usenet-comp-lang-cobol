I finally got an application running which uses APPC calls from
command-level CICS to communicate directly to a networked OS/2
workstation. I use direct calls to APPC.DLL from within Micro Focus
Cobol for the program which runs under OS/2.

The configuration of CICS and VTAM was a bitch!! But we did it.

I used the sample APPCDEMO BATTLE programs which come with Workbench as
the skeleton for the PC program. The CICS program code I had to figure
out for myself but it is pretty easy on the CICS side.

I would be delighted to keep in contact with other developers who have
done or are doing APPC programming.
