In most PC and UNIX versions of COBOL, define your printer file as follows:

SELECT PRINT-FILE
ASSIGN PRINT "PRINTER"
STATUS PRINT-STAT.


FD PRINT-FILE.

01 PRT-REC PIC X(132).

Load whatever you want to print in PRT-REC and use WRITE statements to output.
Other WRITE modifiers include BEFORE or AFTER ADVANCING ## LINES or PAGE.
