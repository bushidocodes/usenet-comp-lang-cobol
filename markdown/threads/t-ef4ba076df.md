COBOL is a high level business programming language.

Assembler is a low-level everything programming language. If you know
assembler, you can probably program in just about any language.

COOLGEN is a case/code generation tool (formerly IEF by Texas Instruments)
that generates gibberish COBOL.

Noel Ng
Consultant, Toronto.
