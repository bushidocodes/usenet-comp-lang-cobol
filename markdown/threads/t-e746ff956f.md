I think it is FUNCTION RANDOM.

I have used another method.  Get a seed by polling the system clock by doing
an ACCEPT <data-grp> from TIME.  Parse it out to get the hundredths of a
second.  This gives you a fairly random seed.

Then plug it into a homemade function.  You could do a random letter like
this:

 01  SYS-TIME.
*--->>> THIS GRP MAY DEPEND ON OS.
     05  SYS-HH   PIC 9(2).
     05  SYS-MM   PIC 9(2).
     05  SYS-SS   PIC 9(2).
     05  SYS-HS   PIC 9(2).
 01  POLL-CLOCK-SEED  PIC 9(2).
 01  LETTERLINE       PIC X(26) VALUE 'ABCDEFGHIJKLMNOPQRSTUVWXYZ'.

 PROCEDURE DIVISION.
     MAINLINE SECTION.
     POLL-CLOCK.
        ACCEPT SYS-TIME FROM TIME.
        IF ( SYS-HS > 78 )
           GO TO POLL-CLOCK
        END-IF.
     COMPUTE-SEED.
        COMPUTE POLL-CLOCK-SEED = SYS-HS / 3.
        ADD 1 TO POLL-CLOCK-SEED.
     SHOW-NUMBER.
        DISPLAY LETTERLINE ( POLL-CLOCK-SEED : 1 ).
     SHUT-DOWN-SHOP.
        GOBACK.
