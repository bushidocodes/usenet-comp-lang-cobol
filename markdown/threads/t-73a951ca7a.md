I have made some subprograms using DL/1, in a VSE ambient.
The subprogram have the normal 'ENTRY DLITCBL'
as any batch program, as the first instruction, and 
all the EXEC DLI instructions.
Then you pre-compile the subprogram with DL/1 command precompiler.
The output, with all EXEC DLI expanded to Cobol CALL, must be then
compiled as a relocatable module, and at last linked with the main.
In this mode the subprogram automatically see the database through
the PSB loaded from main.

Giandomenico De Sanctis
