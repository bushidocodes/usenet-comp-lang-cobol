- **Message-ID:** `<3cbafbc8@news.mt.net.mk>`

```
Hi,

Anybody know if there is a simple way (with intrinsic function or
using justify clauses etc) to justify left a numeric field in a report.
For ex:

num-field pic 9(6).
rep-field pic Z(5)9.

move num-field to rep-field.

this gives:
num-field    rep-field
25                "    25"

and I want:    "25    "
```

```
I usually do something like:

05 tc pic 99.  (tally counter)

05 num-field pic 9(6) value 25.
05 rep-field pic z(5)9.
05 rep-field-x redefines rep-field pic x(6).
05 left-field pic x(6).

move num-field to rep-field.
move zero to tc.
inspect rep-field-x tallying tc for leading spaces.
move rep-field-x (tc + 1 : length of rep-field-x - tc) to left-field.
```
