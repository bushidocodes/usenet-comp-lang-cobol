- **Message-ID:** `<#RitLp0D#GA.142@upnetnews05>`

```
I am using Dialog 3.0.01 along with NetExpress 2.0.  I need to show two
windows and have the two entry fields on the first window enabled (they are
the only objects on that window) and have all of the objects on the second
window disabled.  If I use DISABLE-OBJECT WIN2 it will disable all of the
objects on WIN2 with the exception of the pages on a tab control and that is
where my problem comes in.  Since the pages won't disable, the user can
click on them with the mouse and the LOST-FOCUS event is triggered in my
entry field on the first window.  I've tried disabling the tab control, the
individual pages, as well as the entire window and nothing seems to work.
Is there a way to disable the pages?  Right now I have it coded to not show
the second window until the user finishes with the first window, but it
makes the screen look crappy when I do that.  If anyone can give me a
suggestion, I'd appreciate it.

bill
```
