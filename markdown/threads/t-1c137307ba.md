```
There is a environment variable COBPATH what says to Cobol where the
programs should be searched

export c_prog=/dir1/dir2/c_routines
export cob_prog=/dir1/dir6/programs

export COBPATH="$c_prog:$cob_prog"
or directly
export COBPATH="/dir1/dir2/c_routines:/dir1/dir6/programs"
```
