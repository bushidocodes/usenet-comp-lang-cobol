The following code fragment should work:


01 TEST-DATA1 PIC X(10) VALUE 'ABCDEFGHIJ'.
01 TEST-DATA2 PIC X(10) VALUE 'abcdefghij'.
01 OUTPUT-AREA PIC X(10) VALUE SPACES.


PROCEDURE DIVISION.

START-PROCESSING.

MOVE SPACES TO OUTPUT-AREA.
MOVE FUNCTION LOWER-CASE(TEST-DATA1) TO OUTPUT-AREA.
DISPLAY OUTPUT-AREA.

MOVE SPACES TO OUTPUT-AREA.
MOVE FUNCTION UPPER-CASE(TEST-DATA2) TO OUTPUT-AREA.
DISPLAY OUTPUT-AREA.


STOP RUN.


Mike Dodas
