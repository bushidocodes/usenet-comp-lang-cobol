You didn't actually say what message you are getting., but if you are
DECLARING, then you *must* 'DECLARE' something as follows :-

EXEC SQL BEGIN DECLARE SECTION END-EXEC.
01  SQLSTATE                    pic x(5).
EXEC SQL END DECLARE SECTION END-EXEC.

or

    EXEC SQL DECLARE

    `Clients` TABLE
    ( "CliId"               VARCHAR(3)
     ,"FacId"               VARCHAR(2)
     ,"Shortname"           VARCHAR(30)
     ,"LifeYears"           CURRENCY
     ,"Percent1"            CURRENCY
     ,"Percent2"            CURRENCY
     ,"Range1"              SMALLINT
     ,"Range2"              SMALLINT
     etc......
    ) END-EXEC.

The latter example is taken from the copyfile generated by ESQL
Assistant.
