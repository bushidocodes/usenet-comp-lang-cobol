```
For Linux here are the Open Source Compilers currently available.

COBOL compilers and tools for Linux: open source
Tiny COBOL 
http://tiny-cobol.sourceforge.net/ 
The Tiny COBOL project is a COBOL compiler being developed by the free
software community. The goal is to produce a COBOL compiler, which is
ANSI COBOL-74 compliant and later ANSI COBOL-85 compliant. It is being
developed on the Linux OS, using a mix of C and LEX/YACC to produce a
compiler that will generate GNU x86 assembler code. It only runs on an
i386 Linux platform. The Tiny COBOL compiler and runtime library are
licensed under the GNU General Public License
http://www.gnu.org/copyleft/gpl.html/.

GNU COBOL2C 
http://www.gnu.org/software/cobol/ 
GNU Cobol is a Cobol to C translator. GNU Cobol is an open source
project still in alpha release. The GNU Cobol2C project is an effort to
create a COBOL compiler that will have the ability to interact with
existing COBOL libraries and data files. A free compiler download is at
http://metalab.unc.edu/pub/Linux/devel/lang/cobol/.

CobCy 
http://web.tiscalinet.it/albertosantini/cobcy/ 
This is a COBOL to C translator intended to compile ANSI COBOL sources.
The current version is not complete. The parser is YACC based. The rest
of the translator is written in C++.

Tim Josling COBOL 
http://www.geocities.com/timjosling/ 
A GNU Compiler Collection-based COBOL compiler is provided here.

CPB 
http://users.skynet.be/jpa/ 
CPB is a general-purpose COBOL source code analysis tool.
```
