```
Hi All,

The TinyCOBOL compiler version 0.60 is available for download at the TC
home page (1) and iBiblio (2)..

This release contains mainly bugs fixes, and some enhancements.  It
includes updates to the main compiler and run-time.
With this release, improvements have been made in the following areas:
locking for relative files, better support of print files (write after
and
before advancing) and better compatibility with 85 Standard for
some indexed file I/O operations.


Please report any bugs and/or problems to the TC mailing list.


Bernard Giroud

1) TinyCOBOL Home Page
   http://tiny-cobol.sourceforge.net/

2) iBiblio (formerly Metalab)
   http://www.ibiblio.org/pub/Linux/devel/lang/cobol/
```
