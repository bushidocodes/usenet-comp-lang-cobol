Without knowing how your electronic scales output the data or the
cobol you're using or the environment it is running in it's hard to
tell, but two possible methods which come to mind are:

A callable sub-program written in the low level language of your
choice (C? assembler?)

If the scale can output the data on a serial port you could try a
process running a Cobol program with standard in being the serial port
and try using ACCEPT statements. You could then log the data to a file
for access by the actual application.
