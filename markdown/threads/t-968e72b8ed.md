like

something like

  05  FAX  PIC X.

it should work.

```
Many compilers are very case senstive. Try BY instead of by. Also, if you are
doing my favorite trick of using the same COPY statement for both an input and
output file (for example), try puting the REPLACING on the first occurance of
the COPY, not the second (this has tripped up some older compilers I've
used....
i.e. try
COPY INPUTFD REPLACING ==INFILE== BY ==OUTFILE==.
COPY INPUTFD.
instead of
COPY INPUTFD.
COPY INPUTFD REPLACING....
```
