```
Announcement from Casegen Systems Ltd.


"Casegen COBOL for Windows Professional is now available on Beta and we are
looking for a small number of beta testers.  We especially want to hear from
you if you already have Casegen COBOL for Windows or you are interested in
writing ODBC Windows applications in COBOL.  Casegen will provide the Beta
FREE on CD, but those wishing to develop ODBC applications MUST have a copy
of Fujitsu Version 4 which we do not supply.  We can supply a trial version
of SQL Anywhere upon request with the Casegen CD.  It must be noted that
this is a Beta version and does NOT include latest documentation or help
facilities and elements of the features and facilities could change.  A
second Beta MAY be made available subject to problems/enhancements added to
the product".

Casegen COBOL for Windows Professional includes the following features over
the existing product:


Full ODBC support including "point and click" SQL specification. After
selecting an ODBC Data Source, all table import, keys, Cobol variable
definitions etc. are automatic.

New "Explorer-style" List View - new Message View window.

Support for bit-mapped buttons.

Dynamic bitmaps providing support for loading .bmp files at execution with
simple syntax. For example - click on a country to see its flag.

Toolbar support (not docking) with tooltips on all controls.

Modified and simplified generator dialogs.

Launch and support sub-forms and subroutines from main form

Double-click on compiler messages takes you straight to error line in event
code.

Several fixes and removal of some size limitations.


The new product looks better and is easier to use. SQL/COBOL applications
are now easier to write than ever before.

Please email us at support@casegen.co.uk even if you have already advised us
of your interest. Please indicate which, if any, relational databases you
are interested in testing.


Simon Cordingley
Development Manager
Casegen Systems Ltd
www.casegen.co.uk
```
