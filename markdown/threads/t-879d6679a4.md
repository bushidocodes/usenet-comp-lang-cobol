I'm not sure what you're saying, but could this be what you want?  My
table is indexed by a TABLE-INDEX and a TABLE-SIZE, I never change
table-size.  Variable length tables work almost the same way.

    SET TABLE-SIZE TO 1
    SET TABLE-SIZE UP BY 1
    READ TABLE-FILE INTO TABLE-RECORD
        AT-END SET TABLE-EOF TO TRUE.
    PERFORM 0100-BUILD-TABLE 
        UNTIL TABLE-EOF.
    DISPLAY TABLE-VALUE (TABLE-SIZE).

0100-BUILD-TABLE.
    SET TABLE-SIZE UP BY 1.
    MOVE TABLE-RECORD TO TABLE-VALUE (TABLE-SIZE)
    READ TABLE-FILE INTO TABLE-RECORD
        AT-END SET TABLE-EOF TO TRUE.
