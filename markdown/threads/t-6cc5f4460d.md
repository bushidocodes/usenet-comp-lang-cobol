Fixed Assets System

IBM ES-9000, CICS / COBOL II, DB2, MVS-ES

Involved in interacting with the users to get the detailed requirements and in

Share Accounting System

IBM ES-9000, CICS / COBOL II, DB2, MVS-ES

to maintain shareholders information. It is also used for transferring,
splitting and consolidation of shares. Transfer, split and consolidation can
be done online as well in batch mode. It is used to compute dividends and
print dividend warrants. It has querying features to view shareholders
information. It also has an interface to update the database in batch mode for
all new allotment of shares.
