```
As a long-time student of human behaviour, I found the following link to be 
very interesting.

http://www.itworld.com/print/67857

Anyone here got any "rituals" they use to assist concentration?

I hadn't realised it before, but reading this made me aware of some that I 
have.

I like to sit in a particular armchair (when I thought about it I realised I 
never use any other chair to write code :-)) with the notebook on my lap, 
and a table handy for papers within reach. And I like it quiet. (No 
background music, which is odd because I really enjoy music of a wide 
variety.)

Occasionally, (in the Summer, not at the moment) I'll take the notebook to a 
remote beach, but when I thought about it, I realised I never write code at 
the beach, it is always documentation or diagrams. And I've decided NOT to 
do that any more. I need to leave the bloody notebook at home and simply 
enjoy the beach :-)

Curious creatures, we people.

Pete.
```
