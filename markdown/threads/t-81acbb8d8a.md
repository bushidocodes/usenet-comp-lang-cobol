The Fujitsu COBOL V4 Internet support packages include two different ways to
build and deploy Web-based COBOL applications.

The Fujitsu COBOL V4 "prototype HTML" feature make it easy to build dynamic
websites with the power of COBOL. Database access and business logic can all
be done in familiar COBOL. For more information see:
http://www.adtools.com/version4/WebApplications.htm


The Oracle COBOL Cartridge provides an easy way to build dynamic and
scalable COBOL-base web applications which fully integrate with Oracle's Web
technology. The COBOL Cartridge is implemented as a shared library and
provides seamless access to the Web Request Broker which is the central
component of the Oracle Web Application Server. For more information on the
COBOL Cartridge see:
http://www.adtools.com/news/COBOLcartridge.htm and
http://www.adtools.com/news/cr-1.htm
