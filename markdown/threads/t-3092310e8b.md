```
Hello,
I'm looking for an Intelligent File Conversion Program to Interface
AS/400 applications.

To link AS/400 applications to other AS/400 applications an interface is
required. This interface should cater for the transfer of data from the one
application to the other and vice versa.

I'm looking for some software transformation module that converts data
from one file to another. It must additionally be possible to manipulate the
fields (converted, truncated, moved, calculated, etc.) based on user defined
settings. E.g. It should be possible to add data by means through parameter
driven tables. Also cross referencing of data, i.e. giving data another value,
should be an option. The software module should work in a way that it is
fully parameter driven. Preferably the parameters can be set online realtime
catering for an entirely new and different output when rerunning the module
instantaneously. The module preferably should not be a software generator.
It is preferably written in COBOL for the AS/400, however mainframe COBOL
written software may be an option as well.

Program is meant for commercial banking application and will be paid for.

Please may I have your reactions?

Marcel NL
```
