```
(Sorry for the delay in replying - I was hoping someone else was going to
give you the easy way.)

*ASSUMING* you are using a "current" compiler (with Intrinsic Functions) and
are running in an ASCII environment (so characters are stored in ASCII),
then you can use:

  * convert hexadecimal to decimal - which is "simple arithmetic" and then

 Move Function Char (decimal-ordinal-position-in-collating-seq) to
Character-field

(Remember, that the CHAR function starts at "1" while ascii starts at "0" -
which is why you need to add 1 in figuring out the place in the collating
sequence.)

(FYI, the opposite of the CHAR function is the ORD function - which can be
used to convert a character to an ASCII value.)
```
