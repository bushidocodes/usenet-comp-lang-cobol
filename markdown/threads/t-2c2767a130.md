               3. HINT AND TIPS
               ==============

               Printing with Printer Escape
Sequences

--------------------------------------------------
---

               Some customers, particularly those
coming from other COBOL vendors, have
               developed applications that achieve
particular
               printing effects by writing printer
escape sequences to their files. Fujitsu
               COBOL's print files are directed
through the Windows
               printer drivers, so escape
sequences do not work.

               In the long term we would recommend
getting away from using printer escape
               sequences, as they are difficult to
code and
               tie applications to particular
printers. PowerFORM is probably a much better
               solution for the long term.

               However, if you are currently using
printer escape sequences, we may have a
               solution for you. We have created a
program
               that can take text files with
embedded escape sequences and use Windows system
               calls to pass these files directly
to the
               printer. If you can create the text
files, we can help you print them!

               If you are interested in seeing if
this program can work for you contact us at
               support@adtools.com.
