```
You may want to look at the LE "Bit Manipulation" routines.  See

   http://www2.s390.ibm.com/bookmgr-cgi/bookmgr.cmd/BOOKS/ceea305/4%2e3

Look for the following documented subroutines:

    4.3.1 CEESICLR--Bit Clear
    4.3.2 CEESISET--Bit Set
    4.3.3 CEESISHF--Bit Shift
    4.3.4 CEESITST--Bit Test

Note:  I have NOT tried these myself, so I don't know exactly how easy or
hard they are to use from an LE-enabled COBOL application program.
```
