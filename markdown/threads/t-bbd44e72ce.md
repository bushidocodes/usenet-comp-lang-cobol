```
I received the following note (in reply to a note that I sent) - that I
thought I would pass along to all of those who have asked about COBOL on
LINUX.

"Hi,

The current web page is http://members.xoom.com/rpragana/

Follow the links to Tiny Cobol.

We are looking for people to beta test as well as to help develop the
whole compiler.

Currently is tends to be COBOL 74 based but it is being changed to be
COBOL 85 compliant.

There are currently only 3 of us busy with the coding.

Any contribution is welcome including adding this link to the COBOL FAQ.
Perhaps a posting on the alt.cobol*** lists asking for developers, Beta
Testers, Documenters Etc, Cobol Programmers are also needed to write test
programs.

Regards
Andrew Cameron"
```
