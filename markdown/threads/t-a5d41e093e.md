COBOL sp2 is a tool used by COBOL programmers to develop and maintain
contemporary user interfaces for their programs.  It allows the
complexities of such an interface to be somewhat isolated from a COBOL
program, yet still enables a COBOL programmer to have close control
over the interface directly from a program using standard COBOL code.

It will interface with Fujitsu Cobol and there is a verion on the CD
that came with Thane's book.
