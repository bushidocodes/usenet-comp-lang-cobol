          STOP RUN.

      * The purpose of the 200-Headings Module is to  *
      * print out the Headings for the report         *
      *************************************************        
       200-HEADINGS.
          MOVE HEADING-OUT TO RECORD-OUT
          WRITE RECORD-OUT AFTER ADVANCING PAGE
          MOVE SPACES TO RECORD-OUT
          WRITE RECORD-OUT AFTER ADVANCING 1 LINE
          MOVE FIRST-LINE-OUT TO RECORD-OUT
          WRITE RECORD-OUT AFTER ADVANCING 1 LINE
          MOVE SECOND-LINE-OUT TO RECORD-OUT
          WRITE RECORD-OUT AFTER ADVANCING 1 LINE.
      **************************************************
      * The purpose of the 300-Process-Records Module  *
      * is to process each record and compute the      *
      * sales tax if there is one. It also calls the   *
      * 400-Calculations Module.                       *
      **************************************************             
       300-PROCESS-RECORDS.                 
          MOVE ACCT-NUM TO ACCT-NUM-OUT
          MOVE ITEM-NUM TO ITEM-NUM-OUT
          MOVE ITEM-PRICE TO ITEM-PRICE-AMT
          MOVE QUANTITY-SOLD TO QUANTITY-AMT
          COMPUTE NET-SALES-AMT = ITEM-PRICE-AMT * QUANTITY-AMT             
          MOVE NET-SALES-AMT TO NET-SALE-OUT   
          IF TAX-STATUS EQUAL 'N'
            MOVE ZERO TO SALES-TAX-OUT            
            MOVE 'TAX' TO TAX-STAT-OUT            
          ELSE
            COMPUTE TAXES = (ITEM-PRICE-AMT * QUANTITY-AMT)* .08
            MOVE TAXES TO SALES-TAX-OUT                         
            MOVE SPACES TO TAX-STAT-OUT
          END-IF
          MOVE SALES-DATA-OUT TO RECORD-OUT
          WRITE RECORD-OUT AFTER ADVANCING 1 LINE
          PERFORM 400-CALCULATIONS.
      ***************************************************
      * The 400-Calculations module keeps a running     *
      * total of the number of items sold, sales,       *
      * taxes and net-sales                             *
      ***************************************************     
       400-CALCULATIONS.
       COMPUTE WS-TOTAL-COUNT = QUANTITY-AMT + WS-TOTAL-COUNT   
       COMPUTE WS-TOTAL-SALES-AMT = ITEM-PRICE-AMT + WS-TOTAL-SALES-AMT
       COMPUTE WS-TOTAL-TAXES = ((ITEM-PRICE-AMT * QUANTITY-AMT)* .08
        + WS-TOTAL-TAXES)
       COMPUTE WS-TOTAL-NET-SALES = (ITEM-PRICE-AMT * QUANTITY-AMT)
        + WS-TOTAL-NET-SALES.
      ****************************************************
      *  The 500-End-of-Job module closes out all files  *
      *  that are opened up at the begining of the       *
      *  program                                         *
      ****************************************************              
       500-END-OF-JOB.
           CLOSE SALES-RECORDS
                 FILE-OUT.
