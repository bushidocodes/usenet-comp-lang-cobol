I wonder if you can help me. I need to find out if RM COBOL can use
shared memory areas in the same way that MF COBOL can ?
If not, I need to find out how easy/difficult it is to pass data from RM
COBOL to a C program which can then handle the shared memory area.

The reason for all this is that I need a way to access data which
currently being written and updated by RM COBOL. I can't use a converted
copy of the data because it changes all the time. I can't convert
