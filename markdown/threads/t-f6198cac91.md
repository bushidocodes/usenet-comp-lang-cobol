All the regular screen section stuff works with Fujitsu, as well as the 
standard accept and display.  Simply display something, and a window 
will open.  You can set the screen size and font in the CBI file. When 
you set the project up, you can also specifiy wheather you want the 
accept/display treated as a window, or as an operator's console.
