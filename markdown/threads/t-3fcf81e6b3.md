Stuttgart, 29 May, 1995 -- multiX Software GmbH


PathFinder, multi-language Visual source code Browser/Analyzer
is released for
HP, SUN, IBM, LINUX platforms.
The kit is stored on the ftp server (multix.n-e-t.de 194.45.42.26).

PathFinder is able to handle over 1 million lines of code, it offers an increased
understanding of software under development, it gives you a higher productiv-
ity whether you are writing, reengieneering or maintaining C/C++, Tcl, iTcl,
Fortran, Pascal, ApplixWare ELF, COBOL, ABAP/4 or other languages.

The product package includes code parsers for diverse languages, symbol-hier-
archy browser, source code editor, retriever, diff, make, RCS-SCCS-... support.

Tcl/Tk acts as a glue language to integrate PathFinder with other frameworks or
extend base functionality with new applications. The well separated parsers
included by PathFinder can be easily replaced/extended for new languages.

Free of charge Internet evaluation kit for HP, SUN, IBM, LINUX is on the ftp server:
ftp.multix.n-e-t.de (194.45.42.26).
