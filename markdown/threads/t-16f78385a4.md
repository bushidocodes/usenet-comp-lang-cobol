J. Ranade VSAM book says:

You get this return code if the way a file is defined in the program and
the way it is allocated by Access Method Services are
incompatible.  Some of the conflicts which return this status code are:
       - A KSDS file is being opened as an ESDS or an ESDS is being
opened as KSDS.

       - The prime key length and the offset as defined in the file
description (FD) area do not match the actual file attributes.

       - An alternate key length and its offset in the file description
do not match the actual file attributes.

       - The record length in the file description is different from
that of the actual file.
