Hello everyone,

As OpenCOBOL moves toward a 1.1 release, I've been experimenting with
various mixed-language wrappers and interfaces.

Last weeks entry is an interface to Lua 5.1, embedded in OpenCOBOL.
An alpha quality release was posted to the opencobol.org forum with
sources posted to add1tocobol.com.

Single instance, single threaded, very high level embedding for this
cut.

The three trial source files
http://opencobol.add1tocobol.com/luacaller.cob
http://opencobol.add1tocobol.com/oclua.c
http://opencobol.add1tocobol.com/oclua.lua

Initial (very initial) documentation, using ocdoc extractor and ReST.
http://opencobol.add1tocobol.com/oclua.html

Cheers,
Brian Tiffin
