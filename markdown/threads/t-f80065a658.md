```
I just upgraded my MicroFocus Cobol from v3.0 to v4.1 running on an AIX
4.3.2 platform. Since then, my EOL (Erase to End of Line)  key on my
ibm3151 terminals does not operate properly.  The terminfo file looks
fine. I re-tic'd the file. Using keybcf to set the ADIS keys seems to
accept the keystroke. But instead of  deleting from the cursor to the
end of the line, it clears the field and puts a capital "I" in the first
space. The sequence for Erase to End of Line is "ESC I".  At a unix
prompt, pressing the key sends an "ESC I" as it is suppose to.

Why is Cobol not handling it properly?  An suggestions appreciated.

Phil Ross
pross@freeway.net
```
