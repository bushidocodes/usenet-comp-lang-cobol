rn> perform until no-more-masters and no-more-trans
rn> perform 02100-process-records

This is a logic error. If you perform both reads until you reach EOF on
both files you'll read past EOF on one of them if they don't have the same
number of records. You have to use the clasic "Master Update" coding to do
this.
