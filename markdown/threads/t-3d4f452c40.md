```
The newsgroup is bit.listserv.ibm-main .  The corresponding list is
IBM-MAIN and is run by BAMA.UA.EDU.
You can read the newsgroup and post to it, but if you post to the
newsgroup it will only be seen by others who access ibm-main via the
newsgroup, not by readers of the list (I have no idea what proportion of
the readership this would be).  For postings to be seen by both readers
of the list and readers of the newsgroup, you must formally subscribe to
the IBM-MAIN list, which will then allow you to post to the list (by
emailing your posting to ibm-main@bama.ua.edu)  Postings to the list are
automatically sent to the newsgroup as well.  The reverse is not true in
order to isolate list receivers from SPAM and other undesirable posts. 
If you prefer to read via the newsgroup, one of the options after
subscribing is to set a NOMAIL option, which allows you be a member of
the list without receiving any list postings via email.
  Should you want to subscribe to the list, the procedure for formally
subscribing to the list is to send email to LISTSERV@BAMA.UA.EDU and, in
the  text of your message (not the subject line), write: SUBSCRIBE
IBM-MAIN  .  The procedure for leaving the list is similar, except that
the text of the message needs to be SIGNOFF IBM-MAIN .  Note that this
is a different address than the one used for posting items to the list.  
```
