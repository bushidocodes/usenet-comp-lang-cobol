too many useful features to list and is good for a number of languages in
addition to Cobol.  I started with it in .html, but it has syntax highliting
already set up for C and C++, etc.  I find this to be a very useful feature.
Cobol syntax hightliting was not yet setup by default but there is a simple text
file to edit it in (now supporting up to 10 or 15 langs total) and even I was
able to accomplish this easily, esp. due to the adequate help file.

with F-2.  I put thin vertical column markers on 8 and 71.  Newer versions have
autoindenting on CR.  Also a column editing feature that is not quite as
powerful as I would like:  wish it allowed multiple inserts (sounds like what

UEdit is not free ware, but a time limited shareware.  Not too  expensive.
