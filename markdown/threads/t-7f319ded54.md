```
Use inspect or a subscripted byte by byte examination of the field to
determine the length of it. Then either:

1. Move spaces to the myfield and use
      MOVE sourcefield (n) TO myfield (n/2+1, n)
   if your compiler allows subscripted moves.

or

2. Move spaces to myfield then use a subscripted byte by byte move n 
   times to move the source field into myfield appending after the
   spaces.

or

3. Try using String. I don't know if this would work.
```

```
There isn't a language feature to do this, but you can do some character
manipulation by hand.  Use INSPECT to determine starting and ending
character position and length of the field to be moved, calculate where
to move it, and then use reference modification to move it to the
desired spot.
```

```
      * !                                                            !

      * !                                                            !
      * !   Type: Cobol source file                                  !
      * !                                                            !
      * !   Desc: center a string                                    !
      * !                                                            !
      * !   Useage: CALL 'CENTER' USING BY REFERENCE In-str          !
      * !                               BY CONTENTS LENGTH OF In-str !
      * !                               BY REFERENCE Out-str         !
      * !                               BY CONTENTS LENGTH OF Out-str!
      * !                                                            !
      * !   Creation date: 5 Sep 1999, creation time: 20:35:46       !
      * !                                                            !
      * !   Author: (c) Volker Bandke                                !
      * !                                                            !
      * +------------------------------------------------------------+

       Identification Division.
        Program-Id. CENTER IS INITIAL.
       Environment Division.
       Data Division.
        Working-Storage Section.
         77 Left-spaces          PIC S9(9)  BINARY.
         77 Right-Spaces         PIC S9(9)  BINARY.
         77 Input-ptr            PIC S9(9)  BINARY.
         77 Output-ptr           PIC S9(9)  BINARY.
        Linkage Section.
      * String to be centered
         01 Text-in-string.
            05 Text-in-chars     PIC X OCCURS 1 to 32767 TIMES
                                       DEPENDING ON Text-in-length.
         01 Text-in-length       PIC 9(9)    BINARY.
      * Area to receive centered string
         01 Text-out-string.
            05 Text-out-chars    PIC X OCCURS 1 to 32767 TIMES
                                       DEPENDING ON Text-out-length.
         01 Text-out-length      PIC 9(9)    BINARY.
       Procedure Division USING Text-in-string
                                Text-in-length
                                Text-out-string
                                Text-out-length.
       CENTER-entry.
           MOVE SPACES TO Text-Out-String
           IF (Text-in-string NOT = SPACES)
               PERFORM VARYING input-ptr
                       FROM Text-in-length
                       BY -1
                       UNTIL Text-in-string(Input-ptr:1) NOT = SPACE
               END-PERFORM
               COMPUTE Right-spaces = Text-in-length - input-ptr
               PERFORM VARYING Input-ptr
                       FROM 1
                       BY 1
                       UNTIL Text-in-string(Input-ptr:1) NOT = SPACE
               END-PERFORM
               COMPUTE left-spaces = Input-ptr - 1
               COMPUTE Output-ptr = 1 + (  text-out-length
                                     - text-in-length
                                     + left-spaces
                                     + right-spaces
                                    ) / 2
               IF Output-ptr < 1 THEN
                  COMPUTE Input-ptr = Input-ptr
                                     - output-ptr
                                     + 1
                  MOVE 1 TO output-ptr
               END-IF
               MOVE Text-in-string(input-ptr:)
                 TO Text-out-string(output-ptr:)
           END-IF
           GOBACK
           .
```
