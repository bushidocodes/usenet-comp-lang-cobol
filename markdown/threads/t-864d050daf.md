I just read an announcement that Fujitsu will release a COBOL compiler
next year which will take advantage of the recently announced
Microsoft.Net framework's common runtime support for multiple
development languages, including COBOL, Visual Basic and C++.

tools, in conjunction with the framework, will enable developers to more
easily build Web sites that incorporate existing COBOL applications".
