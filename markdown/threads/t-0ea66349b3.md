Several years ago I had a sales brochure for a product to do this. I
l believe the product was called RECODER and the company was called
Language Technology, Inc. I don't know if they are still in business
under that name.

I have used IBM COBOL Structuring Facility, but my company cancelled
the license because not enough programmers were using it. It required
TSO and the PL/1 runtime library to execute. It had quite a few
options you could change for each translation. It did an admirable
job of removing GO TO's and ALTER's, but I was a little disappointed
with it. In my opinion it does not create well-structured programs,
it simply regenerates the program with pure structured procedural
control. If the input program has bad datanames and paragraph names
going in, it will still have bad names after restructuring. After
restructing, the output program generally has FEWER paragraph names
than the input program, but you can request that the missing paragraph
names be generated as comments in the output program.

It was quite educational using it, however. It can generate paragraph
numbers or prefixes to indicate the heirarchy of the program, but they
were not like my shop's standards. You can select an option to
generate "called by" comments at the beginning of each paragraph (I
thought this was fairly useful). If your input program had really bad
cases of PERFORM THRU, Cobol/SF would generate flags in the data
division to manage conditional calls for the generated paragraphs.

One of the biggest surprises was how often Cobol/SF would report dead
code in the input program that it could NOT generate into the output
program! It would place all the dead code at the end of the output
program, commented out of course. Cobol/SF also produces extensive
metrics reports on both the input and output programs.

I wanted my company to keep it around, but the bean-counters said no.
We were not doing any mass conversions, but occasionally we have to
add some maintenance to some really old and ugly program. I thought
it was a nice tool to have.
