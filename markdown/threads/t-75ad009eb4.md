If no one has the text file, check if one of your friends has a scanner to

Incidentally, reserved words are a pet peeve of mine. In the first place
the only reason reserved words are used in any language is to make life
easy for he compiler writers, not the users. The text can be syntactically
analyzed to determine what the meaning really is. I've written enough
interpreters, assemblers, and compilers to know this can be done.

More than once I've had to change a variable or subroutine / function
because the language vendor decided to add another reserved word on an
update.

Secondly, if reserved words ARE used, don't make them the same as people
might want to use.
