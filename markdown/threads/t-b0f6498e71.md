There isn't one.  One of the main reasons is that each of the COBOL
vendors supports the interlanguage calls differently.  The COBOL
standard for calling programs does not (yet) support a standard method
for calling, C, Pascal, C++ etc programs.  Each is different.

However, once you learn to interpret the API you can usually figure it
out.  (With some help from everyone here!).  What compiler are you
using?  There are specific hints for each.
