I wrote an assembler routine to do this - I can't give you the code (due to
copyrights), but I can give you hints on how to do it.

Check out the VSAM "ENDREQ" macro in the "MVS Auth Assembler Services Guide"
bookserver manual.  The output ACB is allocated to the Internal Reader and
the JCL records are written to it.  Upon close, the ENDREQ macro will return
the JES JOB number that was submitted.  I believe there is also example code
in one of the Assembler manuals on this.

HTH - TL

If you really want a method, not the one I'ld use of course, but 
something simple (to write), run the COBOL program as a batch TSO/ISPF 
session, call the TSO environment from the COBOL program to run a 
CLIST/REXX.  CLIST/REXX uses OUTTRAP, SUBMIT and then analyses the 
trapped output to extract the jobid.  Inserts JOBID into ISPF variable, 
and returns.  COBOL program extracts JOBID from ISPF variable.

Otherwise it's into assembler.

James Campbell
