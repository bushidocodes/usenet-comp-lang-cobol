If you are in a Windows environment, Azalea Software and Worthington Data
Solutions offer TrueType barcode fonts.

Another solution would be a downloadable soft font. We carry font
generators for Code 128 and Code 39; you specify bar width, height, etc.
and the program builds the font. Then just copy it to the printer and
change to the barcode font whenever you need it.

If you have a choice of barcode types, Code 39 is easy to use because it
does not require the addition of a check digit.

Check our page at http://www.mecsw.com for more information.
