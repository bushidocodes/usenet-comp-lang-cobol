Yes. You cannot "create" the icon with PowerCOBOL (you need an icon editor
for this and there are several good ones on the Web, I recommend
MicroAngelo), but, once you have it, you can put it into your Project as an
icon. Alternatively, simply install your application using the PowerCOBOL
installer (read the addenda to "COBOL" and "PowerCOBOL") and then make your
shortcut on the desktop in the usual way with Windows. You will then have a
default icon (it's actually the COBOL97 icon) which you can change to be
your selected icon, using the normal facilities of Windows.

I have hand-crafted icons for all my PowerCOBOL applications (I'm very proud
of some of them :-) ) and install them by either of the 2 methods described.

If this is not enough for you to get it working, or you need more help, mail
me via my Web Site or personally to: pete_dashwood@compuserve.com

Pete.
