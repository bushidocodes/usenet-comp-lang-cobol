Hi Scott,
Yes, Micro Focus does support using MTS. You could do it using the
API/interfaces for it, which is fairly painful, or you could use the
Object COBOL run-time support provided in MF NetExpress (1.0, or
preferably the newer 2.0). This is much easier, especially using
the wizards available in 2.0.

There's an example of using MTS with Object COBOL in the product,
plus another on WebSync (the web update/contact site for product users).

Web Site: http:\\www.microfocus.com contains information about
NetExpress (see the Distributed Computing option).

Hope that helps,
Steve.
Micro Focus.
