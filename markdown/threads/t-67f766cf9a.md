› Is there any COBOL for Windows and if yes , how much it costs.

Do you want to develop Windows Applications, or just use an
editor/compiler etc. in a Windows environment to write DOS
applications?

Microfocus Workbench 3.2 has a very nice Windows development
system for writing DOS applications, although it is very
memory intensive. I believe you can also use the same
package for writing Windows applications, but I have never
done this, so don't know what it is like. There is an
add on product called Dialog which I believe is pretty
decent as a screen painter. I think the cost of WB 3.2 is
around 2500 uk pounds.

An alternative is to use Visual Basic as the screen handler
and link this in to either Microfocus or Microsoft COBOL
applications. The Microsoft COBOL compiler is actually
the Microfocus one but cost around 500 uk pounds!! The
development tools are no where near as good though, as
you just get PWB (as with MS C v6).

Hope this helps

Scott Clark
