MsgDeliverySequence is a queue definition property.  Even if the definition
is changed on the fly, the messages currently in the queue are not
reordered.

You can influence the order in which messages are retrieved from the queue
by using the Priority field in the message descriptor when the message is
inserted.   Check the manuals for details.
