```
Hello,
I have a problem reading a BTRIEVE file by calling EXTFH. When I call EXTFH
with special code "06" (return file information) i get the information that
the file is BTRIEVE / ANSI format. This works.
But when i want to open the file calling code "00" (open input) or "02"
(open I-O) i get status 9/141 (already open).
I tried to close the file --> status 9/142 (not open) !
The error occured with NetExpress 3.1.11. The program works with workbench
4.0.32.
BTRIEVE version is 6.15.

Thank you for your help,

Markus
```
