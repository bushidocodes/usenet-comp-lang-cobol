> Is there a way to move an entire PDS to a server splitting up each PDS
> member into individual files?  Please provide example code.

Convert your PDS to a flat file with IEBPTPCH, like this:
---------------------
//PUNCH  JOB stuff
//COPY1    EXEC PGM=IEBPTPCH
//SYSPRINT DD SYSOUT=*
//SYSUT1   DD DSN=PDS.INPUT,DISP=SHR
//SYSUT2   DD DSN=FLAT.OUTPUT,DISP=(NEW,CATLG),
//    ... you'll need RECFM=FBM,LRECL=81 ...
//SYSIN    DD *
      PUNCH   TYPORG=PO,MAXFLDS=999
      RECORD  FIELD=(80,1)
      LABELS  DATA=NO
/*
-------------------
I use Easytrieve to format it into IEBUPDTE input records and reformat into
FB 80 recs.

//STPU010  EXEC PGM=EZTPA00
//SYSPRINT  DD SYSOUT=*
//INFILE    DD DSN=&&TEMP1,DISP=SHR
//OUTFILE   DD DSN=GRPH.SEL.FLAT,
//       .. this one is FB 80...
//SYSIN     DD *
FILE INFILE
    INFILE-HEADER     2  11  A
INFILE-FILENAME  15   8  A
    INFILE-RECORD     2  80  A
FILE OUTFILE FB(0 0)
    OUTFILE-RECORD    1  80  A
WS-OUT              W 29  A
WS-OUT1   WS-OUT      21  A VALUE './       ADD    NAME='
WS-INFILE WS-OUT +21   8  A
JOB INPUT(INFILE)
    IF INFILE-HEADER  = 'MEMBER NAME'
        WS-INFILE     = INFILE-FILENAME
        INFILE-RECORD = WS-OUT
    END-IF
    PUT OUTFILE FROM INFILE-RECORD
/*
--------------------------
I do a lot of mass modification to JCL PDSs, so I like the output of this -
an entire PDS as a flat file.  I can edit with SPF using excludes, overlays,
everything.  It's faster for me than FileAid.  I use IEBUPDTE to make a PDS
out of it again.

To get to my PC, I download this flat file from the mainframe, then use this
Fujitsu COBOL program:
--------------------------
 IDENTIFICATION DIVISION.
  PROGRAM-ID. EXPLODE.
  AUTHOR. CLIFF PETERSON
          PETERSON TECHNOLOGY, INC.

 ENVIRONMENT DIVISION.

  CONFIGURATION SECTION.
  SPECIAL-NAMES.
     ENVIRONMENT-NAME  IS ENV-NAME
     ENVIRONMENT-VALUE IS ENV-VAL.

  INPUT-OUTPUT SECTION.
  FILE-CONTROL.
       SELECT IN-FILE
         ASSIGN TO INFILE
         ORGANIZATION IS LINE SEQUENTIAL
         FILE STATUS IS IN-STATUS.
       SELECT OUT-FILE
         ASSIGN TO OUTFILE
         ORGANIZATION IS LINE SEQUENTIAL
         FILE STATUS IS OUT-STATUS.

 DATA DIVISION.

  FILE SECTION.
  FD  IN-FILE.
  01  IN-RECORD.
      05  FILLER            PIC XX.
          88  IN-HEADER               VALUE "./".
      05  FILLER            PIC X(19).
      05  IN-FILE-NAME      PIC X(8).
      05  FILLER            PIC X(51).

  FD  OUT-FILE.
  01  OUT-RECORD.
      05  FILLER            PIC X(80).

  WORKING-STORAGE SECTION.
  01  IN-STATUS             PIC 99    VALUE ZERO.
      88  INFILE-EOF                  VALUE 10.
      88  INFILE-OK                   VALUE 00 10.
      88  INFILE-NOT-OK               VALUE 01 THRU 09
                                            11 THRU 99.
  01  OUT-STATUS            PIC 99    VALUE ZERO.
      88  OUTFILE-EOF                 VALUE 10.
      88  OUTFILE-OK                  VALUE 00.
      88  OUTFILE-NOT-OK              VALUE 01 THRU 99.
  01  FILE-NAME             PIC X(12).
  01  FILE-NAME-SUF         PIC X(4)  VALUE ".TXT".

 PROCEDURE DIVISION.

      PERFORM OPEN-INFILE.
      PERFORM OPEN-OUTFILE.

      PERFORM READ-INFILE.
      PERFORM UNTIL INFILE-EOF
          IF IN-HEADER
              PERFORM CHANGE-DEST
          ELSE
              PERFORM WRITE-OUTFILE
          END-IF
          PERFORM READ-INFILE
      END-PERFORM.

      PERFORM END-PROGRAM.

 CHANGE-DEST.
      CLOSE OUT-FILE.
      STRING IN-FILE-NAME
             FILE-NAME-SUF
             DELIMITED BY " "
             INTO FILE-NAME
      END-STRING.
      DISPLAY "OUTFILE" UPON ENV-NAME.
      DISPLAY FILE-NAME UPON ENV-VAL.
      OPEN OUTPUT OUT-FILE.

 OPEN-INFILE.
      OPEN INPUT  IN-FILE.
      IF INFILE-NOT-OK
          DISPLAY "ERROR OPENING INFILE " IN-STATUS
          PERFORM END-PROGRAM
      END-IF.

 READ-INFILE.
      READ IN-FILE.
      IF INFILE-NOT-OK
          DISPLAY "ERROR READING INFILE " IN-STATUS
          PERFORM END-PROGRAM
      END-IF.

 OPEN-OUTFILE.
      OPEN OUTPUT OUT-FILE.
      IF OUTFILE-NOT-OK
          DISPLAY "ERROR OPENING OUTFILE " OUT-STATUS
          PERFORM END-PROGRAM
      END-IF.

 WRITE-OUTFILE.
      WRITE OUT-RECORD FROM IN-RECORD.
      IF OUTFILE-NOT-OK
          DISPLAY "ERROR WRITING OUTFILE " OUT-STATUS
          PERFORM END-PROGRAM
      END-IF.

 END-PROGRAM.
      CLOSE IN-FILE
            OUT-FILE.
      EXIT PROGRAM.
