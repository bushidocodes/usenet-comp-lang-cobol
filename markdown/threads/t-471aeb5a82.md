>Without the LOCK MODE IS AUTOMATIC phrase in the SELECT statement, the
>file is opened as if LOCK MODE IS EXCLUSIVE was specified. A file so
>opened (EXCLUSIVE) is inaccessible to other programs.
>

One Minor correction.  There is a difference with Fujitsu and
specifying Lock Mode Exclusive and not specifying a lock mode at all.

If LOCK MODE EXCLUSIVE is specified, files are locked EVEN if open
Input.  With no lock mode specified, files are locked when opened for
I/O - but are in shared mode if open input.
