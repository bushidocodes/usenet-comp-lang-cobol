Fcb not found means that you are using a *very* old compiler.  FCB's
dissappeared completely with windows.  DOS version 5 was the last that
supported them, and they have been obsolete since DOS version 2.

What message do you get with Fujitsu?  I would suspect that it will not link
because your path is not set correctly, but it is hard to know without an
error message.  Did you do a full install from the CD?
