just fine. You have to be carefull using the RTE .int and .gnt format for shipping as there is
an annual license fee to pay for each machine it is run on, which is ok if it is only to be run
on 1 or 2 machines, but gets a bit expensive when you get up 3000 PCs.

Use the COB command for linking the items - it seems about the easiest way.
