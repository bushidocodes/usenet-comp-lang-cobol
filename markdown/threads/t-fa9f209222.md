```
We have a MF COBOL package that can produce printed output...currently
plain and simple text but we have plans to improve this in the not too
distant future.

At the moment we are looking for a quick improvement to our support of
page printers with pre-printed stationery - and this simply relates to
micro-adjustments to ensure alignment within the pre-printed boxes
etc. Obviously on continuous stationery there can be as much
adjustment, by tractor positioning etc., as the user can be bothered
to make, but that's not the case with page printers. And so we're
looking for a software solution. The output data is simple text
without control characters (so it should print anywhere) and it's easy
to introduce an extra character(s) or line(s) at the left and top of
the page (which we have done), but are there any printer controls that
allow for finer control ?

I've looked through some Epson ESC P and HP PCL specifications but
without success.

There doesn't seem to be an easy route to the full specs. on the web
(although I suppose buying a manual may get more information but I
don't want to do that if the controls we want aren't available!) but
while I have found a lot of information nothing appears to relate to
our requriements.

Ideally some control command to start printer "y" things down from
standard top of page and "x" things in from the left (either globally,
or we could issue this with each data line).

Any suggestions gratefully received.

(BUT, please don't mention PDF, XSLfo etc - I did mention longer-terms
plans, didn't I?)

graham
Olympic Computers Ltd.
```

```
Yes, one way would be to use escape, or control sequences.  For
example:
*p520x690Y @ADDR1---------------------------------@
*p520x740Y @ADDR2---------------------------------@
will allow you to print 2 lines of an address at an exact position.

I use this technique for printing giro payment slips, with different
fonts, boxes, circles, optical character recognition fonts, etc.  It
is fairly simple once you have learnt the techniques.  I will post on
my website a more detailed example - within the next day or so:  see
www.willowtreecomputing.com

Another method might be to do the printing via Microsoft Word, giving
you very good control where everything goes.
```
