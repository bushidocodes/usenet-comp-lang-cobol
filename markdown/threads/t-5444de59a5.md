Hmm. first, its *bad* to mix programming languages. I suppose you are
porting Access Basic(and embedded SQL) to COBOL and SQL.

So what I would do first is abstract two calls

CALL EXECSQL USING SQL-STRING
CALL PREPARESQL USING SQL-STRING

then compare compilers, and choose whichever you want.
