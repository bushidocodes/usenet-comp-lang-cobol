"Drops the string value..." drop it where? You can't place text just
anywhere on a form; you have to place text into a control. Since you can't
_create_ a control on the fly, you have to use some other technique. You
could, I guess, have an invisible or off-screen text box and, at the
mouse-up event, move the receiving box to the mouse location and insert the
text.

To answer your question directly, you cannot associate text with a mouse
pointer.

What you want to do is use the actual text in the text box as a mouse
pointer, right?

If it is so, you will have to do some hoop jumping, i.e. you will have to
make your string into a bitmap and use it as a drag-drop mouse pointer.  You
can do that by getting the DC of the text box, create a compatible bitmap
and copy the DC into it.  Make it transparent (not sure how with Fujitsu,
with NetExpress you would use a transparent image list) and assign it as
your mouse pointer while you're dragging.

Don't want to get into technical details, but that's how I would do it.

Good luck!
