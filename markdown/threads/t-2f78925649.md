```
XMLBooster for COBOL - Press Release - November 2001
* * * * * * * * * * * * * * * * * * * * * * * * * * *

XMLBooster is an XML parser generator which is used
to integrate XML parsing services in COBOL applications.
Version 2.3.11 supports:

- Generation of high-performance parsers (XML to a COBOL
  data structure) and unparsers (COBOL data structure
  back to an XML character stream) in the form of plain
  COBOL copy books to be included royalty-free in
  application programs.
- XMLBooster-generated parsers take the incoming XML
  messages (or files), and fill an easy to use, strongly
  typed COBOL structure that can be used by application
  programmers without XML-specific knowledge.
- Generation of standard COBOL code, that can be used
  with most compilers on most platforms, including Win32,
  Unix, MVS and VMS.
- Support for user-defined entities.
- Support for regular expressions which are used to
  validate incoming data at parse time.
- Generates HTML-based cross-referenced documentation,
  explaining the intricacies of the XML format it parses,
  far more efficiently than plain DTD's.
- Improved support for schemas and DTD's.
- Supports COBOL, C, C++, Java and Delphi for convenient
  multi-language infrastructures.

More information available at http://www.xmlbooster.com
```
