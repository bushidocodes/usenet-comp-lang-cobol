Depends on the file system, data, and phase of the moon. In some cases the
file size increase by 100% or more(1), in other cases the file size may
shrink by 50% or more(2).

For example:
(1) File consists of data that are all keys and alternate keys (NAME, ZIP,
PHONE)
(2) File is mostly repeating characters (like blanks) and the file system
does run-length-encoding(RLE) to compact the data.
