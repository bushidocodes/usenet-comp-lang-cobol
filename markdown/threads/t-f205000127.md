Hi Bernado.

Find the version  by typing

cob -V

If you are using COBOL 4.1 or earlier, you can only animate a .int file, if
you are using ServerExpress, then you can animate .gnt or executable files
also.

If you have a .int file try typing

anim source_program.int

Hope this helps.
