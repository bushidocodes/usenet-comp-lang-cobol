Jin Zhang,

Any code you write using Acucobol can be used in a Client/Server mode.

The client/server setup is transparent to your cobol code and there are
no unique coding requirements necessary if you are using the Acucobol
Vision file system.

Whether or not you need bridging software like ACU4GL or ACUServer
depends how your system is configured and really is not a concern for
the cobol programming techniques.
