In addition to what Bill Klein said on the subject of file opens and closes,
if you have opened a file for some reason in processing before a
USING/GIVING sort, where that file will be input to the sort, you must close
the file _before_ the sort is executed.

I also note that _all_ your files appear to be assigned to a floppy disk
drive (a:). Files on a floppy disk cannot be opened simultaneously.

I don't know the details of your homework assignment, but it appears to me
that the logic in your program won't accomplish whatever it is that you are
intending to do without some significant rework. Good luck on getting it
