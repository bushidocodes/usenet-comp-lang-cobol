Forwarded note:

1. The ITEM-ACCEPTED event is generated by doubleclick on a list
item.

2. There is no easy way to do this. You could create the window
in DS, but you will have to move it yourself (e.g. using
the dsmovsz canned dialog).

3. DS Data Access is reasonably easy to use. I would expect the
reads done using Data Access to be similar in efficiency to reads
you code yourself, but, obviously, this depends on the circumstances.
