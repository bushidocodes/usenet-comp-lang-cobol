You must handle ALL 01-levels in Linkage Section (other than the 01-level
that just has a list of BLL-cells in it - this entire 01-level should be
deleted along with any code that referenced it or its elementary items).
There really is no need (that I know of) EVER having the S9(08) COMP field
(and probably few reasons to even have any USAGE POINTER item).

What you want to do is change your EXEC CICS statements so that they
"automatically" set the address of each "real" 01-level item FOR YOU -
without any computes or explicit SET ADDRESS OF statements.

The only time you should need to use an explicit SET ADDRESS OF statement is
if you are being passed a pointer (for example in your COMMAREA).

Does this help? If not hopefully one of the people who is doing actual CICS
coding today (I haven't done it in years) can give you some more concrete
examples of what to code.
