```
One way is to have a THIRD program (FILEIO) which has the FD and I/O
section and all that. Thus take the I/O out of PRG1 and PRG2. Then
call FILEIO to do the I/O. You need to set up a linkage section with
1) some control information (do you want open/close/read/write, etc)
2) the record layout.
```
