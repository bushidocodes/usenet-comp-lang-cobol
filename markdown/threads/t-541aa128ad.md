Tommy,

Accessing arrays of Strings is more complicated because, as I am sure you
have discovered, Strings are not native types they are Objects. Therefore,
what you need to do is use the ObjectArray functions such as
GetObjectArrayElement and SetObjectArrayElement. Updating a String in the
array will require use of functions such as NewStringUTF etc to create new
strings to put into the array.

Have fun...
