```
Are you defining the file properly in the FILE SECTION? Your FD
must contain a RECORD VARYING clause. For example, to define a 1500
byte variable length file your FD should look something like this:

environment division.
input-output section.
file-control.
select varlength-file assign "varlength.dat"
organization line sequential.
data division.
file section.
fd varlength-file
record is varying in size from
0 to 1500 characters.
01 varlength-rec pic x(1500).
```
