Vic,

The 9022 error code is telling you that it is unable to open or close the
file. In this case, the file for printing.  The 90 is an error about File
opening and closing and the 22 is the type of error related to 90. I suspect
it is the error of FORM RTS, which is related to Power COBOL. You can read
this in the Fujitsu COBOL Users Guide 4.0 on Appendex B which starts on page
927, which is either in the manual or on the CD.  Hope this helps.
