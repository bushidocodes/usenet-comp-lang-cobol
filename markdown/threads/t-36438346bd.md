Se si deve importare dei file dal formato COBOL all'Access, o viceversa, si
deve avere una minima conoscenza del linguaggio COBOL, perche' si deve fare
un programma di lettura/scrittura in questo linguaggio in quanto i file
COBOL sono in formato fisso e devono essere dichiarati nell'apposita sezione
del linguaggio, dopodich� con comandi SQL si importa nel DataBase.

Per l'importazione il programma COBOL deve leggere dal suo archivio (formato
proprietario) e con i comandi SQL inserire nel database.

Per esportare dal database il programma COBOL deve leggere dal database con
i comandi SQL appropriati e poi scrivere nel file COBOL.

Ricordarsi che nel pannello di controllo alla voce ODBC Driver deve essere
dichiarato il collegamento al database perch� il COBOL si connette con il
comando CONNECT TO <nome_collegamento> dove <nome_collegamento> e la
descrizione del collegamento da pannello di controllo.

P.S.
Non so se l'MSCobol ha la possibilita' di gestire i driver ODBC in quanto si
tratta di un vecchio compilatore (per DOS). Per fare questo consiglio di
usare o il NetExpress (COBOL della Micro Focus per Windows) oppure quello
della Fujitsu (anche quello per Windows). Tutte e due i compilatori hanno la
possiilit� di collegarsi con i drivers ODBC.
