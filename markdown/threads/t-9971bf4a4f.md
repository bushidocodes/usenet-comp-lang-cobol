MS Cobol version 5 was a repackaged version of MicroFocus Cobol/2 3.0.  MF 
may well have something to say if you have 'stolen' it or gave it away.

However, based on an assumption that you bought the product yourself years 
ago then neither MS nor MF could complain about you continuing to use it 
even if you had shelfed it for a decade or so.

The licence for MS Cobol 5 includes right to distribute the programs you 
compile with this product without any run time charges.

It can produce DOS and Windows 3.x programs.

Windows XP apparently does not support this version of Cobol, but Win98 and 
W2K should be OK.
