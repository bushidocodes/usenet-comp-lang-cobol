Liant sells a product called "Relativity" that permits any ODBC-enabled
database or tool (i.e.  MS Access, Excel, Visual BASIC, Crystal) to
directly access COBOL data.  Going the other way, Liant also sells a
product (RM/PlusDB) that allows COBOL to transparently reach into
relational databases like Oracle and Ingres.

If this is what you're looking for, you can call 1-800-RMCOBOL for more
info or write to info@rmc.liant.com.
