You are mistaken, the example that I attached is in fact at field level,
which is what you asked?.

Take the code again and understand it.

           move p2ce-button3down to EventValue
           move "mouseclickedC " to methodName
           invoke EntryCallback "new" using methodname
               returning aCallback
           invoke oTextEditField "setEvent" using EventValue aCallback

There is a callback which is set to oTextEditField (an instance of an
particular edit field)

So when p2ce-button3down in oTextEditField, method methodName in this case
"mouseclickedC " is invoked.

However there is an easier way from doing this(depending on your structure)
code as follows:

           map event p2ce-button3down upon oTextEditField to  
               method z"mouseclickedC" upon self 
  
map event is syntax defined from the standard MF guibase.if file.
