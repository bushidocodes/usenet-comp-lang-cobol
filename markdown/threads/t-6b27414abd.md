CBX, The COBOL Apprentice

At ftp.worcester.com (which is 207.60.25.1), I have placed an MS-DOS and Linux
executable for making systematic source code enhancements and alterations to
COBOL and COBOL related languages, such as DB/2 and CICS imbedded in COBOL.

This program employs expert-system technology (it's rules based) to make
changes, and is not an editor based system. The program exists as shareware
and various kinds of upgrades are available to increase functionality.

With this technology, it is very easy to build executable scripts which can
correct source code with YEAR-2000 problems. These scripts are actually rules
(and make use of CLIPS, a NASA built expert-system.)

During this process, source code to be operated on is temporairily converted
into facts, modified, and converted back into source code. All whitespace and
comments are preserved intact, of course.

The shareware version does not provide any provision for writing or building
rules, and so each rule must be designed and hand-written by the maintainer.

But the marketed version has no such limitation, and in fact, the COBOL
Apprentice ('CBX') gains enough information from examining the hand-prepared
examples (in most cases, these are just collections of 'before' and 'after') to
build complete 'programs'. These are rule scripts, really, and are suitable
for correcting COBOL and related sources (such as DB/2 and CICS) without manual
editing.

Rules constructed in this way can be saved and shared with others to correct
many different kinds of source code maintenance problems. In fact, YEAR-2000
issues are only one kind of source code maintenance this technology addresses.

If you fetch this program, please let me know what you think. It's free.

The program is at 207.60.25.1 in sub-dir '/guest/jules/apprentice/cobol'. The
alias for 207.60.25.1 is ftp.worcester.com
