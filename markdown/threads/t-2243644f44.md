For those who don't understand this question, this refers to an IBM
extension that is quite old - but still supported in the latest IBM S/390
compilers.  It is similar to Panvalet and Librarian (should I say Endeavor)
library management system.  It allows you to do "virtual" updates to your
COBOL source code during a compile.

In answer to the question, both when I worked with the vendor of Xpediter
and when I worked for Micro Focus, we provided "support" for this IBM
feature - however, I never found any customers who actually used it.  I
think the basic reason was that the 3rd party products (Librarian and
Panvalet in their PRE-CA days) were considered superior and provided greater
functionality. I do remember there being 1 or 2 IBM APARs against this, so
someone must have been using it - but I sure question any common usage.
