```
A file error with a prefix of 4 means Error in Application Logic.
Specifically File Error 46 means that that a sequential read was attempted
but the end of file was already reached.

You probably need the "AT END" clause on the READ statement.

Example:

READ IN-FILE-RECORD
       AT END MOVE "YES" TO END-OF-FILE.

PERFORM 2000-PROCESS-LOOP
       UNTIL END-OF-FILE  = "YES".
```
