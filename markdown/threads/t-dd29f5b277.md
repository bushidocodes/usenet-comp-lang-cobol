```
I think every program should contain the full name of the author.
Further, if specs were written by someone else, his/her name should be
included as well.
Also, whenever a program is modified (for any reason after it goes
into production), a change log should show the date, programmer, and
reason for change. This log should be permanent no matter how long it
gets.

In the same vein, I think all program documentation, user guides, and
the like should contain the writer's full name.
Many companies seem to discourage that, but I think it's better.
Helps give pride to people in their work.
```
