```
David:

The standard way of doing this is as follows:

Code your CONTROLS ARE or CONTROLS IS clause in the RD (highest first),
e.g.:

CONTROLS ARE my-year my-month

Now code a dummy control footing group (no LINE, no COLUMN, just a
01-level entry), with the NEXT GROUP NEXT PAGE clause, e.g.

01 TYPE CF my-year NEXT GROUP NEXT PAGE.

That's all!!

There's a terrific manual available with the IBM versions of COBOL R.W.
You should be able to get a printed or a machine-browsable copy.

John Piggott.
```
