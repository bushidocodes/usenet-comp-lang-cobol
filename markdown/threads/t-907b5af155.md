You cannot call an EXE, as an EXE is the main module of an application. If
you want to call a program it has to be a DLL. However, if you want to start
another application you can execute the .EXE by using the run-time call
x"91" function 35 or CBL_EXEC_RUN_UNIT .

I hope this helps.
