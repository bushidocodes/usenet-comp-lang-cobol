Bill,
I finally got around to writing the COBOL code to get the mac address.
I used CA-Realia COBOL Workbench 3.0. The code is in the Realia cbl
format which drops columns 1-6. You will probably have to insert 6
bytes at the beginning of each line of code unless your compiler
supports this format. Obviously, if you use another compiler,  you'll
have to translate the data defs and api call syntax. I included code in
the sample program code to get the computer and user names as well. If
you or anybody else out there wants the code, email me and I'll send
it to you.
