MVS (or OS/390 or z/OS) are all IBM *mainframe* operating systems.  If you are 
using Fujitsu .NET products, you probably have no need to know more about it - 
than the fact that it isn't relevant to your environment. If you are working 
with Fujitsu to develop programs intended to RUN in an MVS (or IBM mainframe) 
environment, then you will probably need some significant training.

The COBOL source is *relatively* portable; the actual uses and interfaces are 
not.
