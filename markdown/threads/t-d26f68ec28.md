A simple Ascii FormFeed will work fine.  However, if you want to
code up the escape sequences, I do it like so:
      *The HP setup strings
       01 PAGE-SETUP.
      *set page orientation Portrait/Landscape
           02 FILLER               PICTURE X VALUE X"1B".
           02 FILLER               PICTURE XX VALUE "&l".
           02 PAGE-ORIENTATION     PICTURE 9 VALUE 0.
           02 FILLER               PICTURE X VALUE "O".
      *set fixed font (non proportional)
           02 FILLER               PICTURE X VALUE X"1B".
           02 FILLER               PICTURE X(4) VALUE "(10U".
      *set to row 0
           02 FILLER               PICTURE X VALUE X"1B".
           02 FILLER               PICTURE X(4) VALUE "&a0R".
      *set to column 0
           02 FILLER               PICTURE X VALUE X"1B".
           02 FILLER               PICTURE X(4) VALUE "&a0C".
           02 FILLER               PICTURE X VALUE "O".
      *set number of copies
           02 FILLER               PICTURE X VALUE X"1B".
           02 FILLER               PICTURE XX VALUE "&l".
           02 NUMBER-OF-COPIES     PICTURE 99 VALUE 1.
           02 FILLER               PICTURE X VALUE "X".
      *set character size
           02 FILLER               PICTURE X VALUE X"1B".
           02 FILLER               PICTURE X(4) VALUE "(s0p".
           02 CHARS-PER-INCH       PICTURE 99.99 VALUE "16.67".
           02 FILLER               PICTURE X(11) VALUE "h8.5v0s0b0T".
      *lines per inch
           02 FILLER               PICTURE X VALUE X"1B".
           02 FILLER               PICTURE XX VALUE "&l".
           02 LINES-PER-INCH       PICTURE 99.99 VALUE "08.80".
           02 FILLER               PICTURE X VALUE "D".
       01 PAGE-SETUP-ARRAY REDEFINES PAGE-SETUP.
           02 PAGE-BYTE            PICTURE X OCCURS 57 TIMES.
