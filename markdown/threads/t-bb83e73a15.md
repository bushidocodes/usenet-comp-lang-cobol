to even think about, Micro Focus COBOL allows user-written file

possible to write a file handler which handles ACU-COBOL file
formats (assuming you have access to the technical documentation for
those formats).

probably best to do a one-off conversion using an intermediate text
based file and two COBOL programs.

This is not a simple yes or no question.

Firstly, MicroFocus can either use their own ISAM format or C-ISAM files.

AcuCobol can use many different file formats in addition to their own Vision
system, including C-ISAM.

Therefore if both MicroFocus and AcuCObol are set up to use C-ISAM files the
answer is yes, they are compatible and programs can be written in either
cobol to access the data.

AcuCobol's vutil utility will also convert MicroFocus cobol files to Vision
files.

Mike.
