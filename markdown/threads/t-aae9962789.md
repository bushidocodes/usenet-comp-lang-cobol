Hi y'all,

I just translated a SAS macro to a COBOL program that takes a string
and converts it to upper and lower case. I originally wrote it to
handle strings passed from JCL parameters, but I'm sure y'all can
adapt it for other I/O functions. The indentations might be a little
uneven since I had to copy this from a notepad, but it does work.
Enjoy.


000100 IDENTIFICATION DIVISION.
000200 PROGRAM-ID. UPLOW01.
000300 AUTHOR. BOYCE G WILLIAMS, JR.
000400 INSTALLATION. VIRGINIA COMMONWEALTH UNIVERSITY.
000500 DATE-WRITTEN. 11/10/95.
000600 DATE-COMPILED.
000700*----------------------------------------------------------------
000701*
000710* CREDITS: THIS PROGRAM WAS TRANSLATED FROM A
000720* SAS MACRO WRITTEN BY LAWRENCE H MUHLBAIER
000721* (L.··.@BIO··E.EDU) POSTED IN SAS-L ON
000730* 08/26/1994.
000731*
000732* THIS COBOL-2 PROGRAM TAKES A PARAMETER STRING,
000733* CAPITALIZE THE FIRST LETTER IN EACH 'WORD' AND
000734* LOWER CASED THE REMAINDER THAT IS COMMONLY SEEN
000735* IN COMPUTER FILES. 'WORD' IS A GENEROUS TERM
000736* ALLOWING TYPICAL SEPERATION BY BLANKS, HYPENS,
000737* PERIODS, AND APOSTROPHIES. THIS PROVIDES FOR
000738* THE VARIETY OF CAPITALIZATION SEEN IN NAMES AND
000739* DEGREES. IT ALSO CAPITALIZES THE LETTER AFTER
000740* 'Mc' AND 'Mac' AND THE SUFFIX 'II' AND 'III'.
000741*
000742* AFTER THE PARAMETER STRING IS CONVERTED TO
000743* UPPER/LOWER CASE, THE PARAMETER IS PASSED BACK.
000745*
000746* EXAMPLE:
000747* BEFORE: o'daViS-mAcInTiRe, iIi, jOhN a.
000748* AFTER: O'Davis-MacIntire, III, John A.
000749*
000750* MODIFICATIONS: THE TEST LIMIT WAS SET TO FIFTY
000751* CHARACTERS. THIS CAN BE CHANGED TO THE LIMIT OF
000752* THE PARAMETER SIZE. THE LIMIT CAN BE ALSO
000753* CHANGED IF THE PROGRAM IS CONVERTED TO A CALLED
000754* SUBPROGRAM OR EVEN TO A RECORD LENGTH BY
000755* INCLUDING THE REQUIRED CODE FOR FILE HANDLING.
000759*
000760*----------------------------------------------------------------
000800 ENVIRONMENT DIVISION.
000900*----------------------------------------------------------------
001000 CONFIGURATION SECTION.
001100*----------------------------------------------------------------
001200 SOURCE-COMPUTER. IBM-3081.
001300 OBJECT-COMPUTER. IBM-3081.
001400*----------------------------------------------------------------
001500 DATA DIVISION.
001600*----------------------------------------------------------------
001700 WORKING-STORAGE SECTION.
001800*----------------------------------------------------------------
001900 01 WS-WORK-AREAS.
002000 05 WS-COUNT PIC S9(02) VALUE +0.
002010 05 WS-COUNT-LIMIT PIC S9(02) VALUE +50.
002100 05 WS-PREVIOUS-CHARACTER PIC X(01).
002200 88 WS-PREVIOUS-CHARACTER-CHECK
002300 VALUE ' ' ',' '-' ''''.

002700 05 WS-LOWER-CASE PIC X(26)
002800 VALUE 'abcdefghijklmnopqrstuvwxyz'.
002900 05 WS-UPPER-CASE PIC X(26)
003000 VALUE 'ABCDEFGHIJKLMNOPQRSTUVWXYZ'.
003100*----------------------------------------------------------------
003200 LINKAGE SECTION.
003300*----------------------------------------------------------------
003400 01 LS-PARM-AREA.
003500 05 LS-MSG-CODE PIC S9(04) COMP.
003600 05 LS-FIELD PIC X(50).
003700*----------------------------------------------------------------
003800 PROCEDURE DIVISION USING LS-PARM-AREA.
003900*----------------------------------------------------------------
004000 000-START-OF-PROGRAM.
004010*------------------------------
004200 INSPECT LS-FIELD (1:1)
004300 CONVERTING WS-LOWER-CASE TO WS-UPPER-CASE
004400 PERFORM VARYING WS-COUNT FROM 2 BY 1
004500 UNTIL WS-COUNT IS GREATER THAN WS-COUNT-LIMIT
004600 MOVE LS-FIELD (WS-COUNT - 1:1) TO WS-PREVIOUS-CHARACTER
004700 IF WS-PREVIOUS-CHARACTER-CHECK
004800 THEN
004900 INSPECT LS-FIELD (WS-COUNT:1)
005000 CONVERTING WS-LOWER-CASE TO WS-UPPER-CASE
005100 ELSE
005200 INSPECT LS-FIELD (WS-COUNT:1)
005300 CONVERTING WS-UPPER-CASE TO WS-LOWER-CASE
005400 IF WS-COUNT IS GREATER THAN OR EQUAL TO 3
005500 THEN
005600 IF LS-FIELD ((WS-COUNT - 2):2) IS EQUAL TO 'Mc'
005700 THEN
005800 INSPECT LS-FIELD (WS-COUNT:1)
005900 CONVERTING WS-LOWER-CASE
005910 TO WS-UPPER-CASE
006000 END-IF
006100 END-IF
006200 IF WS-COUNT IS GREATER THAN OR EQUAL TO 4
006300 THEN
006400 IF LS-FIELD ((WS-COUNT - 3):3)
006410 IS EQUAL TO 'Mac'
006500 THEN
006600 INSPECT LS-FIELD (WS-COUNT:1)
006700 CONVERTING WS-LOWER-CASE
006710 TO WS-UPPER-CASE
006800 END-IF
006900 END-IF
007000 END-IF
007100 END-PERFORM
007200 INSPECT LS-FIELD
007300 REPLACING ALL ' Iii' BY ' III'
007400 INSPECT LS-FIELD
007500 REPLACING ALL ',Iii' BY ',III'
007600 INSPECT LS-FIELD
007700 REPLACING ALL ' Ii' BY ' II'
007800 INSPECT LS-FIELD
007900 REPLACING ALL ',Ii' BY ',II'
008100 STOP RUN
008200 .
008300*----------------------------------------------------------------
008400* END OF PROGRAM
008500*----------------------------------------------------------------
