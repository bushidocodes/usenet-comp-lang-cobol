The book you want is the '3270 data stream programmers guide' (IBM). If you do
a search you should be able to find it online.

It's kind of heavy going and ranks up there with ESA/390 Principles of
Operation as a good book for insomniacs, but it does have everything you need
to know about writing, using, and debugging 3270 data streams.
