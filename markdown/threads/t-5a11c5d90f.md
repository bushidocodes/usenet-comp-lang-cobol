```
Hi Eric,

There are a couple of ways to do this.  One way is to create a file called
runcobol.ini and use the in2reg utility (see Appendix G of the User's
Guide).  The runcobol.ini file should look something like this:

[Defaults]
; Set it for all programs
CloseEnable=0

[MyProgram]
; Set it for one program
CloseEnable=0


Then execute the ini2reg utility with the command:

ini2reg  <pathname>\runcobol.ini

This can also be accomplished in the program by calling the C$GUICFG
subprogram (see Appendix F of the User's Guide).  For example:

      01 WS-Exit-Code       pic 9(9) binary value 0.
      01 Ws-Value.
          03                 pic x(13) value "Enable Close=".
          03                 pic x(05).
             88 EnableClose   Value "True" When False "False".
       PROCEDURE DIVISION.
       A.
           Set EnableClose to False.
           CALL "C$GUICFG" USING WS-Value, WS-Exit-Code.

Hope this helps.
```
