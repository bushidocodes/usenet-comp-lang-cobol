Although it's true that we don't have a UNIX data file editor, it is
possible to edit data files resident on a UNIX machine using the GUI
Data File Editor tool which is part of COBOL Workbench. If you have a
Windows or OS/2 pc with access to your UNIX machine.

The Data File Editor will run on OS/2 or Windows and access the UNIX
file using Fileshare which in turn uses CCI for whatever protocols you
have for connecting the two machines (typically TCP/IP).

I hope this is of some use. Please mail me if you need any more
information.
