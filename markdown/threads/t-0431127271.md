On Thu, 11 Mar 1999 22:34:28 GMT, dhawkins <dhawkins@home.com> wrote:

>1 more thing, A few of you out there have requested the "directory
>listing", is this the directory structure? or something else?

On MSDOS box, type:
dir/s "appDir">listingFile
listingFile is the directory listing of "appDir". It will contain the
list of all the files. If we can see something like "*.idx", may be we
can help you.

If you will see something.idx, look at for the same name with ".dat"
or ".fil" or other as the suffix of the name, and type:
debug something.dat
d
q
You will see the header of the file. It's useful. It can help to find
the version and the vendor of the ISAM engine
