```
The ANSI Standard itself meets this requirement.  It includes the following
statement in it,

"ACKNOWLEDGMENT
Any organization interested in reproducing the COBOL standard and
specifications in whole or in part, using ideas from this document as the
basis for an instruction manual OI for any other purpose, is free to do so.
..."

Read the full acknowledgement and act upon it as your legal advisor
recommends.

OF COURSE, it is also true that this is now an "OBSOLETE" Standard and any
documentation that you produce should indicate that.
```

```
From what I can see, you can now order a downloadable copy of the official
INCITS/ANSI 2002 COBOL standard from ANSI's standards store at
http://webstore.ansi.org/ansidocstore/ for $18.    I did a Standards Search
for the keyword COBOL to get to it.

It is (or it darned well better be!!) identical in technical content to the
ISO/IEC version also downloadable from ANSI as well as ISO for $273.

ANSI X3.23-1985 (the '85 standard itself), ANSI X3.23a-1989 (the Intrinsic
Function Amendment) and ANSI X3.23b-1993 (the Corrections Amendment) are
also downloadable from the ANSI web site for $18 apiece.
```
