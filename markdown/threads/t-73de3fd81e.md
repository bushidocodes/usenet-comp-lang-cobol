If it is a DOS box or console under Windows just make it 'windowed'
instead of full-screen with Alt-Enter then use the 'System Manu'
(click on icon on left end of caption) to select 'Edit' 'Mark' then
'Edit' 'Copy'.  This can be pasted into, say, Notepad, or an actual
text editor, and saved.

If it DOS then it is OK if it is DR-DOS as this is built-in to the
task switcher. For MS-DOS only you may be SOL.
