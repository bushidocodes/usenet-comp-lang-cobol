Micro Focus currently has "Micro Focus COBOL for Macintosh" in
field test.

Mac COBOL is a native (68k) implementation of COBOL on the Mac and supports
a full Mac look-and-feel user interface as well as execution with
the Mac Programmers Workbench. The product includes: the Checker,
Animator, Screens, Panels, Profiler, Data File Conversion tools,
native Mac graphical editor, project definition tool and session
recorder and playback utility. Support for native PowerPC execution
will be released in H2, 1995 (that is, August onwards).
