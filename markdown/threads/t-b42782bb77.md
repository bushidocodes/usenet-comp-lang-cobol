A crude but effective way to do this manually would be to define a table in
memory of 12 occurrences.  Assign each occurrence the value equal to the
number of days of the previous month since the beginning of the year.  For
example, occurrence 1 = 0, occurrence 2 = 31, occurrence 3 = 59, etc.  To
convert mmdd to ddd, add the value of the occurrence number of the month to
the number of days in that month.  For example, 05/20 would translate to 120
+ 20 = 140.  Remember to adjust for leap years.  There are several
variations very similar to this example.
