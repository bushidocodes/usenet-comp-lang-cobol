### Semantic Designs Announces COBOL CloneDR

- **From:** "Dr. Klone" <snedunuri@hotmail.com>

- **Newsgroups:** comp.lang.cobol
- **Message-ID:** `<rv2bbmk3i588@corp.supernews.com>`

```
Semantic Designs announces the availability of COBOL CloneDR -
enterprise-class software for the automatic detection and removal of COBOL
clones. Designed to handle large-scale applications (up to1 million lines of
code), the CloneDR helps organizations reduce maintenance costs, increase
programmer productivity, and raise reliability and availability.


Feature Set
--------------
Automatic Clone Detection AND REMOVAL

UNICODE support

Large-scale application support

SMP support for true parallel processing

Full-program processing support:
 Preprocessor Directives
 Data Declarations
 Procedural Code
 Comment Lines

Windows NT Runtime Environment

Evaluation Copy Availability
----------------------------
An Early Adopter Program (EAP) version of the CloneDR that detects clones
and provides an informative report is available at
 www.semdesigns.com/Products

(Note: CloneDR  Requires a Pentium class machine with at least 128M memory
running Windows NT 4.0)

Further Information can be obtained from the author or www.semdesigns.com
```
