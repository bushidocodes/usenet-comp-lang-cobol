Hi Everybody,

We are trying to upgrade one of our customers from Microsoft Cobol for SCO
Unix to Micro Focus OCDS 4.1.40 on RedHat Linux and we have one difficulty
concerning printing to the spooler or to a text file.

The case is the following:
Using Microsoft Cobol, the customer can start a program that uses the
printer using a specific switch that redirects all print jobs to a file (for
example: ./prog -pf). Without this switch every print job goes to the
spooler.

Now that he moves to OCDS he wants to keep this functionality, but there is
no such switch in OCDS (as far as I know). Is there anything we can do to
achieve this without changing the code?

Thanks in advance for any response.

Panos
