http://primacomputing.co.nz/COBOL21

New videos have been added, some of which show COBOL code before and after 
being transformed.

There is a new 5 cent tour of Code Conversion with explanation of 
Transformation and Werewolf COBOL.

A new portal is being added which will deal with converting standard COBOL 
so it is "object aware". (It is able to use objects, although it is not 
actually a true OO Class itself.) This moves on to making it actually OO 
COBOL. Some of the new videos discuss this and actually do it so you can see 
the code changes that needed. The "Refactoring" portal is still under 
development but the groundwork for much of it is in the new videos.

If you have no interest in OO COBOL or refactoring legacy into a new COBOL 
asset, then don't visit :-) (That's for Dilworth...)

If you encounter any problems with loading or viewing any of the videos 
please post to this thread.

As always, any comments, observations or questions are welcome.
