I'll be nice and supply the JCL to run the COBOL/370 compiler. It's
simular to calling COBUCL but has some differences. Check your local
mainframe computer center and see if you the latest and greatest.

//?ZUSERA JOB (####,####,99,999), ACCOUNTING INFORMATION
// '?ZUSER IGYWCL', PROGRAMMER NAME AND DELIVERY BIN
// PRTY=12, JOB PRIORITY
// CLASS=A, INPUT QUEUE CLASS
// MSGCLASS=D, PRINT DESTINATION
// MSGLEVEL=(1,1), HOW MUCH MVS SYSTEM PRINT DESIRED
// NOTIFY=?ZUSER WHO TO TELL WHEN JOB IS DONE
/*ROUTE PRINT RMT5 JES2 PRINT DESTINATION
//*******************************************************************
//* DESCRIPTIVE NAME: COBOL/370 COMPILER CALL
//*
//* NAME: IGYWCL
//*
//* FUNCTION: THIS COMPILES A COBOL/370 PROGRAM TO A LOAD LIBRARY
//*
//* INPUT: COBOL SOURCE CODE.
//*
//* OUTPUT: UPDATED COBOL LOAD MODULE.
//*
//* NOTE:
//*
//*******************************************************************
//*------------------------------------------------------------------
//*- SET PARAMETERS
//*-
// SET LIBSORC='SOURCE.CODE.COBOL', SOURCE CODE LIBRARY
// MEMSORC='(F270M010)', SOURCE CODE MEMBER
// LIBCOPY='NULLFILE', COPYBOOK LIBRARY
// LIBLOAD='COMPILE.LIBRARY.LOAD' LOAD LIBRARY
//*------------------------------------------------------------------
//*- CHECK LIBRARIES FOR EXISTANCE
//*
//STEP0001 EXEC PGM=IEFBR14
//DD1 DD DSN=&LIBSORC.&MEMSORC., SOURCE CODE LIBRARY
// DISP=SHR
//DD3 DD DSN=&LIBLOAD., LOAD LIBRARY
// DISP=SHR
//*-------------------------------------------------------------------
//*- IF OK, PROCESS SOURCE CODE THRU COBOL/370 COMPILER
//*
//STEP0002 EXEC IGYWCL,
// COND.COBOL=(0,LT),
// PARM.COBOL=('APOST', USE APOSTREOPHE AS QUOTE
// 'DYNAM', USE DYNAMIC LOADING
// 'FASTSRT', USE FAST SORT OPTION
// 'FLAG(I,E)', ALL MSGS; IMBED ERROR MSGS
// 'LIB', COPY LIBRARY OK
// 'NUMPROC(PFD)', PREFERRED SIGN HANDLING
// 'LANG(UE)', HEADING/MSGS UPPERCASE
// 'TEST(NONE,SYM)', ENABLE COBTEST HOOKS
// 'TRUNC(OPT)', OPTIMIZE TRUNCATATION
// 'OFF', MAKE CONDENSED LISTING
// 'OPT', OPTIMIZE OBJECT CODE
// 'XREF') PROVIDE IMBEDDED CROSS REF
//COBOL.SYSIN DD DSN=&LIBSORC.&MEMSORC., SOURCE CODE
// DISP=OLD
//COBOL.SYSLIB DD DSN=&LIBCOPY., COPYBOOK LIBRARY
// DISP=SHR
//LKED.SYSLMOD DD DSN=&LIBLOAD.&MEMSORC., LOAD LIBRARY
// DISP=SHR
//*------------------------------------------------------------------
//*- IF OK, DISPLAY MESSAGE OF SUCCESSFUL COMPLETION
//*
//STEP0003 EXEC PGM=IKJEFT01, TSO BATCH PROGRAM
// COND=(5,LT)
//SYSTSPRT DD SYSOUT=* STATUS MESSAGES
//SYSTSIN DD * INSTRUCTIONS
SEND 'BATCH COBOL/370 COMPILE COMPLETE' U(?ZUSER) LOGON
//*------------------------------------------------------------------
//*- IF NOT OK, DISPLAY MESSAGE OF FAILURE
//*
//STEP0004 EXEC PGM=IKJEFT01, TSO BATCH PROGRAM
// COND=(EVEN,(0,EQ,STEP0003))
//SYSTSPRT DD SYSOUT=* STATUS MESSAGES
//SYSTSIN DD * INSTRUCTIONS
SEND 'BATCH COBOL/370 COMPILE FAILED' U(?ZUSER) LOGON
//
