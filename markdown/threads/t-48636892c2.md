Hi Jack,

If your template contains JCL, you might want to use: 

//MYTMPLT DD DATA

If this occurs at the end of the stream a /* is not required.
If the data in MYTMPLT contains a /* stmt, code:

//MYTMPLT DD DATA,DLM='cc'

where, cc can be any chars that don't appear in the DD DATA stream.
The aposts are only required if cc contains special chars.

Regards, Jack.
