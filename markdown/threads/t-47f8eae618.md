I have NOT tried this (compiled this) - so my syntax may be slightly off.
However, as I understand the request, you would like a "simple" statement to
get a random number from 1 to 6. The following is a single COBOL
statement - how "simple" it is, depends on how much you LIKE the COBOL
intrinsic functions.

Compute Int-field = ((Function Mod (
Function Integer (
Function Random (
Function NumVal (
Function Current-Date
(9:6)

)

)
* 10)
5)
+ 1)
End-Compute

(The statement would be a LOT easier if you were willing to take COBOL's
"pseudo-random" numbers - but most applications are NOT,)

I hope that the indentation helps (a little) to make clear what the
functions are doing.

004000 .
004100

004300 COMPUTE RANDOM-PRODUCT = RANDOM-MULTIPLIER * RANDOM-SEED

004600 COMPUTE RANDOM-SEED = RANDOM-PRODUCT

004800

005000 DIVIDE 6 INTO RANDOM-NUMBER
005100 GIVING RANDOM-NUMBER
005200 REMAINDER RANDOM-REMAINDER
005300

Here is how I am doing it now and it works okay with only one call to the
seed method at the start of a game.

Identification Division.
Class-Id. Dice
inherits from Base with data.
*
Environment Division.
Object Section.
*
*****************************************************************
* Identify all of the classes that will be used **
*****************************************************************
*
Class-Control.
Dice is class "Dice"
Base is class "base"
.
*
Class-Object.
Object-Storage Section.
*
Method-Id. "New".
Data Division.
*
Working-Storage Section.
*
01 diceValue pic 9.
88 initDice value 0.
88 validRoll value 1 thru 6.
*
Linkage Section.
*
01 lsObjRef object reference.
*
Procedure Division returning lsObjRef.
*
*****************************************************************
* Instantiate the class so that we have object data **
*****************************************************************
*
Invoke Super "New" returning lsObjRef.
*
*****************************************************************
* set the default value for the player's name **
*****************************************************************
*
set initDice to true.
invoke lsObjRef "setDiceValue" using diceValue.
*
End Method "New".
End Class-object.
*
Object.
Data Division.
Object-Storage Section.
*
01 seed pic 9(8).
01 aRandomNumber pic 9v9(8).
01 filler redefines aRandomNumber.
05 n1 pic 9.
05 n2 pic 9.
05 n3 pic 9.
05 n4 pic 9.
05 n5 pic 9.
05 n6 pic 9.
05 n7 pic 9.
05 n8 pic 9.
05 n9 pic 9.
*
01 diceValue pic 9.
88 initDiceValue value 0.
88 goodDiceValue value 1 thru 6.
*
01 diceNumber pic 9.
88 goodDiceNumber value 1 thru 5.
*
Method-Id. "setDiceValue".
Linkage Section.
01 lsDiceValue pic 9.
Procedure Division using lsDiceValue.
*
move lsDiceValue to diceValue.
*
End Method "setDiceValue".
*
Method-Id. "setDiceNumber".
Linkage Section.
01 lsDiceNumber pic 9.
Procedure Division using lsDiceNumber.
*
move lsDiceNumber to diceNumber.
*
End Method "setDiceNumber".
*
Method-Id. "getDiceValue".
Linkage Section.
01 lsDiceValue pic 9.
Procedure Division returning lsDiceValue.
*
move dicevalue to lsDiceValue.
*
End Method "getDiceValue".
*
Method-Id. "getDiceNumber".
Linkage Section.
01 lsDiceNumber pic 9.
Procedure Division returning lsDiceNumber.
*
move diceNumber to lsDiceNumber.
*
End Method "getDiceNumber".
*
Method-Id. "showDice".
Procedure Division.
*
display 'Dice Value ('diceValue')'.
*
End Method "showDice".
*
Method-Id. "seed".
Working-Storage Section.
*
01 osCurrentDate pic x(21).
01 filler redefines osCurrentDate.
05 osCurrentCC pic 9(02).
05 osCurrentYY pic 9(02).
05 osCurrentMM pic 9(02).
05 osCurrentDD pic 9(02).
05 osCurrentHours pic 9(02).
05 osCurrentMins pic 9(02).
05 osCurrentSec pic 9(02).
05 osCurrentHS pic 9(02).
05 osCurrentGreenwich pic x(01).
05 osCurrentGreenwichHours pic 9(02).
05 osCurrentGreenwichMins pic 9(02).
*
Procedure Division.
*
move FUNCTION CURRENT-DATE to osCurrentDate.
move FUNCTION RANDOM (osCurrentHS) to aRandomNumber.
*
End Method "seed".
*
Method-Id. "roll".
Working-Storage Section.
*
Procedure Division.
*
set initDiceValue to true.
perform until goodDiceValue
move FUNCTION RANDOM () to aRandomNumber
move n9 to diceValue
end-perform.
*
End Method "roll".
*
End Object.
End Class Dice.
