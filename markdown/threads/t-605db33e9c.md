Commercial Product Announcement

I thought that some of the readers of comp.lang.cobol would be
interested in the following announcement.

Flexus has developed a conversion tool for assisting Micro Focus
Dialog System users to convert their screensets into COBOL sp2 panel
definitions.  Your converted COBOL sp2 panels can then be used with
your existing Micro Focus Net Express development system.

The conversion service is available immediately.

Micro Focus Dialog System is an outstanding COBOL user interface
development tool.  In the past, some Micro Focus Net Express customers
have communicated their desire to use the Flexus COBOL sp2 user
interface development tool so that they could use one or more of the
Flexus COBOL Tools such as Thin Client or Web Client.

Because of the extensive features in Dialog System, it can be time
consuming to manually convert your user interface screens to an
alternate format.  Our conversion service can save a considerable
amount of time by helping you avoid manual conversion.

For more information, please contact me at rtwolfe@flexus.com.


Bob Wolfe
