To use READ ... NEXT you need ACCESS IS DYNAMIC in the SELECT. (And a START,
which will fail on a sequential file anyway).

Just use READ PROV-FILE
                AT END
                   MOVE "Y" TO EOF-FLAG.


For extra style points, open PROV-FILE for INPUT rather than I-O, as you
don't need to write to it.

MCM
