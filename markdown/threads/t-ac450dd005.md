```
The long-term solution, of course, is to change the program which is locking
the record for this inordinate amount of time. A single program should not,
for ordinary maintenance (add/change/delete), require a lock for more than
the time it takes to read the record into some working-storage area, then
immediately free, delete or rewrite it.
```
