Just because SOAP has come up previously in CLC, FYI at:

    http://www.ibm.com/isource/cgi-bin/goto?it=usa_annred&on=203-199

is an announcement that starts out:

"IBM SOAP for CICS feature delivers fully supported SOAP access to CICS

At a glance
The SOAP for CICS feature provides a mechanism which allows CICS
applications, written in any supported programming language, to communicate
via the Simple Object Access Protocol (SOAP). Transports are provided over
Hypertext Transfer Protocol (HTTP) and WebSphere MQ. Both inbound and
outbound function is provided. The implementation supports SOAP 1.1."
