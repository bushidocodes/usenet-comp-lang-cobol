AcuCorp has a Linux runtime - per runtime license fee is my objection 
- it's too high.

MicroFocus has a RedHat Linux full development version - $2800 
(approx) per US seat, and $800.00 per 10 users runtime - and you have 
to know who they are  - Inadequet for Internet/CGI work - and way too 
expensive.

ICOBOL has a Linux development platform, High runtime fees.

Liant may have one - I'm not sure, Tom Morrison would know the status.

Legacy J has a Cobol to JAVA compiler that will run under Linux.  No 
runtime fees, but high acquisition cost.   Best solution to date 
because of the runtime.

There are two "free" efforts under way, one by members of the Linux 
community - to first develop an Ansi 74 COBOL for Linux, and the other
by the GNU team (and they seem to have some recent momemtum!).
