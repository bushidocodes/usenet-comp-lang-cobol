```
This is a question about how best one should do a string substitution in Cobol.

Let us assume that we have string on which we want to operate. Let's call it TARGET. We also have an OLDSTR variable and a NEWSTR variable. All of these are null terminated strings.

What we want is logic that will replace all occurrences of OLDSTR in TARGET with NEWSTR. The function should operate properly even when the OLDSTR is a different length than the NEWSTR.

I have a solution that works for my purposes, but I knew you lot would do it much more cleverly than I could have done. I am perfectly willing to replace your logic with mine. (I remember the days of Weinberg's egoless programming.)

Thank you for your time and cycles you spend thinking on this.
```

```
I considered four procedures but selected UNSTRING/STRING for
implementing due its simplicity.

Some Micro Focus extensions were used.
Micro Focus COBOL 3.2.51, last updated in 1995.

While only eleven statements are used to implement the find/replace,
there is a lot of overhead for test setup and results. Some might
find the use of Report Writer to record the results 'interesting'.


identification division.
program-id. sub-str.
environment division.
input-output section.
file-control.
select report-file assign to "sub-str.txt"
organization line sequential.
data division.
file section.
fd report-file
report rpt.
working-storage section.
1 comp.
2 counters.
3 work-fld-len pic 9(4).
3 oldstr-len pic 9(4).
3 newstr-len pic 9(4).
3 target-len pic 9(4).
3 temp-len pic 9(4).
2 pointers.
3 work-pos pic 9(4).
3 target-pos pic 9(4).
2 misc.
3 test-max pic 9(4).
3 test-num pic 9(4).
1 work-fld pic x(40).
1 target pic x(40).
1 oldstr pic x(16).
1 newstr pic x(16).
1 temp pic x(40).
1 delim pic x(16).
1 comment pic x(40).
report section.
rd rpt.
1 rpt-detail type de.
2 line plus 1.
3 column 1 pic x(40) source comment.
2 line plus 1.
3 column 1 pic x(40) value all "-".
2 line plus 1.
3 column 1 pic x(7) value "target:".
3 column 9 pic x(40) source target (1:target-len).
2 line plus 1.
3 column 1 pic x(7) value "oldstr:".

2 line plus 1.

2 line plus 1.

3 column 9 pic x(40) source work-fld (1:work-fld-len).
2 line plus 1.
3 column 1 pic x value space.
procedure division.
main section.
open output report-file
initiate rpt
move 7 to test-max
perform
varying test-num from 1 by 1
until test-num > test-max
evaluate test-num
when 1

when 2

when 3
perform test-3-prep
when 4
perform test-4-prep
when 5
perform test-5-prep
when 6
perform test-6-prep
when 7
perform test-7-prep
end-evaluate
perform sub-str-proc
end-perform
terminate rpt
close report-file
stop run
.

sub-str-proc section.
initialize counters
move 1 to target-pos work-pos
inspect oldstr tallying oldstr-len
for characters before low-value
inspect newstr tallying newstr-len
for characters before low-value
perform until exit
unstring target
delimited by oldstr (1:oldstr-len) or low-value
into temp delimiter delim count temp-len
with pointer target-pos
end-unstring
string temp (1:temp-len) delimited by size
into work-fld
with pointer work-pos
end-string
if delim (1:1) = low-value
then
string low-value delimited by size
into work-fld
with pointer work-pos
end-string
exit perform
else
string newstr (1:newstr-len) delimited by size
into work-fld
with pointer work-pos
end-string
end-if
end-perform
inspect target tallying target-len
for characters before low-value
inspect work-fld tallying work-fld-len
for characters before low-value
generate rpt-detail
.
```
