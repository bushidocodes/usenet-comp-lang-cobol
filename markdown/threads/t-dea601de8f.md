### Announce: TinyCOBOL release (0.1.5.3)

- **From:** David Essex <dessex@arvotek.net>
- **Date:** 2000-08-20T00:00:00
- **Newsgroups:** comp.lang.cobol,comp.compilers
- **Message-ID:** `<00-08-097@comp.compilers>`

```
A new version (0.1.5.3) of the Tiny COBOL compiler is available for
download at the Tiny COBOL home page (1) and at Metalab (2).

In this release, along with some bug fixes, the INITIALIZE, CURRENCY,
PERFORM WITH TEST BEFORE/AFTER option statements have been added to the
TC COBOL grammar.

Please forward any comments, suggestions to the TC mailing list (3).


David

1) Tiny COBOL home page
   http://tiny-cobol.sourceforge.net/
2) Metalab
   http://metalab.unc.edu/pub/Linux/devel/lang/cobol/
3) Tiny COBOL mailing list
   tiny-cobol-users@sourceforge.net
```
