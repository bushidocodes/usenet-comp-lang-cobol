```
I did this on the Fujitsu cobol compiler, because I don't have access
to IBM LE COBOL.  I was wondering if someone could test this and see
if it works?  I got all the compile errors out of it, but the version
of the COBOL compiler I have doesn't allow recursive subcalls in it's
execs so I can't test it.

Hope this is interesting to someone...I'm kind of doing different
things you don't normally see in COBOL just to see if they could be
done...

The main program
 CBL NOSSRANGE
 IDENTIFICATION DIVISION.
 PROGRAM-ID. QSRTMAIN.
*++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
* QUICKSORT (RECURSIVE?) MAIN PROGRAM.
* CREDIT "Sky_Eagle1 on comp.lang.cobol"
*++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
 ENVIRONMENT DIVISION.
 DATA DIVISION.
 WORKING-STORAGE SECTION.

 01  SORT-TABLE-ITEMS.
     04  SORT-TABLE       PIC 9(5) OCCURS 1000 TIMES
                          INDEXED BY INTERVAL-LEFT INTERVAL-RIGHT.
 01  TABLE-LEFT           INDEX.
 01  TABLE-RIGHT          INDEX.

 01  TEST-VARS.
     04  IN-TIME          PIC 9(8).
     04  GEN-COUNTER      PIC S9(5) COMP-3.
     04  COUNTER-1        PIC S9(5) COMP-3.
     04  COUNTER-2        PIC S9(5) COMP-3.

*----------------------------------------------------------------
 PROCEDURE DIVISION.
 0000-START-SECTION.
     ACCEPT IN-TIME FROM TIME.
     DISPLAY "BEFORE: " IN-TIME.
     PERFORM VARYING GEN-COUNTER FROM 1 BY 1 UNTIL GEN-COUNTER > 1000
        MOVE 1 TO COUNTER-1
        PERFORM VARYING COUNTER-2 FROM 1000
               BY -1 UNTIL COUNTER-2 < 1
           MOVE COUNTER-2 TO SORT-TABLE (COUNTER-1)
           ADD 1 TO COUNTER-1
        END-PERFORM
        SET INTERVAL-LEFT TO 1
        SET INTERVAL-RIGHT TO 1000
        SET TABLE-LEFT TO INTERVAL-LEFT
        SET TABLE-RIGHT TO INTERVAL-RIGHT
        CALL "QSORT5" USING SORT-TABLE-ITEMS TABLE-LEFT TABLE-RIGHT
     END-PERFORM.
     ACCEPT IN-TIME FROM TIME.
     DISPLAY "AFTER: " IN-TIME.
     PERFORM 0600-DISPLAY.
     GOBACK.

 0600-DISPLAY SECTION.
     PERFORM VARYING GEN-COUNTER FROM 1 BY 1 UNTIL GEN-COUNTER > 1000
        DISPLAY GEN-COUNTER ': ' SORT-TABLE (GEN-COUNTER)
     END-PERFORM.

The subroutine

CBL NOSSRANGE
 IDENTIFICATION DIVISION.
 PROGRAM-ID. QSORT5. RECURSIVE.
*++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
* QUICKSORT RECURSIVE SUBPROGRAM.
* CREDIT "Sky_Eagle1 at comp.lang.cobol"
*++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
 ENVIRONMENT DIVISION.
 DATA DIVISION.
 WORKING-STORAGE SECTION.
  01  SORT-PROC-VARS.
      04  TEMP-SWAP-VALUE  PIC 9(5).
      04  INTERVAL-PIVOT   PIC 9(5).
      04  INDEX-T1         INDEX.
      04  INDEX-T2         INDEX.

 LINKAGE SECTION.
  01  SORT-TABLE-ITEMS.
      04  SORT-TABLE       PIC 9(5) OCCURS 1000 TIMES
                           INDEXED BY INTERVAL-LEFT
                                      INTERVAL-RIGHT
                                      SWEEP-LEFT
                                      SWEEP-RIGHT.
  01  TABLE-LEFT       INDEX.
  01  TABLE-RIGHT      INDEX.

*----------------------------------------------------------------
 PROCEDURE DIVISION USING SORT-TABLE-ITEMS TABLE-LEFT TABLE-RIGHT.
 0000-START-SECTION.
     IF TABLE-LEFT < TABLE-RIGHT
        SET SWEEP-LEFT TO TABLE-LEFT
        MOVE SORT-TABLE (SWEEP-LEFT) TO INTERVAL-PIVOT
        SET SWEEP-LEFT UP BY 1
        SET SWEEP-RIGHT TO TABLE-RIGHT
        PERFORM 2100-SHIFT-INTERVAL UNTIL SWEEP-LEFT > SWEEP-RIGHT
        MOVE SORT-TABLE (SWEEP-RIGHT) TO SORT-TABLE (INTERVAL-LEFT)
        MOVE INTERVAL-PIVOT TO SORT-TABLE (SWEEP-RIGHT)
        SET SWEEP-RIGHT DOWN BY 1
        SET INDEX-T1 TO SWEEP-RIGHT
        SET INDEX-T2 TO SWEEP-LEFT
        CALL "QSORT5" USING SORT-TABLE-ITEMS TABLE-LEFT INDEX-T1
        CALL "QSORT5" USING SORT-TABLE-ITEMS INDEX-T2 TABLE-RIGHT
     END-IF.
     GOBACK.

 2100-SHIFT-INTERVAL SECTION.      
     PERFORM UNTIL (SORT-TABLE (SWEEP-LEFT) <= INTERVAL-PIVOT)
         SET SWEEP-LEFT UP BY 1
     END-PERFORM.      
     PERFORM UNTIL (SORT-TABLE (SWEEP-RIGHT) > INTERVAL-PIVOT)
        SET SWEEP-RIGHT DOWN BY 1
     END-PERFORM.      
     IF SWEEP-LEFT < SWEEP-RIGHT         
        MOVE SORT-TABLE (SWEEP-RIGHT) TO TEMP-SWAP-VALUE
        MOVE SORT-TABLE (SWEEP-LEFT) TO SORT-TABLE (SWEEP-RIGHT)
        MOVE TEMP-SWAP-VALUE TO SORT-TABLE (SWEEP-LEFT)
     END-IF.
```
