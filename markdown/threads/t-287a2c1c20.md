The PATH command in your AUTOEXEC.BAT file, and/or some of the environment
SET commands might have been fragged. This sometimes happens when installing
some programs that are unfriendly to your existing AUTOEXEC.BAT/CONFIG.SYS
files.

This is what is needed for MF COBOL 3.2:

AUTOEXEC.BAT:

path=..............;c:\cobol;
^^^^^^^^\
\
set cobcpy=c:\cobol\source \
set cobdir=c:\cobol \
set cobhnf=c:\cobol\help \
^^^^^^^^^^^^^^^<------ change these as needed for your system
