Using the intrinsic functions you could simply convert your date to
integer format (FUNCTION INTEGER-OF-DATE or FUNCTION INTEGER-OF-DAY),
add or subtract as needed, then convert back to your original format via
FUNCTION DATE-OF-INTEGER or FUNCTION DAY-OF-INTEGER.

Perhaps not the most efficient method, but very simple.
