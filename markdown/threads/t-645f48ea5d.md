Barcodes are only a special way to code
numbers. So you can take any data structure
which fits.
Barcodes can be read with wands, e.g. HP or
DataLogic . This Wand translates usually the
Code in ASCII-Codes which can be treated by
programs. Wands are usually connected via a
serial Interface ( COM-Port) of the PC. They
will not be used in big Iron environments.
