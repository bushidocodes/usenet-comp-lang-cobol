```
The following code-snippet might prove useful.

       IDENTIFICATION DIVISION.
       PROGRAM-ID.  SAMPLE.
      *
       ENVIRONMENT DIVISION.
      *
       CONFIGURATION SECTION.
       SOURCE-COMPUTER. IBM-PC.
       OBJECT-COMPUTER. IBM-PC.
      *
       DATA DIVISION.
      *
       WORKING-STORAGE SECTION.
      *
       01  SYSTEM-PARMS.
           05  CMD-LINE                    PIC X(130)
                                           VALUE SPACES.
           05  RDRW-CODE                   PIC X(1)
                                           VALUE "N".
           05  EXIT-CODE                   PIC S9(4) BINARY
                                           VALUE ZEROES.
      *
       PROCEDURE DIVISION.
      *
       Main-Para.
      *
           MOVE "%comspec% /c echo ABCDE" TO CMD-LINE.
           CALL "SYSTEM" USING CMD-LINE RDRW-CODE EXIT-CODE.

           IF EXIT-CODE NOT ZERO
             DISPLAY "Non-zero exit code."
             MOVE 1 TO RETURN-CODE
           END-IF

           EXIT PROGRAM.
```
