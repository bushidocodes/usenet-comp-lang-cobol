```
The most common problem I have had with Windows NT and setup/update is
that it tries to run them as OS/2 programs. You should run them with
the FORCEDOS command to stop this.

e.g.

forcedos setup

or

forcedos update

Also, once installed, you need to change the setting of Os2LibPath to
include the EXEDLL directory of the installed product(s). You can do
this from the control panel (system icon).
```
