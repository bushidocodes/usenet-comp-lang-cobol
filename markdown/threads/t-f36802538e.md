This is going to vary by language and country.  In Canada, a common
rule of thumb is to allow 25% more space for French text than its
English equivalent.

Since any system that we design these days can end up being used on a
global basis, whether intended or not, we should be designing for the
entire world.  This brings up some related questions:

1. Country Code

I believe that there is a standard code to identify countries although
I can't remember whether it is 2 or 3 characters.

2. Province/State/etc.

The Canadian and US postal systems have adopted standard 2 character
abbreviations for provinces and states.  However, how many characters
should we allow for the equivalents in other countries that form a
normal part of the address?

3. Postal Code

The longest postal code that I know of is the expanded US zip code - 10
digits.  If we use a 10 character field, is this long enough for the
rest of the world?

4. Phone Country Code

North Americans are used to a single character (1).  However, it can be
2 (44 for the UK) or 3 characters (886 for Taiwan).  But there is not
necessarily a one-to-one relationship between a political country and
the phone country code.  Canada and the US share 1.  Germany has (or
had) 2 - one each for East and West.  Are there any rules for
translating a political country code into a phone country code?

5. Phone Number

The North American standard is 10 digits - area code (3) + exchange (3)
+ local number (4).  However, other countries break down the phone
number differently.  You could allow 12 characters in a single field
including punctuation.  Is this enough for the rest of the world?

The North American phone companies can already foresee running out of
numbers with the current system and are planning to expand it.  Does
anyone know what their future plans are?  We could be looking at
another Y2K situation here.

lumped under the 809 area code), with the actual country access codes
used in international dialing. People in North America & the Caribbean

the local phone no.

999-9999

Some country codes do not have city codes.

1 is the country code.  If you were dialing the US from London, I

999 999 9999.

country its in.  For North Americans, 1 is both the country code and
