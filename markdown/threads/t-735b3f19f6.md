```
With MS 4.5 you need to call DOS using the CALL X"84" call-by-number
routine to do this as in:

IDENTIFICATION DIVISION.
PROGRAM-ID. GETFILE2.
*==================================================================

DATA DIVISION.
WORKING-STORAGE SECTION.
COPY "I:versdate.ws".

* ----------------------------------------------------------------
* Cobol 2 DOS areas
01 DOS-INT21 PIC X VALUE X"21".

01 REGISTER-FLAG PIC X.

01 DOS-AX.
03 DOS-AH PIC X.
03 DOS-AL.
05 DOS-AL-9 PIC 99 COMP.
01 DOS-AX-9 REDEFINES DOS-AX PIC S9(4) COMP.

01 DOS-BX.
03 DOS-BX-VAL PIC 9(4) COMP.
01 DOS-BX-9 REDEFINES DOS-BX PIC S9(4) COMP.

01 DOS-CX.
03 DOS-CH PIC X.
03 DOS-CL PIC X.
01 DOS-CX-9 REDEFINES DOS-CX PIC S9(4) COMP.

01 DOS-DX.
03 DOS-DH.
05 DOS-DH-9 PIC 99 COMP.
03 DOS-DL.
05 DOS-DL-9 PIC 99 COMP.
01 DOS-DX-9 REDEFINES DOS-DX PIC S9(4) COMP.

01 IN-INDEX PIC S9(4) COMP-5.
01 FILE-NAME.
03 FN-CHAR PIC X OCCURS 65.

01 DIR-ATTRIBUTE PIC X VALUE X"10".
01 R-ATTRIBUTE PIC X VALUE X"01".
01 A-ATTRIBUTE PIC X VALUE X"20".
01 S-ATTRIBUTE PIC X VALUE X"04".
01 H-ATTRIBUTE PIC X VALUE X"02".

01 WS-ATTRIBUTE PIC X.

01 DTA.
03 FILLER PIC X(21).
03 DTA-ATTRIBUTE PIC X.
03 DTA-TIME PIC 9(4) COMP-5.
03 DTA-DATE PIC 9(4) COMP-5.
03 DTA-SIZE PIC 9(9) COMP-5.
03 DTA-NAME PIC X(13).
03 FILLER PIC X(75).

*==================================================================

LINKAGE SECTION.

COPY "I:getfile.ws".

*==================================================================

PROCEDURE DIVISION USING ACTION FILE-LINK RETURN-DATA.
MAIN SECTION.
PROGRAM-GET-FILES.

D DISPLAY ACTION " " FILE-LINK

MOVE SPACES TO RETURN-DATA

IF ACTION = "S"
PERFORM START-SEARCH
MOVE "N" TO ACTION
PERFORM MOVE-RETURN
ELSE
IF ACTION = "N"
PERFORM NEXT-SEARCH
PERFORM MOVE-RETURN
ELSE
IF ACTION = "F"
PERFORM GET-FREE-SPACE
.

EXIT PROGRAM
.

MOVE-RETURN.

MOVE DTA-ATTRIBUTE TO WS-ATTRIBUTE
CALL "CBL_AND" USING DIR-ATTRIBUTE
WS-ATTRIBUTE
BY VALUE 1

IF DTA-NAME = SPACES
NEXT SENTENCE
ELSE
IF WS-ATTRIBUTE = DIR-ATTRIBUTE
MOVE "" TO RETURN-DATA
ELSE
PERFORM MOVE-FILENAME
MOVE DTA-SIZE TO RT-SIZE
PERFORM MOVE-DATE
PERFORM MOVE-TIME
PERFORM MOVE-ATTRIBUTES
.


* ----------------------------------------------------------------
MOVE-ATTRIBUTES.

MOVE "____" TO RT-ATTRIBUTE

MOVE DTA-ATTRIBUTE TO WS-ATTRIBUTE
CALL "CBL_AND" USING R-ATTRIBUTE
WS-ATTRIBUTE
BY VALUE 1
IF WS-ATTRIBUTE = R-ATTRIBUTE
MOVE "R" TO RT-ATTRIBUTE(1:1)
.

MOVE DTA-ATTRIBUTE TO WS-ATTRIBUTE
CALL "CBL_AND" USING A-ATTRIBUTE
WS-ATTRIBUTE
BY VALUE 1
IF WS-ATTRIBUTE = A-ATTRIBUTE
MOVE "A" TO RT-ATTRIBUTE(2:1)
.

MOVE DTA-ATTRIBUTE TO WS-ATTRIBUTE
CALL "CBL_AND" USING S-ATTRIBUTE
WS-ATTRIBUTE
BY VALUE 1
IF WS-ATTRIBUTE = S-ATTRIBUTE
MOVE "S" TO RT-ATTRIBUTE(3:1)
.

MOVE DTA-ATTRIBUTE TO WS-ATTRIBUTE
CALL "CBL_AND" USING H-ATTRIBUTE
WS-ATTRIBUTE
BY VALUE 1
IF WS-ATTRIBUTE = H-ATTRIBUTE
MOVE "H" TO RT-ATTRIBUTE(4:1)
.

* ----------------------------------------------------------------
MOVE-FILENAME.

UNSTRING DTA-NAME
DELIMITED BY "." OR X"00"
INTO RT-NAME
RT-TYPE
IF RT-TYPE NOT = SPACES
MOVE "." TO RT-DOT
.

MOVE-DATE.

DIVIDE 32 INTO DTA-DATE
GIVING DTA-DATE
REMAINDER RD-DD
DIVIDE 16 INTO DTA-DATE
GIVING RD-YY
REMAINDER RD-MM
ADD 80 TO RD-YY
MOVE "/" TO RD-S1
RD-S2
.

MOVE-TIME.

DIVIDE 32 INTO DTA-TIME
GIVING DTA-TIME
REMAINDER RT-SC
MULTIPLY 2 BY RT-SC
DIVIDE 64 INTO DTA-TIME
GIVING RT-HR
```
