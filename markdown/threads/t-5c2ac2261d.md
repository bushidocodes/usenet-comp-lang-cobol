It reads like you have just gotten into the EBCDIC ASCII (ISO)
conversion problem.  This is because the code for +0 or -0 on the
mainframe using your default codepage doesn't translate to +0 or -0 in
codepage on the target system.  Using SIGN IS SEPARATE on your data and
doing the appropriate conversion should work.
