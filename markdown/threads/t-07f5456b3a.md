```
Hi,

You have not linked in the runtime support, the best way of linking
COBOL and C together is to use 'cob' for creating the exe.

eg: cob -x appel.c testcob.cob

A quick example would be:

--START: bld --
cob -vx appel.c TESTCOB.cob
--END: bld --

--START: appel.c --
#include <stdio.h>

extern int TESTCOB();

int main(int argc, char **argv)
{
#ifdef _AIX
	cobinit();
#endif

	TESTCOB();

	cobexit(0);
}
--END: appel.c --

--START: TESTCOB.cob --
	display "Hello world".
--END: TESTCOB.cob --


You might also want to take a look at a demo called 'cmain.c' 
in $COBDIR/demo.

Yours
```
