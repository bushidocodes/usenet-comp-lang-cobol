Response version 1:

On the front of the computer case (the metal box thing under the desk,
or follow the keyboard cable) you will find two buttons.  The larger one
should be marked '0|1' or similar, the smaller one 'Reset'.  Press the
smaller one.

If, upon trying this program again several times, you get the same
message each time, then press the larger button once only.  You will
then not get this message again.


Reponse version 2:

This is not a MicroFocus message that I have seen before, but that may
only be because I never misused it that way.

If this is a program that you wrote yourself then you should be able to
animate it, or debug it, to locate where it is occuring.  

If it is not a program that you wrote then it may be a message written
by the programmer for reasons that only he would know.
