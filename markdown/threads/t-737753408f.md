```
In article <318··.@mhd··s.edu>, guest (gu··.@mhd··s.edu) writes:
› I just got a job and they use COBOL at this company. What would be the
› best source to learn COBOL. I have used other languages but not COBOL.

My guess would be that any beginner's book will get you started,
and the more interesting things will be the vendor-specific
extensions, which will be described in the compiler's manual.

You definitely will need some sort of book, as some things are
wildly different from other languages I know or heard about. But
the basic (standardized) features of Cobol aren't that difficult,
you'll figure them out in no time from any book.

I happen to be in a similar situation. A useful feature of the
compiler we use are macros (simple ones, with no arguments), which
allow to make Cobol code "look like" Pascal or C so it would seem
more familiar to you, yet the compiler would think it's Cobol and
understand it with no difficulty. :-)

For example, the following code would compile and run well with the
compiler we use:

identification division.
program-id. HelloWorld.

replace var by 01, | that's going to look like Pascal :-)
const by 78,
bool by 88,

byte by ==pic X usage Comp-X.==,
word by ==pic XX usage Comp-X.==,
==unsigned long== by ==pic X(4) usage Comp-X.==,

do by perform,
repeat by ==perform with test after==,
while by ==until not==, | use with extreme caution!
done by end-perform,
fi by end-if, | well, that's more like Unix shell scripts

==()== by ==section.==, | that'll look like C
==//== by ==*==, | this will work only in the first column!
==++== by ==add 1 to==,
==--== by ==subtract 1 from==,
== != == by == not = ==,
== <> == by == not = ==. | Pascal again
// Not all of these are actually used in the little program below,
// but from these you probably get the idea
// how Cobol code can be made more readable through macros
// if Cobol is not your first language.

data division.
working-storage section.

var i byte;
var j word;

var Message-Text pic X(40).
var Message-Template pic X(40).

procedure division.
main()
*{
perform Message-Init;
perform Message-Copy;
perform Message-Write;
GoBack.
*};

Message-Init()
*{
move spaces to Message-Template;
StrCat "Hello", " ", "world!"
delimited by size
into Message-Template.
*};

Message-Copy()
*{
set i to 0;
set j to size of Message-Template;
do while (i < j)
++i;
move Message-Template(i:1) to Message-Text(i:1)
done.
*};

Message-Write()
*{
display Message-Text.
*};
```
