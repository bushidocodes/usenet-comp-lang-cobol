Hi Tommy,

If you want to get/post a request to a web server from COBOL you 
can use any of the following APIs:
 1) w3c has a set of 'generic' APIs that allow you get/post
    information to your server.
      http://www.w3.org/Library/

   (ideal if you need a cross platform solution)

 2) windows has a set of wininet APIs that allow you 'get' a
    document. This can be used to submit a simple CGI request.
    The apis are: HttpOpenRequest, HttpSendRequest... etc.. 
    these are documented in the SDK

Hope this points you in the right direction.
