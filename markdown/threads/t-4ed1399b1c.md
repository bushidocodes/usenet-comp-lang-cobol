Hello, everyone

Just posted up a small seed work sample of GnuCOBOL as a WebSocket server.

Uses libwebsockets along with a little jQuery for the web page example.

Managing the request and response text for the ws protocol callback is about 40 lines of COBOL. The entire seed work sample is about 300 lines. Demo GnuCOBOL executable weighs in at about 35K.

https://sourceforge.net/p/open-cobol/discussion/cobol/thread/025d79c9/

libwebsockets supports http, https, ws, and wss network protocols. Advertised as performing well with over 250,000 active connections, but that hasn't been verified by benchmark with GnuCOBOL as a server yet.

Have good, make well,
Brian
