```
I do *not* know this for a fact, but the current IBM TCP/IP support requires
the C run-time, not compiler - which "just so happens" to be LE (Language
Environment) which just so happens to be the COBOL run-time.  THEREFORE, I
would think that you would be a WHOLE lot safer trying to interface COBOL
with the web - if you used the IBM TCP/IP interface, not the CA one.

Having said that, I would check with CA to see if this is or is not supported
with their environment (and which COBOL would be required, i.e. VS  COBOL II
or IBM COBOL for this-and-that)
```
