I took the online C# tutorial and nearly finished it but got sick and
now I need to go back and do it again because too much time has gone
by and I forgot much of it.  It isn't too terribly hard.  Some of the
concepts are quite foreign if you've never done OO type programming
before.

As for Python, you might start here:
http://www.python.org/

Lots of free doc and downloads.

Also this site:

http://en.wikibooks.org/wiki/Python_Programming

Has everything you need to learn Python for free.

But if it is money you are looking to spend, then look at:

http://www.amazon.com/Python-Programming-Introduction-Computer-Science/dp/1590282418

The text offered my be a bit too basic for an experienced programmer,
but you certainly could learn Python from it.

They also offer these:

http://www.amazon.com/Python-Languages-Tools-Programming-Books/b?ie=UTF8&node=285856

Some free ones here:

http://www.coderholic.com/free-python-programming-books/

and

http://www.computer-books.us/python.php

Others here:

http://shop.oreilly.com/category/browse-subjects/programming/python.do

There are also some tutorials on YouTube at:

http://www.youtube.com/playlist?list=PLEA1FEF17E1E5C0DA&feature=playlist-comment

Also

http://www.tutorialspoint.com/python/


HTH
