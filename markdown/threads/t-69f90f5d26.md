```
Your condition flag defines a name for a specific relational statement, namely

VALID-SWITCH = 'Y'

That statement can be substituted with the word

GOOD-RECORD

Therefore, the conditional statement

IF GOOD-RECORD
    do something
ELSE
    do something else

is equivalent to the statement

IF VALID-SWITCH = 'Y'
    do something
ELSE
    do something else

Your code excerpt never tests VALID-SWITCH for equality to anything, and it
doesn't use the substitute condition flag GOOD-RECORD either, so neither your
VALID-SWITCH nor your 88 condition flag are doing anything in your data
validation (at least in the excerpt you quoted).

RoadRaat
```
