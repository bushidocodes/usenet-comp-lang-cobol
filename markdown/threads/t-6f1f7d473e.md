Well i once was troubled with the same problem using Acucobol...this
will solve your dos window problem - In version 4.3 and up maybe in
lower versions as well, use this line

CALL "C$SYSTEM" USING "COMMNAD LINE ARGUMENTS", 32.

I always like to using a pre-written batach file to be called in the
command line arguments.  This is in the documentation.
