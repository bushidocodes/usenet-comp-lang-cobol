The GetDefault() API-call is not implemented in win9x, only in windows
2000 and newer versions. Instead, you can use the EnumPrinter() call
(http://msdn.microsoft.com/library/en-us/gdi/prntspol_9fjn.asp) and set
the PRINTER_ENUM_DEFAULT flag. This works in windows 95 and newer
versions. The same call can be used to get the printernames of all
printers installed in windows (change flags).
