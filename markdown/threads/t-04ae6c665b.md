```
The hexidecimal value for the Euro symbol in EBCDIC is X'9F'. If the S/390
hasn't been patched with the character set changes then X'9F' may only show
up as the international currency symbol.

If you haven't got the keyboard map in your emulator set up so that one of
the keys inputs a hex 9F byte then you will have to SET HEX ON when you are
in edit mode in ISPF and manually enter the hex value of the byte to see
what character set you machine is using.
```
