ICOBOL v1 was written by Data General, who handed over to Egan
Systems/Envyr,
who produced version 2. Where compiler and utility switches are
concerned,
there is no resemblance between between the two versions.

If I remember correctly, the command "icobol filename" should compile a
crt
source file, with errors and warnings displayed on the screen.
"icobol/l=listing filename" will create a compiled listing named
"listing".

To reorg an ISAM file to sequential, use "reorg isamfile seqfile/s".

ICOBOL 2 has a switch(es) that will produce ICOBOL version 1 object
files.
