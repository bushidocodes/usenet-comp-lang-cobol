*.dat is a text file.  You will still have to know if it has carriage returns
or line feeds.  Some *.dat files may be continuous data.  No matter what the
data is you need to know what kind of fields are contained.  Sometimes the
fields are fixed length and sometimes the fields are comma delimited or have
variable length fields.

Keep in mind there is no such thing as a COBOL file.

COBOL programs use the available file types available for the operating
system.
All of the COBOL files we use are fixed length records, with fixed length
fields.  

You still have to know the file record structure.
