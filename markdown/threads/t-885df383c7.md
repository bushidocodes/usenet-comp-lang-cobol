```
You can only call Windows API functions from a real Windows
program, and not from a DOS program in a Windows DOS box. You
need to link in the MSWindows Library LIBW.LIB. If you are
not doing this then it is not a Windows program, if you
are then doing a WinExec() is just like calling any other
Windows API:

CAL WINAPI "__WinExec"
USING by reference "RUNME.EXE" & x"00"
by value SW-SHOWNORMAL size 2
RETURNING Return-Int

usual conventions apply.
```
