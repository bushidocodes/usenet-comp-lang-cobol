```
Doc had asked about this a bit back and I said I'd ask around the shop.  Well
everyone that uses it gives it a big thumbs up.  Much easier to use than QMF
and not as 'kludgy' as one of my cohorts put it especially for fixing rows on
the fly or adding rows.  I was watching my cubemate work on a table this
afternoon and it appears to be as easy as Fileaid for the rest of the world. As
the systems I'm supporting have very little DB2 in them (for the time being) I
haven't had the opportunity to use it myself mostly using QMF to do the queries
I required.

As for the cost that was another story.  Seems they (the company I'm consulting
at) leases the mainframe and all software so the cost of Fileaid is included in
a bundle number and not readily available.  It was suggested that if you use
multiple CA products you can probably negotiate a good bundle rate on all of
them including Fileaid for DB2 next time the contract is negotiated.

Hope this sheds some light on the subject.

Eileen
```
