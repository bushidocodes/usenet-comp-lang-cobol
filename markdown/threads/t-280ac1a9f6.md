subroutine that makes the appropriate Windows API calls:

* -------------------------------------------------------------- *
$SET ANS85 CHECK-CLEAR-BREAKS CASE
* -------------------------------------------------------------- *

IDENTIFICATION DIVISION.

PROGRAM-ID. WSHELL32.

ENVIRONMENT DIVISION.

CONFIGURATION SECTION.

SOURCE-COMPUTER. IBM-PC.

* -------------------------------------------------------------- *

SPECIAL-NAMES.

CALL-CONVENTION 74 IS WinAPI32.

* -------------------------------------------------------------- *

WORKING-STORAGE SECTION.

01 WS-Wait-Timeout PIC X(4) COMP-5 VALUE 258.

VALUE "Extended Error Status: ".

* -------------------------------------------------------------- *

LINKAGE SECTION.

01 LS-Shell-Parameters.
03 LS-Shell-Command-Line PIC X(128).
03 LS-Shell-Function PIC X(5).
03 LS-Shell-Handle PIC X(4) COMP-5.
03 LS-Shell-Return-Code PIC X(4) COMP-5.

* -------------------------------------------------------------- *

PROCEDURE DIVISION USING LS-Shell-Parameters.

CALL "COB32API".

* -- Convert function to upper case so that evaluate always
* -- works correctly.

CALL "CBL_TOUPPER"
USING LS-Shell-Function
BY VALUE 5 SIZE 4
END-CALL.

* -- Test the function. If the function passed is not START
* -- or STOP return 31 as the error code.

EVALUATE LS-Shell-Function
WHEN "START"
WHEN "DOMAX"
PERFORM Proc-Start-App
WHEN "STOP"
PERFORM Proc-Stop-App
WHEN OTHER
MOVE 1 TO LS-Shell-Return-Code
END-EVALUATE.

EXIT PROGRAM.
STOP RUN.

* -------------------------------------------------------------- *

Proc-Start-App SECTION.

MOVE LS-Shell-Command-Line TO WS-Calc-String.

* -- Calculate the length of the given string. This is done
* -- to place the null terminator at the end.

MOVE 0 TO WS-Counter.
MOVE FUNCTION REVERSE(WS-Calc-String)
TO WS-Calc-String.

INSPECT WS-Calc-String
TALLYING WS-Counter
FOR LEADING SPACES.

SUBTRACT WS-Counter FROM 129 GIVING WS-Counter.
MOVE FUNCTION REVERSE(WS-Calc-String)
TO WS-Calc-String.
MOVE x"00" TO WS-Calc-String(WS-Counter:1).
MOVE WS-Calc-String to LS-Shell-Command-Line.
MOVE LOW-VALUES TO WS-Startup-Info.
MOVE LENGTH OF WS-Startup-Info TO WS-SI-cb
SET WS-SI-lpReserved TO NULL.
SET WS-SI-lpTitle TO NULL.
if ls-shell-function = "DOMAX" *> Show window
MOVE 0 TO WS-SI-dwFlags
else
MOVE 1 TO WS-SI-dwFlags *> Minimised
end-if
MOVE 6 TO WS-SI-wShowWindow

* -- Start the Windows NT/95 process.

CALL WinAPI32 "CreateProcessA"
USING BY VALUE 0 SIZE 4
BY REFERENCE LS-Shell-Command-Line
BY VALUE 0 SIZE 4
BY VALUE 0 SIZE 4
BY VALUE 0 SIZE 4
BY VALUE 0 SIZE 4
BY VALUE 0 SIZE 4
BY VALUE 0 SIZE 4
BY REFERENCE WS-Startup-Info
BY REFERENCE WS-Process-Info
RETURNING LS-Shell-Return-Code
END-CALL.

IF LS-Shell-Return-Code = ZERO
PERFORM Get-API-Error
ELSE

MOVE WS-PI-hprocess to LS-Shell-Handle
MOVE ZERO TO LS-Shell-Return-Code

CALL WinAPI32 "WaitForInputIdle"
USING BY VALUE LS-Shell-Handle
BY VALUE 2000000 SIZE 4
RETURNING LS-Shell-Return-Code
END-CALL

IF LS-Shell-Return-Code = WS-Wait-Error-C5
PERFORM Get-API-Error
END-IF

END-IF.

Proc-Start-App-Exit.

EXIT.

* -------------------------------------------------------------- *

Proc-Stop-App SECTION.

CALL WinAPI32 "GetExitCodeProcess"
USING BY VALUE LS-Shell-Handle
BY REFERENCE WS-Exit-Code
RETURNING LS-Shell-Return-Code
END-CALL.

IF WS-Exit-Code = WS-Still-Active

CALL WinAPI32 "TerminateProcess"
USING BY VALUE LS-Shell-Handle
BY VALUE 10 SIZE 4
RETURNING LS-Shell-Return-Code
END-CALL

IF LS-Shell-Return-Code = ZERO
PERFORM Get-API-Error

ELSE
MOVE ZERO TO LS-Shell-Return-Code
END-IF

ELSE
MOVE ZERO TO LS-Shell-Return-Code
END-IF.

Proc-Stop-App-Exit.

EXIT.

* -------------------------------------------------------------- *

Get-API-Error SECTION.

CALL WinAPI32 "GetLastError"
RETURNING LS-Shell-Return-Code
END-CALL.

Get-API-Error-Exit.

EXIT.

CALL WINAPI "WinExec"
USING by reference "NOTEPAD filename" & x"00"
by value SW-SHOW size 2
RETURNING mIReturn
