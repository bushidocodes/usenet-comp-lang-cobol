I'm only a student in my last semester of COBOL, but I recently finished a
program where I had to verify dates which included a leap year check. I'll
give you what I did and I hope it helps.

First, I set up a one-dimensional table that held all of the valid number
of days for each month:

01 MONTH-TABLE.
05 FILLER PIC X(24) VALUE '31293130313130313031'.
01 VALID-MONTH-TABLE REDEFINES MONTH-TABLE.
05 DAY-OF-MONTH OCCURS 12 TIMES PIC 99.

With this table set up, I could read a record into working storage and
compare the month to the table:

Record Description:
TRANSACTION-MONTH PIC XX.
TRANSACTION-DAY PIC XX.

I redefined the month to:

TRANS-MONTH-X PIC 99.
This will allow me to use it as a subscript when I check the Table.
When checking the days of the month, I ask:

IF TRANSACTION-DAY <= DAY-OF-MONTH(TRANS-MONTH-X)
* Here I continue and check if the month is = 02
IF TRANS-MONTH-X = 02 and TRANSACTION-DAY = 29
* I call a sub-routine that checks for the leap year, basically all that is
doing is
* dividing the year by four and checking for a remainder. No remainder =
* leap year. If leap year, then TRANS-MONTH-X can = 29 days. to do this,
you need
* to use the REMAINDER clause. DIVIDE YEAR BY 4 GIVING WS-TOTAL
REMAINDER
WS-REMAINDER
IF REMAINDER = 0
leap-year flag =
'Yes'
END-IF.

I know this is partially pseudo code and hopefully you will get enough to
help.
Remember, I'm a student and I'm sure some of the regulars of this group
will jump in
and blast my code but hey, how else do you learn. Good Luck.
