That's one way, but you should enclose the tested item in quotes:  

ie:  If Print-File-Status = "00"
.....

One way I prefer is a set of 88 levels on the file status statement.

01  File-Status  Pic XX.
      88  File-Success Value "00" "02" "05".
      88  End-0f-File    Value "10" "14".

etc etc...
