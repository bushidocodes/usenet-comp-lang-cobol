Assuming that you are using a standard COBOL 85 compiler, your 
native collating sequence is ASCII (I assume so because of your use 
of LOW-VALUES - it is not guaranteed to be binary zeros) and you have 
no PROGRAM COLLATING SEQUENCE clause, just do

   COMPUTE ws-work-number = FUNCTION ORD (ws-pass-field) -1

No need for REDEFINES, strange data types, and so on.
