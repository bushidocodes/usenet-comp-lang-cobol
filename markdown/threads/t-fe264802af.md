Not quite right. Report writer has no such thing. It does allow you to
define the size of a page and specify automatic generation of pages.
What
you are thinking about is LINAGE, which is supported in all
implementations
that conform to the standard. With it, you define a logical page and use
WRITE ... AT EOP (or END-OF-PAGE) ... . This does not let you sense the
printer control mechanism. Just a logical page that you defined with
LINAGE.
