[Acucorp Support has contacted Jim directly; I'm summarizing the key points for
anyone else in the same situation - cadams]

* Office 97 installs a new version of ODBC which is not backwards compatible.
  Early AcuODBC versions need to be upgraded to work with it.

* You should check that all of the files listed in XFDRMT.DIR exist and are
  listed in the correct case.

* Call 1-800-399-7220 or send email to support@acucorp.com.
