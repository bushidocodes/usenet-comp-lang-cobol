Yes, Object/methods have a comparable (not same) feature called "inline
invocation".  Once again, I don't know of any vendor who supports it yet.
Look for syntax where there is a double colon (e.g. "::" ) in the middle of
a source line.  Again, this will return any type of "parameter" - so it will
depend on Boolean data types before you can use it in True/False tests.
