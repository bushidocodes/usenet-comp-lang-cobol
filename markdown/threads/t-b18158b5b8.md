Hi..... I created a word doc called C:\NETEST.DOC with a macro in it called
MACRO1 which displays a "hello world" message box.  Stepping through the
following code in your animator should work..........   I am using NE 3.1
but hopefully this will work for you.   It took a while to figure out when I
needed it way back when.......

Later,
Mike Kinasz


       $set ooctrl(+P) mfoo

       PROGRAM-ID. WRD000.
       OBJECT SECTION.
       CLASS-CONTROL.
       COM-OBJECT IS CLASS "$OLE$Word.Application".

       WORKING-STORAGE SECTION.
       01 WRD-APP-OBJECT                        OBJECT REFERENCE.
       01 WRD-DOCS-OBJECT                       OBJECT REFERENCE.
       01 WRD-DOC-OBJECT                        OBJECT REFERENCE.
       01 WRD-SOURCE-FILE                       PIC X(128).
       01 WRD-MACRO-NAME                        PIC X(128).
       01 OLE-TRUE                              PIC 9(1) VALUE 1.
       01 OLE-FALSE                             PIC 9(1) VALUE 0.

       PROCEDURE DIVISION.
        100000-MAINLINE.
           MOVE "C:\NETEST.DOC" TO WRD-SOURCE-FILE.

           INVOKE COM-OBJECT "new" RETURNING WRD-APP-OBJECT.

           INVOKE WRD-APP-OBJECT "setVisible"
                USING OLE-TRUE.

           INVOKE WRD-APP-OBJECT "getDocuments"
                RETURNING WRD-DOCS-OBJECT.

           INVOKE WRD-DOCS-OBJECT "Open"
                USING BY REFERENCE WRD-SOURCE-FILE.

           INVOKE WRD-APP-OBJECT "getActiveDocument"
                RETURNING WRD-DOC-OBJECT.

           INVOKE WRD-APP-OBJECT "Run"
                USING "MACRO1".

           INVOKE WRD-DOC-OBJECT "Close"
                USING OLE-FALSE.

           INVOKE WRD-APP-OBJECT "Quit".

           STOP RUN.
        100099-MAINLINE-EXIT.
