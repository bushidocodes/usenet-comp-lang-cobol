I believe you shuold examine a product we distribute, called XperCASE,
produced by SIEMENS. It allows conversion of COBOL code to and from any
number of languages, including DBase, Clipper, ADA, C/C++, etc.

You could view demos of XperCASE on SimTel, at OAK.OAKLAND.EDU, or any
other SimTel mirror site. Simply telnet to SimTel, enter directory
PUB/MSDOS/WINDOWS3 and look for files beginning XPER...

Note that demos do not allow saving, so you won't be able to actually
translate, but they should give you a good feel for the program.

The procedure you would use is to read the COBOL code into XperCASE COBOL
PACK, which would break the code into Structure Diagrams. You then would
take the Structure Diasgrams and import them into XperCASE SPX (for DBase
or Clipper), which would convert the diagrams into DBase, and compile
said code.

Please note the current release of XperCASE is Version 4.01, so look for
files beginning XPER4...
