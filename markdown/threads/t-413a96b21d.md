Hi Terri,


Here is the reply how to code table in COBOL.

Define table in your Working-Storage Section. Following is the example of 
table in working storage.

01 Table-XXX. 
   05 Table-01 Occurs 100 Times.
      10 EMP-CODE         Pic 9(5).
      10 EMP-Rate         Pic 9(8)v99.

To load this table from say empoyee rate file you will have a paragraph in 
your procedure division something like this.

Load-Rate-Table.
      Read EMP-Rate-File At End 
            Perform End-Of-Program.
      Add 1 to S1.  (here this S1 is the subscript which you will define 
                    working storage section numeric 9(3) value zeros.
      If S1 <= 100
      Move Emp-File-Code to Emp-Code(S1)
      Move Emp-File-Rate To Emp-Rate(S1)
      End-If.

This will load up your table from the file. 


To read this table code something like following: 

      Move Zeros to S1.   (Initialize the Subscript)
      Perform Retrieve-Table-Information 
              Varying S1 By 1
              Until S1 >= 100.

Retrieve-Table-Information. 

      If Emp-File-Code = Emp-Code(S1)
         do your processing what you want to do when an equal condition 
      End-If.

This is one way of doing it. Other way is you define your table and 
indexed it with one of your data item. 

       You can perform Search on the table with the indexed item.
       Binery search would be faster and easy.
