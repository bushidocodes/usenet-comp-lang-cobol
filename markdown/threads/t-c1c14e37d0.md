We use logic after the ACCEPT statement:

   ACCEPT 8-DIGIT-DAT FROM DATE.

   IF 8-DIGIT-DAT-YYYY > 50
       ADD 1900 TO 8-DIGIT-DAT-YYYY
   ELSE
       ADD 2000 TO 8-DIGIT-DAT-YYYY
   END-IF
