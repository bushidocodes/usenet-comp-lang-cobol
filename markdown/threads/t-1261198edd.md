problems with some DETAILS and I'm sure you'll get help.

One quick tip - are you getting file errors when you attempt to run. Go to your
manuals and look at the detail of file-status codes and some of this is
associated with the way in which you are accessing files,  for example you can't
REWRITE if you have opened the file only as INPUT.  You should also find a table
referencing when you can/can't use which file VERBS with the various OPEN
statements associated with file type, i.e. Relative, Sequential ISAM(indexed).
