SELECT CSV
ASSIGN TO "TIMEDATA.CSV"
ORGANIZATION LINE SEQUENTIAL
....
FD CSV.
01 CSAV-Record PIC X(512).

.....

MOVE SPACES TO CSV-Record
STRING QUOTE DELIMITED BY SIZE
First-Field DELIMITED BY SIZE
QUOTE
","
Numeric-Field DELIMITED BY SIZE
","
........
INTO CSV-Record
WRITE CDV-Record

(oops typing errors in above)

It may be useful to use the pointer option and have several
STRINGs to build the data to the output. Reducing the field
sizes to the minimum can be done quite easily using reference
notation.
