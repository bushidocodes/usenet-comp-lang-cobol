Just a guess, but the default for ORGANIZATION is SEQUENTIAL.  There may
be no line terminators after each write so the 'file' is just a long
string of characters with no line feeds or carriage returns (the AFTER 1
is ignored).  When you look at the file with a text editor it looks like
there is only one line, try scrolling right.  When it is sent to the
printer the line overflows and is lost.

Or try ORGANISATION LINE SEQUENTIAL, or whatever it is for your system.
