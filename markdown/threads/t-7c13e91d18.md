definition.

compiler. To avoid confusion, you might try looking at the
data with a hex editor, comparing the bytes with known values,
and deducing (or verifying) the format by inspection.

Usage COMP-0, in Micro Focus, is covered under
Microsoft COBOL V1.0 and V2.0 Syntax Support.

I do not use COMP-0; but the Micro Focus LRM says:

"USAGE IS COMP-0 is allowed only with numeric
items. If an item is described with a PICTURE no
larger than S9(5), the effect is a data item whose
description is PIC S9(4) USAGE COMP; that is,
a 2-byte signed binary item. Otherwise, the effect
is a data item with the PICTURE clause actually
specified and with USAGE DISPLAY."

I located this statement by referring to the Master
Index.
