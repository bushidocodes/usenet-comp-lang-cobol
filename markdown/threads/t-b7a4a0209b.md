Questions:

1) Is there anything in MyModule that moves 67 to RETURN-CODE?

Failing the obvious things get more difficult...

1) What operating system and compiler are you using?
2) Is MyModule actually executing? The call format looks like it's a static
call but compiler directives may cause it to be dynamic. Perhaps a 67 means
that the module failed to load.
3) If it's a static call, did the linker report any errors?

- Karl
