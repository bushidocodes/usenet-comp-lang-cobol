```
Reference modification might help.  The first token in the parenthesis is
the starting column (1 for first column) and the token after the colon
represents the length.  The 2nd token isn't required (the colon still is) if
you need to refer to the variable starting at a specific point through its
end, e.g. MY-VAR(11:) would give the last ten characters of MY-VAR.

01  SUB-1      PIC 999.
01  MY-VAR    PIC X(20) VALUE "       NAME    ".
- - - - - - -

   MOVE 1 TO SUB-1.
LOOP-1.
   IF MY-VAR(SUB-1:1) = SPACE
      ADD 1 TO SUB-1
      GO TO LOOP-1.
   IF MY-VAR(SUB-1:4) = "NAME"
      PERFORM DO-YOUR-THING
   ELSE
      PERFORM WHINE-POUT-SEQUENCE.
```

```
If you are checking for various things in a list, check for the longest
items first.
For example:
IF MY-VAR(SUB-1:5) = "JOHNS"
    PERFORM whatever
ELSE
   IF MY-VAR(SUB-5:4) = "JOHN"
      etc.

A check for JOHN by itself would find those characters in JOHNS as well.
Moving MY-VAR to a longer field might help so you could include a trailing
space as well.

MOVE MY-VAR(SUB-1:) TO AUX-FIELD.
IF AUX-FIELD(1:5) = "JOHN "
   etc.
```
