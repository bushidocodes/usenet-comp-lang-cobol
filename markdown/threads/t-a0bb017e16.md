Your post doesn't rally tell what you are doing. I assume that you are
going from either OS/VS COBOL or DOS/VS COBOL (with the LANGLVL(1) compiler
option) to COBOL for OS/390 and VM (with the NOCMPR2 compiler option). If
that isn't what you are doing, please explain exactly what you are doing..

Assuming that is what you ARE doing, then you should start with the IBM
Migration Guide (which you probably will need in hard copy but) can be seen
online at

http://ppdbooks.pok.ibm.com/cgi-bin/bookmgr/bookmgr.cmd/BOOKS/IGYMG200/CCONT
ENTS

(This should be all on one line - but my newsgroup processor puts in a line
break after the hyphen.)

It talks about many of the services, products, and tactics for such a
migration. There are also a variety of other books (I wrote one that is
available from Micro Focus Publishing) that talk about the issues - but I
would still start with the IBM publication.
