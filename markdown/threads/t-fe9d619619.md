```
I was wondering if COBOL 97 will have the following and if any besides
my self are interested is such features.

A) An easy and safe way to convert a pic x field to a pic 9 field.
By easy I mean a single call (perhaps a function).
By safe I mean something that returns an error when appropriate.
It should handling pic formatting codes (leading $/*/- ... commas
that kind of thing).

B) A string function to strip leading or trailing spaces (or both).
I'd perfer a function but something like:
"inspect str removing leading spaces"
"inspect str removing trailing spaces"
would be ok, too.

C) A string funtion that returns the length of string, by 'length' I
mean the position of the last non-blank character.
I'd perfer a function but something like:
"inspect str tallying i for characters before trailing space"
would be ok, too.

I consider the above in the nice to have, but not critical category.

Greg
```
