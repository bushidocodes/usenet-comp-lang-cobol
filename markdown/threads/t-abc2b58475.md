First of all the attribute values set by the map can be overridden from
with the program. Say the map has a field named RATE.  The symbolic map
will have a RATEA field that you can use to set the attribute. 

Also the MDT (modified data tag) is set automatically when the user keys
data into an unprotected field.  AAFAIK CICS never transmits (send/receive)
LOW-VALUES.
