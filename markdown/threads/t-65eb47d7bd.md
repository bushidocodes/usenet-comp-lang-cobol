```
NetCOBOL 6 is 'protected' by a licence number and the need to get a
'key' from the Fujitsu (or some other) server. Given that 6 was more
than a decade ago, it may be difficult to get a usable key, especially
if you aren't the registered owner.

It is possible to move the key from one working machine to another,
but it needs to initialize the move media on the destination and then
write the key on the original (which disables that copy) before
putting the key onto the new copy.
```
