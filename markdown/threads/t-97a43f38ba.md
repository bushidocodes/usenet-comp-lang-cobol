You can use the CLIPBOARD class available in the GUI class library that will
give you access to Window's clipboard.

It offers two class methods, read and write that do this. Write, which is
the one you need, takes a CharArray as a parameter so make sure you create
this chararray object from your pic x(20) string, before using it there.

regards

Panos
