Hello Arild,

You should check the version you are working with. The first 32-bit
version that was send out is version 4.0.07. That version and later
version until 4.0.26 (if I'm not mistaken) has not got all the
functionalities you would find in the 16-bit 3.2.50 (the latest 3.2
version). Especially these functionalities were not included, they
took the functions availble in 3.2.43 of 3.246 and build a 32-bit
version with those.

You better check on the Micro Focus ftp server (ftp.mfltd.co.uk) for
the latest update. I almost sure all the PC_WIN_PRINTER... functions
were up to date in version 4.0.26.

Success,
