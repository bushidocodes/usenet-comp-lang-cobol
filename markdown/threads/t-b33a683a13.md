Several products come to mind....all of these for the IBM MVS environment

Pinpoint - previously offered by Language Technologies, Inc. (since acquired by some other firm). A static analyzer that looks at source code.

RECODER - again a Language Technologies, Inc offering. While a restructurer, it would also look at the structure of the code as it existed.

COBOL/SF - IBM's restructuring engine. Still offered by IBM (which has not been bought out by CA), again a restructurer, but provides lots of "current structure" analysis. An interesting sidelight is that a version of this is part of the Visual Age COBOL product for OS/2 and (I beleive) NT / Win 95. Mainfram version and PC version have the same functionality. Take your choice of platforms.
