```
How you might obtain the 'BIOS Date' is entirely dependent on your
compiler.  But you can obtain the system date in a variety of ways.
I prefer the COBOL 85 intrinsic function CURRENT-DATE:

  CURRENT-DATE.
    CCYYMMDD    PIC  9(08).
    HOUR        PIC  9(02).
    MINUTE      PIC  9(02).
    SECOND      PIC  9(02)V9(02).
    GMT-SIGN    PIC  X(01).
    GMT-HOUR    PIC  9(02).
    GMT-MINUTE  PIC  9(02).

  MOVE FUNCTION CURRENT-DATE TO DW-CURRENT-DATE.

If GMT-SIGN = "0" system does not know GMT
              "+" time is same or ahead of GMT
              "-" time is behind GMT

Fields GMT-HOUR and GMT-MINUTE specify how far this time is ahead or
behind GMT (if any).  See documentation for more details.
```
