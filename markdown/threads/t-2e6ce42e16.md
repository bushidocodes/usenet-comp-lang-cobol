Look at it this way: You defined storage for 100 different
"500-EntireName" entries. Your STRING statement needs to know WHICH of
the 100 possible gets the result of the string operation. Once you
code "dataname OCCURS.." in Working Storage, every occurrence of
"dataname" in the program (including the STRING verb) must be followed
by an index to particularize which entry you mean.

So,
           String   400-LastName delimited by space
                     ', ' delimited by size
                     400-FirstName delimited by space
                     into 500-EntireName (300-SS)

Of course your code won't work if the last name has an imbedded space
(Van Zandt) or a trailing modifier (Jr., M.D., Ph.D.) let alone trying
to put the name back the way it was. But it's close enough for
government work (I assume you are attending a public school).
