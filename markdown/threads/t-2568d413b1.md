It refers to the PERFORM verb. Basically, if  PERFORM is executing a
paragraph or a section, it is 'out of line'. If the code is contained
within a PERFORM...END-PERFORM construct, it is 'inline'.

*out of line *
       PERFORM some-para-or-section ...

*inline*
      PERFORM ...
           [optional statements]
      END-PERFORM
