Are you asking "Is there a utility for Windows XP to compile and
execute COBOL programs?" or are you responding to an earlier question?

If the former, then not exactly.

Windows XP does not provide such a utility.

You can buy compilers for Windows from various companies such as
Fujitsu and Microfocus.  There are also some free compilers, I believe
TINY COBOL is a free compiler.

Whether or not you need a utility for executing programs depends upon
the compiler, some compilers produce directly executable code, while
others may need a run time environment to execute programs.

Bill Klein's FAQ regularly listed in this newsgroup contains much more
detail.  You might find a teach yourself COBOL book an asset, Bill's
FAQ contains a list of those too.

Robert
