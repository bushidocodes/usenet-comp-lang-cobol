You can program most anything you want in COBOL, as in any other general
purpose programming language. But some things are programmed easier and more
efficiently in one language Vs another. For example, COBOL has scaled
decimal data and powerful data formatting features which make it good for
writing business programs which handle money and print reports, and C/C++ has
powerful memory allocation and referencing features which are good for
writing system software which handle binary data blocks and such.

Micro Focus COBOL comes with a demo TicTacToe game, but it is copyrighted, so
I won't include it here. I'm sure there are others out there.
