Fujitsu COBOL supports dynamic loading DLLs and also dynamic loading
multiple Entry Points in the DLL.  To dynamically call a DLL or Entry Point
in the DLL, use the DLOAD compiler directive.

Fujitsu COBOL V2 for Windows 3.1 (16-bit) includes Microsoft's 16-bit
IMPLIB.EXE utility.

For creating LIB files for Windows 95/NT (32-bit) see:
http://support.microsoft.com/support/kb/articles/q131/3/13.asp
