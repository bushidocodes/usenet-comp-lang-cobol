You presented a program and did not show that you had attempted to
compile it, nor state what the problem, if any, might particularly be.

program what you apparently have not?

My suggestion, which I thought helpful, was to have separate EOF flags
for each file, so that checking the EOF flag will actually let you know
just which file it is that has been read completely.  

So go and compile your program; fix the syntax errors (or ask here for
interpretation of the reported errors); then run the program and come
back with your results. 

Everyone has to start somewhere, true, but they have to start !!
