You load and execute a COBOL (.EXE) program from BASIC by using the
SHELL command.

You load and execute a BASIC (.EXE) program from (MF) COBOL by using
CALL X"91" function 35.

It is not possible to call or chain to a BASIC program from (MF) COBOL.

It may be possible to call (not chain) a (MF) COBOL (.OBJ) from BASIC
but it depends upon the BASIC. See the (MF) COBOL User Guide for
information on "Interfacing and Mixed-Language Programming."
