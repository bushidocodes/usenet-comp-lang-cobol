```
Hello,

For the ZUSE//A Project, we need a list relating the
compiler feature number with the human name.

E.g. PL/1 Compiler 1.4 puts as identifyer in the
loadmodule '5734-PL1'. We only have PL/I 1.4,
1.5 and 2.3, some releases of Assembler H, COBOL II, and
OS/VS COBOL at our disposal but are still left with a number
of unknown compilers.

We would be very grateful if some of you could take
the trouble to have a look in a loadmodule created by a known
compiler, and could pass us the id written in the module, and the
compiler version and name.

This is also posted on the folklore list because we actually
have unknown compiler id's from modules last compiled
in 1968.

For those of you interested, we intend to put the results of
this translation - as run against 16.000 loadmodules in graphical
form on the Net. Point your browser to URL
http://www.club.innet.be/~janjedsp/zuse.htm
and click on the graph icons in the text.

In order not to clutter up this list, yoy can send me
this information by e-mail. I will make it available
to anybody who requests.

Thanks,
```
