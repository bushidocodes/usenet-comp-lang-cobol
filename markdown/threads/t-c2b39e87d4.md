```
Rodney Thompson wrote:

> I havn't been able to find resources on the net so have come here for
> help. Could any one tell me how to call a DOS command from COBOL-85?
> I tried SYSTEM "type output.dat |more"
> and it spat back at me, not knowing what SYSTEM was.

Perhaps that should have been:
01  CALL-PARM.
       05 FILLER      PIC X(22) VALUE "type output.dat |more".
       05 FILLER      PIC X(01) VALUE LOW-VALUE.
-------------
       CALL "SYSTEM" USING CALL-PARM.

Check the documentation for the compiler on the system you're using to
see if you need to specify "$SYSTEM" instead or if you can specify your
command line after USING instead of setting up another variable in
working-storage.  HTH
```
