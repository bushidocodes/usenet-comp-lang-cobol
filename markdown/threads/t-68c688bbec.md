Take a very large number, then square it. You may be close.

One major corporation that I am familiar with has perhaps 40,000,000 LOC,
mostly COBOL. One note - a fair chunck of the new code for the last 10
years or so was not manually written, but was generated by TELON and IEF.
Yet it is legacy, running on good ole T. Rex type iron.

This corporation has a gross of $3,000,000,000 or so, which is a tiny
percentage of the "big corporation" contribution to GNP. Say .01% or
less? From this could one extrapolate 1,000 times this amount of code (or
maybe 10,000 times).

One might make a WAG at 40,000,000,000 lines of legacy code - only 1000
data centers of the size of the one that I have hard numbers on.

Rex Widmer

survive in a legacy based world.
