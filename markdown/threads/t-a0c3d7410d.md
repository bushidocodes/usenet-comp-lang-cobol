```
This is NOT really a problem with your code.  It is a common "new to Fujitsu
problem".  You need to tell Fujitsu (unlike Micro Focus) which is the MAIN
program in your application.  There are several ways to do this.  If you are
using Power - I think you can just "click" on it and mark it as "MAIN".
Otherwise, you can put an @OPTIONS MAIN statement as the first line of your
program.
```
