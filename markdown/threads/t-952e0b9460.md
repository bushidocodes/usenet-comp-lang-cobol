I haven't used 'Personal' Cobol but I understand that it only
generates .INT programs and uses the run-time to execute them.
It is likely that there insufficent tools in the package to
allow you to build a callable file from C code.

You will probably need to buy the full compiler to enable this
to be done.

'Persoanl' Cobol is a package for use in learning Cobol, not
for producing production programs, or for advanced techniques
such as mixed language programming.
