Think of it this way.
CDENT-PROD and CDENT-PRD are both the same level (02).

So therefore, CDENT-PRD and CDENT-PROD are both pointing to the same
location in memory.  You can alter the value in memory, which is 2 bytes, by
using any of the variables.

Hope this helps.
