You could define different record layouts for each different record type and
then choose which record you want to use by testing the record type, or you
could use the redefines clause to change the way various fields in a single
record layout can be accessed:

e.g.

Either

01 Rec1.
  03 date1 pic x(8).
  03 scode1 pic x(3).
  03 trancode1 pic x(1).
       88 is-rec1  value 'A'.
   03 rec1-fld1 pic x(12).
   03 rec1-fld2 pic x(10).

01 Rec2.
  03 date2 pic x(8).
  03 scode2 pic x(3).
  03 trancode2 pic x(1).
       88 is-rec2  value 'C'.
   03 rec2-fld1 pic x(3).
   03 rec2-fld2 pic x(33).
... etc ...

or

01 Rec.
  03 date pic x(8).
  03 scode pic x(3).
  03 trancode pic x(1).
       88 is-rec1  value 'A'.
       88 is-rec2  value 'C'.
  03 rec1-flds.
       05 rec1-fld1 pic x(12).
       05 rec1-fld2 pic x(10).
   03 rec2-flds redefines rec1-flds.
       05 rec2-fld1 pic x(3).
       05 rec2-fld2 pic x(33).

... etc ...
