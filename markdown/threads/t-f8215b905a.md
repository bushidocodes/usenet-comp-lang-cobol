One recomendation for a code formatter if you have a PC with
means to file transfer to/from mainframe (an assumption) is
Ironsoft COBOLXplorer. Their phone number is 608-695-1896.
This product can format (indent) the PROCEDURE DIVISION clauses
according to a presubscribed pattern (sorry you can't change
it). You also perform "desk checking" the code in an
interactive session for syntax errors. It works with copybooks,
COBOL 2, Micro Focus, and Panvalet Delta system via TCP/IP.

This is DOS-based product but can run in multiple sessions in
windows. The verbs and keywords appears in color for easy
spotting.

In a more simpler form the COBOLXplorer is the code analyzer
option in CTC's SPF/PC version 4.0.

It's simple, robust and useful for routine COBOL maintenance.

Good luck,
Boyce Williams
