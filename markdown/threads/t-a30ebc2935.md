I think there's no problems with the COBOL650.LIB, but rather with your old
version of the linker... Perhaps you should try the linker which comes with
OS/2 Warp... ;-)

More seriously : Beginning with version 3.30, the linker comes with the
"DOS Technical Reference", on a diskette along with LIB, EXE2BIN and
so on. What is funny for me, is that you get the "dual-mode" version
of the linker with the "Reference Manual" for DOS version 4.0 :
mine does display an "IBM Linker/2" message on start, indicating that
the linker was able to run on DOS & OS/2 along (in 16 bit-mode of course).
With this one, I had no problems linking the code for COBOL650.
