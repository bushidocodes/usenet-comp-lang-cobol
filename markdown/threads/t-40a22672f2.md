Our IDE is writen on Java. It uses Eclipse as UI base (but it isn't
Eclipse's COBOL subproject) so it can work on Windows, Unix, Linux,
MacOS (all platforms that Eclipse supports). We don't develop it for
any particular customer. We orient on IBM, Microfocus, UniSYS versions
of COBOL.

The main goal of our IDE is to provide useful operations with sources
(syntax analysis, errors detection, browsing, usage reports,
refactoring...). Not all features are ready yet but we are working.

Could you please send me more details about your proposal on
sng@lemosoft.com

Thanks.
