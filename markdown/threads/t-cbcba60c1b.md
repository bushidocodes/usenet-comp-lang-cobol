Notice to COBOL programmers on MVS and OS/390:

I recently updated the Tools&docs section of my Web page to add a sample
COBOL program that retrieves some pieces of information about the MVS (or
OS/390) system it's running on.  Among other things, the program finds out
the CPU type and model, the amount of real storage it has, its speed
(SU/s, or service units per second), as well as the julian date of the
last IPL and various system names.

In the Tools&docs section also, you'll find two previously posted sample
COBOL programs that retrieve info about the current job and issue TSO
commands.
