EXPERIENCE:
MOTOROLA ELECTRONICS PTE LTD., Singapore Feb. 1993 to Present
Manufacturing System: Bill of Material , Inventory Control, Purchasing,
MRP/MPS, Shop Order Processing, Customer Order Interface Cost Accounting
and Message Management.
Part of a team that analyzed, designed, coded and tested programs for
the Customer Order Interface (COI) module. The objective of the project
was to downsize the manufacturing control system from a mainframe based
system (CICS, COBOL, IMS, DL/I) to a UNIX platform (HP UX, CICS, COBOL,
INFORMIX, SQL).
Automated the faxing of non-production purchase orders to different
suppliers by allowing buyers to fax directly from the Manufacturing
Control Systems (MCS).
Helped implement a new Manufacturing Software (MCSLite) in the different
manufacturing sites (India, China, Singapore). Provided on-site support in
the implementation of MCSLite in Korea.
Converted cost roll module from Motorola Computer Group (MCG) to cater
for Paging Product Group's (PPG) requirements (i.e. make it frequency
sensitive).
Worked closely with users (planners, buyers, logistics personnel, CIM
engineer), to gather new requirements for the ever changing demand of the
paging business.

LEVERAGE SYSTEMS TECHNOLOGY June 1992 to Nov. 1992
Systems Analyst
For Singapore Network Services PTE. LTD., Singapore
Phoenix Message Management System: The purpose of the project was to
effectively managed the traffic of messages.
Assisted the systems analyst in designing the screen layout, writing
programs specifications, and in conducting system/integration testing.
Coded, prepared test plan, test cases, test data to be used in testing
phase. Conducted a preliminary evaluation of three CICS debugging tools,
namely Compuware's Xpediter/CICS, Intertest, and SmartTest. All work done
using IBM 9121, MVS/ESA, TSO/ISPF, CICS command level, VS COBOL II,
NETVIEW, FileAid, SDF2, VSAM Application: MAIL Management System

INFOLINK CORPORATION March 1991 to May 1992
Systems Analyst/Programmer
For Bank of The Philippine Islands (BPI), Philippines
Banking Software (Systematics) System:
Coded additional programs to meet new user requirements. Revised
programs to incorporate enhancements. Prepared test plan and test data.
Performed unit testing of these programs. Assisted the system analyst in
conducting system and integration testing. Assumed maintenance
responsibilities for existing programs, which include trouble shooting,
problem determination and resolution, and improving program efficiency and
logic. All work done using : IBM ES/9121, MVS/ESA, TSO/ISPF, CICS command
level, VS COBOL II, VSAM.

CITIBANK, NA, Philippines Jan. 1992 to Feb.
1992
Mortgage Power Loans System:
Improved the performance (i.e. to enable the system to run faster and
consume less resources) from CICS programs already running in production.
Improved program logic and clarity to ensure easy maintainability. Coded
two new on-line programs for the Mortgage Power System in my short stay
with the company. Tested and debugged these programs to ensure conformity
with specifications. All work done using IBM 4381, ICCF, CICS command
level, COBOL, VSAM.

P.T. BANK NIAGA - INDONESIA Mar. 1991 to Dec.
1991
Current Account/Savings Account (CA/SA), Term (Fixed) Deposits:
Converted programs that utilizes CICS Basic Mapping Support commands, to
COBOL call commands so that it could interface with front-end programs
running on Unisys.
Tested and debugged these converted programs using CA-Intertest to
ensure proper communication with front-end UNISYS teller terminals. Coded,
tested, and documented file maintenance programs. Painted and generated
maps using SDF II. All work done using IBM 4381/9121, VSE/ESA ICCF, CICS
command level, COBOL, VSAM, CA-Intertest, CA-VOLLIE, SDF.

DYNAMIC SOFTWARE SOLUTIONS, INC. May 1989 to Feb. 1991
Analyst Programmer
For Security Bank and Trust Company, Philippines
Retail Banking Software Package (JSRBS):
Customized JS-RBS's 3270 administrative terminal system to meet SBTC's
requirement
Gathered user requirements by coordinating with the bank's managers and
supervisor as to what areas that can, and can not be supported by the
software package. Identified areas (programs, maps, tables, communication
areas etc.) that will be affected by SBTC's requirements.
Wrote specification for programs that will be affected by the
customization and also specification for new programs. Designed new screen
layout. Conducted systems testing, user's training and supervised revision
of user manuals. IBM 9221, VSE/ESA, ICCF, CICS command level, COBOL, VSAM.
Supervised five programmers:
Gave schedules for completing the programs so as to meet the bank's
deadline
Assisted the programmers in logic formulation, program efficiency and
debugging.

FAREAST BANK & TRUST COMPANY March 1990 to Oct. 1990
Remittance, Correspondence Banking, Foreign Rates, Exports:
Coded on-line program using CICS/COBOL/VSAM, debugged, prepared test
plan that stress tested the programs and documented programs. Created JCL
that will be used to run batch programs. Assisted the system analyst in
designing screen layouts. All work done using IBM 3090, MVS/XA, CICS
command level, VS COBOL II, TSO/ISPF, VSAM, MVS/JCL.

PHILIPPINE NATIONAL BANK Jan. 1990 to Mar. 1990
Retail Banking Software Package (JSRBS):
Was part of a team that customized a retail banking software package
(JSRBS) using IBM 3090, MVS/ESA, CICS command level, VS COBOL II,
TSO/ISPF, VSAM.


JOINT SERVICES PTY., LTD., Australia Sept. 89 to Jan 90
Current Account, Savings Account, Term (Fixed) Account:
Worked closely with the vendor of the retail banking software package.
The objective of the trip was to help the vendor expand/extend all balance
fields (anything involving currency). These balance fields had to be
expanded because the length allocated is not enough in some instances to
accommodate Philippine Peso.
Tested and debugged these field extended programs, wrote program
specifications for programs that will be affected by the field extensions.
Assisted in conducting systems testing. IBM 3090, MVS/ESA, CICS command
level, VS COBOL II, TSO/ISPF, VSAM, CA-Intertest.

PHILIPPINE NATIONAL BANK May 1989 to Sept. 1989
Credit Card System:
Designed screen and report layout for PNB credit card system, wrote
program specifications for some of the report generation requirement of
the system. Coded and tested programs. Conducted intensive month long
training in CICS command level programming for 5 staff with no previous
CICS programming experience. IBM 3090, MVS/ESA, CICS command level, VS
COBOL II, TSO/ISPF, VSAM, MVS/JCL.

SYSTEMS RESOURCES INC. Oct. 1987 to April 1989
Programmer
For United Airlines Corp.
Airline Operation System (Human Resources, Property, Accountability,
Inventory Management, Warehouse Control).
Coded programs for AAFOSS (Advanced Automated Facilities Operation)
project using COBOL as host language, SQL as database access facility and
CICS as data communication facility. Prepared test data and test plan.
Tested and debugged programs as per specification. Assisted the systems
analyst in conducting system testing. IBM 4341/9370, IBM S/36, VSE/SP,
ICCF, CICS command level, COBOL, SQL.
