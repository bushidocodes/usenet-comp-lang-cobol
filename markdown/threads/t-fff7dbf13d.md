### Free COBDATA tool now supports Micro Focus

```
The popular free tool "COBDATA" has been enhanced so that it can analyze
Micro Focus COBOL data definitions. Previously, COMP (binary) fields in
Micro Focus record layouts were reported with incorrect lengths for that
environment. This caused subsequent fields offsets/positions and the
overall record length to be reported incorrectly. Although Micro Focus
provide a compiler directive to enforce the IBM lengths (IBMCOMP), some
shops do not allow this.

If you already have COBDATA you need to uninstall it from "Add/Remove
Programs" (BOTH components... COBDATA Tool and COBDATA Delivery), THEN:

http://primacomputing.co.nz/PRIMAMetro/newCOBData.aspx

Run BOTH PHASES of the install...

This install (and the components) are now code signed so it should be
easier than before.

Please post any comments here or to in··o@pri··o.nz

Cheers,

Pete.

"I used to write COBOL...now I can do anything."
```
