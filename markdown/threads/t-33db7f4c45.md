### Fujitsu COBOL for .NET Beta is now available for download

- **From:** "Ron Langer" <cobol@adtools.com>
- **Date:** 2002-01-11T11:05:56-08:00
- **Newsgroups:** comp.lang.cobol
- **Message-ID:** `<a1nd3t$r14$1@slb0.atl.mindspring.net>`

```
I am pleased to announce that the Fujitsu COBOL for .NET Beta is now
available for download from http://www.adtools.com/dotnet/index.html.



The beta supports most COBOL85 syntax and is ready for those who would like
to start migrating their COBOL code into the .NET Framework. Also the syntax
extensions supporting .NET coding have been firmed up so you can work with
the syntax that will be in the released product.



To obtain your copy of the beta product, connect to the Download link on the
http://www.adtools.com/dotnet/index.html page, register with our new
download service, and answer a few survey questions.



I would be most interested to hear your feedback on this beta product.



Please note that you will need the Visual Studio .NET Release Candidate,
which requires Windows 2000 or XP. Applications can be deployed on Windows
98, Me, and NT 4.0 as well.



Sincerely,



Ron Langer,

Director of Languages

Fujitsu Software Corporation

COBOL@ADTOOLS.COM

http://www.adtools.com/
```
