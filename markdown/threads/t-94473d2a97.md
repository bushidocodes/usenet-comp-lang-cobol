I can give you a clue. It is probably due to the data set organization.
Your program is probably expecting a fixed format (10 bytes) record,
while the actual file format might be "line sequential". That is each
record terminated by a "carriage return" and/or "line feed".

If you can dump the beginning of the file in hex (or octal) you should be
able to easily find this out.

Hope this will help,
