Take a look on Amazon for "Cobol for dummies"  ( no insult intended! ).  It
teaches the basics of the language, and comes with a CD that contains about
4 or 5 compilers:   The book is worth buying for the CD alone.   I'd heard

few compilers and seeing which one you're the most comfortable with.  I've
been using Microfocus (Merant) on and off for about 8 years, and I find it
the most productive.

Dieter:

Here is an answer to all three of your questions:

The best place to get an inexpensive compiler or to learn how to write
COBOL code is to buy Thane Hubbell's book, "Teach Yourself COBOL In 24
Hours!"

The book only costs about $20.00 US$ and includes the compiler at no
additional charge.

It is not only a great book, but also incudes 16 and 32 bit versions
of the Fujitsu COBOL compiler.  The ISBN number of Thane's book is:

0-672-31453-3
