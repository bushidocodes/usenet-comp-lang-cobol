Here's some pseudocode that might help.

    parse the address (http://hostname/path) into it's component parts
    resolve the hostname to an IP address using gethostbyname()

    set up a request buffer
        string "GET " delimited size path delimited space x"0d0a" delimited
size into buffer
    send a TCPIP request (port=80)
    get the response

Some API functions that are required for TCP/IP communications:
        socket()    % create a socket
        connect()
        send()
        do recv() until all the document is received
         closesocket()    % destroy socket
