```
RM Cobol85 has a set of utility programs to rebuild a
corrupted RM database. The procedure for a file named
MYFILE is as follows:

RECOVERY MYFILE MYFILE1 MYFILE2

This procedure will rebuild the index in place and put
any dropped records in either MYFILE1 or MYFILE2.

Sam Butler
```
