```
You use DISPLAY and ACCEPT statements in conjunction with
SPECIAL-NAMES for ENVIRONMENT-NAME and ENVIRONMENT-VALUE;
e.g.,
       SPECIAL-NAMES.
           ENVIRONMENT-NAME  IS ENV-NAME. 
           ENVIRONMENT-VALUE IS ENV-VALUE. 
       WORKING-STORAGE SECTION.
       01  ENV-VAR-NAME   PIC X(5)   VALUE "PATH".
       01  ENV-VAR-VALUE  PIC X(256) VALUE SPACE.
       01  ENV-VAR-SWITCH PIC X      VALUE "0".
           88  ENV-VAR-OKAY          VALUE "0".
           88  ENV-VAR-FAILED        VALUE "1".
       PROCEDURE DIVISION.
           DISPLAY ENV-VAR-NAME UPON ENV-NAME
               ON EXCEPTION
                   SET ENV-VAR-FAILED TO TRUE
           END-DISPLAY
           IF ENV-VAR-OKAY
               ACCEPT ENV-VAR-VALUE FROM ENV-VALUE
                   ON EXCEPTION
                       SET ENV-VAR-FAILED TO TRUE
               END-ACCEPT
           END-IF
           .

Use the exception handler, especially under WNT 3.51 or
when setting values (DISPLAY ... UPON ENV-VALUE).  See
the relevant sections of the Language manual for full
information.
```
