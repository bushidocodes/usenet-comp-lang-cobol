```
Here is Proj0401 r05-drvr.cbl :

       $set sourceformat "free"
*>************************************************
*> Room Rental System
*> W. Price  Nov 23, 1999            R05-DRVR.CBL
*>
*> This driver provides the user a menu allowing 
*> for full room prices, discounted room prices, 
*> or room capacity.  It illustrates invoking 
*> different methods of the room class.
*> The following classes comprise this project:
*>   r05-room   Room
*>************************************************

Identification Division.
       Program-ID.  Room05Driver.

Environment Division.
       Object Section.
         Class-control.
          RoomClass is class "r05-room"
         .
Data Division.
  Working-Storage Section.

    01  room-prices.
        10  std-config-price      pic 9(03).
        10  spec-config-price     pic 9(03).
    01  room-capacity             pic 9(02).

    01  menu-choice               pic X(01).
        88  discounted-prices       value "D" "d".
        88  standard-prices         value "S" "s".

    01  theRoomHandle   usage object reference.

Procedure Division.
  000-process-room-data.

    *> Create an instance of the room object
    Invoke RoomClass "new"
                     Returning theRoomHandle
    Perform 850-get-menu-selection
    Evaluate TRUE
      When discounted-prices
        Invoke theRoomHandle
                    "get-discounted-room-prices"
                    Returning room-prices
        Perform 900-display-room-prices
      When standard-prices
        Invoke theRoomHandle "get-room-prices"
                    Returning room-prices
        Perform 900-display-room-prices
      When other
        Invoke theRoomHandle "get-room-capacity"
                    Returning room-capacity
        Perform 950-display-room-capacity
    End-evaluate *>TRUE
    Stop run
    .
  850-get-menu-selection.
    Display " "
    Display "The options are:"
    Display "  C - Room capacity (default)"
    Display "  S - Standard room prices"
    Display "  D - Discounted room prices"
    Display "   "
    Display "Your choice <C,S,D>? " no advancing
    Accept menu-choice
    .
  900-display-room-prices.
    Display " "
    Display "Standard configuration price: " 
                               std-config-price 
    Display "Special configuration price:   " 
                              spec-config-price 
    .
  950-display-room-capacity.
    Display " "
    Display "Room capacity: " room-capacity
    .
```

```
Here is Proj040` r05-room.cbl  :
       $set sourceformat "free"
*>*************************************************
*> Room Rental System
*> W. Price  Nov 23, 1999             R05-ROOM.CBL
*>
*> This class is similar to R04-ROOM except it 
*> utilizes the OBJECT paragraph for the room
*> object rather than the FACTORY paragraph.
*> Thus it is necessary to create an instance of
*> the room object.
*> Methods.
*>   get-discounted-room-prices
*>   get-room-capacity
*>   get-room-prices
*>*************************************************

Identification Division.
       Class-id.  RoomClass 
                  inherits from BaseClass
                  .
Environment Division.
       Object Section.
         Class-control.
          RoomClass is class "r05-room"
          BaseClass is class "base"
         .
*>=================================================

OBJECT.
  Data Division.
    Object-Storage Section.
    01  room-data.
        10  room-number     pic X(03)  value "100".
        10  std-config-price  pic 9(03)  value 225.
        10  spec-config-price pic 9(03)  value 385.
        10  room-discount     pic v9(02) value .05.
        10  room-capacity     pic 9(02)  value 38.

  Procedure Division.

    *>---------------------------------------------
    Method-id. get-discounted-room-prices.
    *>---------------------------------------------
      Data Division.
        Local-Storage Section.
          01  lo-price-factor         pic 9(01)v9(02).

        Linkage Section.
          01  ls-room-prices.
              10  ls-std-config-price   pic 9(03).
              10  ls-spec-config-price  pic 9(03).

      Procedure Division Returning ls-room-prices.
        Move std-config-price to ls-std-config-price
        Move spec-config-price to ls-spec-config-price
        Subtract room-discount from 1.0
                                 giving lo-price-factor
        Multiply lo-price-factor 
                             by ls-std-config-price
        Multiply lo-price-factor 
                             by ls-spec-config-price
          .
    End Method get-discounted-room-prices.

    *>---------------------------------------------
    Method-id. get-room-prices.
    *>---------------------------------------------
      Data Division.
        Linkage Section.
          01  ls-room-prices.
              10  ls-std-config-price    pic 9(03).
              10  ls-spec-config-price   pic 9(03).

      Procedure Division  Returning ls-room-prices.
        Move std-config-price to ls-std-config-price
        Move spec-config-price to ls-spec-config-price
        .
    End Method get-room-prices.

    *>-----------------------------------------------
    Method-id. get-room-capacity.
    *>-----------------------------------------------
      Data Division.
        Linkage Section.
          01  ls-room-capacity             pic 9(02).

      Procedure Division  Returning ls-room-capacity.
        Move room-capacity to ls-room-capacity
        .
    End Method get-room-capacity.
    *>-----------------------------------------------
END OBJECT.
END CLASS RoomClass.
```
