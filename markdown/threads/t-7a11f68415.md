```
In Fujitsu V4, you can specify an environmental variable
which will eliminate the task bar entry.  I'm not sure
if it works with V3 (doubt it, actually)

@EnvSetWindow=UNUSE

After entering the item, you must save the environment.
It won't take effect until the *next* time you execute the
program.

The environmental data is saved in a ".cbr" file.  The
file must be in the directory you are running from.  If you
are testing in the debug mode, and are running from
the "debug" subdirectory (this is the default), you need
to make sure the file is saved in that directory.
```
