Once you've saved a COBOL Source file, you need to select 'Tools' from the
menu bar and click on 'Compile' to syntax check the source file, click on
'Link' 
to generate an EXE/DLL from the OBJ created by a successfull compile, click
on 'WinEXEC' to run the compiled/linked EXE program.

Hope this helps
