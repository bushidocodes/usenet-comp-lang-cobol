This is an error that occurs when attempting to 'check-out' a Micro Focus
Application Server (Runtime) license. Here's a possible explanation:

This error message may be caused by a missing semaphore file (aslmfsem) in
/var, or if the permsissions on the file, or on /var are inadequate.
Since the file is zero length, it can be simply created using the following
command, having cd'ed into /var.


   touch aslmfsem
The file should have R/W permissions for all, and should be chowned to root.
If the file is present, with the correct permissions and the problem
persists, check that the permissions on /var itself allow users to access
this file.
