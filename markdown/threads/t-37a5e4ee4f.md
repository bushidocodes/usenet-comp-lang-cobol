Big Think host a series of interviews covering IT subjects with Bjarne Soustrup and Larry Wall being asked about which five computer languages a person should know. Larry touches on Cobol. Link:

http://bigthink.com/videos/big-think-interview-with-larry-wall

When asked about which 5 computer languages people should know he replied:

"Java is sort of the Cobol of the 21st century, I think. It's kind of heavyweight, verbose, and everyone loves to hate it, though not everyone will admit that. But managers kind of like it because it looks like you're getting a lot done, you know, if 100 lines of Java code accomplish a task, then it looks like you've written 100 lines, even though in a different language, it might only take 5 lines."
