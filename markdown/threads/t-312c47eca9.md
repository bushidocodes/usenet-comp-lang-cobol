In 'Program A', I am calculating the amount of time it has been since
'Program B' has previously executed.

Every time 'Program B' starts, I write that time to a file.  I read this
file and move the time to WS-PREV-RUN-TIME.  I move the system time to
WS-SYS-TIME.  I convert both times from hhmmss into seconds.  I then
subtract WS-PREV-RUN-TIME-SECS from WS-SYS-TIME-SECS.  I take the
difference a convert it back to hhmmss.

I've tested the code and I know it works.  Is there a better calculation
I can use?  I'm also open to any suggestions regarding my style of code.

Thank you.

Below is a sample of my code :

 01  WS-WORK-FIELDS.
     05  WS-TEST-DATE            PIC 9(08) VALUE 20001229.
     05  WS-CICS-FIELDS.
        10  WS-ABSTIME           PIC S9(15) COMP.
        10  WS-RESPONSE-CODE     PIC S9(08) COMP.
        10  WS-RESPONSE-CODE2    PIC S9(08) COMP.
        10  WS-DAY-OF-WEEK       PIC S9(08) COMP.
            88 SUN-ISS           VALUE 0.
            88 MON-ISS           VALUE 1.
            88 TUE-ISS           VALUE 2.
            88 WED-ISS           VALUE 3.
            88 THU-ISS           VALUE 4.
            88 FRI-ISS           VALUE 5.
            88 SAT-ISS           VALUE 6.
******SYSTEM TIME
        10  WS-SYS-TIME          PIC 9(06).
        10  WS-SYS-TIME-R        REDEFINES WS-SYS-TIME.
            15  WS-SYS-TIME-HH   PIC 9(02).
            15  WS-SYS-TIME-MM   PIC 9(02).
            15  WS-SYS-TIME-SS   PIC 9(02).
        10  WS-DATE              PIC 9(08).
        10  WS-RIDFLD            PIC S9(09) COMP.
****PREVIOUS RUN TIME
     05  WS-PREV-RUN-TIME        PIC 9(06).
     05  WS-PREV-RUN-TIME-R      REDEFINES WS-PREV-RUN-TIME.
         10  WS-PREV-RUN-HH      PIC 9(02).
         10  WS-PREV-RUN-MM      PIC 9(02).
         10  WS-PREV-RUN-SS      PIC 9(02).
****DIFFERENCE BETWEEN PREV-RUN-TIME AND SYS-TIME
     05   WS-DIFF-TIME           PIC 9(06).
     05   WS-DIFF-TIME-R         REDEFINES WS-DIFF-TIME.
         10  WS-DIFF-TIME-HH     PIC 9(02).
         10  WS-DIFF-TIME-MM     PIC 9(02).
         10  WS-DIFF-TIME-SS     PIC 9(02).
***MISCELLANEOUS TIME FIELDS.
     05  WS-PREV-RUN-TIME-SECS   PIC 9(05).
     05  WS-SYS-TIME-SECS        PIC 9(05).
     05  WS-REMAINDER-MM         PIC 9(05).
     05  WS-REMAINDER-SS         PIC 9(05).
************************************************************

 READ-LAST-START-ADS0600.
**OBTAIN PREVIOUS RUN TIME.
     MOVE 'ADIT' TO COD-KEY.
     PERFORM GIT-COD-REC.
     MOVE COD-NBR (2) TO WS-PREV-RUN-TIME.

 READ-LAST-START-ADS0600-EXIT.     EXIT.

 COMPUTE-TIME-DIFF.
**OBTAIN SYSTEM TIME.
     PERFORM CICS-ASK-TIME.
**CONVERT PREV-RUN-TIME INTO SECONDS.
     COMPUTE WS-PREV-RUN-TIME-SECS =
        (WS-PREV-RUN-HH * 3600) + (WS-PREV-RUN-MM * 60) +
        WS-PREV-RUN-SS.
**CONVERT WS-SYS-TIME INTO SECONDS.
     COMPUTE WS-SYS-TIME-SECS =
        (WS-SYS-TIME-HH * 3600) + (WS-SYS-TIME-MM * 60) +
        WS-SYS-TIME-SS.
**CALCULATE WS-DIFF-TIME.
     SUBTRACT WS-PREV-RUN-TIME-SECS FROM WS-SYS-TIME-SECS.
     DIVIDE WS-SYS-TIME-SECS BY 3600
        GIVING WS-DIFF-TIME-HH
        REMAINDER WS-REMAINDER-MM.
     DIVIDE WS-REMAINDER-MM BY 60
        GIVING WS-DIFF-TIME-MM
        REMAINDER WS-REMAINDER-SS.
     MOVE WS-REMAINDER-SS TO WS-DIFF-TIME-SS.

 COMPUTE-TIME-DIFF-EXIT.         EXIT.
