Are you new to the COBOL newsgroups?
If so, please read the COBOL FAQ before posting your questions, comments,
or input. It can be viewed at:

http://www.netcom.com/~wmklein/cobolfaq.htm

Some of the questions that it will answer for you are:

Where can I get a (free) COBOL compiler?
Can I post job openings in this newsgroup (and what should I put in it)?
Can you help with my homework project?

If you don't have access to a web browser, a text version is posted to the
newsgroup about twice a month.
