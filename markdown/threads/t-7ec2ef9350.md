Ah, especially check to see what you have in column 7.  The above characters are

> *    37:JMN1004I-W    Reserved word 'PERFORM' must start in area B. Assumed
> to start in area B.

More of the same.   Your program isn't aligned correctly.   While some compilers
have variations, the standard is to have 1-6 for sequence numbers (the compiler
doesn't look at these), column 7 for those symbols above, column 8-11 for the
start of commands which must start in area A (DIVISION, SECTION, FD, 01,
PARAGRAPHS, etc), 12-72 for everything else, and 73-80 as a redundant sequence
number place.

Lots of shops put change control comments in the sequence number columns.
