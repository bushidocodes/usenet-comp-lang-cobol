```
Error 105 is a memory allocation error, not a file allocation error. It
means that the run-time system cannot allocate memory for an operation being
executed. You will normally only get that if a CBL_ALLOC_MEM call specified
a huge size, or if a memory leak or corruption had occurred.

Have you tried animating the program to find out on exactly which COBOL
statement the error occurs, and have you also tried the same
open/close/delete without any other code in between to see if the error
still happens? Doing both should help you track down the problem.
```
