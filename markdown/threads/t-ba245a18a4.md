```
I think that you are asking how you access the data items from two
tables in a single select via a cursor.

Something like:

   DEFINE ... CURSOR FOR
    SELECT * FROM A, B WHERE A.value > 1000.00 and B.code = A.code

    FETCH ... (A.code, B.name) INTO :code, :name
```
