It sounds like the old DOS machine had partitioned drives to
overcome the old MS-DOS limit of 32Mbyte disks and thuis had
C:, D:, E:, .... and the drives were included in the filenames,
or possibly in a configuration file.

You now have a machine with only a C: drive and thus it can't find
E:.  Try doing some drive mapping.  If you don't have a network
use SUBST E: C:\ to map E: to C:, or similar.
