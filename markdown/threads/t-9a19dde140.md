JC:

  First to answer Oscars question yes you can rename wrun32.exe anything you
want.  Starting with Acucobol-GT version 4.3 there are two files: wrun32.dll
and wrun32.exe.  When you remake the runtime you now remake the .dll.  You
cannot change the name of the .dll since the .exe calls it.  However you can
still rename the .exe anything you like.

  Now as far as your invaid page fault is concerned I would suggest that you
forward your question to support@acucorp.com.  They'll need to know the
release you're running under, what the program is doing at the time is
occurs, whether it reproduces consistantly or is random, etc.  If you are
doing gui programming and the version is less that 4.3 we had some memory
leaks which could cause this error.  If it's one of them it would appear to
occur randomly over time.  Upgrading to 4.3 would resolve it assuming that's
the problem.  However Tech Support can tell you better.

Peter Withey
