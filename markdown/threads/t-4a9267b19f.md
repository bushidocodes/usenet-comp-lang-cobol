It is available the new release 0.1.5.5 of the Tiny Cool compiler.

This minor release contains some bugs fixes, some code enhancements
including updates to the PERFORM statements. With this release, all
PERFORM statements are functional and conform to the COBOL 85 standard.

Tiny Cobol is a free cobol compiler under GPL license. It can be found
at http://tiny-cobol.sourceforge.net
Mirror inquiries are welcome.
