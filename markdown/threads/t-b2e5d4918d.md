```
Assuming you've got a SPECIAL-NAMES paragraph with a CALL-CONVENTION 74 IS
WINAPI statement then the would look like this.

	CALL WINAPI "Sleep" USING BY VALUE 10 size 4 

You must use the CASE compiler directive to allow case-sensitive entrypoints.
The entrypoint in the API is Sleep@4, not SLEEP@4.
```
