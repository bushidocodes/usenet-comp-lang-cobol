```
Your code is fine, except for

> 002740 REPORT-PARAGRAPH.
> 003060             GENERATE report1
> 003061    READ sorted-file NEXT RECORD
> 003062          AT END SET end-of-file-flag TO TRUE
> 003066                     END-READ.

No need to READ Sorted-File in this paragraph. After you generate the report
line, just let the paragraph exit, and the next RETURN will get you the next
line of detail to report.

i.e., when you RETURN a line,  your NOT AT END should look something like...

   RETURN sort-record
     NOT AT END
      PERFORM   REPORT-PARAGRAPH

...

REPORT-PARAGRAPH.

     IF lines-this page GREATER  THAN  MAX-LINES-PER-PAGE
        PERFORM PAGE-HEADER-ROUTINE
    END-IF
    IF  CONTROL-BREAK-CONDITION     (new name?)
         PERFORM  CONTROL-BREAK-ROUTINE
    END-IF
    MOVE  (fields of sort record) TO (fields of report-line)
    WRITE REPORT-LINE
    ADD    1  TO Lines-this-page

   CONTINUE.

OTHER-PARAGRAHS.


MCM
```
