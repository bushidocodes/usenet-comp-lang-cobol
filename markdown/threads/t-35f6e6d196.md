from one vendor format to another. There are four different file types and

arguments. I have
specified the ARGUMENT-NUMBER and ARGUMENT-VALUE in the SPECIAL-NAMES
SECTION
of the main form. The code to retrieve the arguments has been attach to the
'OPEN' event of the form
but this does not work because, I guess, it is in a sub program. I don't see
how to get this code to
be in the main program and be executed prior to the form being displayed.
I'm sure there is a simple
solution but it seems to be eluding me.

Thanks for any help.

> What does this mean, Right-clicking on POW-SELF ?

Type the word POW-SELF in your program. Highlight the word by
double-clicking. Right-click on the highlighted word. There are two lines in
the info box of interest: Methods and Properties. Clicking on each provides
a list of things that are doable.

Right-clicking on the name of any form element does the same thing. Suppose
you have a Text Box named MY-TEXT-BOX. Type MY-TEXT-BOX in your program.
Double-click. Right-click. Whoopee!

But that's not all, folks!

Click on the Method or Property of your choice in the pop-up menu.
PowerCOBOL lays in the skeleton of the command. In the case of the property
of interest above, clicking on "CommandLine" as a property changes

POW-SELF

to

 "CommandLine" of POW-SELF

You now put MOVE in the front and "TO MY-COMMAND-LINE" at the end. This is
not the best example. Some Methods and Properties can get verbose, so you'll
appreciate not being allowed to forget an essential parameter.
