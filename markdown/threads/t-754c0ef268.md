The difference is because the MVS compiler permits the non-standard
form, and Fujitsu does not.  The COBOL 85 standard Format 3 PERFORM
syntax is as follows (on page VI-109):

    PERFORM [ procedure-name-1 [ {THRU|THROUGH } procedure-name-2 ] ]
    -------                       ---- -------

        [ WITH TEST {BEFORE|AFTER} ]  UNTIL condition-1
               ----  ------ -----     -----

        [ imperative-statement-1  END-PERFORM ]
                                  -----------

In other words, you can use END-PERFORM, if and only if, you include
an imperative statement.  This requirement made is on all the PERFORM
statement formats.  The purpose of END-PERFORM is to delimit the code
included in the inline PERFORM.  If there is no inline code, there is
no need for an END-PERFORM. :-)
