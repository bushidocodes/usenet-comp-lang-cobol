```
Announce: TinyCOBOL release 0.52

The TinyCOBOL compiler version 0.52 is available at the TinyCOBOL home
page (1) and iBiblio (2).

This release contains bugs fixes, and enhancements, including updates to
the preprocessor, math verbs, MOVE (CORRESPONDING option) statement,
COMPUTE (exponent operator)
statement, enhancements to the regression test suite.

Support for the BLANK WHEN ZERO clause, SYNC clause, qualification,
OCCURS...DEPENDING ON... clause (variable-sized tables), RECORD VARYING
... DEPENDING ON ... (variable-sized records) clause, multiple record
defines for FD entries, sequence of source programs, have been added to
the compiler.


Please report any bugs and/or problems to the TinyCOBOL mailing list. 


1) TinyCOBOL Home Page
   http://tiny-cobol.sourceforge.net/

2) iBiblio (formerly Metalab)
   http://www.ibiblio.org/pub/Linux/devel/lang/cobol/
```
