Hi:

A lot of people in this forum don't like answering questions like
this because they think that you want them to do your homework for you.

You have to define multiple record types in your file. Each record
description defines the fields according to the type. When you read the
file, since you probably wouldn't know what record was coming next, you
have to check the record code and use the record description appropriate
to the record just read.
