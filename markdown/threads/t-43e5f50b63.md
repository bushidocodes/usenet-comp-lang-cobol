Fujitsu's error message parser has some REALLY big holes. You can try:

1. Rearranging the order of the FDs / SDs.
2. Rearrange the order of the SELECT statements.
3. Add/Remove optional clauses (RECORD CONTAINS, BLOCK CONTAINS,
LABEL RECORDS STANDARD, etc.)
4. Add extra records to the FD/SD that have no funcitonal use:
   FD  MYFILE.
   01  MYRECORD                          PIC X(80).
   01  DUMMYRECORD                 PIC X(80).

Whatever technique you find will not necessarily work on the next
occurrence of the same message.

Good Luck.
