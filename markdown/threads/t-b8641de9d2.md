```
*actually* you can (but don't have to) change where DISPLAY output goes.
This is done by

A) Using the "UPON" phrase in the DISPLAY statement
   and/or
B) Associating another name with "SYSOUT" in the Special-Names paragraph.
   and/or
C) using the OUTDD compiler option.

Having said that, Michael's answer (below) will certainly work
```

```
When/if you see "SYSOUX" in an IBM Mainframe shop, this is (usually) a
"historical" left-over from OS/VS COBOL days - when you could ONLY change
the last character of "SYSOUT".  It wouldn't surprise me if Howard's shop
has set the *default* (possibly not even modifiable by application
programmers) for the OUTDD compiler option to SYSOUX.  If you/he looks at a
compiler listing (assuming he is using VS COBOL II or later), my guess is
that this is what OUTDD will show.

*EVEN* when you use the OUTDD statement, it is still possible to over-rider
where DISPLAY output goes - by using the "UPON" phrase of the DISPLAY
statement.
```
