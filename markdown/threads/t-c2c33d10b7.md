You don't want this period ---------------|
This is likely what is causing the problem, though the interpreter should
flag it.

› END-PERFORM.
› QUITPROGRAM.
^^^^^^^^^^^
What in the world is this? Use the standard STOP RUN or, if you are
'dropping out' of the bottom of the program like this, a STOP RUN is
implied, and you don't have to code it.

Although the VARYING option will work here, you don't use I for anything
else, so the TIMES option is more fitting. Here is a slightly improved
version, which definitely compiles and runs under Micro Focus COBOL.

IDENTIFICATION DIVISION.
PROGRAM-ID. Sig.
ENVIRONMENT DIVISION.
DATA DIVISION.
WORKING-STORAGE SECTION.
PROCEDURE DIVISION.
GET-OFF-MY-FOOT.
PERFORM 3 TIMES
DISPLAY "You're on my foot!"
END-PERFORM.
