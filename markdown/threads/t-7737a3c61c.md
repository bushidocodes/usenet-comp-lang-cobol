```
First:

When you're reporting needs get complex - use CoBOL, not EasyTrieve.

Second - if you don't have an appropriate language, use the following code
(from a report description) as an example:


 HEADING HOLD-LAMT ('LOAN RECEIVABLES' 'OUTSTANDING' 'AMOUNT')
 LINE 1 COL  20 FOPPS                                     +
        COL  55 DESCRIPTION                               +
 -  -  -  -  -  -  -  -  -  -  -  -  -  -  -  -  4 Line(s) not Displayed

 BEFORE-BREAK. PROC
  IF LEVEL EQ 4
     DISPLAY '              '
     DISPLAY 'FINAL TOTALS '
  END-IF

   IF LEVEL EQ 3
      DISPLAY 'FOPPS TOTAL                                      ' +
       '      --------------    ------------------    --------'  +
       '  ----------------   ------'
   END-IF
 END-PROC
 AFTER-BREAK. PROC
   IF LEVEL EQ 3
      DISPLAY '              '
   END-IF
 END-PROC
```
