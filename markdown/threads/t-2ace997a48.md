```
I'm running the latest version of mf cobol on
a unix system. I have a printer queue which
I set up as "BANDLP", the name of the raw
device is "/dev/lp0". I can get a program to
print to the raw device, but I can't get it to
print to the queue. Any suggestions would
be appreciated. Example follows:

SELECT REPORT-FILE 
     ASSIGN TO PRINTER PRINTER-NAME
     ORGANIZATION SEQUENTIAL
      etc.

MOVE "/dev/lp0" TO PRINTER-NAME.  


The above works, but I need to print the report
to the queue.
```

```
You can proceed like the following:

select report-file assign to external myPrinter.

Then, in the .profile of the appropriate user add the following:
DD_myPrinter = "> lp -d ThePrinter"
export DD_myPrinter

It should work for you as it works for me without any problem every
day.
```
