```
In case anyone who hasn't seen this yet is interested...

The Micro Focus User Conference this year is online. That means no
trip to the legendary conference location of Dallas; but on the plus
side, it's free (for the main content; there are some optional
for-charge workshops). It's happening 11-13 May.

See http://www.microfocuslive.com for details.

There are tracks for developers, managers, and ISVs. If you use MF
tools (including the Acu products), it's a chance to see what's in the
new releases and pick up some tips. There's a bunch of stuff on the
newer technology such as .NET support, Eclipse, and COBOL support for
cloud computing.
```
