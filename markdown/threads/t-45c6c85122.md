```
Hi,

I'm trying to write a routine in COBOL to generate a table of six digit
numbers. Each number must include all the digits from 1 to 6, and I need
to generate all 720 possible combinations from 123456 to 654321
automatically.
My attempts so far have failed, as I can't work out the appropriate
algorithm to do it neatly. So far, I've tried code like:

      01  ws-shuffle-area.
          02  ws-sh pic 9 occurs 6.

      f-shuffle.
           perform until ws-shuffle-area = "654321" 
              if sh(6) > sh(5)
                 move sh(5) to temp
                 move sh(6) to sh(5)
                 move temp to sh(6)
                 perform c-calculate            
              end-if   
              if sh(5) > sh(4)
                 move sh(4) to temp
                 move sh(5) to sh(4)
                 move temp to sh(5)
                 perform c-calculate            
              end-if   
              if sh(4) > sh(3)
                 move sh(3) to temp
          etc....
This approach, however, bubbles the high numbers backwards through the
number far too quickly.

I'm using RM-COBOL so I can't call on any intrinsic functions.

Can anyone think of a fast and compact approach?

No it's not homework, it's for fun!

Thanks in advance.
```

```
How neat do you want the code?  There are a couple of brute force
methods, including creating every possible number, and discarding those
which don't qualify.  

There are more efficient algorithms than what follows, but look at it.
PERFORM VARYING VALUE-1 FROM 1 BY 1 UNTIL VALUE-1 > 6
       PERFORM VARYING VALUE-2 FROM BY 1 UNTIL VALUE-2 > 6
       IF VALUE-2 = VALUE-1
           CONTINUE
       ELSE
           PERFORM VARYING VALUE-3 FROM 1 BY 1 UNTIL VALUE-3 > 6
           IF VALUE-3 = VALUE-1
           OR VALUE-3 = VALUE-2
               CONTINUE
           ELSE
    ....
            MOVE VALUE-1 TO WS-SH(1)
            MOVE VALUE-2 TO WS-SH(2) ...

Clean up the code to make it readable with all 6 levels of performs.
```

```
move 123456 to ws-shuffle-area.
set your table index to +1.
perform until ws-shuffle-area greater than 654321
  if (ws-sh1 + ws-sh2 + ws-sh3 + ws-sh4 + ws-sh5 + ws-sh6) equal 21
       move ws-shuffle-area to your table
       set your table index up by +1
  end-if
  add 1 to ws-shuffle-area
end-perform.
```
