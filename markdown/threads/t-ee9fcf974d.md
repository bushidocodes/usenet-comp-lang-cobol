```
"Peter Alluyn" <peter_alluyn@hotmail.com> wrote 

> Our company is searching a Linux Cobol Environment with cheap runtime
> licences.  All ideas are welcome.

Fujitsu for Linux - but it has no SCREEN SECTION or X/Open
ACCEPT/DISPLAY (ie DISPLAY .. AT or UPON CRT).  It is usable with SP/2
and/or CGI or other interactive systems.  Good SQL and file system
support.  Zero runtime cost.

OpenCobol - a work in progress.

TinyCobol - apparently usable in many ways, probably not in all ways.

Kobol - $59.95 per machine.  No file sharing/ record locking so may be
a problem with multi-user.  Apparently SQL will be available any day
now, which should solve multi-user access.

RM, MF, Accu.  require (AFAIK) runtime licences.
```
