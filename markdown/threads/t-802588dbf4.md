Please could you tell me if the program below works,If it doesn't any advice
where Iam going wrong would be greatly appreciated.
A program is required to validate batches of transactions that are held on a
disk file Each batch should consist of up to 100 transaction records
followed by a batch control record Each field of the transaction record
needs to be
validated and hash totals of all the numeric fields calculated

If the accumulated hash totals match those of the batch control record and
there are no invalid records and the number of transaction records in the
batch does not exceed 100, then the batch is to be classed as valid All
valid records are to be written to a valid file An erroneous batch is to be
printed out on an error report that contains appropriate headings, with each
batch starting on a new page It is also required that the accumulated hash
totals for that batch are printed after each invalid batch A separate disk
file is to be used as temporary storage of a batch until the validity of the
batch is ascertained
       IDENTIFICATION DIVISION.
       PROGRAM-ID.                BATCH-VALIDATION.

       ENVIRONMENT DIVISION.
          SELECT OPTIONAL TRAN-FILE ASSIGN TO 'TRANS.DAT'
             ORGANIZATION LINE SEQUENTIAL.
          SELECT TEMP-FILE ASSIGN TO 'TEMP.DAT'
             ORGANIZATION LINE SEQUENTIAL.
          SELECT VALID-FILE ASSIGN TO 'VALID.DAT'
             ORGANIZATION LINE SEQUENTIAL.
          SELECT PRINT-FILE ASSIGN TO PRINTER.

       DATA DIVISION.
       FILE SECTION.
       FD TRAN-FILE.
       01 TRANS-REC.
          03 TYPE-T            PIC X.
          03 REC-PART          PIC 9(8).
          03 REC-SUP           PIC 9(6).
          03 REC-DATE          PIC 9(6).
          03 REC-QUANT         PIC 9(5).

       01 HASH-REC.
          03 TYPE-B            PIC X.
          03 HASH-PART         PIC 9(8).
          03 HASH-SUP          PIC 9(8).
          03 HASH-DATE         PIC 9(8).
          03 HASH-QUANT        PIC 9(8).

       FD TEMP-FILE.
       01 TEMP-REC.
          03 T-TYPE-T          PIC X.
          03 TEMP-PART         PIC 9(8).
          03 TEMP-SUP          PIC 9(6).
          03 TEMP-DATE         PIC 9(8).
          03 TEMP-QUANT        PIC 9(5).

       FD VALID-FILE.
       01 VALID-REC.
          03                   PIC X.
          03 VAL-PART          PIC 9(8).
          03 VAL-SUP           PIC 9(6).
          03 VAL-DATE          PIC 9(6).
          03 VAL-QUANT         PIC 9(5).

       FD PRINT-FILE LINAGE IS 60 LINES.
       01 PRINT-REC            PIC X(80).

       WORKING-STORAGE SECTION.
       01 HASH-PART-TOT        PIC 9(8) VALUE ZERO.
       01 HASH-SUP-TOT         PIC 9(8) VALUE ZERO.
       01 HASH-DATE-TOT        PIC 9(8) VALUE ZERO.
       01 HASH-QUANT-TOT       PIC 9(8) VALUE ZERO.

       01 EOF                  PIC XXX VALUE 'NO '.
          88 NO-MORE-RECORDS           VALUE 'YES'.


       01 W-HASH-FILE.
          03 W-HASH-TYPE       PIC X VALUE 'B'.
          03 W-HASH-PART       PIC 9(8) VALUE ZERO.
          03 W-HASH-SUP        PIC 9(8) VALUE ZERO.
          03 W-HASH-DATE       PIC 9(8) VALUE ZERO.
          03 W-HASH-QUANT      PIC 9(8) VALUE ZERO.

       01 W-HEADING.
          03                   PIC X(20) VALUE SPACES.
          03                   PIC X(30) VALUE
               '***** ERROR REPORT *****'.

       01 W-HEADINGS-2.
          03                   PIC X(5) VALUE SPACES.
          03                   PIC X(7) VALUE 'PART NO'.
          03                   PIC X(8) VALUE SPACES.
          03                   PIC X(11) VALUE 'CUS/SUP NO.'.
          03                   PIC X(8) VALUE SPACES.
          03                   PIC X(12) VALUE 'DATE SUP/REC'.
          03                   PIC X(8) VALUE SPACES.
          03                   PIC X(8) VALUE 'QUANTITY'.

       01 W-TOT-LINE.
          03                   PIC X(5) VALUE SPACES.
          03                   PIC X(8) VALUE 'PART TOT'.
          03 PRINT-PART-TOT    PIC X(10) VALUE SPACES.
          03                   PIC X(7) VALUE 'SUP/TOT'.
          03 PRINT-SUP-TOT     PIC X(10) VALUE SPACES.
          03                   PIC X(8) VALUE 'DATE TOT'.
          03 PRINT-DATE-TOT    PIC X(10) VALUE SPACES.
          03                   PIC X(10) VALUE 'QUANTITY'.
          03 PRINT-QUANT-TOT   PIC X(10) VALUE SPACES.

       01 W-DETAIL-LINE.
          03                   PIC X(5) VALUE SPACES.
          03 INVAL-PART        PIC X(8) VALUE SPACES.
          03                   PIC X(7) VALUE SPACES.
          03 INVAL-SUP         PIC X(6) VALUE SPACES.
          03                   PIC X(13) VALUE SPACES.
          03 INVAL-DATE        PIC X(6) VALUE SPACES.
          03                   PIC X(14) VALUE SPACES.
          03 INVAL-QUANT       PIC X(5) VALUE SPACES.

       01 REC-COUNT            PIC 999 VALUE ZERO.

       01 W-ERROR-FLAG         PIC X VALUE 'N'.

       PROCEDURE DIVISION.

       MAIN-CONTROL.
          PERFORM INITIALISE
          PERFORM TRANS-PROCESS UNTIL NO-MORE-RECORDS
          PERFORM CLOSE-DOWN
          STOP RUN.

       INITIALISE.
          OPEN INPUT TRAN-FILE
               OUTPUT TEMP-FILE VALID-FILE PRINT-FILE
          PERFORM HEADINGS.

       TRANS-PROCESS.
          READ TRAN-FILE AT END
             MOVE 'YES' TO EOF
             NOT AT END
             EVALUATE TYPE-T
                WHEN 'T' PERFORM TRANS-PROCESS-2
                WHEN 'B' PERFORM HASH-PROCESS
             END-EVALUATE.

       TRANS-PROCESS-2.
          IF REC-PART NOT NUMERIC
             OR REC-SUP NOT NUMERIC
             OR REC-DATE NOT NUMERIC
             OR REC-QUANT NOT NUMERIC
                MOVE 'Y' TO W-ERROR-FLAG
          ELSE
             ADD 1 TO REC-COUNT
             ADD REC-PART TO W-HASH-PART
             ADD REC-SUP TO W-HASH-SUP
             ADD REC-DATE TO W-HASH-DATE
             ADD REC-QUANT TO W-HASH-QUANT
             WRITE TEMP-REC FROM TRANS-REC
          END-IF.

       HASH-PROCESS.
          IF HASH-PART NOT NUMERIC
            OR HASH-SUP NOT NUMERIC
            OR HASH-DATE NOT NUMERIC
            OR HASH-QUANT NOT NUMERIC
               MOVE 'Y' TO W-ERROR-FLAG
               PERFORM PRINT-DETAIL
          ELSE
                  PERFORM TEMP-VALIDATE
          END-IF.
       TEMP-VALIDATE.
          CLOSE TEMP-FILE
          OPEN INPUT TEMP-FILE
             READ TEMP-FILE AT END
             MOVE 'YES' TO EOF
             NOT AT END
                IF REC-COUNT <= 100
                   AND W-HASH-SUP = HASH-SUP
                   AND W-HASH-PART = HASH-PART
                   AND W-HASH-DATE = HASH-DATE
                   AND W-HASH-QUANT = HASH-QUANT
                   WRITE VALID-REC FROM TEMP-REC
                ELSE
                   MOVE 'Y' TO W-ERROR-FLAG
                   PERFORM PRINT-DETAIL
                END-IF
                   CLOSE TEMP-FILE
                   OPEN OUTPUT TEMP-FILE
                      MOVE ZERO TO REC-COUNT
                      MOVE ZERO TO HASH-PART
                      MOVE ZERO TO HASH-SUP
                      MOVE ZERO TO HASH-DATE
                      MOVE ZERO TO HASH-QUANT.

       PRINT-DETAIL.
          IF W-ERROR-FLAG = 'Y'
             OR REC-COUNT > 100
             IF W-DETAIL-LINE = SPACES
                MOVE REC-PART TO INVAL-PART
                MOVE REC-SUP TO INVAL-SUP
                MOVE REC-DATE TO INVAL-DATE
                MOVE REC-QUANT TO INVAL-QUANT
                ADD HASH-PART TO HASH-PART-TOT
                ADD HASH-SUP TO HASH-SUP-TOT
                ADD HASH-DATE TO HASH-DATE-TOT
                ADD HASH-QUANT TO HASH-QUANT-TOT
                WRITE PRINT-REC FROM W-DETAIL-LINE
                WRITE PRINT-REC FROM W-TOT-LINE
             END-IF
          END-IF.

       HEADINGS.
          WRITE PRINT-REC FROM W-HEADING AFTER PAGE
          WRITE PRINT-REC FROM W-HEADINGS-2 AFTER 1.

       CLOSE-DOWN.
          IF REC-COUNT > THAN ZERO
             OR W-ERROR-FLAG = 'Y'
             PERFORM PRINT-DETAIL
          ELSE
             CLOSE TRAN-FILE TEMP-FILE VALID-FILE PRINT-FILE
          END-IF.

Thank you.
