The pointer is incremented for each character examined. There is
no need to manipulate the pointer between UNSTRING statements.

Try the following:

       CALL "SYSREAD"
           USING CALL-Error-Explanation, Input-Pipe-FD,
                 Read-Buffer, Read-Len.
* Initialize pointer
       MOVE 1 to Read-Ptr.
       UNSTRING
           Read-Buffer DELIMITED BY ";"
           INTO Test-Count WITH POINTER Read-Ptr.
* Pointer now contains 5, 1 plus 4 characters examined
       UNSTRING
         Read-Buffer DELIMITED BY ";"
         INTO Storage-Record WITH POINTER Read-Ptr.
* Pointer now contains 25, 5 plus 20 characters examined
