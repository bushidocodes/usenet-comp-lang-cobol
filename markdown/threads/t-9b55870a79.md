```
Hi all,

I hope someone has come accross the following problem :

We are using Merant's Net Express 3.0 to develop some character based
applications that are running on a Windows NT server network. The clients
are both Win 95b and Win 98 PCs. We are using the Windows runtime to run
this applications (runw.exe).

The problem is that when we print from a Windows 95 PC to a Network printer,
we occasionally receive a 9/13 error (File not found) and the application
hungs! There is no rule under which this problem occurs,  which means that
the same PC can issue any number of prints before the problem occurs.

The same problem never appears from a Windows 98 client, or when we use the
character runtime (run.exe) from a Windows 95 application! (but
unfortunately we have to use run.exe to overcome some difficulties with our
local character set).

Has anyone come accross this situation ?

Thanks in advance for your concern.

Regards

Panos Zotos
```
