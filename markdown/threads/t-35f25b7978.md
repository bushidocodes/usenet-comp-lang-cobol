```
Use something like
        02 alt-key.
            03 nth-liv pic 99.
            03 qt-liv pic 99.
            03 qtr-liv pic 99.
with
        alternate record alt-key

Or, if the compiler recognizes split-keys
(Micro Focus, for example)
        alternate record alt-key = nth-liv qt-liv qtr-liv
```
