You need to set some XM switches for the COMPAQ machines.

Try this one first:

SET XM=/P1+JRUU1

See if that works. If not, try just
SET XM=/P1, then /P2, then try /P3 etc, thru /P6

If these do not get you there, post again. There is a method provided
by MicroFocus to test for proper XM settings.
