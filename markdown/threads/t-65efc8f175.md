```
If the "other cobol program" was not created with the same compiler, the
data format could be different because "COMP-3" is "Implementor-defined"
meaning not all compilers will use the same format. (e.g., some might use
BCD, others binary integers).

As far as not creating either BCD or binary integer, I would think if your
:WS-COLUMN var (what you are SELECTing INTO) column were defined as USAGE
DISPLAY that should eliminate the problem.
```
