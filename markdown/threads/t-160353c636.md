Siber Systems is pleased to announce availability of 
ver. 3.2 of CobolTransformer Library, Tools and Converters.

CobolTransformer is a complete toolkit that makes
developing Cobol program analyzers and converters
an easy and satisfying job. 

If you need to automate analysis and/or conversion 
of your legacy Cobol code and you want the automated 
solution to be delivered fast and have high quality, 
then it is time to get a CobolTransformer license.

To get more information, please direct your browser
to http://www.siber.com/sct/

Starting with ver. 3.2 CobolTransformer is not just
one product but a family of products that consists of:

- CobolTransformer Library. This is the base library
on which other tools are built. Licensable to 3rd parties.

- Cobol Tools: General purpose tools that use CobolTransformer
Library: CobolBeautifier, CobolGrep, CobolReporter.

- Cobol Converters: Cobol dialect converters and Y2K tools
that address specific Cobol conversion paths.


Major New features in CobolTransformer 3.2
------------------------------------------

* CobolGrep tool and Query Language (QL) used in it.

We invented CobolGrep and Query Language (QL) so that 
Cobol developers can perform program searches such as
"Show me all the lines of code that contain data item 
declarations that have VALUE clause and belong 
to LINKAGE SECTION or FILE SECTION".

The above search looks as follows in Query Language:
 E.OP==DECL_DD_ENTRY AND 
(E.ISDESCOF(SECTION_WORKING_STORAGE) OR 
 E.ISDESCOF(SECTION_LINKAGE)) AND
 E.ARG(DD_VALUE) != NULL

It means: "Find a node in Cobol Program Tree such that
this node is data item declaration and
this node is a descendant of Working-Storage Section node
or of Linkage Section node and this node has Value Clause node".

QL is a powerful SQL-like language.
Using QL you can write both simple and complex queries
that search your Cobol sources for Cobol constructs that 
you need to find.


* Data Item Length And Offset computation.

Now for every data item we compute:
- its length,
- its offset from 01 level start,
- its offset from the surrounding section start.

This data is necessary for figuring out
data item aliasing information.

* CobolReporter tool.

Reports lengths and offsets of every data item.
Also generates a wealth of other useful information
about your Cobol program.


* Performed usability analysis of command line tools
and reworked the set of available command line options.
Allow multiple files in converters.

* Use Kenonic Controls CrypKey License Manager software 
to enforce CobolTransformer licensing agreements.

* CobolTransformer Library manual is now distributed
for free to prospective licensees.

* Developed formal manual for mf2fsc and ibm2fsc converters.

* Converted all manuals to Adobe PDF format.

* Implemented Micro Focus $IF/$ELSE/$ENDIF/$SET preprocessor 
directives to further enhance MF Cobol parser.

Vadim Maslov
Siber Systems
