I don't know of any.  Have you considered running SoftWindows, the
Windows emulator for the Macintosh?  Which Mac do you have, a 68k or
a PPC?  If you can get the Windows emulator running, you should be
able to instal any Windows-compatible COBOL.  I've got an Amiga here
and I emulate both MS-DOS & Windows 95 with PC- Task 4.4 and
Macintosh OS ver. 8.1 with Shapeshifter 3.10.  Obviously, if you run
something under MS-DOS, as I do with MicroFocus COBOL, it runs fairly
fast, but under Windows, it will run slower.  Lots of big-endian -
little-endian translations with instruction between CPUs, even though
my PC-Task is a dynamic emulator.  I've also got NRCOBOL for the
Amiga.  Since the Amiga is a 68k CPU, maybe its author might be
interested in compiling a version for the 68k Mac if no COBOL
compiler exists.  But, that's a *big* maybe, 'cause there's
differences between Amigas & Macs.

Jay
