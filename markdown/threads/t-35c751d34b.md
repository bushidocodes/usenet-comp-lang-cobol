Easytreve has a utility to generate a definition from your copybook which you
can then run easily against the file.  It can be saved and you can manipulate
that definition into a fancy report eventually. Some shops have this built in
to ISPF (that is optional), you will need to explore sub menus if it is at all
present in your ISPF main menu.

If you have Easytreve, but not installed online, the manuals will probably
reveal a way to do it in batch.
