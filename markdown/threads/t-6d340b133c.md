If you visit our web site, you can find a "freebies" section
which includes a binary of the University of Deleware's
UDSMTP program.

UDSMTP can be used to send batch SMTP traffic to
an SMTP (mail) server.

If you can ALLOC DD's and run programs from your
COBOL environment you should be able to invoke
UDSMTP to accomplish the task.

For more info - see http://www.dignus.com.

    - Dave Rivers -

p.s. There is an alternative, you can call Systems/C
      generated code from COBOL.   If you have Systems/C
      at your site, you could redo UDSMTP to make it
      a callable function and simply CALL it..
