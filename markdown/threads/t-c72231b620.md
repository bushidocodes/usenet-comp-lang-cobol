If you are using the LE COBOL compilers,  define the structure(s) you want to
work with in you Linkage Section, allocate the storage using a call to CEEGTST,
and SET ADDRESS OF your_linkage_section_definition TO the pointer returned by
CEEGTST. 

Something like this:

working-storage section.
01  storage-stuff.
      05  heap-id  pic s9(9) binary value 0.
      05  fc  pic x(12) value low-values.
      05  stor-amount pic s9(9) binary value ------.
      05  stor-loc pointer.

linkage section.
01   my-dyn-storage-area.
       05 ...

procedure division.
.
.
.
          call 'ceegtst' using heap-id,
                                            stor-amount,
                                            stor-loc,
                                            fc
           set address of  my-dyn-storage-area
                                to stor-loc
.
.
.
