Hi Bhavesh,
NetExpress comes with a sample application (called olerep) which gives an 
example of calling MS Excel from NetExpress COBOL using Excel's OLE 
Automation interface.

If you need help on what methods Excel contains, then look in VBX_XL.HLP 
in Office 95 (in the Excel directory on the CD) or VBAXL8.HLP in Office 97 
(in the Office directory, I think).
All the documentation is in VB, which requires some trivial translation.

Micro Focus does not document Microsoft products, but there is full, 
on-line help on using OLE automation included with the product in the 
Distributed Computing book (start with OLE in the index, and follow 
links).

Hope that helps,
Steve.
Micro Focus.
