I have been using the Realia compiler for 8+ years, even before
Computer Associates bought Realia. I also use the Flexus SP2 product
for the Windows portion.

I'm not sure what kind of compiler problems you are having, but
would be happy to help if I can. There is a debugger for debugging
Windows applications on the CA-Realia BBS. Unfortunately, I believe
this is more of a "beta" version of the debugger to be released in
version 5.0 of the compiler. I am a beta tester of the 5.0 compiler
and have found the debugger to be much superior in the 5.0 version.
I do not know when the 5.0 release is scheduled for general use.
However, even the available debugger on the BBS will help you a great
deal solving coding problems.

I have had very good results with the SP2 product. I recently converted
a DOS character-based applications that was 35,000-40,000 lines of
code to use the SP2 product. I also use the beta versions of SP2 and
have been very pleased with the progress and enhancement of the product.
