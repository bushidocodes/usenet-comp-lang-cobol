```
Hello to all,

I am working on a school project where the user inputs the date
as mm/dd/yy along with some number between 1 and 5 inclusively,
let us say 5, and then the program reports the name of the month
that was 5 months ago, in this case 'August' since 5 months
before January is August.

This month subtract n = [name of what month]

At first, I thought I could do this by creating an internal table
that indexed the 12 months by name, and then I could set the
index down by five to find August.

But that idea doesn't work if the current month has a lower index
number than the month to be found.

For example, January would be indexed as '1' and August would be
indexed as '8,' but if I set the index down by 5 from 1, I would
not get 8!

At least I don't think I would!

Also, the teacher did not assign this part specifically. It is
something I am trying to add as extra.

I am using RM Cobol 85, student version.

Thank you for any help on this perplexing problem!

Sincerely,


B.Yates
```

```
I can't think of any particularly "clever" ways of doing it, but it can be achieved
using only 1 extra line of code, remember to define your new month variable as signed.

SUBTRACT 5 FROM WS-MONTH-IN GIVING WS-MONTH-OUT.
IF WS-MONTH-OUT < 1 ADD 12 TO WS-MONTH-OUT.
MOVE WT-MONTH-TABLE(WS-MONTH-OUT) TO WS-MONTH-OUT-TEXT.

The other option is to define the months table with 24 months, and add 7 to the index to
get the new month.
```

```
Take the input number, subtract it from the initial index month and
check to see if the number is less than zero. If it is then add (yes,
add...it is a negative number at this point), the number to 12 giving
you the correct index number. To use your example...

January (Index 1) minus the input of 5 would give you -4. The test
would find it to be less than zero and add the -4 to 12 giving you an
index of 8, thereby pointing to August as required...
```

```
What I suggest you do is to have a signed integer.

For instance:
Index PIC S99.

Then subtract the input number from it.

If the number is less than 1 then add 12 to it and then you have the correct month.

In your situation 1 - 5 = -4

Then add IF INDEX < 1 THEN INDEX = INDEX + 12.

Will give you 8 or August
```
