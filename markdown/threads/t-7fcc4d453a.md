There is a version of the Telon product for the PC (sold by Computer Associates)
This is not a COBOL compiler or anything like that, but a code generator which
uses a database of code fragments.

The PC version of Telon is compatible with MicroFocus COBOL and includes
some interfacing with MicroFocus COBOL and MicroFocus IMS, but the COBOL
compiler is still required.
