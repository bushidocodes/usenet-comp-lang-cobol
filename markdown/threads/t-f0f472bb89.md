Now THAT'S a system I haven't heard of in a while! I programmed for TI
990 under the DX10 operating system for 10 years, although I haven't
even seen a 990 since 1991. I heard that the original COBOL compiler
for TI was written by Ryan McFarland, so if you can find any
documentation on ANSI 74 RM COBOL, then that should be a help. The
biggest difference you'll find is in the ACCEPT/DISPLAY statements.
I'm just now becoming familiar with a non-RM COBOL compiler (Realia,
to be specific) after being out of programming for 4 years, but I
believe many, if not most, other major COBOL compilers have an RM
compatibility mode.

I'll try to dig up some old TI COBOL I saved for posterity :) and see
if I can see anything else that would probably have to change.
