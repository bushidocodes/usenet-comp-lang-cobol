(This tool converts your existing PowerCOBOL Sheets into Windows Forms,
(by default, but other GUI systems and Web Forms are also possible), and
refactors all of your COBOL Scriptlets for the sheet, into a single OO

automatically wired up to the WinForm to become a new code-behind.)

The overall effect is that the new WinForm has all of the functionality
of the old PowerCOBOL sheet, but is now a standard Windows .Net
application and is no longer reliant on PowerCOBOL.

The new release of PC2N is a "no-code" tool and you don't have to write
ANY CODE yourself. It is all just mouse clicks.

There is a video Overview which shows a PowerCOBOL application running
under Windows XP, (of particular interest to the Spanish speakers in
this forum :-)) and then shows the converted result running under
Windows 10 .Net.

There are advantages to using our tools for your Migration, and PC2N can
add value to your applications, as is shown in the video.
