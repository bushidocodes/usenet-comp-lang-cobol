Hi Dave.

By default, the Net Express compiler will search for copy files using either
'.cbl' or '.cpy' extensions. This is determined by the COPYEXT compiler
directive. Search for COPYEXT in the Net Express Help for more information
on this.

I believe that if you were to specify COPYEXT "cbl,cpy,ws" within either the
Project Properties or Build Settings, it *should* resolve your problem.
