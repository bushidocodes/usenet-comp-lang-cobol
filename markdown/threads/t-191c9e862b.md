```
Probably home work, but the following will work (not Standard) with many
compilers

   Display "Hello World".

(Many compilers allow omitting Identification Division, Procedure Division
Header, and Stop Run.   - I don't recommend it, but it does work.)  Only
slightly more complicated and Standard.

Identification Division.
 Program-Id.  ReallySimple.
Data Division.
Working-Storage Section.
01   GetName    Pic X(80).
Procedure Division.
Mainline.
    Display "Enter Your Name: " upon Console with no Advancing
    Accept GetName from Console
    Stop Run
    .
```
