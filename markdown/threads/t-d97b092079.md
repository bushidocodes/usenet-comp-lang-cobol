```
You can use ACCEPT but I've had problems getting it to work. I use the
following method:


CALL "_EXTNAME" USING ENV-VAR-NAME, ENV-VAR-VALUE

where the variables in the using clause are PIC X(n). Fill ENV-VAR-NAME
with the name of the environment variable you want to read. The value
will be in ENV-VAR-VALUE after the call.
```
