* To do a windows callback with NetExpress, simply set a procedure pointer
to the address of the routine windows is to call...In this case ePointer is
set to the entry point "CF001030AS"...

       01  ePointer                    procedure-pointer.

           Set ePointer to entry 'CF001030AS'
           Call WinAPI 'EnumChildWindows'
             using by value AutoScalehDialog
                   by value ePointer
                   by value asPointer
             returning WinBoolResponse.

          .....do whatever is next.....


* Code the entry point that windows will call using the parameters windows
will pass to the procedure...

          Entry 'CF001030AS' WinAPI using by value lshControl
                                                   lsPointer.
          .....do whatever.....

          Exit program returning +1.

There is NO finagling involved...Pretty simple once you do one...Certainly
can be done by even the timid...
