We're using at my businessplace. I have to admit, I like using it, but
have noticed that I have had to do alot of workarounds to make its
limitations go away. By limitations I mean:

1) Forms Workbench to generate after a certain 'amount' of code is
already generated. My workaround to that is to create a file that
contains reusable action code and delete that action code from the
objects that use it and toss in a RUN "falnkbeg.rlz" where the stuff
between the quotes is the copied code.

2) Lots of GPFs and other errors. No workarounds to them, other than
take down Windows. Some errors, though, I can get out of - such as
the "Cannot start host interface" error when I try to compile some
COBOL source from the Project Manager without the COBOL workbench
running. To get out of it, try starting the workbench on its own,
and then go back and compile from the PM.

3) "DLL in use by another application". Happens to me frequently. No
way out, as I see it, just restart Windows. BTW, usually only
happens when the app I made aborts abnormally, or the DLL text window is
closed improperly...

4) CAVR seems kinda like a kludge. Like they adapted Realia to work
with CABLE/Windows, but really haven't perfected too well yet...

5) The Install function doesn't pull over the correct files when I want
to install my package on other machines. Usually I have to add some
other files that it isn't picking up which it definitely SHOULD.

But don't get me wrong, I don't dislike this package. It's expensive,
but it is definitely a BIG step up from what I did before. I
particularly like the CARET reporting engine. Very slick and simple.
I am actually pretty impressed with it, but it does need to grow a little
bit. I hear that CA is making an upgrade to it soon. I'd like to to see
what changes will be made and how much the upgrade will cost when we
already paid so much for the initial package!


I AM writing really nice apps with it. And aside from the problems
I mentioned above and some others, I think that it could be a
real strong product.

Please keep in touch here or through email, I'd like to know your
experiences as well!
