```
Using SQLite3, this example wrapper comes in two flavours.  An API
interface and a clone of the sqlite3> shell command prompt.

Requires sqlite3 and sqlite3-dev packages

ocsqlite.c and sqlcaller.cob for one way of getting to the SQLite API

ocshell.c and sqlscreen.cob for the shell capture and a simple screen
display of a
.mode column; .width n n n ; select * from ; command string.

http://www.opencobol.org/modules/newbb/viewtopic.php?topic_id=378&forum=1#forumpost2182

http://add1tocobol.com/tiki-list_file_gallery.php?galleryId=12 for
less chat more source.

Treat these sources as alpha.  Names will change to preferred
OpenCOBOL standards soon.

Cheers,
Brian
```
