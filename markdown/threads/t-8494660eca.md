This IS a string with lenght 3 bytes. The sign however would be stored
in rightmost or units place zone. If you are using it purely for
Display purpose why don't you try SIGN IS LEADING/TRILING SEPARATE
clause? In that case the string will be of 4 characters with sign
appearing before or after the digits depending on the LEADING or
TRAILING cluase used.
