> What 'p' mean. Plus ? or is it an error in the record ?

The "S" in that picture means one byte in the field will contain both
the value of that digit as well as the sign of the field.  In this case
the last four bits of the letter "p" indicate the value of that digit, a
zero so the value of your field is 151.15.  Not knowing your system, it
is impossible to know if that value is positive or negative.  If you
have a "p" in every occurance of that field you can probably ignore it.
If you find something else there do your best to track that value
down--that would help answer a lot of your questions.  HTH

If the PICTURE specifies "SIGN IS LEADING" the P, having code x'D7' would
indicate that the value is -700151.15. x'C0' and A-I and indicate +0 to +9 and
