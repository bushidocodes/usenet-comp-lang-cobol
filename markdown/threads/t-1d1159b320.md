Have a look at the sample project in netexpress\base\demo\guisamp. In the
mainwin.cbl you'll find a method called 'PutBitmapInClientArea' which loads
a bitmap (in this case the background for the SDI frame) from a resource
(.rc) file, using the BitmapData class. If you replace the 'fromResource'
method by the methods 'fromFile' or 'fromFileZ', that should do it. Of
course you'll need to be in GUI mode, so you might want to use the guisamp
project as the basis for your work; take trigger.cbl, mainwin.cbl, and one
of the gui control demo programs, textedit.cbl would be the obvious choice,
and you're away. Have fun.
