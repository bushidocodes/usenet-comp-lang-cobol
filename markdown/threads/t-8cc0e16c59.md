Erwin,

You did not indicate which COBOL vendor you are using, but in RM/COBOL a
procedure error 204
indicates that the runtime can't find the COBOL object file on the Runtime
command or on
a CALL statement. Make sure that the object is in your RUNPATH. It could
be that the object is corrupt.
Or is could be that you are trying to execute a program witch was compiled
to an object version level higher
than your runtime allows.
