This number is not packed, but has what is called an overpunched sign.
The trailing position indicates both sign and value, as follows:

  digit   +   -
    0     {   }
    1     A   J
    2     B   K
    3     C   L
    4     D   M
    5     E   N
    6     F   O
    7     G   P
    8     H   Q
    9     I   R
    
The value of that field is -43050
