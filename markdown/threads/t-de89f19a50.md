Most IBM mainframe (MVS) to PC download uses IND$FILE. This supports
automatical EBCDIC to ASCII conversion. This won't help you if the data
structure contains packed decimal or binary numbers.

Best bet is to write a small COBOL program which reads the file as is and
writes it out in pure display. Then an EBCDIC -> ASCII conversion won't
hurt you.
