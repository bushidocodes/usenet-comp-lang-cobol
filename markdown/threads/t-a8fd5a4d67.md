```
The '74 Standard was done on a typewriter (as I recall) - it is certainly NOT
available "on-line" (and I sure hope has NOTHING to do with anything that is
being created now).

The '85 Standard was done in NROFF (with Amendments in various word
processing software). I do not know of any where it is generally available
"on-line" - but you could try contacting the J4 chair to see if it is. (I
think there is a CD "floating around" with some variation of it).

There is a test suite from NIST for determining if a compiler is "conforming"
or not.  Again, the J4 chair could (probably) point you to how to contact
them.  They have gone out of the business of validating ANY programming
languages, so I don't know what has happened to the test suite.

The J4 chair (who is out of town this week), can normally be reached at:
   doncobol <at> mediaone.net
```
