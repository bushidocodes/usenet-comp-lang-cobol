Two approaches may be taken. One reduce the number of open files in your
application. If opening an indexed file means opening a data file and an
index file, that counts as two files.

The other approach is to reconfigure your UNIX kernel for the number of files
open per process. Another may be the number of files open per user. Please,
do not share logins under UNIX. The per user limits accumulate per user, not
login session.

The unfortunare reality is that some UNIXs do not allow you to change the per
process file limit. Another note is you start out with 3 files open: Standard
In, Standard Out, Standard Error, and often a 4'th, your program, before you
open any files.
