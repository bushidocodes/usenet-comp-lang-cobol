What do you mean by 'compile 2 source files together' ? Each
program should be separately compiled and marked as 'main' or
not as appropriate. They must have different names and the
name must be the PROGRAM-ID.

The called programs will be DLLs. They will also produce a
.LIB, this must be linked into the main program. The manual
(on CD) covers this.

Fujistsu uses a DLL model for dynamic calls on Windows, unlike
MF it needs to know of all these DLLs when linking the main
routine.
