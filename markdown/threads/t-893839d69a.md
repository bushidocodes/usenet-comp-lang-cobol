I don't believe you can do this.  In IBM mainframe COBOL, each issuance
of a DISPLAY advances a line, and the only way to get 2 (or more) on a
line is to
DISPLAY VARIABLE-1 '         ' VARIABLE-2,  and I don't think this is
what you are looking for.  DISPLAY was not intended to produce pretty
pages.

An alternative might be to store the items to be displayed in a table,
and then output the table at the end-of-job. But this is a lot of work,
just for debugging output.

Bob
