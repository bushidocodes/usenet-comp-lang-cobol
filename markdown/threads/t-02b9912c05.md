```
       IDENTIFICATION DIVISION.
       PROGRAM-ID.  GETRECT.
       ENVIRONMENT DIVISION.
       CONFIGURATION SECTION.
       Special-Names.
           call-convention 66 is APIentry .

       DATA DIVISION.
       WORKING-STORAGE SECTION.
       01  hWnd        pic  9(9) comp-5.
       01  bResult     pic  9(9) comp-5.
       01  Rect.
         05  R-left    pic s9(9) comp-5.
         05  R-top     pic s9(9) comp-5.
         05  R-right   pic s9(9) comp-5.
         05  R-bottom  pic s9(9) comp-5.

       PROCEDURE DIVISION.
       MAIN SECTION.

           Call APIentry "GetDesktopWindow"
               returning        hWnd
           End-Call
           Call APIentry "GetWindowRect" using
               by value         hWnd
               by reference     Rect
               returning        bResult
           End-Call
           If bResult = 0
               Call APIentry "GetLastError"
                   returning       bResult
               End-Call
               Display "GetWindowRect returned (" bResult ")"
           Else
               Display "left  (" R-left ")"
               Display "top   (" R-top ")"
               Display "right (" R-right ")"
               Display "bottom(" R-bottom ")"
           End-If
           STOP RUN.
```
