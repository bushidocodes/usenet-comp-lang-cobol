Can anyone give me any ideas of suitable Report Writing Tools which will
integrate into a COBOL2 / OS390(MVS) environment (i.e not PC based in
any way)

The tool should be able to generate printed reports (with titling, page
numbering, grouping, subtotaling etc), as well as 'data' reports (for
example Comma Separated Value datasets or just a plain old sequential
file). Printing can be simple fixed-font, or more sophisticated layout
(i.e AFP or Postscript datastreams) using variable font sizes, bold,
italic, form overlays etc.

Input could be either a list of sequential records, or a DBMS (e.g DB2),
so SQL access would be needed.

A *theoretical* application would be.

Input: list of employees records detailing department, country, salary

Output1: Printed Report suitably titled showing total salary, total
number of employees by department/country.
Output2: Same set of data, but in CSV (or whatever) for input to
spreadsheet, database etc.

The only things I can think of are either CA-Easytrieve, or bespoke
COBOL programs. Any other ideas ?
