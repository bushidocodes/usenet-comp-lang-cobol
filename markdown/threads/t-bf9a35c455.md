MOVE 42 TO Data-Name-1

MOVE 9 TO Data-Name-1
WRITE Any-Record (writes record 9 assuming it
didn't exist previously)

Relative files are very good where you can guarantee a unique
number for each record and be able to find it by this. I use
them for menu files and for work files where the record number
is held in a working-storage array. Access the array element
to get the relative number, read record then all the data is
available.
