You can "talk" to your phone via a serial comm port IF the phone has an
internal modem. (Most do...).  (You might be able to do this even if it
hasn't; I haven't tried it.) You can also "talk" to your phone via infrared
link (I use this all the time). A modern computer will detect the phone's
modem using Plug 'n Play, once you connect it.

If you check the Motorola and Nokia Web sites you can probably download
software to manage the connection. (I did this for my Ericsson. They provide
a connection to an Outlook address book that automatically lets you maintain
phone numbers, addresses, and e-mail addresses in sync with your computer.
You can also extend this interface to use other fields if you want. While
connected, it is possible to send SMS messages using the computer keyboard,
also.)

If you really want to do it all yourself, the first step is to establish a
connection (see above) then "listen" to the phone. I don't think there is
any special protocol when connected over a serial comm port, but the
manufacturer's web sites should tell you, or you could post there.

Pete.
