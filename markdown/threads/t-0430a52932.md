I don't know if I'm the last person in the world to find out about this 
or not, but in case not, I thought I'd share.  The website below has 
"standalone" versions of various releases of Internet Explorer.  Using 
them, you can have IE 7 installed, but see how a website renders with IE 
6.  I imagine they would run in wine as well, as the necessary DLL's 
come with them - but, I don't have wine set up on this machine, so I 
can't test that at the current time.

I put a new theme on my blog, and it didn't render correctly in IE 6 - I 
downloaded the IE 6 version, and I was able to see the problem and come 
up with a hack... er, solution for it - so, I know it's true to form.  I 
did have trouble getting a cookie-dependent feature to work, but I also 
didn't try very hard.

I know many of you do web development - maybe this will help make things 
easier.  (Of course, I know many of you have also said "Screw IE 6!" 
heh...  I thought about it, but I checked my logs, and I've got about 
18% of my hits coming from IE 6.  Oh well...)

http://browsers.evolt.org/?ie/32bit/standalone
