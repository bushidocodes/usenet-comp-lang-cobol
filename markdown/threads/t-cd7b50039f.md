The current version (7.0) of the RM/COBOL User's Guide is available in PDF
format on the Liant web site here:

http://www.liant.com/docs/

This is 32 bit Windows documentation, but much of the information is still
applicable.

As for why the your DOS box is closing, I'm not really sure what's going on.
How are you invoking the compiler?  Are you using Windows' Explorer and simply
clicking on the file rmcobol.exe, or are you typing rmcobol.exe from a already
open DOS box (recommended)?

Certainly try using the "l" option to cause the compiler to create a listing
file.  This would at least give you the ability to capture the error(s) or
message(s) for viewing.  You could also contact Liant at 1-800-RM-COBOL - I'm
sure sales would be able to help you purchase a manual or upgrade.

Hope this helps,
-Michael
