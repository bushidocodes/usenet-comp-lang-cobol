This is done not in the JCL, but in the program itself.  You simply set up a
parameter list (consisting of DSname, disposition, etc.), then call the
DYNALLOC macro (SVC 99), pointing to this parameter list.   You  can then
change the disposition in the list, point to it again, and de-allocate the
dataset.  Cobol programs can call a user subroutine sort of like this:
       call SUBR using (dsname, disposition).   The subroutine is very
simple to write.
