### FileShare: Use it for all files?

- **From:** "greg coles" <ua-author-17071942@usenetarchives.gap>
- **Date:** 1997-10-19T20:00:01+00:00
- **Newsgroups:** comp.lang.cobol
- **Message-ID:** `<01bcdd87$a8d0a220$8464a8c0@gcoles>`

```

We are using Microfocus 16-bit COBOL with FileShare on a Novell Netware
4.xx network. Currently, we only use FileShare to handle a half-dozen or
so of our busiest files. My question: would we be better off to run all
files thru FileShare? Every once in a while the system slows down to a
crawl. I've tried to determine if it's people running I-O instensive
programs, but that doesn't appear to be it. I got to thinking that maybe
FileShare gets to the point where it fights with Netware for cache and that
is the reason for the slowdown. (Maybe it's time to invest in a good
network analyzer, huh?)

Greg
```
