Contact Fujitsu through their web page (www.adtools.com) for a definitive 
statement.

As far as I know, they have not stated that PowerCOBOL is being dropped, but 
it won't run under .NET as winForms would be required.
NetCOBOL for .NET therefore does not include PowerCOBOL.

Versions of NetCOBOL up to 7 still come with PowerCOBOL and it runs OK under 
Windows XP. I have run PowerCOBOL apps on a 64 bit machine, but not under a 
64 bit OS.

There are many PowerCOBOL users concerned about the future of it.

I am already looking at converting some PowerCOBOL applications to C# for 
use with .NET. If enough people show interest, I may develop a general 
toolset to do this.

Pete.
