Alain,

from a quick check of the Postgres docs, it looks like using their
ODBC interface is your best bet ; see the Postgres FAQ at
http://www.postgresql.org/docs/faqs/FAQ.html#2.1 (also section
2.4).

If you're running either Net Express or Server Express, you can use 
OpenESQL (EXEC SQL syntax) to communicate to the RDBMS via ODBC. See the 
Database Access manual for more details.
