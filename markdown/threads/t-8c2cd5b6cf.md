Later editions of the book have this problem fixed.  It is NOT a 
Fujitsu problem - but rather a problem with the way Sams put the CD 
together.  When developing TY COBOL in 24 Hours I was aware of this 
problem and I took great pains to ensure that the CD was a reliable 
copy and install of the Fujitsu compiler.  You CAN order a CD copy for
a price from the Fujitsu web site (http://www.adtools.com) - or if you
want, you can buy my book ($20.00 or so from the online book sellers).
 Or you might be able to call MacMillan (Sams) and have them send you 
a corrected CD.
