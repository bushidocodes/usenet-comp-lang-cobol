I kept hoping that someone else would help you with this - but as no one
has, let me show you a small example program - and see if this helps. The
program reads the value in one environment variable and then depending on
what value it has, it sets another environment variable. If you have any
problem with the program logic (which is pretty useless), let me know.

$set MF quote noflag
Identification Division.
Program-Id. CobEnv.
Data Division.
Working-Storage Section.
01 Env-Values.
05 Env-DoW Pic X(10).
88 Week-Day Value
"MONDAY" "TUESDA" "WEDNESDAY" "THURSDAY" "FRIDAY".
88 Week-End Value
"SATURDAY" "SUNDAY".
05 Env-Type Pic X(10).
Procedure Division.
Mainline.
Display "whatday" upon environment-name
Accept Env-DoW from environment-value
If Week-Day
Move "WeekDay" to Env-Type
Else
If Week-End
Move "WeekEnd" to Env-Type
Else
Move "unknown" to Env-Type
End-If
End-If
Display "whattype" upon environment-name
Display Env-Type upon environment-value
Display Env-Type upon console
Goback.

--------

NOTE: I know that this is pretty awful syntax - but don't blame Micro Focus.
This is the syntax that was originally developed by X/OPEN. You can use
this same syntax on any platform with any compiler that claims to be X/Open
conforming.
