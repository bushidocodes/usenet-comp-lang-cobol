```
Hi,

I have a small query regarding the calling of common procedures
from event procedures in Fujitsu PowerCOBOL (v4).

I wish to be able to call a common procedure (in this case to update the
fields on a form) from more than one event procedure (such as the
clicking on several different navigation buttons).  It seems that each
event procedure is a separate entity unto itself, and seems unable to
share any common code -- apart from cutting and pasting the required
procedure code into each and every event procedure... a very wasteful
and inefficient method.

Is there any way to have a set of common procedures, which are specific
to a certain form, that can be utilised from any of the form's other
procedures - including event procedures?  They don't need to be
available outside the form.

Thanks.

Mark Hallett
mark@bizwerkz.com
```
