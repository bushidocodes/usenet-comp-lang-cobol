The library routine you're interested in is W$GETC. It's in Appendix I.
Briefly summarized, you call W$GETC USING a PIC X(2) data item. On return it
will contain the character entered, a two digit abreviation for special keys
(e.g. under VT100 pressing F1 returns "k1"), or "-1" if an EOF occurs.

Here's a very quick-and-dirty example:
PROGRAM-ID. WGETC.
DATA DIVISION.
WORKING-STORAGE SECTION.
01 Got-Char PIC X(2).
PROCEDURE DIVISION.
Main-Logic.
INITIALIZE Got-Char.

PERFORM 100 TIMES
CALL "W$GETC" USING Got-Char
DISPLAY Got-Char
END-PERFORM

GOBACK 0.
