Changing the printer within a COBOL program under UNIX can
be accomplished like this:

1. In the program's SELECT statement, make the printer name a
variable:

SELECT PRINT-FILE
ASSIGN TO PRINTER PR-NAME.

During the execution of the program, you can move
different values to PR-NAME before OPENing the print file.
The different values could be something like PRINTER1,
PRINTER2, PRINTER3, etc.

2. Add DEFINE-DEVICE statements to the runtime configuration
file for each printer definition:

DEFINE-DEVICE device=PRINTER1 path="lp -s -dLPDEST1" pipe=yes
DEFINE-DEVICE device=PRINTER2 path="lp -s -dLPDEST2" pipe=yes
DEFINE-DEVICE device=PRINTER3 path="lp -s -dLPDEST3" pipe=yes

This example assumes that LPDESTn is a valid UNIX device name.
