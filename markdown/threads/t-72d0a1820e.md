There is a set of tools provided by Siber Systems
that would allow you to do it.

Data2Flat/Data2Dbf convert MicroFocus data files
to text (CSV) or DBF (DBase) format.

DataGuess guesses the record layout of the file
if you do not have source for Cobol program
that can read or write the data file.

These tools are not free of course (but I guess
your Cobol files were not written by GNU either), and 
as I heard, free tools for this do not exist, not 
for MF index files that are compressed.

More details here http://www.siber.com/sct/datafile/
