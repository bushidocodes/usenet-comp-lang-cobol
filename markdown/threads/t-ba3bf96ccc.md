"Data Management System (DMS)
The Unisys database management software product that conforms to the
CODASYL
(network) data model and enables data definition, manipulation, and
maintenance in mass
storage database files."

The "network" means that the set relationships allow to be defined in
a "network" like structure and not hierarchical.

As far as my memory goes, IDMS and DMS have the same roots.
