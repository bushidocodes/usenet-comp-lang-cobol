```
Here is some feedback I got from MF which may be of interest to those of
you considering buying Personal Cobol.

›
› 1. Is personal COBOL fully compatible with Windows 95?

Yes, because Windows 95 will run programs designed for Windows 3.1 or
for DOS.
On a small percentage of Windows 95 installations (probably less than
5%), a bug in Windows 95 prevents Personal COBOL for Windows ('PCW')
running. Personal COBOL for DOS ('PCD') is not affected by the bug. If
you buy Personal COBOL from us it is covered by a money-back guarantee,
so if PCW will not run for you we either refund your money or exchange
it for PCD at your choice.
› 
› 2.   Does it have a CICS interface so I can test stand alone CICS screens >on my PC?
 
› No.  CICS is supported only by the Micro Focus COBOL Workbench.
› 
› 3.   Does it have a file formatter/editor like mainframe Fileaid?.
 
› No
› 
› 4.   Does it have an interface for the Windows 95 version of DB2.

No
```
