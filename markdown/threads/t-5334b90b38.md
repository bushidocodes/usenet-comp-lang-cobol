You must in Postscript place the text and print it with a command.
I use the following definition (Header) for Postscript, that is to
send
to the printer before it beginns to print:
%!PS-Adobe-2.0
%%Creator: (c) AS-DATA

/mainvm save def
(\032) cvn {} def
/bd {bind def} bind def
/ff {/vpitch exch def findfont vpitch scalefont setfont} bd % 'ff'
is find-font
/x  {currentpoint exch pop moveto} bd           
% 'x' is absolute x positioning
/n  {currentpoint exch pop 0.0 exch vpitch sub moveto} bd
% 'n' is newline
/v  {/vpitch exch def} bd                       
% 'v' is vertical line height
/p  {show} bd                                   
% 'p' is print
/rp {exch 1 index stringwidth pop sub x p} bd       
% 'rp' right just print
/newcolor {pop 6 setlinewidth} bd                        
%neue Farbe setzen
% Print Dieresis ÖÄÜöäü
/die {1 index stringwidth pop (\310) stringwidth pop sub 2 div
%Hälfte diff Die/Letter
      currentpoint exch 3 -1 roll add
exch               
%zentrieren
                       
3 -1 roll {vpitch 5 div add} if   %versatz Doppelpunkt 1/5 nach
oben
      3 -1 roll p currentpoint                           
%drucken Letter
      4 -2 roll mt (\310) p mt} bd                       
%Doppelpunkt über Buchstaben
% overprint a Character
/ovp {2 index 2 index stringwidth pop stringwidth pop
      sub abs 2 div                                      
%Hälfte diff Zch/Letter
      currentpoint exch 3 -1 roll add
exch               
%zentrieren
                       
3 -1 roll {vpitch 5 div add} if   %versatz Zch. 1/5 nach oben
      4 -1 roll p currentpoint                           
%drucken Letter
      4 -2 roll mt 3 -1 roll p mt} bd                    
%Zch über Buchstaben
/pp {showpage} bd                                        
%print page
/mt {moveto} bd
/lt {lineto} bd
/np {newpath} bd
/li {mt lt}  bd
/str 45 string def
/#C {userdict begin /#copies exch def end}bd             
%Anzahl Kopien
/sym {currentfont /Symbol vpitch ff exch p setfont} bd   
%Zeichen in Symbols
/logo {} bd
%%EndProlog
%% END HEADER file
