CA-Realia's code is pretty much an "emulation" of IBM mainframe code.
Fujtisu provides "full" support for all IBM dialects *thru* VS COBOL II.
Therefore, I don't think that you should have much difficulty with the COBOL
source conversion (possibly NONE).

As far as ScreenIO vs sp2 - I think that Bob Wolfe probably can help you
with what tools are available for that - or how much effort it would take.
