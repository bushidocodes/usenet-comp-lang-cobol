You cannot connect the CICS Client directly to a mainframe system via
tcp/ip. You can only use SNA (APPC) or TCP62 which is SNA over TCP/IP
and requires Anynet support on the mainframe side.

For configuration information use the helpful guides downloadable from:
http://www-4.ibm.com/software/ts/cics/library/manuals/ctg31dl.html

(See the 2nd section, configuration PDF's).
