Two reasons:

1) It is read my SIGNIFICANTLY more readers (and is more active).  This is a
"re-enforcing" situation.  You are more likely to find someone posting (or
who has posted) on any specific topic in C.L.C. And because there are more
people posting and responding in C.L.C., it gets even more readers/posters.

2) alt.cobol is *not* "carried" by all ISPs.  This is because you have to go
thru a "voting" process to get a NG with "comp" as the first part.  As part
of that vote (that took place many years ago), there was supposed to be an
end to "alt.cobol".  This has happened with some servers - but not all.

   ***
Certainly ANYONE who wants to keep using alt.cobol (and has an ISP that
supports it) may continue to do so.  However, I think you find a more limited
audience - and a more limited number of threads of potential interest.  If
you use something like www.deja.com and compare the number of posts and
readers in the two groups, I think you will find this to be a true
representation of current usage.
