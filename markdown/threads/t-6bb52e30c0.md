The first part of the file is probably generated because the file is
stated as an "IDX FORMAT 4" or similar in the select statement. If the
files are from Win'95 (DOS) they also are a bit weird and you need to
add the following after the FD 

FD FILE-NAME
     RECORDING MODE IS V.

The V stands for variable and you should be able to read them as a
sequential file thereafter. Alternatively cut out the first part of
the file using an editor that doesn't expand or contract any embedded
control characters (ie a specially written C program !)
