```
'Attributes' are hardware dependant and will vary when used with
different systems. I don't think that many will still be using MDA or
Hercules cards on an IBM-PC these days, or even CGA, so you are more
likely to want to know the VGA text attributes.

However attributes also apply to Unix consoles and terminals (and
others) so MF has a 'Generic' mechanism described in the "Programmer's
Guide to Creating User Interfaces" in the set that I have.  YMMV with
NE.
```
