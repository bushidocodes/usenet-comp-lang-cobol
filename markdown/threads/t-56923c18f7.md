In one of my papers I mention that with the advent of LE and USS,
you can call virtually any C or UNIX service directly from COBOL
(as well as from LE-enabled Assembler, PL/I, etc.).

One member of the list asked if he could call regcomp and regexec
from a COBOL program. So I took up the challenge.

It was a challenge, since the parameters and processing are not
clearly documented, but I was able to accomplish the task in at
least a minimal way.

Thought some of you might want the challenge of fixing this up,
to make it more general or proper. It's, perhaps, a good example
of my point, though. So enjoy or ignore, as you choose.


       process test(sym,none) numproc(pfd) opt rent
       process flag(w,w) map xref(short)
      *  Copyright (C) 2003 by Steven H. Comstock          Ver4
       Identification division.
       program-id.  'COBREG'.

       environment division.
       data division.
       working-storage section.
       01  res         pic s9(9)   binary  value 0.
       01  dest        pic s9(9)   binary  value 2.
       01  work-bin    pic s9(9)   binary  value 0.
       01  n           pic s9(9)   binary  value 0.
       01  fc          pic x(12)   value   low-values.
       01  dtitle      pic x(80)   value   'Snap shot'.
       01  dopts       pic x(255)
            value 'regst(0) nocond notrace page(0)'.
       01              pic x(4)    value   'Ver4'.
      *
       01  Pattern            pic x(20) value z'in*'.
      *
       01  In-string          pic x(80)
                value z'In the beginning...'.
      *
       01  Match-msg.
           02                 pic  s9(4) binary value +81.
           02                 pic  x(09) value 'Match at '.
           02  match-start    pic  zz9.
           02                 pic  x(09)  value ', found: '.
           02  match-val      pic  x(60)  value spaces.
      *
       01  Inmess.
           02                 pic  s9(4) binary value +10.
           02  inmess-string  pic  x(10) value
                  'In COBREG'.
       01  Outmess.
           02                 pic  s9(4) binary value +15.
           02  outmess-string pic  x(15) value
                  'Leaving COBREG'.
      *
       01  regex-t.
           02  re-nsub        pic  s9(9) binary.
           02  re-comp        pointer.
           02  re-cflags      pic  s9(9) binary.
           02  re-erroff      pic  s9(9) binary.
           02  re-len         pic  s9(9) binary.
           02  re-min-col-val pic  s9(4) binary.
           02  re-max-col-val pic  s9(4) binary.
           02  re-cols occurs 10 times.
               03  re-lsub    pointer.
               03  re-esub    pointer.
           02  re-map         pic  x(256).
           02  re-shift       pic  s9(4) binary.
           02  re-dbcs        pic  s9(4) binary.
      *
       01  reslts.
           02   starts occurs 20 times.
                03  start-ptr pointer.
                03            pic s9(9) binary.
      *
       01  binary-reslts redefines reslts.
           02   startx occurs 20 times.
                03  start-off pic s9(9) binary.
                03            pic s9(9) binary.
      *
       procedure division.
       thecode.
             call 'CEEMOUT' using Inmess, dest, fc

             call 'regcomp' using regex-t,
                                  pattern,
                                by value 0
                   returning res

             if res not = 0
             then display 'regcomp failed'

             else

             call 'regexec' using regex-t,
                                  In-string,
                                by value 9,
                            by reference reslts,
                                by value 0
                   returning res

                   if res = 0
                   then
                     display 'match found'

                     perform varying n from 1 by 1
                        until start-off(n) = -1

                        move start-off(n) to work-bin
                        add 1 to work-bin
                        move work-bin to match-start
                        unstring In-string(work-bin:)
                             delimited by x'00'
                             into match-val
                        call 'CEEMOUT' using Match-msg, dest, fc

                     end-perform

                     call 'CEE3DMP' using dtitle, dopts, fc

                   else
                     display 'no match found'
                   end-if

             end-if

             call 'CEEMOUT' using Outmess, dest, fc

           goback.
