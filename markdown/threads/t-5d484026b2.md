I was able to install this software on Win XP SP2 without difficulty
and without using any compatibility mode.

The package I used was COBOLV3.EXE obtained from Fujitsu's web site.

There was no request to reboot. After installation the programs reside
in c:\FSC\PCOBOL32. Installation creates an entry on the start menu
"Fujitsu COBOL 3.0". Try "Programming Staff".

This version does contain its own IDE, such as it is. You may find it
tolerable in time as I did eventually. One key point is that you have
to declare one program as the MAIN program. Google in this newsgroup
for more explicit directions.

Otherwise see:

http://www.adtools.com/student/index.htm

and

http://www.adtools.com/student/2000compatibility_tips.htm

Note that I have not used the software you eventually find by following
this link to Microsoft's web site. So beware!
