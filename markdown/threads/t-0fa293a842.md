The word "function" for BOTH an intrinsic and a user-defined function is 
OPTIONAL in the '02 Standard (and I would guess in MF) *if* the proper 
"repository paragraph" option is used.

e.g.

Compute Numfield = Function NumVal (ABC)
    is the same as
Compute NumField = NumVal (ABC)

If

  Repository.
      Function ALL Intrinsic.

appears in the Environment Division / Configuration Section.

Similarly,

 Compute XYZ = Function User-Defined-with-Numeric-Returned (ABC XYZ)

is the same as

  Compute XYZ = Function User-Defined-with-Numeric-Returned (ABC XYZ)

if you code

 Repository.
    Function User-Defined-with-Numeric-Returned.

*and* you created a "function prototype" for User-Defined-with-Numeric-Returned

   ***

Does this answer your question Jimmy?
