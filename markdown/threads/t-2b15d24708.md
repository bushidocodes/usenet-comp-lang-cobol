```
This is probably Windows-related, rather than a Fujitsu issue.

Microsoft Knowledge Base Article - 296742
2351 and 2355 Internal Errors When Installing Software from CD-ROM
http://support.microsoft.com/default.aspx?scid=KB;en-us;q296742

While not specific to your problem, this Office 2000 KB article
outlines multiple workarounds, any of which may grant you relief,
such as copying the files to the hard drive or turning off your virus scan.

Microsoft Knowledge Base Article - 236437
OFF2000: Error Message: "Internal Error 2351" or "Internal Error 2355" During Setup
http://support.microsoft.com/default.aspx?scid=kb;EN-US;236437

The most likely scenario IMO is that the error is due to a bug in the Windows Installer Setup engine
when the Setup engine extracts files from the compressed cabinet files (CAB) on the CD-ROM. 

To help resolve this, you should, if appropriate, install the latest Windows OS service pack
(For example, Microsoft included a fix in Windows 2000 SP2). 

As a workaround, you can copy the files from the CD-ROM to your hard drive,
and/or boot into Safe Mode and install from there.
```
