Hi,

I am unable to compile Microfocus cobol/cics programs. The CICS translator is
introducing a special name 'litlink' in between CICS calls.

For Example, EXEC CICS ASKTIME ABSTIME(WS-TIME) END-EXEC statement will be
translated to CALL litlink 'CICSAPI' USING CICS-ARGS. Where litlink is the
special name introduced by the cics translator which will inturn give the
compilation errors.

Actually it should be translated to just CALL 'CICSAPI' USING CICS-ARGS.

Can anybody please help me in compiling these programs. Thanks in advance.

Thanks,
