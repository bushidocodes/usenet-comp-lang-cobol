Hi Jags,  (I would never say that in a plane  SCNR;)

I think you should try the INTERCEPT Command from within the main program.
You will find syntax and 'what-it-does' via RTFM, so here just a few words:

Type "INT <sub-program-name>" when yor Xpediter session starts or put the
command in your script file. If everything is okay (source and DDIO of
main program and sub-programs were found, you will see the 'source jump'
after the sub-program call.

Hint: Also put 'stop-before' mark on the main program line that calls
the sub-program(s)

By using INT two stop tags are set into the intercepted sub-program 
automatically. One at the PROCEDURE DIVISION USING... statement, the
other at GOBACK or RETURN.
