If you made the telnet or ssh server use port 80 it will compete with
the httpd server - these services all have different ports for a good
reason.  You will have to convince the firewall maintainer to open ssh
port to the machine with your system.

> Can any one give me detail procedure to use cgi-bin etc with some
> examples.

No, it is rather too much of a subject to explain in a single message.
 It would require a complete redesign and redevelopment to work as
cgi.  If you wanted to do this there are plenty of books and tutorials
which will explain how CGI works, though they will be for C, Perl or
Python.  Try and get one that is for C only, it will be the most
relevant.

The major problem is that the CGI programs are stateless so you would
have to create some state information for each user.
