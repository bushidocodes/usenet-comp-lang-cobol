That's because the screen and form designing utilities are for creating
character mode applications. These can be run in either native DOS, DOS
with our memory manager or using Windows itself as the memory manager.

To create GUI applications, you use either Dialog System or Dialog System
Professional depending on the type of application you want or the money
your willing to pay.

once hence the differentiation. You have the choice rather than us always
overiding the current program your debugging. The run button will be
disabled if your not in "Edit+Execute" mode.
