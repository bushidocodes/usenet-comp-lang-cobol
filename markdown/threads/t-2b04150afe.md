I would suggest you take a look at Acucobol (www.acucorp.com) When I did the move a number of years ago it was right out of the box. You do not get new source though, the Acucobol runtime allow you to run your applications as is and then you can migrate the user interface step by step as your old ui are allowed to coexist with your new one. You are even able to distribute one single code as the runtime will distinguish this where it is necessary.

Acucobol has in my opinion a very good implementation of graphical user interface. The syntax is very similar to what you are used to.

What cobol are you currently using?
