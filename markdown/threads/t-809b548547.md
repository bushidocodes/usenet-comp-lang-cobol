01 TEST-INITIAL-VALUE PIC X(8) VALUE "1234ABCD".
01 FILLER REDEFINES TEST-INITIAL-VALUE.
03 TEST1 PIC X(4) OCCURS 2.

DISPLAY ' 1ST TEST1 VALUE = ' TEST1(1).
DISPLAY ' 2ND TEST2 VALUE = ' TEST1(2).
