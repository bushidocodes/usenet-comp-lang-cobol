Frik, I will assume for now that you want to display the bitmap in a window
(rather than on a printer). If so, you should either look at the demos for
writing Windows or OS/2 API programs or the documentation on Panels2 in the
product. Either will allow you to do this, as the COBOL language itself has
no support for displaying bitmaps.

If you wanted to print the bitmap there are PC_WIN_ functions available to
do so. However, they are Windows only. As you didn't mention which
operating system you are running I don't know whether that would be
suitable.

I hope this helps.
