```


Hello, we are using Micro Focus COBOL on our campus network and
were wondering how to turn compressed printing on when printing
to a laser printer. We know how to turn on compressed printing
when printing to a DOT Matrix printer, but that does not work
when I try to print to a HP LaserJet 4.

Example: (works for DOT Matrix Printer)
01 PRINT-LINE PIC X(120).
01 PRINT-LINE2.
05 FIELD-27 PIC 99 USAGE BINARY.
05 FIELD-15 PIC 99 USAGE BINARY.

MOVE 27 TO FIELD-27.
MOVE 15 TO FIELD-15.
WRITE PRINT-LINE2.

I know the problem is not with the COBOL code, but that each printer
can use a different set of escape sequences to turn on its individual
functions. However I can not seem to find the manuals for our laser
printers to see what escape code sequences are used to set it into
compressed print (the numbers to MOVE into FIELD-27 and FIELD-15).

Could someone tell us the numbers we have to MOVE in order to get
compressed printing on/off on a HP 4MV LaserJet and a
HP 4Si/Mx + (plus) LaserJet?

PS. Please respond back via email if possible.

Thanks in advance,

Peter Zuge
Student at Univ. of Wisconsin - Stevens Point
```
