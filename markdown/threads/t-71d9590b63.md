There are only two rational possibilities:
1. The required DLL does not exist anywhere on your system, or
2. If it does exist, the program cannot find it.

Can YOU find F3BIPRCT.DLL?

If you cannot find the DLL ("Find" command), then the error is
probably in the installation process. That is, something failed during
the installation of the compiler.

If you can find the DLL by manual searching, then the error probably
lies in the directions you gave the compiler as to the DLL's location
(the PATH statement is a good place to start).
