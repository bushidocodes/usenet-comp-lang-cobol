As far as program code their isn't any function which will do this: hint:
you simply have to accept the time then test the second's variable for
change, if so then update the static text if GUI is used.

You will have to register this method as idle, in order for the rest of the
application to work while the counter is counting down.
If you dont understand what I mean by idle then if I were you (being a
newbie) ignore the timer.
I often use this method for status bars, updating the system status or time,
and I think it's the best method for using Cobol.
