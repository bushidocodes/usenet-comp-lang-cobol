I grew up northeast Indiana and used to drive up to Detroit a lot to
see baseball games and concerts and stuff.  Lansing is much like
northeast Indiana.  Typical midwest mid sized town.  It is the capital
of Michigan so there is a lot of activity there.  

It is also next door to East Lansing, the home of Michigan State
University so there is that vibe as well.

If you are more used to mid to small town living, then I think you'd
find Lansing more than acceptable.  If you are used to big city life,
then you'd probably be disappointed in Lansing.

You might, if you haven't already, check out this web site:
http://www.lansing.org/visitor/things-to-do/
