The IBM System 34/36/38 and 5280 had something similar. It used PIC 1
but not the USAGE clause. The common use (as far as the Sys36 was
concerned) was to set screen attributes since SDA (Screen Design Aid)
used them to control things like cursor positioning, highlighting,
reverse video, and field protection.
