Hi,

The following code may be helpful in eliminating the table lookups to
convert from printable hex to internal hex.  I used the old compiler
max for a numeric PIC [PIC 9(18)].  Converting to the lastest max
[PIC
9(31)] should not be a problem.

 
******************************************************************
       01  WS-WORK-FLDS.
 
******************************************************************
      *===>
           05  WS-PACKED               PIC  9(017)     COMP-3.
           05  WS-INTRNL-HEX           REDEFINES
               WS-PACKED               PIC  X(009).

           05  WS-UNPACKED             PIC  9(017).
           05  WS-PRNTBLE-HEX          REDEFINES
               WS-UNPACKED             PIC  9(017).

           05  COMP-INFO               PIC  X(21) VALUE
SPACES.         00000910
1     /
 
******************************************************************
       PROCEDURE DIVISION.
 
******************************************************************
       000-MAINLINE.
 
******************************************************************
      *===> RTN-1 COVERTS INTERNAL HEX DATA TO PRINTABLE HEX.
*
      *===>
*
      *===> RTN-2 COVERTS PRINTABLE HEX DATA TO INTERNAL HEX.
*
      *===>
*
      *===> FOR EXAMPLE THIS INTERNAL HEX DATA: X'12ACF45F'
*
      *===> PRINTS/DISPLAYS AS: >Ú4^<. TO PRINT/DISPLAY IT AS
*
      *===> >12ACF45F<, USE ROUTINE 1. TO CONVERT PRINTABLE HEX DATA
*
      *===> (E.G. 12ACF45F) TO ITS INTERNAL FORM, USE ROUTINE 2.
*
      *===>
*
      *===> I USED THE OLD PIC LIMIT OF 18 FOR DIGITS; IF YOU WANT TO
*
      *===> USE THE NEW LIMIT OF 31, USE THE ARITH(EXTEND) COMPILER
*
      *===> OPTION AND ADJUST THE PIC OF THE WORK FIELDS BELOW.
*
      *----------------------------------------------------------------
*

       000-MAIN.
           MOVE    X'12ACF45F'         TO WS-INTRNL-HEX
           DISPLAY 'INTERNAL  HEX >'   WS-INTRNL-HEX  '< '
           PERFORM RTN-1
           DISPLAY 'PRINTABLE HEX >'   WS-PRNTBLE-HEX '< '

           DISPLAY ' '

           MOVE    '12ACF45FB6980A3B'  TO WS-UNPACKED(1:17)
           DISPLAY 'PRINTABLE HEX >'   WS-PRNTBLE-HEX '< '
           PERFORM RTN-2
           DISPLAY 'INTERNAL  HEX >'   WS-INTRNL-HEX  '< '
           GOBACK
           .
       RTN-1.
 
*-----------------------------------------------------------------
      *===>  ROUTINE TO CONVERT INTERNAL HEX DATA TO PRINTABLE HEX.
 
*-----------------------------------------------------------------
           MOVE    WS-PACKED           TO WS-UNPACKED
           INSPECT WS-UNPACKED            CONVERTING
                   X'FAFBFCFDFEFF'     TO 'ABCDEF'
           .
       RTN-2.
 
*-----------------------------------------------------------------
      *===>  ROUTINE TO CONVERT PRINTABLE HEX DATA TO INTERNAL HEX.
 
*-----------------------------------------------------------------
           INSPECT WS-UNPACKED            CONVERTING
                   'ABCDEF'            TO X'FAFBFCFDFEFF'
           MOVE    WS-UNPACKED         TO WS-PACKED
           .
 
******************************************************************
      *===>  END OF PGM CODE
 
******************************************************************
