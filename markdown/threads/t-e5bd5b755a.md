Sorry, I got a telephone call while I was finishing the above post

You can download Fujitsu COBOL 3.0 free from www.adtools.com.

But a better option is to buy the Sams book "Teach Yourself COBOL
in 24 Hours" by Thane Hubbell, which includes a CD with both 16 and
32 bit versions of the same compiler.

Or, if you would like a more advanced COBOL book, buy Sams "COBOL
Unleashed" by Jon Wessler, et al., which also comes with the same

excellent CD that contains a number of COBOL compilers and
interpreters, including Fujitsu 3.0.  But "COBOL for Dummies"
is not as detailed and thorough as either of the above books.
Not a criticism; the 'for Dummies' books are supposed to be
light and easy introductions.

Both books are readily available at good book stores and online.
