Sven,

I suspect that you linked your application as graphical. If so, the icon is
not used by the application and if the run-time system creates a window for
text output it has its own icon for that window.

If your application is a text-based program, link the application as
character. Then,  the operating system (if running WindowsNT) will display
the icon you linked in.
