Easiest way example:
01 wCtrl.
   03 wState	PIC X(3)
   03 wCity	PIC X(20)
   03 wLast	PIC X(15)
   03 wFirst	PIC X(15)
01 MaxKey	PIC 9(4) COMP.


MOVE ZERO TO MaxKey
IF State NOT= SPACE
   ADD 3 TO MaxKey
   IF City NOT= SPACE
      ADD 20 TO MaxKey
      ...
   END-IF
END-IF

MOVE wCtrl TO FileCtrl
START File KEY NOT < FileCtrl NEXT [REGARDLESS]
READ File NEXT [REGARDLESS]
PERFORM UNTIL EndOfFile OR wCtrl(1:MaxKey) NOT= FileCtrl(1:MaxKey)
   <Your action code>
   READ File NEXT [REGARDLESS]
END-PERFORM
