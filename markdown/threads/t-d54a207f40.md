Someone else has already given a partial answer, but I think the bottom-line
is that for the SORT syntax

1) After the sort verb, you place the name from an SD
2) In the Using/Giving phrases (if used) you put the names from the FD
3) The fields in the Ascending/Descending keys, need to be defined in a
record under the SD.
4) You only need to get into Release and Return if you are doing Input
and output procedures (which it sounds like you do NOT need for your case -
but should learn about in the next lesson).

David,
You cannot sort a FILE from within a COBOL pgm.. You must check out the
SORT examples better in your book. You can only SORT storage areas
referenced by SD's (Sort Descriptors) set up in the file section.
Basically you read your file into an SD area RELEASE the records to the
SORT and RETURN them from the sort and process them.
