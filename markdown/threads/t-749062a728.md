```
I would appreciate knowing what various compilers do with the following 
code.

with any '85 Standard compiler.  If you capitalize the lower-case letters, 
it should work with '74 compilers too (I think - I haven't tried it).

I know the results (already) with Micro Focus and IBM Enterprise COBOL, but 
would like to know what other compiers do (and more importantly, what the 
run-time resutls are).

Feel free to post your replies or send it to me directly at
   wmklein <at> ix.netcom.com

>>>> Code follows - hopefully, there won't be any "space" problems <<<

      * $SET Dialect(ANS85) FLAGAS(W) AREACHECK
       Identification Division.
        PROGRAM-ID. SIVSXX.
       DATA DIVISION.
        WORKING-STORAGE SECTION.
         1 A PIC XX VALUE IS "1 ".
         1 B PIC XX VALUE IS "10".
         1 I PIC 9P VALUE IS 10.
       PROCEDURE DIVISION.
        Mainline.
           IF I EQUAL A
             DISPLAY "This implementation is standard-compliant."
      *       MOVE 1 TO RETURN-CODE
                .
           IF I EQUAL B
             DISPLAY "This implementation has the alternative "
                 "interpretation."
      *       MOVE 2 TO RETURN-CODE
                 .
           IF I NOT EQUAL A AND I NOT EQUAL B
             DISPLAY "This implementation has yet another "
                 "interpretation."
      *       MOVE 3 TO RETURN-CODE
                .
           STOP RUN.
```
