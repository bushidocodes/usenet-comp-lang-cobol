Line 1114 specifies the highest line your are trying to display. It is line
29.  Likely, you have not set a screen size.  The default is an 80 column by
24 line screen. At run time, the program bombs, because it cannot write the
output.  Note that the screen size is set when you run the program, so you
are quite correct with your code.  What you have to do is set the run time
screen to fit.  I like a blank line at top and  bottom, I think it looks
better, so I would suggest you try a screen size of 30*80.  Once you see
what it looks like, you can narrow it down so the text looks centered.

Run time parameters are specified in a file called cobol85.cbr that sits in
the run area alongside the exe.  Here is a typical one for a system. The
name in the square brackets is the PROGRAM-ID that you used in the program.
(BTW, Yours won't work.  Change:
> 000030 PROGRAM-ID.   LOTTERY.
to
> 000030 PROGRAM-ID.  " LOTTERY".
)
