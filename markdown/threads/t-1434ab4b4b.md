If you post your example I am sure we would all like to tell you what is
wrong with it.

Otherwise, if you are using a newer compiler, OS370 mainframe, chances are
there are some nice ones implemented as intrinsic functions.

I have written some in REXX as well as COBOL and one thing that always helps
is to convert all dates to a BINARY type of date, number of days, then do
your operations, then convert back. This method works nicely when just
converting date formats. It also easily generates algorithms for number of
weeks between and of course number of days between. Of course if you don't
have a BINARY date function this could be a little difficult. In that case
JULIAN makes a good substitute. But again you need a good JULIAN date
function or your program will get rather complicated.

Yes.  A calculation for changing a date into a count of days since a given
baseline which includes Y * 365 + Y / 4 (ignoring months and days for the
moment) would assume all years divisible by 4 were leap years and such logic
would work from 1901 until 2099.  If the calculation was Y * 365 + Y / 4 - Y
/ 100 + Y / 400 , it would work for any Gregorian year.  Only if Y / 400 was
missing would the program not treat 2000 as a leap year.

 CONVERT-FROM-STANDARD.
                MOVE ZERO TO NEW-JULIAN-YEAR, JULIAN-DAY, BASE-DATE.
  MOVE 1 TO DAY-WEEK-NUMB.
                IF NEW-SYSTEM-YEAR IS LESS THAN 1986
                        MOVE 1986 TO NEW-SYSTEM-YEAR.
  IF SYSTEM-MONTH IS LESS THAN 1
   MOVE 1 TO SYSTEM-MONTH.
  IF SYSTEM-MONTH IS GREATER THAN 12
   MOVE 12 TO SYSTEM-MONTH.
  IF SYSTEM-DAY IS LESS THAN 1
   MOVE 1 TO SYSTEM-DAY.
  IF SYSTEM-DAY IS GREATER THAN 31
   MOVE 30 TO SYSTEM-DAY.
  PERFORM SYSTEM-TO-BASE.
  PERFORM BASE-TO-JULIAN.
  PERFORM BASE-TO-DAY-WEEK-NUMB.


 SYSTEM-TO-BASE.
                SUBTRACT 1985 FROM NEW-SYSTEM-YEAR GIVING SUBSCRIPT.
  ADD YEAR-START (SUBSCRIPT), BASE-DATE
   GIVING BASE-DATE.
  ADD MONTH-START (SYSTEM-MONTH), BASE-DATE
   GIVING BASE-DATE.
  ADD SYSTEM-DAY, BASE-DATE GIVING BASE-DATE.
                IF NEW-SYSTEM-YEAR IS EQUAL TO 1988
   MOVE 1 TO SUBSCRIPT
  ELSE
                IF NEW-SYSTEM-YEAR IS EQUAL TO 1992
   MOVE 1 TO SUBSCRIPT
  ELSE
                IF NEW-SYSTEM-YEAR IS EQUAL TO 1996
   MOVE 1 TO SUBSCRIPT
                ELSE
                IF NEW-SYSTEM-YEAR IS EQUAL TO 2000
                        MOVE 1 TO SUBSCRIPT
  ELSE
   MOVE ZERO TO SUBSCRIPT.
  IF SYSTEM-MONTH IS LESS THAN 3
   MOVE ZERO TO SUBSCRIPT.
  ADD SUBSCRIPT, BASE-DATE GIVING BASE-DATE.


 BASE-TO-JULIAN.
  MOVE 1 TO SUBSCRIPT.
  PERFORM BASE-TO-JULIAN-TWO.
  MOVE BASE-DATE TO DATETEMP1.
  SUBTRACT YEAR-START (SUBSCRIPT) FROM DATETEMP1
   GIVING JULIAN-DAY.
                ADD 1985, SUBSCRIPT GIVING NEW-JULIAN-YEAR.


 BASE-TO-JULIAN-TWO.
  MOVE BASE-DATE TO DATETEMP1.
  SUBTRACT YEAR-START (SUBSCRIPT) FROM DATETEMP1.
  IF DATETEMP1 IS GREATER THAN ZERO
   ADD 1 TO SUBSCRIPT
   GO TO BASE-TO-JULIAN-TWO.
  SUBTRACT 1 FROM SUBSCRIPT.


 JULIAN-TO-BASE.
  MOVE ZERO TO BASE-DATE.
                IF NEW-JULIAN-YEAR IS LESS THAN 1986
   NEXT SENTENCE
  ELSE
                        SUBTRACT 1985 FROM NEW-JULIAN-YEAR
                                GIVING SUBSCRIPT
   ADD YEAR-START (SUBSCRIPT), BASE-DATE
    GIVING BASE-DATE
   ADD JULIAN-DAY, BASE-DATE GIVING BASE-DATE.


 BASE-TO-DAY-WEEK-NUMB.
  DIVIDE BASE-DATE BY 7 GIVING DATETEMP1
   REMAINDER DAY-WEEK-NUMB.
  IF DAY-WEEK-NUMB IS LESS THAN 5
   ADD DAY-WEEK-NUMB, 3 GIVING DAY-WEEK-NUMB
  ELSE
   SUBTRACT 4 FROM DAY-WEEK-NUMB.
  MOVE MONTH-LITERAL (SYSTEM-MONTH)
   TO MONTH-YEAR-LITERAL.
  MOVE DAY-LITERAL (DAY-WEEK-NUMB) TO DAY-WEEK-LITERAL.


 CONVERT-TO-STANDARD.
  PERFORM BASE-TO-JULIAN.
  PERFORM JULIAN-TO-SYSTEM.
  PERFORM CONVERT-FROM-STANDARD.


 JULIAN-TO-SYSTEM.
                MOVE NEW-JULIAN-YEAR TO NEW-SYSTEM-YEAR.
  MOVE 1 TO SUBSCRIPT.
  PERFORM BASE-TO-SYSTEM-TWO.
  MOVE SUBSCRIPT TO SYSTEM-MONTH.
  SUBTRACT MONTH-START (SUBSCRIPT) FROM JULIAN-DAY
   GIVING SYSTEM-DAY.
  IF SUBSCRIPT IS LESS THAN 3
   NEXT SENTENCE
  ELSE
                IF      ((NEW-JULIAN-YEAR IS EQUAL TO 1988)
    OR
                        (NEW-JULIAN-YEAR IS EQUAL TO 1992)
    OR
                        (NEW-JULIAN-YEAR IS EQUAL TO 1996)
    OR
                        (NEW-JULIAN-YEAR IS EQUAL TO 2000))
    SUBTRACT 1 FROM SYSTEM-DAY.


 BASE-TO-SYSTEM-TWO.
  MOVE JULIAN-DAY TO DATETEMP1.
  SUBTRACT MONTH-START (SUBSCRIPT) FROM DATETEMP1.
  IF SUBSCRIPT IS LESS THAN 3
   NEXT SENTENCE
  ELSE
                IF      ((NEW-JULIAN-YEAR IS EQUAL TO 1988)
    OR
                        (NEW-JULIAN-YEAR IS EQUAL TO 1992)
    OR
                        (NEW-JULIAN-YEAR IS EQUAL TO 1996)
    OR
                        (NEW-JULIAN-YEAR IS EQUAL TO 2000))

  IF DATETEMP1 IS GREATER THAN ZERO
   ADD 1 TO SUBSCRIPT
   GO TO BASE-TO-SYSTEM-TWO.
  SUBTRACT 1 FROM SUBSCRIPT.
