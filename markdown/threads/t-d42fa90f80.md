On 08 Aug 2000 03:20:07 GMT, horsupport@aol.com (Horsupport) wrote:

>Any way to use COBOL to write CGI scripts in developing websites?

Sure, 

Create an html file similar to this:
 <html>
 <head>
 <title>Trying a CGI program</title>
 </head>
 <body>
 <h1 align="center">Trying a CGI Program</h1>
 <p>Fill in the appropriate data, then press the "SUBMIT" button
 </p>
 <form method="get" action="/cgi-bin/mycgi.exe">
 <p><strong>Your name</strong> <input name="name" size=30></p>
 <p><strong>Age</strong> <select name="age">
 <option value=0>Less than 20
 <option value=1>between 20 and 40
 <option value=2>between 40 and 60
 <option value=3>older than 60
 </select>
 </p>
 <p><strong>Sex </strong>
 <input type=radio name=sex value=0>female   
 <input type=radio name=sex value=1>male      
 <input type=radio name=sex value=2>not enough
 </p>
 <input type="submit" value="Send request">
 <input type="reset"  value="Clear form">      
 </form>
 </body>
 </html>


The CGI Program is in VA COBOL, as this is the only COBOL I use on my Windows/NT system:


       IDENTIFICATION DIVISION.
        PROGRAM-ID. "MYCGI".
       DATA DIVISION.
        WORKING-STORAGE SECTION.
      * ----------------------------------------------------------
       01  URL-data.
           02 URL-data-char PIC X 
                            OCCURS 1 TO 999 TIMES 
                            DEPENDING ON URL-data-length.
       01  Environment-var.
           02 Environment-var-name PIC X(12) VALUE Z'QUERY_STRING'.
           02                      PIC X     VALUE LOW-VALUE.
       01  URL-data-length         PIC 9(4)  BINARY VALUE 999.
       01  F1                      PIC X(20).
       01  F2                      PIC X(20).
       01  F3                      PIC X(20).
       PROCEDURE DIVISION.
           PERFORM Send-header
           PERFORM Get-url-data
           IF Url-data-length > 0 
              INSPECT URL-data REPLACING ALL '+' BY ' '
              UNSTRING URL-DATA DELIMITED BY '&' 
                       INTO F1
                            F2
                            F3
              END-UNSTRING              
              PERFORM Send-html
           ELSE
              DISPLAY 'Error retrieving URL DATA'
           END-IF
           PERFORM Send-trailer 
           GOBACK
           .

       SEND-HEADER.
           DISPLAY 'Content-type: text/html'
           DISPLAY SPACES
      * Note, if using DOMINO WEB SERVER or APACHE WEB SERVER, 
      *       use DISPLAY X'0A0D' instead!
           DISPLAY '<html>'
           DISPLAY '<header>'
           DISPLAY '<title>'
           DISPLAY 'My first COBOL CGI Program'
           DISPLAY '</title>'
           DISPLAY '</header>'
           DISPLAY '<body>'
           .

       Get-url-data.
           MOVE 999 TO Url-data-length
           CALL 'GetEnvironmentVariableA' 
                 Using BY REFERENCE Environment-var
                                    URL-data
                           BY VALUE LENGTH OF URL-data
                 RETURNING URL-data-length
           IF URL-data-length NOT = 0
           ELSE
              DISPLAY 'ERROR GETTING URL'
              PERFORM Send-trailer
              STOP RUN
           END-IF
           .

       Send-html.
           DISPLAY '<h1>My first attempt at a CGI program</h1>
           DISPLAY '<P>'
           DISPLAY '<br>Parm1=' F1
           DISPLAY '<br>Parm2=' F2
           DISPALY '<br>Parm3=' F3
           DISPLAY '</P>'
           .
 
      Send-trailer. 
           DISPLAY '</body>'
           DISPLAY '</html>'
           .
