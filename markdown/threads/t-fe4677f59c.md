>I have a need to dynamicly allocate a dataset at open time and not by

Have you considered TSO services? You call IKJTSOEV at the start of your
program to set up the TSO environment, then you can issue 'tso alloc'
statements using IKJEFTSR. I'm pretty sure there's a complete example in
bookmanager if you do a search on the module names. I can send you a
stripped-down example if you can't find it.

As far as the body of the program is concerned, it has no idea you're
goofing with the allocations, provided you do the allocs before you open
the file, which is normal ASSIGN/FD type coding in the program itself. *
