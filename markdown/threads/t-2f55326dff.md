IDX suffix cannot be used for an indexed file in Micro Focus.
"filename.IDX" is used by Micro Focus to contain the actual index
to the corresponding data file.

>                   ORGANIZATION IS INDEXED
[...]
>001560     OPEN INPUT  CUSTOMER-MASTER    *error reported here*
>                       SORTED-FILE
>                       STOCK-MASTER-FILE
The error is reported on the OPEN statement, any of the files
could be the source of the problem. In this case, it is the last one.

word the problem went away.
