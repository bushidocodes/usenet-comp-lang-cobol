› Can someone please tell me the difference between the two?


OS/VS COBOL is based on the ANSI-74 definitions for the language and
COBOL/II is based on the '85 standards. COBOL/II allows features such
as scope delimiters, in-line performs, evaluate statement and reference
modification. COBOL/II dropped direct support for the Report Writer
functions and requires a pre-processor. There are more issues,
including technical issues compiling programs.

>From "VS COBOL II: A guide for Programmers & managers" by Anne Prince:

Figure 1-1, Features of VS COBOL II that aren't available under OS/VS
COBOL:

New language elements
Stuctured programming enhancements
(in-line PERFORM, IF, EVAULATE, etc.)
Easier CICS coding

Performance improvements
31-bit addressing
Reentrant programs
Improved optimization
Faster sorting

Compiler improvements
Checking for out-of-range conditions
Increased compiler limits

Improved debugging features
VS COBOL II debug
Improved compiler listings
Improved dump format

Other features
Controls added for reserved words
Controls added for compiler options
More feedback from VSAM requests
More flexibility for preloading COBOL programs under IMS/VS
Extended Graphic Character Set

Figure 1-3, Features of OS/VS COBOL that aren't supported by VS COBOL
II:

1974 standard features
Report Writer
Communications support
Advanced capabilities of the USE FOR DEBUGGING statement

Language elements (mostly IBM extensions to '68 and '74 standards)
(EXAMINE, EXHIBIT, NOTE, READY TRACE, ALTER, etc.)

ISAM and DBAM files

The lister feature

Hope this helps,

Boyce G. Williams, Jr.
