You will need to set a compiler option that states that your program is
MAIN.  This is to distinguish it from supporting files that are compiled
separately.  To do this add the following line to your code beginning in
column 8:

	@OPTION MAIN

This must be the very first line of your program and must be all
caps.  Because this a compiler directive, and not a COBOL statement, case
does matter.

I think this is in the fact; but if not, it is covered in Chapter 2 of
Thane Hubbell's book "Teach Yourself COBOL in 24 Hours".

Good luck.

Floyd Johnson
