```
I am using Wang's COBOL compiler and we have tools for screen formatting.
I use a tool called EZFORMAT where I can put all of a screen's text and
fields on the screen and EZFORMAT would generate a COBOL section that 
can be inserted into my program. WANG's COBOL also have a display and
read function that can display a whole screen, does data verification and
gets user input using one single function.

My question here is does Micor Focus have similat tools and what's the
name of the tool/program that is similar to our EZFORMAT?

Abdu
```

```
Yes, the Micro Focus compiler includes a similar tool, called 
SCREENS, which will allow you to paint your screens and generate 
the COBOL code necessary to use them.  There are, also, third 
party services & packages available to convert Wang applications 
to Micro Focus applications or to emulate some of the key Wang 
facilities on the PC, in conjunction with a Micro Focus comiled 
application.  If you have access to the Micro Focus Forum on 
CompuServe, these are listed in one of the Library files, whose 
search keyword is WANG.  If you are interested and you don't have 
 access to CompuServe, let me know and I will forward a copy of 
the information.

Regards, Alan.
```
