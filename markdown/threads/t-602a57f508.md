This is an old thread. But one was looking for a javadoc like for Cobol.

You can have a look at robodoc at
http://www.xs4all.nl/~rfsber/Robo/robodoc.html

It generates a documentation in html, rtf, etc... You can use it with
cobol and personalize it if you do not use standard comments. Il
personally prefer *> instead of * in column 7 for comments.

It generates the hierarchy of you project (if you commented correctly
the project of course).

So, it is worth seeing for programmers who are looking for an automatic
documentation system.

Regards.

Alain
