Yes, I do. Used MF COBOL for nine years but we changed to VisualAge and
loved it. I had a client that we setup with an MF COBOL, CICS, DB2 workstation
environment to develop host systems. We always had some minor "annoying"
problems when migrating from the workstation to the host for final testing and
implementation.

One thing we liked about VisualAge is that we could use the same set of
manuals for both workstation and host code. It's also a good envrionment for
developing workstation and client/server application as well as OO if you're into
that. Feel free to E-mail me if I can help.

I think the win '95 and NT versions come out next month.
