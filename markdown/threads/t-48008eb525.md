The compiler notes that a program has Local-Storage and generates code
as required.  This allows for re-entrancy.  This is does whether the
program has any CALLs in it or not.

Recursion is 'self re-entering' and is only detectable from re-enter
by seeing that the caller and the called are the same, which is not
done by the compiler.  That is a program may be re-entered or it may
recurse.  L-S provides for both/either.
