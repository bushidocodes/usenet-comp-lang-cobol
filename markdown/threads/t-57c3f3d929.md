The goal is not to avoid GO TO.  The goal is to avoid having multiple ways to
"come from".   The goal is to avoid drop through code.  The goal is to avoid
errors.

If your code already has EXIT paragraphs, sure there is no difference.   But
with an EXIT-PARAGRAPH command, there will no longer be an excuse for PERFORM
THRU logic in new programs.
