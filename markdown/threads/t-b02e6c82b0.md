Well, I should have tested, because I made a big boo-boo here.

While you *do* need to add the second index in the WHEN clause, you
do NOT search a subscripted or  indexed area using the subscript; you set
the subscripts only in the WHEN clause of the search.

Corrected code follows, and sorry for any extra time spent by anyone
following this road.

Michael Mattias
Racine WI USA
