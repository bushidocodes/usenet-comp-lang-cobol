```
From your description I take it that you are looking for a way to remove
embedded spaces from text fields (e.g. "A C E G" into "ACEG   ").

Using 2 work areas:

       IDENTIFICATION DIVISION.
       ENVIRONMENT DIVISION.
       DATA DIVISION.
       WORKING-STORAGE SECTION.
       77  IN-TEXT                     PIC  X(07) VALUE "A C E G".
       77  OUT-TEXT                    PIC  X(07).
       77  IN-PTR                      PIC  9(02) COMP.
       77  OUT-PTR                     PIC  9(02) COMP.
       PROCEDURE DIVISION.
       DO-IT-TO-IT.
           DISPLAY ":" IN-TEXT ":".
           PERFORM CRUNCH-TEXT.
           DISPLAY ":" OUT-TEXT ":".
           STOP RUN.

       CRUNCH-TEXT.
           MOVE SPACES TO OUT-TEXT.
           MOVE 1 TO OUT-PTR.
           PERFORM VARYING IN-PTR FROM 1 BY 1
                   UNTIL IN-PTR > FUNCTION LENGTH(IN-TEXT)
               IF IN-TEXT(IN-PTR:1) NOT = SPACE
                   MOVE IN-TEXT(IN-PTR:1) TO OUT-TEXT(OUT-PTR:1)
                   ADD 1 TO OUT-PTR
               END-IF
           END-PERFORM.

Using 1 work area:

       IDENTIFICATION DIVISION.
       ENVIRONMENT DIVISION.
       DATA DIVISION.
       WORKING-STORAGE SECTION.
       77  WORK-TEXT                   PIC  X(07) VALUE "A C E G".
       77  IN-PTR                      PIC  9(02) COMP.
       77  OUT-PTR                     PIC  9(02) COMP.
       PROCEDURE DIVISION.
       DO-IT-TO-IT.
           DISPLAY ":" WORK-TEXT ":".
           PERFORM CRUNCH-TEXT.
           DISPLAY ":" WORK-TEXT ":".
           STOP RUN.

       CRUNCH-TEXT.
           MOVE 1 TO OUT-PTR.
           PERFORM VARYING IN-PTR FROM 1 BY 1
                   UNTIL IN-PTR > FUNCTION LENGTH(WORK-TEXT)
               IF WORK-TEXT(IN-PTR:1) NOT = SPACE
                   MOVE WORK-TEXT(IN-PTR:1) TO WORK-TEXT(OUT-PTR:1)
                   ADD 1 TO OUT-PTR
               END-IF
           END-PERFORM.
           PERFORM UNTIL OUT-PTR > FUNCTION LENGTH(WORK-TEXT)
               MOVE SPACE TO WORK-TEXT(OUT-PTR:1)
               ADD 1 TO OUT-PTR
           END-PERFORM.
```

```
If you have the latest report writer syntax in your COBOL, you can use 
relative columns and PRESENT WHEN clauses, e.g.
  03  LINE + 1.
    05  COL 1     VALUE "fixed bit".
    05  COL + 2   VALUE "company A bit"  PRESENT WHEN co-A-flag = 1.
    05  COL + 2   VALUE "company B bit"  PRESENT WHEN co-B-flag = 1.
```
