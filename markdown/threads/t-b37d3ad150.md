For the Realia COBOL compiler:

ISBN 0-13-140179-3 COBOL: from Micro to Mainframe, Robert T. Grauer,
Carol Vazquez
Copyright 1992 by Prentice-Hall, 1-800-223-1360 before 4:30 PM, about
$70 now.

In 1992 the book still included the compiler and examples on 5-1/4 inch
PC-DOS floppies. I hope they've converted to 3-1/2 if the book is still
in print.

You had to provide your own linkage editor (I used Microsoft LINK from
QuickBasic 4.5) and the compiler was limited to 500 lines of code,
excluding comments, and files were limited to 64K bytes. You could
generate standalone executables for DOS.

You could get around the 500 LOC limits by using subprograms. If you
had access to the manuals for the $800 Realia COBOL 4.2 compiler, you
could get documentation on the full set of library calls for MS-DOS
macros, which still worked from the educational version of the
compiler. I wrote a version of the TOUCH utility using Educational
Realia COBOL.
