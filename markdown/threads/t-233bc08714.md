GnuCOBOL 3.0-rc1.0 MinGW 64-bit (MSYS2 for Windows) is now available for
download as an installable binary. The MSYS2 MinGW GCC level is 7.3.0.

https://www.arnoldtrembley.com/GC30B-64bit-rename-7z-to-exe.7z (36.6
megabytes)

https://www.arnoldtrembley.com/GnuCOBOL-3.0-MSYS2-64-bit-Build-Guide-draft.odt

https://www.arnoldtrembley.com/GnuCOBOL-3.0-MSYS2-64-bit-Build-Guide-draft.pdf

Install to a folder (named something like C:\GC30B-64), open a CMD.EXE
window, navigate to that folder, and then run "bin\cobenv --verbose" to
update path and environment variables, then run "cobc --info" or
"cobcrun --info".

You should also be able to configure OpenCobolIDE (OCIDE) on Windows to
work with this compiler.

https://sourceforge.net/p/open-cobol/discussion/help/
