Do you mean an z/OS or MVS console command?
If so, you cannot do it from COBOL natively.
You will have to either:
1. Invoke a TSO CONSOLE command via the TSO Interface API.
2. Attach an EMCS console and issue the command via MGCR system macros
in an assembler subroutine - which must run authorized.
