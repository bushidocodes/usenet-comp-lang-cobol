COBOL was invented in a world of five and six bit common language
machines. The battle for a common programming language was preceded by a
battle for a common machine language, i.e., five bit paper tape on all
kinds of machines. First generation computers used for business were
patterned after the tab card systems where the code system left no space
for the character set we take for granted today. So, while the
discussion probably included using CoBOL, cooler heads prevailed in
favor for something easier to type

medium. That, and it was the same as the source code requirement for the
same reason. The other input medium was paper tape which only had five
bits per character, and Unityper which used a RemRand typewriter that
only typed upper case letter because that was all you could record or
print, or punch out. Change is a fact of life.
