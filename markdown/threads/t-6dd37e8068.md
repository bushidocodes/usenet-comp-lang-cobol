You really don't need the varying on either table.

The search of the second level table will be based on the current
value of the index of the first table.  Your search line is correct.
I didn't see a WHEN for the second search, however.  For the WHEN you
must fully qualify the element being tested (all index levels).

You can see an example of this in my book, Sams Teach Yourself COBOL
in 24 Hours, Chapter 12.  If I can help you further, let me know!
