Can anyone enlighten me as to how MF handles GSAM files.
A few years ago, we used Stingray IMSVS86 to handle GSAM
files. This entailed quite a bit of work to animate even a small
batch program. We had to download the PSB, specify the RECLEN
of the file and then gen the PSB. We then used the DBUTIL function
to load the file and had to repeat this step each time we animated
the program.

We are now running MF COBOL version 3.2.46. I was looking in
the documentation to see if GSAM files were handled any differently
with this newer version but could barely find a mention of GSAM in
the Micro Focus IMS Option Technical Reference Guide.

If anyone can explain how MF currently handles GSAM files, I
would greatly appreciate it.

Thanks.
