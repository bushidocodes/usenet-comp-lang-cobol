What does this mean?  If it is a CICS program, i.e. contains EXEC CICS Commands, it must
be compiled and linked to a DLL, a transaction code has to be set up in CICS NT, and then
the transaction has to be run under CICS

If you do NOT have a an EXEC CICS call, then you don't need to have CICS running anyways.
