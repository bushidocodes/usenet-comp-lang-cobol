4. Is there a free COBOL compiler?

Just for the record, no COBOL tools are listed in the Catalog of compilers, 
interpreters, and other language tools posted to comp.compilers and 
comp.lang.misc. This probably means that there are no freely available 
COBOL compiler sources.  There is, however, a current ongoing project to 
produce an "open source" (GNU) compiler.  See

http://www.gnu.org/software/cobol/index.html

another project �COBOL for GCC� and a status on various other �open source� 
projects, see
