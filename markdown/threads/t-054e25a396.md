```
It is real easy.  Your hardest task will be to create your JCL to run this.
You only need to do this once and then crib for all other EasyTrieve
programs.  (Of course, if this is already available, then you have
EasyTrieve programs available to crib as well, and don't need to be asking
us).

Here's a program:   (Sorted reports are almost as easy)
//Z46ACCT    JOB (12651,'EZT'),'U37   JONES,
//             CLASS=N,MSGCLASS=S
/*ROUTE PRINT R0010
//EASY#1     EXEC PGM=EZTPA00
//STEPLIB    DD DSN=EASY.LOADLIB,DISP=SHR
//EZTVFM     DD UNIT=SYSDA,SPACE=(4096,(100,100))
//ACCTFD     DD DSN=BRS.ACCTDT,DISP=SHR
//SYSPRINT   DD SYSOUT=*
//OUT1       DD DSN=UMS.D92747.ACCTF4,DISP=(NEW,CATLG),UNIT=DEVEDISK,
//             SPACE=(CYL,(5,1),RLSE)
//SYSSNAP    DD SYSOUT=*
//SYSOUT     DD SYSOUT=*
//SYSIN      DD *
FILE OUT1 FB (270 27000)
  OUT-REC       1     270   A
  OUT-AMT    55        11   A
  OUT-R2     66       142   A
FILE ACCTFD
ACCTFD-REC      1     396   A
  ACCT-FILL     1     129   A
  ACCT-SID    130       9   A
  ACCT-REST   139     258   A
  ACCT-AMT  ACCT-REST +54   8 P 2
  ACCT-R2   ACCT-REST +62 142 A
WK-AMT   W   11  N  2
JOB INPUT ACCTFD
IF ACCT-SID = '524794886'
   OUT-REC  = ACCT-REST
   WK-AMT   = ACCT-AMT
   OUT-AMT  = WK-AMT
   OUT-R2   = ACCT-R2
   PUT OUT1
END-IF
```
