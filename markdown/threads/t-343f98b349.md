First understand this:

ALL COBOLs when doing 'CALL xxxxxxx USING'
pass ONE ITEM ONLY.

REPEAT ONE ITEM ONLY.

INCASE SOMEONE MISUNDERSTOOD,
COBOL ONLY PASSES ONE ITEM.

The item will be EITHER an address length item of 0 / zero
Or it will be the address of the first BYTE of the FIRST thing PASSED.

01  MY-LIST.                memory address at run-time 0100A0
...
01  MY-RC.
011A00

CALL 'MYSUB' USING MY-RC, MY-LIST

will casue the compiler to put the address of MY-RC in memory,
put the address of MY-LIST RIGHT BEHIND it.
So you will have  at some location we'll say 20200:  011A008100A0   (The 8
means bit leftmost is on
to signify the END-OF-LIST) - 99./100 times that is how it is done.)
The COBOL routine will load 20200 into the 'parameter register' and Branch
to the routine.

All you need to do, is grab the 4 bytes at the 'parameter' register,
check for END-OF-LIST, and while not EOL,
get whatever is pointed to by that address.

The "attached" is an incredibly "implementation specific" definition.  What
the ANSI COBOL Standard says is that if you pass a parameter (explicitly or
implicitly) BY REFERENCE, that you get "addressability" to that item (with or
without any indication of the length of the item passed).   What was in the
"attached" note is certainly how many implementations work (ignoring for the
moment the assumption that a "pointer" is 4 bytes long).

/* ChangeCase subroutine.  Makes all items in a string which are letters
*  into the selected case: all to upper or all to lower. Ignores others.
*  Accepts: switch (1 char), short count of string length, string of chars
*  &, a short ret-code.
*  Modifies the string as is.  Max string length is 128.
*/

#define SUCCESS         0
#define FAILURE         -1
#define NUMBER_OF_ARGS  4

void  ChangCas( Passed )  /* COBOL passes one Address)
/* actually ( swtch, count, str_i_o, ret_code )    CALL 'CHANGCAS' using
WS-SWITCH, */
/* WS-SIZE, WS-STRING, WS-RC  */
{
   char *swtch, *str_i_o, *Ptr_to_Char, *Ptr_to_int_as_Char;
   char Two_Bytes[3];
   short count, *ret_code;
   int  List_Array[NUMBER_OF_ARGS], i;
   /* args come in as  AddressOfSWITCH,..AddressOfRC */
   int  *Start_Of_List;
   Start_Of_List = &Passed;     /* put Address of Passed Address into
StartOfList */
   for ( i = 0; i < NUMBER_OF_ARGS; i++ )
         List_Array[i] = *Start_Of_List++;

   swtch = List_Array[0];
   Ptr_to_Char = List_Array[1];
   for ( i = 0; i < sizeof(short); i++ )
      Two_Bytes[i] = *Ptr_to_Char++;
   Ptr_to_int_as_Char = &count;
   for ( i = 0; i < sizeof(short); i++ )
         *(Ptr_to_int_as_Char + i) = Two_Bytes[i];
   str_i_o = List_Array[2];
   ret_code = List_Array[3];

  if (count > 128)
      { Ptr_to_int_as_Char = ret_code;
        for ( i = 0; i < sizeof(short); i++ )
              *(Ptr_to_int_as_Char + i) = 0xff;
        return;
      }

  for ( i = 0; i < count; i++ )
        if ( *(str_i_o + i) >= 'A' && *(str_i_o + i) <= 'Z')
            {
            if ( *swtch == 'L' )
                  *(str_i_o + i) += 32;
            }
        else
           if ( *(str_i_o + i) >= 'a' && *(str_i_o + i) <= 'z')
               {
               if ( *swtch == 'U' )
                    *(str_i_o + i) -= 32;
               }

 Ptr_to_int_as_Char = ret_code;
 for ( i = 0; i < sizeof(short); i++ )
     *(Ptr_to_int_as_Char + i) = '\0';

return;
}
