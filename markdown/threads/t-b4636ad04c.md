Hi Scott,
The examples to look for are in the \netexpress\base\demo\oledemos\server 
and (in version 3.0) the \netexpress\base\demo\oledemos\dcom directories.
The first provides examples of creating both in-process and out-of-process 
COM servers, and the second shows a simple DCOM client and server. No code 
changes are required to move from local COM to DCOM.

There are also full docs available (in the Distributed Computing on-line 
book; search for OLE in the index and follow links).

In addition, the NetExpress Class Wizard (File/New/Class) will allow you 
to create skeleton DCOM (OLE Automation) programs with mimimum effort. 
Full context help is provided for every dialog.

Finally, our on-line AnswerLab service (on our support web site) has 
already had many questions about this area and serves as a useful 
knowledge base for technical queries that have come up in the past.

Hope this helps,
Steve.
Micro Focus Development.
