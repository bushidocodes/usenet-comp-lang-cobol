 One disadvantage is that IEFBR14 will do a recall of a migrated  file. IDCAMS
does not do a recall.
IEFBR14 can be used to delete  VSAM clusters and entire GDG generations too:

// EXEC PGM=IEFBR14
//DD00   DD   DSN=your.vsam,DISP=(OLD,DELETE)  <<delete VSAM
//DD01    DD   DSN=your.gdg,DISP=(OLD,DELETE)    <<delete all GDG generations.

When deleting a data set IDCAMS returns a CC of 8 if the data set is not found.
If you want a zero return code you can always do:
    IF LASTCC = 0 THEN    SET MAXCC = 0
