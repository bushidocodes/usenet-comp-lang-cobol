While possible, using the Micro Focus Screen Section - you really are
taking on a big challenge to try and emulate a scrollbar. Years ago I did
use the Screen Section to create a Menu Bar and Pop-Up menus - but
honestly, would suggest that's a bit of a toughie for beginners.

E-mail me if interested - I can give you a zipfile of some M/F DOS programs
that illustrate using Screen Section coupled with the Mouse. I also have a
second 'junk' zipfile which has some of my old Screen stuff - if you want
to wade your way through these, you might also get some ideas.

You didn't say which compiler - the DOS version 3.0 ??? Doubt you can get
your hands on a manual - but try the on-line books at the Micro Focus site.
Although now geared to Net Express etc., the appropriate manuals do include
all the Screen syntax plus use of ADIS(AcceptDISplay), directives etc.,
which you will need.

http://supportline.microfocus.com/documentation/books/nx31sp1/nx31indx.htm

The book you need is "Character User Interfaces'
