I needed something like this the other day.
Try 'random' from a UNIX command line.

My cobol code has something like this.

CALL "SYSTEM" USING "random 255 > xxx"

This will generate a random number between 0 and 255.
If you only want a two digit random number then replace 255 with 99.
You can select any range not exceeding 255.

Then I read xxx for my random number that was generated.
