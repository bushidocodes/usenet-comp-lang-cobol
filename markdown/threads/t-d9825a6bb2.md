The latest release of the open source Sonar platform for continuous inspection of code quality includes extended functionality for real-time detection of quality issues introduced with new code. It also provides a related review process as well as personalized dashboards configurable through widgets. In total, version 3.0 of the Sonar platform includes 40 improvements and just as many bug fixes while maintaining compatibility with over 50 plugins. COBOL engineers can use the COBOL plugin to perform objective and automated COBOL code reviews against pre-defined or custom coding best practices in Sonar 3.0.

To read more about the new features and get the latest release, visit http://www.sonarsource.com/3-0.
