Usually, I run Micro-Focus workbench, but today I had to create an
application with my old compiler, MBP-visual cobol. The task is:
I need to write records to a sequential file. All alfanumeric fields
should look like this: "Alfatext",129.50,3000,"Alfatext2".....
My record looks like this:
01 Customer-Record.
02 Cust-number PIC 9(7).
02 Filler PIC X value """.
02 Cust-name PIC X(30).

Because cobol use quotation marks around text, it seems impossible
to have a quotation mark between two quotation marks.
The Filler line should look like 02 Filler PIC X value """.

Any suggestion ?

Yes. To create one quote you need to put two. Therefore, you say
PIC X VALUE """" to get one quote. Or, which is easier, you can say
PIC X VALUE QUOTE. The two-quote thing is best for longer strings
like PIC XXXXX VALUE "AB""CD" where you end up with AB"CD. I hope
nobody suggests using apostrophies on the outside and the quote
inside. That is not standard and not portable to other
implementations.
