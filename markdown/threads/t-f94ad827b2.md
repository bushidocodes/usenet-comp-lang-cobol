We converted around 1.3 million lines of Wang COBOL to Micro Focus using
a product called VUPORT from San Soft, Inc. The system went "live" in
October 1995 and has been very stable. Our users experienced the
conversion as a non-event. I've been very satisfied with the product.
Visit their site at http://www.sansoft.com for details.

Your project is very "doable." As you are no doubt aware, there are two
schools of thought in the Wang migration area: 1.) Convert all VS code
into native format using the compiler of your choice, or 2.) use an
emulator product to provide a Wanglike environment and system services
on the target platform. Both methods have good and bad points.

VUPORT uses the middleware/emulator method. It is an "integrated
preprocessor" to Micro Focus, so additional code is inserted in your
source program as you compile. (The inserted code is essentially
invisible to you.) After compiling, programs run under a VUPORT runtime
that provides the Wang "shell", including the familiar command
processor, etc.

Good luck...
