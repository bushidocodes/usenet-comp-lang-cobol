OK then, I'll tell you :-)

There is no easy way of doing that with Dialog System.  That object that you
see in IE is an auto-complete drop-down.  You would have to write all the
functionality yourself.  You have to trap the characters as they are typed
in the entry portion of the drop-down, look through an array of possible
values for what you need and empty the drop-down and refill it with the
right values.

Quite a job, don't you think?

Alternative?  I'm sure someone out there sells an ActiveX control that does
that for you.  I don't recommend using ActiveX controls in Dialog System for
various reasons, but IT IS much easier than to do it yourself.

Cheers,
Calin, TORONTO
