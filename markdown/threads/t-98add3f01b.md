```
To comp.lang.cobol people:

I know that they were LATE in getting posted, but the presentation
(slides/handouts) from last August's SHARE in San Francisco are now
available online at:

  http://www.share.org/proceedings/sh99/share.html

(it takes a long time to load)

Of particular interest to (some) CLC readers would be:

   "What's New in COBOL for z/OS"

at:
 http://www.share.org/proceedings/sh99/SHARE/data/S8211.pdf

Some CLC readers might want to "expand" the "Sessions by Number" option on
the homepage  and do a search on "COBOL" and/or "LE" and see such sessions
as:

 8203 - 1001 Things to Do With a Mainframe COBOL App.
 8207 - LE Setup and Customization
 8208 - Diagnosing Application Problems Under LE
 8211 - What's New in COBOL for z/OS
 8216 - Assembler Issues When Migrating to LE and/or Amode 31
 8219 - Introduction to IPCS for Application Programmers
 8230 - Abend-AID and LE
 8231 - How to Perform an LE Migration Assessment
 8234 - Writing User Condition Handler Routines for Language Environment
 8239 - What Is ILC and How to Use It
 8240 - Migrating COBOL & PL/I Run-times to LE - Common Issues
 8241 - Migrating to COBOL Compilers Under LE
 8244 - Using the Edge Portfolio Analyzer for LE Migration

Obviously, other sessions (such as CICS or IMS or DB2 - or even Java) might
interest some of you.
```
