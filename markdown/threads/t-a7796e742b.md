```
Announce: TinyCOBOL release 0.55

The TinyCOBOL compiler version 0.55 is available at the TinyCOBOL home
page (1) and iBiblio (2).

This release contains mainly bugs fixes,and enhancements.  

It includes updates to the main compiler, run-time, the regression test
suite.

The grammar was modified to enable compilation with Bison.
The MOVE run-time routines have been updated and improved.
ScreenIO has been updated to handle screen status and function keys.
Support for [SIGN IS] {LEADING/TRAILING} [SEPARATE] clauses, have been
added to the compiler run-time.
Preliminary work as started on the NIST test programs.


Please report any bugs and/or problems to the TinyCOBOL mailing list. 


1) TinyCOBOL Home Page
   http://www.tinycobol.org/
   http://tiny-cobol.sourceforge.net/

2) iBiblio (formerly Metalab)
   http://www.ibiblio.org/pub/Linux/devel/lang/cobol/
```
