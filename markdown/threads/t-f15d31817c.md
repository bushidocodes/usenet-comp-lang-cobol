Yes, we do this for similar reasons.

//JS040  EXEC PGM=IEBGENER,                                           
//            COND=(1,GT,JS020)                                       
//SYSPRINT DD SYSOUT=*                                                
//SYSIN    DD *                                                       
//SYSUT2   DD SYSOUT=A,DEST=(destination,NETISMTP)  
//SYSUT1   DD DATA                                                    
HELO smtp.yourserver.com
MAIL FROM:<you@yourcompany.COM>
RCPT TO:<you@yourcompany.COM>
DATA                                                                  
TO:<you@yourcompany.COM>
SUBJECT: ERROR PROCESSING SOMEJOB                                    
THERE WAS AN ERROR PROCESSING SOMEJOB.  FIX IT RIGHT AWAY!
.                                                                     
QUIT                                                                  
/*                                                                    
//*
