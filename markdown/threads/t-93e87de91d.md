The one I used on VM/CMS was COBOL. That was the name of the compiler.
But you had to bring in the libraries with coblib command. Then you had
to type in LOAD or some such. The cobol compile had to be followed
by the Load step.

Example:

filename filetype drive

mytest cobol a


type in:
cobol mytest

Then do mod or load:
load mytest

run mytest.

Good luck...

Try typing in the COBOL command followed by your input source code.

Then try using filedef statements to tie your ddnames in your program
to the input data files.
