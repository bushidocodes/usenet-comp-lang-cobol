Be patient with me; I'm a newbie.

This is a question concerning a project at my job. I need to develop a
program designed to assist private agencies who do not have a
communication link with our Mainframe environment. We would like them to
be able to send in diskettes with preformatted data files, rather than
sending in hardcopy forms which force us to input the data manually. Is
there a COBOL compiler available which would allow me to create a
stand-alone program for the PC? I'm sure it would be easier to choose
another language. I have Borland C++ at home and I have considered

authorized to request any other programming language (or atleast I doubt
if they would allocate the funds for something other than COBOL.)

Any help would be appreciated. Thanks in advance.

Cobol compilers have existed for years for the PC. CA-Realia, Micro
Focus, Acucobol, to name a few. The type of application you describe
sounds like a typical data entry program. Cobol is a perfectly
appropriate language for this project. I developed a rather significant
system for employers to create wage information using CA-Realia Cobol.
This information is sent to us on diskette, or can be up-loaded to us
via a BBS.

Today's Cobol can do just about anything you can think of, including WWW
applications, not to mention GUI. Cobol is probably the most portable
language there is.
