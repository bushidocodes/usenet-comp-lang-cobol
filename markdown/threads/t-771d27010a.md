Hi

We are looking for a way to perform mass compiles to generate int and/or gnt
files for use in the MicroFocus runtime system.

Does anybody know of a way to get any of the MicroFocus compile tools to

I use a version of 'make' for DOS. The makefile contains all the
dependancies so that when a file is edited the make program can
determine what requires to be compiled based on the dates and times
of all these files.

Often a 'make' program will remain resident while it fires off each
compile. This can reduce memory available to the compiler. I use
the '-N' option so that make doesn't execute the commands but
just outputs them to stdout. This is then captured using redirction
to a .BAT file, which is then executed. Thus in my 'RECOMP.BAT'
I have:

[various initialization]
make -N >Doit.Bat
DoIt

There are PD versions of make available which will run on DOS.
I even modified one to run as a CP/M-86 version (for the multi-user
Concurrent-DOS).
