I don't use ShellExecute, I use CreateProcess and you can have it wait
until the executed item finishes to return to your program or you can
do it Async.  Here is the Fujitsu version of the API call.

     MOVE "\SCRIPTS\PINGIIX.BAT" to command-line
     COMPUTE CB = FUNCTION LENGTH(STARTUPINFO)
     move 2 to wshowwindow		
     CALL "CreateProcessA" WITH STDCALL USING
                            BY VALUE 0 
                            BY REFERENCE COMMAND-LINE
                            BY VALUE 0
                            BY VALUE 0
                            BY VALUE BLNHERITHANDLES
                            BY VALUE DWCREATIONFLAGS
                            BY VALUE 0
                            BY VALUE 0
                            BY REFERENCE STARTUPINFO
                            BY REFERENCE PROCESSINFO
                            RETURNING RETURN-VALUE
      END-CALL

working.
