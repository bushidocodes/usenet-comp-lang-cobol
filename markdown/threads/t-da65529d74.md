And I hate (even more) to disagree with Bill ... WFL can be a useful
data creation/editing tool. Create a set of test data using your
favourite text editor (the source editor provided in Workbench, Net
Express and Mainframe Express would be a good choice), and then use WFL
to convert it to the required format (ISAM, relative or whatever). The
source editor makes it easy to create many records of similar data by
using cut/copy/paste. In Net Express and Mainframe Express, WFL is
replaced by the cunningly named "Data File Convertor" for doing
conversions on data files.

DFED of course is an excellent tool and GUI versions are provided with
Workbench, Net Ex and Mainframe Express (the latter products having the
nice Windows native look-and-feel. Mainframe Express V2.0 (with
AddPack) and the next release (look out for it soon) include a version
specifically for editing IMS databases - something you won't find
anywhere else.
