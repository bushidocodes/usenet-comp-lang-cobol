Tom,

Assuming Win NT as OS you can't access memory allocated from another
program in that way. It's not "shared". NT always keeps a look at who
is the owner of data in mem. That's why you get an access denied error.

It is very easy to code something in that way to get access violation
errors. You will get the same result using "every" language.


HTH - Stefan Meyer
