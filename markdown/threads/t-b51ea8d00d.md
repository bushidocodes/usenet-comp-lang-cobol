```
OK, we've got a couple of tools:

1. A PowerCobol project to adjust MajorVersion, MinorVersion, and
ReleaseNumber to a bevy of PPJ files so they don't have to be opened
one-by-one simply to upgrade the version/release number. Maj.Min.Rel
numbers for 100 projects can be adjusted in ten seconds with no button
pushes or mouse-clicks. Credit to FJ technical support for providing
the clues for this technique.

2. Finally figured out the batch-compile gizmo so that one batch file
can compile 100 projects without intervention (this takes a bit longer
than ten seconds).

Used together, they provide a trouble-free and more efficient
mechanism to generate a new product release.

Email me if you want copies.

Jerry Peacock
```
