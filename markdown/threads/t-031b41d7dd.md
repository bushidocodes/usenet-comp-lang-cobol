to research what is best to buy.  Some people recommend books like learn
COBOL in 24 days or something similar which usually comes with a compiler
from Fujitsu or some other places.  If you are taking a first basic
programming class an older compiler may work fine.  I have used a compiler
from Merant (Microfocus) that was designed for Windows 3.1 and it seemed to
run fine on Windows 95, but I have never tried it on Windows 98.  I have not
seen any sites dedicated to how to program for beginners in COBOL.  I have
