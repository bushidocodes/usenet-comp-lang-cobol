› I need a YYMMDD to YYDDD conversion program in COBOL...Lost Mine!!!

John, get a current compiler. Any compiler that supports the ANSI '89
amendment to the COBOL '85 standard (for example, COBOL/370, COBOL for MVS
& VM, MicroFocus COBOL) includes the intrinsic functions, including the
functions "Integer-of-date" which converts YYYYMMDD (yes, four-digit
years, too) to an integer date; then "Day-of-Integer" converts integer to
YYYYDDD. Other functions available, too.
