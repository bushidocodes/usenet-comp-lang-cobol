It really doesn't have anything to do with your cobol
experience. To attach a cobol program or any other file to
an email you need to open your email program (i.e. netscape
or outlook, or other that you may use if yu are in a
different environment). type your message, then find the
attach option and search for your file then send the email.
thats it.

If you are working in a linux or unix environment from a
University that is text based, and no graphical user
interface it is still possible I'm sure but you would be
better off asking one of the network administrators how to
do it...
