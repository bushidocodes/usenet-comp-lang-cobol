I can't believe that I have to ask this, but here it is. We started
teaching COBOL using MF Workbench COBOL. The problem is getting
printouts. We are using the MF COBOL over a network, and it seems to
want to print directly to printers. When we "spool" the output to files
and then print them, they are some funky format that does not print
correctly: either from Notepad, Wordpad, Word, or anything else we have
tried.

And, why does Animator not have a "Print" command? The Personal version
does. Do professional programmers not have to printout programs or
output?

Has anyone figure out a simple solution to this problem?

Jack
York College
