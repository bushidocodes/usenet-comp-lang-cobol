Ryland,

There is nothing in the language do do what you require. However, you could
use the Win32 API 'Sleep' function which will sleep for a certain time
period but not until a specific time of day.

Alternatively , if you actually want to start the program at a particular
time of day you could use the 'AT' command on Windows NT.
