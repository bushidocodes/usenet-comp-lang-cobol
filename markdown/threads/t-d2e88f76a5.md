Hi there :-)

we had a assembler program which would NOT save register 10 and use it in
the program (apparently some trick from  the old days....) Under Cobol 1
this worked fine - under LE we crashed 0c4.... so if you have old assembler
code you might check if they save ALL registers and restore (like they
should in the first place).
