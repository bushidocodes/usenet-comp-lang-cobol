>We used to use Realia. Is it not true that Realia is no longer

Jerry:

This is not 100% correct.  The CA-Realia II Workbench is still
officially supported for the purpose of offloading development and/or
maintenance of mainframe COBOL applications.

Although I really wish that CA would change their position on
this.....and we have encouraged them to do so, they do not officially
support PC application development.

Page 1-1 of their Getting Started Guide states very clearly that:

"Usage of the product to create PC applications is possible but is
beyond the scope of the product....

...Computer Associates technical support will only be able to provide
limited support for creating PC-based application."

Personally, I think that the COBOL compiler in the Workbench is a
great tool for PC COBOL development.  The compiler is fast and
efficient and those who have used it generally love it.  If CA would
officially support PC development with the Workbench, I really believe
that more programmers would feel much more comfortable using it for
that purpose.


Bob Wolfe, flexus
