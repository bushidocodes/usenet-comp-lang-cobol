Indeed the Fujitsu file system was licensed from Liant.  However,
since that time Liant has made changes that are not reflected in the
Fujitsu version, and Fujitsu has made their own changes as well.  For
SOME things they may be compatible but this is by no means guarenteed.


Transoft corp has built an ODBC driver for Fujitsu COBOL format files
that may be available or may still be in testing.  It is expensive.

Fujitsu suggests licensing Btrieve and using that file system if ODBC
access is required.
