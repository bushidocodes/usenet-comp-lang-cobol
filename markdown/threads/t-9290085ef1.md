Hi Gene,

The OpenESQL Assistant will generate your Host Variable declarations
(DCLGENs).

You can do it as follows:-

- Start up the Assistant from the View menu option.
- Connect to the relevant data source and then build the required Query
- Go to the Auxiliary Code tab and select the Host Variable check box
- Go to your source code working storage
- Press the Insert Source code button on the assistant. This will create a
CPY with the DCLGENs and also put a EXEC INCLUDE in the source code.

You can also use the assistant to insert the SQL , CONNECTS code into your
program as well.

Hope this helps
David.
