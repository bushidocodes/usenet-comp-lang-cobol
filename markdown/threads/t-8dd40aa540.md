I don't know COBOL/400 specifically, but in COBOL-85 with the
intrinsic functions you write:
MOVE FUNCTION CURRENT-DATE TO identifier

This gives you the current date and time in the format
YYYYMMDDHHMMSSHH0000 with the various letters representing digits of
the year, month, day, hour, minutes, seconds, 1/100's of a second, and
the last 4 zeroes being reserved for future use.

The function result is alphanumeric so the move is from left to right.
If you define identifier as pic x(8), you'll get the date.

You can use reference modification, too. So for example if you wanted
to display just the hour you could write:
DISPLAY FUNCTION CURRENT-DATE(9:2)

George Trudeau
