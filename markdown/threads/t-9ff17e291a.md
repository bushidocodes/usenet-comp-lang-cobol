The leftmost bit, when = 1, is a flag indicating a 31-bit address. If
you want to get rid of it, use "LA  Rx,0(,Rx)". We aren't limited to PIC
S9(8) on Big Blue, try "S9(9) BINARY SYNC" - no truncation there. I
haven't had a problem manipulating 31-bit addresses with this
specification.
