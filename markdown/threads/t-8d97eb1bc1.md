Hi,

Deskware (www.deskware.com) has released "cobolscript"
which allows cobol programmers to write cgi programs. It runs on
win32, linux, and Sun, for real cross-platform flexibility. 

Pat Newberry, who does cobol  programming on mainframes,
has started a maillist for cobol programmers who want to  use their
cobol skills in cgi development. He is currently developing cgi
programs using cobolscript on a linux-apache platform.

If anyone is interested in this maillist, here is the url:
http://www.onelist.com/community/cobolscript

goshawk@gnat.net
