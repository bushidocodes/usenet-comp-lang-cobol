Usually specifying the file name as a literal in the ASSIGN statement 
causes the compiler to ignore the FILE-ID clause of the FD statement.
In this case the FILE-ID is treated as a comment.

You can change the ASSIGN to some generic value like DISK
or some compilers let you use an identifier in the ASSIGN statement
like SELECT PAYABLES ASSIGN TO PAY-FILE-NAME.

Good Luck.

Carefully review the manual for your compiler to determine how
it treats the values for the ASSIGN clause and the FILE-ID clause.
