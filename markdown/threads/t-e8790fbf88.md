Bill,

The answer is Yes.

In RM/COBOL one would code something like this in the FILE-CONTROL
paragraph:

    SELECT A-FILE ASSIGN "C:\PVSW\samples\sample.btr"
        ORGANIZATION INDEXED
        RECORD KEY <appropriate record key description>

and in the FILE SECTION:
    FD  A-FILE.
    01   A-RECORD.
         <appropriate record description>

and in the PROCEDURE DIVISION:
    OPEN I-O A-FILE.

Check Chapter 4 in the RM/COBOL User Guide:
http://www.liant.us/download/pdf/rmcug80.pdf
for additional information.
