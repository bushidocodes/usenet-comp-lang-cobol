Hi Michael,

Line sequential in COBOL Set for AIX and Micro Focus should be pretty much
the same. My suspicion is that your Java program might not be terminating
the record with the correct CR and/or LF character/s so the COBOL will then
fill the buffer with whatever comes after the end of your data and the size
of the buffer itself. The file will also need terminating with the correct
end-of-file marker also.

If you use Micro Focus Server Express 2.0 or later as your Micro Focus COBOL
then you could call a COBOL program from Java to write out the record for
you and then you would have no problem.

Regards,
Paul
