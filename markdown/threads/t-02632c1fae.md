What I did (on AIX) was to write a C-wrapper over the most basic socket
functions (definitely available on all UNIX flavors). These wrappers had two
purposes:
1) Their parameters were ones that COBOL could pass very "naturally" (e.g. a
couple of records)
2) They did some work that the sockets functions don't do (e.g. loop until a
message of a known length is fully received)

I found Stevens latest book on UNIX Network Programming very useful. It's
the one that describes the various socket calls and gives some tips on using
them.

- Darius Cooper
