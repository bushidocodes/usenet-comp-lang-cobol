```
It's a COBOL Function, viz:

COMPUTE NUM-DAYS =
    FUNCTION INTEGER-OF-DATE(END-DATE) -
    FUNCTION INTEGER-OF-DATE(BEG-DATE)

Where END-DATE and BEG-DATE are PIC 9(8) fields of containing YYYYMMDD
```
