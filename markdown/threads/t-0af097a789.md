```
Dear Andrew.

Currently there is no way to read or undeleted a deleted record from a
Vision file, as the runtime removed the record pointer from the Vision
header segment, and the space used for the record is marked as available, so
that the space can be reused.

Also, if newer records have been added since some records are marked for
deletion, the deleted record space could already be used up by the newer
records, which will mean that there is no way to get the deleted records
back.

So, unfortunately, this is not possible.
```

Technically it is feasible, but not every time.  A lot depends on whether or
not
the original record was compressed or variable-size.  If so, then some
information
about the record is permanently lost when it is deleted, making
reconstruction
not entirely automatic.  It is this reason why Acucorp doesn't provide a
tool to do so.

However, that being said, it is technically possible to do much of the time.
