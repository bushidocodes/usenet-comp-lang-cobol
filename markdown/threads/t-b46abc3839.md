I prefer to use the character based editor, and check/animate from
there.  This editor is editor.exe. Invoke it with the cobol source file
name on the commandline. Press F2 a couple of times to set the
check/animate options.  After checking/compiling, you can run the program
at the commandline by typing 'run <progname>' where progname is either the
.int or .gnt filename.

HTH
