It is to do with 16-bit co-existence.
Had this problem lots of times in Win 95.
Don't know Win ME yet, but this inelegant solution fixed most cases:
FIRST load another 16-bit application.
We set up WPS.EXE (Windows Process Status) to kick off (minimised) in the
Startup group.
(The R&D guys have now written some little 16-bit thing which has the same
effect but invisible to the user.)
