In our surveys of commercial IBM mainframe accounts, it appears that a typical COBOL population is in the 85 to 95% range of "application programs". This measurement is by logical program count, not lines of code. The measurement is based on analysis of compiled, executable code using the Edge Portfolio Analyzer (blatant commercial - see http://www.edge-information.com).

Sample size - 10 to 20 major commercail sites where Edge has done site-wide measurements of all application code executables.

Observations were made on 10 to 1000 libraries at each site, with a total of perhaps 1,000,000 application load modules studied.
