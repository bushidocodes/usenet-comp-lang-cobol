```
"COBOL Unleashed" is an intermediate to advanced book on programming
in COBOL.  Going much farther than simply covering COBOL, 'Unleashed'
is about 'how' to write good COBOL.  The author is Jon Wessler, but
much of the book was written by other experienced COBOL programmers,
several of whom frequent this newsgroup.  I wrote chapter 25, and the
sample program for that chapter covers most things you might want to
do with screens.  Chapter 26 covers GUI interfaces.  Unleashed comes
with a CD that contains all the source code in the book, plus a number
of third party tools.  You can get "COBOL Unleashed" from most good
book stores.  You can get it at www.bookpool.com for about $33 U.S.,
or www.amazon.com for about $40 U.S.

"COBOL Unleashed" is not a beginner's book, but for people who know
how to program in COBOL already, but want to know more.  If you want
a beginner's tutorial, try Thane Hubbell's excellent "Teach Yourself
COBOL in 24 Hours", also from Sams.  'TYC 24 Hours' covers screen
handling, but not in as much detail as 'Unleashed'.  'TYC 24 Hours'
is about $17 U.S. from www.bookpool.com.

Both books include the Fujitsu COBOL 3 compiler, and all the sample
code in 'TYC 24 Hours' is written using that compiler.
```
