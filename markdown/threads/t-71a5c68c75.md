```
I can only take a SWAG since I don't know what kind of compiler/runtime (if
any) you have, so you may need a different name on the call statement.

01  CHMOD-COMMAND  PIC X(??) VALUE "chmod 600 /bla/bla/bla/MyFile".

------------
    OPEN OUTPUT MY-FILE.
    CLOSE MY-FILE.
    CALL "SYSTEM" USING CHMOD-COMMAND.
    OPEN OUTPUT MY-FILE.

Then do a chmod 666 to your new file after you close the file (or 644 if
others can only read it).
```
