```
Some people talk about compiler listings as being irrelevant.  I like to
use them for a few features, even if I don't print them out:

1.  I want to get a list of all fields from within copy members.   Often
I will print these out.
2.  I want to see actual field sizes and displacements (would prefer
position).  I pick an option which will put these on the same line as
the field.  With my IBM compiler, this also adds a bunch of junk on the
bottom, which wastes paper when I occasionally print it out.
3.  I like options where I can find unused or unchanged fields.  This
can be useful when you cut and paste from program to program - find out
why that field is unused.
```
