Moreover, the claim "XML ... presumes a code set" is incorrect.  XML
permits explicitly declaring the encoding in the XML Declaration which
begins the document prolog, if present.  An XML document must contain a
prolog, which must contain an XML Declaration, which must contain an
encoding declaration, if the document does not begin with a UTF-16 Byte
Order Mark (BOM) and is not UTF-8 encoded.

Since the presence of a BOM is equivalent to explicitly declaring that
the document is UCS-2 encoded, the only encoding XML is willing to
assume is UTF-8.  Explicitly declaring the encoding is preferred.

See
<http://www.w3.org/TR/2000/REC-xml-20001006.html#NT-EncodingDecl>.
