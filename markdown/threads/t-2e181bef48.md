Maurizio,

I'll send you an example tomorrow. Meanwhile, did you get a copy of Will
(Wilson) Price's book "Elements of COBOL Web Programming with M/F N/E",
when you received your N/E package for V 3.1 ?  If you go to Page  295 -
Relational Databases and SQL - he gives you a pretty good intro to SQL. He
sticks to the basics, no reference to Normalization etc.

N/E's own on-line help book about SQL and DBs is a damned good reference
too - and again spells out ESQL Assistant.

At first glance it looks complicated, but when you get into ESQL Assistant
and get the hang of clicking on the appropriate buttons/icons, (there are
only THREE !) it does a LOT of work for you,  creating your copy books for
host, cobol and null variables, plus formatting your queries which you can
auto paste into your source code..

If you don't have Will's book, look at the N/E demos, they provide tables
set up using MS Access which you can use.

There are some sites you will find useful for general SQL syntax,
regardless of programming language. Although I've downloaded hard copies I
didn't make a note of the sites. (Do a web search using Web Ferret or
Google on "ESQL" or "SQL"). This one is interesting - the author is a movie
nut !

            http://www.fluffycat.com/sql/

Until tomorrow.
