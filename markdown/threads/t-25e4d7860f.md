```
Dear Augie.

It is possible to connect to two different Oracle databases being hosted on
different server machine.   All that needs to be done within the COBOL code,
is to ensure that the ORACLE_HOME, ORACLE_SID and LD_LIBRARY_PATH are set up
correctly prior to the opening of the file, and the Runtime will connect to
the correct Database on the correct Server machine.

Please do not hesitate to contact your local Acucorp office if you need any
further assistance in doing this.
```
