I see this all the time when I use the Template.  It puts the EXPLICIT
Path to your panel file in the template.  (I've modified mine not to
do that now!).  Anyway, check that in your source you don't have the
full path to the .pan file specified in the Open Panel call.  If
that's not it, sometimes I see this when I forget to include fonts.sp2
with the runtime.
