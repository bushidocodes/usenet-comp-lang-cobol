```
hello:

you need review a screen definition, if you give a name to each field,
you will be able read each field individualy, there are some examples to
do this in your samples.

example:

if you put this:

 SCREEN SECTION. 01 DISP-SCR BLANK SCREEN BACKGROUND-COLOR IS 7
 FOREGROUND-COLOR IS 0. 02 LINE 5 COLUMN 13 VALUE "ID-NUMBER:". 02
 ID-NUMBER-AREA LINE 5 COLUMN 25 FOREGROUND-COLOR IS 1 PIC X(6) TO
 ID-NUMBER AUTO.

the field ID-NUMBER-AREA is a -NAME- of field on screen, and you can
accept this field individualy
```
