› z

This is a file compressed by the UNIX `pack` program. To decompress it
on UNIX, type `unpack xx.z`. Note that uppercase .Z is different:
these are produced by `compress`, and are decompressed (on UNIX) with
`compress -d`.

› gz

This is a file compressed with GNU ZIP (`gzip`). It works a lot like
`compress`. To decompress it, type `gzip -d`. gzip can also
decompress .z and .Z files, is a free program, and is also available for
DOS and OS/2. One stop shopping.

› tar

This is an archive file that contains multiple files. On UNIX, you can
extract its contents with `tar -xvf filename`. Under DOS or OS/2, use
GNU tar (I renamed mine `tar`, so I don't know the original name, but
think it's something like `gtak`).

Both GNU programs (which are all you really need) are available on
internet via anonymous FTP. Use archie to find them. I know they
are on ftp.cdrom.com, and lots of other places. They are also certainly
available on AOL, and I know they are on Compuserve (use GO IBMFF).

None of these are actually text files. They are wrappers that can be
placed around any kind of program, including text files. You extract
the text (or other) file from these .z, .Z, .gz, and .TAR files with the
above tools.
