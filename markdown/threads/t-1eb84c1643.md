I work for a company who is at present converting DOS based COBOL code
into Windows based environment. We use MF COBOL and are using software
called DIALOG which is produced by MICROFOCUS and enables you to convert
the screen section of DOS code into pretty windows and all those flashy
windows boxes. All you do basicly is draw you boxes, etc add the screen
variable and remove you COBOL screen section. Then you replace all you
calls in COBOL ( DISPLAY, ACCEPT) with calls to DIALOG, EASY !
I hope this helps

Edd.
