- **From:** "Gazaloo" <gaz@home.com>

- **Message-ID:** `<Ybbv6.2496$ad1.229268@news1.frmt1.sfba.home.com>`

```
from John Billman, Net Express Product Manager at MERANT:

======

if you have Net Express V3.1 and you have a current maintenance contact,
this new feature pack is available for download at:

http://support.merant.com/websupport/websync/netx31.asp

white paper at:

http://www.merant.com/products/microfocus/product_focus/whitepapers/COBOL_En
terprise_JavaBeans.asp

highlights include:

� Creating COBOL-wrapped Enterprise JavaBeans (EJBs) has become easier.  Net
Express wizards now create not only source code but also an EJB 1.1
compliant JAR file.

� Using a new data interface type class more complex structures can be
implemented in the COBOL/Java interface reducing the need to change existing
COBOL code.

� The Net Express Unix Option now supports publishing COBOL/Java
applications to Server Express (you need Server Express V2.0.10 to execute
applcations using this enhanced functionality).
```
