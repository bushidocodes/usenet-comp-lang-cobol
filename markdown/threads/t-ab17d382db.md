Hello Andrew,

Sounds like a Micro Focus call-by-number routine to me. The MF COBOL/2 3.1
manual lists some call-by-number routines, but the first call-by-number
routine is X"84". Are you sure you mean X"81"? All call-by-number routines
have been replaced by call-by-name routines. Call-by-name routines are
easier to handle and remember.
Microsoft COBOL 3.0 (made by Micro Focus), discontinued a very long time
ago, supported additional call-by-number routines. Didn't find them in the
MS COBOL 3.0 manual though.

Hope this helps.
