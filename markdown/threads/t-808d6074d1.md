You should be able to find your previous version of winsock.dll in
/Windows/Sysbckup. Try recopying that version of winsock.dll back into
/Windows/System. Be forwarned though, you can't do this through Windows
Explorer.  You'll have to restart in Dos mode, go to /Windows/System and
rename the current winsock.dll to something else.  Then copy, from the DOS
prompt, the winsock.dll from /Windows/Sysbckup to /Windows/System.   I've
had similar problems (usually with Netware Clients) with programs replacing
my winsock and replacing the current version with the version in the
directory specified above almost always seems to solve the problem.
