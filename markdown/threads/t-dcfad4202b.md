Please help me!
I have written a program in Acucobol which is printing a form to a laser. It
is done in 3 passes using PCL to control the output. The first pass is
printing the layout of the form using line characters. The 2nd is printing
the box headings in a variety of fonts etc and the 3rd pass is printing the
actual data. The application is running on an AIX box and is used by a
number of remote sites using their own printers. 1 site is printing to the
laser from AIX via a Novell server, and it prints OK. Another site are
printing from the AIX to an HP laser directly, through a JetDirect card. The
output is slightly diffferent. The 2nd and 3rd passes seem to be padding the
output lines with spaces. If the print job is captured to disk and these
edited, the output is OK.
Has anybody got any ideas what I can do, without affecting the output on the
other printer. Is it a programmatic thing or is it a hardware issue. Any
guidance would be appreciated.
