Fileshare Version 2

Fileshare Version 2 is a client/server system designed to provide more
efficient processing of data files that are shared concurrently by a
large number of users.

In addition, Fileshare Version 2 offers features to file handling
applications that are not available with the base COBOL file handler.
These include transaction processing and rollforward recovery logging to
help ensure integrity of data held in files....
