Merant, previously known as Micro Focus, sells various compilers.  
Some versions of compilers are designed for the web and some are not.
Usually a free one or cheaper student edition does not make object code.
I think the Micro Focus Cobol for windows (Win3.1), which is the older 
version came in a student edition for about $49.00 and a version that 
could make Object code ran about $100.00 or so.  The newer compilers are
a little more expensive because they have ways to interact with the 
Application Proragrammer Interface(API) for windows and are designed with
internet applications in mind.

If you want to learn basic coding skills maybe try one of the Books like 
Learn COBOL in 24 Hours which comes with a Fujitsu student edition compiler.
Try looking in places like Amazon.com or other online book stores if you 
can not find the books in a mall bookstore.  I think it is easier to order
books online than to go to a mall.
