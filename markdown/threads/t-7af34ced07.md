There used to be (circa early 1970s) a batch program called "Autoflow", mfg by ADR,
which read a S/360 COBOL source program and produced a flowchart diagram on the
character-based line printer. Does this sound like what you want? AFAIK, ADR stood for
Applied Data Research. At the time, it ran on a predecessor to DOS/VSE. I don't
remember if MFT/MVT was supported or not.

I've asked recently in other newsgroups and received no responses. Autoflow seems to
have vanished. One hope is that someone who still has a copy would be willing to
"share". If you ever get a non-public response, perhaps you'd let me know. It would be
useful for year 2000 reconstruction.

Rgds,

Bill -- Boston
