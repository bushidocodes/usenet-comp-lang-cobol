I think the problem he's having (overlaying the entire file and not
just one record) has to do with REUSE/REPLACE. REUSE will overlay the
whole file, while REPLACE will add/overlay just the one record. WRITE
only causes a problem if you try to add a record that already exists,
and REWRITE only causes a problem if you try to overlay a record that
doesn't exist. Neither, I believe, causes the entire file to be
overwritten.
