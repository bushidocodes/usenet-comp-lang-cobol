Here are links for new 32-bit MinGW builds of GnuCOBOL 3.1-rc1 (Release
Candidate 1, 30JUN2020), using GMP 6.2.0 (Gnu Math Package), PDCurses
4.1.99 for extended screen I/O, and Berkeley DataBase 18.1.32 for ISAM
support. GMP 6.2.0 is configured with "fat binary" to optimize support
for various x86 processors.

I've also updated the build guide.

https://www.arnoldtrembley.com/GnuCOBOL-3.1-rc1-MinGW-Build-Guide-V7.pdf


There is also a 64-bit version of GnuCOBOL 3.1-rc1 for Windows:

https://www.arnoldtrembley.com/GC31B-64bit-rename-7z-to-exe.7z

https://www.arnoldtrembley.com/GnuCOBOL-3.1-MSYS2-64-bit-Build-Guide-draft.docx
