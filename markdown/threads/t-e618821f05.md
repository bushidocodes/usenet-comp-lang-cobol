```
What's happening is the cobol program thinks it's the main program
instead of a subroutine. You need to put some code in asm to first
inform cobol it's a subroutine and not a main program. This is the
way I've handled this in the past. If anyone has other approaches
please post:


L 15,ILBOSTP0
BALR 14,15
ILBOSTP0 DC V(ILBOSTP0)
ILBOSRV DC V(ILBOSRV)


Do this call ONE TIME ONLY before any cobol calls. It covers all
cobol subroutines and all calls. You never actually call ilbosrv,
it just needs to be linked into the load mod.
```
