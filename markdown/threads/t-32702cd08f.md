There is no such statement. Also, you cannot use "initialize" as a
procedure-name. Try the following:

WORKING-STORAGE SECTION.
01 PIC X VALUE "y".
88 more-input VALUE "y".
88 no-more-input VALUE "n".

PROCEDURE DIVISION.
startt.
PERFORM UNTIL no-more-input
READ data-file-in AT END
SET no-more-input TO TRUE
NOT AT END
MOVE etc.
WRITE output-line FROM detail-line
END-READ
END-PERFORM
...

This syntax is COBOL85 and works on any conforming compiler.
