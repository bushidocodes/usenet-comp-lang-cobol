```
A COBOL program can return a parameter using the linkage section.
See the CALL,  PROCEDURE DIVISION USING, and LINKAGE SECTION portions of
your manual for details on your system, but here's a C&D example:

PROGRAM-ID. FIRST.
..
...
PROCEDURE DIVISION.
    MOVE 9 TO A B C.
    DISPLAY A B C.
    CALL 'theprog' using a b c.
    DISPLAY A B C.
    STOP RUN.

...
END PROGRAM FIRST.
PROGRAM-ID. THEPROG.
...
LINKAGE SECTION.
 01   X     PIC 9.
 01   Y     PIC 9.
 01   Z     PIC 9.
..
  PROCEDURE DIVISION USING  x y z.
    MOVE 1 to X
    MOVE 2 to y
    move 3   to Z
    GOBACK.

END PROGRAM THEPROG.
```
