```
Just as a "follow-up" on another thread,

In a "conforming" 2002 ANSI/ISO COBOL compiler, the following are BOTH
valid:

05  Switch-Field  Pic 1.
     88   True-Value 1 When False 0.

 05 Flag-Field    Pic X.
     88   Type-One   Value "A" "B" "C"  "D" when false high-value.
     88   Type-Two   Value "1" "2" "3" when false high-value.

I have never (personally) cared much about the "flag" versus "switch"
distinction, but with the introduction of boolean fields and boolean
operators, the 2002 Standard allows (but doesn't force) a programmer to make
such distinctions.
```
