problems:

1) The only time that you PERFORM
     F16-CHECK-COURSE
is in the middle of
    F15-CONTINUE-ADDING.
*but* you have COMMENTED out that perform statement - so it will NEVER get
executed

     F10-NEW-RECORD.
seems "odd" to me - in that you OPEN this file for EVERY write statement
(and do NOT close it after each record).  You are not (yet) checking the
file status codes after all of your I/O statements so the system MIGHT be
trying to "tell" you something that you aren't listening to.
