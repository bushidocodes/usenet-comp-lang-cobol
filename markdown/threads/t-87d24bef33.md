It is a fairly simply thing to do, but can be a very printer-dependent
operation. I don't have a laser jet book showing all the escape/font
codes, so the example I have shown is for a an IBM proprinter (DOT matrix
printer).

In working-storage, define the printer-code:

01 FONT-IDS.
03 FONT-COMPRESSED PIC X VALUE X"0F".

Prior to writing any printout, issue:

WRITE PRINT-LINE FROM FONT-COMPRESSED AFTER ADVANCING 0 LINES.

X"0F" will turn on the printer's compressed font. You might try this
example on your HP Laserjet. Some of the printer codes might be
somewhat standard. However, a lot are not. That's exactly why there
are hundreds of priter drivers for the software you may purchase.

Mike Dodas
