Jaye,

I use DEC COBOL on a VMS system. The manual quite clearly indicates that
reference modification can be used on a display item - so yes you can
use FIELD-A(3:1) because PIC 9(09) is a display item (but if you had
PIC 9(09) COMP it won't get past the compiler because PIC 9(09) COMP
is NOT a DISPLAY item).
I tried it in a program and the snag is that the term FIELD-A(3:1) is
alphanumeric not numeric
    and so  ADD   1  TO FIELD-A(3:1)  will not compile
    BUT     MOVE "7" TO FIELD-A(3:1)  will compile and execute as expected.

Hope this is of interest - but I am not really sure why you would want to
be doing such a curious thing. :-)

Good programming,
Brad Henry.
