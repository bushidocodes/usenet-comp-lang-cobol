Re: printer not printing.

I think I have this nailed down, but I do not like the answer
much.

It appears that V4 cannot write generic ASCII to a printer.
A print file *must* be associated with a font, and that font
*must* be available for the printer that you are using. That
also means that you *must* have the full blown, up-to-date
windows driver for the printer ... there is no "generic ASCII"
anymore.  When I try to do things with everything set up that
way, *and* the assignment is to "printer" *and* the printer that I
want is the windows default printer, then the program works. Use
any "fontless" printer, and you get an error reading the font
file.

I have listed the test program below, BTW.  There are a whole
slew of problems with this.  First of all, the default is about point
6 text, so you also have to set the font to get a legible print
out.  Secondly, the driver inserts a form-feed on close, like
it or lump it.  When you are printing single page invoices, this
creates a problem in that not all printers can be set up for the
correct page size. One is forced to manually adjust the paper
after each page in this case.

Lastly, I cannot get it to work with *anything* in the assignment
statement except "PRINTER" .  In the case of a second printer
there seems to be no way to print.  The documented PRINTER-2
and "LPT2:" all give a runtime error of "unsupported", as does the
assignment to a network printer.  The only way to change printers
seems to be to exit, and change the windows default.

I'll tell you a funny story about such patches.  In the late seventies
or early eighties, I had a microprocessor based concrete plant.

a car battery), and connected them back to back to give me
plus/minus 12.  I then grabbed a handful of diodes, and soldered
then end to end (each one will drop the voltage about 1.1 Volts)
and onto the plus end to give me plus five.  I hooked it up,
turned it one, and told them to call when the power supply came
in.

Three years later, I got the next service call.  Turned out the
batteries were dead.  By that time the power supply had come,
gone, and was no longer being made.  We changed the
batteries, and it ran like that until 1992, with BI-yearly
maintenance.

two gold mines (they blended ore for uniformity, and it froze

standard, red, number eight wire.  He phoned the surface,
