In the IBM Mainframe world, JCL stands for "Job Control Language". It's
an interpreted language for running programs in batch jobs. It is
roughly equivalent to using ".BAT" files in MS-DOS. The JCL for IBM VSE
is different from the JCL for IBM MVS/ESA.

I suppose JCL would be equivalent to shell scripts in Unix, but I don't
have enough Unix experience to say that with certainty.
