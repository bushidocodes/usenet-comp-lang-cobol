```
I am using Fujitsu COBOL v6.  The following subprogram, when called,
displays some simple demonstration data from a line sequential
file into the console window.  Is there some way to close the console
window after (or before) the "program exit" statement is executed and
control is returned to the calling program?  

The calling program uses a COBOL screen to control a menu (and in a later
version of the program, to gather some input data).  The length of the
input file is unknown.

Thank you for any advice.

Floyd Johnson


*>===============================================================

identification division.
program-id.                DisplayResults is initial.

environment division.
configuration section.
source-computer.           ibm-pc.
object-computer.           ibm-pc.

input-output section.
file-control.
    select datafile assign to "currdata.dat"
       organization is line sequential
       access mode is sequential
       file status is inp-file-status.
       
data division.
file section.
fd datafile.
01 input-record                     pic x(5).

working-storage section.
01 temp                             pic x(5).
01 EOF-flag                         pic x
               value "N".
01 inp-file-status                  pic xx value "00".
    88 file-success      value "00" thru "09".               

procedure division.
display-paragraph.
   *> Open the data file
   open input datafile
   
   *> Do an initial read
   perform read-record-paragraph
   
   *> Process the record until EOF
   perform until EOF-flag = "Y"

      *> Display the record
      move input-record to temp
      display temp
  
      *> Read the next record   
      perform read-record-paragraph
   end-perform
   
   *> Close the file 
   *>    and return to the calling program
   close datafile
   accept temp from console
   exit program
   .      

read-record-paragraph.
   read datafile
      at end move "Y" to EOF-flag
   .    

*>===============================================================
```
