```
G'Day,

I am currently learning COBOL (only just started) and wish to know how to
use the DISPLAY statement to cause text to be displayed at the start of a
blank screen when I want it to and also how to advance it as many lines as
I want before displaying.

I want to achieve the same results on screen as I would using the following
code involving the printer: "WRITE what-ever AFTER ADVANCING PAGE"
or "WRITE what-ever AFTER ADVANCING 2 LINES"

I notice that I can't just swap the word DISPLAY with the word WRITE in the
above examples as it wont work.

I am using COBOL-85.

Please send replies by EMAIL ONLY.

Thanks in advance,

Brendan Farthing.
```

```
Two ways to do this: (1) adjust the position on the screen by means of
DISPLAY SPACES repeated as necessary to produce the proper amount
of blank lines, or (2) define a FILE assigned to the screen and use WRITE...
AFTER ADVANCING... as needed.
```
