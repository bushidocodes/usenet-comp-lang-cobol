As far as the '85 Standards is concerned, this is NOT a "J4 decision" but
has to do with ANSI *claiming* that they have a copyright (which is dubious
at best - if you read the acknowledgement, know its history, and have a
basic understanding of copyright law).  The '74 Standard was written with a
type-writer (as I recall) - so it is hard to get it electronically (unless
someone wanted to "scan" it - and then you would run into the copyright
issue again).

The draft of the next Standard *was* available electronically.  I think (and
this may be mis-stating the real issue) that there are ISO (not ANSI) rules
about "drafts" between official CD (Committee Drafts) that are impacting J4
and WG4 on their NOT making the current version available electronically.  I
think (but won't swear to this) that you can still get hard-copy versions by
contacting the J4 chair at

   doncobol <at> mediaone.net
