```
Tony B. Shepherd (tbs@iii1.iii.net) wrote:
: And now for a trickier question: how can one tell if the program is 
: executing on an Ascii or Ebcdic machine at execution time?

Under ASCII, "0" has value 48, while "A" has value 65.  Under EBCDIC,
"0" has value 240, while "A" has value 193. So just compare:

    02  CHARACTER-0       PIC X   VALUE "0".

    IF CHARACTER-0 < "A"
        we are on an ASCII system
    ELSE
        we are on an EBCDIC system
    .
```
