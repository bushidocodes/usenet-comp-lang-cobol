Hi Bob,
Windows 95 is not a true multi-tasking system, it only multi-tasks if the
programs have be written to behave and occasionally give control back to
the system - the system cannot pre-empt a running program. Your first
recourse is to use -- which should bring up a task menu.
Choose the one you wish to terminate and click on END TASK. If this does
not work you are left with the choice of rebooting the system and the
subsequent loss of all running programs and any open files.

Good luck,
Jim
