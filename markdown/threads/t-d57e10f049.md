COBOL is typically oriented towards 80-column cards. It has a very specific
line format:

columns 1-6 Optional sequence numbers
column 7 Is the "indicator area". An asterisk "*" indicates that the
line is a comment. A hyphen could appear here to indicate
the continuation of a non-numeric literal.
columns 8-11 Area-A. Division headers, section headers, paragraph names,
FD's, and 01's all begin in Area-A.
Columns 12-72 Area-B. Everything that cannot begin in Area-A.
Columns 73-80 Optional program identification. Some installations use line
numbers here, and some echo the program-name here.

TLINK should work for linking the .OBJ files.
