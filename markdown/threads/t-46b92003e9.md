If you have Micro Focus COBOL, you can call the CODESET program (the source is
supplied in $COBDIR/src (on Unix, not sure about other platforms, but it will
be there) so you can alter the mapping tables for any local quirks). Note that
the altered source, if compiled and added to the product, will also affect the
COBOL syntax for CODESET IS EBCDIC etc, as this is the support program that is
used at runtime to convert data.
