I have used the following on a dEC ALPHA but since it is in
Don Nelson's book I presume it is implemented on all  '85
CObOL . I determined what the actual length of the search string
was at run time. I only needed to know if the search string occurred
at least once within the target string: set the tally 
counter to zero before the INSPECT and check it afterwards.

The length of the search string depended on how much of the vendor's
name the operator had to work with. My code handled embedded spaces
as well.


===========================================================
+                                                         +
+ INSPECT target-string TALLYING                          +
+         tally-counter FOR                               +
+         search-string . . . (see reference manual for   +
+                              the charming details)      + 
+                                                         + 
+                                                         +
===========================================================

I hope this helps,

rrg

here be some snippets of code (if it looks like a C programmer doing his
first COBOL program...well, it isn't... it was my 3rd COBOL program! and 
no, there is no amount of money that will persuade me to show anything
of #1 or #2): 

========================= am Anfang =======================================
1400-FETCH-BY-NAME.
     MOVE SPACE TO W2-SRCH-EMPLOYEE-NAME.
     MOVE "NAME" TO SRCH-KEY.
     DISPLAY "PLEASE ENTER THE NAME:       ".
     PERFORM 1500-GET-THE-SEARCH-KEY.
     MOVE "N" TO DONE-FLAG.
     PERFORM 1600-SRCH-ON-KEY VARYING TST-IX FROM 1 BY 1 UNTIL DONE.
     IF FOUND
      PERFORM 1410-GET-BY-NAME.
     IF FOUND AND
        GOOD-READ-KEY
          PERFORM 1420-DISPLAY-THE-RECORD.

1500-GET-THE-SEARCH-KEY.
     ACCEPT  SRCH-VALUE FROM SYS$INPUT.
     INSPECT SRCH-VALUE  CONVERTING LOWER-ALPHA TO UPPER-ALPHA.
     MOVE    "N" TO DONE-FLAG.
******now determine the length of the search string ********************
     PERFORM 1510-GET-SRCH-VALUE-LENGTH VARYING J FROM SRCH-VALUE-SIZE BY
                                   -1 UNTIL DONE.      *************** 
     ADD 1 TO J.                                       max possible size 
                                                        of search string     
     MOVE J TO SRCH-VALUE-LENGTH.
               *****************
               actual length of search string

*this starts at the end of input search string variable and moves towards
*the non-space character that is at the end of the actual search string
1510-GET-SRCH-VALUE-LENGTH.
     EVALUATE SRCH-VALUE(J:1)
       WHEN NOT " " MOVE "Y" TO DONE-FLAG
     END-EVALUATE.



1600-SRCH-ON-KEY.
i     IF TST-IX > TST-IX-MAX
g        MOVE "Y" TO DONE-FLAG
n        MOVE "N" TO MATCH-FLAG
o        DISPLAY "END OF TABLE REACHED WITHOUT FINDING THIS ENTRY:",SRCH-VALUE
r        DISPLAY "PRESS <RETURN> TO CONTIUE ... "
e        ACCEPT DUMMY-PROMPT FROM SYS$INPUT.


*now go and do the search ***
     IF NOT-DONE
       MOVE "N" TO MATCH-FLAG
       MOVE 0 TO SUBSTRING-INDICATOR
                  *set tally counter to zero before search*
       EVALUATE SRCH-KEY
         WHEN "NAME"  *when we are searching on a name*
                     
                     target string            shows # of substring
                                              occurrences i.e tally cntr
                     ****************         *******************
             INSPECT TST-NAME(TST-IX) TALLYING SUBSTRING-INDICATOR FOR ALL
                                            SRCH-VALUE(1:SRCH-VALUE-LENGTH)
                                            **********   *****************
                                           search string var   this is J
                                           ********************************
                                           together these constitute the
                                           embedded string searched for

i         WHEN "SSN "
g             INSPECT TST-TID (TST-IX) TALLYING SUBSTRING-INDICATOR FOR
n                                    ALL SRCH-VALUE(1:SRCH-VALUE-LENGTH)
o       END-EVALUATE.
r
e

  ELSE
