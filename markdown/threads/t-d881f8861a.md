This really isn't a COBOL question, but it is a MicroFocus question, so
hopefully nobody will mind.

Actually, not even a very important question, but is there any way to turn
off the bell/beeps in the editor? I am using the DOS version, and whenever

beep, beep" for a few seconds.

Thanks for any help to this minor question. :-)
