Use GetKeyState() or GetKeyBoardState() to query the value of VK_NUMLOCK.
You can't use SetKeyboardState() toggle the keyboard, only the state of the
keyboard for the current thread. You'll have to simulate a keypress using
keybd_event().

- Karl Wagner
