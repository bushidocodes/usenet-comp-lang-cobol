You can use the SIZE option to limit the length of the display.

Examples:

DISPLAY FIELD-A AT 0160 SIZE 20.

or

DISPLAY FIELD-A LINE 01 POSITION 60 SIZE 20.

This wil limit the display to 20 characters.
It works in RM/COBOL and MF-COBOL.
