Just a quick note to announce an important update to our
class "Enterprise COBOL: Unicode and XML Support".

With the release of V3R3 of the compiler two weeks ago, you can now
generate XML from a COBOL program using the XML GENERATE statement.
This completes the package for creating COBOL wrapper programs.

That is, you can now do the following all in native COBOL:

* Accept XML input transaction
[* Convert from UNICODE if necessary]
[* Convert from ASCII if necessary]
* use XML PARSE to convert the XML transaction
  to classic COBOL record layout
* call existing COBOL transaction program
  passing the converted record layout
  [this is the program being wrapped]
* get response from transaction program
  in classic record layout
* convert response to XML format using
  XML GENERATE
[* Convert to ASCII if necessary]
[* Convert to Unicode if necessary]
* Transmit XML response to original source

Our course has been updated to include the XML GENERATE piece of the puzzle.
