David:
This isn't a problem is you're using Acucobol 3.0 GT. You have access
to all of the printers in Windows including any fax drivers. You simply
do as any other program does: select the FAX as your printer and then
print. It goes to the fax driver. The cblconfig entry would be:

-P spooler

going to the win95 print spooler.


Peter Withey
Acucobol Technical Support
