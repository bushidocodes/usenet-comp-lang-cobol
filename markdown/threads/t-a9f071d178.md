```
You can input 'compiler directives' into Microsoft COBOL 4.5 (a
Microsoft modified and relabeled version of Micro Focus COBOL 2.x)
several ways, but the simplest way to get a listing is through the
compiler command line.

    COBOL [source-file-name[,[obj-file-name][,[source-list-name]
        [,[object-list-name]]] [directives][;]

Assuming your program source name is PROGRAM.COB, and you want to
create object PROGRAM.OBJ and list file PROGRAM.LST, you can use
the syntax:

    COBOL PROGRAM.COB,PROGRAM.OBJ,PROGRAM.LST;

Object-list-name is the name of the object listing file.  If you want
one, then append that too.  Specifying the source listing name will
create a listing file.  But if you have $NOLIST embedded in your
source file, the listing will be turned off at that point.  You can
turn on the listing again by using $LIST.
```
