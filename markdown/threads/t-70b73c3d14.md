      ***********************
      *peekmessage

      *BOOL return value is 16bit

      *msg structure
      *since peekmessage with no remove, just a place to dump
      *temp info

      *filterlast and first are two message boundaries.

      *msg is far pointer

       02 no-remove    pic xx  value   h"0".
       02 remove-pos   pic xx  value   h"1".

       02 msg-structure.
           05 hwd  pic xx  comp-x.
      *    HWD 16bit value

           05 message  pic xx  comp-x.
           05 wparam   pic xx  comp-x.
           05 lparam   pic x(4)    comp-x.
           05 time     pic x(4)    comp-x.
           05 point.
      *        x and y are type int
               10 x    pic xx  comp-x.
               10 y    pic xx  comp-x.
