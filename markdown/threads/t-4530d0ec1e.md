```
No, you do not have to change anything. In fact,
if you recompile, there should be no change in
the object produced.

With a large system, in fact, the standard way to find out
what has to be changed is to simply change a data name
and recompile everything. The syntax errors will then point
you to all the places code has to be checked if you change
a field size.
```
