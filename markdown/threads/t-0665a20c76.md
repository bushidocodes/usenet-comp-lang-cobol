```
April 18, 2001

This is a repost of something I posted about six months ago. Those who work
in the mainframe environment and are responsible for defining and allocating
VSAM datasets may find this of interest.

In approximately October, 1999, I asked if anyone could suggest a 'VSAM for
Dummies'- type tool to enable a calculation of the space allocations which
should be used during an IDCAMS DEFINE of a VSAM dataset. No one could, so I
decided to write one myself. Thanks to several people who helped me with
this project, that software is now available.

In October, 2000, I released Version 1.1 of "VSAMSpace." VSAMSpace is a
single 55K Windows(r) 32-bit (NT/9x/2K) executable program file used to
calculate the cylinder or track allocations required for various types of
VSAM datasets under various conditions.

The user selects the type of file (KSDS, RRDS or ESDS), the allocation unit
(cylinders or tracks), CISIZE, CI% and CA%, the record length and a couple
of other parameters, clicks the "Space" button and the software returns a
little message of the form:


ESDS on 3390
CISZ= 512  CI: 10 CA: 10
Var  LRECL= 200/400
Number of Records:   10,000
Requires      8  Cylinders

        OR

KSDS on 3390
CISZ= 512  CI: 15 CA: 10
Var  LRECL= 200/400 Key  24
Number of Records:    5,000
Requires     59  Tracks

Click the "Copy" button to copy this data to the Windows clipboard, and do
what you will with it. (This is how I got the output into this note). Change
parameters, click "Space" again and the space is recalculated; you can see
how CISIZE and other parameters affect your DASD space allocation
requirements.

When you click the "Doc" pushbutton, it extracts the doc file to a text file
in the same directory where the executable is and launches NOTEPAD to view
it. All the arithmetic of "how space is calculated" is in the documentation.

Operation of the software is so simple even a mainframer can handle it.

VSAMSpace does not require any support DLLs, does not read from or write to
the system registry or do any other annoying stuff. To install it, you "save
e-mail attachment." To run it, you click on it twice in Windows Explorer. To
un-install, you delete the file. (Although I doubt anyone would EVER
de-install it!).

And, the best thing about this software - at least insofar as I can tell
about the BBS/Usenet crowd - is that the price is right: free.

If you'd like a copy of this software, just send an e-mail request to
michael.mattias@gte.net . The request should have a valid "reply to" address
which I do not need to edit. I can attach the file to the email as an EXE or
as a ZIP file (about 25K) if you need to get past a firewall or just feel
better about it that way; so let me know if you you want the software
attached as "EXE" or "ZIP." (Default shipping format is ZIP format).

Note: For those who already have a copy of this software, this is NOT a new
version. You can check your version using the "properties" button against
your VsamSpace.exe file using Windows Explorer. The current version is
1.1.0.
```
