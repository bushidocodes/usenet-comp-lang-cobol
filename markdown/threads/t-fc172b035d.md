Don't know how Fujitsu COBOL 6.0 does its thing (I never intend to
install the thing, even though I have a license), but if it's like
Net Express, you only get the 30 days the first time you install it
on a machine.  If you have ever installed and uninstalled Net Express
on a machine, even an earlier release, you get a 'license expired'
message if you install it again later, until you get a new response
key.  This, I believe, is inexcusable.  If you install a later version
on a machine after having uninstalled an earlier version, is it not
obvious that you aren't trying to simply extend the original 30 day
trial period, but are at least evaluating the new version?  It also
means that you cannot use a given machine for backup purposes more
than once, unless you completely reformat the hard drive.  Since few
of us have an unlimited number of backup computers, and none of us
want (or probably have the time to) format the hard drives and reload
all the software regularly on our backup machines, this severely
limits the 30 day trial period as a backup tool, in real practice.
