I have had no problem using these routines, though you should upgrade them
before using.  There is an update of Fujitsu's website.

To use them, set up your project to include the library
C:\FSC\COBOL97\F3BICBLR.LIB. (I am presuming a standard installation). The
LIB file will link your calls to the appropriate DLL.
