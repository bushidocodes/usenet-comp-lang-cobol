```
Steve,

There should be manuals at your workplace that explain all of these
software products. The manuals could be online, either in
BookManager, or in some other online product. Ask one of your
co-workers.

Definitions:
MVS - The IBM large computer operating system
CICS - One of the transaction processing systems that facilitate
application programs interacting with the computer.
JCL - Job Control Language. The language for submitting batch
work to the computer.
IMS DC - Another transaction processing system that facilitates
application programs interacing with the computer.
IMS DB - A hierarchical data base system.
DB2 - A relational data base system.
Xpeditor - A third-party COBOL and Assembler animator debugging tool.
FileAid - A third party file access tool. Matches files with COBOL
data description copybooks. Allows easier access to VSAM and IMS
files that would otherwise be possible.
Workbench - Don't know. Could be several products.

Hope this helps,
Dilbert
```

```
MVS - Multiple Virtual Storage - refers to an IBM operating system for the
large mainframe computers.

CICS - Customer Information Computer System - IBM's interactive online
transaction processor

JCL - Job Control Language - How you tell an IBM computer what programs
and utilities you're running.

IMS - Information Management System - a Heirarchecal (sp) Data Base
Management System

DB2 - Database 2 - Relational Database Management System

Xpediter - a package that allows interactive debugging of programs.

Fileaid - a package that allows the editing of files (very large files,
VSAM files, etc)

Workbench - A MicroFocus product that allows mainframe development to take
place on the PC.
```
