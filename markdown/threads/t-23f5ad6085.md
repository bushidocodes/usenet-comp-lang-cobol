```
The latest draft (and it is STILL A DRAFT) of the COBOL Technical Report adding
"native language support" to COBOL for XML is available for review at:

     http://www.cobolportal.com/j4/files/04-0008.doc

This document will be discussed at the J4 meeting on Tues, Jan 27.

If you have any comments that you can send (via email) to the J4 chair at:

   Don.Schricker <at> microfocus.com

before then, I would think that they would be interested in input.

I should note that this document is written as a "change" to the 2002 Standard,
so MUCH of it is pretty hard to follow.  HOWEVER, if you do a "find" on
"CONCEPTS" and read that section, it will show you the direction they are going.

This "project" is on a (relatively) fast track (or as fast as ANSI, ISO, et al
can go), so I would think that any and all comments that you can get to the
Standards groups SOONER than later would be appreciated.
```
