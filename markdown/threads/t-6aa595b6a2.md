I am not a c programmer, so I don't have all the information you request,
but....

struct                            = Group item
double                          =  COMP-2  (assuming your COBOL uses COMP-2
for IEEE standard 32-bit float)
char  anything[number]  =   PIC X(number) or maybe PIC X(number +1)
long                              =  32 bit signed binary integer, COMP-4
(implemenation dependent)
int                                 =  EITHER 16-bit or 32-bit (depends on
the compiler and /or environment) signed binary integer, COMP-4
double[number]              =   USAGE COMP-2 OCCURS number  (or maybe OCCURS
number+1)


c stdcall for  int myfunction ( INPUT *in, OUTPUT *out)  expects two
paramerters, each a pointer to a structure; parameters pushed on the stack
right to left. The function returns an int value which, as per above, is
either 16 or 32 bits.
