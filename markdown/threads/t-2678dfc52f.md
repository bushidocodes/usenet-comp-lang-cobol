If you are compiling in, or writing a file to, the root directory,
it is possible that the root directory is full.  Root directories of
all hard drives, floppy disks, zip disks, etc., are finite, the number
of file *names* that it can contain is fixed and determined at format
time.

P.S.- If you have put so many files in the root directory of your
hard drive that you have filled it up, then God help you, for either
you don't have a clue about subdirectories, or have a sense of order
better suited to a packrat, and everything you own is probably lying
on the floor at home. ;-)
