```
It is a separate product you need to pay for. 

When you install the development tools you automaticaly get the
Multi-threaded runtime.  When you DEPLOY, however, the multi-threaded
runtime is not included.  You need to pay for it separately.  The MOST
you pay is $40.00 per machine, and the price goes down from there
based on volume.  (See http://www.adtools.com for pricing info).

When I inquired about it, Fujitsu told me there is no reason to use
the MTS (Multi-Threaded System) unless you are actually doing
Multi-threaded programming using OO COBOL.  If you are, you must
install the MTS on the target system or your code will not work.
```
