MicroFocus systems usually require a runtime in order to execute
programs. This runtime is licenced for a fee. There were some earlier
compilers (MS-DOS, OS/2 and Win3.x and Win95) that did allow an
executable to be built that could be distributed without requiring a
runtime licence. Alternately the Microsoft compiler, a version of
MicroFocus, could be used to produce executables for MS-DOS and Win3.x
without requiring a runtime licence.

I would suggest only distributing source code or using OpenCobol or
CobolIT to distribute executables.
