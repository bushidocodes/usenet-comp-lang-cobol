I suspect that this is an F3xxxxxx.DLL file and you can edit and
compile but cannot run your program.  USUALLY this is because the
Fujitsu installer with v3 has a difficult time amending the path
statement in your autoexec.bat or your autoexec.bat is not running.

Please tell us which OS you are running and we will try to help.

Bottom line, if you took the defaults on install, you need to get:

C:\FSC\PCOBOL32 into your path.
