As far as I've heard, the __concept__ of viruses originated on mainframes
in the 1970s when computer games were written to be used between programmers
during off hours. Once PCs started proliferating, these nasty programs were
redesigned to be run on PCs in great numbers. Unlike mainframes, which are
pretty much shut up in computer rooms where no one can get to them, PCs are
everywhere, and the assembly language that can be used to create virus
programs, is easily available.
