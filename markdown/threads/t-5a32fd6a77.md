It is not entirely clear from your message, but if you are trying to link with an
import library, the command line looks something like:

cbllink -v -opgm.exe pgm.obj a.lib

If this is what you want to do, you do not "package .OBJ modules into .LIB file". You
want to create a .DLL by using the -D option with cbllink; the import library will be
created for you, but you need to specify the -K option; otherwise, it will be deleted
when cbllink is done.
