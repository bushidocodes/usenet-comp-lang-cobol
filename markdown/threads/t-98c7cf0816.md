That's a good point, Jerry. I have "Base versions"; clean Virtual Machines 
with Windows XP and Windows 7 on them. When I want a new VM, I either clone 
an existing VM or build what I need using a snapshot from a base machine. I 
have never actually "transferred" the existing real environment into a VM 
client. I can say that the reliability is outstanding and I have never had a 
VM collapse or be unrecoverable. And they all are recognised on the wireless 
LAN.

I think it would be possible to take the real Boot partition and copy it to 
a VM that had nothing on it. This would achieve what you are asking, but I 
have never tried it.

Given that I have never paid a penny for any of this software, I reckon it 
is a pretty good deal.
