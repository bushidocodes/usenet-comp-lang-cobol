"ricky" <rveach@comcast.net> wrote in message news:<iw7e9.480721$2p2.19029157@bin4.nnrp.aus1.giganews.com>...
> Are there any good, free Cobol compilers that has a GUI for making window
> based programs?

Yes! (Despite the response from Jimmy, who apparently slipped a
cog...<G>)

You can get Fujitsu powerCOBOL from http://www.adtools.com

I started COBOL development for Windows using MF DCOBOL and DIALOG but
switched to PowerCOBOL and have used it ever since.

It is an IDEAL COBOL tool for GUI deveopment.

You build your GUIs using drag and drop, specify your event processing
in COBOL, your main processing in COBOL (if any) and then compile to
either .EXE or .DLL. It is simple, quick, and fun. You do not need a
lot of knowledge of the Windows environment but if you have such
knowledge you CAN use it, with API calls etc. all part of the standard
compilation (no need to address separate libraries (you simply drop
them on your project)and grapple with the compiler environment.)

Highly recommended.

Pete.
