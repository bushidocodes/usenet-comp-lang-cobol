Well I always use MS DOS editor!

But the twist is to program in 2 Windows MSDOS sessions:


Open two DOS windows, One in the COBOL directory and the other where you
want to save.
After an edit, merely switch (Alt Shift) to the 'run' window, compile and
run prog.
Then switchback to edit is necessary.
This way you don't have to keep opening and closing editors.

Regards


Ian
