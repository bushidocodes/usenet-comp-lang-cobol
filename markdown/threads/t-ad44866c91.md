Windows NT comes with Winsock. This is the Windows socket interface and
application programming interface. This allows you to write TCP/IP
applications for Windows without worrying about any low-level TCP/IP
details. Winsock is provided as a .DLL and thus you can make calls to it's
functions from within you COBOL programs. There are several good books and
Web sites dedicated to Winsock programming. Check'em out!

John
