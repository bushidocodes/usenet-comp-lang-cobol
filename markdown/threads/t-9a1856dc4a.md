› For a LINK, the address of curent commarea is passed, and both

Incorrect.
If A links to B and A is AMODE 31, while B is AMODE 24, then CICS creates a copy of A's commarea in an area below the 16MB line and will
pass the address of this copy to program B. When B returns, then CICS again copies the commarea back to the area above the 16 MB line

Also: "address space" is not the quite correct term, as all of this should happen in the same address space, unless DPL is involved. In
this case, XCTL is not allowed, and the commarea is physically copied under all circumstances. You probably meant "copied to a new area",
though
