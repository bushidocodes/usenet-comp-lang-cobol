Bob,

NetExpress contains a copy file called WINDOWS.CPY which contains the
definitions of the Windows data types and constants needed for API
programming. To code the example you mentioned you would need something like
:

SPECIAL-NAMES.
    CALL-CONVENTION 74 IS WINAPI.

01 nDspElements int VALUE 1.
01 DspElements.
    03 FILLER         int VALUE COLOR-BACKGROUND.
01 RgbValues.
    03 FILLER         COLORREF VALUE h"00bbggrr".

* replace the 'bbggrr' with the hex values you want. Black will be zeroes.

CALL WINAPI 'SetSysColors' USING BY VALUE nDspElements
                                                                 BY
REFERENCE DspElements
                                                                 BY
REFERENCE RgbValues

Hopefully that should give you a start point. Note that I put this together
from looking briefly at the Win32 SDK, so I can't guarantee that it is
accurate.
