```
Can somebody direct me to some online documentation on COBOL syntax
rules, limits, tokenizing etc. I am trying to put together a COBOL
interpretor to be used for educational purposes (including my own
COBOL education).

I have started with a WORKING STORAGE class in C++, which can also be
used standalone to scan COBOL source files and process COBOL data
files, but I need a lot more technical information. I have decided to
be lenient on myself and not specify that the interpretor produce
object code, it must just do whatever the COBOL program would do,
allowing the student to step/trace through it.

Basically, I need to know what I should display if a PIC clause is
incorrect, or if an invalid verb is encountered etc.
```
