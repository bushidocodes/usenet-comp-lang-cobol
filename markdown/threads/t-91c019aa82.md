File status codes in the range of 00 thru 89 have standardized definitons
that should hold true across various 1985 standard COBOL implementations.


The base document is the ANSI / ISO COBOL standard, which shows the
standardized definitions for code values in this range. Code values 90
thru 99 are reserved for vendor defined conditions.

Code 47 is defined as

"4" - Logic Error

"7" - READ issued with the file not open for INPUT or I-O.

Typically an error of this type relates to failure to issue the OPEN at
all, or more likely, an OPEN which failed.

Suggestion, check file status code after open, should be 00, if not fix
that problem first, the 47 will likely go away.
