I have asked Fujitsu the same question twice.

The first time was some months ago when we first received version 4.0 and I
was told that the ODBC driver for use with Crystal Reports should be ready
by about Dec '98 and got the impression it was being developed by Fujitsu
themselves.

When I enquired again in late November I was told -

"Crystal Reports access to Fujitsu COBOL files is still be developed. The
support is directly to Crystal Reports. We are not building an ODBC
interface.

The current plans call for the tools to be release in February/March."

Regards, Robert Kovacic.
