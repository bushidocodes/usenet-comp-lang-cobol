SPECIAL-NAMES.
     CALL-CONVENTION 74 IS WIN32API.

* Sleep for 1 second
CALL WIN32API "Sleep" USING
    BY VALUE 1000 size 4
END-CALL.

Relevant compiler directives are "CASE" and "LITLINK".

Link using  -- cbllink  -L -R4.0 -S -omyProg.exe myProg.obj
kernel32.lib

If you're planning on waiting for a long time you may be better off
using WaitForMultipleObjects() rather than Sleep, so you have a
mechanism for waking up the sleeping task.
