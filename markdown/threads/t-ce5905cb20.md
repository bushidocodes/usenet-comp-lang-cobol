```
Hi all,

we need to call a legacy COBOL program on an IBM/390 mainframe from a
C++ code on Solaris. The COBOL program takes two string parameters,
the first one is an input parameter, the second one is an output
parameter - the C/C++ signature would be: void unxxx(const char* in,
char* out)

I wonder what is the standard way to do this? Are there some
recommended products to do this? I think that every solution will
involve some programming on both sides (mainframe, Solaris), the ideal
solution for us would be one with minimal programming on the mainframe
side.

Thanks for your suggestions!

Dusan
```
