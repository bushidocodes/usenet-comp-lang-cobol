A generic parser like ParseRat (http://www.guysoftware.com/parserat.htm) might be able 
to help you convert to something else.  If your data is 2-digit-years it can convert to 
4-digit-years based on a user defined "window".  e.g. you can tell it that any 2 
digit year below 20 is post 2000, so  09 gets converted to 2009 but 21 gets converted to 
1921.
