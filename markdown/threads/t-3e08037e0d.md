Maybe you forgot to put the record under the FD entry. As in:
data division.
File section.
FD my-file
data record is my-record.

No record here????!!!

Compiler gets confused...


Or maybe your select/assign statement is pointing at the wrong file:
select myfile assign to thefile.

The file should be the name of the ms-dos file depending on your
compiler directive on how to resolve file names. Maybe it is looking
for a file named "thefile" or maybe it is looking for an environment
variable: Set thefile c:\thefile.txt.
