The problem with encryption is that, except under very rare 
circumstances, what is encoded can be decoded.  We routinely get 
databases in from the field to test problems - and, we have a 
"scrambler" routine that actually de-links records, and zeroes out any 
SSN's that might be there.

If the test file would still work if everyone had the same SSN, that's 
what I'd do - make a copy of the file, then change all of them to zeroes 
(or all 9's, or 123456789, or whatever).
