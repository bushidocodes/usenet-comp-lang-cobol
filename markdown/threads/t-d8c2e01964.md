```
A) "COBOL 1" is an (incredibly) inaccurate name for IBM's OS/VS COBOL
product.  In that context, "COBOL 2" usually means "VS COBOL II" (a product
that IBM no longer supports).  The current IBM COBOL (mainframe) compiler is
IBM COBOL for OS/390 & VM V2R2.

B) It sounds like you haven't used the IBM "migration guide" which is ALWAYS
a good place to start when doing such conversions.  I suggest you look at:
  http://publib.boulder.ibm.com/cgi-bin/bookmgr/BOOKS/IGYMG202/CCONTENTS

C) for this specific issue, look at:
 http://publib.boulder.ibm.com/cgi-bin/bookmgr/BOOKS/igymg202/4.1.6?

and look for the topic, "Relation conditions" which includes discussion of
this "undocumented extension" and some others, e.g.
    = TO
    > (<) THEN
    > (<) THAN
```
