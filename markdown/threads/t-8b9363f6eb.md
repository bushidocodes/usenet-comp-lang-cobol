Hi All,

The department I support is having it's files moved from a Novell
NetWare 4.1 server to a NetWare 5.1 server, and we are encountering MF
COBOL file status error 9/027 (Device unavailable) when attempting to
access our Btrieve master files. The programs were developed with
Micro Focus COBOL 3.1 for DOS.

After various experiments we have ruled out file path and directory
permissions problems. Is anyone aware of file access problems related
to a Novell 5.1 migration, and any fixes/work-arounds. (We haven't
done any new COBOL development for several years, so don't have a
support contract with Merant/Micro Focus, and their website won't
accept trouble reports without an existing contract).

Thanks!
Pamela
