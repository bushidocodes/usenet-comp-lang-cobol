line sequential file made with notepad, I get an Open error acc-method

Thats a known problem to be fixed in the next release.  The Fujitsu RTS is
treating it as Unicode instead of ASCII.  In the meantime you can either use
the StreamReader class or use Notepad to change the encoding.
