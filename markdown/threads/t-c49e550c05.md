```
Announce: TinyCOBOL release 0.53

The TinyCOBOL compiler version 0.53 is available at the TinyCOBOL home
page (1) and iBiblio (2).

This release contains mainly bugs fixes, and enhancements.  

It includes updates to the preprocessor, math verbs, the regression test
suite, gdb debug stabs,  STRING/UNSTRING statements, ACCEPT from
COMMAND-LINE and ENVIRONMENT-VARIABLE statements. 

Support for the dynamic CALL statement, ACCEPT  DAY, DAY-OF-WEEK, ON
EXCEPTION and ERASE clauses, ADD and SUBTRACT (CORRESPONDING option)
statements, EXTERNAL clause in the SELECT statement, BINARY-CHAR, 
BINARY-SHORT, BINARY-LONG, BINARY-DOUBLE and COMP-1/COMP-2 (preliminary)
floating point types, support for the Debian package format have been
added to the compiler.

Please report any bugs and/or problems to the TinyCOBOL mailing list. 


1) TinyCOBOL Home Page
   http://tiny-cobol.sourceforge.net/

2) iBiblio (formerly Metalab)
   http://www.ibiblio.org/pub/Linux/devel/lang/cobol/
```
