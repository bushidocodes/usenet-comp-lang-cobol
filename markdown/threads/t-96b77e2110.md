```
Visual Cobol by fujitsu is free.  I love it.  It makes applications that
have a beautiful windows interface.  The design method is just like
microsoft's visual basic.  You just click and drag buttons, list box's,
labels, edit boxes, etc.  But it has many more features than that, including
SQL, DDE, OLE, Animation, etc.  Very powerful.  You can design gui apps and
compile them (make executables) on the pc, then take the same code, and port
it to AIX, HPUX, Solaris, for a nice gui app there...  This product is
incredible.  After you click and drag your items to the sheet, you then
right click them and set each items procedure using COBOL !!!!  It's
awesome!
```

```
using the free compiler as it is an academic licence.
```
