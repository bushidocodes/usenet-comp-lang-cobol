I've been assigned the task of adding email address to an on-line
system. Input will be via CICS and VS Cobol II release 3.2. As I
understand it, Cobol II (Cobol 85 in my textbook) can handle case using
'ALPHABETIC-LOWER' and 'ALPHABETIC-UPPER' so storing lower case
shouldn't be a problem. The question I have is will CICS convert lower
to upper case as the map is received? Likewise, if the data is stored
in lower case, will CICS convert it to upper case when the map is sent
to the terminal? Any help will be much appreciated.

Thx, Kent

You can use or avoid the Upercase convertion in CICS using the "UCtran"
parameter in the Typeterm definition. This convertion is only done in
one direction, at the input and refers to a terminal associated with
this definition. You should talk to your CICS manager in order to now
how the terminal definitions in your site are.
