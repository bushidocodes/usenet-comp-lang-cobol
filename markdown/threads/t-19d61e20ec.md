If all you want is to know whether or not the string is present,
you can use something like this:
========================================
01 plen PIC 9999 COMP VALUE 0.
01 thekeyword.
05 kchar PIC X occurs 1 to 70 depending on plen indexed by px.
01 in-area PIC X(70).
01 junk PIC X(70) VALUE SPACES.
....
move 'car' to thekeyword.
move 3 to plen.
unstring in-area
delimited thekeyword
into junk.
if junk not = in-area
display "yes, string is present".
========================================
Now, if you want to know the string's position,
then you must do further processing to figure it out.

Lots of different techniques work....
