       01 name2                         pic x(22).
       01 p1                            pic x(4) comp-5.
       01 p2                            pic x(4) comp-5.

*         05 pic x(30) value "04,Line Heater/Furnace   ,06".
*         name1         "Line Heater/Furnace   "
*         name2         "Line &Heater/Furnace  "
*                        0........1.........2..
*                        1...567..0.........0.2

       Linkage section.
       01 n                             pic x(4) comp-5.
       01 lnk-text                      pic x(22).

       Procedure Division using     n
                          returning lnk-text.

        move wt-descrip(n)   to name1
        move wt-ampersand(n) to ampersandCount
        evaluate true
           when ampersandCount = 1
                move ampersandChar to name2(1:1)
                move name1         to name2(2: )
           when other
                subtract 1 from ampersandCount giving p1
                add 1      to   ampersandCount giving p2
                move name1(1:p1)            to name2
                move ampersandChar          to name2(ampersandCount:1)
                move name1(ampersandCount:) to name2(p2: )
        End-evaluate
        move name2 to lnk-text
       End Method "get-menu-line".

         move wt-descrip(lnk-code) to lnk-name
       End Method "get-typename-text".

       Method-id. "get-typename-string".
       *>--------------------------------------------------------------
       Local-storage section.
       01 ls-length                         pic x(4) comp-5.
       01 ls-text                           pic x(22).
       Linkage section.
       01 lnk-code                          pic 9(02).
       01 lnk-string                        object reference.

         move wt-descrip(lnk-code) to ls-text

            using ls-length, ls-text returning lnk-string
       End Method "get-typename-string".

       *>--------------------------------------------------------------
       Procedure Division.
         invoke os-Popup "showat" using ws-x ws-y
       End Method "show-popup".
       *>--------------------------------------------------------------

       End OBJECT.
       End CLASS VesselTypes.

Peter,

You need to use the "popupmnu" GUI class for the popup menu
then, when your end user right clicks of left clicks the text field
a callback is invoked to execute the relevent commands held within the
popup menu.


You need to use the window event      p2ce-button3down
to trigger the event. for example:

          move p2ce-button3down to EventValue
          move "mouseclickedC " to methodName
          invoke EntryCallback "new" using methodname
              returning aCallback
          invoke oTextEditField "setEvent" using EventValue aCallback

You would then have a method called "mouseclickedC "
which would create and show the popup menu.


To add items after you have created an instance use:

           move "Display Setup" to ws-text
           move length of ws-text to StringLength

           invoke characterArray "WithLengthValue" using StringLength
                                              ws-text
                                              returning aString


           invoke opopup "addItem" using aString returning bString

Without going to OO, you can achieve the result by placing the following 
code in each window's dialog
 MOUSE-EVENT
     IFNOT= $EVENT-DATA 7 QUIT
     INVOKE-CONTEXT-MENU $WINDOW CTX-MENU $NULL
   @CTX-FIELD-HELP
     MOVE "HF" COMMAND-FLAG
     RETC
(ctx-field-help is assigned to the popup item "Field Help")

and placing code such as this in your event loop back in your program
           EVALUATE COMMAND-FLAG
               WHEN "HF"
                    PERFORM 80-HELP-FIELD
           END-EVALUATE

and the help-field paragraph
       80-HELP-FIELD.
           MOVE 0 TO ON-LINE-CONTEXT-NO
           EVALUATE DS-OBJECT-NAME
               WHEN "EF-ID"
                    MOVE 1 TO ON-LINE-CONTEXT-NO
           END-EVALUATE
           IF ON-LINE-CONTEXT-NO NOT = 0
              CALL "ON-LINE" USING ON-LINE-PB
                                   ON-LINE-TOPIC
              END-CALL
              IF ON-LINE-RETURN NOT = 0
                 DISPLAY "on-line error: " ON-LINE-RETURN
              END-IF
           END-IF

If you need a special menu for a field, just put the mouse-event in that 
field's dialog but let field-help roll-up to the ctx-field-help in the 
window's dialog.

Hope this helps. I tried to keep it short. If it was too short, email me 
at rayleech1@home.com without the number 1.
