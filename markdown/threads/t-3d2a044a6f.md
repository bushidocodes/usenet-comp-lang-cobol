```
To All,

I have written a Window's program that can be very usefull if you are
a programmer. "2-Do Plus" is designed to make it easy for you to
track tasks you are working on and tasks you have completed. It
consolidates a to-do list, solutions list, client-history list, and
time-worked list, into a simple to use Windows application. You can
track tasks by priority, client, or start date. The tasks are divided
into a Current list (to-do), and a history list. They can be moved
from current to history, and vice versa. The program is file driven,
so you can have multiple task files. Also, each employee can have
their own task files. You can further break the task files down by
month and year, or have one for each month of the year. ShareWare

I wrote the application to allow you to use it in any way you wish,
thus not tying you into rigid rules for tracking tasks.

You can download it from my web page at:

http://shoga.wwa.com/~pdubois/webpages/examples.html

If you beta test it for me, feel free to keep it at no charge. All I
ask is that you contact me with any bugs reports or suggestions.

Thanks.

Paul
```
