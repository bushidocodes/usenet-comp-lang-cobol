```
>I know I need a compiler but what else?

If you bought COBOL for Dummies you already have several COBOL compilers
and interpreters on the CD which comes with the book.  Beyond that, you
need knowledge and understanding.  Study to get the knowledge, practice
to get the understanding.  If you have specific questions, ask them here.

If you want a good COBOL tutorial, get "Teach Yourself COBOL in 24 Hours"
by Thane Hubbell.  $20 US from www.amazon.com.  It includes Fujitsu COBOL
3 on the CD, and all the example code compiles and runs using it.  If you
don't like it, you can complain to the author, who often frequents this
newsgroup. :-)
```
