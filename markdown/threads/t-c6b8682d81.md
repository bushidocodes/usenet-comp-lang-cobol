Do you have any experience with compilers at all, or you a complete
neophyte?  There are usually some examples on a CD of that sort.
The norm is to find a program called "setup" or "install", then start
working your way through the examples.

If you have only used a language like Basic, then the first thing you are
going to have to get the hang of is the structure.  Cobol is a language
of it's own.  You write a program using a text editor, then compile
and link it ... do you understand those concepts?
