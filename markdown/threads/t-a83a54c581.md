Paulo,

This is what I came up with for the compile command:

The line for compiling is:

      xm cobol program_name, gnt_name, list_name, NUL

	prgram_name is the name of the program you are compiling

	gnt_name is the name of the object file created

	list_name is the name of the error listing created

	The last parameter is unkown to me for its use.

So for you to compile the program emp0000 try:

  xm cobol emp0000, emp0000.gnt, emp0000.lst, NUL

This should get you on the track,

Paul
