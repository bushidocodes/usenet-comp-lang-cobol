One answer is "enough so that picking up yet another language isn't too
hard".

A better answer is "enough so that you can readily distinguish between
programming concepts and their implementations in different languages"(and
as a result, find it easier to pick up new languages - it's just a matter
of figuring out how the new language implements the programming concepts
you already know, and then picking up any new ones it introduces).  Alas,
not everyone who learns several languages gets that far.  Hence the old
jokes about people who can "write Fortan in any language".
