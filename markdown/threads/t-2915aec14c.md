You do not specify which Micro Focus COBOL product you are using,  Server
Express or Object COBOL Developers Suite

In the docs for Micro FOcus Object COBOL Developers suite the cobcall()
function is documented as follows:

long cobcall(name, argc, argv)

indicating that cobcall() returns a long integer

The Server Express docs use Micro Focus data prototypes to document the same
function

cobrtncode_t cobcall (const cobchar_t *name, int argc,  cobchar_t **argv);

The definiution for cobrtncode_t can be found in the
$COBDIR/include/cobcall.h
