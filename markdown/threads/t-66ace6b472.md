You need to pass the structure by its address; a "C" pointer:  In
COBOL, you specify this using BY REFERENCE.  Values are passed BY
CONTENT, like this:

       CALL "cfunc2" USING BY REFERENCE  ESTRUCTURA
                                            BY CONTENT      counter
                                            RETURNING Return-Pointer

Since COBOL passes BY REFERENCE by default, your counter was not being
passed correctly; rather, COBOL was passing a pointer to the variable
"counter".

There are generally a couple of other minor things concerning the
format of calls to C from COBOL, and they should be in your COBOL
manual; they vary by compiler.
