I have been told by some MF internals that MF is working on a Linux
version of its MF suite for Unix. No date of issue has been planed to
the moment.

You can run the binary of your code under Linux with the IBCS
compatibility option. I think you have to relink two SCO libraries
(tcp/ip? I can't remember) under Linux in order to make it run
properly.

Anyway, I didn't try the procedure myself, but I have been confirmed
that it works. Of course, without support from MF...
