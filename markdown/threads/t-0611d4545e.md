Given the info above, that's a tough question to answer. If you
purchased the educational compiler and runtime, you can certainly
use it personally for any development project you are asked to do. If
you are asking if you can give a copy of the runtime away to someone
who isn't licensed to use it, then the answer is no. If you are asking
if you can include a copy of the runtime with your assignment so your
instructor can see if the program really runs, then there are two
possibilities:

1. Your instructor already has a license because he owns an educational
development system (presumably he owns the textbook)--no problem.
2. Your instructor doesn't have a license--just don't use the runtime
until he/she gives it back with your grade. :-)

BTW, this isn't the final word, as I don't know exactly how the
educational license agreement is written (it was probably written
by the textbook publisher). Read the fine print--nobody here can
give you permission to violate a contract.
