```
Well the smallest, fastest and simplest would be NotePad.  But I don't 
think that you will like it, you probably need to add more criteria such as 
syntax highlighting, auto-indenting, block indenting, key style you would 
be happy with.

Editors are something that you have to train yourself to fit, or vice 
versa. You need to get several and see which you like and which you can 
make do what you want.

I use TSE (www.semware.com) because I have made it fit my needs with syntax 
highlighting, tabbing, and key short cuts (eg F9+P -> PERFORM).

On Linux I use SETEdit (setedit.sourceforge.net) which is free and I can 
get used to it.  There is a version for DOS which will run in Windows.
```
