```
If you're using MF COBOL, use the run time routine x"91" function 69 to scan
the directory. loop around keeping a count until you get a 'No more files'
status returned. It takes wildcards too so you can do things like just look
for all the *.CBL files for instance if you want.

It's documented in the online help and the manual.
```

```
Here is a code segmrnt that does the trick.

01 cntr pic ....
01 result.
05 f-error pic x comp-x.
05 f-handle pic x(2) comp-x.
05 f-attrout pic x comp-x.
05 f-time pic x(2) comp-x.
05 f-date pic x(2) comp-x.
05 f-size pic x(4) comp-x.
05 f-fileout pic x(80).
01 function-code pic x comp-x value 69.
01 parm.
05 f-action pic x comp-x.
05 f-attrin pic x comp-x.
05 f-filein pic x(80).
procedure division.
move "*.*" to f-filein
move zero to f-action
f-attrin
cntr
call x"91" using result function-code parm
move 1 to f-action

perform until f-error not = zero
add 1 to cntr
call x"91" using result function-code parm
end-perform
```
