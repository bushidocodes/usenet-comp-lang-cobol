Yes Micro Focus Server Express 1.0 does support files > 4 gig. One needs to
set up a configuration file extfh.cfg.

A quote from the File Handler Documentation book from Section 6.2.1.1

"The configuration option you need to set in order to create
and access a large file is the FILEMAXSIZE option. This option works for
sequential, line sequential, relative and indexed files"

Chapter 6 in the file handler book tells you all about how to set up the
configuration file.

(I've never actually done this myself, I'm just quoting the documentation).

Steve.
