```
Assuming you have a "current" COBOL compiler (with Intrinsic Functions)

A) Move your input field to a YYYYMMDD field
B) Convert this date via FUNCTION INTEGER-OF-DATE
C) Subtract 60 from that result
D) Convert this integer via FUNCTION DATE-OF-INTEGER

(Some of those steps can be "combined" in a single COMPUTE statement)
```
