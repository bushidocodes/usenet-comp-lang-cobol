I imagine that the "IMS" in the "COBOL IMS" refers to IBM's hierarchacle
(sp?) database "IMS". Two types exist: 1) IMS/DB, which is the database
and 2) IMS/DC which is the communications product.

Both run on the mainframe, so comparing them to the PC is a little
tricky.
