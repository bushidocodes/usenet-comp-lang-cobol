SINGLE (like DOUBLE) is a floating point representation.  In the draft of
the next standard there is a data usage called FLOAT-SHORT which is the
same as SINGLE, which has been implemented in some newer compilers.  There
is no floating point in the current (1985) standard but many compilers
implement floating point under the name COMP-n (where n is an integer thru
9 which varied from one implementation to the next).  What compiler are
you using?
