```
Creating the dll should be relatively simple. Write your code as
though it is going to be statically linked. At compile time, compile
all of the programs into OBJs as usual. Next, link the proposed DLL,
but instead of specifying a filename with an EXE extention, use a DLL
extention. The linker will then create a DLL as an output file.
Then, compile your EXE as normal (remember not to link in the OBJ that
you created the DLL from). MicroFocus cobol does not require that you
use the .LIB file that corresponds to the DLL, but your compiler
might.

Good luck.
```
