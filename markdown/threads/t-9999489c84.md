If you are using an up-to-date compiler, use the intrinsic functions that
are
designed to do just that. First, you need a 4-digit year. To convert
from
gregorian to julian you do:

COMPUTE jul = FUNCTION DAY-OF-INTEGER (FUNCTION INTEGER-OF-DATE
(greg))

The other way:

COMPUTE greg = FUNCTION DATE-OF-INTEGER (FUNCTION INTEGER-OF-DAY
(jul))

Simple.

Don Nelson
