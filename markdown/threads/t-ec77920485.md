It does seem to me that some programmers endlessly write the same
program over and over, with only a few differences (such as file and
report layout), and have done so for decades.

Personally I avoid that as much as possible.  For example I have one
program that prints most reports using a script file that specifies the
file(s) (accessed via a called module into a standard linkage area), the
selection criteria (based on dictionary names that decode to field
