```
Over the last few months there have been several threads about what you can
and cannot do with the Fujitsu V4 editor.  I think the following note that I
received from their support may help a bit.

"The editor of COBOL V4 has two modes for the search command. One is the V3
mode, the other is the V4 mode. The V4 mode can search for dashes, asterisks
and spaces without tabs.

If you had installed COBOL V3 into your PC before you installed V4, the
search command is the V3 mode. Of course, you can exchange the V3 mode to
the V4 by the following command.
 [Options]menu of Editor
 -> [Set Enviroment for Editor (Basic Status)]command
 -> [Use Find/Replace Commands Of PowerGEM Plus V3 Version] option

When you install COBOL V4 into PC at the first time, the search command is
the V4 mode.

The editor and the builder of COBOL are components of the PowerGEM Plus."
```
