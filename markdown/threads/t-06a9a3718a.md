```
Hi Andy,

I use Fujitsu-Cobol too.

The Child-Window can be called using the Opensheet command as
described in the manual or the demos.
the control is beeing returned back to the calling sheet even if the
child window is not closed.

When you want to transfer data between the two windows you have to use
the external clause in the data division where the items have to be
the same.
the best way is to use a counter (describe in the data division as
external in both sheets) in the close-window procedure of the calling
sheet.

Example:

sheet1 (calling sheet)
Data division
01 childready pic 9 global exernal.

procedure division:
move 0 to childready
call opensheet of sheet1 using "sheet2"

close-window procedure:
...
if childready = 1
do something
end-if

sheet2 (the called sheet)
data division:
01 childready pic 9 global exernal.

when to close sheet:
move 1 to childready

hope that helps.
```
