› The problem is that I DON'T have an instructor! Anyone knows the

You received a correct answer to this quest ion from Del Archer, viz:
SELECT fd-name ASSIGN "dos-pathname" ...
The educational version of RM/COBOL does not do long file names.

An alternative would be:
SELECT fd-name ASSIGN file-name-var ...

where file-name-var is defined in WORKING-STORAGE:

01 file-name-var pic x(60) value "C:\DIR\FILE.TXT".

This allows the ability to change the pathname of the file connection at
runtime before the file is opened.
