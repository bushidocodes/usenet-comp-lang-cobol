COMPUTE HALF-TEST = TESTNUM / 2.
PERFORM VARYING NUM FROM 3 BY 2 UNTIL NUM > HALF-TEST
     IF MOD (TESTNUM NUM)  = 0

        END-IF
   END-PERFORM.
