```
Hi, All,

The original release of GUI ScreenIO lacked support for fonts and
colors.  We'd planned for this (as well as many other features) but
did not have time to get it into the initial release.

We just posted a new release to our web site that provides font and
color support; you can easily set the font and/or colors (and other
things) of your screen objects by clicking the right mouse button
while pointing at the object in the editor.  We've also enhanced a
number of other things.

If you're converting a legacy ScreenIO application, you can either
choose to convert your color scheme or to use the default Windows
color scheme for your converted panels.  (Sometimes the old legacy
color schemes don't look as "Windows" as they should.)

You can get more information and download GUI ScreenIO for a 45 day
free trial from:

http://www.norcomsoftware.com/newsoftware/GUI_ScreenIO/GUI_ScreenIO.htm

Questions and comments are welcome.

Thank you.

Kevin
NORCOM COBOL Programming Tools
GUI ScreenIO; a cool COBOL/Windows Screen Manager
http://www.ScreenIO.com
```
