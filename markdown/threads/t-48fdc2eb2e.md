```
I have several versions of Microfocus Cobol which require the use of a
security key.
Unfortunately, it is rather old and grey and has the following raised
markings :

    SOFTWARE
    SENTINEL-C TM
    RAINBOW
    TECHNOLOGIES, INC.
    IRVINE CA

I have tried to get this key to work using various Key enablers supplied by
Rainbow without success.  This may be due to the key being broken, but I
suspect that it is the key enabling software which no longer supports keys
of this type.  When I used keys like these in the good 'ole days, I remember
using a command called INSTKEY to install it.  Does anybody on the planet
have an old disk or a functioning version of MF Cobol which uses this type
of key and knows anything about how it the key was installed ?

This is my last effort to get it to work before I throw the lot in the bin
and get to grips with the Fujitsu alternative that everybody keeps
mentioning.

Please send replies / attachments to justinroberts@btinternet.com

Thank you.
```
