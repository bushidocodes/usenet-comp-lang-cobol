Not that we're short of work, but we have had an ongoing project to Migrate
PowerCOBOL projects to .NET.

The screens would be converted to .Net Forms (and maintained using the
design surface in Visual Studio) and the existing event scriptlets would be
hooked back to the forms as code-behind.

The main advantages in doing this would be:

1. You are no longer locked in to a system which has no future and support
for which could be withdrawn at any moment.
2. The code-behind doesn't HAVE to be in COBOL, so if you wanted to
gradually migrate your system to a .NET language, you could. (New controls
and event processing can be added to forms in C# or VB.Net without having to
convert or rewrite the existing COBOL scriptlets. Or, you can continue using
native COBOL.
3. It would be a simple matter to web-enable the forms if that was required.
(It is NOT a simle matter to do this with PowerCOBOL screens).

I spent some time considering this problem a while back and I did some
experiments and tests which persuaded me that, although non-trivial, it is
viable to do this. But the project got pushed down the priority stack. There
is a fair bit of work in it and it isn't something I want PRIMA to undertake
unless there is a chance of people using it.

I'd like to make a decision as to whether we will do this or not, and remove
it from the priority stack if it isn't worth doing.

So, the question comes down to: "Is anybody still using PowerCOBOL?"

We have a couple of sites who have expressed interest, but it would be good
to get a feel for how much PowerCOBOL is still out there.

If you are currently using PowerCOBOL please drop a mail to
in··o@pri··o.nz, with the subject: PowerCOBOL. You won't be
spammed or have your email address misused.

PRIMA already provides a very useful tool for PowerCOBOL which allows you to
batch compile PowerCOBOL (and NetCOBOL) projects, and it multithreads so you
can see results as they happen.
(http://primacomputing.co.nz/PRIMAMetro/PwrCobBatchCompile.aspx)

I'll make a free copy of this tool available from a draw of the people who
respond by the 1st of December, 2013.
