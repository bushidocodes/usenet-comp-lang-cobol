I did not find ShellExecuteEx, but here is the setup for ShellExecuteA


      * ShellExecute

       01  HANDLE-TO-PARENT-WINDOW             USAGE HWND.

       01  lpOperation.
           03  lpOperation-Data                PIC X(20).
           03    FILLER                        PIC X(01)  VALUE C-NULL.

       01  lpFile.
           03  lpFile-Data                     PIC X(20).
           03    FILLER                        PIC X(01)  VALUE C-NULL.

       01  lpParameters.
           03  lpParameters-Data               PIC X(20).
           03    FILLER                        PIC X(01)  VALUE C-NULL.

       01  lpDirectory.
           03  lpDirectory-Data                PIC X(20).
           03    FILLER                        PIC X(01)  VALUE C-NULL.

       01  nShowCmd                            uns-int.



           CALL "ShellExecuteA" using
                            BY VALUE hWnd-HANDLE
                            BY REFERENCE lpOperation
                            BY REFERENCE lpFile
                            BY REFERENCE lpParameters
                            BY REFERENCE lpDirectory
                            BY VALUE nShowCmd
                RETURNING RETURN-VALUE.

      * ShellExecute.

       ESEGUI-SHELLEXECUTE.
           MOVE LOW-VALUE TO WS-ShellEx-Info.
           MOVE LENGTH OF WS-ShellEx-Info TO WS-ShellEx-cb.

           MOVE SEE-MASK-FLAG-DDEWAIT TO WS-ShellEx-fmask.
           MOVE 0                     TO WS-ShellEx-hwnd.
           SET WS-ShellEx-Comando TO ADDRESS OF WK-COMANDO.
           SET WS-ShellEx-FileName TO ADDRESS OF WK-FILENAME.
           SET WS-ShellEx-Parametri TO NULL.
           SET WS-ShellEx-Directory TO NULL.
           MOVE SW-SHOWMINIMIZED       TO WS-ShellEx-Show.
           MOVE 0                      TO WS-ShellEx-Histance.
      *Opzionali
           SET WS-ShellEx-IDList       TO NULL.
           SET WS-ShellEx-Class        TO NULL.
           MOVE 0                      TO WS-ShellEx-KeyClass
                                          WS-ShellEx-HotKey
                                          WS-ShellEx-Icon
                                          WS-ShellEx-Process.


      *
           call WinAPI32 "ShellExecuteExA" using
                by reference WS-ShellEx-Info
                returning WS-ShellEx-Return-Code
           end-call.
      *
           IF WS-ShellEx-Return-Code = ZERO
              PERFORM Get-API-Error  THRU Get-API-Error-Exit
              MOVE  LS-Shell-Return-Code    TO WS-Disp-Code
              DISPLAY WS-Disp-Error
           END-IF.
      *
       FINE-TRAS.
           STOP RUN.

      *
           CALL WinAPI32 "GetLastError"
              RETURNING LS-Shell-Return-Code
           END-CALL.
      *
       Get-API-Error-Exit.
