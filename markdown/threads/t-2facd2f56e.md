I'm trying to preserve some of the history of IBM 1401. I'm hoping to find
Autocoder, Cobol, Fortran, RPG, Sort 6, Sort 7, IOCS, ... tapes or decks,
manuals, user-developed applications, the engineering drawings that CE's
used for field maintenance, or any other IBM 1401 memorabilia (including

Do you have any of this stuff, or know anybody who might?

My list of manuals is http://math.jpl.nasa.gov/~vsnyder/1401/1401-docs.html.
Do you have any manuals that I don't have, or newer editions of the ones I

was output from the assembler.  It has images of object cards interspersed.  It
doesn't load correctly.  I can give you the details of how it fails if you're
interested.  Comments in the listing (July 1962) have the initials RBR.  Do you
have any idea who that was?

Thanks in advance,

Guess there aren't any 1401 programming language newsgroups!  I would love 
to hear follow-ups on this one.  1401 was the first computer I programmed 
(using SPS) and I still remember some of the tricks and the bizarre design 
e.g. card reader input went into loc 001-080, punch went from 101-180, 
printer 201-232, X1/X2/X3 hard-mapped to specific core memory locations.

We used to code initialization logic in the reader/punch/printer locations 
before doins I/O operations... you didn't have much choice with 16K (max).

And... I'll never forget the impact of missing a wordmark...

Now... THAT was programming!  One shot at a compile, entering fixes at the 
console, patching object decks... the "good old days"...
