Martin ... I know of various projects that have "rightsized" mainframe
based applications to PCs using either MicroFocus or Realia COBOL ... They
met with various levels of success ... I believe the City of Orlando
Florida did this with very good success ... Anyway, there is a lot more to
a dependable operational system than just COBOL programs and it is these
things that make the project difficult, e.g. operational control, job
scheduling, user interfaces (CICS?), etc. ... Be very clear about these
things and how they will be handled in the new environment ... Also, I
sincerely hope this is not being considered as an adjunct to a Y2K
remediation project ... Good Luck.
