Read COBOL/400 manual it explains sub file processing in detail and in 
easy to understand manner with examples. 

If you dont know how to make sub files using DDS or SDA than consult these 
two manuals. 

Its really simple. Define the control record where you define the sub file 
page, control indicators, end of sub file indicators etc.. Actual subfile 
where you will define the fields for your sub file.

COBOL program.....

Perform initialization... where you will clear the sub file turn the 
indicator to write the control record on.

Read your Physical file... keep loading Physical fields into sub file 
fields until Physical file reach EOF (End of File) or subfile reached EOF.

Write the control record. This will display all the records written to the 
subfile with the capability of Roll Up and Roll Down.
