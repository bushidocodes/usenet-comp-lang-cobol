Hi Fabio,

I don't believe COMP or COMP-3 have any directly corresponding OLE types.
However you can easily convert them to the relevant data type:-

01    mycomp3        pic 9(5) comp-3.
01    myVT-I2           pic x(2) comp-5.

move mycomp3 to myVT-I2

COBOL handles the translation of type for you.

You can use a OpenESQL connection that you previously set up by using the
following syntax:-

EXEC SQL
    SET CONNECTION connectionname
END-EXEC

If you want to reuse connections for performance reasons you should consider
using the CONNECTIONPOOL OpenESQL directive.

Regards
David.
