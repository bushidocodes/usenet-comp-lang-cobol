The ETK toolset includes a screen painter that you use to define a map.
A runtime module (SCRNIO) then generates the 3270 datastream on the
fly. No BMS map is needed. From the map, SCRNIO can also generate
HTML instead of the 3270 datastream. Go to http://www.etk.com for more.
