The only option to specify this is on the OPEN verb:

Open EXTEND file-name.

This will allow adding records to the end of the file. In an IBM
mainframe Cobol environment, only QSAM file may be processed in this
manner. Some other Cobol's for other platforms do allow EXTEND to
indexed files. IBM mainframe JCL also allows appending to a QSAM
file by adding DISP=MOD.

Mike Dodas
