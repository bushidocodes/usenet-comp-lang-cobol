Well, there is at least one piece of compiled software which does this. You
can download the freeware COBFD.ZIP from
http://www.flexus.com/downloads.html .

This is an MS-DOS program I have been threatening to bring into the 21st
century (e.g., Windows(r)) for about two years. I had not considered making
the source code available, but if you want it, I will e-mail it do you for
your personal use.

That source code is written in PowerBASIC (um, I feel dumb saying this, but
PB is a BASIC dialect) , and uses some features of that language not
available in other BASICs (it also uses PB/DOS version 3.5, and some of the
features I used are not available in prior versions of PB). But it is
well-written BASIC code, which looks a lot like well-written COBOL code, and
if nothing else would give you an idea of some of the traps, pitfalls and
other goodies you might encounter on your trip.

If interested, drop me a note explaining your application and we'll go from
there. (Applies to anyone).
