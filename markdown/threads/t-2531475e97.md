Question:
WRITE A PROGRAM TO PROCESS TWO
FILES. ONE OF THE FILES IS A TRANSACTION FILE AND IT WILL BE USED TO
UPDATE THE SECOND FILE WHICH IS A MASTER FILE. THE UPDATED MASTER FILE
RECORDS WILL BE WRITTEN AS A REPORT INSTEAD OF A NEW MASTER FILE AND
THE ERRORS WILL ALSO BE WRITTEN AS A SEPARATE REPORT. THE REPORT
LAYOUTS ARE LEFT UP TO YOU BUT SHOULD HAVE PROPER SIZING AND LOOK
PROFESSIONAL. MAIN LOGIC OF THIS PROCESS WILL BE DISCUSSED DURING ONE
OF THE LABS AND IS DISCUSSED IN CHAPTER 11 OF TEXT.

THE TWO INPUT FILES ARE:

ITEM MASTER FILE (CM2310S)
     ITEM NO              1  -  3
     ITEM DESCRIPTION     4  - 19
     UNIT PRICE          22  - 26  (2 DECIMALS)
     UNIT COST           27  - 31  (2 DECIMALS)
     ON HAND QUANTITY    32  - 36  (0 DECIMALS)
     STATUS              46  - 46  (A - ACTIVE, D - DELETED)
======================================================================
SALES ORDER DETAIL (CM2300S)
     QUANTITY SOLD       26  - 29
     ITEM DESCRIPTION    30  - 45
     ITEM NUMBER         48  - 50
     TRANSACTION CODE    67
======================================================================

THE TWO FILES MUST BE MATCHED ON ITEM NO AND THE MASTER RECORDS
UPDATED ACCORDING TO THE FOLLOWING SPECIFICATIONS:

  1) IF THE MASTER IS ACTIVE AND THERE IS NO MATCHING TRANSACTION,
     THEN WRITE THE MASTER RECORD TO THE REPORT

  2) IF THE MASTER IS ACTIVE AND THERE IS A MATCHING TRANSACTION
     RECORD, THEN UPDATE THE MASTER AS FOLLOWS:

        A)  IF THE TRANS CODE IS "S", SUBTRACT QTY SOLD FROM THE ON
            HAND QUANTITY
        B)  IF THE TRANS CODE IS "C", ADD QTY SOLD TO THE ON HAND
            QUANTITY

  3) WHEN ALL TRANS RECORDS HAVE BEEN PROCESSED AGAINST A MASTER
     RECORD, PERFORM THE FOLLOWING CALCULATIONS AND INCLUDE ON DETAIL

        EXTENDED PRICE   = ON HAND QUANTITY TIMES UNIT PRICE
        EXTENDED COST    = ON HAND QUANTITY TIMES UNIT COST
        PROFIT/LOSS      = EXTENDED PRICE - EXTENDED COST

  4) ALL OTHER NORMAL ERROR CONDITIONS MUST BE CAUGHT AND REPORTED ON
     THE ERROR REPORT.

  5) IF THE MASTER STATUS IS "D", THEN AN ERROR MESSAGE SHOULD APPEAR
     SAYING "DELETED-MASTER", IF THE MASTER STATUS IS "D" AND HAS
     CORRESPONDING TRANSACTIONS, THEN THESE TRANSACTIONS ARE IN ERROR
     AND SHOULD BE REPORTED ON THE ERROR LISTING.


Answer:
       IDENTIFICATION DIVISION.
       PROGRAM-ID. ASSIGN3.
      **********************************************************

       ENVIRONMENT DIVISION.
       CONFIGURATION SECTION.
       INPUT-OUTPUT SECTION.

       FILE-CONTROL.
  SELECT SALES-ORDER-DETAIL ASSIGN TO DISK 'C:\COBOL\CM2300S.DAT'
            ORGANIZATION IS LINE SEQUENTIAL.
            SELECT ITEM-MASTER-FILE ASSIGN TO DISK 'C:\COBOL\CM2310S.DAT'
            ORGANIZATION IS LINE SEQUENTIAL.
  SELECT OUTPUT-FILE ASSIGN TO DISK 'C:\COBOL\REPORT1.RPT'
            ORGANIZATION IS LINE SEQUENTIAL.


       DATA DIVISION.

       FILE SECTION.

       FD   ITEM-MASTER-FILE
            RECORD CONTAINS 45 CHARACTERS
            DATA RECORD IS IN-SALS-REC.

       01   MR-MASTER-RECORD.
            05 MR-REC-TYPE  PIC X.
   88 SR-TRANSCODE  VALUE "S".
   88 SR-TRANSCODE  VALUE "C".

   10 ITM-NUM              PIC X(2).

 01 SALES-RECORD

 FD  OUTPUT-FILE

       WORKING-STORAGE SECTION.

 01-WORK-AREAS.
  05  WS-ACTION-CODE      PIC (X).
  05 WS-SALES-ORDER-DETAIL PIC X(100).

 01 PROGRAM-COUNTERS       COMP.

 PROCEDURE DIVISION.


 100-PROCUDE ITEM-MASTER-FILE.
  PERFORM 200-START-UP
  PERFORM 210-PROCESS-UPDATE
   UNTIL NO-MORE-RECORDS AND NO MORE-DETAIL-RECORDS.
  PERFORM 220-FINISH-UP.
  STOP RUN.

 200-START-UP.
  OPEN INPUT SALES-ORDER-DETAIL
   OLD-MASTER-FILE
     OUTPUT OUTPUT-FILE
  PERFORM 300-INITALIZE-ITEMS.
  PERFORM 310-WRITE-REPORT-HEADERS.
  PERFORM 400-READ-SALES-ORDER-DETAIL.
  PERFORM 410-READ-DETAIL-RECORD.

 210-PROCESS-UPDATE.
  MOVE SPACES TO ADUDIT-LINE
  IF OLD-MR-NAME = DETAIL;-NAME
   THEN
    IF UPDATE-RECORD
     PERFORM 320-UPDATE-OLD-MASTER-RECORD
    ELSE
   PERFORM 350-DELETE-OLD-MASTER-RECORD
 ENDIF

ELSE
    IF OLD-MR-NAME ( DETAIL-NAME
     PERFORM 330-COPY-OLD-MASTER-RECORD
    ELSE
     PERFORM 340-ADD-NEW-RECORD
   END-IF

  ENDIF
  PERFORM 360-PREPARE-AUDIT-LINE.

 220-FINISH-UP.
  PERFORM 370-PREPARE-TOTALS.
  WRITE PRINTLINE FROM END-LINE.
  AFTER ADVANCING 5 LINES.
  CLOSE DETAIL-LINE
  SALES-ORDER-DETAIL
  ITEM-MASTER-FILE
  OUTPUT-FILE.

 300-INITIALIZE-ITEMS.
  ACCEPT WS-DATE FROM DATE.
  MOVE WS-DATE TO O-DATE.
  INITIALIZE PROGRAM-COUNTERS.

 310-WRITE-REPORT-HEADERS.
  WRITE PRINTLINE FROM MASTER-FILE
   AFTER ADVANCING PAGE.
  WRITE PRINTLINE FROM REPORT-NAME-1
   AFTER ADVACNING 3 LINES.
  WRITE PRINTLINE FROM REPORT-NAME-2
   AFTER ADVANCING 2 LINES.
  WRITE PRINTLINE FORM HEADINGS1
   AFTER ADVANDCING 2 LINES.
  WRITE PRINTLINE FROM HEADINGS2.
   AFTER ADVANCING 1 LINE.
  MOVE 2 TO WS-PROPER-SPACING.

 320-UPDATE-OLD-MASTER-RECORD.
  MOVE "S" TO WS-ACTION-CODE.
  MOVE OLD-ITM-NO TO NEW-ITM-NO.
  MOVE DETAIL STATUS TO NEW-DETAIL-STATUS.
  MOVE OLD-ITM-DES TO NEW-ITM-DES.
  MOVE OLD UNT-PRC TO NEW-UNT-PRC.
  MOVE OLD ON-HND-QTY TO NEW-HND-QTY.
  ADD 1 TO WS-SALES-ORDER-DETAIL-CTR
     ITEM-MASTER-FILE.
  PERFORM 400-READ-SALES-ORDER-DETAIL.
  PERFORM 410-READ-DETAIL-RECORD.

 330-COPY-SALES-ORDER-DETAIL.
  MOVE "C" TO WS-ACTION-CODE.
  MOVE OLD-MR-RECORD TO NEW-MR-RECORD.
  WRITE NEW-MR-RECORDS.
  ADD 1 TO WS-OLD-RECORD=-COPIED-CTR.
  PERFORM 400-READ-OLD-MASTER-RECORD.

 340-ADD-NEW-RECORD.
  MOVE "A" TO WS-ACTION-CODE.
  MOVE DETAIL-ITM-NO TO NEW-ITM-NO.
  MOVE DETAIL STATUS TO NEW-DETAIL-STATUS.
  MOVE DETAIL-ITM-DES TO NEW-ITM-DES.
  MOVE DETAIL-UNT-PRC TO NEW-UNT-PRC.
  MOVE DETAIL-ON-HND-QTY TO NEW-HND-QTY.
  WRITE NEW-MASTER-RECORD
  ADD 1 TO WS-SALES-ORDER-DETAIL-CTR
     ITEM-MASTER-FILE.
  PERFORM 410-READ-DETAIL-RECORD.

 350-DELETE-STATUS-ITEM-MASTER-FILE
  MOVE "D" TO WS-ACTION-CODE.
  MOVE OLD-ITEM-MASTER-FILE TO WS-OL=MR-RECORD.
  ADD 1 TO WS-OLD-RECORD-DELETED-CTR.
  PERFORM 400-READ-OLD-MASTER-RECORD.
  PERFORM 410-READ-DETAIL-RECORD.

 370-PREPARE-TOTALS.
  WRITE PRINTLINE FROM SUMMARY-HEADER
   AFTER ADVANCING 5 LINES.
  MOVE WS-OLD-RECORD-COPIED-CTR TO O-OLD-RECORD-COPIED-CTR.
  WRITE PRINTLINE FROM SUMMARY-LINE-1.
   AFTER ADVANCING 2 LINES.
  MOVE WS-OLD-RECORD-DELETED-CTR TO O-OLD-RECORD-DELETED-CTR.
  WRITE PRINTLINE FROM SUMMARY-LINE-2.
   AFTER ADVANCING 1 LINE.
  MOVE WS-NEW-RECORD-CTR TO O-NEW-RECORD-CTR.
  WRITE PRINTLINE FROM SUMMARY-LINE-3.
   AFTER ADVANCING 1 LINE.
  MOVE WS-NEW-MASTER-FILE-CTR TO O-RECORD-CHANGED-CTR.
  WRITE PRINTLINE FROM SUMMARY-LINE-4.
  MOVE WS-NEW-MASTER-FILE-CTR TO O-NEW-MASTER-FILE-CTR.
  WRITE PRINTLINE FROM SUMMARY-LINE-5.
   AFTER ADVANCING 1 LINE.
  MOVE WS-AUDIT-CTR TO O-AUDIT-CTR.
  WRITE PRINTLINE FORM SUMMARY-LINE-6.
   AFTER ADVANCING 1 LINE.
  EXD-PRICE = WS-ON-HND-QTY * UNT-PRC.
  EXD-COST = WS-ON-HND-QTY * PRC.
  PRF-LOSS = EXD-PRICE - EXT-CST

 400-READ-OLD-MASTER-RECORD.
  READ OLD-MASTER-FILE NEXT RECORD
   AT END
    MOVE HIGH-VALUES TO OLD-ITM-NO.

 410-READ-DETAIL-RECORD.
  READ DETAIL-FILE-RECORD
   AT END
    MOVE HIGH-VALUES TO DETAIL-ITM-NO.
