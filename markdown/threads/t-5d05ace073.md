I would write a COBOL program (but then this is a COBOL NG for COBOL
programmers).

Starting at the first position of your field, you will need to check
your digits (*for signs, for commas and so forth); moving each byte
group (if numeric) to a holding field that is only numeric and defined
in an arithmetic field (so you can add, subtract or what ever).
