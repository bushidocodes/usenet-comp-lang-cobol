It not possible to use a Fujitsu Cobol serial number, specially if you
are installing v6.1.. The protection scheme is enough difficult to
crack, because the site key given during the installation, can change
from machine to another, also using the same serial number. There is
also another consideration: supposing your are able to install the
product, it will run only for 30 days, then it needs to be activated
sending a request to Fujitsu registration server, to obtain a valid
activation key to unlock the product for an unlimited use.
The Fujtsu cobol proctection scheme is hard to crack.
There is final consideration that each programmer should be think: Do
you like to see your work freely distributed on the web? I think not!
So a product copyrighted must be purchased and regularly licensed.

An advice: don't waste your time..
