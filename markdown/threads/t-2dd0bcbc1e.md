Windows has a fundamental flaw dating back to its earliest days that
any file can be an executable. It also has a flaw that it (by default)
hides the real type of the file so what looks like 'knickers.img' is
actually 'knickers.img.exe' and clicking on this may cause it to
execute even if it is, say, an email attachment.

These flaws, and several others, means that Windows opens attack
vectors that are closed in other systems. Windows is broken by design
and is, apparently, unfixable.

Actually the anti-virus industry is so large that MS seems to have no
intention of fixing the causes. Each new version or product is claimed
to be 'more secure' and this generates revenue by users upgrading in
order to try to fix the problems of their older and dying
installations, or by having their machines so slow and clogged by
malware that they toss the machine out and buy a new one.

My daughter has a laptop that had Vista on it. It became slower and
slower over time and eventually locked up completely, each reboot came
to a point where the machine simply locked.

The 'solutions' offered were: spend some hundreds to have it de-
malwared and Vista re-installed (requires recovery CDs from
manufacturer); throw it away and buy a new laptop; spend some hundreds
and buy Windows 7 (and hope the devices were supported);

I put Ubuntu on it. Everything works, it does what she needs: skype,
firefox, digikam (or Picassa), OpenOffice.org, it even runs her
Windows based banking program under wine. It boots and runs faster
than it ever ran with Vista (not hard).

The computer industry loves Windows and malware because they generate
so much revenue and repeat business.

If you must run Windows then FireFox + NoScript + AdBlock + FlashBlock
will remove most of these malware attacks from even starting. Turn
Cookies off (or ask) to get rid of tracking.
