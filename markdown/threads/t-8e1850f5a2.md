This looks like your program is attempting to work with a file that you 
did not setup a runtime environment variable for.

    Select My-File Assign to file1.

In the mainframe world , you would have a JCL with a DDName FILE1
followed by the full file name to be processed.   In Fujitsu, the operations
are similar.   When the program executes, a run-time environment screen
is presented where you can set environment variables to help your program
associate internal assignments to actual external files.

  FILE1=C:\TEMP\SOMEFILE.DAT
  BSORT-TMPDIR=D:\TEMP;E:\TEMP;P:\REALLYBIGSPACE
