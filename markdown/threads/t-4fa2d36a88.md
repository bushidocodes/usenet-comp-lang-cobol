```
A few years back I was in a shop which went from Univac to DEC.  The
COBOL we used was changed along with everything else.  It wasn't until I
worked at an IBM shop again that I learned that the DEC COBOL was COBOL
II.  Knowing that may have been useful in job hunting.

Does anybody have a list of various COBOL versions used by various
manufacturers with ways to determine which versions were comparable to
which other versions?

(My current shop has COBOL 68, COBOL 72, & COBOL 370, but as far as I
can tell, no COBOL II (they only switched to be Y2K compatible, and
COBOL II is not Y2K compatible)).

Some shops use names such as COBOL 82/89 and COBOL 2000.  Others user or
did use their own names or implied names (Honeywell COBOL around 1974,
etc).  Others have different COBOLS for different sized machines
(Burroughs).
```
