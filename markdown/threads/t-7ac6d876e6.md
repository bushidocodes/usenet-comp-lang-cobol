Yes, you can 'override' every line of your JCL if you don't code it directly,
but rather include this statement. You can perfectly INCLUDE one or another IF
statement via symbolic parameters. A sample:

//XXXX PROC IF1=$$IF1,IF2=$$IF2
//  INCLUDE MEMBER=&IF1
//S1 EXEC PGM=IEFBR14
//  INCLUDE MEMBER=&IF1.END
//  INCLUDE MEMBER=&IF2
//S2 EXEC PGM=IEFBR14
//  INCLUDE MEMBER=&IF2.END
//   PEND
//GO1 EXEC PROC=XXXX
//GO2 EXEC PROC=XXXX,IF1=X1,IF2=X2

This sample-members contain:
$$IF1....:  //COND1 IF RC = 0 THEN
$$IF1END.:  //COND1   ENDIF
$$IF2....:  //COND2 IF RC = 0 THEN
$$IF2END.:  //COND2   ENDIF
X1.......:  //COND1 IF RC < 4 THEN
X1END....:  //*
X2.......:  //COND1 ELSE
X2END....:  //COND1 ENDIF

And the results:
GO1      S1          00
GO1      S2          00
GO2      S1          00
GO2      S2       FLUSH

Was it that what you were looking for?
