The following is a note from page 65 of the PDF Reference
Manual, version 1.3, that is available from the Adobe web site,
< http://partners.adobe.com/asn/developer/acrosdk/DOCS/PDFRef.pdf >.

"Note: RC4 is a copyrighted, proprietary algorithm of RSA
Security, Inc. Adobe Systems has licensed this algorithm for
use in its Acrobat products. Independent software vendors may
be required to license RC4 in order to develop software that
encrypts or decrypts PDF documents. For further information,
visit the RSA Web site at <http://www.rsasecurity.com> or send
e-mail to <products@rsasecurity.com>."

Is there any reason you cannot use the Acrobat SDK
< http://partners.adobe.com/asn/developer/acrosdk/main.html >?

I have not looked at the details; but this seems that it may be
the easiest means for creating PDF from COBOL.
