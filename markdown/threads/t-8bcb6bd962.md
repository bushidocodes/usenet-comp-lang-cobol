You can use any editor as long as you can save as an ASCII file.
Word, WordPad, NotePad, vi (UNIX), etc.. will all work.

If you are using Word you can still save your document in an
ASCII file. Try to save it under a new filename and then Word will ask
what kind of file. Tell it text, not rich text or word format, but
just text. This is what I use and it always seems to work well for me.
