Hi Oscar,

To get the name of your application to appear in the task list on NT
all you need to do  set the window title. There are various ways to
do this, but probably the simplest two are:

If you explicitly declare a window, e.g.

        DISPLAY STANDARD WINDOW.

you just need to add a TITLE to that statement
        
        DISPLAY STANDARD WINDOW TITLE "CANE".

If you let the runtime create the window, then you can do:

        DISPLAY "CANE" UPON GLOBAL WINDOW TITLE.

Either should get you what you need.

Nigel
