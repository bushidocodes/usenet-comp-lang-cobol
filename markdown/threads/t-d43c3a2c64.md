The very least you need is a good book and a compiler.  Teach Yourself COBOL in
21 Days is the best selling COBOL book, and it's used in colleges and training
programs across the world, because its VERY easy to understand.  It assumes you
know nothing about programming and teaches it very simply giving lots of
analogies, and defining new terms and concepts at they arise.  It's very good
for teaching the basics of COBOL, which is what you want right now.

The next thing you need is a compiler.  We recommend the MicroFocus COBOL
Compiler for DOS, as thats what TYC21D was written to go with originally.  The
student version is $50.00.

Some people have trouble locating errors in their code and do not have anyone
to help them, so for these people we recommend the disk of code for Teach
Yourself COBOL in 21 Days, and the EZ-Compare Utility.  Then, if you write the
practice lesson and there's an error you just can't find, you can compare the
code you wrote to the code that's on the disk, and it will show you where the
error are.

For those that need a more structured approach or even more help with the
compiler and editor, we recomend the COBOL, Just in Time course, which has
explicit lessons on how to install and use the compiler and editor. If you
don't want to bother with trying to find these locally, give me a call.
