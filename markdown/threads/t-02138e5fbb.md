```
Several parts to answer:

1) IBM COBOL for OS/390 & VM is truly the release (actually version as I
recall) that follows IBM COBOL for MVS & VM (both of which follow IBM SAA
AD/Cycle COBOL/370 - which in turn followed VS COBOL II).  Therefore you have
the latest documentation - but not the latest software. (You might want to
check your site to see if using a different Steplib/Joblib would actually
point to the newer compiler.)

2) There are a couple  of other ways to get there, but one way to see the
online version of the "latest and greatest" IBM COBOL manuals is to go to:

   http://www.ibm.com/software/ad/cobol/library.htm

This includes pointers to online dox for BOTH of the compilers that you are
looking for. (Unlike some other links, this allows you to download the
manuals in PDF format - or use them online via Bookmanager)

3) If you check the "Summary of Changes" in any of the manuals, it will give
you a summary of what has changed between products, versions, releases, or
whatever. I usually look at the Programming Guide for this to get the most
"user-friendly" summary - but really any of the manuals will do.  For
example, check out:


http://www.s390.ibm.com/bookmgr-cgi/bookmgr.cmd/BOOKS/IGYPG204/FRONT_2%2e1%2e
1

   ***

I hope this helps.
```
