The Relativity product was NOT designed to convert users away from the
COBOL language. It was designed to allow COBOL users to access their
COBOL data using ant ODBC client product therfore allows them to report,
analise and graphically display their COBOL data.

By defination if the data can be accessed by any ODBC product, it can
also be converted to any other format supported by the ODBC client
product.

The FDs are required to the fact that a COBOL source with the FDs
included must be compiled with a special switch using RM/COBOL-85. The
compiled programs can then be used as the based for ODBC database tables.
