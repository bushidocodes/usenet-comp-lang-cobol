David, the best way to plug these two together is by using the Micro
Focus ESQL Toolkit. this is a pre-compiler specifically for Microsoft
SQL Server. so in your COBOL code you write embed SQL statements
which at compilation time produce code to directly go against the
SQL Server database. the Toolkit contains all the necessary in-depth
dox and demos.
