Bob, the INTERRUPT-FLAG indicates how the parameters are passed through
from the COBOL parameters to the system registers on entry and back into
the parameter block on exit. You will need to set the flags according to
the parameters MS-DOS requires for interrupt 1Ah. I don't have any
documentation for that interrupt so I can't tell you exactly what you need
to set though unfortunately.

If the interrupt requires the fields in BCD, the easiest way to set up the
information will be to redefine the appropriate field as COMP-3, which is
BCD, and move the required value to it. eg

05 INTERRUPT-CH PIC X(01) COMP-X.
05 INTERRUPT-CH-BCD REDEFINES INTERRUPT-CH
PIC X(01) COMP-3.
.....
MOVE TO INTERRUPT-CH-BCD
