```
First of all, let me clear some things up....

With my compiler, i received a copy of England Tech. Services' Cobol-CGI.
It included Source, Object, and EXE. for 2 programs.  The EXEs were created
with the cobol object, the standard libraries and a special C program which
is called to "read" and to "write" the cgi.  The source for the C prog was
however not provided.

I put the HTML doc into the wwwroot folder and the EXEs into the scripts
folder.  And corrected the path to the programs in the HTML doc.

When i load the page, it returns:

"JMP00261-W NO 'ON EXCEPTION' STATEMENT EXISTS, WHEN EXCEPTION OCCURRED ON
'ACCEPT ARGUMENT-VALUE' STATEMENT"

and the browser returns:

"CGI Error
The specified CGI application misbehaved by not returning a complete set of
HTTP headers. The headers it did return are"

This is the majority of one of the programs.  If you need me to post the
rest of it I will upon request.  It is supposed to accept env-variables from
the client, send them to the server, and be sent back to the client in a
text format (not HTML) on the browser.

 IDENTIFICATION DIVISION.
 PROGRAM-ID. SHOWENV.
*
 ENVIRONMENT DIVISION.
 CONFIGURATION SECTION.
 SPECIAL-NAMES.
     ARGUMENT-VALUE IS COMMAND-LINE.
*
DATA DIVISION.
WORKING-STORAGE SECTION.
01 FILE-NAME            PIC X(80).
01 WEB-DATA.
     05 REQUEST-PROTOCOL PIC X(20)
                         VALUE SPACES.
     05 FILLER           PIC X
                         VALUE LOW-VALUES.
     05 REQUEST-METHOD   PIC X(20)
                         VALUE SPACES.
     05 FILLER           PIC X
                         VALUE LOW-VALUES.
     05 EXECUTABLE-PATH  PIC X(80)
                         VALUE SPACES.
     05 FILLER           PIC X
                         VALUE LOW-VALUES.
     05 LOGICAL-PATH     PIC X(80)      VALUE SPACE.
     .
     .

 PROCEDURE DIVISION.
 0000-MAINLINE.
     ACCEPT FILE-NAME FROM COMMAND-LINE.
     CALL "CGISETUP" USING FILE-NAME.
     PERFORM 1000-GET-ENV-VALUES.
     PERFORM 2000-SEND-REPLY.
     GOBACK.
 1000-GET-ENV-VALUES.
     CALL "CGIREAD" USING
         BY CONTENT "REQUEST PROTOCOL" & X"00"
         BY REFERENCE REQUEST-PROTOCOL.
     CALL "CGIREAD" USING
         BY CONTENT "REQUEST METHOD" & X"00"
         BY REFERENCE REQUEST-METHOD.
     .
     .

 2000-SEND-REPLY.
     PERFORM 2100-SET-PLAIN-TEXT.
     PERFORM 2200-WRITE-HEADING.
     PERFORM 2300-WRITE-ENV-VALUES.
 2100-SET-PLAIN-TEXT.
     CALL "CGIWRITE" USING
         BY CONTENT "Content-type: text/plain" & X"0D0A" & X"00".
 2200-WRITE-HEADING.
     CALL "CGIWRITER" USING
         BY CONTENT "Here are the values of the CGI items " & X"00".
     CALL "CGIWRITE" USING
         BY CONTENT "returned as a plain text file:" & X"0A" & X"00".
 2300-WRITE-ENV-VALUES.
     CALL "CGIWRITER" USING
         BY CONTENT "REQUEST PROTOCOL: " & X"00".
     CALL "CGIWRITE" USING REQUEST-PROTOCOL.
     CALL "CGIWRITER" USING
         BY CONTENT "REQUEST METHOD: " & X"00".
       .
       .
```
