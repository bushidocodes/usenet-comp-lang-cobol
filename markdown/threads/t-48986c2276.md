You need to put the libraries on your search path and point to them
with environment variables.

In my set up is the compiler is on drive X: which is on the search
path, and I have the following settings in my autoexec.bat file:-

SET COBDIR=X:
SET COBHNF=X:HELP
SET LIB=X:LIB
SET INCLUDE=X:INC
