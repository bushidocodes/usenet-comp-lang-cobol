```
If your needs are quite simple and you know Cobol then CobolScript
(www.cobolscript.com) is quite cheap and is almost Cobol.  It is easy
to get it to access the web and most bits required are built in.

It is relatively easy to get Cobol to work with CGI without any fancy
interface routines.  All that is required is to be able to ACCEPT FROM
ENVIRONMENT-VALUE and to ACCEPT FROM CONSOLE where this is stdin.

However, for simple, free, web accessing in a nice modern
multi-platform language try Python (www.python.org).
```
