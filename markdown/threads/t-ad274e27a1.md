Ok, we figured it out. It seems that we were using the bundled 'cc'
compiler, but were using the GNU assembler because the user had the
GNU tools first in her path and the GNU assembler was called 'as'
instead of 'gas'. Renaming the GNU assembler to 'gas' fixed the
problem.
