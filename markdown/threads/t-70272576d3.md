 Cyrillic is utf-8 Unicode. Bill is right about the picture to accommodate 
it.

However the text box itself needs to have its properties amended so that 
Cyrillic can be input.

You need to set the following properties for a Cyrillic text box:

charset = "windows-1251"
Font.Name = "Arial" (other fonts are also possible, but this is simple and 
easy)
Font.charset = 204

My COBOL server is currently offline so I can't test whether the PowerCOBOL 
text box control has these properties. If you can't make it work in
