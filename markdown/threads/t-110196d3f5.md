```
Go to the MSLFILES area of Microsoft's site and download the
LNK563.EXE
file, which is a self-extracting archive containing MS's last 16 bit
linker,
version 5.63 (surprise, surprise!) and see if that works better.  V3.05
is a really OLD version of LINK.

Alternatively, download Fujitsu's free win16 targeting Cobol, which is
a stable and mature product.  COBOL 6.50 was never finished as
the developer died.
```
