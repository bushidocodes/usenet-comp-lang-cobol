You can use DataReader tools from Siber Systems.

They allow to read Cobol daat files produced
by popular PC Cobol compilers (MF, FSC, RM, Acu)
and to convert this data to non-Cobol formats
(CSV, DBF, read into Crystal Reports).

Source for Cobol program helps but it is not required.

More info at http://www.siber.com/sct/datafile/
