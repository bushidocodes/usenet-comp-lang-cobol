You should have no technical problems going from Win 7 to Win 8 as far as
installation goes.

The biggest problem you will encounter will be getting used to having an app
surface as well as the desktop and having no START button in Win 8; it
reappears in 8.1.

In fact, I used win 8 for a couple of months and never missed the START
button. I just pressed the Windows button to flick to the app surface and
began typing the name of what I wanted. People don't realize that on this
surface you don't have to position a cursor into a text box in order to
input data. The whole surface is a giant textbox and you can type anywhere
on it. It is progressive like Intellisense so usually it takes a couple of
key depressions to get what you want.

It is important to do the update to 8.1 (not for the START button, but
because there are internals which are much improved in 8.1.)

As far as the Acer goes, it may be a special case. You said you ran the
Windows Adviser, but there is an ACER upgrade tool you should definitely
look at here:

http://us.acer.com/ac/en/US/content/upgrade-assistant-tool

I note they save the BIOS which is a bit drastic, so there might be some
specific things required for Acer. (I upgraded a Toshiba and an HP Probook
without having to do any of that.)

Once you have upgraded and stabilized the configurations across the
machines, you should have a consistent user interface with your personal
settings available on each machine if they are internet connected and you
allow login with your Microsoft account.

Give it a month or two to get used to it and you'll be glad you upgraded.
Boot times are much faster, BSOD seems to be a thing of the past, screens
are better, stability is better (mine have never crashed or hung), and the
live tiles do grow on you. I boot into the app surface (previously Metro)
and run from there. Desktop applications automatically swap to the desktop
so it is all pretty seamless really.

The jury is still out for me (and I am a big fan of XP) but Win 8.1 is
shaping to be my favourite Windows of all time. (However, I still do desktop
development on XP; Win 8 is used for the Web (services and pages) and will
be used for mobile and surface when I get around to it.

HTH,

Pete.
