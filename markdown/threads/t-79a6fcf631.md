Acucorp, Inc., an international provider of legacy extension solutions
for COBOL applications, is proud to announce the release of extend 5.1
with ACUCOBOL-GT. The extend 5.1 family offers the most comprehensive
set of technologies for transporting inherited applications into modern
systems. These solutions include Internet deployment, graphical COBOL
transitions, data access, distributed computing, and programmer
productivity. 

extend 5.1 with ACUCOBOL-GT offers some significant new features and
capabilities that help to make the development process easier and faster
for COBOL programmers. Some of the new features in 5.1 include: 
  


*	Character-to-GUI Wizard
*	Heightened plug-in security
*	Extended support for ActiveX and OLE Objects
*	Bundle configuration files with helper application library
*	Oracle 8i support
*	Redesigned AcuOBDC configuration interface
*	New multi-record mode in AcuServer
*	General Preprocessor Interface
*	Support for error and exit procedures
*	File I/O redirect facility
*	Numerous enhancements to AcuBench?, including;
*	Ability to Build or Rebuild just one project in the open
workspace
*	Screen Designer for graphical and character-based screen
creation
*	Simplified Version Control configuration interface
*	COBOL-intelligent source Code Generator

Internet: http://www.acucorp.com <http://www.acucorp.com> .
