relating to using MAPI in an mf cobol application...the last one I remember
on there was that normal file io verbs no longer work once you use the MAPI

datasets...the file status returns as normal, but no actual data record is
returned after say a read verb...interesting and pretty critical problem for
sure...still unresolved as far as I know...
