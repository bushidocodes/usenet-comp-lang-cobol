```
Paul,
  Your statement is correct when you say
    "Yes, you can compile on a PC"

but what you are missing is that there are LIMITED numbers of "PC" operating
systems that are supported for VisualAge COBOL.

See:
  http://www-4.ibm.com/software/ad/cobol/va/

which explicitly LIMITS the compiler to running on a Windows NT desktop (no
Windows 2000, no Windows XP, no other Windows or OS/2 operating systems).

 IBM announcement 200-107  (dated May 9, 2000 )

viewable at:

http://www1.ibmlink.ibm.com/cgi-bin/master?request=announcements&parms=S&xh=
xM53F%24AuHb3GLy1USenGnN9332&xhi=usa.main&type=ANNO&smonth=02&sday=27&syear=
2001&emonth=02&eday=27&eyear=2002&searchwords=&content=WORDS&n%240=200-107&n
%241=&n%242=&n%243=&n%244=&n%245=&n%246=&n%247=&n%248=&n%249=#@3h@061@

goes on to distinguish between program object code execution vs compiler
execution and says,

"For application execution, use one of the following:
  a.. Microsoft Windows NT 4.0 plus Service Pack 4
  b.. Microsoft Windows 95
  c.. Microsoft Windows 98 "
No where does it list Windows 2000 or Windows XP
```
