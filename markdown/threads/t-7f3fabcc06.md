In my bookshelf:

James G. Janossy : CICS/ESA Primer - ISBN 0471309915

Is intended for programmers, but gives a nice CICS overview in chapter 2

Eugene S. Hudders: CICS A Guide to Internal Structure - ISBN 0471521728
Is starting point for sysprog type info

Josepg J. Le Bert: CICS Essentials for application developers and programmers - ISBN
0070358699
Chapter 3: Basic concepts and terminology
