I have a bunch of heritage code running in win95 (Fujitsu Cobol).  Several
of the reports go 132 columns, and were handled in the past by a dot matrix
printer with buttons.  To print, the user set the print size to 20 CPI, and
away it goes.  They are now being printed on a HP laser 6.

I do not have a list of printer codes for that printer. Does anybody have
either a list, or better yet, a subroutine that will allow basic printer
setup?  Two or three basic fonts and sizes are plenty.  I want to set up the
printer, then copy a basic ASCII file to it.

---------- *** START PRINTER.CBL *** ----------
       Identification division.
       Program-id. PRINT-A-FILE.

       Working-storage Section.

       01 i                            pic x(4) comp-5.
       01 Printer-retcode              pic 9(4) comp-5.
       01 Printer-handle               pic x(4) comp-5 .
       01 Window-handle                pic x(4) comp-5.

       01 Printer-file-read-information.
           03 fi-file-name             pic x(80).
           03 fi-access-mode           pic x comp-x.
           03 fi-deny-mode             pic x comp-x.
           03 fi-device                pic x comp-x.
           03 fi-file-handle           pic x(4).
           03 fi-file-offset           pic x(8) comp-x.
           03 fi-byte-count            pic x(4) comp-x.
           03 fi-flags                 pic x comp-x.
           03 fi-buffer-size           pic 9(4) comp-5 value 128.
           03 fi-buffer                pic x(130).

       01 Document-title.
           03 Document-title-length    pic x(2) comp-5 value 8.
           03 Document-title-value     pic x(8) value "Document".


       *> set of bits defining printer options
       *>    1  bit 0  display a printer control dialog
       *>    2  bit 1  display a font dialog box
       *>    4  bit 2  print portrait (not with bit 0 or bit 3)
       *>    8  bit 3  print landscape (not with bit 0 or bit 2)
       *>   16  bit 4  display progress indicator dialog box
       *> rest of bits must be 0
       01 Flags                        pic x(4) comp-5.

       01 Font-family-name.
           03 Font-name-length         pic x(2) comp-5.
           03 Font-name-value          pic x(11).

       *> Font size in points
       01 Font-size-value               pic x(4) comp-5.

       *> set of bits defining font style
       *>    1  bit 3  bold
       *>    2  bit 2  strikeout
       *>    4  bit 1  underline
       *>    8  bit 0  italic
       *> Rest of bits must be 0
       01 Font-style                   pic x(4) comp-5.


       Linkage section.

       01 Printer-param.
          Copy PrintPar.

       Procedure Division using Printer-param.

       Main.
           Move zero to Printer-retcode

           Move Font-size to Font-size-value

           Move Font-name to Font-name-value
           Move 11 to Font-name-length

           *> General print options:
           Move 0 to Flags

           If Control-dialog-on
               Add 1 to Flags
           End-if

           If Font-dialog-on
               Add 2 to Flags
           End-if

           If (Print-orientation-set = "P")
               Add 4 to Flags
           Else
               Add 8 to Flags
           End-if

           If Progress-bar-on
               Add 16 to Flags
           End-if

           *> Font style options:

           Move 0 to Font-style

           If Font-style-bold-on
               Add 1 to Font-style
           End-if

           If Font-style-strike-on
               Add 2 to Font-style
           End-if

           If Font-style-underline-on
               Add 4 to Font-style
           End-if

           If Font-style-italic-on
               Add 8 to Font-style
           End-if

           Move File-to-print to fi-file-name

           Perform Print-file

           Perform Error-routine
           Goback
           .


       Print-file.
           Move 1 to fi-access-mode
           Move 3 to fi-deny-mode
           Move 0 to fi-device

           Call "CBL_OPEN_FILE"
             using fi-file-name,
                   fi-access-mode,
                   fi-deny-mode,
                   fi-device,
                   fi-file-handle
           End-call

           If (return-code not = zero)
               Move 90 to Printer-retcode
               Perform Error-routine
               Goback
           End-if

           Call "PC_PRINTER_OPEN"
             using Printer-handle,
                   Document-title,
             by value Flags,
             by value Window-handle,
             returning Printer-retcode
           End-call

           Call "PC_PRINTER_SET_FONT"
             using Printer-handle,
                   Font-family-name,
             by value Font-size-value,
             by value Font-style,
             returning Printer-retcode
           End-call

           If (Printer-retcode = zero)
               Move zero            to fi-file-offset
               Move zero            to fi-flags
               Move fi-buffer-size  to fi-byte-count

               Call "CBL_READ_FILE"
                 using fi-file-handle,
                       fi-file-offset,
                       fi-byte-count,
                       fi-flags,
                       fi-buffer
               End-call

               Perform Read-file
                 until (return-code not= zero)

               If (fi-buffer not= spaces)
                   Perform varying i from fi-buffer-size
                     by -1 until (fi-buffer (i:1) not = space)
                       Continue
                   End-perform
                   Move x"0D0A" to fi-buffer (i + 1: 2)

                   Call "PC_PRINTER_WRITE"
                     using Printer-handle,
                           fi-buffer
                     by value fi-buffer-size
                     returning Printer-retcode
                   End-call
               Else
                   Move x"0D0A" to fi-buffer (1:2)
                   Move 2 to  fi-buffer-size
                   Call "PC_PRINTER_WRITE"
                     using Printer-handle,
                           fi-buffer
                     by value fi-buffer-size
                     returning Printer-retcode
                   End-call
               End-if

               Call "PC_PRINTER_CLOSE"
                 using Printer-handle
                 returning Printer-retcode
               End-call
           End-if

           Call "CBL_CLOSE_FILE"
             using fi-file-handle
           End-call
           .


       Read-file.
           Call "PC_PRINTER_WRITE"
             using  Printer-handle,
                    fi-buffer
             by value fi-buffer-size
             returning Printer-retcode
           End-call

           Add fi-buffer-size to fi-file-offset
           Move space to fi-buffer

           Call "CBL_READ_FILE"
             using fi-file-handle,
                   fi-file-offset,
                   fi-byte-count,
                   fi-flags,
                   fi-buffer
           End-call
           .


       Error-routine.
           Move Printer-retcode to Error-code

           Evaluate Error-code
           When  0
               Move "Printed successfully"      to Error-mssg
           When  1
               Move "Failed to open printer"    to Error-mssg
           When  2
               Move "Bad printer control code"  to Error-mssg
           When  3
               Move "Printer not open"          to Error-mssg
           When  4
               Move "Out of memory"             to Error-mssg
           When  5
               Move "File not found"            to Error-mssg
           When  6
               Move "Disk full"                 to Error-mssg
           When  7
               Move "Printer aborted"           to Error-mssg
           When  8
               Move "Bad info structure"        to Error-mssg
           When  9
               Move "No default printer found"  to Error-mssg
           When 10
               Move "No COMMDLG.DLL found"      to Error-mssg
           When 11
               Move "Write Failure"             to Error-mssg
           When 90
               Move "File not found"            to Error-mssg
           When other
               Move "** Error unknown **"       to Error-mssg
           End-evaluate
           .
       End program PRINT-A-FILE.
---------- *** E N D PRINTER.CBL *** ----------
---------- *** START PRINTPAR.CPY *** ----------
       13  File-to-print                 pic X(80).
       13  Error-code                    pic 9(4) comp-5.
       13  Error-mssg                    pic X(30).
       13  Control-dialog-set            pic X(1) value "F".
           88  Control-dialog-off            value "F".
           88  Control-dialog-on             value "T".
       13  Font-dialog-set               pic X(1) value "F".
           88  Font-dialog-off               value "F".
           88  Font-dialog-on                value "T".
       13  Progress-bar-set              pic X(1) value "F".
           88  Progress-bar-off              value "F".
           88  Progress-bar-on               value "T".
       13  Print-orientation-set         pic X(1) value "P".
           88  Print-portrait                value "P".
           88  Print-landscape               value "L".
       13  Font-name                     pic X(30) value "Courier New".
       13  Font-size                     pic X(4) comp-5 value 8.
       13  Font-style-bold-set           pic X(1) value "F".
           88  Font-style-bold-off           value "F".
           88  Font-style-bold-on            value "T".
       13  Font-style-strike-set         pic X(1) value "F".
           88  Font-style-strike-off         value "F".
           88  Font-style-strike-on          value "T".
       13  Font-style-underline-set      pic X(1) value "F".
           88  Font-style-underline-off      value "F".
           88  Font-style-underline-on       value "T".
       13  Font-style-italic-set         pic X(1) value "F".
           88  Font-style-italic-off         value "F".
           88  Font-style-italic-on          value "T".
---------- *** E N D PRINTPAR.CPY *** ----------
---------- *** START PRINTTST.CBL *** ----------
       Identification division.
       Program-id. TST-MAIN.

       Data division.

       Working-storage section.

       01 Prt-Param.
          Copy PrintPar.

       Procedure division.

       Main.
           Move "c:\AUTOEXEC.BAT" to File-to-print

           Call "PRINTER"  using Prt-Param
           Display Error-code
           Display Error-mssg

           Stop run
           .

       End program TST-MAIN.
---------- *** E N D PRINTTST.CBL *** ----------
