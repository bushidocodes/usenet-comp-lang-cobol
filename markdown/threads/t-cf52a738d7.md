Hi Donald

The Pdf995 driver, as Joe Hunter has mentioned, in the simple and easy
way to obtain automatically a .pdf file (at low cost)

I use it with Netcobol for Windows and what you have to do is only to
pass before run the printing, the print name that must be assumed the
job, like "Invoice_report"..It will create a .pdf file called
"Invoice_Report.pdf" and, if you did set the pdf995 options, you can see
the preview automatically after the print is executed.

If you use Netcobol for Windows, you can create your own .prt file
information, like below:

*

*    Printer Information File
*

PRTDIALG N                       * Print Dialog : YES
PRTID    "Invoice_Report"
PRTDRV   PDF995
FORMKIND C                       * Kind of form : cut paper
PRTFONT  FAC                     * Font face specification
FONTACE  "Arial"                 * Default font face name

It costs about $20 for the complete suite...
You can also set in the PDF995 options, the directory where to store the
pdf generated.

Hope in this help.

Gianni
