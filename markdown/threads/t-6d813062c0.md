here are fragments from the PRINTESC.CBL sample program supplied with
Personal COBOL, showing a couple of ways to do this...

select print-file
organization line sequential
assign "LPT1".

01 enable-landscape.
05 filler pic x value x"1B". *> escape
05 filler pic x value "&".
05 filler pic x value "l". *> "ell"
05 filler pic x value "1". *> one
05 filler pic x value "O". *> "oh"

01 lines-per-inch-11 pic x(4) value x"1B43000B".

print-landscape section.
open output print-file
write print-record from enable-landscape



best
Al

Here's a few ways to define hex codes. YMMV depending on compiler and
computer.

identification division.
program-id. esc.
* ways to define an escape character in COBOL.
environment division.
configuration section.
special-names.
* part of one of the right ways
symbolic esc-val is 28.
data division.
working-storage section.
* not especially portable because of byte order
* differences. This one works on a low-order first
* machine like a VAX or Alpha
1 nn pic 9999 comp value 27.
1 redefines nn.
3 esc-1 pic x.

* one of the right ways
1 esc-2 pic x value esc-val.

* don't try this at work
1 esc-word.
3 pic 9999 comp value 27.
1 esc-3 pic x.

* is this portable?
1 esc-4 pic x value x"1B".

1 esc-5 pic x.

1 esc-6 pic x.

1 esc-7 pic x.

procedure division.
begin.
* byte order dependent
move esc-word to esc-3.

* is this portable
move x"1B" to esc-5.

* a right way
move function char(28) to esc-6.

* another right way
move esc-val to esc-7.

One of the easiest ways to send escape charaacters to a print is by
enclosing them in quotes in a value statement of a move statement.
Microfocus lets you use the Microsoft extended character method inside
quotes. To use it type a quote then hold down the ALT key and on the
numeric keypad type the decimal value for the character you want.

ESC sequence, and here is an example of how I do it. It may not work under
a different environment but give a try.


01 PRT-ITEM-ETA-DAT.
05 PRT-IN-RED PIC X(3) VALUE "^[H".
05 PRT-ITEM-ETA-MM PIC 9(2).
05 FILLER PIC X VALUE "/".
05 PRT-ITEM-ETA-YY PIC 9(2).
05 PRT-IN-BLACK PIC X(3) VALUE ^[G".
