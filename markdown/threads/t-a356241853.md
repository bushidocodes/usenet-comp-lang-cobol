```
All these can be handled with the ANSI 74/85 INSPECT verb:

MOVE ZERO to TALLY-VAR
INSPECT  THE-DATA
    TALLYING TALLY-VAR FOR ALL 'A'.

INSPECT NOME
    REPLACING ALL 'A' WITH 'F'.   <<< Might be "BY" F I get confused

TALLY is a special register in older/other COBOLs;using ANSI 74/85 you must
specify and initialize the dataname where the tally (count) will be stored.
```
