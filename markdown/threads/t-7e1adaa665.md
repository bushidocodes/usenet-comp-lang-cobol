```
GnuCOBOL 2.2 Release Candidate (20JUL2017) is now available.

You can download a Windows MinGW build from here:
http://www.arnoldtrembley.com/GC22RC-BDB-rename-7z-to-exe.7z

The compiler is packaged as a 7-Zip self-extracting file, but for
security reasons my web hosting service does not allow hosting .EXE
files. The download can be extracted using the open source 7-Zip
compression utility for Windows, or you can rename the ".7z" file
extension to ".exe" and run it to install the GnuCOBOL folder.
http://www.7-zip.org/

Source code for the compiler and discussion forums can be found on the
SourceForege project pages:
https://sourceforge.net/projects/open-cobol/

Manuals and Programmer's Guides can be downloaded from this page:
https://open-cobol.sourceforge.io/

Older install versions, such as GnuCOBOL 2.0 Release Candidate 2
(06NOV2016) and GnuCOBOL 1.1 (20JAN2014) are available from my GnuCOBOL
page:
http://www.arnoldtrembley.com/GnuCOBOL.htm

Kind regards,


http://www.arnoldtrembley.com/
```
