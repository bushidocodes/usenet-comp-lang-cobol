There are a few other COBOL vendors.

CA-Realia AllFusion - no runtime fees, but the compiler is pretty 
expensive for a home user.
http://www3.ca.com/solutions/Product.aspx?ID=1475

Liant might still sell RM COBOL (the old Ryan-McFarland COBOL)
http://www.liant.com/products/

Acucorp might still sell AcuCOBOL for the PC or Linux.
http://www.acucorp.com/
http://www.acucorp.com/solutions/datasheets/

The Kompany sells a PC COBOL compiler for as little as $60, although I 
believe it requires either Mingwin or Cygwin (Unix environment for 
Windows).  I think they have a version for Linux too.
http://www.thekompany.com/products/kobol/

But these vendors are much smaller players in a market that doesn't 
seem to be very big.

Here's a website where you might be able to download Microsoft COBOL 
4.65 for DOS (based on MicroFocus COBOL), or Nevada COBOL for CP/M.
http://www.retroarchive.org/cpm/lang/lang.htm

A few brave souls have even attempted writing their own COBOL 
compiler.  Free source code in C:
http://members.aol.com/CobolHouse/index.htm

Most COBOL programmers work in the business world.  They write code 
for money, not for fun.  Perhaps that's why there's no market for a 
personal COBOL compiler.
