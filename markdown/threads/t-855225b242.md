```
I tried to post this once but it didn't make it:

With Fujitsu and MicroFocus

Select Byte-File Assign To "filename.dat".

FD  Byte-File.
01  Byte-Rec Pic X.

Open Input Byte-File
Perform Until All-Done
   Read Byte-File
            At End Set All-Done To True
            Not At End
                  Perform Process-Byte
    End-Read
End-Perform
Close Byte-File


With Realia, change the select to:

Select Byte-File Assign To "filename.dat[B]".
```
