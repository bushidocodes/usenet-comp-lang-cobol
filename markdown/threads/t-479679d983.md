AFP is a forms programming language used to make nice laser printer
forms from flat files.  You basically write out a record of all the data
to appear on the report and the AFP programmer builds a form and maps
the data to the form.  The language is arcane and tied closely to page
position.  We had a guy here who did this full-time and he always
started by laying out the form on a transparency and then worked with
the data using the transparency to make final adjustments to the
position where each field should fall.

Joe
