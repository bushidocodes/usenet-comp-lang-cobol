I assumed that the original question was using "compiler" loosely and they
really mean that the Micro Focus (MEANT) "product" provides numeric
validation "checking."  This is determined (in INT) by the setting of the
+F/-F run-time switch.  With the "correct" setting numeric fields including
"incompatible data" get a run-time message (when referenced) - with the other
setting, they do not.

I have never used it, but I believe that using the Fujitsu

         CHECK(NUMERIC)

directive, MAY provide the same functionality.
