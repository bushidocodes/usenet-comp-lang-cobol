```
I don't know if your "MF COBOL" was intended to be "Micro Focus" or <IBM>
"Mainframe" - but luckily, the answer is the same.  As I understand it, your
COBOL program is calling a C program that returns a pointer and you want
that pointer to point to an integer that you can use in COBOL.  If so, do
the following:

Data Division.
 Working-Storage Section.
 01  Passed-Pointer Pointer.
 Linkage Section.
 01  Int-Field  Pic 9(08).     *> I don't know how big an integer you are
getting
Procedure Division.           *> do NOT put Int-Field in a USING statement.
   ...
  Call "C-program" Using Passed-Pointer.        *> you may need returning
instead
*                            Returning Passed-Pointer    *> depending on how
the C DLL is defined
 Set Address of Int-Field to Passed-Pointer
 Display "The integer field is not 'addressed' and has the value:" Int-Field
```
