Hi again

It's near to previous solution....

Suppose you need to insert 10 space chars before some values:

WORKING-STORAGE SECTION.
01  strspace          pic x(10) value spaces.
01  strvalue          pic x(10) value "Yauyau".
01  strResult         pic x(20).

PROCEDURE DIVISION.
      STRING   strspace
                    strvalue          DELIMITED SIZE
                    INTO              strResult

You should see the result like: "          Yauyau".

Another way is insert a variable number of spaces like:

WORKING-STORAGE SECTION.
01  strspace          pic x value space.
01  strvalue          pic x(10) value "Yauyau".
01  strResult         pic x(20).

PROCEDURE DIVISION.
STRING   strspace    (1:10)   *> this means, put 10 spaces
              strvalue          DELIMITED SIZE
                    INTO              strResult


Hope in this help

The WITH POINTER clause of the STRING statement specifies the position to 
start stringing.

       MOVE SPACES     TO StringLine
       STRING "abc" 
          INTO StringLine
          WITH POINTER SpaceNo

As the pointer is updated it can be use to concatenate using several 
separate STRING statements.  Output areas for STRING and UNSTRING should 
always be set to an initial value, usually spaces.
