**** New Version Of Screens2 With Extra Facilities ****

o Screens2 is a tool that will allow you to paint a coloured screen

display using a 'standard' .dds file within you application

o The two cobol programs that make up Screens2 are available as SOURCE

code

o Screens2 functions in an almost identical fashion to the
Microfocus(tm) utilty screens as in toolbox

o The program will output the drawn screen in a similar format to
the output of the DOS/windows based forms utility

o The program works with Generic Display Attributes - this
allows you to decide what colour a partilcular attributes means
e.g. x"1f" can mean white on a blue background as default
but you can change it to be green on red. This facility
means an end user can re-colourize an application to
suit their own needs with no extra coding from you

o The SOURCE code needs compiling with MF Cobol 3.2 / 4.0

o Use the embedded 'C' and shell script routines to capture existing
screens directly into Screens2

o Automatically centre titles

o For details of price and other features available within Screens2
please e-mail : ni··.@ala··o.uk
