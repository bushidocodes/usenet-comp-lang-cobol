This is a Cobol Problem using NetExpress, not an HTML problem.

What you must do is go to Project Build Settings (ALT F7).
Select Link tab, and make sure that under Runtime Library you have select
static. I have tried this to create a standalone single program that does not
need the runtime libraries.

Otherwise, you must find the Runtime files that it need and copy them to the
server, but try to change the Runtime Libraries to static.
