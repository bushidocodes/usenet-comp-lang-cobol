I am doing some research on behalf of a client who is a Micro Focus
customer. They'd like to move users (several hundred) off a Unix
machine, but leave their indexed files there. They're considering using
Micro Focus's Fileshare capability to access the data from user's PC
(where the code will be running).

Does anyone have any experience with large numbers of users accessing
server-based data from Windows clients using this technology? Will
performance be adequate or will their users complain about poor
response?
