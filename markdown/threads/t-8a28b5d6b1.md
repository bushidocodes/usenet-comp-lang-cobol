Looks like homework, but....

To pass a table, define the table the same way in both the working storage
of the callING program and in the linkage section of the callED program.

If you define the table with a group item enveloping  the elements, as in...

     01  THE-TABLE.
           05 TABLE-ELEMENT     PIC X(01) OCCURS 100.

When you CALL:

CALL programname using THE-TABLE

Or, if the table is described without an enveloping group-item...

01  BIG-WS-AREA
     05 first-thing        pic whatever
     05 second-thing  pic whatever
     05 table-element  pic whatever  occurs 100

you can..

CALL programname USING table-element (1)


(This second method is not covered in textbooks very well. It works because
CALL USING passes the address of the parameter unless you go out of your way
to do otherwise. All the callED progam knows when it starts is that whatever
is that the parameter is an address where whatever is defined as the USING
parameter the PROCEDURE DIVISION header begins. Tables start at element
one).

MCM
