Two things:

1.  You must OPEN a file before you can read from it or write to
    it.  You did not OPEN the file in your program.

2.  If you need to create a text file, you probably do not want to
    use a file whose organization is INDEXED; you would probably
    want to use SEQUENTIAL or LINE SEQUENTIAL instead.  This will
    depend on the particulars of the program reading the file.  Note
    however, that if you change this, then your "Account Number Taken"
    logic will also need to be changed.
