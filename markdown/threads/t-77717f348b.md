```
ccitcp2 is the MF process that keeps track of how to locate fileshare
servers...there must be at least one CCITCP2 program running somewhere in
the network...I usually run it on the fileshare server machine...Unless it
is running, your apps cannot locate the server, and fileshare cannot be
started on the server...Using NT, both CCITCP2 and FSSERVICE can be set up
as services and started automatically on bootup...

kenmullins
```
