```
In case other have missed it, IBM (which is already competing with such
products as File-Aid, Xpediter, InterTest, etc) has announced a product that
(as I read it) is intended to compete with STROBE and InTune.  See:

 http://www.ibmlink.ibm.com/usalets&parms=H_202-065

which says (in part),

"Application Monitor collects samples from the monitored address space and
analyzes the system or resource application in the following categories:

  a.. Total address space utilization of all modules in the address space
  b.. CSECT utilization within each load module
  c.. Instruction or statement utilization within each CSECT
  d.. COBOL statements utilization within each COBOL module
If the DB2� option is installed, Application Monitor analyzes the following
DB2 categories for a DB2 application:

  a.. Total address space utilization of all modules in the address space
  b.. CSECT and SQL utilization within each load module
  c.. Detailed DB2 delay information "
```
