```
I am converting a bunch of old COBOL programs from CTOS platform to MF
COBOL to run in MS-DOS box under Win95. The programs make extensive use
of communications byte streams where system calls are used to define
baud rate and parity, raise/lower DTR, send/recived buffered data.

In the past I have used Microfocus DOS interrupt calls to send and
recieve single bytes by addressing proper MS-DOS ports. The compiler
manuals show lots of CBL_ calls, but nothing like what I need. Do newer
versions of MF COBOL have additional calls?

Is there a third party tool kit available that provides the
functionallity that I need?

Charlie Goodman
```
