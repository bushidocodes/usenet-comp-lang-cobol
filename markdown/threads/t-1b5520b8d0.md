```
As I'm learning COBOL, I already have a project in mind. I want to write
plain, text based programs, but I want to be able to run those programs
remotely over a terminal emulator, just like the real systems. I had one
idea. There are a couple of free COBOL compilers for Linux, OpenCOBOL and
TinyCOBOL, does anyone have any experience with these?. These compilers
generate C code which is then compiled for Linux. I guess the COBOL programs
run in the Linux console, in which case I could use PuTTY remotely. Then I
could even run VIM, edit and compile my code anywhere! TinyCOBOL makes a
Windows version too, but I don't know how you would host the programs over
the internet.

Once I lay this framework, it will be fun to build my system up. I can keep
it general purpose, where users can customize their system by storing
preferences in data files. The options are endless.

Matt
```
