It is not really targeted toward total beginners, but you might try Sams
new book COBOL Unleashed.  It has a complete COBOL 85 syntax chart and
comes with a CD containing lots of code and good stuff, such as Fujitsu's
COBOL v3 compiler and documentation.  There is also a PDF file of the
COBOL 85 syntax on the CD.  You can buy COBOL Unleashed from The Bookpool
www.bookpool.com for $32.50 (35% off) plus shipping.
