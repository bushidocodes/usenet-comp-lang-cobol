```
If I understand you correctly, you have linked your program using
RealDLW.LIB, which is used to create executables for Windows from
DOS-style programs. This method emulates the DOS environment.

There are no facilities to manipulate fonts directly with Realia or the
ScreenIO products. I believe you can change the font attributes from
within the Window your DOS application is executing from within.

It really sound like it may be time to convert your application to a
true Windows applications. I use CA-Realia Cobol and the Flexus Sp/2
product for my GUI's to do this with great results.

Mike Dodas
```
