+ TF-TT.

/ 100.
COMPUTE EDIT-STOP =(TIME-STOP-SECONDS - TIME-START-SECONDS)
/ 100.
DISPLAY " INPUT ", EDIT-INPUT,
" ELAP ", EDIT-STOP.
STOP RUN.

SORT-INPUT-RTN-01 SECTION 03.
SIR-SECTION-01.

MOVE IN-C TO SF-2.

GO TO SIR-SECTION-01.
END-INPUT-01.
ACCEPT TIME-INPUT FROM TIME.
SORT-OUTPUT-RTN-01 SECTION 04.
SOR-SECTION-01.
RETURN SORTFILE AT END GO TO END-OUTPUT-01.
MOVE SF-BODY TO OUT-FILE.
WRITE OUT-FILE.
GO TO SOR-SECTION-01.
END-OUTPUT-01.
