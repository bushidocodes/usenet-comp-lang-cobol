As long as we are now WELL past the two-hour limit and I *hope* I am not
doing someone else's homework,   a few changes from this reply (probably
correct for the '85/'89/'91 Standard) that reflect enhancements in the draft
Standard (expected to be final by the end of this year)

 - variable_name is now valid (with underscore)
 - Pointer    (yes)
 - Boolean expressions (b-xor, b-not, etc)
 - Recursion  (yes)
 - Parameter-passing methods     (BY REFERENCE, BY CONTENT, BY VALUE)
 - Functions    (both intrinsic and user-defined functions supported)
 - Exception handling        (many types including user-defined and
implementor-defined "exception conditions")
 - Object oriented support    (yes)

I probably missed some other changes based on enhancements
