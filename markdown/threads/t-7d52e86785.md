```
Announce: TinyCOBOL release 0.58

The TinyCOBOL compiler version 0.58 is available at the TinyCOBOL home
page (1) and iBiblio (2).

This release contains mainly bugs fixes, and some enhancements.  
It includes updates to the main compiler, run-time, the regression and
NIST test suites.

Support for inline comments, DEBUGGING lines, declaratives
(preliminary), intrinsic functions (preliminary), INSPECT CONVERTING,
concatenation (preliminary), have been added to the compiler.

The compiler front end and resource options have been enhanced to to
define the default compiler action (create executable or library). The
compiler can also create an static/shared libraries, from COBOL sources,
directly from the command line.

And finally, embedded SQL can be used with TC, by using a vendor
supplied SQL pre-processor and libraries.


Please report any bugs and/or problems to the TinyCOBOL mailing list. 


1) TinyCOBOL Home Page
   http://tiny-cobol.sourceforge.net/
   http://www.tinycobol.org/

2) iBiblio (formerly Metalab)
   http://www.ibiblio.org/pub/Linux/devel/lang/cobol/
```
