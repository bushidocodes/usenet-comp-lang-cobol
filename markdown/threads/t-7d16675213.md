File organization for COBOL varies with the platform on which you are
running (MVS, VMS, PC, etc.), what brand of COBOL you are using (IBM,
MicroFocus, CA-Realia, DEC), and what type of file definition you did
(VSAM, QSAM, ISAM, KSDS, ESDS, RRDS, record sequential...). While the way
in which COBOL looks at an individual record is basically similar once it
is in program torage, the physical storage on disk, tape, etc. varies
wildly. Therefore, you must ask your question in the context of a
particular platform, compiler, and file definition to get a meaningful
answer.
