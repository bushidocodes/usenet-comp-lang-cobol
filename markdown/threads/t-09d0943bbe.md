```
The data values are only placed on the screen when you use the DISPLAY
verb to display the screen.  'FROM' fields are output to screen only,
'USING' fields are both input from and output to the screen, and 'TO'
fields are input from the screen only.  No data fields are displayed
by the ACCEPT verb, even USING or FROM fields or VALUE literals.
```
