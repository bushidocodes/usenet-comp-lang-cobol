You'll need 8 meg of Ram, but if you have it, the CD that is in the
book "Sams Teach Yourself COBOL in 24 Hours" includes the 16 bit
version of the Fujitsu COBOL compiler, and the book covers how to
install and use it.

explain any differenced between that and the 16 bit Windows 3.1
version).
