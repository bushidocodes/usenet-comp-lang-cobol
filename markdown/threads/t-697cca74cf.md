- **Message-ID:** `<3A08D6C7.12C247B2@arvotek.net>`

```
There appears to be several file extensions (.cob|COB .cbl|.CBL) used by
COBOL source files.

It appears that source files with (.cbl|.CBL) file extensions are
usually in fixed (1-80) format, while source files with (.cob|.COB) file
extensions are usually in free format.

Is this is a commonly accepted standard or just just a guideline
followed by some compilers ?


David
```

the extension and the fixed/free form of the source code, on any
compiler I know of.  You use compiler options to specify to the compiler
which form you are using.
