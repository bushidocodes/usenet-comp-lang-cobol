Hope this helps.
--------------
       program-id. port300.
       data division.
       working-storage section.
       01 port-addresses.
         03 port-301 pic 9(5) value 301. *> b+1
         03 port-303 pic 9(5) value 303. *> b+3
       01 output-values.
         03 relay-1-on pic x value x"80".  *> 128
         03 relay-2-on pic x value x"08".  *> 8
         03 relay-5-on pic x value x"40".  *> 64
         03 relay-off pic x value x"00". *> 0
       01 keyboard-input pic x.
       procedure division.
           display space upon crt. *> CLS
           call x"88" using port-303 relay-on. *> OUT

       start-of-loop.
           display "ON". *> PRINT
           call x"88" using port-301 relay-1-on. *> OUT
           accept keyboard-input. *> INPUT$ not the same as INPUT$(1)

           display "OFF". *> PRINT
           call x"88" using port-301 relay-off. *> OUT
           accept keyboard-input. *> INPUT$

           go to start-of-loop. *> GOTO
           stop run.
