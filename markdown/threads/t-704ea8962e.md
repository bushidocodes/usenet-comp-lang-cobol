A little promotional piece, begging your indulgence:

Sunday Feb 27 I will be teaching an all day
COBOL Boot Camp at SHARE in Annaheim.

If you, or a colleague, want a fast but
thorough introduction to modern COBOL,
consider picking it all up in one day.

This will be lecture / discussion / examples,
but no hands on labs. All content is based on
the current Enterprise COBOL compiler. Big
picture of the agenda:

COBOL Basics
  Character set, coding rules
  Program structure
  Data types
  Editing
  File processing
    sequential
    VSAM files and random processing

COBOL Works
  Calling subroutines
  various ways of passing data
  static vs dynamic calls
  Intrinsic functions
  Tables / arrays
  String handling

Modern COBOL
  Unicode support
  XML support
  Possible callable routines universe
    Language Environment
    C functions
    z/OS UNIX kernel services
    DLLs
  Object oriented COBOL and Java-COBOL interface

So if your organization is sending folks to SHARE
in Annaheim, give some thought to whether someone
in your group can benefit from this fast paced
look at COBOL.

Thanks for your consideration.

Kind regards,

-Steve Comstock
The Trainer's Friend, Inc.
