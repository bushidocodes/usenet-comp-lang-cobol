Siber Systems sells a library that you can use to access
Cobol data files produced by MicroFocus compilers.
You can read more about it at http://www.siber.com/sct/datafile/

The library interface is in C++, but I understand that
accessing C++ libraries from Java or SmallTalk is a lot
easier than reading the Cobol data directly.

You can download some of our Cobol-data-file to everything-else
converters from our site to see if they can read your data.
If they can, then the library will read them too, because
these converters are based on this DataAceess library.
And if they cannot, then we want to see your data
to make sure we can read it.
