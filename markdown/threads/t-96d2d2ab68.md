18.3: What's a free or cheap C compiler I can use?

A: A popular and high-quality free C compiler is the FSF's GNU C
 compiler, or gcc.  It is available by anonymous ftp from
 prep.ai.mit.edu in directory pub/gnu, or at several other FSF
 archive sites.  An MS-DOS port, djgpp, is also available; it can
 be found at ftp.delorie.com in pub/djgpp, or at the various
 SimTel mirrors (e.g. ftp.simtel.net in pub/simtelnet/gnu/djgpp;
 ftp.coast.net in SimTel/vendors/djgpp).
-------

You might also consider MS VC++ 6.0 Standard. It is priced
at about $100 US for the compiler and IDE. The support provided
by the IDE might save enough of your time to justify the expense.

I never heard of COBOL++, but if you want a simple program to
statically link to a FJ PowerCobol project, you can use the COBOL97
compiler that came with PowerCobol, generate OBJ files, put 'em
in a LIB file, include the LIB file with the PC link. Works swell.

Further, copying a file from within a COBOL program can be
accomplished with CBL routines.
