First check out www.merant.com for complete details on all the MERANT COBOL
(and other) products.

However, "Personal COBOL" is still available, but I believe is
"strategically" replaced by their "Academic" products.  This is a
(relatively) limited product.

The current "strategic" MERANT/Micro Focus COBOL products are (with
APPROXIMATE target audiences):

    Net Express V3.1 (for PC development AND deployment of applications)
    Server Express (primarily run-time support for UNIX-like production
systems)
    Object  COBOL Developer Suite (development environment for UNIX and LINUX
environments)
    Mainframe Express (for PC development of IBM mainframe targeted
applications)

P.S.  FYI - all of this is outlined in a little more detail in the COBOL FAQ
at:
 http://cobolreport.com/faqs/cobolfaq.htm
