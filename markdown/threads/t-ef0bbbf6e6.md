Most errors of that type are because the source contains actual errors
which are allowed by some other compiler.  That is you have not
constructed your program correctly but the compiler you used didn't
enforce correctness.  Alternately you used extensions that were not
implemented by Fujitsu.

Get the downloadable Fujitsu manuals.  The error message will tell you
the line number, look at that line in the source code, compare that
with the syntax in the manual, see where the code is wrong - or should
be different.  This is a basic programmer skill which you should be
trying to develop before moving on to more advanced issues.

> Any other complier I could try? 

Fujitsu is OK.  There is a free student edition.  For others, how much
are you willing to spend ?
