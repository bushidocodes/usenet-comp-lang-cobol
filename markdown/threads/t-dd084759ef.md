```
Due to some recent security tightening at PRIMA, some of you who
downloaded the free COBDATA tool
(https://primacomputing.co.nz/PRIMAMetro/newCOBData.aspx) may find it
denies access if you haven't used it for a while.

I apologize to all concerned. (It is caused by our move to https: and
really has nothing to do with the actual package...)

It can be fixed by copying the file:
https://primacomputing.co.nz/downloads/COBDATAfix/RAVSAF-COBDATA.bin to
the folder C:\RAVComponents\xmlfiles, on machines where the tool is
installed.

Rename it to be RAVSAF-COBDATA.xml, replacing the existing file with
that name.

(I simply don't have time at the moment to fix the install, so if you
install this tool fresh, you should run the above fix.)

Cheers,

Pete.
I used to write COBOL; now I can do anything...
```
