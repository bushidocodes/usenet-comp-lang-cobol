to answer your question: yes, siemens mainframes are 390 machines - at
least the 'big ones', which are manufactured by Fujitsu, so they should
be real PCMs; the 'small mainframes' are risc machines, emulating 390
Code or translating it to risc code. The operating system used on these
machines is OSD V3, which has been developed out ouf BS2000, which
itself has ist roots in a RCA timesharing system from about 1969.
Besides, unix systems on risc processors and NT systems on intel
processors are offered by SNI (note: the siemens computing buisness has
been concentrated in 1990 in a separate firm called Siemens Nixdorf
Informationssysteme, short SNI).
For further information you might contact the web site
http://www.sni.de/public/sni.htm
