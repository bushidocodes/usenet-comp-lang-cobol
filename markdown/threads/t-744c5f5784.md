Not sure exactly what editing you need to do.
You might consider using a redefine or level 88 under the variable.

Example:

05  ARRIVAL-AS-PROMISED     PIC X(1).
       88   ON-TIME    VALUE 'Y'.
       88   LATE          VALUE 'L'.
       88   LOST          VALUE 'Z'.

05  PHONE-NUMBER     PIC X(10).
        10  PHONE-AREA-CODE       PIC X(3).
        10  PHONE-PREFIX                 PIC X(3).
        10  PHONE-LAST4                   PIC X(4).
05  PHONE-IN-COMPANY REDEFINES PHONE-NUMBER
        10  FILLER                                 PIC X(5).
        10  COMPANY-PHONE           PIC X(5).

Gene

You can enforce various PICTURE clauses if defined in a SCREEN SECTION.
(e.g, PIC 999,999 will force the user to enter digit,
digit,digit,comma,digit,digit,digit).

Other than that, I think all you can force COBOL to handle is class
condition (e.g., IF X IS NUMERIC|ALPHABETIC..)
