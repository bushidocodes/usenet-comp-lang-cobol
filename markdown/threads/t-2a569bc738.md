Tool announcement: Grammar Recovery Kit 1.0 released

GRK --- Grammar Recovery Kit --- is a software toolkit for semi-automatic
grammar recovery. For example, using GRK one can build an authoritative
Cobol definition from IBM's definition. GRK is provided with a BSD-style
license. It is implemented in 99.99% SWI-Prolog and 0.01% gmake.

http://www.cs.vu.nl/grammars/grk/

Ralf Laemmel
CWI & VU Amsterdam
