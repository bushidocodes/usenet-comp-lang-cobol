I think that Fujitsu Cobol comes with a MF to Fujitsu converter called
MFTO85 on the START menu. Otherwise it is known as MF Cobol/2 Source
Conversion Utility. This contains all the help text you should need.

Fujitsu Cobol and PowerCobol come with their own manuals online (or you
can download them from www.adtools.com). These include quickstart
guides.

Fujitsu Cobol is Cobol 2 (i.e. Cobol 85) compatible so any book about
Cobol II syntax from IBM should help.
