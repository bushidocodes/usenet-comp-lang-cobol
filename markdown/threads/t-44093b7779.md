```
According to,
  http://publibz.boulder.ibm.com/cgi-bin/bookmgr_OS390/BOOKS/dfhs1p01/2.1.2

"The first thing that the transaction abend code can indicate is whether or
not this was a CICS abend. CICS transaction abend codes begin with the
letter "A". A user program or another product might also use abend codes
beginning with "A". However, if the transaction abend code begins with
anything other than "A", it is an abend code belonging to a user program or
to some other product. For the sake of convenience, all such non-CICS abend
codes are referred to in this section as user abend codes."

Therefore, this sounds like it is either an application ABEND code - or an
ABEND code from some other product.
```
