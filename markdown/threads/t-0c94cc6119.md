--------------------------------------------------------------------
Q. How do I upload something to Demon Internet Services' ftp site?

A. Use FTP. Uploads go into /incoming on ftp.demon.co.uk.
You then need to notify us by sending mail to "upl··.@de··n.net".
Please include:
The filename
size
a one line description (start the line with 'Short: ')
a multi-line description
a suggestion as to which directory the file should be placed

It is also recommended that you provide a copy of the message with the
upload. Try to keep names similar, so xyz33.zip would have xyz33.txt or
xyz33.readme.

/pub/amiga, antivirus, archimedes, atari, diabetes, ibmpc/winsock and
riscpc all have their own "incoming" directories; please use these when
appropriate, instead of /pub/incoming.

Problems to int··.@de··n.net
