One other little item I'd like to post is the fact that COBOL's "RETURN-
CODE" is treated by IBM COBOL as a HALFWORD, S9(04) BINARY. So, if
anyone has ANY ideas of using R15 as a means to pass back a fullword-
value to the Caller, forget it, because only the low-order halfword is
addressable. Therefore, the greatest value for this purpose (and I'm
not advocating this by no means) is H'32767'.
