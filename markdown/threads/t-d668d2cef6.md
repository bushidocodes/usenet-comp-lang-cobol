Phil Mitchell wrote:
›
› I was just curious, what language are the COBOL compilers you work on written in? On
› Tandem, I would guess C or Assembler?? Does anyone write compilers in good ole'
› COBOL?

The compiler is written in TAL, a proprietary langauge that is sort of
like Pascal (and sort of like C and some other stuff as well). I've
written others in assembler (about three different assemblers), CYBIL
(a Control Data language very much like Pascal - the best
implementation language I ever used), Ada, SYMPL (a subset of Jovial),
and now will be using C and C++ (much to my dismay since I hate them
both).

Yes, Micro Focus wrote most of their compiler in COBOL. I think
Realia (now CA) did as well. I've also heard that a few others did,
too.

Don Nelson
