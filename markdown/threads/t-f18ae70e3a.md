```
For what it's worth, if you WANT to use COBOL and want to "trim a string", 
contact your "vendor of choice" and ask them if (and when) they plan on 
implementing the TRIM intrinsic function defined on page 702 of the current 
draft of the next COBOL standard.  It has "leading" and "trailing" optional 
phrases and if you specify neither, then it trims from both ends.
```
