One solution is to code a gained-focus event in the window.  Problem? 
Ds uses a heirachy to process events.  If the event is not coded on 
each object it will look in the window dialog then the global dialog 
for the event.  When necessary, where I used to work, we coded a dummy
gained focus on each object that did a noop, then we coded a gained 
focus event on the window.
