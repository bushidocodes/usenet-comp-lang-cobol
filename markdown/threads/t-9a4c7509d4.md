The first four bytes of an IMS message must be LLZZ, where LL = the length
of the message (PIC S9(04) COMP) and ZZ = binary zeros.  ZZ is reserved for
the use of IMS, and *must* contain binary zeros (LOW-VALUES) when the
message is inserted.  If ZZ contains any other values, you get an 'XC'
status when you insert the message.
