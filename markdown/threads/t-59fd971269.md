There are TWO display screens in Fujitsu, which can be a bit confusing.
I use one as an operators console, and the other as a "screen". Common
display statements go on one, and Screen section stuff goes on the other.
I think you are specifying the wrong one.  Try DISPLAY SPACE AT
LINE 1 COLUMN 1, and I think you will get the correct screen to blank. You
night also check the word CONSOLE as opposed to CRT.  There are ways
to set which of the two are used in the SPECIAL NAMES section. (I am using
GUI, and only use the console/screen for debugging).  Back when using
screens, I always used a SCREEN SECTION, which will also work.
