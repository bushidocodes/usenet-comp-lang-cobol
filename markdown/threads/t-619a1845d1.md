Mark,

You can use embedded SQL within Micro Focus COBOL programs. To compile the
source you need a pre-compiler and these are available from the major
database vendors (Oracle, Informix etc). In the case of Microsoft SQL
Server and DB2, Micro Focus provides the support.

Our Visual Object COBOL and soon to be released NetExpress product lines
include an ESQL to ODBC pre-compiler (called OpenESQL) which allows you to
access any database that has an ODBC driver. In the initial release of
NetExpress we have officially tested SQL Server and Oracle but others will
follow.

I hope the above is of help.

Paddy Coleman
PC Client/Server Support
Micro Focus Europe.
