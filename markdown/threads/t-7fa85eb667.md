> sqlcheck=full

Oracle checks the SQL against the database when this directive is used.
If it doesn't have access permission (Userid=regu34/aut) then it is
going to give some sort of error.

Perhaps that AIX machine isn't in the list of machines allowed to
access, but the Windows machine is.
