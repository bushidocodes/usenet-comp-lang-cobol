<snip program>

After the "accept data-entry-screen" the fields in working storage 
(screen-items) will contain the data entered.

You will have to define a file (a select statement and an FD) with 
fields in the data record that will be used to store the data. You 
have to open the file as ouput, move the working storage data fields 
to the FD's record fields, write the record to the file and then close
the file.
