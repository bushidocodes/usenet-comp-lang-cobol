I don't believe 999v12 is a legal picture clause.  Did you perhaps mean 
999V99?

A level 77 data item in COBOL is always an elementary item.  In some 
early COBOL compilers a level 77 item might have been allocated to a 
different type of memory.

A level 01 data item can be an elementary item but is usually a group 
item.  It can be subdivided into lower lever group and/or elementary items.

77  77-DATE            PIC X(08) VALUE SPACES.

01  WS-EDITED-DATE.
     05  WS-YEAR        PIC X(04) VALUE SPACES.
     05  FILLER         PIC X(01) VALUE '-'.
     05  WS-MONTH       PIC X(02) VALUE SPACES.
     05  FILLER         PIC X(01) VALUE '-'.
     05  WS-DAY         PIC X(02) VALUE SPACES.

In the example above, 77-DATE, WS-YEAR, WS-MONTH, and WS-DAY are all 
named elementary data items.  They are not further subdivided. 
WS-EDITED-DATE is a group item, containing five elementary items.

First, "v12" and ".12" are illegal.

If you are asking about the difference between a "v" and a "period," the 
difference is:

"v" asserts the variable is used internally for computation.

"period" is actually a decimal point and is used when the variable is to be 
viewed by a human.

In general, "v" variables look like crap when you try to print or view them 
and "decimal point" variables cannot be used in computations.

Use one or the other depending on the sentence.
