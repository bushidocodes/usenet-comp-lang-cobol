Agenda
8:00 Registration and COBOL Magic

9:00 Presidents Message - Fred Young
The benefits of GLAMUG membership - Sigs (special interest groups), Input into
the COBOL 97 standard, Job Board on our web site (coming this year), Electronic
Newsletter (E-Mail required), Video conferencing with developers, Video
presentation lending library (being defined).

9:30 Keynote Address - Marcelo Gumucio
The new chairman of Micro Focus will share with us his vision of the future of
applications development, and the role Micro Focus will play. There are a lot
of new things happening at Micro Focus, so don't be surprised if there are a few
major announcements.

10:45 A Micro Focus Technology and Directions - Clive Beavis
As the Vice President of Development, Mr. Beavis will present the Micro Focus
enterprise Client/Server and GUI strategy. Learn how to move to the future with
Micro Focus products.

10:45 B Object Oriented Training - Edmund Arranga
This session begins UCLA Professor Edmund Arranga's full day of Object COBOL
Training. Learn the syntax and semantics of Object-Oriented COBOL including
classes, objects, and message passing.

11:45 Lunch

1:30 A How to write good COBOL - Jerry Garfunkel
Dr. Garfunkel is one of the foremost authorities on the COBOL language, also a
consultant and educator. His is able to present a complex topic, in a very
understandable manner. This session is a must for anyone who writes COBOL. Dr.
Garfunkel delivers one of the most informative, entertaining, and useful
presentations in the business. This is a must see for anyone involved in COBOL
Application Development.

1:30 B Object Oriented Training - Edmund Arranga
UCLA Professor Arranga's full day of Object COBOL Training continues. Following
this session, you will Understand how inheritance, polymorphism, and abstract
classes are implemented in OO COBOL.

2:45 A Revolve, an amazing productivity tool - Mickey Rosen
This is the productivity tool used by every programer every day. It's not just
for Date 2000 issues, it is a tool to help programmers understand existing
systems. It will visualize and explain large software applications.

2:45 B Object Oriented Training - Edmund Arranga
Professor Arranga's full day of Object COBOL Training continues. In this
session, you will be exposed to the Object Oriented COBOL class library and
learn how to create collection classes.

4:00 A COBOL and the Web - Gary Crook
Beginning with Visual Object COBOL, and perhaps moving into workbench, COBOL
syntax will support HTML forms and other extensions directly. Find out from the
expert, how all this fits together.

4:00 B Object Oriented Training - Edmund Arranga
In this session of our full day of Object Oriented Training, you will
participate in a small object-oriented analysis and design session and learn why
programming with objects is not necessarily object-oriented programming.
