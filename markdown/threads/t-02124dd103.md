```
Depending on the version of rebuild (and the file layout) you are 
using you could try the "/D" switch for the rebuild command.

rebuild originalFile.dat, newFile.dat /D

This will copy the original file to the new file and create an index 
by reading the data records only.

After it is finished , make a copy of the original .dat and .idx files
and copy the newFile .dat and .idx files to the original file name.
```
