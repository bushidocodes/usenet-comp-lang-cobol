>Is there a way to add to the value of a variable or am I way off the mark?

Yes, you can add to a variable, if it is in DISPLAY format.  In this
case, you could use the STRING verb.

As an alternative, you might want to write your results to a work
file.  Then, when the user is finished the times table exercise,
re-read the information in your work file, and, for each one that is
incorrect, display an appropriate message.

If the total number of questions in the dialog is small, then you
could store the results in a table, instead of using a work file.
Process the table similarly.

Good luck with your program.
