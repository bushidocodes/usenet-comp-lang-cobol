```
Morning,

Below is a very simple example using Micro Focus COBOL v3.4 [16bit].

The application works in .int/.gnt/.exe.

Stephen Gennard
Just a developer @ Micro Focus

$ set ans85 noosvs mf defaultbyte"00" case
identification division.
program-id. wexec.
special-names.
call-convention 11 is WINAPI.
data division.
working-storage section.
01 myHandle pic s9(4) comp-5.

01 mypreloadptr procedure-pointer.
procedure division.

set mypreloadptr to entry "user"
set mypreloadptr to entry "gdi"
set mypreloadptr to entry "kernel"

call WINAPI "WinExec" using
by reference "notepad" & x"0",
by value 1 size 2
returning myHandle
end-call

if myHandle < 32
display "Failed to start notepad"
else
display "Notepad started!"
end-if
```
