Key word is "syntax". The general syntax for a SQL "SELECT" in COBOL
is:

EXEC SQL SELECT columnname <,columnname...>
...>
FROM tablename

...>

...>
END-EXEC.

I wish I knew more but all my SQL coding have been in SAS and
MS-Access. Perhaps someone more experienced could post a general
skeleton of a SQL COBOL program to illustrate the features needed for
such programs (such as DECLARES, FETCH, SQLCA, etc., placement).

Hope this helps some,
