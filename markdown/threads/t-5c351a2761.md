```
I have a simple little problem.

I want to move a certain number of bytes to a field with out changing the
rest of the 'to' field.  I must be able to move the bytes to any position on
the 'to' field without altering the rest of the 'to' field.  Normally, I
would use a string command with a  pointer but on the AS/400 there is a 32K
limit on the 'to' field (this limit does not exist on MVS) and this field is
50K.    Any ideas on how I can do this?

Why does the AS/400 COBOL put this limit on the String command?  The from
field does not have it..only the 'to' field....
```
