int subp(struct record-details)

int some_var = record-details.varible;

C language is standardized
COBOL language is standardized
C to COBOL interface is not standardized
Details vary by implementation.
The COBOL side may be able to handle C-speak, but may not.
The individual compilers and run-times involved will set the rules.

To help the environment (operating system), the C compiler, the COBOL
compiler, and the respective run-time packages must be known.
