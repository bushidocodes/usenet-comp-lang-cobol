It seems that I've overlooked the most obvious place to look at. So

around with TCP/IP socket programming with COBOL on z/OS:

   1. CS for OS/390 TCP/IP Implementation: MVS Applications
      http://publibz.boulder.ibm.com/cgi-bin/bookmgr_OS390/BOOKS/ez3aby01/CCONTENTS
   2. MVS TCP/IP Sockets
      http://publibz.boulder.ibm.com/cgi-bin/bookmgr_OS390/BOOKS/ez306700/CCONTENTS
   3. TCP/IP Tutorial and Technical Overview
      http://publibz.boulder.ibm.com/cgi-bin/bookmgr_OS390/BOOKS/ez306201/CCONTENTS
   4. TCP/IP V3R2 for MVS: CICS TCP/IP Socket Interface Guide
      http://publibz.boulder.ibm.com/cgi-bin/bookmgr_OS390/BOOKS/EZAP1004/CCONTENTS?DT=19970730010745

In addition there are some more documents especially for TCP/IP
programming techniques under CICS or IMS respectively.
