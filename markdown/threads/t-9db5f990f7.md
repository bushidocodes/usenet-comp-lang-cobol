Minora wrote:
> 
> Are there any good freeware Cobol editors available for windows.

IIRC there was an editor that came with the free Fujitsu version 3
compiler.  It was adequate but not good.

If you will do a lot of programming then invest in the best chair and
the best editor that you can find because you will spend a lot of time
in both.  You could work in notepad and sitting on an orange box, but it
is not a good idea.

Of course, what is 'best' depends on what you are used to and what you
have trained yourself to use.

I have used WordStar like editors for 20 years (WS, Turbo-Pascal, rtc)
and so am most comforatable with that basic key set.  Others prefer
vi-like or emacs-like keys.  Get an editor that can be configured to any
key set and try out all of them.  Definately get one that doen't _need_
the mouse.

I use TSE and have it configured with colouration for Cobol, short keys
for most reserved words (eg  F9-P giver PERFORM, F0-V gives VARYING).

Many editors have limited time or limited facility versions for
downloading.  For example www.semware.com have a yestdrive version of
TSE for DOS or Windows that you can download.
