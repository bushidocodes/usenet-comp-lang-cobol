Hello,

It sounds like your comms functions are NOT protect mode
enabled, which is a requirement for XM.

You thus have three options:
o make the functions protect mode aware
o use the XM APIs that allow you to call a real-mode function
these are: PC_XM_INSTALL_REAL_MODE_COBOL, PC_XL_CALL_REAL_MODE_COBOL,
PC_XM_CANCEL_REAL_MODE_COBOL
(Technical Support should be able to help you with these functions)
o get some new functions...

The second option might be you quickest solution.

Hope this helps.
