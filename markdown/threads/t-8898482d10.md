3. stack check disabled

Not sure you'll be able to do this.  The specs you give (e.g. Model Large) imply that it 
expects something from an earlier C or C++ system. Back then you had the ability to 
specify the "memory model" (essentially the addressing scheme and data segment 
locations) as "tiny", "small", "large" and "huge".  I don't see that ability in the 
instructions or help file for C++ Builder and don't know which one it defaults to 
(possibly none of them as it is a 32 bit only system and won't run in a 16 bit system). 

Additionally, it is more like Delphi in nature, but with the event handlers written in 
C++ rather than Pascal.

Probably nobody ever asked either of them your question before and they are still 
thinking about it.
