I've written two DDS records to a display file. I have update capable
fields on both record. When I read the display file, I'm able to update
both records but the data returned from the read is only from the last
record written (see code) if any one could give a possible solution to
this. Thanks in advance.



WRITE DISPLAY-FILE-REC FROM SFKEYSCR-O
FORMAT IS "SFKEYSCR"
INDICATORS ARE INDICATOR-TABLE
END-WRITE.

WRITE DISPLAY-FILE-REC FROM SFOPTION-O
FORMAT IS "SFOPTION"
INDICATORS ARE INDICATOR-TABLE
END-WRITE.

READ DISPLAY-FILE INTO SFKEYSCR-I
FORMAT IS "SFKEYSCR"
INDICATORS ARE INDICATOR-TABLE
END-READ.
