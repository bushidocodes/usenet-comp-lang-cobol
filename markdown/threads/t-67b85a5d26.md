```
On Mon, 23 Aug 2004 14:44:24 -0500, "JerryMouse" <nospam@bisusa.com>
wrote:

>Does anybody have a process to install PC and applications in an "All Users"
>environment, particularly on XP?
>
>We've got one user with ten employees and six machines. Our current
>technique - the only one we know - necessitates 60 installs for his shop.

You want to put the ten users into a Group, which is authorized to run
the application. Go to Control Panel\Users\Advanced\Advanced\Groups to
define a group and add users at the same time. Next go to
Explorer\right click on directory\properties\security\advanced to add
the group. If the Inherit checkbox is selected on subordinate files,
they will all inherit group from the directory.

Better yet, if the app is not copy protected, install it on one
machine and let everyone run a single copy. If it doesn't do
record/row locking, centralize only the executables and read-onlys.
That reduces the number of installs from 60 to 1, simplifying futue
upgrades. 

Ease of installation is WHY people run Terminal Server and Citrix.
```
