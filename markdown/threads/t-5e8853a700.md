The fact that it is no longer supported or maintained does not mean
you can not ask (and GET) the file layout. You do not need (normally)
to have any source program, but only the FD. If you can get this
information then it will be easy for a COBOL programmer to create
a program to extract the information.

also Relational Databridge ( and a very good programmer).
