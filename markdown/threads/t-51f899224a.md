Ben,
When calling the dll, pass the variable by reference and not by value.
This will pass the pointer to the variable and the function will figure out
it's data type. If this isn't working for you, trying moving the variable
to a float(comp-1) or double(comp-2) data type variable and then passing it
by reference. Good luck.
ie. call "MyDll".
call "MyDllFunction" using by reference MyDecimalValue returning
ReturnValue
on exception display "Not working!"
non on exception "It works!"
end-call.
cancel "MyDll".
