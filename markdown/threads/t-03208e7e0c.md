Dear Sir,

The COBDIR and PATH environment variables should be set as follows:

set cobdir=c:\cobol\lbr;c:\cobol\exedll;
set path=c:\cobol\exedll;<...>

Where <...> are the other settings on your PATH e.g. c:\dos. You also need
to substitute the correct drive and directory names in the above.

Hope this helps.
