Floyd, 

The colorization options are stored in the registry here:


\\HKEY_CURRENT_USER\Software\Fujitsu\PowerFRAMEVIEW\PFV\COBOL\COLOR

The values for the different compiler versions are different.

So - one way to fix this is to uninstall - then clear this registry entry - then
reinstall.  I would in fact delete the whole Fujitsu Tree.

Alternately - I will email you a .reg export file.  You can delete this branch
then import the .reg file.
