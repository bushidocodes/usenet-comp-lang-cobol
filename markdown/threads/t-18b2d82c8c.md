from the deskjet manual ? is:

0 = line printer, 1= pica, 2=elite, 3=Courier, 4=Helv, 5=times roman,
6=gothic, 7=script, 8=prestige, 11=presentations, 85=universe condensed.

stroke weight is bold=escape '(' 's' '3' 'B', extrabold= escape '('
's' '7' 'B', normal= escape '(' 's' '0' 'B'.

size is determined by escape '(' 's' # 'V' where # is the vertical size
in 72's of an inch.
