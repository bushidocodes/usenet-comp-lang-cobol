If you're on UNIX, try the "strings" command on the object file. Or,
you can hex-dump the object file: RM/COBOL object files contain the
the date and time compiled (in ASCII), the string "RM/COBOL-85", the
compiler version, and the platform where the portable object was created
starting at offset 0xCF in the object file. The file suffix defaults
to ".cob", but can be changed.
