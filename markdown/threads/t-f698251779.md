First you have to know what the problem is.

I've mentioned it before, maybe it was too simple to try.

Set the terminal's serial line speed to 'real slow', 110 or 300 baud, 
and then watch the screen display.  You'll be able to see everything 
that happens.  Perhaps some unnecessary and bandwidth clogging stuff is 
there.

When using a PC or whatever with a terminal emulator, you're using 
ethernet, much higher speed, and that can mask plenty of things.
