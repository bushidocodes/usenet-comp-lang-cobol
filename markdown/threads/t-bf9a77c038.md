Thane:

You are absolutely correct.  A web browser interface is a step
backwards.  It was never designed to be an interactive user interface
environment and we are all waiting for it to mature into one.

I agree that a thin client Internet architecture is a far superior
environment than a web based Internet architecture.

In fact, Microsoft is starting to crow about it as well.  If you
access the Microsoft web pages and search on Windows NT Terminal
Server Edition, lo and behold, Microsoft actually endorses the use of
a thin client architecture.

....but I *thought* that only a year or two ago, they proclaimed the
web to be the latest and greatest business application
architecture??!!??

I don't want to climb on a soapbox, but it really does make sense to
look at the alternatives to a web based application when you consider
running across a TCP/IP network......Internet or intranet.
