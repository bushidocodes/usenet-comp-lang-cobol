Passport has a "Power Page" feature designed for reading long lists.  It
sounds like you may have it turned on.

When you press the PF8 key the first time it only gets the page data, when
you press it a second time Passport displays the page and request the next
one.  This way you can read the first page while the second page is coming
over the wire.

You can turn this on and off via the menus. I forget which one.  My version
of Passport is at home so I can't check right now.
