In general, you can not use the serial port in the manner you describe.
To use the serial port, you program the UART and the UART handles the
input and output of serial data automatically. You don't, in general,
tell the serial port 'raise this signal' or 'lower that signal'.

The parallel port, on the other hand, is very well suited to this kind
of use. The signals are, for the most part, raised and lowered by
command from the CPU. And since you have parallel data, there are a
number of output pins and input pins which you can use. The signal is
TTL level, which makes it easy to interface. You would need to build a
switch for the 24v power, which you would need to provide.
