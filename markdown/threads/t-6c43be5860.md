(from memory)
It sounds to me like you don't have write permission to the directory
containing filename.$$$.  If I recall correctly, MF will append .$$$ to
the file name if you don't have write permissions on the *file* you're
editing.  If you also don't have write permission to the *directory* it
can't write the $$$ version either.

When in doubt, always check permissions!
