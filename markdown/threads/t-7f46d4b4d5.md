```
While creating a mini-server,  I'm attempting to call a windows api call for
CreateMailslot.   The error code returned in the handle is
Error_Invalid_Data.   I've tried about everything I can think of.   I have a
feeling that it's probably tied to the ws-ms-secattributes, but am not sure.
Compiles clean, runs, but returns a -1 in the ws-ms-handle and the extended
error code is a 13, error_invalid_data.

Any assistance here would be appreciated.

Code follows.

Working Storage

    01  ws-ms-name  pic x(72).
    01  ws-ms-maxmsgsize  pic S9(9) comp-5 value 0.
    01  ws-ms-readtimeout pic S9(9) comp-5 value  0.
    01  ws-ms-secattributes.
         05  ws-sec-att-Length pic S9(9) comp-5.
         05  ws-sec-descriptor pic S9(9) comp-5.
         05  ws-sec-bhandle    pic S9(9) comp-5.
    01  ws-ms-handle  pic S9(9) comp-5.

Procedure Division

   string "\\.\mailslot\monitor"
      delimited by size into ws-ms-name.

    call winapi32 "CreateMailslotA" using
        by reference ws-ms-name
        by value ws-ms-maxmsgsize
        by value ws-ms-readtimeout
        by reference ws-ms-secattributes
        giving ws-ms-handle
    end-call

    if ws-ms-handle = -1
       call winapi32 "GetLastError"
            giving ws-lasterror
       end-call
    end-if
    .
Robert Sellers
```
