        IDENTIFICATION DIVISION.
        PROGRAM-ID. PROCESS-EXERCISE-RECORDS.
      * AUTHOR. KA KHIONG KWOK.
      * DATA SOURCE. EXERCISE.NSF (LOTUS NOTES)
      * PURPOSE. EXPORT DATA FROM NOTES. PERFORM BATCH WORK ON DATA.

      * THINGS TO CONSIDER:
      *                         MULTI PAGE REPORTING.
      *                         PROVIDE OPTION FOR ENTERING FILE NAME.
      *                         FIX UP SOME OF THE SUCKY SECTION DUMMY.

      * ORDER OF PROCESSING.
      * 1) READ EXERCISE TYPE FILE TO DETERMINE WHETHER THERE
      * ARE NEW EXERCISES. DONE.
      * 2) PROCESS EXPORTED FILE1.
      * 3) PROCESS EXPORTED FILE2.

        ENVIRONMENT DIVISION.
        INPUT-OUTPUT SECTION.
        FILE-CONTROL.
            SELECT ET-EXERCISE-TYPE ASSIGN TO "EXTYPES.DAT"
              ORGANIZATION IS LINE SEQUENTIAL.
            SELECT EM-EXERCISE-MAIN ASSIGN TO "MAIN.DAT"
              ORGANIZATION IS INDEXED
              ACCESS IS SEQUENTIAL
              RECORD KEY IS EM-EXERCISE-KEY.

        DATA DIVISION.
        FILE SECTION.
        FD  ET-EXERCISE-TYPE.
        01  ET-EXERCISE-TYPE-RECORD.
            02      ET-EXERCISE-KEY         PIC 9(04).
            02      FILLER                  PIC X(02).
            02      ET-EXERCISE-NAME        PIC X(20).
            02      ET-EXERCISE-COMMENTS    PIC 9(03).

        FD  EM-EXERCISE-MAIN.
        01  EM-EXERCISE-MAIN-RECORD.
            02      EM-EXERCISE-KEY         PIC 9(04).
            02      EM-EXERCISE-NAME        PIC X(20).
            02      EM-EXERCISE-TALLY       PIC 9(03).

        WORKING-STORAGE SECTION.
        01  WS-USER-FILE-NAME               PIC X(20).
        01  WS-TEST-KEY                     PIC 9(03).
        01  WS-REDEFINED-KEY
            REDEFINES WS-TEST-KEY           PIC ZZ9.
        01  WS-TRUE                         PIC X(01) VALUE "T".
        01  WS-FALSE                        PIC X(01) VALUE "F".
        01  WS-END-OF-FILE-FLAG             PIC X(01) VALUE "F".
            88  WS-END-OF-FILE              VALUE "T".
        01  WS-WRITE-MAIN-RECORD            PIC X(60).
        01  WS-COUNT-DELETED-RECORDS        PIC 9(03).
        01  WS-COUNT-NUMBER-OF-RECORDS      PIC 9(03).


        PROCEDURE DIVISION.
        100-MAIN.
            MOVE ZEROES TO WS-COUNT-DELETED-RECORDS
            MOVE ZEROES TO WS-COUNT-NUMBER-OF-RECORDS
            MOVE ZEROES TO WS-TEST-KEY
            PERFORM 200-PROCESS-MAIN-FILE
            STOP RUN
            .


        200-PROCESS-MAIN-FILE.
            OPEN INPUT ET-EXERCISE-TYPE
            OPEN OUTPUT EM-EXERCISE-MAIN
            PERFORM 300-GET-EXERCISE-RECORD
              PERFORM UNTIL WS-END-OF-FILE
                IF ET-EXERCISE-TYPE-RECORD = SPACES
                  DISPLAY "SPACES - RECORD IGNORED"
                ELSE
                  ADD 1 TO WS-COUNT-NUMBER-OF-RECORDS
                  DISPLAY WS-COUNT-NUMBER-OF-RECORDS
                  PERFORM 300-WRITE-MAIN-RECORD
                END-IF
                PERFORM 300-GET-EXERCISE-RECORD
              END-PERFORM
            CLOSE ET-EXERCISE-TYPE
            CLOSE EM-EXERCISE-MAIN
            .


        300-GET-EXERCISE-RECORD.
            READ ET-EXERCISE-TYPE
              AT END MOVE WS-TRUE TO WS-END-OF-FILE-FLAG
            END-READ
            DISPLAY "=== Start ==="
            DISPLAY ET-EXERCISE-TYPE-RECORD
            .


        300-WRITE-MAIN-RECORD.
            MOVE ET-EXERCISE-KEY TO EM-EXERCISE-KEY
            DISPLAY "MAIN KEY: " EM-EXERCISE-KEY
            MOVE ET-EXERCISE-NAME TO EM-EXERCISE-NAME
            DISPLAY "MAIN NAME: " EM-EXERCISE-NAME
            MOVE ZEROES TO EM-EXERCISE-TALLY
            DISPLAY EM-EXERCISE-MAIN-RECORD
            WRITE EM-EXERCISE-MAIN-RECORD
              INVALID KEY PERFORM 400-DISPLAY-BAD-KEYS
            END-WRITE
            .


        400-DISPLAY-BAD-KEYS.
            DISPLAY "=== bad keys ==="
            DISPLAY "TYPE KEY: " ET-EXERCISE-KEY
            DISPLAY "OTHER KEY: " EM-EXERCISE-KEY
            .
