CHAR-VALUE = FUNCTION ORD(DATA-CHAR)

i.e.,
FUNCTION ORD("A") = 65

---
the reverse function is CHAR

DATA-CHAR = FUNCTION CHAR(CHAR-VALUE)

i.e.
FUNCTION CHAR(65) = "A"

```
FIRESTARTER <member31992@dbforums.com> wrote:

>
>How do you do XOR in COBOL. I am trying to do CRC in COBOL for a project
>I am working on which needs XOR, and on Mainframe it kinda tough as
>there is no XOR function. If you can share the info with me it will be
>really great.

* Compute c = a xor b
  01  unqualified-variables. 
       05  a.
             10  a-n    comp  pic 9(2).
       05  b. 
             10  b-n    comp  pic 9(2).
       05  c.
             10  c-n    comp  pic 9(2)..
       05  a-bit         comp  pic 9(2).
       05  b-bit         comp  pic 9(2)

     move zero to c-n
     perform 8 times
         divide 2 into a-n giving a-n remainder a-bit
         divide 2 into b-n giving b-n remainder b-bit
         if  b-bit = 1
             compute a-bit = 0 - a-bit
         end-if
         compute c-n = (c-n * 2) + a-bit
     end-perform.

Think of b as a mask or sieve. If the b bit is off, the a bit passes through
unchanged. If the b bit is on, the a bit is flipped.
```
