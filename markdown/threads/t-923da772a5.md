The easiest way to this is to write a cobol program that reads the file data
and writes to a new file that contains display usage data instead of packed...

i.e.

01 the-record-packed-stuff.
05 fun-field-one pic x(05).
05 fun-packed-field pic 9(04) usage comp-3.
05 fun-ima-integer pic s9(04) usage comp.

01 unpacked-record-stuff.
05 fun-field-one-unpacked pic x(05).
05 fun-packed-field-unpacked pic 9(04) usage display.
05 fun-ima-integer-char-arized pic 9(04).

move fun-packed-field to funner-under-unix-field-display.
move other fields appropriate
write out the record.
