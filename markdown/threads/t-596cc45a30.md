You should really write a program. Performing a conversion without knowing how
the comp fields are laid out is very dangerous. Some comp field could have signs
others not, a simple conversion program cannot determine which field is signed
and which is not. The easiest way is to write a skeleton program and then use
the COBOL copy statement (or include) to include your "01" for the file. Then
make a copy of the "01" without the COMP. You can then use move corresponding to
perform the data conversion. This is much safer and you now have a skeleton
program to perform the conversions for you.

Hope this helps...
