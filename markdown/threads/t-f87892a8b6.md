Donald,

From your post I am presuming that you are using the EXPANDTAB=OFF file
handler parameter.

A possible solution would be to set this parameter for your line sequential
file only instead of setting it globally.

Your EXTFH.CFG would look something like this;

[XFH-DEFAULT}
BASENAME=ON

[MYFILENAME.EXTENSION]
EXPANDTAB=OFF

Hope this helps.
