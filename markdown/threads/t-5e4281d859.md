1.Create a log_in id to log on to mainframes:


All you need to do is go to the Dezhi Mainframes Internet-Website, at http://zos.efglobe.com/cgi-bin/mainframe/mainuser and register yourself for a new TSO User-ID and Password.

2. Access using browser  :

Now-a-days, installing a separate 3270 Terminal-Emulator is not required. In fact, today it is possible to access Mainframes using a  Web-Browser Application like Internet Explorer or Mozilla Firefox. To get started, you can go to Open BPM Project(on Hosted Hercules) at http://mvs.open-bpm.org/web3270.fcgi

3. You can also download and install an emulator for the same purpose  :

http://www.winsite.com/Internet/Utilities/Z-Scope-Terminal-Emulator/

a.download / & install
b.configure : mainframes by giving telnet/mainframes/host_name which will be provided by 1st step.
c.click on terminal/host_name/connect on left side.
