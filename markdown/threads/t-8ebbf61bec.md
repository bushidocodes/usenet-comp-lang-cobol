```
It may be best to just cut and paste.  If you send a bunch of
stuff from something like a Fujitsu compiler or a MAC computer it can not
be read unless it is in text (ASCII) format.  Never had a problem with a
compiler.  I always did a copy inside the compiler and opened notepad and
pasted what I copied.  Most of the time the file extension is program.cbl
or something like that and that confuses notepad Or past the code right
into the message itself. 

Usually a dos compiler saves the source as text and then changes the file
extension so just try changing it to: program.txt(make a copy first).
Or try "copy a:\program.cbl a:\program.txt"
If you are using a specific compiler find out what it is and tell the
newsgroup.

Charles Hammond, BSIM Undergraduate Program
```
