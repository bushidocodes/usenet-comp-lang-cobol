Phoenix, AZ. (May 8th 1997) -- Unikix Technologies Java WebClient

located at http://www.unikix.com.
The WebClient creates full Internet GUI's for any 3270 based
applications (CICS, TSO, IMS, UniKix, etc.). It consists of two parts:-

- A Java application provides a visual editor to create the
GUI from the 3270 screen using drag and drop techniques. No
changes to the host application are required.

- A Java applet provides the runtime connection over TN3270
to the host application.
