GOT IT FIXED....
All I did was create a file called "ADISKEY.MAP" and placed it on my
directory.
The content of the file was as follows:
------------------------
Meta-Char is "/"
-----------------------
that is right......just in line of text. 
Hope this helps if anyone else runs into this problem. Found the
answer in "COBOL V4.0 for UNIX    System Reference - Voulume 2" page
O-4.
