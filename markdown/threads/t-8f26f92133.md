› How I can sort a table of name ( alphabetic ) ?

If your compiler is COBOL 85 or later, it should support the extensions to
the SORT verb which sorts tables as well as files. If not, below is a
template for a Shell Sort in COBOL.

The data fields declared BINARY can be any numeric type you want; BINARY is
just most efficient on my compiler. Remember, this is a template, but it
is also a working example. The OCCURS of ST-ELEMENT can be changed, of
course, but be sure to set ST-ELIM to the actual number of populated table
entries to sort before you perform the sort routine. I used GO TOs within
the loops, because this is the type of routine where the need for
efficiency can justify it.
