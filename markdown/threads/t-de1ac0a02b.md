I have, over the past several years, produced two FREE tools for COBOL
programmers. While these are available on Compuserve, I would like to make
them more broadly available by posting them "somewhere" on the Internet.

Can someone suggest a suitable for site or sites for me to contact?

Here are the summaries of the files as they appear on Compuserve:

COBDATA.ZIP
Text and graphics introduction to COBOL numeric data types. Designed for
intermediate to advanced programmers not familiar with the COBOL language or
COBOL numeric data storage. Windows(r) Write or compatible word processor
recommended (but ASCII text file included). Includes executable program file
as well as PB 3.2 source code. Copyright but free. From Michael Mattias.
A compiler is NOT required to use this product.

COBFD.ZIP
Tal Systems' COBFD COBOL FD Analyzer v 1.40 (Nov '98). Freeware for
PC/MS-DOS;
parses COBOL source or copylibs and reports the size and offset within each
'01' level of
all datanames (group and elementary). Supports any level OCCURS/REDEFINES
and most
specialty compiler directives. EXE, documentation and demo file included.
Version 1.40 adds
command-line options for use in batch files. Conversions, sort keys, new
record definitions:
Why do it by hand? From Michael Mattias.


Thanks in advance,
Michael Mattias dba Tal Systems
Racine WI USA

PS: I'm still kind of new to the newsgroup software, so if you want to
really make sure I get your message, please send e-mail.
