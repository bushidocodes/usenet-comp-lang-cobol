```

Language Environment for MVS and VM gives you the ability to write
"Condition Handlers" that will, for example, intercept a data exception
(OC7) and allow the program to continue. The basic version of a Condition
Handler simply skips the instruction in error, and is very easy to code.
I've taken this slightly further, displaying some information about the
instruction such as the offset and the instruction itself. Again this is not
real hard to do.

What I want to know is - has anyone taken this further ? For example I
would like to display the COBOL line number where the instructions is, and
the BL cell # and displacements of the sending and receiving fields.
Ideally, I would like to correct the data, in the same way that CA-Optimizer
does, using the DETECT option.

I guess what I really need to know is the Control block structure of a COBOL
program. Is this available anywhere ?
```
