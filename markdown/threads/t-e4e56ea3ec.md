Sorry but there isn't one. If you have the CD then that gives you
supporting on-line help. Now if you want to refer to specific features,
say Language Reference Manuals, go to :-

http://supportline.microfocus.com/documentation/books/nx31sp1/nx31indx.htm

The books I'm referring you to are for Net Express, which is an
updated/revamped version of VISOC. But so long as you keep to the
features in the VISOC CD help you will find the N/E on-line books
useful. (An example of what you can't use - N/E has support for invoking
Java - VISOC never had this feature).

Jimmy, Calgary AB
