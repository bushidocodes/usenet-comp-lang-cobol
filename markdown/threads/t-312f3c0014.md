Cobedit runs on MS-Windows 95 and up. It is designed to edit COBOL
programs and also to launch a number of tools using either menu
choices or tool bar buttons. It has a number of helpful features:

- highlighting of reserved words (but not real syntax coloring)
- a tree structured view of some parts of the COBOL program
- several ways of indenting code
- You can add your own choice of compilers, other tools or runtimes.
(Read the online help *first*.) You need to specify the proper command
line plus arguments to invoke those programs.
- option of saving the source file before editing (*nice*)
- option of capturing (standard) output from the compile step and
displaying it or displaying the contents of a selected error or log
file in a secondary window

It is set up to work with compilers that produce code which is
interpreted by a runtime module. It may work with compile, link and
execute style programs as well, but this probably requires adding the
link step and the execute step to the tool menu.

It may handle groups of files as a main file plus COPY files, but I
have not tried this feature. While I usually edit, compile, link and
run from the command line, this program is tempting. If you can figure
out the correct command lines to use, this might be an attractive
alternative to the "Programming Staff" supplied with the free version
of Fujitsu Cobol (Ver 3.0) which may be of interest to students.
