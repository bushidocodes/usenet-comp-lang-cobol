Following are the functions used with the registry:

RegCloseKey
RegConnectRegistry
RegCreateKey
RegCreateKeyEx
RegDeleteKey
RegDeleteValue
RegEnumKey
RegEnumKeyEx
RegEnumValue
RegFlushKey
RegGetKeySecurity
RegLoadKey
RegNotifyChangeKeyValue
RegOpenKey
RegOpenKeyEx
RegQueryInfoKey
RegQueryValue
RegQueryValueEx
RegReplaceKey
RegRestoreKey
RegSaveKey
RegSetKeySecurity
RegSetValue
RegSetValueEx
RegUnLoadKey

If you can call 'em from COBOL, you can use 'em.

If you can translate from BASIC, Alan Earnshaw wrote a real nice article on
using the Registry in the September, 1999 issue of "BASICally Speaking"
magazine. Reprints may be ordered for $4.00 at http://www.infoms.com .

Peter,

Here's some other Windows APIs that might be of help. These are in CA-
Realia WB 3.0 format.

 01  SystemInfo.
     03  dwOemId                         PIC 9(09)  BINARY.
     03  dwPageSize                      PIC 9(09)  BINARY.
     03  LpMinimumApplicationAddress     POINTER.
     03  LpMaximumApplicationAddress     POINTER.
     03  dwActiveProcessorMask           PIC 9(09)  BINARY.
     03  dwNumberOfProcessors            PIC 9(09)  BINARY.
     03  dwProcessorType                 PIC 9(09)  BINARY.
     03  dwAllocationGranularity         PIC 9(09)  BINARY.
     03  dwReserved                      PIC 9(09)  BINARY.
 01  OSVERSIONINFO.
     03  dwOSVersionInfoSize             PIC 9(09)  BINARY
                                                    value 148.
     03  dwMajorVersion                  PIC 9(09)  BINARY.
     03  dwMinorVersion                  PIC 9(09)  BINARY.
     03  dwBuildNumber                   PIC 9(09)  BINARY.
     03  dwPlatformId                    PIC 9(09)  BINARY.
     03  szCSDVersion                    PIC X(128).
 01  DISKFREESPACEINFO.
     03  lpSectorsPerCluster             PIC 9(09)  BINARY.
     03  lpBytesPerSector                PIC 9(09)  BINARY.
     03  lpFreeClusters                  PIC 9(09)  BINARY.
     03  lpClusters                      PIC 9(09)  BINARY.
     03  lpszRootPathName                PIC X(121).
 01  DriveType                           PIC 9(09)  BINARY.
         88  DriveType-Unknown                      value 0.
         88  DriveType-DoesNotExist                 value 1.
         88  DriveType-Floppy                       value 2.
         88  DriveType-Hard                         value 3.
         88  DriveType-Network                      value 4.
         88  DriveType-CDRom                        value 5.
         88  DriveType-RamDisk                      value 6.
 01  LogicalDrivesMask                   PIC 9(09)  BINARY.
 01  Result                              PIC 9(09)  BINARY.
 01  MEMORYSTATUS.
     03  memorystatus-length             PIC S9(09) BINARY
                                                    VALUE 32.
     03  memoryload                      PIC S9(09) BINARY.
     03  totalphysmemory                 PIC S9(09) BINARY.
     03  availphysmemory                 PIC S9(09) BINARY.
     03  totalpagefile                   PIC S9(09) BINARY.
     03  availpagefile                   PIC S9(09) BINARY.
     03  totalvirtual                    PIC S9(09) BINARY.
     03  availvirtual                    PIC S9(09) BINARY.

 PROCEDURE DIVISION.

 8400-GSI.
     CALL 'S_GetSystemInfo'
         USING BY REFERENCE SystemInfo.
 8400-GSI-EXIT.
     EXIT.
 8405-GVE.
     CALL 'S_GetVersionExA'
         USING BY REFERENCE OSVERSIONINFO
         GIVING Result.
 8405-GVE-EXIT.
     EXIT.
 8410-GDFS.
     CALL 'S_GetDiskFreeSpaceA'
         USING BY REFERENCE lpszRootPathName
                            lpSectorsPerCluster
                            lpBytesPerSector
                            lpFreeClusters
                            lpClusters
         GIVING Result.
 8410-GDFS-EXIT.
     EXIT.
 8415-GDT.
     CALL 'S_GetDriveTypeA'
         USING BY REFERENCE lpszRootPathName
         GIVING DriveType.
 8415-GDT-EXIT.
     EXIT.
 8420-GLD.
     CALL 'S_GetLogicalDrives'
         GIVING LogicalDrivesMask.
 8420-GLD-EXIT.
     EXIT.
 8425-GMS.
     CALL 'S_GlobalMemoryStatus' USING
         BY REFERENCE MEMORYSTATUS.
 8425-GMS-EXIT.
     EXIT.
