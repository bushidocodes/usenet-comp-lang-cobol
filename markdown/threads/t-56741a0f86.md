The Windows version of RealDebug is available in both CA-Realia II WB
and CA-Visual Realia. To be able to debug a program under windows you
need a "CVX" file for this program. The CVX file contains analysis
information about your program and allows you to debug and view the
structure of the program at the same time.
When you are not using windows to compile, but the native CA-realia
compiler under DOS, the /V option for the compiler will create a CVX
file which is readable under windows.
Since you customer has already purchased CA-Realia II WB he can use
this product to debug under windows.
The usage of the debugger is explained in the User Guide chapter 8.
