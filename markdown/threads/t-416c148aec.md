If I'm not mistaken COBOL has always supported exponentiation, so try
this:

COMPUTE SQ-ROOT = SOME-NUMBER ** (1 / 2)

Similarly cubic root would be:

COMPUTE CB-ROOT = SOME=NUMBER ** (1 / 3)

Personally I've never had to use exponentiation in COBOL, but I see no
reason why it wouldn't work.
