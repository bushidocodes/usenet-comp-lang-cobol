The Micro Focus compiler allows the use of integrated preprocessors (using the
PREPROCESS directive). When using such a preprocesor, the animator can animate
using the original source file. Is it possible to get a version of the CCP
precompiler which is able to be used as an MF integrated preprocessor ? That
would give you the functionality you require.
