I have printed various barcodes using Cobol.  Some I drew using raw
matrix printer graphics, some are output as a series of rectangles in
PCL (HP Laser) codes, others use specific printer's barcode interface.

OTOH I have also used various form printing systems creating the merge
data in Cobol and having the form define how the barcode is made.  For
example see PowerForm from Fujitsu and FormPrint from Flexus.

As you haven't specified which Cobol and which OS it is difficult to
give specific help.
