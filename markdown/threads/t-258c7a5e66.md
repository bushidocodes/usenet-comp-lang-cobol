Steve,

PowerCOBOL uses the ANSI COBOL-85 standard NESTED PROGRAM model.

If you want to access a data item from multiple events in the same sheet,
then you want to declare that data item in the main DATA DIVISION as GLOBAL.

If you want to access a data item in multiple sheets, then you must declare
that data item as EXTERNAL. EXTERNAL data items can be declared in either
the main DATA DIVISION or in the specific event.

If you want to access a data item from both multiple sheets and multiple
events, then you must declare it in the main DATA DIVISION as GLOBAL
EXTERNAL.

For a detail demonstration of EXTERNAL, see the DIALOG sample at
http://www.adtools.com/support/samples.htm
