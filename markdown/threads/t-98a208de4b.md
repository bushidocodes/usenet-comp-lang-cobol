You could try something like

EXEC CICS RETURN
TRANSID('CESF')
IMMEDIATE
INPUTMSG('LOGOFF')
INPUTMSGLEN(6)
END-EXEC

I've never tried this with CESF, but it works in two other circumstances
1) To get a CESN logon screen displayed
2) To transfer control to another program in a different MRO region.

I think the IMMEDIATE option came in with CICS 3.3.

Yours faithfully

Colin Lucas
