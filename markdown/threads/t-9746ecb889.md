The "label" is nothing but paragraph name (column 8, followed by
a period). And you can just use the word "GO" as "TO" is optional
(one of the dozens of "noisy" words in COBOL).

e.g.:

100-PROCESS-CUSTOMER SECTION.
100-START.
MOVE....
PERFORM....
IF .....
GO 999-ABORT-PROGRAM
END-IF.
....
....
100-EXIT.
EXIT.
......

999-ABORT-PROGRAM SECTION.
999-START.
...
....

The GO TO can be used to branch to any paragraph name, but it is normally used
to branch to an error processing section in case of a fatal error or
to exit a section/perform through. And it should never be used to branch
"upwards", although that is not a restriction of the language but a good
programming practise.
