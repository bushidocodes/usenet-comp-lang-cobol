Hi Ann,

a) runcobol rmmapinx
b) enter the name of the file. This will show the file structure,
record length, and all the keys. This is all the info you need. Write
them down
c) write a programm and "select/fd" a dummy file with the above info
and another text file which will be your output. Read the dummy file,
move record to text file and write.

PS. If you don't have the original FD, there is no way for you to know
what fields exist in the record.

Hope it helps,

Hi Ann,

If you don't have the original FD, there is no way to know what fields
exist in the record. You CAN however extract the records from ANY
indexed file to an ascii with the following procedure:

a) runcobol rmmapinx
b) enter filename
c) info about te file's structure, record size and keys displayed.
WRITE them down.
d) write a program that "select/fd" a "dummy indexed" file with the
above info and a text file which will be your output. Read dummy file
move record to text and write text.

Hope it helps,
Theophilos.
