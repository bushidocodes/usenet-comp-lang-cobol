Not your SYSLMOD, your SYSLIB on the Link step is data set not found; you need
to find out the dsnames of all the relevant files; for example, you use
IGY.V1R2M0.SIGYCOMP for the library holding the compiler and
CEE.V1R5M0.SCEELKED for the LE link time library; if your shop has installed a
later release than V1R5 (and that is pretty old), then the dataset name has
probably changed - a real reason not to have the version and release number in
your data set names!
