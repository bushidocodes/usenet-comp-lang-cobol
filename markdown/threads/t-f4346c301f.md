Absolutely!   Ada is used in commercial systems for software where a high
level of reliability is important.    There are free compilers on the web sites
I indicated, a book titled "Ada Distilled" that can get you started (although
it is simply a starter book not a comprehensive treatise on the language),
libraries for windows programming, Linux programming, and lots of
interesting projects.

Ada even has a decimal type that can be used where we use the
PICTURE 999V99 syntax in COBOL and a complete set of
picture editing clauses that closely follow those found in COBOL.

The most recent version of Ada is Ada 95.  It is currently going
through a revision process for Ada 200X.

Ada is no longer an exclusively military language.   It has benefited
from being more widely adopted in non-military software.
