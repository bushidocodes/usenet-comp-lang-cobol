```
I thought others in this NG might be interested in some information that I
provided to another COBOL group.

    ***

With NetExpress V4, Micro Focus now provides OO support that fully (or so
they claim) supports the syntax approved last year as an ISO COBOL Standard.

To see the differences between these two "syntaxes" - go to:


http://supportline.microfocus.com/supportline/documentation/books/nx40/oppubb.htm

and (find) read the chapter:

    "Chapter 12: Micro Focus OO COBOL Alternative Syntax"

If you aren't familiar with OO "concepts" in general - as used in COBOL, you
might also want to read the chapter,

     "Chapter 2: OO Programming Concepts"

in the same (online) documentation.
```
