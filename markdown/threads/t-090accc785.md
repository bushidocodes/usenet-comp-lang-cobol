I am just playing with a similar thing (for future use) using the Novell
Groupwise API on a PC DOS/Windows platform on a Netware using M$-Cobol
v4.5.

Sending is easy, just create 2 text files in the API directory, one is a
control file, the other contains the body of the message.

Receiving was a bit trickier, I ended up writing a small assembler
sub-routine to check for the presence of a control file (of random name)
in the interface directory and passing it's name back so that it can be
opened.

The idea is then to process the content of an email (file name in
control file) generated by FormMail on a web page and entering the data
into system automatically. Certain of web pages will also be refreshed
by the interface, without needing a full CGI/online system.

If you need anymore info - please email.

Brian
