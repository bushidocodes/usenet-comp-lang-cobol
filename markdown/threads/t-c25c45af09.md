I'm not sure if this is the group to ask in but here goes...

We've got a suite of systems written in Cobol running on mainframes and Unix boxes. We
need to move forward and get them looking modern! My current line of thought is a 2 stage
approach: 1) Pretty up the front end, 2) Gradual redevelopment using a RAD
tool/methodology.

The questions is which are the best tools for the jobs?

Front ends: One solution seems to be screen scraping to give PC terminals a GUI look and
feel. The problem is that we supply systems for different platforms and so simple screen
scraper stuff would mean different solutions for each of the platforms we support.

Redevelopment: Has anyone got any suggestions/recommendations for a RAD tool which
works with Cobol or which will allow us to do piecemeal redevelopment of existing systems
as a gradual move to another language?

Any comments gratefully accepted.

based systems, I would highly recommend a product called FlexGen. The

object code on any target platform. You can migrate to their environment
