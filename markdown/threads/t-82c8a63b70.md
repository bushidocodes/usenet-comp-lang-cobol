```
ab··.@ago··p.com (Sphinx) writes:

› We use a WANG VS 300 system at work. I would like to know how to send
› a message to a user through a program instead of through the OS.
› The broadcast a message to a single user option.

Very good question. You already know that HELP-PF11-PF14-PF6 lets
you send messages to users from the operating system. There
is a way to send a message to the operator from within a Procedure
Language program. But the only way I think of to send a message
to a user from a program would be to use the Inter-Task Messaging
Facility (ITMF) calls... VSSUB: MESSAGE. That will require
writing this into your application. If you use Wang Office you
can send messages thru its API so that you get the "M" alert
on the screen. But as far as (from an application) forcing
a user into HELP and displaying a message... can't help.
```
