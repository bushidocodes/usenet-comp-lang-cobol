Stephan,

What COBOL are you using? If it's Acucobol-GT you can empty a list box 
by using the MODIFY verb to set the RESET-LIST property to 1, as in:

MODIFY MY-LIST RESET-LIST = 1.

To remove a single item you use the ITEM-TO-DELETE property, just set it 
to the number of the item you want to remove (1 is the first, 2 the 
second and so on):

MODIFY MY-LIST ITEM-TO-DELETE = WS-ITEM

or

MODIFY MY-LIST ITEM-TO-DELETE = 3

You can INQUIRE on the SELECTION-INDEX property if you need to know 
which item number is currently selected, or the VALUE property if you 
want to know what text is in the selected item:

INQUIRE MY-LIST SELECTION-INDEX IN WS-SELECTION,
                        VALUE IN WS-ITEM-TEXT.

Of course, if you're *not* using Acucobol-GT, this is all useless to 
you...  ;^)

Best regards

Nigel
