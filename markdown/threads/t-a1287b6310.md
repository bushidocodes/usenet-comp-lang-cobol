```
> Can you be clearer please?I know the String command but the unstring, how

> 
> I need to replace  the string xxxxxxxxxx01 with 09/09/1976 in a field for

You may be better with reference notation:

     MOVE ZERO   TO I1  
     INSPECT string1 TALLYING I1 FOR CHARACTERS BEFORE INITIAL "xxxxxxxxxx01"
     MOVE string1(1:I1)   TO string2
     MOVE "09/09/1976"    TO string2(I1+1:)
     MOVE string1(I1+11:) TO string2(I1+11:)
of course unstring and string could also be done.

     MOVE ZER        TO I1 I2
     UNSTRING string1
         DELIMITED BY "xxxxxxxxxx01"
         INTO work1 COUNT IN I1
              work2
     MOVE SPACES     TO string2
     STRING Work1(1:I1)
            "09/09/1976"
            Work2
         DELIMITED BY SIZE
         INTO string2
```
