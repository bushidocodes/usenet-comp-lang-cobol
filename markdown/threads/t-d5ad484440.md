My experience has been yes.  There are a couple ways you can do it.

#1

       select print-file  assign to print "lpt1".


#2  This is what I'm doing with RM/COBOL


        select print-file assign to print "printer1"


     Note:  printer1 referes to a line in the RM/COBOL configurtion
            file that actually points to lpt1.
