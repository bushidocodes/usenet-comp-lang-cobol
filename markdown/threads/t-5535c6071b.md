If you have dBASE (most any version from dBASE-III on), or FoxPro (pretty much any version) you
should be able to create a SDF file with the copy command as in:

USE <file name>
COPY TO <output file> TYPE SDF

The output will be straight text, each field set to the width as defined in the dbf file.  Decimal
points are explicit for numeric fields.  If you can't get your hands on dBASE or FoxPro then use the
Clipper DBU utility.
