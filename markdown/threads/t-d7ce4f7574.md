newly there is a DD-Statement to change the COBOL options fur run-time

The following list shows you the major changes that have been made in 
z/OS V1R7.

 	Language Environment supports the ability to provide additional 
run-time options at invocation through a DD statement named CEEOPTS.

You can not only add run-time options, you can, if allowed, change 
options

before this version you can only submit options via the /-parm exec 
statement.
