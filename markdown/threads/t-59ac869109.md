In reference to Fujitsu COBOL 6.1, I'd say it's quite powerful, especially
if you decide to purchase the professional version which includes
PowerCobol.

PowerCobol allows you to "drag and drop" database controls (such as the one
we use for MS Access) on to your form, add some lines of code, and update
the database without ever opening MS Access per se.  Once you get the hang
of how the database field names are set up, it's really quite slick.

Also, we have applications that map network drives to our Unisys host
(mainframe), and pull data from the host without using any file transfer or
FTP software.

All in all, in my opinion, it's quite a powerful product and allows those of
us who have been working with COBOL a number of years to stay in familiar
territory.  Qualified by the fact that there is a bunch of new stuff to
learn regarding Windows Methods, Controls, Properties, etc.  And that's not
all bad!!!

Denny
