When you use SCREEN SECTION or do a DISPLAY or ACCEPT AT or UPON
CRT then the Cobol run time requires ADIS modules to do this.
You can add these to your link, such as:

LINK program+adis+adisint+adiskey, ...

Or you can link ADIS separately and has ADIS.EXE in the current
directory. The same applies to EXTFH if you use ISAM files.
