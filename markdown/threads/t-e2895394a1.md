No offense intended to you, Mr Gunshannon, or to anyone else - even 
myself, not that such a thing might ever happen, mind you - but this is 
something I have noticed and has caused me to muse about self-fulfilling 
criteria.

A person has a physical object, call it X... and there comes a time when X 
no longer serves an immediate need.  Relying on experience ('that which is 
not needed now has been later') this object gets set aside.

Time passes... sometimes *much* time passes.  For whatever reason this 
same person then stumbles across the X, which has not served a use for a 
goodly while.

What is it that goes on when the person says 'Well, I haven't used this in 
a goodly long while... and there's even less of a possibility that I'll 
have need of it in the future... but since I've saved it so very long it 
must be Very Important, I can't throw it out now!'?

(This happened to me a few years back when I ran across a still-in-the-box 
copy of some manner of 'all in one' software suite... manuals, 5.25" 
floppy disks, never used.  I thought 'this is of recent Oldene Dayse, it 
is of interest, it... it is a piece of obsolete software for an obsolete 
operating system and I ain't no museum, away with you!')
