The following is (a slightly modified version of) what I posted on teh 
OpenCOBOL forum,

     Another new (free) COBOL available
I have just become aware of another (new I belive) FREE COBOL product. If 
you are interested in seeing it (and comparing it, check out:

https://www.elasticcobol.com/

The documentation on getting started is NOT very good. (I am still working 
my way thru it).

It seems to have two free options.

1) There is a 14-day trial which allows you to test the more extensive 
version. I think you just select one of the full products and "cancel" 
after/before 14 days. (This includes both SQL and CICS support)

2) The version that I am playing with runs on the Cloud (the IDE and the 
compiler and the executables ALL run on the cloud, not on your machien. The 
response time on this is medium slow, but it does seem to be VERY full 
featured (but not CICS or SQL - but does include HTML and some remote 
features)

Checking the LRM, this compiler accepts a WIDE range of extensions (most 
Micro Focus and IBM mainframe extensions, and a number that I have never 
seen before.  I think it has signficant COBOL/400 and HP COBOL language 
features)

So far, their support has been pretty good in helping me with "getting 
started".

This is NOT an OPEN SOURCE product. If you are doing "real work" with their 
product and go with the non-free version, the pricing scheme is interesting 
(to me). check it out at:

  https://www.elasticcobol.com/index.php/pricing/



Special note to those uinterested in OO COBOL:

  As far as I can tell the CLOUD (free) version has full OO COBOL support 
(both ISO 2002 and MF compatible) as well as "close" integration with Java 
(but not C#)
