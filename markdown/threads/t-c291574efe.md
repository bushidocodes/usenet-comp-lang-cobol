Well Life can get better. If you have Exit Perform in your compiler,
then it makes life easier. My preference is :-

(a) 

Set DontEndPerform to true
PERFORM VARYING SUB FROM 1 BY 1
      UNTIL SUB >= MAX-ENTRIES or EndPerform

    if	RecordFits
        MOVE WS-HOLD-REC (SUB) TO WS-REC-DEFINITION
        more stuff

     else set EndPerform to true
 END-PERFORM

(b) 

PERFORM VARYING SUB FROM 1 BY 1
      UNTIL SUB >= MAX-ENTRIES

    if	RecordFits
        MOVE WS-HOLD-REC (SUB) TO WS-REC-DEFINITION
        more stuff

     else EXIT PERFORM
 END-PERFORM

Whether you test using = or >=  depends upon whether you are using Test
After or Test Before.
