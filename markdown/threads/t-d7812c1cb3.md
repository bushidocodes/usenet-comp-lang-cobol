(from the Fujitsu manual) Environment variable information

may be specified with the compiler option MAIN (MAIN). In this
case, when an application is started using a method other than
input to the command prompt window, such as by clicking an
icon, a system console is generated soon. In addition, a COBOL
console is also generated for input-output use.

A lot is going to depend on which compiler you have. Fujitsu is giving
away version 3 and selling version 10? also there are .Net compiler
versions that match each on the Microsoft .NET versions. So with that
in mind.

This program

000010 PROGRAM-ID. HELLO.
000010 PROCEDURE DIVISION.
000010     DISPLAY "HELLO BACK".
000010     STOP RUN.

When compiled with this batch file.

CALL "COBOLC.EXE" %1.CBL /WC:SRF(FIX,FIX) /TARGET:EXE /MAIN:%1 /PRINT:
%1.ERR


Looks like this:
C:>hello
HELLO BACK
