I work a lot with Real World accounting software.  The RM85 version
comes with file utilities.

Just go to your main Realword directory and type:

   run85 FIX??

where ?? is the module that contains the file you want to fix.  For
example, Customer file is in AR so you would type run85 FIXAR

Then you will be presented with a menu.  Select "Recover a file in
place" and then you will be presented with a list of files to choose
from.  Choose your file then this will run the RM utility to rebuild
the file.  

I would recomend making a backup of the file before doing this just in
case something happens.
