Sure Steve - though I admit you would be a lot better off with COBOL. :) 
For an AS/400 compatible RPGII style compiler, you will probably need to
use 'Baby/400 From California Software Products.' This is an AS/400
emulator that comes with CL and RPG compilers, but... I would not
recommend buying from the company.  Cost is something like $4K I think.

For an AS/400 RGG IV compatible compiler, you can pick up VisualAge RPG,
which lets you do a lot of things, including client/server front ends on a
PC running Windows95 or Windows NT. It comes from IBM and much more
information  is available on their web site about it. Cost is around $2700
I think. 

Hope that helps. 

-Paul
