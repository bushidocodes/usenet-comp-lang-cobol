In general, if we cannot parse a particular Cobol dialect,
it needs to be fixed in the parser. We could try to pass
the source code to the beautified output 
without really understanding (parsing) it,
but this is more dangerous than replacing it with 
error message, because if smth is lost from the code,
you will find it at run time at a great cost.

So we decided to make such things explicit at compile time
where they are easy to catch.

Vadim Maslov
