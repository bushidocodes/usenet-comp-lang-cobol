David,

I don't want to dampen your enthusiasm - but softly, softly, catchee
monkee. Personal COBOL is a learning tool, and you are an amateur
learning. Use the package to get a firm handle on the basics, screen
displays (I don't know whether it provides you any OO for Windows) and
flat file-handling, indexed, relative, sequential, line sequential :-

	- Display on screen as appropriate
	- Add new record			- WRITE new
	- Change existing record		- READ & REWRITE
	- Delete existing record		- READ & DELETE

And of course, produce reports from the data files.

You could then move on to NetExpress University Edition (about $US 100 )
which is same as professional NetExpress, giving you considerably more
features than 'Personal', but can't be used to sell applications.

If 'Personal' contains PC_PRINT routines, look at syntax for printing a
Bitmap, (your business logo).

Charts and Graphs - you could do Mickey Mouse ones in COBOL - BUT - this
is one area where you DEFINITELY don't reinvent the wheel. There are
chart packages, produced by those with mathematical expertise, to cover
this. (They are invariably written with C or C++ type languages in
mind).

Jimmy, Calgary AB
