Have you tried setting the noalphal option?  If not, place the following so
that the ampersand is in column 7.  Cobol is not case sensitive.
Unfortunately, C++ libraries *are* case sensitive at the link level, so
lower case link names will not be resolved.  The above option will fix that
particular problem.

       @OPTIONS NOALPHAL
