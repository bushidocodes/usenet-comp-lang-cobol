SCO Unixware and Open Server have an SMB server VisionFS that will allow
the files on the Unix system to be accessed as a sharable drive from
Windows.  Or install Samba to do this for free (get the SCO binaries
from www.sco.com/skunkware).  Then you can run a Cobol program in the
Windows environment to access the Unix file.  The Cobol program should
use the same brand of compiler and run-time that is used on Unix to
ensure compatability.
