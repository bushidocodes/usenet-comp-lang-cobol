I received an error 114 when running MF Cobol under rtsora (Oracle
Pro*Cobol runtime support) after upgrading to Solaris 2.5. In case
anyone else encounters this, it is a Solaris 2.5 problem that is fixed
by patch 103238-01. The error does not occur in Solaris 2.5.1.
