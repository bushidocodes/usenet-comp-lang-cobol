```
Using Micro Focus 4.1. Would like to create callable programs or ENTRY
points that take char parameters that are of variable length that can
be coded like the DISPLAY statement using literals. For example, I can
  DISPLAY "This is a test"
or
  DISPLAY "123"

How can I code parameters so that I can
  CALL "MY_FUNC" USING "This is a test"
or
  CALL "MY_FUNC" USING "123"

The problem is defining the LINKAGE parameters. Micro Focus has a
library call that returns the number of parameters passed, but it
doesn't return any other information about the parameters. I know that
Micro Focus stores the length of the literal in the object code, I
believe as a 16-bit integer immediately preceeding the text itself. Is
there any way to do this or am I just out of luck?

Thanks,
```

If you are passing a literal (by CONTENT - I assume), then this means that
YOU (the program writer) knows the length at compile-time.  Therefore, there
is no reason that you can't pass the LENGTH field as another parameter.
Then use that as the "object" of an OCCURS DEPENDING ON in the receiving
program's Linkage section item.

Having said that, passing an alphanumeric literal seems a FAIRLY "odd" thing
to do.  However, not knowing the real "business needs" of your application
there MIGHT be a reason to do it.
```

Let me qualify - it's some years since I coded in non-OO COBOL, but the
problem is essentially the same whether you are CALLING, ENTRY-POINTS or
INVOKING. Below might appear as OO 'gibberish', but I think you can
probably sort it out to suit your own needs. Two examples :-

    1. Passing Data to a Dialog for display
    2. Returning Data (Entryfield) from Dialog when user presses OK or TAB

1. PASSING DATA

1A. Business Invoking class :
*>--------------------------------------------------------------
Method-id. "pass-entryfield".
*>--------------------------------------------------------------
Local-storage section.
01 ls-display-3                         pic zz9.
01 ls-display-31                        pic zzz.9.
01 ls-length                         pic x(4) comp-5.
01 ls-text.
   05 occurs 1 to TextLength
      depending on ls-length         pic x(01).
Linkage section.
01 n                                 pic x(4) comp-5.
Procedure Division using n.
 move n to Field
 Evaluate true

   when Field-percent1
        move length of ls-display-31 to ls-length
        move CLIENT-percent(1) to ls-display-31
        move ls-display-31 to ls-text

   when Field-Address1
        move length of CLIENT-Address1 to ls-length
        move           CLIENT-Address1 to ls-text
    when etc.....
 End-evaluate

 invoke os-Dialog "set-entryfield"
        using Field, ls-length, ls-text

End Method "pass-entryfield".
*>--------------------------------------------------------------

1B. Dialog Invokee class

*>--------------------------------------------------------------
Method-id. "set-entryfield".
*>--------------------------------------------------------------
Local-storage section.
01 ls-object                         object reference.
01 ls-string                         object reference.
Linkage section.
01 lnk-Field                         pic x(4) comp-5.
01 lnk-length                        pic x(4) comp-5.
01 lnk-text.
   05 occurs 1 to TextLength
      depending on lnk-length    pic x(01).

Procedure Division using lnk-Field
                         lnk-length
                         lnk-text.

  invoke CharacterArray "withLengthValue"
         using     lnk-length, lnk-text
         returning ls-string
  invoke os-ControlCollection "at"
         using lnk-Field returning ls-object
  invoke ls-object "setText" using ls-string
  invoke ls-string "finalize" returning ls-string

End Method "set-entryfield".
*>--------------------------------------------------------------

2. RETURNING DATA

2A . Dialog Class returning the data:
*>--------------------------------------------------------------
Method-id. "return-entry-object".
*>--------------------------------------------------------------
Local-storage section.
01 n                             pic x(4) comp-5.
01 ls-field                      pic x(4) comp-5.
01 ls-id                         pic x(4) comp-5.
01 ls-length                     pic x(4) comp-5.
01 ls-object                     object reference.
01 ls-string                     object reference.
01 ls-text.
   05 occurs 1 to TextLength
      depending on ls-length     pic x(01).

Procedure Division.

perform varying n from 1 by 1 until n > MaxEntryFields
    invoke os-ControlCollection "at"
           using n returning ls-object

    if  os-CurrentObject = ls-object
         invoke os-CurrentObject "getText" returning ls-string
        invoke ls-string "trimblanks" returning ls-string
        invoke os-CurrentObject "Getlength" returning ls-length
        invoke ls-string "getValueWithSize"
             using ls-Length returning ls-text
        invoke ls-string "finalize" returning ls-string
        invoke os-Caller "return-entryfield"
           using n, ls-length, ls-text
         EXIT METHOD
    End-if
End-perform

End Method "return-entry-object".
*>--------------------------------------------------------------
2B. Business Class receiving the returned data

*>--------------------------------------------------------------
Method-id. "return-Entryfield".
*>--------------------------------------------------------------
Working-storage section.
78 MNS                       value 25.     *> MethodnameSize
01 mn1.
   05 pic x(MNS) value "EF-01-client-number".
   05 pic x(MNS) value "EF-02-client-shortname".
   05 pic x(MNS) value "EF-03-percent1".
   *> this is a list of method names ("paragraphs") being used
    05 etc...

01 mn2 redefines mn1.
   05 ws-MethodName occurs MaxEntryField pic x(MNS).
Linkage section.
01 lnk-Field                    pic x(4) comp-5.
01 lnk-length                   pic x(4) comp-5.
01 lnk-text.
   05 occurs 1 to TextLength
      depending on lnk-length   pic x(01).

Procedure Division using lnk-Field
                         lnk-length
                         lnk-text.

  move lnk-Field to Field

  if     not Field-ClientNumber
         set RecordChanged to true
  End-if

  invoke self ws-MethodName(Field)
         using lnk-length lnk-text

Exit Method
End Method "return-Entryfield".
*>--------------------------------------------------------------
```
