Count the "IF"s and the "ELSE"s - the indentation of the COMPUTE verb

> DIVIDE W-TOTAL BY THIRTY-SEVEN GIVING W-DIV
> ADD 1 TO W-REMAINDER

This looks like it's missing a remainder clause.  The DIVIDE verb, in 
and of itself, does not fill a remainder variable unless you tell it 
"REMAINDER W-REMAINDER" after the "GIVING W-DIV" and before the ADD 
statement.

> I can understand most of the syntax but there are a few keywords i am stuck with.

I can understand why you'd be stuck - this certainly isn't the most 
elegant COBOL I've ever seen.  It's far from the worst, though.  :) 
Below is a conversion of your code above.  I've turned the variables 
such as THREE and THIRTY-SEVEN into numbers...  I haven't compiled this, 
but it should be close.  I've also made the division fill the remainder 
before adding 1 to it, but I've commented that line out by using a 
single quote.  I also reversed the NOT logic in the code above, by 
making the positive condition stuff on the If leg instead of the Else leg.

(Of course, anything other than "If", "Trim", "Then", "Or", "isNumeric", 
"Do", "Until", "Loop", "End", and "Mod" is a variable you'll have to 
declare somewhere.)

-=-=-=-=- Begin VB Code -=-=-=-=-

If Trim(RegNo_Digit_Div(W_Count)) = "" Then
     W_Count = W_Count + 1
     If RegNo_Digit_Div(W_Count) = "T"
       Or RegNo_Digit_Div(W_Count) = "V" Then
         W_Count2 = W_Count + 1
         If isNumeric(RegNo_Digit_Div(W_Count2)) Then
             RegNo_Digit_Div(1) = RegNo_Digit_Div(W_Count)
             RegNo_Digit_Div(2) = ""
             W_Count = 3
             Do Until W_Count = W_Count2
'               You'll need to make this subroutine and pass the two
'               counter variables, unless they're global
                 Call F000_Fill_In_Space
             Loop
          End If
'       You've got this line on both the true and false legs of the
'       If statement - it only needs to be there once
         W_Count = 0
     Else
         W_Count = 0
'       If the compute should be at a different level, add "End If"s
'       as needed here, and remove the same number of "End If"s from
'       the bottom - also, correct the indentation for the sake of
'       future maintainers.  :)
         W_Total = (Value_Digit(1) + Value_Digit(4) + (Value_Digit(2) _
           + Value_Digit(5)) * 3 + (Value_Digit(3) + Value_Digit(6)) * 7)
         W_Div = W_Total / 37
'       Here's where the remainder would be filled
'       W_Remainder = W_Total Mod 37
         W_Remainder = W_Remainder + 1
     End If
End If
