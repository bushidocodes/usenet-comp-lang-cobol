Is there a software product that allows development and testing of CICS
COBOL programs on a PC network (NT-based)? It must be possible to then take
the COBOL source and re-compile on a MVS system for productionising (i.e. no
development work should be done on the mainframe).

Alternatively, has anyone got PC COBOL programs to talk to MVS CICS COBOL
programs, perhaps using CICS/NT?

My company is considering various options for replacing our existing 3270
terminal/MVS CICS COBOL systems with PC GUI/CICS COBOL, so I would welcome
any suggestions. We currently use DB2 databases and native VSAM files.

Thanks,

Steve
