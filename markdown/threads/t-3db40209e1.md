It is running just fine on XP for me so I know it can be done.
There is a setup fix for Y2K that you need to install it. I don't
think you can install it at all without this fix. I installed from the
diskette. I had to set the compatibility setting of setup.exe on the
diskette to Win 95 or NT. Then put in the CD and use this command:

AT THE C: PROMPT: 
C:\>A:\setup -from x:\cobolwb   (where x: is your CD drive)

This is to initially install Workbench 4.0. After that the update
CD did not require any special settings.
