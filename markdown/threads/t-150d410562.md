```
03  WS-FIELD PIC X(11).
03  WS-STRING PIC X(05).
03  WS-SUB PIC S9(04) BINARY VALUE +1.
03  WS-INDEX PIC S9(04) BINARY VALUE ZERO.

PERFORM UNTIL WS-SUB > (LENGTH OF WS-FIELD - 4)
           IF  WS-FIELD (WS-SUB:LENGTH OF WS-STRING) = WS-STRING
                MOVE WS-SUB TO WS-INDEX
                COMPUTE WS-SUB = (LENGTH OF WS-FIELD - 3)
           ELSE
                     ADD +1 TO WS-SUB
                     END-IF
           END-PERFORM.

IF WS-INDEX > ZERO "Then you found the String" and WS-INDEX contains the
field-position.
```
