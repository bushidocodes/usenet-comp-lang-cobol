I work for a team that has done some porting of system Z COBOL and CICS/DB2 systems over the years with pretty good success. These systems typically port over to *IX systems using IBM's TXseries and IBM's DB2 or even Oracle, with MF Server Express or similar as a COBOL compiler.

I've not been doing this for some time and I am curious if some of the newer COBOL compilers emerging also now have compatibility with TXseries on *ix or windows, with DB2 or Oracle as supported and even mySQL or Postgres as options.

TXseries for multiplatforms seems to be a pretty dominant application server for the rehosted mainframe app. I guess there is not much that does what it does in open source. Looking to see what other COBOL and RDBMS options can be working with it, particularly the open source offerings.

Thanks for any suggestions and updates.
