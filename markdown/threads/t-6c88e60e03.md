CDD (Common Data Dictionary) is a DEC product on VMS. It serves as a
repository for schema information related to files and databases,
including RMS files. DEC COBOL has the ability to use file
description information that resides in the CDD system as an
alternative to SELECT statements and FD's. It's done with a special
kind of COPY directive.
