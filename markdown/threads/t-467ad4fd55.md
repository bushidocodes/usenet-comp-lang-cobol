You should be writing COBOL programs, not C programs in the COBOL
language.

COBOL programmers have survived for several decades without using
pointers for creating dynamic memory structures - they are far to
prone to error for sensible use in business systems.

You should look at using an array (see OCCURS) and using the index
or subscripts as 'pointers', even better use an indexed file.
