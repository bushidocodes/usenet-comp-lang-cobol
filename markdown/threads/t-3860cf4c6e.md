Animator Version 2 is the source level debugger and editor. It's not a
linker.

>
>Is there somebody who can tell me what to do?
>Is there somebody who programmed a program making *.exe files himself ?

Indeed there is but I suggest the manuals are a better option. Try the
System Reference 'Linking/Library' chapter. Depending on what version
of compiler you have, you may find a demo program with a batch file
called TTDEMO.BAT which will go through most of the stages of
compiling and linking a program for you.

You bought Animator?  By itself?  If so, that's really odd, since Animator is
simply one piece of the MF product package and, to my knowledge, is not sold
as a separate product.

In any event, I'm not surprised you can't get an .EXE out of Animator. 
That's not what it's for.  Animator is an interactive source code level
debugger.  To get an .EXE file you need a compiler (to produce object files)
and a linker (to link the object files into EXEcutables).

Opinions are mine and not necessarily those of my employer.

- Ed Stevens
