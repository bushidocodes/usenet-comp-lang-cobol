Sorry for disturbing but I would like to offer you our free tool to
develop reports: Rpv Printing System.

Rpv is an easy to learn, freeware Windows based application oriented
to those professional developers that need to produce high quality
reports (including barcodes and images if needed) using a Windows
programming language, a DOS programming language or any language.

To obtain a report, Rpv reads a simple ascii file, then Rpv transforms
it into a professional report.If you know how to create an ascii file
using your programming language, then you know how to program an Rpv
report.

The tool is available in FREE, Plus and PRO editions in english,
spanish and portuguese. Italian and Greek soon in
