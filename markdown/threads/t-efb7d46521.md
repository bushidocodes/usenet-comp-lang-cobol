Look up Appendix M (M.2.2) in the Appendices Manual (book 4).  Also
check out the KEYSTROKE config variable.

You will be adding entries to the config file like so :

keystroke exception=11 U1   

(this will return an exception value of 11 for function key f11)

Good luck.

Matt Robinson
Mentis Technologies
