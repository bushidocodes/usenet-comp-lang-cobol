Conceptually they are this:

Sequential access is reading records from first in
file towards the last, or from last to first, but
always in one direction.
Random access is supplying a key for each record
desired and retrieving just that record.
Dynamic is a combination of the two.
You can supply a key (or a portion of a key) and
start reading sequentially at that point or
you can supply a key for each read. This is by
far the most versital.

For a more precise definition, any good text on
COBOL should suffice. (If it does not suffice it
was not a good text).
