Several years ago, Applied Data Research produced a macro-processor and a
set of macros for structured Cobol, as a way to add structured programming to
Cobol 74, on IBM machines at least.
These macros allowed a number of structured programming constructs,
such as loops, case (select), and some others, some of which
were included in Cobol 85.

Is there anyone out there still using ADR Cobol/SP - metaCobol?
