```
It is likely that you would need to CALL something to fetch the file
onto your local disk and then open and read it.

For example you could use wget or snarf:

       CALL "SYSTEM" USING "snarf http://www.myURLsite.com/reports/SAMPLE.txt
SAMPLE.txt"

Or you could write a simple Python script such as:

import httplib
conn = httplib.HTTPConnection("www.myURLsite.com")
conn.request("GET", "/reports/SAMPLE.txt")
r1 = conn.getresponse()
data1 = r1.read()
conn.close()
```
