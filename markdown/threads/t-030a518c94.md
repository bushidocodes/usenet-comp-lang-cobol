I have had no problems running Fujitsu Cobol from the command line. When 
I wish to start a command line session working on a Cobol program, I run 
the following batch file (in W2K or WXP).

	S:\>type e:\lang\Cobol\PCOBOL32\pth.bat
	path e:\lang\cobol\pcobol32;%path%
	set lib=e:\lang\cobol\pcobol32;%lib%
