```
An additional reason that I joined this group is that we are looking
for some help from someone with current COBOL expertise. By current, I
mean familiar with some of the more recent features, such as
generating XML. I am somewhat hesitant to post what we need because of
the rather negative response received by the poster of the "Quick
COBOL Assignment Needed". So I hope this query will not result in the
same "don't be so lazy" type of responses.

But here's what we need, and if you can pass this along to someone who
can help, we'd appreciate it. Also, if you think there is a better
place to post this, please let us know.

We need some software written in z/OS COBOL, and our COBOL skills are
out of date. So we are looking for someone to help us. This isn't a
full-time, nor even a part-time job. It might be a one-time
assignment, but it might also turn into an on-going part-time support
job. We have dismissed Java, REXX, and assembler for a variety of
reasons, so COBOL is our first choice. All work can be done from your
home by linking into our IBM ISV test site located in Dallas. All
communication would be via phone and email. As an example, here are
some of the first items to be delivered:

1. Set up our test site with PROCs for compiling and linking COBOL,
with the most efficient options. That is, set up our testing
environment.

2. Help create an overall program flow for a product that we've
designed.

3. Create a skeleton program to perform the basic function (read a
complicated file and create XML output), but designed to later add
sections to parse parameters and create reports.

There is no specific time frame for this, although we're ready to get
started. If you are interested, please send your resume, time
availability (hours per week and when you might become available), and
pay requirements to us at admin@watsonwalker.com. If you have a sample
COBOL program of yours, we'd love to see it. Before we discuss
anything further with you, we would need you to sign a confidentiality
agreement.

Many thanks,
Cheryl Watson
Watson & Walker, Inc.
Sarasota, Florida
www.watsonwalker.com
```
