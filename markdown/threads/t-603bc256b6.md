pgm 1
--------------------------------
       IDENTIFICATION DIVISION.
       PROGRAM-ID.  GETOBJA.
      *-------------------------------------------------------------*
      * The purpose of this program is to retrieve ILE program      *
      * information that is not attainable with the command         *
      * DSPOBJD.                                                    *
      *                                                             *
      * This program makes use of the following AS/400 APIs.        *
      *                                                             *
      * QUSCRTUS - Creates user space for storing data.             *
      * QBNLPGMI - List object information to the user space.       *
      * QUSRTVUS - Retrieve entry information from the user space.  *
      *                                                             *
      *-------------------------------------------------------------*
       ENVIRONMENT DIVISION.
       CONFIGURATION SECTION.
       SOURCE-COMPUTER. IBM-AS400.
       OBJECT-COMPUTER. IBM-AS400.
       SPECIAL-NAMES. LOCAL-DATA IS LOCAL-DATA-AREA.
       DATA DIVISION.
       WORKING-STORAGE SECTION.

       01  WS-WORKING-STORAGE-START   PIC  X(038)
           VALUE 'GETOBJA  - WORKING STORAGE BEGINS HERE'.

       01  STDWS-AREA.
           COPY STDWS REPLACING ==XXXXXXXXXX== BY ==GETOBJA ==.

       COPY QUSGEN   OF U801741-QCBLLESRC.
       COPY QBNLPGMI OF U801741-QCBLLESRC.

      *-------------------------------------------------------------*
      * The following is copied from QSYSINC/QCBLLESRC member QUSEC *
      * so that the variable length field EXCEPTION-DATA can be     *
      * defined as 100 bytes for exception data.                    *
      *-------------------------------------------------------------*
       01  QUS-EC.
           05 BYTES-PROVIDED          PIC S9(00009) BINARY.
           05 BYTES-AVAILABLE         PIC S9(00009) BINARY.
           05 EXCEPTION-ID            PIC  X(00007).
           05 RESERVED                PIC  X(00001).
           05 EXCEPTION-DATA          PIC  X(100).

      *-------------------------------------------------------------*
      * VALUES USED FOR THE QUSCRTUS PROGRAM                        *
      *-------------------------------------------------------------*
       01  CRTUS-INFO.
           05  CRT-SPCNAME            PIC X(20)
                       VALUE 'RTVOBJA   QTEMP     '.
           05  CRT-EXTATTR            PIC X(10) VALUE SPACE.
           05  CRT-SPCSIZE            PIC S9(9) BINARY VALUE 1024.
           05  CRT-INITSPACE          PIC X VALUE ' '.
           05  CRT-AUTHORITY          PIC X(10) VALUE '*CHANGE   '.
           05  CRT-DESCRIPTION        PIC X(50) VALUE SPACE.
           05  CRT-USRRPL             PIC X(10) VALUE '*YES      '.

      *-------------------------------------------------------------*
      * VALUES USED FOR THE QUSRTVUS PROGRAM                        *
      *-------------------------------------------------------------*
       01  RTV-START-POS              PIC S9(9) BINARY VALUE 1.
       01  RTV-LENGTH                 PIC S9(9) BINARY VALUE 140.
       01  QBN-SECTIONA.
           05  USE-SPACEA             PIC  X(00020)
                                 VALUE 'RTVJOBA   QTEMP     '.
           05  FOR-NAMEA              PIC  X(00008)
                                 VALUE 'PGML0100'.
           05  PROLIB.
               10 PRO-NAME            PIC  X(00010)
                                 VALUE SPACES.
               10 PRO-LIB             PIC  X(00010)
                                 VALUE SPACES.
           05  PGM-ERRORA             PIC  X(256)
                                 VALUE SPACES.

      *-----------------------------------------------------*
      * WS-HOLD-FIELDS                                      *
      *-----------------------------------------------------*
       01  WS-HOLD-FIELDS.
           05  WS-HOLD-PROG-NAME      PIC  X(010)  VALUE 'GETOBJA '.
           05  WS-HOLD-PARA-NAME      PIC  X(030)  VALUE 'STARTUP'.
      *-----------------------------------------------------*
      * LOCAL DATA AREA DEFINITION                          *
      *-----------------------------------------------------*
       01  LDA-WORK-AREA.
           COPY LDA100.

      *-----------------------------------------------------*
      * PROGRAM STATS                                       *
      *-----------------------------------------------------*
       01  PROGSTATS.
           COPY PROGSTATS.

      *-----------------------------------------------------*
      *  PROGRAMS CALLED                                    *
      *-----------------------------------------------------*
       01  CP-CALLED-PROGRAMS.
           05  CP-ABEND               PIC  X(010)  VALUE 'ABEND    '.

      *-----------------------------------------------------*
      * STANDARD FILE STATUS CODES.                         *
      *-----------------------------------------------------*
       01  WS1-FILE-STATUS.
           COPY FILESTAT.
      *-----------------------------------------------------*
      *  DATABASE DESCRIPTION                               *
      *-----------------------------------------------------*
       01  WS-GETOBJPF.
           COPY DDS-GETOBJAR OF GETOBJPF.

      *-----------------------------------------------------*
      * SQL STATUS INFORMATION                              *
      *-----------------------------------------------------*
       01  SQL-STATUS-INFORMATION.
           COPY SQLSTAT.

      *-----------------------------------------------------*
      * SQL COMMUNICATIONS AREA                             *
      *-----------------------------------------------------*
           EXEC SQL
               INCLUDE SQLCA
           END-EXEC.

       01  WS-WORKING-STORAGE-END     PIC  X(038)
           VALUE 'GETOBJA  - WORKING STORAGE ENDS   HERE'.

       LINKAGE SECTION.
       01 PARM-NAME                   PIC X(10).
       01 PARM-LIB                    PIC X(10).

       PROCEDURE DIVISION USING PARM-NAME,
                                PARM-LIB.

       MAINLINE.

           MOVE PARM-NAME TO PRO-NAME OF PROLIB.
           MOVE PARM-LIB  TO PRO-LIB  OF PROLIB.

      *-------------------------------------------------------------*
      * INITIALIZE THE DATA STRUCTURE.                              *
      *-------------------------------------------------------------*
            INITIALIZE WS-GETOBJPF.

      *-------------------------------------------------------------*
      * INITIALIZE ERROR CODE STRUCTURE.                            *
      *-------------------------------------------------------------*
            MOVE 116    TO BYTES-PROVIDED.
            MOVE 0      TO BYTES-AVAILABLE.
            MOVE SPACES TO EXCEPTION-ID.
            MOVE SPACES TO RESERVED OF QUS-EC.
            MOVE SPACES TO EXCEPTION-DATA.

      *-------------------------------------------------------------*
      * CREATE THE USER SPACE USING INPUT PARMS FOR THE CALL        *
      *-------------------------------------------------------------*
           CALL 'QUSCRTUS' USING CRT-SPCNAME,
                                 CRT-EXTATTR,
                                 CRT-SPCSIZE,
                                 CRT-INITSPACE,
                                 CRT-AUTHORITY,
                                 CRT-DESCRIPTION,
                                 CRT-USRRPL,
                                 QUS-EC.

      *-------------------------------------------------------------*
      * LIST THE OBJECTS TO THE USER SPACE OBJECT.                  *
      *-------------------------------------------------------------*
           CALL 'QBNLPGMI' USING CRT-SPCNAME,
                                 FOR-NAMEA,
                                 PROLIB,
                                 PGM-ERRORA

      *-------------------------------------------------------------*
      * RETRIEVE ENTRY INFORMATION FROM THE USER SPACE.             *
      *-------------------------------------------------------------*
           CALL 'QUSRTVUS' USING CRT-SPCNAME,
                                 RTV-START-POS,
                                 RTV-LENGTH,
                                 QUS-GENERIC-HEADER-0100,
                                 QUS-EC.

           IF NUMBER-LIST-ENTRIES OF QUS-GENERIC-HEADER-0100
                                  GREATER THAN ZERO THEN
               ADD 1 TO OFFSET-LIST-DATA OF QUS-GENERIC-HEADER-0100
                                  GIVING RTV-START-POS.
               PERFORM 100-RTV-AND-INSERT THRU
                       100-RTV-AND-INSERT-EXIT
                             NUMBER-LIST-ENTRIES
                          OF QUS-GENERIC-HEADER-0100 TIMES.

           STOP RUN.

      *-------------------------------------------------------------*
      * RETREIVE AND INSERT INFROMATION INTO FILE GETOBJPF          *
      *-------------------------------------------------------------*
       100-RTV-AND-INSERT.

           MOVE  '100'              TO  WS-HOLD-PARA-NAME.

           CALL 'QUSRTVUS' USING CRT-SPCNAME,
                                 RTV-START-POS,
                                 SIZE-EACH-ENTRY OF
                                  QUS-GENERIC-HEADER-0100,
                                 QBN-LPGMI-PGML0100,
                                 QUS-EC.

           MOVE PROGRAM-NAME                   OF QBN-LPGMI-PGML0100
                             TO PGMNAME        OF GETOBJAR.
           MOVE PROGRAM-LIBRARY-NAME           OF QBN-LPGMI-PGML0100
                             TO PGMLIB         OF GETOBJAR.
           MOVE BOUND-MODULE-NAME              OF QBN-LPGMI-PGML0100
                             TO BNDMODNA       OF GETOBJAR.
           MOVE BOUND-MODULE-LIBRARY-NAME      OF QBN-LPGMI-PGML0100
                             TO BNDMODLIB      OF GETOBJAR.
           MOVE SOURCE-FILE-NAME               OF QBN-LPGMI-PGML0100
                             TO SRCFILNA       OF GETOBJAR.
           MOVE SOURCE-FILE-LIBRARY-NAME       OF QBN-LPGMI-PGML0100
                             TO SRCFILIBNA     OF GETOBJAR.
           MOVE SOURCE-FILE-MEMBER             OF QBN-LPGMI-PGML0100
                             TO SRCFILMBR      OF GETOBJAR.
           MOVE MODULE-ATTRIBUTE               OF QBN-LPGMI-PGML0100
                             TO MODATT         OF GETOBJAR.
           MOVE MODULE-CREATION-TIMESTAMP      OF QBN-LPGMI-PGML0100
                             TO MODCRT         OF GETOBJAR.
           MOVE SOURCE-FILE-UPDATED-TIMESTAMP  OF QBN-LPGMI-PGML0100
                             TO SRCFILUPD      OF GETOBJAR.
           MOVE SORT-SEQUENCE-TABLE            OF QBN-LPGMI-PGML0100
                             TO SRTSEQTAB      OF GETOBJAR.
           MOVE SORT-SEQUENCE-TABLE-LIBRARY    OF QBN-LPGMI-PGML0100
                             TO SRTSEQLIB      OF GETOBJAR.
           MOVE LANGUAGE-ID                    OF QBN-LPGMI-PGML0100
                             TO LANGUAGE       OF GETOBJAR.
           MOVE OPTIMIZATION-LEVEL             OF QBN-LPGMI-PGML0100
                             TO OPTLEVEL       OF GETOBJAR.
           MOVE MAXIMUM-OPTIMIZATION           OF QBN-LPGMI-PGML0100
                             TO MAXOPT         OF GETOBJAR.
           MOVE DEBUG-DATA                     OF QBN-LPGMI-PGML0100
                             TO DEBUGDTA       OF GETOBJAR.
           MOVE MODULE-CREATED-ON              OF QBN-LPGMI-PGML0100
                             TO MODCREATE      OF GETOBJAR.
           MOVE MODULE-CREATED-FOR             OF QBN-LPGMI-PGML0100
                             TO MODCRTFOR      OF GETOBJAR.
           MOVE RESERVED1                      OF QBN-LPGMI-PGML0100
                             TO RESERVED1      OF GETOBJAR.
           MOVE USER-MODIFIED                  OF QBN-LPGMI-PGML0100
                             TO USERMOD        OF GETOBJAR.
           MOVE LICENSED-PROGRAM               OF QBN-LPGMI-PGML0100
                             TO LICPGM         OF GETOBJAR.
           MOVE PTF-NUMBER                     OF QBN-LPGMI-PGML0100
                             TO PTFNUM         OF GETOBJAR.
           MOVE APAR-ID                        OF QBN-LPGMI-PGML0100
                             TO APARID         OF GETOBJAR.
           MOVE MODULE-CREATION-DATA           OF QBN-LPGMI-PGML0100
                             TO MODCRTDTA      OF GETOBJAR.
           MOVE MODULE-CCSID                   OF QBN-LPGMI-PGML0100
                             TO MODCCSID       OF GETOBJAR.
           MOVE OBJECT-CONTROL-LEVEL           OF QBN-LPGMI-PGML0100
                             TO OBJCTLLEV      OF GETOBJAR.
           MOVE PERFORMANCE-DATA               OF QBN-LPGMI-PGML0100
                             TO PERFROMDAT     OF GETOBJAR.
           MOVE PROFILING-DATA                 OF QBN-LPGMI-PGML0100
                             TO PROFDATA       OF GETOBJAR.
           MOVE RESERVED4                      OF QBN-LPGMI-PGML0100
                             TO RESERVED2      OF GETOBJAR.
           MOVE NUMBER-PROC                    OF QBN-LPGMI-PGML0100
                             TO NUMPROC        OF GETOBJAR.
           MOVE NUMBER-PROC-BLK-REORDERED      OF QBN-LPGMI-PGML0100
                             TO NUMPROCBR      OF GETOBJAR.
           MOVE NUMBER-PROC-BLK-ORDER-MEASURED OF QBN-LPGMI-PGML0100
                             TO NUMPROCBO      OF GETOBJAR.
           MOVE RESERVED3                      OF QBN-LPGMI-PGML0100
                             TO RESERVED3      OF GETOBJAR.
           MOVE NUMBER-SQL-INST                OF QBN-LPGMI-PGML0100
                             TO NUMSQLINS      OF GETOBJAR.
           MOVE RELATIONAL-DATABASE            OF QBN-LPGMI-PGML0100
                             TO RELDBA         OF GETOBJAR.
           MOVE COMMITMENT-CONTROL             OF QBN-LPGMI-PGML0100
                             TO COMMCTL        OF GETOBJAR.
           MOVE ALLOW-COPY-DATA                OF QBN-LPGMI-PGML0100
                             TO ALLCDATA       OF GETOBJAR.
           MOVE CLOSE-SQL-CURSORS              OF QBN-LPGMI-PGML0100
                             TO CLOSQLCUR      OF GETOBJAR.
           MOVE NAMING-CONVENTION              OF QBN-LPGMI-PGML0100
                             TO NAMCONV        OF GETOBJAR.
           MOVE DATE-FORMAT                    OF QBN-LPGMI-PGML0100
                             TO DATFOR         OF GETOBJAR.
           MOVE DATE-SEPARATOR                 OF QBN-LPGMI-PGML0100
                             TO DATSEP         OF GETOBJAR.
           MOVE TIME-FORMAT                    OF QBN-LPGMI-PGML0100
                             TO TIMFOR         OF GETOBJAR.
           MOVE TIME-SEPARATOR                 OF QBN-LPGMI-PGML0100
                             TO TIMSEP         OF GETOBJAR.
           MOVE DELAY-PREPARE                  OF QBN-LPGMI-PGML0100
                             TO DELPERPARE     OF GETOBJAR.
           MOVE ALLOW-BLOCKING                 OF QBN-LPGMI-PGML0100
                             TO ALLBLOCK       OF GETOBJAR.
           MOVE DEFAULT-COLLECTION-NAME        OF QBN-LPGMI-PGML0100
                             TO DEFCOLLECT     OF GETOBJAR.
           MOVE SQL-PACKAGE                    OF QBN-LPGMI-PGML0100
                             TO SQLPACK        OF GETOBJAR.
           MOVE SQL-PACKAGE-LIBRARY            OF QBN-LPGMI-PGML0100
                             TO SQLPACKLIB     OF GETOBJAR.
           MOVE DYNAMIC-USER-PROFILE           OF QBN-LPGMI-PGML0100
                             TO DYNUSEPRF      OF GETOBJAR.
           MOVE SQL-SORT-SEQ-TABLE             OF QBN-LPGMI-PGML0100
                             TO SQLSRTSET      OF GETOBJAR.
           MOVE SQL-SORT-SEQ-TABLE-LIB         OF QBN-LPGMI-PGML0100
                             TO SQLSRTSETL     OF GETOBJAR.
           MOVE SQL-LANGUAGE-ID                OF QBN-LPGMI-PGML0100
                             TO SQLLANG        OF GETOBJAR.
           MOVE CONNECTION-METHOD              OF QBN-LPGMI-PGML0100
                             TO CONNECTM       OF GETOBJAR.

           EXEC SQL
           INSERT INTO GETOBJPF VALUES(:GETOBJAR.PGMNAME,
           :GETOBJAR.PGMLIB, :GETOBJAR.BNDMODNA, :GETOBJAR.BNDMODLIB,
           :GETOBJAR.SRCFILNA, :GETOBJAR.SRCFILIBNA,
           :GETOBJAR.SRCFILMBR, :GETOBJAR.MODATT, :GETOBJAR.MODCRT,
           :GETOBJAR.SRCFILUPD, :GETOBJAR.SRTSEQTAB,
           :GETOBJAR.SRTSEQLIB, :GETOBJAR.LANGUAGE,
:GETOBJAR.OPTLEVEL,
           :GETOBJAR.MAXOPT, :GETOBJAR.DEBUGDTA, :GETOBJAR.MODCREATE,
           :GETOBJAR.MODCRTFOR, :GETOBJAR.RESERVED1,
:GETOBJAR.USERMOD,
           :GETOBJAR.LICPGM, :GETOBJAR.PTFNUM, :GETOBJAR.APARID,
           :GETOBJAR.MODCRTDTA, :GETOBJAR.MODCCSID,
           :GETOBJAR.OBJCTLLEV, :GETOBJAR.PERFROMDAT,
           :GETOBJAR.PROFDATA, :GETOBJAR.RESERVED2, :GETOBJAR.NUMPROC,
           :GETOBJAR.NUMPROCBR, :GETOBJAR.NUMPROCBO,
           :GETOBJAR.RESERVED3, :GETOBJAR.NUMSQLINS, :GETOBJAR.RELDBA,
           :GETOBJAR.COMMCTL, :GETOBJAR.ALLCDATA, :GETOBJAR.CLOSQLCUR,
           :GETOBJAR.NAMCONV, :GETOBJAR.DATFOR, :GETOBJAR.DATSEP,
           :GETOBJAR.TIMFOR, :GETOBJAR.TIMSEP, :GETOBJAR.DELPERPARE,
           :GETOBJAR.ALLBLOCK, :GETOBJAR.DEFCOLLECT,
:GETOBJAR.SQLPACK,
           :GETOBJAR.SQLPACKLIB, :GETOBJAR.DYNUSEPRF,
           :GETOBJAR.SQLSRTSET, :GETOBJAR.SQLSRTSETL,
           :GETOBJAR.SQLLANG, :GETOBJAR.CONNECTM)
           END-EXEC.

           MOVE SQLCODE OF SQLCA                TO SQL-RETURN-CODE.

           EVALUATE TRUE
             WHEN SQL-GOOD-RESULT
               CONTINUE
             WHEN OTHER
              MOVE WS-HOLD-PROG-NAME            TO ERROR-PGM-NAME
              MOVE WS-HOLD-PARA-NAME            TO LDA-PARA-NAM
              MOVE 'GETOBJA '                   TO CALLING-PGM-NAME
              MOVE SQLCODE OF SQLCA             TO PROGRAM-ERROR
              MOVE 'ERROR ON INSERT TO GETOBJPF'
                                                TO
                                                PROGRAM-MESSAGE
              PERFORM Z900-ABNORMAL-END
                      THRU Z900-ABNORMAL-END-EXIT
           END-EVALUATE.
           ADD SIZE-EACH-ENTRY OF QUS-GENERIC-HEADER-0100 TO
                                  RTV-START-POS GIVING RTV-START-POS.

       100-RTV-AND-INSERT-EXIT.
            EXIT.

       Z900-ABNORMAL-END.

            MOVE WS-HOLD-PARA-NAME              TO LDA-PARA-NAM.
            MOVE WS-HOLD-PROG-NAME              TO LDA-CALLING-PGM.
            MOVE ERROR-PGM-NAME                 TO LDA-ERR-PGM.
            MOVE PROGRAM-ERROR                  TO LDA-ERR-NBR.
            MOVE PROGRAM-MESSAGE                TO LDA-ERR-MSG.
            DISPLAY LDA-WORK-AREA UPON LOCAL-DATA-AREA.
            CALL CP-ABEND USING PROGSTATS.

       Z900-ABNORMAL-END-EXIT.
           EXIT.
