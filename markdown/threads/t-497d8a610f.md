Check out Oracle's SQL*Loader process. Works great if you're just trying to
load a flat-file to a database table. If you need to preprocess the data,
consider loading the raw data to some temporary tables on the database with
SQL*Loader and then use SQL*Plus or PL/SQL to populate the real table from
the temp(s). We've seen significantly better performance using some of these
techniques as opposed to using a COBOL-ESQL program.
