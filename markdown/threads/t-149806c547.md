If you're asking "On what day of the week did Jan 23, 2001 fall?" that can 
be computed.

Something like:

COMPUTE MY-DAYS = FUNCTION DAY-OF-INTEGER (MY-DATE-YYYYMMDD)

Which gives you the number of days since December 31, 1600. Then,

DIVIDE MY-DAYS BY 7 GIVING JUNK REMAINDER DAY-OF-WEEK

where DAY-OF-WEEK is a number between 0 and 6.  You now have only to figure 
out what days of the week the numbers 0 through 6 represent. Using any old 
known date as an example, you can determine that by experiment.
