IBM's COBOL for MVS and VM also has an intrinsic function for square
roots. Example (Page 73 of the programming guide):

Compute Z = Function Log(Function Sqrt(2 * X + 1)) + Function Rem(X 2)

A little excessive, but you get the idea.
