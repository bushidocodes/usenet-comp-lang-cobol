Actually, it is a requirement that the two fields be the same size.
Therefore, the length must be verified before the first execution
of the routines. Consider the following:

       procedure division.
       start-of-program.
           if function length ( input-field )
                   not equal function length ( output-field )
               display "Right justify field lengths are different"
               stop run
           end-if
      * Now get on with the reset of the program

Or make certain they are the same size when defined.
