```
Peter wrote in message ...
>I'm researching the migration from MF-COBOL to Fujitsu-COBOL. I need to
>be able to get the escape-key as TERMINATOR.
>Anyone know how???

You can define all the keys in a keyboard file. Search the
manual for the file extension KBD.  (The sample file given
is called COBOL.KBD)

You can set the terminator keys in the environment as well,
but it is better (in my opinion) to set up the file, then specify
the file in the environment instead.  That allows you to nail
them all in one place.  After I/O, it will put the exception
keystrokes into the key status register, and you can look at
them.  If you have a problem, tell me and I will post an example.
```
