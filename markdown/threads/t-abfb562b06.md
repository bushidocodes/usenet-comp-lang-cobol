```
Does anyone know of a softcopy (BookManager, Adobe Acrobat Reader, etc.)
COBOL Language Manual that our programmers in Northern Ireland could
download from the Web, or otherwise obtain, that would contain
documentation of the COBOL Report Writer feature?

My company is using these programmers to do a lot of conversion of OS/VS
COBOL programs to COBOL for OS/390 & VM.  We have roughly 800 programs
that contain Report Writer code (out of tens of thousands).

In California, we have old copies of OS/VS COBOL and even MicroFocus
COBOL language manuals that contain documentation of this feature, but
newer IBM manuals that are easily obtainable on the Web don't include
the feature, because IBM dropped it when the 1985 ISO / ANSI Standard
made it "optional" for a "Full" compiler.

Please reply to "ccampbe2@csc.com", and many thanks in advance.
Colin Campbell
```
