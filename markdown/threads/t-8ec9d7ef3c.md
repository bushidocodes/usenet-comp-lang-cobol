One idea:  Accept (after prompt) an unsigned 5digit integer representing
the number of days since the beginning of the 20th century (19010101 = day
zero) and display the resulting date in CCYYMMDD format.  Opposite problem,
accept any date between 1901-2099 and return (or display) the first
number.  Good practice in MOD arithmetic and Julian date conversion.  Have
fun!
