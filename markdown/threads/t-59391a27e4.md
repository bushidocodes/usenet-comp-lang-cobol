The answer to this depends entirely on two things: A) what mainframe compiler are you using and B) how many extensions are you
using. If you use only ANSI syntax on the mainframe, then you should be able to port your code very easily. (There should be a
flagging directive with your mainframe compiler to tell you what extensions you have used.) Even if you have used some (lots?) of
extensions in your mainframe COBOL program, there are several Unix COBOL compilers that provide "compatibility mode" options -
usually these are for IBM mainframe COBOL extensions, but I think some exist for other mainframe dialects as well.
