Probably NOT what you want, but you might want to check into it anyway.
IBM offers a productset called "SMARTdata Utilities for AIX. Assuming that
you are using AIX for your UNIX, this would give you a method of getting
your UNIX files to something that is an IBM VSAM-like dataset - readable on
all IBM platforms (MVS, OS/2), Windows, etc. I know that Micro Focus can
also read these files. As I say, this is probably NOT what you really
want - but it would do the trick (with a bit of overkill).

Arnel,
Try dumping your ASCII file ( I assume it is indexed) to a flat
(sequential) file in key sequence. Then use the REBUILD utility that
came with MF COBOL to build an ISAM file. I have done this in the past
and it works fine as long as the number of duplicate keys does not
exceed 64,000 (this is not pertinent if you do NOT have dupe keys)..

Good Luck!
