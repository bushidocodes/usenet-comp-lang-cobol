### PowerCOBOL Migration tool is available (at last...)

```
The new PRIMA tool to migrate PowerCOBOL projects to .Net is finally
downloadable from the PRIMA web site. (http://primacomputing.co.nz)

This tool converts PowerCOBOL sheets (forms) into Windows Forms and
extracts ALL of the existing COBOL behind the sheets (scriptlets) so it
can be used with the new Windows Forms. (It is about salvaging what can
be a heavy investment in PowerCOBOL, and also providing a path for the
future with new languages AND/OR COBOL.)

The tool also automatically detects all references in the code to
PowerCOBOL controls/objects and Transforms them to equivalent invokes
against the new WinForms.

The downloadable DEMO will run for 1 year but it is limited to 12
screens. (2 of these are expected to be the demo baseline that we
provide with the install, so you get 10 of your own...)

You can extend the number of screens by purchasing more at the new
PowerCOBOL online store. Each screen the tool converts, saves around 4 -
6 hours of programmer time, so we are giving you a programmer/week to
get you started... :-)

You can upgrade to a licensed version which comes with support and free
upgrades, at any time, by purchasing the license at the online store
"Upgrade" option, or contacting PRIMA.

The whole site will be officially relaunched when the COBOL21 stuff is
completed (getting close now...) but, in the interim, adventurous types
can download the tool and play with it.

Enjoy!

Pete.
I used to write COBOL; now I can do anything...
```
