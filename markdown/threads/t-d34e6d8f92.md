I don't think this will work, however have you considered using the string
statements to do it?

UNSTRING src DELIMITED BY "ABC"
INTO a1, a2, a3,a4, a5, a6, a7 [...]
ON OVERFLOW

END-UNSTRING


Then
STRING a1 DELIMITED BY SPACE
"XYZ123" DELIMITED BY SIZE
a2 DELIMITED BY SPACE
"XYZ123" DELIMITED BY SIZE
a3 DELIMITED BY SPACE
"XYZ123" DELIMITED BY SIZE

...
INTO dest
ON OVERFLOW

END-UNSTRING

This of course assumes that you can guess the max number of occurances of
ABC and that there are now spaces in the src string.

More complicated would be to use the
MOVE 1 TO tallyho
PERFORM UNTIL (tallyho =0)
MOVE 1 TO tallyho
INSPECT src TALLYING tallyho FOR CHARACTERS BEFORE INITIAL "ABC"
IF (tallyho > 0)
ADD FUNCTION LENGTH ("ABC") to tallyho GIVING tallyhe
SUBTRACT 1 FROM tallyho
STRING src(1:tallyho) DELIMITED BY tallyho
"XYZ123" DELIMITED BY SIZE
src(tallyhe:maxsrc) DELIMITED BY SIZE
INTO dest
END-STRING
END-IF
END-PERFORM

The above hasn't been check but it's should be close enough to give you
the idea.

Greg
