```
The TinyCOBOL compiler version 0.2 is available for download at the TC
home page (1) and Metalab (2).

This release contains some bugs fixes, and some code enhancements,
including updates to the IF/ELSE relational, class, name, sign
conditionals statements. The CALL BY VALUE has been added, as an
extension to the standard COBOL grammar.

Please report any bugs and/or problems to the TC mailing list. 



David

1) TinyCOBOL Home Page
   http://tiny-cobol.sourceforge.net/

2) Metalab
   http://metalab.unc.edu/pub/Linux/devel/lang/cobol/
```
