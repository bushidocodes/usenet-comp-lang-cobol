The Micro Focus X"91" call only returns whether the command was submitted to
the DOS command processor - NOT the actual return-code of the executable (or
.BAT / .CMD) file.  I have tried numerous ways to get this to work, but to
no avail.  The only thing I would suggest is to write a .CMD that invokes
the .EXE and put the errorlevel logic in the .CMD file.

Hope this helps...
