```
nishankjain.csc@gmail.com wrote:
> I want to define -.5 in COBOL .. .how can i do that

01  NUMERIC-CONSTANTS.
     05  MINUS-POINT-FIVE    PIC SV9  VALUE -.5.

or PIC S99V9 COMP-3 VALUE -00.5.

Do you have any concerns about precision, or how the number is stored in 
memory or in a file (DISPLAY USAGE, PACKED-DECIMAL (IBM COMP-3), or binary)?

Things might work a little differently if you want to display that number.

05  EDITED-NUMBER  PIC ZZ9.99-.

MOVE MINUS-POINT-FIVE TO EDITED-NUMBER
DISPLAY "MINUS-POINT-FIVE = " EDITED-NUMBER

With kindest regards,
```
