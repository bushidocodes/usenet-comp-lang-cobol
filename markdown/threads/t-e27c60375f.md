Are you using WRITE .... or WRITE ... FROM ... without adding the ADVANCING syntax.

COBOL has pagination and spacing control built in to it, depending on how well this has been implemented the problem should be easily solved by using the

WRITE ... FROM ... AFTER ADVANCING PAGE or
WRITE ... FROM ... BEFORE ADVANCING PAGE

depending on when you decide that the page is full or a new page is desired.

You may find the LINAGE IS clause of the FD to be usefull in specifying page geometry, and to automatically provide a line counter for your usage.
