How is the field defined?

If the field is comp this is usually a binary integer of some sort.

Others are comp-3 (packed with each digit a nibble low order nibble is
the sign) 
comp-1 which is floating point (I think)
and regular signed ASCII (this might be complicated by ASCII sign 
indicators or EBCDIC signs)

Some compilers (Microfocus) support comp-5 which is "native" binary.
