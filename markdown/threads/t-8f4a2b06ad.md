```
I don't think that this is off-topic - or certainly not completely.  FYI, I
just put the following information out on the IBM-MAIN list server and
thought that it might be of interest to some readers of this NG as well.

If you aren't aware of it, the SHARE handouts from the session,

 "How to Solve Abends using Language Environment Dumps"

are now available (in Acrobat PDF format) at

 http://www.s390.ibm.com/le/conference/s8209.pdf

Obviously, the foils don't cover everything that BEING there would do (End
Plug for SHARE) - but I think these will help a lot of the common questions
on IBM-MAIN.

P.S.  These foils do NOT cover the new TERMTHDACT options:
        - UAONLY
        - UAIMM
        - UATRACE

For information on those, see
 http://www.s390.ibm.com/bookmgr-cgi/bookmgr.cmd/BOOKS/ceea306/2%2e1%2e53

or for the web-less, the section, "2.1.53 TERMTHDACT" in
 Title: OS/390 V2R7.0 Lang Env for OS/390 & VM Prog Ref
 Document Number: SC28-1940-06

but to "wet your appetite" the Summary of Changes describe these as,

"Three new TERMTHDACT sub-options that will allow the supression of CEEDUMP
while obtaining a SYSxDUMP. This will allow the capture of a system dump
prior to Language Environment condition management processing. ..."
```
