I take it that these are 'QuickWin' programs now. The size of 2/3
of the full screen is about what I get too.

Leaving them as DOS programs running a a DOS box allows multiple
copies to be run and the font-size can be changed for various
window sizes, there are even additional DOS fonts that are better
than the supplied ones. It is also possible to change to full
screen using ALT-ENTER. None of this can be done with a QuickWin
program, at least not by any means that I have tried.

With DOS programs they also can pre-emptively multitask under
Win3.11 with appropriate settings.

Overall, I see not much point in QuickWin, though I do have
some that are used by clients. DOS versions of these are aslo
available, but they use the QuickWin version, don't know why,
perhaps because the DOS version requires a password and the
Win versions bypass this.
