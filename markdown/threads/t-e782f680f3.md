### Microfocus and Btrieve ??

- **From:** "ifb" <ua-author-4214812@usenetarchives.gap>
- **Date:** 1997-07-16T20:00:01+00:00
- **Newsgroups:** comp.lang.cobol
- **Message-ID:** `<01bc92f5$ddfb7ba0$ba5163c3@nt>`

```

we use micro-focus cobol version 3.2.31 for dos with btrieve engine.
we use these directives :

ANS85
NOSMALLDD
SEGSIZE(65536)
64KSECT
ERRQ
CALLFH(XFH2BTR)

COMPILE AND LINK
SET COBSW=+S5-N-F+P
XM COBOL MYPROG;
LINK MYPROG+XFH2BTR+EXTFH+EXTERNL+DBTRV,,,COBLIB+COBAPI /NOD;

AFTER HAVING OPENED A FILE,WE READ IT MANY TIMES WITHOUT ERROR.
AND SUDDENLY WITHOUT EXPLICATIONS AFTER EACH "READ NEXT" WE GET AN ERROR
CODE "47"
WHICH MEANS THAT THE FILE IS NOT OPENED.
WHEN WE CLOSE THE FILE,WE GET AN ERROR CODE "42" WHICH MEANS THE FILE IS
ALREADY CLOSED.

IF YOU HAVE ANY EXPLICATION ABOUT THIS,PLEASE CONTACT US TO I.··.@I··.FR

thanks
```
