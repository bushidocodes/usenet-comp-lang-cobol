Because Encina servers are started administratively, rather than from the
command line, debugging a server is a non-trivial task. As a starting point
I would suggest looking in the Monitor Programmer's Guide, section 5.6.4
"Debugging Processing Agents" for pointers on using different debuggers with
Encina servers.
