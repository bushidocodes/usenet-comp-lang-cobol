Hi

I am busy doing, at work, a conversion from VS COBOL to COBOL II and
one of the things that is giving a lot of trouble is changing the:

01 PRT-LINE.
05 ASA PIC X(001) VALUE '-'.
05 LINE PIC X(132) VALUE SPACES.


WRITE XXXXXX FROM YYYYYYY
AFTER POSITIONING ASA

In Cobol II it does not allow the positioning command and ASA.

My question is: Is there a way Cobol II accepts the ASA?
