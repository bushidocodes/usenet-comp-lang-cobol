```

Look at this:

› cobol: Severe: prog3.cob, line 213: Undefined name
› PRINT B-DETAIL-LINE.
› ------------^

The ^ error pointer is not pointing to B-DETAIL-LINE, but to the word PRINT,
with the message "Undefined name". There is no 'PRINT' command in COBOL.
The compiler pointed to the word "PRINT", and told you it was undefined. You
must pay attention to what the compiler is telling you.

The correct syntax is, for example:

WRITE B-DETAIL-LINE BEFORE 1.

Which prints the B-DETAIL-LINE and advances to the next line.
```
