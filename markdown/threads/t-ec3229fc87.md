If you can write (unload) the DAT files into a flat-file (sequential) and
unpacked format, you should be able to load them into a DBase DBF using the
'Append from filename type sdf' command.

I am vaguely familiar with MF Cobol on the pc and have dabbled seriously with
DBase for the last five years and have used this many times.
