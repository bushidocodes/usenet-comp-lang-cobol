       program-id. test-disp.

       author.     di systemer as / emo.

       environment division.
       configuration section.

       input-output section.

       file-control.

       data division.
       file section.

       working-storage section.
       77  ws-disp                  pic 9.9.

       01  ws-number                pic 9(02).
       01  filler redefines ws-number.
           03  ws-number-a          pic 9v9.

       procedure division.

       main-010.


           move    38               to ws-number.
           move    ws-number-a      to ws-disp.
           display ws-disp at 0101.
           accept  omitted.

           stop run.
