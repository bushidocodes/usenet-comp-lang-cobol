Hi.

With all problems like this, it is *far* *far* better to include a
small code snippet (preferably a compilable and runnable one), to
avoid getting too many useless replies based on guesswork.

Having said that, I'll take a guess that the file in question is
declared as LINE SEQUENTIAL ? If so, then change it to just being
a SEQUENTIAL file - line sequential means "text format compatible
with a common text editor", and therefore records are variable
length and terminated with whatever the host system's "newline"
sequence is.

If that doesn't help, you need to supply a lot more information.

Cheers,
Kev.
