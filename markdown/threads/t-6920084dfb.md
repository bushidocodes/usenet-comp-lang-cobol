Hi Christopher.

You require one SELECT statement for each logical file that you want a particular
program to operate on. You can have zero, one, or many SELECT statements
according to the file-handling requirements of the specific program and each one
can have a different organization, access mode etc.
You attach each logical file to a physical file using an OPEN statement.

SELECT statements have nothing to do with the "program" in the sense that
IS INITIAL in the PROGRAM-ID does. It`s not a behavioural thing.

Please shout if that's still not clear.

Cheers,
Kev.
