It looks like you are accessing a pre-existing file. When a file is OPEN'd,
the DCB for the file is filled first from the program, then the JCL (if any
DCB parameters) and lastly the catalog. In your case,  your program says it
has  a VB format and loads that info into the DCB. The  catalog knows it to be
FB, but the program takes overrides the catalog, and you get your mismatch.
Either change  your program to a FB format, or delete/redefine the file with
the appropriate attributes.  The system does not default an RECFM.

Regards
John
