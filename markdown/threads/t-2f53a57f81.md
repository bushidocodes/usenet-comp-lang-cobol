Question:
 "Does IBM have any plans to create an interface to allow Cobol programs to
call Java programs and pass parameter information?"

Answer from Tom Ross,

 "Yes. Any other questions? Actually, we are working toward this goal, but
Java has some unique challenges in this area, like not having structures,
records or CALL statements. We are making great progress, and will have some
announcements in this area soon, but it will be a first step.

Just out of curiosity, what platforms/subsystems are you interested in
running Java from COBOL programs in? Batch, CICS, Unix Systems Services,
IMS, or ?"
