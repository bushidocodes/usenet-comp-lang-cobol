```

Hello,

I am using MicroFocus NetExpress Version 2.0 on Windows
95.
I am trying to print to a NCR 7193 thermal receipt
printer. This is a 44 column serial printer that is
attached to COM1.

There are no window drivers provide with this printer
so I have set up the printer as "Generic Text/Only".

I am unable to send control codes to the printer. I can
send text with no problem. I need to be able to send
control codes such as x"1B6D" this allows me to cut the
receipt paper. I also have control codes for Bold and
Centering that i need to be able to send to the printer.

The following is the code that i am using:

set myprt to entry "mfprnt32"

call "PC_PRINTER_OPEN" using printer-handle,
document-title
by value printer-flags

move x"1B6D" to printer-buffer
move 2 to print-buffer-size

call "PC_PRINTER_WRITE" using printer-handle,
printer-buffer,
by value print-buffer-size

call "PC_PRINTER_CLOSE" using printer-handle

As i mentioned above I am able to send text to the printer
with no problem. I have also tried setting the printer
up as a "IBM Graphics Printer" this also does not work.

Could anyone give me some suggestions.

Thanks,
Bob Hennessey
```
