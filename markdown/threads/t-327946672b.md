This depends on the compiler and the op.system.

I use a mouse in Microfocus COBOL 4.0 under OS/2 as follows:

First, you must activate it with

  move 64 to use-mouse-function
  move  1 to activate-mouse

  call X"AF" using use-mouse-function
                   activate-mouse
  end-call

where you have declared

 05  use-mouse-function                   pic 9(2) comp-x.
 05  activate-mouse                       pic 9(2) comp-x.


Later, you can read the position with this AF call:

         call X"AF" using get-mouse mouse-details

with declarations

  01  mouse-details.
   05  mouse-x                              pic 9(4) comp-x.
   05  mouse-y                              pic 9(4) comp-x.
   05  mouse-status                         pic 9(4) comp-x.
  01  get-mouse                             pic 9(2) comp-x  value 67.

It's all described in the manuals.
