```
From Arthur Griffin, who wrote "COBOL for Dummies", posted here in
January:

--------------------------------------------------------------------
This serial number thing was an oversight. The installation
information for the other compilers is complete but, to install the
Fujitsu compiler, you may need to enter a serial number. If you do,
this one will work:
 103-2001 1699-03317-70168
The first part of the number should already be in the window during
the installation procedure, so all you will have to enter is
 99-03317-70168
The same number works for all the Fujitsu compilers.

Please accept my apologies for the omission but, hey, we all have the
right to be a dummy from time to time, right? If you have any other
problems, there is an 800 number at the bottom of page 368 that you
can use to find the answer to any other questions you might have about
the CD.
--------------------------------------------------------------------
```
