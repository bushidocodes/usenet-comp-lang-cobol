Hello fellow COBOL listers,

I am currently in the process of creating a new COBOL course with the (temporary) title
"COBOL for OS/390 - Advanced Programming Workshop"

While I have compiled a few topics that might be of interest to the participants, I wonder
if there are topics that I overlooked or which you would like to be included in such a
course.

This course is not intended to teach programming per se, but the COBOL features that are
around (or will be around soon (?)).

The planned environment is for OS/390 - but non-os/390 suggestions are very welcome

My current list of topics include (no particular order yet)

        Compile time options *)
	Run time options *)
	Compiler directing Statements
		COPY .... REPLACING
	Advanced Table processing
		Searching
		Occurs Depending On
		Hashing
	File processing
		Processing varying length records
	Intrinsic functions
		currently implemented
		functions proposed in the coming standard
	LE/370 callable services
	Sort and Merge
		QSAM
		VSAM
		tables
	Report Writer
	Performance Considerations
		PIC STRINGS
		USAGE clauses
		Statement usage
        Problem Determination
		Source level
			Declaratves
			WITH DEBUGGING MODE
		Post Mortem		
			Understanding the compiler listing
			Working with the Linkage Editor listing
		Post Moretem via Dump
			Basic Dump navigation 
				Instruction executed
				Variables involved
				Finding working-Storage and Linkage Section in the dump
		Post Mortem via LE/370
	
*) These will not be separate Sessions, but will be woven into the other session as needed


As I said, these are the sessions that are (nearly) finished.  If you have any comments
(good) or addtions/proposals (better) ... have a go at it
