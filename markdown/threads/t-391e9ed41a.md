In a CLIST I execute the following:

Phil,
SYSLIB is used by the compiler, not during execution. Your Cobol run-time
library needs to either be in your ISPLLIB (if running under ISPF using ISPEXEC
SELECT PGM), STEPLIB, JOBLIB, or linklist or lpa concatenation(s).
If it's not already in your ISPLLIB concatenation, try doing a ISPEXEC LIBDEF
ISPLLIB DATASET ID('run time library'), then ISPEXEC SELECT PGM(your-pgm).
Hope this helps :-)
