For powerful charts and graphs for your web-based applications, please
take a look at GRAFSMAN/WWW from Soft-tek International, Inc.

Easily built into your applications, GRAFSMAN/WWW generates a GIF image
of your chart and will automatically create an imagemap to go along with
it. You get dynamic graphs and the ability to make them an interactive
element in your application.

You will find a LIVE demonstration which uses a Perl script to prompt
you for data, then displays that data in a chart. An image map is
generated for the chart, allowing you to click on any chart element.
For the purposes of the demo, we are echoing back information about the
chart element that you selected. Of course, in your application, you
would be able to take other action based on the user selection.

You will also find links to demonstrations showing GRAFSMAN/WWW in
action with Web+ from TalentSoft:

http://www.soft-tek.com

You can download evaluation versions of GRAFSMAN for popular UNIX
platforms. We plan to have the Windows version available as a
downloadable module in the near future.

If you need charts and graphs for your web applications, take a look at
the one that can have you on-line in no time--GRAFSMAN/WWW!

Thank you,
