```
I downloaded it, but I haven't tried it out yet.  Documentation is
available in English or French.  It supports an odd format, AREA A
begins in Column 1.  It apparently supports the SORT verb, but does
not support the SEARCH verb or indexed files.  The compiler generates
x86 assembler code, which is then assembled and linked with the
supplied assembler and linker.  I think all the files I downloaded
(most are zipped) will fit comfortably on a single 1.44 meg floppy
disk.

I think a nice feature is that you can download the source code for
the compiler itself, which appears to be written in Borland C.  So it
might be possible to modify the compiler to suit your needs.
```
