Carlos,

I do basically what Gianni suggested regarding the laptop, except that I
don't "transfer in" and "transfer out".  Since I could not afford two 6.1
licenses for home use, I save my 6.1 projects in a unique folder, i.e.
MYPROJECT and I make sure the project file (myproject.ppj) is in the same
folder.  Then I network the folder over to my laptop and recompile the
project using Version 5.0 that I have on the laptop.  Obviously, if you are
using some features exclusive to 6.1, then this will not work.  But for most
of what I do, it's worked fine since 6.1 was released.  I think I've
encountered some minor exceptions with certain control properties, but I
can't recall what they were at this point in time, and I usually found a
different way to accomplish the same task using 5.0.  If I start using 6.1
features, then I'll "transfer in" and "transfer out" as Gianni suggested.

Also, if you ever upgrade hard drives as I did, you may need to call Fujitsu
to get your license reinstated.  Fujitsu was most helpful in doing this, but
it took a day or two. The Maxtor drive image utility that came with the new
hard drive did not copy the Fujitsu protection scheme over properly.  Just
something to keep in mind if your hard drive actually crashes and you have
clients waiting for fixes.

Regards,

Denny
