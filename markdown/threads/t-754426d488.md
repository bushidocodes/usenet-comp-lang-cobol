```
    1.  Backup old file (both pieces).
    2.  Assuming rebuild fails, beg borrow or steal an .IDX of the same
exact type from somewhere else.
    3.  Place old .DAT and new .IDX in same dir, with IDX renamed to same
root name as original.  New IDX can be empty, so long
          as it is valid.
    4.  Run rebuild.  When done, run again, REBUILD OLD.DAT,NEW.DAT /V
         No space around the comma.  /V produces a running count.

    You now have a totally rebuilt file.


    By the way, when you run rebuild, it wants the FILE.DAT, not FILE.IDX
name.
```
