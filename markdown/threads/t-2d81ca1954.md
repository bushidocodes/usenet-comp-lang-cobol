Example of calling Windows API is in DEMO-directory if you are using MF
COBOL...

You have to add a CALL-CONVENTION 74 IS WinAPI in SPECAL-NAMES..

Then just use CALL WinAPI "System" or "Beep" or whatever passing the
correct parameters...

You need i.e. MSDEV (Visual C/C++ and SDK Help-files) to verify the
correct parameters..
