Marc,

ACCEPT NOM is what you have.

ACCEPT NOM FROM DATE (or DAY-OF-WEEK, or TIME, or terminal-id, etc.) is what
you want.

You cannot do ACCEPT NOM without identifying what is getting accepted into
NOM.

The same goes for all the other ACCEPT statements you use in the code.

I'm not sure how your compiler treats that END-PERFORM in Calcul-Rem, but
mine would blow up if I had coded it that way.  Usually END-PERFORM is only
for inline PERFORMS, and this PERFORM is a normal out-of-line PERFORM.
