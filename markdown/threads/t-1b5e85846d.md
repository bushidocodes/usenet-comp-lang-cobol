```
I am New to Cobol Programming. I was given input to a numeric field
through keyboard using ACCEPT in a program. Generally,we know that A
Numeric field will store data with RIGHT JUSTIFICATION, but in my case
this numeric field stores the data with LEFT JUSTIFICATION.
For Example let us assume 
       05 empno pic 9(5). is a Numeric Variable.

If we enter a value 200, we know that it will store 00200 in empno.
But in my case that value is storing like 20000 in emp.
```
