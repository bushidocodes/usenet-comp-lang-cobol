Hi,
I've ordered it but it hasn't arrived yet. However, I followed the
course on CD and it is amazing. If it costed 10 times it's actual
price, it would still be worth it. It doesn't lecture you on theory,
or presents pre-compiled examples; every thing is done online, with
you seeing the typos and compilation errors. Really, a jewel. The only
drawback is that at the time my CDs where published, Fujitsu was on
release 1. Since then, they have a version 1a (with more support for
WinForms and WebForms) and the new features aren't covered on the
course.

best regards
Paulo Vieira
