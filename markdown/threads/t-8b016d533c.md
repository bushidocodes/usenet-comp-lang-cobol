Yes, that works (definately in version 3.2, probably in older versions (I
didn't check any others - if you want to know if it's supported in a specific
version, let me know)) :

working-storage section.
01 x pic 9(8) comp value 106.
procedure division.
display x
stop "Press "
display x at 1010
stop run.

... will display "00000106", then the same at screen coordinate 1010.
