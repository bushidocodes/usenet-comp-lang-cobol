```
depending on your platform, you might be able to do the following:

01  char-handling.
     02  the-value   pic s9(4) comp-4 value zeroes.            (or comp-5)
     02  filler            redefines the-value.
            03  filler        pic x.
            03  the-char pic x.          (may have to switch these two
around)

 01  printable-values.
       02  filler  pic x(15) value    "000000000000000".
       02  filler  pic x(16) value  "0000000000000000".
       02  filler  pic x(16) value  "1111111111111111".
       02  filler  pic x(16) value  "1111111111111111".
       02  filler  pic x(16) value  "0000000000000000".
       02  filler  pic x(16) value  "0000000000000000".
       02  filler  pic x(16) value  "1111111111111111".
       02  filler  pic x(16) value  "0000000000000000".
       02  filler  pic x(16) value  "0000001111100000".
       02  filler  pic x(16) value  "0000000000000000".
       02  filler  pic x(16) value  "0000000000000000".
       02  filler  pic x(16) value  "1111111111000111".
       02  filler  pic x(16) value  "0000000000000000".
       02  filler  pic x(16) value  "0000000000000000".
       02  filler  pic x(16) value  "0000011111111100".
       02  filler  pic x(16) value  "0000000000000000".
 01  filler redefines printable-values.
       02  print-mark pic x occurs 255 times.

01  the-text.
      02  the-text-char   pic x occurs 80 times.

01  the-subscripts.
      02  char-nbr    pic s9(3)  comp.
.....
perform transform-to-printable
    varying char-nbr from 1 by 1
      until char-nbr > 80
....
transform-to-printable.
     move the-text-char (char-nbr) to the-char
     if the-value = zero
     or print-mark (the-value) = "0"
           move "." to the-text-char (char-nbr)          [or whatever]
     .


the table has a 0 for non-printable chars and a 1 for
printable chars. You will have to set these correctly depending
the character set. The above settings are for illustration only.
I have arbitrarily chosen "." as to print for non-printables.
A different approach would be to store in the table what you
want printed for each character.
```
