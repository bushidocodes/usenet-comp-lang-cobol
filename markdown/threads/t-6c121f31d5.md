We just purchased an AIX box and installed the Micro Focus Developer Suite.
My "mission" is to port COBOL programs developed with Micro Focus COBOL in
Windows 95 to the AIX box.  I have no UNIX experience.	I'm trying to draw
parallels between our Windows 95 run-time system and AIX. 1.) PARM PASSING -
We execute our programs and pass parms into linkage via a Visual Basic
front-end.  I understand you run a program on AIX by typing it's name.	How
do you pass the parm ? 2.) EXTERNAL FILES - We SET our files to the system
via a bat file (that executes before the GUI comes up).  Is there such a
thing as a bat file (with SET commands)on AIX ?  Would it be run the same way
as a program - type it's name ? 3.) PACKAGING - On Windows 95, we linked our
programs into DLLs (with the Visual Basic being the exe which called the
first DLL which, in turn, calls other DLLs).  Would we still create DLLs and
have an "executable" program to call the DLLs ?
