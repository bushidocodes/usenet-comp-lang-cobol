Developing with PowerCOBOL is like developing with Visual Basic.  If a
Visual Basic programmer, wanted to do COBOL (Without any prior COBOL
experience) I think PowerCOBOL is the most like what they are used to.

Personally, I use several different COBOL compilers, but a common
interface tool - COBOL sp2 from Flexus.   It's not like developing in
Visual basic at all.  It's much easier!
