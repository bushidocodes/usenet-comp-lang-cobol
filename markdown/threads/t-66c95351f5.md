Take a deep breath, Woody. Help is (probably) on the way.

First step, use MS-DOS to investigate the file. Do you get "reasonable"
results when you enter "TYPE COMPANY.TBL" at the MS-DOS prompt? If so then
the file is most likely a simple text file that you can edit with any
DOS text editor.

Step two, if necessary, is to use MS-DOS debug to look at the first
portion of the file. Enter "DEBUG COMPANY.TBL" at the MS-DOS prompt. You
should get a debug prompt of a hyphen. At this point type "d" which will
dump the
first 256 bytes of the file. Send this info back to me through e-mail
and I will try to get you some additional help. Must warn you, though,
that
this path may lead to A) lots of work and/or B) expense.
