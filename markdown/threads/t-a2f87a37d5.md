We were using AcuCobol to run an application with around 250+ OLTP users. Quite smooth. Now we have converted out data
to Informix Online 5.0 and are using the same Cobol application with the Acu4GL interface (in case you did'nt know, Acu4GL is
an interface between AcuCobol and Informix/Oracle and does not require any changes to be made to the Cobol code).

of which are intermittent, some report programs now take as much as 5 times more time to complete. Our IBM RS/6000 580 with
512MB of memory is now running very slow !
