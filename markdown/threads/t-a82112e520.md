Here's an example:

DISPLAY "Enter your code name: " LINE 5 POSITION 5.
ACCEPT NAME POSITION 0 PROMPT NO BEEP TAB OFF.
DISPLAY " Your code name was: " LINE 7 POSITION 5.
DISPLAY NAME POSITION 0.

The key is the "OFF" control.

We can probably make the reasonable assumption that you are writing
in COBOL (though you may have simply cross-posted this request to
every comp.lang group), but these things tend to be compiler and
system dependant. You should specify which compiler and which
operating system.

If you have X-Open features in screen section then try the SECURE
clause, or WITH SECURE on the ACCEPT.

Some others may have WITH NO-ECHO.
