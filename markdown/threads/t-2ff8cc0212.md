Hi Robert,

Unfortunately the current Micro Focus Unix products are not thread safe,
which is a prerequisite for Java/COBOL integration.  This combined with
the lack of support for callable shared objects in the current Object COBOL
developers suite means you will not able to-do this using JNI. 

However you could use sockets or a similar transport mechanism, if COBOL
and Java were in separate processes.

I would recommend contacting your Micro Focus sales person and ask them
about the next generation of Unix products that will be available as it
may have some features you are interested in.
