```
The first place that I would start to look for "new" features is the
"substantive change" appendix/annex in each ANSI/ISO Standard.  To get a
"free" copy of the 2002 ISO COBOL Standard, check out:

 http://www.incits.org/tc_home/j4.htm

If you want to know what was added in the '85 Standard, then you would need
to buy a copy of that.  If, on the other hand, you want to know what
"non-standard" extensions exist in various vendors' compilers, then you will
need to look for each vendor's "Language Reference Manual".  A few that I
know about easily are:

  IBM:
       http://www-3.ibm.com/software/awdtools/cobol/library/

  Micro Focus:
      http://supportline.microfocus.com//documentation/index.asp

  Fujitsu:
      http://www.netcobol.com/download/manuals_online.htm
```
