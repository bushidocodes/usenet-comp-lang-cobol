01  OUTPUT-BUFFER     PIC X(120)

    STRING
       DATA-1   DELIMITED BY SIZE
       COMMA  DELIMITED BY SIZE
       SPACE   DELIMITED BY SIZE
       DATA-2   DELIMITED BY SPACE
       ...
    INTO  OUTPUT-BUFFER

in working-storage, PIC X(01) VALUE ','.
