>From your error message, I'm guessing you have a MicroFocus COBOL BUILD
application. If so, check the following:

1) Make sure the PSPACCRL.CFG is correct and in the proper directory (this is
probably OK - but check it).

2) Is the COBDIR environment variable set; if you're using MFEXTMAP, is the
COBDIR line in that the DAT file correct?

3) Do you have the whole OSX installed; or do you have all of the needed OSX
files in the COBDIR directory.

4) Does PSPACCRL.LBR exist in the working directory (or where ever the CFG
file points) and does it have a PSPACCRL.GNT initial program?

I suspect 2, 4, or 3 are the culprit, but the best thing for you to do is read
the chapter on the BUILD process in the TOOLSET reference.
