It has been a LONG time since I've worked with this, but I recall there is an
option in AMASPZAP or "IMASPZAP ?!?" to dump a program or module in a program into
hex, displaying the value of the data and giving the machine instruction for that
code if there is one.  Try to determine the entry point addresses for the modules
actually called, your cobol program would generate a call to IGZTRCL0 or IGZTRCL1,
and these wouldn't necessarily be at the start of your module.  HTH
