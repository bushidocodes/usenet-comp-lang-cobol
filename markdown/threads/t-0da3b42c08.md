Mark,
We've been running the VUPort product from San Soft, Inc. for about five
years.  It is extremely Wang-like in every way, including the command
processor, print/procedure queues, nearly all Wang utilities, etc.  In a
nutshell, VUPort is a pre-processor for Micro Focus COBOL and a runtime
system to provide the Wang environment.  It runs on all major flavors of
Unix.  See http://www.sansoft.com for more details (they're re-working
the site so it has a few "gaps").

If you're migrating yourself I highly recommend it.
