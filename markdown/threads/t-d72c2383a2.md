I have done a similar kind of thing, but to be honest I did it in C and
call it from a MF COBOL program. There are quite a number of API calls
required to do a decent job. If you are trying to add a print preview
feature and want to handle multipage documents it can be a little
involved. If you have MS C 2.0 or higher available, there is a sample
program included that does some bitmap printing. You should be able to
cannibalize the sample. Also a company named Flexus has a forms printing
tool available. They specialize in COBOL tools. Check them out at
www.flexus.com

Good luck
