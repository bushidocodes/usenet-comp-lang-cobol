The complete installation takes about 20Mb of disk space.  Version 3.2
is not really the latest. The last version for 3.2 is 3.2.5 and then
went up to v3.4.  Versions 3.2.5 and 3.4 are 16bit versions.

For the Workbench series you should have at least 17 manuals, 7 disks
for the Cobol, 5 disks for Toolset, 5 disks for Workbench, and 1disk
for Early Release.

To install, load the Workbench Disk 1 - Setup and run setup.exe then
follow the instructions on the screen. This will install workbench,
toolset and cobol.

If your version is below 3.2.5 you can download an upgrade to 3.2.5
from MF Website.

If you are using the product to develop software I would recommend you
look at Fujitsu Cobol v4.0 (www.adtools.com) as well as SP2 from
Flexus International (www.flexus.com).

If you need more info, email.

Good Luck.


Tony M. Mina
