You *can* continue to use the Microsoft COBOL version 5 compiler and
create GUI programs.

can call WinAPI functions to display GUI screens. You do need the
Windows SDK to take this route.

You can also try sp2. Download the 16 bit version for Micro Focus
from our web site. It will work with your compiler.
