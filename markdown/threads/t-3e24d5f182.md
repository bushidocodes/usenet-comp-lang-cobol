To do square or cubed roots in RM/COBOL, use this statement:

compute RESULT rounded = NUMBER ** 0.5

(or 0.333333.. for cubed). The ** can be used for exponentiation by
using a whole number instead of a fraction.

Can't speak for RM/COBOL but this works on IBM COBOL (and I think
most ANSI compatible compilers:

COMPUTE SQUARE-ROOT = (ORIGINAL-NUMBER ** 1/2)

This takes advantage of the fact the raising a number to the
inverse of the power results in the root value.

Also, more current compilers implement a function for square root
calculation.
