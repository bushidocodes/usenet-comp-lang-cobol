Greetings,
(We have the full version, so the demo version may not be the same)
Choose 'Generic Release Build' (NOT 'Generic Debug Build') in the IDE
Right Click on the .EXE, choose your 'Build Settings'
Rebuild the Project.
You should find the .EXE in a Sub-directory named RELEASE.
If you chose 'Shared RunTime' you will need to distribute the run time
modules as well as the .EXE.

Under the IDE see menu 'HELP' submenu 'START HERE'.
