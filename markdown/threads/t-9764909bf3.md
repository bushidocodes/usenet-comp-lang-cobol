I hope I can make this clear! .

You know Comp or comp4 is Binary, 1 byte for every 2 bytes of numbers.

So a 255 would be stored as x'00FF'.

In Comp-5 the same rules apply except it uses the "intel" byte reversal
scheme, that confuses many.

The same 255 in COMP-5 would be stored:

x'FF00'

Hope that helps some.
