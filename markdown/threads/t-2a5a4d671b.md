There are at least 3 (maybe 4) user interface options with NetExpress.
 Are you using Dialog System, the HTML template, Screens, or the 
Screen Section?

With the Screen Section you can do what you are trying to do by using 
the "Control" option in the screen definition.  You assign control for
each field to a specific variable, then move the desired display 
attributes into this variable and re-display the screen.
