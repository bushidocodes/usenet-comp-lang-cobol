Most newcomers to Fujitsu encounter this.  When compiling, select options,
add, select "Main" then add.  Now select "Compile program as a main
program" you should now be ok.  The default compile option is subprogram ! 
I have found I need to set this option for each new folder I work in.
If you search the Fujitsu web site there are a series of tutorial's which
get us newbies up and running.  They cover Compile,link,execution --
Editing -- Using the Debugger (again will not work unless you set some
options) -- Runtime filenames -- The file utility.
It is well worth checking these pages out.

The compiler is actually not any more complex!  What you have to do is
indicate that the program is a "MAIN" program.  The Fujitsu Default is
to compile as a sub program.  The simplest way is to insert a line at
the top of the program, case sensitive, starting in column 8 that
says:

@OPTIONS MAIN
