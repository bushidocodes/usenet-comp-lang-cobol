The best place to get an inexpensive compiler or to learn how to write
COBOL code is to buy Thane Hubbell's book, "Teach Yourself COBOL In 24
Hours!"

additional charge.

It is not only a great book, but also incudes 16 and 32 bit versions
of the Fujitsu COBOL compiler.  The ISBN number of Thane's book is:

0-672-31453-3
