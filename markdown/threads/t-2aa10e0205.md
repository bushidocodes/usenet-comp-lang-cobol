It is pretty standard across all systems for the PGT and
TGT, but the linkage editor and executable module formats
differ across platforms. For instance, on S/370, the
module format will not match that found on OS/2 nor MSDOS-
based platforms, nor Windows NT.

The TGT contains the special register words, base-locator cells, sort
parms for the SORT/MERGE facility, etc.
The PGT usually contains global constants and/or vn
cells and perform-save cells.
