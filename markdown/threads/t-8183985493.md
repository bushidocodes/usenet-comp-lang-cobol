Hi,

I'm wondering where you did put the declaration of set coblib= ??? I hope
not in autoexec.bat, because NT does not use that file.

I hope not in autoexec.nt, because NT usually forgets about it to.

The only correct place to declare them is in Control Panel, System,
Environment.  You then have two options.  You can put it under System
variables, so that every user has them, or only under the current logged on
user.
