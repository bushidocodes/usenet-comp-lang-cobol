If you still have access to the Micro Focus COBOL software you can use
the Data File Editor (DFED) to look at the file(s). Additionally you
can use the Workbench File Loader (WFL - called "woofle" by those of us
who have been using it for years) to transfer the format of the
file/record between different formats. Look for a .IDX file with the
same name as a .DAT file to determine if you are working with an indexed
file structure.
