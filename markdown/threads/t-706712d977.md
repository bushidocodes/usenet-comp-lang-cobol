Paul,

To call Btrieve native functions from Fujitsu COBOL 32-bit Windows program
you need to at the minimum the Btrieve 6 Developer Kit
http://www.pervasive-sw.com/products/other/bti_devkit.html

Once you have the Btrieve Developer Kit you can call Btrieve native
functions from within any Fujitsu COBOL using the code sample below.  To
properly link the Fujitsu COBOL to the Btrieve DLL you need to include the
Btrieve W3BTRV6.LIB or W3BTRV7.LIB file in your Fujitsu COBOL P-STAFF
project, or on the LINK.
