```
Your problem is that you are lying to the compiler. :-)  When you use
a numeric PIC, you are telling the compiler that field is numeric,
but in your case, you're not sure.  If the field could possibly *not*
be numeric, never reference it using a numeric PIC.  There are a number
of ways to address this, but one solution is something like this:

    03  MY-FIELD-ALPHA.
        05  MY-FIELD-NUMERIC    PIC  9(6).

Reference MY-FIELD-ALPHA to test for numeric, then if it is, reference
MY-FIELD-NUMERIC to use it. :-)
```
