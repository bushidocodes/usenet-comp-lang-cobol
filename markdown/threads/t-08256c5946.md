languages, actually to c++ and soon to java too. !!!
The resulting code is excelantly maintainable. We charge

0.5 to 1.5 USD per line of cobol code
small portions for demonstration are free


Y E A R 2 0 0 0 - S O L U T I O N

Our solution:

1. We start a cobol program (converted c++) twice. The second task
executes the code identicaly but the current date. A check is made and
links to the origin source lines will be protocolled for solving the problem
interactivly.

2. Double security during runtime. The check between the two tasks
will be made during runtime and automatic correcetion or interactive
correction assisted or non-assisted will happen.
