In Acu/COBOL the format is:

ACCEPT FIELD, LINE ##, POSITION ##, BEFORE TIME ##hundreths seconds##
ON EXCEPTION EXC-FIELD PERFORM EXC-ROUTINE.

IF EXC-FIELD = 99, .. timeout ..

These routines became available with COBOL 85.
