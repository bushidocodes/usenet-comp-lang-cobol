The current way you do it is:

CALL "SYSTEM" USING command-line [repaint-screen] [exit-code]

command line is an alphanumeric data item that contains the
command line to be passed to the operating system.

repaint-screen is an optional one-byte alphanumeric data item that
should be set to "Y" or "N"; the first repaints the screen after the
completion of the command.

exit-code is an optional numeric data item that contains the exit
code of the command upon returning from the operating system. It
must be declared as PIC S9(4) BINARY.

This is the v6 description. I don't think older DOS versions had the
repaint screen nor exit-code options. If you don't see a program
called SYSTEM.EXE in your RM/COBOL directory, then you're probably out
of luck.
