```
There was a question (either in comp.lang.cobol or IBM-MAIN) about finding
the "COBOL file buffers" in a CEEDUMP output file.  I can't find the
original to respond to, but here is the reply that I got back from my
"usually reliable source".

"The file buffers were added to the CEEDUMP starting in LE 5.0 with APAR
PQ07904 (superceded by APAR PQ15428).

So, as long as they are at current maintenance at LE 5,0 or later, they
should get the file record contents displayed for each open file in the
COBOL program (it is located just before the Working Storage for the
program)."

I hope this helps.
```
