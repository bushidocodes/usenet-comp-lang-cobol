Greetings all

I hope this question doesn't spark the controversy my last post did. I
haven't been able to keep up with the reply posting to that one. Thanks
to all how responded with good suggestions.

To the point tho'
We are a simple COBOL shop running several Unysis 2200 systems. Some of
these are written in DPS (those who know the 2200 will know what that
is). We are aware of a company that has ported LINC (another Unysis
COBOL based product) to run on a UNIX box.
What we are curious about is whether or not there is someone out there
who knows if there is a similar product/utility to allow DPS2200
programs to be ported to a UNIX box?

Any info/URL's/suggestions (polite ones Cory) would be much appreciated.

BTW We already know the one about throwing ANSII74 DPS2200 COBOL out the
window and starting again. Not an option I'm afraid.
