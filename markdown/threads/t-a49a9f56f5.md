The control character used in RECORD SEQUENTIAL files (to quote William
Kline) varies depending on whether you use FBA or FBM format on the output
file. Regardless the control character is the first character of the record.
Both formats have a fixed set of characters to indicate SINGLE, DOUBLE AND
TRIPLE spacing as well as a PAGE BREAK.

The only one I know of the top of my head is FBA or FA:
" " single spaced
"1" double spaced

not sure of the others. And of course this may be incorrect, but there
definitely is a fixed set of characters to determine line spacing and page
breaks.
