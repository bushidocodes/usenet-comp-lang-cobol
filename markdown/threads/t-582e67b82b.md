```
Hello.  A friend of mine gave me a COBOL compiler to help me with my business. 
I need to make an inventory report for my small business.  I know nothing of
COBOL at this time, but I'm trying to learn by some books he gave me.  If
someone could come up with a relatively simple Inventory report program, I'd
appreciate it.  Here's some of the guidelines I have worked out.

INPUT

Columns        Data
   9-13            Item Number
  34-48           Item Description
  49-50           Quantity on Hand
   51-52          Quantity Sold
   53-54          Store Number

OUTPUT
   The output will need to be a listing of input fields along with a generated
internal colde.  The internal code is a combination of the store number, the
last three digits of the item number, and the first two characters of the item
description.  Like I need store number 89 and item number 12345 with a
description of HAND SAWS would have an internal code of 89345HA.

Output as follows

Print Positions     Information
 6-7                     Store Number
13-17                   Item Number
23-27                   Item Description
43-44                   Quantity on Hand
51-52                   Quantity Sold
59-65                    Internal Code


I appreciate any help!  Feel free to email me.
AceCobra69
```
