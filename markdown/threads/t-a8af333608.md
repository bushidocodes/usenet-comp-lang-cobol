There is more than one way to select/assign. What you want is the file
name in a variable.

Well: like so:
SELECT INPUT-MSDOS-FILE
ASSIGN TO WS-FILE-NAME
ORGANIZATION IS LINE SEQUENTIAL
FILE STATUS IS WS-FILE-STATUS.

Further down in working storage:
01 VARIOUS-WORKING-VARS.
05 WS-FILE-NAME PIC X(09) VALUE "INPUT.TXT".

Also, you could use environment variables if it is Micro Focus
SET INPUTF=INPUT.TXT
in dos, and then in your program your assign statement would change.
The compiler directives control how you assign stuff.
ASSIGN TO INPUTF

In your compiler directive file (this will probably be different for
another vendor) you add the following:
ASSIGN(EXTERNAL)
this makes the second method work.
ASSIGN"dynamic" makes the working storage method work.

You could use the ACCEPT verb to retrieve the file name.
Or you could hard code it:
ASSIGN TO "INPUT.TXT".
To answer your first question:
You can accept the command line in Micro Focus via:

SPECIAL-NAMES.
mne-COMMAND-LINE IS COMMAND-LINE.
accept ws-hold from mne-COMMAND-LINE.
