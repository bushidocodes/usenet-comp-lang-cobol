CA-Realia cobol uses a special format for .CBL files. The Compiler assumes a special format
whre the first six columes are not present in the source. So if the extension is .CBL then
COBOL column 7 is assumed to be byte one of the source record. This is waht is causing all of
your problems. If you have standard COBOL format programs, change to another extension such as
SRC. Newer versions of our compilers allow you to alter the default extension formats that
would overcome your problem.
