```
OpenCOBOLIDE is the OpenCOBOL/GnuCOBOL Interactive Development
environment (IDE) written in Python for Linux, Windows, and Mac OSX.
The Windows installer version includes MinGW GnuCOBOL 1.1 (20JAN2014) so
you don't have to build the GnuCOBOL compiler from source.

It looks like OpenCOBOLIDE has been updated to version 4.7.2 very
recently. I believe the author is Colin Duquesnoy.

https://pypi.python.org/pypi/OpenCobolIDE

https://launchpad.net/cobcide/+download

You can also install GnuCOBOL 2.0 and change the settings in
OpenCOBOLIDE to use the newer compiler.

Last September I generated GnuCOBOL 2.0 (r624 10JUL2015) using MinGW
(Minimalist GNU for Windows) and source code from the GnuCOBOL project
on Sourceforge.net:
https://sourceforge.net/projects/open-cobol/

GnuCOBOL Manuals and Guides are available here:
http://open-cobol.sourceforge.net/

Recently I generated a newer version of GnuCOBOL 2.0 (r658 25OCT2015),
which can be downloaded from here:
http://www.arnoldtrembley.com/GC20base.zip

It's about 53 megabytes. Create a folder named something like
c:\GnuCOBOL or C:\GC20 and unzip GC20base.zip into it while preserving
the directory structure. Read the CMD files for an idea of how to set
up the environment variables if you want to compile GnuCOBOL 2.0 outside
of OpenCOBOLIDE. Otherwise, simply update the settings in OpenCOBOLIDE
to point to the GC 2.0 compiler in the folder where you installed it.

GnuCOBOL 2.0 is still a work in progress, but they are getting closer to
the final release, hopefully within a few weeks. I plan to rebuild the
GC 2.0 compiler and zip file when the stable build is available.


http://www.arnoldtrembley.com/
```
