In the VMS box please add the following command line juste
before the command run:

$       DEFINE  /USER SYS$INPUT SYS$COMMAND

Here is an example:

$       IF      P3 .EQS. "TEST" THEN DEFINE COB$SWITCHES "2"
$       PGM   = "$EXE:''P1'.EXE"
$       DEFINE  /USER SYS$INPUT SYS$COMMAND
$       PGM     "''P2'"

Note here P1 is the name of executable program where $EXE: is the directory
P2 is the parameter for the program running

You need to use the CLI command parameter.
