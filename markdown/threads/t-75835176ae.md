*.idx are traditionally joined to *.dat. It's ISAM files. You can read
some versions of .dat files on unix. You cannot read others because
their structure is special. This is also true on DOS/Windows. You can
use REBUILD utility to convert files on DOS/Windows without to know
the structure. You can use fhconvert on Unix, but need to know the
structure (latest versions of fhinfo can give you informations about
files)
Others files may be sequential (stream of data as on a tape) or line
sequential (text with delimiter, readable by text editor).
What is your cobol version number ? (type run on DOS, or cobrun on
unix)
