The switch is fairly painless.  There are a few things that
are not, mainly in the screen section.  The cursor control
works completely differently, and very poorly in comparison
to MF.  Clearing the screen has to be changed as the syntax
is different.  Copies have to be put in parenthesis, as does
program id.  I had to pull most of the code into the Fujitsu
editor to get rid of leading tabs and replace with leading
spaces.(just a load save).  First system took about four
weeks for 50 or so programs.  Second about two.

If you have assembler subroutines, that may be a problem.
I rewrote most of mine in Cobol.

Split keys have a different syntax.

The problems are there, but they are mainly simple syntactical
differences.  The code all tested once it compiled.

All in all, a fair amount of work, but straightforward.
