For give you a more precise explanation, I should know which version
is using.
But don't worry: the MICROFOCUS COBOL, for default, performs an
automatic display of the fields when it meet an ACCEPT verb.
If you don't want to see the data introduced in precedence there are 2
ways:
1) clean the fields (with spaces or zeros) before the ACCEPT verb (of
course ...).
2) change the configuration of the run-time using the specific
provided program included in the package, but I should know the exact
version that you are using for point out you the correct name (usually
is name is CONFIG.EXE); with this program you can disable the
automatic display (predisplay).
