```
Your manual is your friend!  The LABEL declarative is an IBM extension. 
In IBM CoBOL for MVS & VM the syntax is:

    USE [GLOBAL] AFTER [STANDARD] [BEGINNING] [FILE]
                                  [ENDING]    [REEL]
                                              [UNIT]

    LABEL PROCEDURE [ON] {file-name} ...
                         {INPUT    }
                         {OUTPUT   }
                         {I-O      }
                         {EXTEND   }


Notice that a label declarative is executed _after_ beginning a file, not
_before_, otherwise there would be no label to process.
```
