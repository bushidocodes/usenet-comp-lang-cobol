```
You gotta read some of the getting started stuff in the compiler product
documentation.  At least get down the basic steps to compile then run a
program.

As for editor, the compiler product probably has a built in editor.  Good
compilers have what they call IDE (Integrated Development Environment) when
editing, syntax checking, compiling, running is all done in one place.  I'm
sure MF Cobol has such an IDE.

As for editor you can basically use anything you want.   We use BRIEF, a
good-ol' DOS type editor for programmers, and I like PFE (Programmer File
Editor).  I've even used BBEdit on my Macintosh when working at home.

As for a book, I recommend "Structured COBOL Programming" by Stern & Stern. 
It's a good textbook that explains things clearly.  It sticks to ANSI standard
COBOL.
```
