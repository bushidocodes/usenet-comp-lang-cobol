05 DAY-OF-THE-WEEK PIC 9(1).


COMPUTE DAY-OF-THE-WEEK = FUNCTION MOD
(FUNCTION INTEGER-OF-DATE(CCYYMMDD), 7) + 1.


Result is 1=sunday, 2=monday etc.
```

```

Compute Which-Day = (Function Integer-of-Date (CCYYMMDD-field)) / 7

The value will be 1-6 - with the same values as are returned from ACCEPT
FROM DAY-OF-WEEK. However, you will get a value of 0 when ACCEPT gives 7.

Note: This assumes that you are using a COBOL product that has the intrinsic
functions in it.
