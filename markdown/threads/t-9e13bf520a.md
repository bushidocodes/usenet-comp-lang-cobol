Hi again William, 

I got it now what you want. 

You want to use function keys in COBOL program and want to associate them 
with the indicators. 

Simply go to the DDS and define your function keys which could be 01 to 24.
Here is the example for you. 
                              CF01(01 'Update Record')
                              CF02(02 'Previous Screen')
                              CF03(03 'Exit Program')

In this above example you are associating indicator 01 to Function key 1 
and CF02 indicator will be 02 and for CF03 indicator 03 is defined. 

Now in COBOL program you can use these indicator. Please note that these 
indicators are booleen values where "1" would be On and "0" would indicate 
OFF. 

When user pressed any of the above function key appropriate indicator will 
be turned On. 

COBOL program would like something like this. 

IF CF01 = On    
   Perform Update-Record
End-If

If CF02 = On 
   Perform Previous-Screen-Routine
End-If

If CF03 = On 
   Perform Exit-Program-Routine
End-IF.
