To write a "control character", such as cr or linefeed, to a file in
ICobol, define the field as PIC 99 COMP. Move the equivalent ascii(?)
value to the field...

e.g move 10 to 'comp-field'

will insert the linefeed/newline character.

I have no proper documentation handy so I don't know the equivalent for
ESC. It shouldn't be difficult to figure it out.

The ASCII ESC character is actually hex 1B. It's decimal 27. I use the following
in my WORKING-STORAGE as setup strings for a printer. The comments beginning
with <- are not valid in COBOL and should be omitted.

02 HP-ON-BLOCK.
03 FILLER PIC 9(2) COMP-X VALUE 27. <-This is ESC character
03 FILLER PIC 9(2) COMP-X VALUE 38.
03 FILLER PIC 9(2) COMP-X VALUE 108.
03 FILLER PIC 9(2) COMP-X VALUE 51.
03 FILLER PIC 9(2) COMP-X VALUE 65.
02 HP-OFF-BLOCK.
03 FILLER PIC 9(2) COMP-X VALUE 27. <-This is ESC character
03 FILLER PIC 9(2) COMP-X VALUE 38.
03 FILLER PIC 9(2) COMP-X VALUE 108.
03 FILLER PIC 9(2) COMP-X VALUE 48.
03 FILLER PIC 9(2) COMP-X VALUE 65.

It's as simple as that.
