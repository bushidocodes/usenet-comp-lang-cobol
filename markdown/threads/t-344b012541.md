```
Try adding 
@EnvSetWindow=UNUSE
@WinCloseMsg=ON
to the COBOL85.CBR file under the 
[program name] section.

Edit cobol85.cbr
[MyProg]
@EnvSetWindow=UNUSE
@WinCloseMsg=ON


The EnvSetWindow=UNUSE parameter turns off the displaying of 
the Run-Time Environment Window.
The WinCloseMsg=ON parameter turns off the displaying of the
'Console Window Closed' message.

The runtime Environment window allows setting variables that are 
needed by your program in order to run.  This is if you are referencing
files without having some form of internal name string area that has
the physical file name that would be found in Explorer.

Hope that this helps.
```
