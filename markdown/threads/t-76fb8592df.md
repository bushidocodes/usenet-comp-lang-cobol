We use MultiEdit - www.multiedit.com
This editor is HIGHLY configurable and will edit HTML, Basic, C, etc.
including color coding, intelligent indents, etc. for each of these
languages.  It even has context sensitive help for some languages like HTML.

For years, to get MultiEdit configured for COBOL you needed the Macro's
created by The Progeni Corporation  www.progeni.com which are still
available for free (in case you own an older version of MultiEdit) on their
website or in the download library at www.aboutlegacycoding.com.  However
the newest version of MultiEdit, version 9, ships with the Progeni COBOL
Macros built in.

We run MultiEdit on all the Windows platforms up to and including 2000.
