Some versions of PowerCOBOL (4 and 5) gave kernel errors if certain .DLLs in
the Operating system had were not compatible with the RTS, under specific
circumstances. I seem to recall having some kernel errors when using OCX
controls under version 5.

There are fixes for this that are downloadable from www.adtools.com and you
should report the problem to Fujitsu support.

Version 6 has no such problems. (AT least, iof it has, I haven't found
them...) and i use third party OCXs as well as my own COM controls with it.

Pete.
