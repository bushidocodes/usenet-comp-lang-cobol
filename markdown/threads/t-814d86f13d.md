Hard drives are slow but still faster than not being able to do it at
all.

Consider the alternatives:

Table in memory: With sufficient RAM=Fast, With insufficient
RAM=Impossible
Table on file: With sufficient RAM=Fast (OS buffers entire file), With
insufficient RAM=Slower

In sum, you lose (almost) nothing using a file for only a few records.
You lose the entire job for huge internal tables.
