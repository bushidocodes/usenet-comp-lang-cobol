You need to edit the file in HEX to change the sign. Hopefully you'll
have the HOURS field defined as signed-numeric, for example: PIC s9(3).
Load the test file into DataFile Editor and click on SHOW HEX. If the
decimal equiv is say 123+, in HEX it would be 1F 2F 3C (a C in the last
byte is for postive, D is for negative, and F is for unsigned or
display). Simply change the C to a D and this number will now be
negative. If you are stepping through your code using Animator, instead
of changing the field in DataFile Editor, you can double-click or
EXAMINE the field to temporarily change the contents of the field.

Mike
