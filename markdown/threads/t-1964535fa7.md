With DB2 v1.2 or v2.1 you can compile and execute on the Windows client.
Prior to these releases, Micro Focus provided a utility called SQLRP which
made it possible to compile on the Windows client by sending the compiler's
requests to the server and sending back the results: both the compiler and
DB2 believed that everything was happening "locally".

I followed up the original posting to this group with a private email to the
sender; if you have any questions, please contact me.
