I have a copy of the Fujitsu GUI PowerCobol. I am a person used to
creating Cobol programs using Microsofts (as was then) compiler to
generate straight forward programs, but I have been messing around with
the Fujitsu Powercobol and generated some very nice GUI front ends, but
my question is, are the any long standing programmers here that have
tried using P-C, but found it very confusing, or is it me. I say
confusing because you generate your GUI screens, then you have to select
options to fill in Procedures etc, I find using the good old SCREENS

interested to know.
