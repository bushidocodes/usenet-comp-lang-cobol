could extract the data from an excel spreadsheet using the excel ODBC driver
with embedded SQL in the COBOL program. Then write out the data in the form
you want and FTP it to the mainframe.

Hope this helps,

Hi,

This can be done but not by using ODBC. It's done using the 'INVOKE'
statement to control excel via it's OLE controls.

If you have Net Express, look in in the
.....\NETEXPRESS\BASE\DEMO\OLEDEMOS\EXCEL for a demo program to do exactly
what you suggest. But, as I said, not with ODBC.

The Excel ODBC driver (supplied by Microsoft) gives you SQL access to the
data in the spreadsheet and NetExpress can use ODBC as it's SQL layer.

Regards,
Paul
