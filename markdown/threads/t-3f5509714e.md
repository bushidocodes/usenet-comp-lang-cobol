```
       IDENTIFICATION DIVISION.
       PROGRAM-ID. COprint.
      *****************************************************************
      *
      *  This program will read student's names and grades from a
      *  file. The program will output the student's names and the
      *  their average grades.
      *
      *****************************************************************
       ENVIRONMENT DIVISION.
       CONFIGURATION SECTION.
       INPUT-OUTPUT SECTION.
       FILE-CONTROL.
           SELECT STUDENT-FILE
               ASSIGN TO DISK-INPUT-FILE
               ORGANIZATION IS LINE SEQUENTIAL.
           SELECT TRANSCRIPT-FILE
               ASSIGN TO "outrec"
      *****************************************************************
      *
      * The ASSIGN statement for TRANSCRIPT-FILE was changed to a
      * file so that we could use the PC_WIN_PRINT_FILE_EXT library
      * routine. The library explaination is below.
      *
      *****************************************************************
               ORGANIZATION IS LINE SEQUENTIAL.
      *****************************************************************
      *
      * The ORGANIZATION IS LINE SEQUENTIAL is the statement that
      * allows printing in line-over-line, instead of one long one.
      *
      *****************************************************************
       DATA DIVISION.
       FILE SECTION.
       FD  STUDENT-FILE
           LABEL RECORDS ARE STANDARD.
       01  STUDENT-REC.
           05  NAME-IN                 PIC X(20).
           05  GRADE1                  PIC 999.
           05  GRADE2                  PIC 999.
           05  GRADE3                  PIC 999.
       FD  TRANSCRIPT-FILE
           LABEL RECORDS ARE OMITTED.
       01  TRANSCRIPT-REC.
           05  NAME-OUT                PIC X(20).
           05  AVERAGE                 PIC 999.
       WORKING-STORAGE SECTION.
       01  DISK-INPUT-FILE        PIC X(12)  VALUE "DATAFILE.DAT".
       01  ARE-THERE-MORE-RECORDS PIC X(3)   VALUE "YES".
      * ***************************************************************
      *
      * The procedure division is divided into two modules. The 100-
      * MAIN-MODLUE will perform until there are no more records.
      * The 200-CALC-RTN will calulate the average grades and write
      * out names and average grades.
      *
      *****************************************************************
       01  MFPRNT16PTR PROCEDURE-POINTER.
      *****************************************************************
      *
      * The PROCEDURE-POINTER above was added to stop the possibility
      * of a printer error caused by the PC_WIN_PRINT_FILE_EXT, that
      * library routine is part of the MFPRNT16.DLL and is not
      * callable without the PROCEDURE-POINTER because there is not
      * a valid entry point for COBOL to locate, causing an error 173
      *
      *****************************************************************
       PROCEDURE DIVISION.
           SET MFPRNT16PTR TO ENTRY 'MFPRNT16'.
      *****************************************************************
      *
      * The setting of the PROCEDURE-POINTER. (above)
      *
      *****************************************************************
       100-MAIN-MODULE.
           OPEN INPUT STUDENT-FILE
                OUTPUT  TRANSCRIPT-FILE
           PERFORM UNTIL ARE-THERE-MORE-RECORDS = 'NO '
               READ STUDENT-FILE
                   AT END
                       MOVE 'NO ' TO ARE-THERE-MORE-RECORDS
                   NOT AT END
                       PERFORM 200-CALC-RTN
                   END-READ
           END-PERFORM
           MOVE x"0C" to TRANSCRIPT-REC
      *****************************************************************
      *
      * Above is a hard code HEX FORM FEED, for an explaination of
      * hard code HEX codes see your documentation or external
      * references to HEX codes.
      *
      *****************************************************************
           WRITE TRANSCRIPT-REC
           .

           CLOSE   STUDENT-FILE
                   TRANSCRIPT-FILE
           PERFORM 300-PRINT-FILE
           STOP RUN.
       200-CALC-RTN.
           MOVE NAME-IN TO NAME-OUT
           ADD GRADE1, GRADE2, GRADE3
               GIVING AVERAGE
           DIVIDE 3 INTO AVERAGE
           WRITE TRANSCRIPT-REC
           .
       300-PRINT-FILE.
            call "PC_WIN_PRINT_FILE_EXT"
                      using by reference "outrec" &x"00"
                            by reference "Output Record" &x"00"
                      BY VALUE 4 SIZE 2.
           .
      *****************************************************************
      *
      * PC_WIN_PRINT_FILE_EXT
      *
      * Sends the specified file to the printer, giving it a document
      * name and displaying the printer-options dialog box.
      *
      *
      * call "PC_WIN_PRINT_FILE_EXT" using     filename
      *                                        doc-name
      *                               by value options size 2
      *                              returning status-code
      *
      * Parameters:
      *
      * filename    pic x(<n>).
      * doc-name    pic x(<m>).
      * options    numeric literal.
      * status-code    See Key
      *
      * On Entry:
      *
      * filename    The name of the file to be printed. The name must
      * be space- or null-terminated.
      *
      * doc-name    The document name to be displayed. The name must
      * be null-terminated.
      *
      * options    A set of bits to set printer options:
      *
      * bit 3    Print in landscape orientation
      * bit 2    Print in portrait orientation
      * bit 1    Display a font dialog box, so you can select the
      *          default font for the document
      * bit 0    Display a printer control dialog box, so you can
      *          set up the printer
      *
      *   CONVERT BIT TO VALUES KEY
      *        BIT 0 = Value 1
      *        BIT 1 = Value 2
      *        BIT 2 = Value 4
      *        BIT 3 = Value 8
      *
      * Bits 2 and 3 cannot be used with bit 0 set.
      *
      *
      * On Exit:
      *
      * status-code    Printer status code:
      *
      * 0    Successful
      * 1    Could not open printer device
      * 4    Out of memory while printing
      * 5    Failed to open file
      * 6    Disk full while spooling file
      * 7    Print job aborted, no file spooled to Print Manager
      * 8    Printer information structure badly constructed
      * 9    No default printer found. Select a printer from Windows
      *      Control Panel.
      * 10    No commdlg.dll found. Should be in Windows system
      *       directory.
      * 11    Write failure
      *
      * Comments:
      *
      *
      * This routine is available only in the Windows environment.
      *
      * Example:
      *
      * To print out the file c:\config.sys in landscape:
      *
      *
      *      call "PC_WIN_PRINT_FILE_EXT"
      *                using by reference "c:\config.sys" &x"00"
      *                      by reference "My Config.sys" &x"00"
      *                      by value 8 size 2
      *                 The hard code HEX x"00" is a NULL.
      *
      *****************************************************************
```
