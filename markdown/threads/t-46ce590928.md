```
rts147= wrong open mode or access mode

An open OUTPUT <filename> erases all the data. You only use open output to
create a file. To write records to a file you must do Open I-O <filename>.
To read records you  must do Open INPUT. (Open I-O lets you read too). If
you intend to read your file sequentialy (sorted by key) you must change
your select clause to
 ACCESS IS DYNAMIC

I hope you understand my english......and don't forget to read your books
about cobol.....
```
