```
Easy answer first, there is no problem running both DOS/VS COBOL
and VS COBOL II (and eventually COBOL for VSE with LE for VSE) on
the same VSE machine.

There are a number of tools and consulting services that can help
you with this. Being a "little self-serving" you may want to
look at a book that I wrote called "OldBOL to NewBOL - A COBOL
Migration Tutorial for IBM Mainframe COBOLs" which is available
from Micro Focus Publishing at (800)661-5269. Pages 74-76 include
a list of tools and vendors.

You certainly will want to get a copy of the IBM publication
GC26-3150, "VS COBOL II Migration Guide for VSE".
```
