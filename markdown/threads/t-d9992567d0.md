If you want particular values initialized, and space considerations are
not important, you may consider having a copy of your table in working
storage, already initialized via value clauses.   Then MOVE
INITIAL-ARRAY TO WORK-ARRAY.

A simple move is very efficient, and value clauses are done at compile
time.
