```
1)Download the Fujitsu v3 (free) compiler.  
2)Get one or more good Cobol books. 
  a) Choose a style from amongst those recommended
  b) draft code fragments according to chosen style
3)START WRITING CODE.(this is over until
it passes the compiler without any warnings when
the compiler is set to "nitpick" mode AND the
runtime behavior is correct -- a few possibilities are:)
  a) Hello World
  b) "Guess" number from 1 to 100
  c) Sieve of Erasthosthenes
4) Try to find someone to CRITICIZE your code (once the compiler
no longer complains and it runs right, NOW the real criticism starts:)
  a) did you stick to chosen style?
  b) did you make any logic errors?

Cycle 3) and 4) with increasingly complicated (and more realistic)
problems.
```
