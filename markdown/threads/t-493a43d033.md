To all

Only to inform you that the Rpv Report Steel has been released from Dasp
Software..

This report generator is very useful to produce reports starting from
templates simply writing the instructions in a line sequential file with
the .rpv estension
RpvReport has an activex/ocx that can be included in your project to run
the application to show (preview) the report or direct it to a printer,
all in GUI format...

On their web site there are some examples written using Powercobol.

You can visit their web site at: www.rpvreport.com
