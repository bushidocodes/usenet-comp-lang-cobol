Hello,

I would like to share a little example of how to create a web services
using Netcobol. It can be consumed by any language, not only Cobol,
but I'm also providing a Powercobol sample of how to call it.

Regards, Emerson


*-----------------------------------------------------------------------
*
*   All Rights Public, Copyleft(C) Emerson
Lopes                        *
*
*
* - Web Service usando Fujitsu NetCobol for
Windows                     *
*-----------------------------------------------------------------------
*
 IDENTIFICATION   DIVISION.
 PROGRAM-ID       webServices.
 ENVIRONMENT      DIVISION.
 CONFIGURATION    SECTION.
 repository.
     class ClassCOM as "*COM"
     class ClassCOMException as "*COM-EXCEPTION".

 DATA             DIVISION.

 WORKING-STORAGE  SECTION.

 01  errorSource                    pic x(1024).
 01  errorMessage                   pic x(1024).

 01  cgiParameters.
     05  resultado                  pic x(512).  *> Defina o tamanho
conforme o necessário
     05  paramName                  pic x(128).
     05  paramValue                 pic x(1024). *> Se necessário,
aumente este limite
     05  keyName                    pic x(128).
     05  webMethod                  pic x(32) value
"NO_WEB_METHOD_PASSED".
     05  indice                     pic s9(9) comp-5 value 0.

 01  nodeName                       pic x(128).
 01  soapPrefix                     pic x(128).
 01  xmlDOMClass                    pic x(30) value
"Microsoft.XMLDOM".
 01  XmlDOM                         object reference ClassCOM.
 01  xmlNodes                       object reference ClassCOM.
 01  xmlFirstChild                  object reference ClassCOM.
 01  XmlNode                        object reference ClassCOM.
 01  documentElement                object reference ClassCOM.

 COPY COBW3.

*
 PROCEDURE        DIVISION.
 declaratives.
 ErrorEvent section.
      use after exception ClassCOMException.

         invoke exception-object "GET-SOURCE"      returning
errorSource
         invoke exception-object "GET-DESCRIPTION" returning
errorMessage

         string errorSource  delimited by size
                " "          delimited by size
                errorMessage delimited by size
           into resultado

         perform setResponseHello
         perform returnResponse
         perform freeResources

         exit program.

 end declaratives.

 main.
     perform initializeCGI
     perform getSOAP
     perform getWebMethod

     evaluate webMethod
              when "helloCobol"
                   perform buildResponseHello
                   perform setResponseHello

           *> when "outro webmethod"
                   *> ...

              when "NO_WEB_METHOD_PASSED"
                   move "SOAP incorreto!!!"   to  resultado
                   perform setResponseHello
     end-evaluate

     perform returnResponse
     perform freeResources

     exit program.


 initializeCGI.
     move    LOW-VALUE                to  COBW3

     call    "COBW3_INIT" using COBW3.

     move    "text/xml; charset=UTF-8"   to  COBW3-CONTENT-TYPE  *>
definindo o content-type de retorno do web service

     call    "COBW3_PUT_HEAD" using COBW3

     set     COBW3-PHYSICALPATH              to  true  *> definindo
caminho físico até o arquivo XML, baseado na localização do programa
CGI

     call    "COBW3_GET_REQUEST_INFO" using COBW3

     move    COBW3-REQUEST-INFO              to  COBW3-HTML-FILENAME.

     move    "\webService.xml"               to  COBW3-HTML-
FILENAME(COBW3-REQUEST-INFO-LENGTH + 1:). *> definindo template XML
SOAP a ser usado

 getSOAP.
     set     COBW3-QUERYSTRING               to  TRUE *> solicitando
web parameter

     call    "COBW3_GET_REQUEST_INFO" using COBW3

     if      COBW3-REQUEST-INFO-LENGTH = zero
             move  "request vazio"    to  resultado
             perform setResponseHello
             perform returnResponse
             perform freeResources
             exit program
     end-if

     invoke  ClassCOM "CREATE-OBJECT"    using xmlDOMClass returning
xmlDOM *> Criando instância do PARSER XML

     invoke  xmlDOM "loadXML"            using COBW3-REQUEST-INFO. *>
Carregando XML recebido com resultado no PARSER

 getWebMethod.
     *> Definindo qual webMethod foi chamado
     invoke  xmlDOM "GET-documentElement" returning xmlNode   *>
Recuperando XML como um objeto DOM

     if      xmlNode not = NULL
             invoke  xmlNode       "GET-firstChild" returning
xmlFirstChild    *> Recuperando elemento BODY
             invoke  xmlFirstChild "GET-firstChild" returning
xmlFirstChild    *> Recuperando node com nome do web method

             invoke  xmlFirstChild "GET-nodeName" returning
nodeName

             unstring nodeName delimited by ":" into soapPrefix
webMethod  *> Recuperando o nome do método

             if      webMethod = spaces     *> se o SOAP não tiver
prefixo...
                     move   nodeName   to  webMethod
             end-if
     end-if.

  buildResponseHello.
     *> Recuperando parâmetro de entrada
     invoke  xmlDOM "GET-documentElement" returning xmlNode   *>
Recuperando XML como um objeto DOM

             invoke  xmlNode       "GET-firstChild" returning
xmlFirstChild    *> Recuperando elemento BODY
             invoke  xmlFirstChild "GET-firstChild" returning
xmlFirstChild    *> Recuperando node com nome do web method

     move    "//name"                    to  paramName

     invoke  xmlDOM "selectSingleNode" using paramName returning
xmlNode   *> Recuperando node com parâmetro

     invoke  xmlNode "GET-TEXT" returning
paramValue.                     *> Recuperando valor do parâmetro

     string  "Hello, "                        delimited by size
             paramValue                       delimited by space
             ", welcome to WebServices World" delimited by size
        into resultado.

  setResponseHello.
     *> Setando parâmetro de retorno
     move    "Mensagem"                      to  COBW3-CNV-NAME

     set    COBW3-CNV-MODE-ADDREP            to  TRUE.

     move   0                                to  COBW3-CNV-VALUE-
LENGTH.
     move   function LENG("Mensagem")        to  COBW3-CNV-NAME-
LENGTH.
     move    resultado                       to  COBW3-CNV-VALUE

     call   "COBW3_SET_CNV_XX" using COBW3.

   returnResponse.
     *> enviando XML para o client do web services
     call   "COBW3_PUT_HTML" using COBW3.

  freeResources.
     call    "COBW3_FREE" using COBW3.

This is XML SOAP returned by the Cobol web services
<?xml version="1.0" encoding="utf-8"?>
     <soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/
envelope/" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
xmlns:xsd="http://www.w3.org/2001/XMLSchema">
	     <soap:Body>
            <Parametros xmlns="http://tempuri.org/">
                <Name>//COBOL//Mensagem//COBOL//</Name>
            </Parametros>
         </soap:Body>
     </soap:Envelope>


This is a simple Powercobol sample that show how to consume the web
services above

 ENVIRONMENT     DIVISION.
 DATA            DIVISION.
 WORKING-STORAGE SECTION.
 01  URI         pic x(256) value "http://www.oocobol.net/
NetCobolWebServices.wsdl".
 01  soapClient  pic x(128) value "MSSOAP.SoapClient30".

 01  webService  usage object reference COM.
 01  webMethod   pic x(256) value "helloCobol".

 01  nome     pic x(256) value "Emerson".

 01  outRetorno  pic x(8192).

 PROCEDURE       DIVISION.

     invoke COM "CREATE-OBJECT" using soapClient returning webService

     invoke webService "mssoapinit" using URI

     move   "Text" of CmText1     to  nome

     invoke webService webMethod using nome returning outRetorno

     INVOKE MainForm "DisplayMessage" USING outRetorno

If you wish to consume the Cobol web services using another language,
here is the WSDL file (yes, it is online!!!)

http://www.oocobol.net/NetCobolWebServices.wsdl
