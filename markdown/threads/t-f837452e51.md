PowerWeb Server++ Version 2.08 release 4 with database support is now available.
______________________________________________________________________


Major new features introduced in this release are:

1. A new Access Control Permissions model which greatly simplifies
administration.

2. New Access Counter facilities for recording and displaying the
date and time the counter was last reset.

3. Access Counters now work across all protocols including FTP. You
can view statistics by protocol or in total.

4. Support for Sybase/Watcom SQL Anywhere.


As usual, read /powerweb/history.htm on your site after installation
for further details.
