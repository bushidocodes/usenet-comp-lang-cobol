  I'm assuming you don't mean link as in compiling and linking your
program into executable code, but rather as in having your program
know the name (and location) of your files.  If this is the case, I
know two ways:

1) You can set your file with a DOS .bat file...
SET TFPGMLOG=C:\MFORGW\TFPGMLOG.TXT
   or, if you're animating, in Mfextmap.dat...
TFPGMLOG C:\MFORGW\TFPGMLOG.TXT
   and then in your program use the Compiler Directive
   ASSIGN"EXTERNAL" and a standard Select Statement:
            SELECT PROGRAM-LOG
                   ASSIGN TO TFPGMLOG.

2) You can have a Working Storage item...
        01  WS-PGM-LOG     PIC X(22) VALUE 'C:\MFORGW\TFPGMLOG.TXT'.
   with a Select Statement:
            SELECT PROGRAM-LOG
                   ASSIGN TO DYNAMIC WS-PGM-LOG.
