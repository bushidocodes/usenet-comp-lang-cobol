Paul,

You should be able to run your old INT code using Net Express.

With GNT code you won't be able to run it if it was created using any 16-bit 
product, or the 32-bit Workbench 4.0 product. The reason for that is that 
the product was sigificantly redesigned in order to support threading etc 
after Workbench 4.0 and therefore the Net Express product line does not 
contain support for the old GNT fomat.

That is probably not what you wanted to hear, but I hope it clarifies the 
situation nonetheless.

Gael.
