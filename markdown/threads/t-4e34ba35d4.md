In the recent discussions concerning FUNCTION REM, there was
no mention that the result of the function is NUMERIC and that the
function can accept NUMERIC (non-INTEGER) arguments.

I have included a small program with word problems to demonstrate
this capability.

-------------------
      $set ans85 flag"ans85" flagas"e"
       identification division.
       program-id. func-rem.
      * demonstrates the functions rem, integer, and numval
       data division.
       working-storage section.
      * a numeric result
       01 nresult pic -z(6)9.99.
      * an integer result
       01 iresult pic -z(9)9.
       procedure division.
       begin.
           perform problem-1
           perform problem-2
           perform problem-3
           stop run
           .

       problem-1.
           display " "
           display "How many feet and inches are in a meter?"
      * there are 100 cm per meter, 2.54 cm per inch
      * and 12 inches per foot
           compute iresult = function integer ( ( 100 / 2.54 ) / 12 )
           display iresult " feet"
           compute nresult = function rem ( ( 100 / 2.54 ), 12 )
           display nresult " inches"
           .

       problem-2.
           display " "
           display "If an 8 foot peice of lumber is cut into"
           display "sections that are 2 1/2 feet long, how"
           display "many sections are there?"
           display "How much is left (in feet)?"
           display "How much is left (in inches)?"
           compute iresult = function integer ( 8 / 2.5 )
           display iresult " sections"
           compute nresult = function rem ( 8, 2.5 )
           display nresult " feet"
           compute nresult = function numval ( nresult ) * 12
           display nresult " inches"
           .

       problem-3.
           display " "
           display "If the lumber was cut into 2 1/4 foot sections?"
           compute iresult = function integer ( 8 / 2.25 )
           display iresult " sections"
           compute nresult = function rem ( 8, 2.25 )
           display nresult " feet"
           compute nresult = function numval ( nresult ) * 12
           display nresult " inches"
           .
