Tom,
Declare a linkage section and put one of these hostent records in there.
Then put a 01 pointer item in the working-storage.
The API returns a pointer to the structure, so the return value is the
01 item.
Then, set the address of the linkage record to the pointer:

01 retPtr usage pointer.
...
linkage section.
01 theHostent   hostent.
...
call winapi "gethostbyname" using "server_name" returning retPtr
set address of theHostent to retPtr

The documentation says that you should copy this structure before
making any other Socket API calls.

Also, bear in mind that the structure contains further pointers, which
you will need to dereference in a similar way. h_aliases and
h_addr_list are actually pointers to pointers, so will need to be
carefully used.
Thane's example might help further here.

Steve.
