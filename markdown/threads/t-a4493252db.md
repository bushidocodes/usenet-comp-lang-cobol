```
There is a distinction between an API call and a system command.
A 'system call' is too ambiguous. Do you mean a call of an API function
or a system command?
As you state your example 'system(...)' it seems that you mean a command
(such as "dir" or "cd \xxx\ccc") as opposed to a DOS API which is an
interrupt 21h function.

Assuming you want a system command you could do
move "dir" to command-line
call "DOS_EXEC" using command-line

But you got to be a bit more precise.
```
