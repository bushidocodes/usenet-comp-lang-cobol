pstruc10.zip Generates structure chart from COBOL source

PSTRUC v1.0 is a must-have utility program for serious COBOL programmer,
employing structured programming techniques, which are universal and
language independant. It accepts the COBOL program source file under the
name of ISOURCE.CBL and it reads this source file, analizes it and
generates the program structure chart in the form of an indented list.
So, you no longer need to maintain the structure charts of your programs
manually. No need either to use paragraph number prefixes to make the
program structure stand out. Everytime you make a major modification, an
enhancement to your program, you can automatically generate the structure
chart of it. So, you can have a global view of your program. When you
are in the source file of your COBOL program for editing it, it is
exactly like if you were in a forest. You see nothing but the trees. If
you really want to see the forest, you should get on a plane or on a
helicopter and glance at the forest from an altitude of hundreds or maybe
thousends of feet above from the ground so that you can see the forest,
not the trees only, with the roads, the lakes, the rivers, the woods of
oak trees or the woods of pine trees.....

PSTRUC also can generate another useful list. It is the list of the
paragraphs that you no longer perform from anywhere in your program. So,
they are dead procedures which do not serve any purpose, maybe left there
because you forgot to eliminate them after having eliminated the calls
to these procedures during a modification. So, by generating a list of
the unused paragraph definitions, you can determine whether you should
keep them for an anticipated use or delete them.
