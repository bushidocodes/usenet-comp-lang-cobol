> 2) some sample programs, tutorials, howtos, etc

Look up Thane Hubbell's book on CObol.  Excellent starting point.  It
contains the compiler in the back, far easier than a download.

> 3) do I need an IDE or should I just use VI?

If you are vi literate there is a good windows version on :
http://www.vim.org/
I use it but would not recommend it to someone that is not already
comfortable with VI.

There is an IDE with the fujitsu download, I find it not too flash in
this version and simply gave up on it.  It is workable but there are
better choices.

> 4) any other advice on where to start, anything else I might need to
> get going

For help try and provide code to tell us what you have already tried,
we will always help if you show us you have worked with it.  It also
makes it easier to understand where you are really having problems
