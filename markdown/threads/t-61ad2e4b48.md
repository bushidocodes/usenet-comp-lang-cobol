That is no problem.

You use the screen section, you declare it in the data division, directly
after the working-storage section.

You use entries like
01  S-RECIPE-LIST HIGHLIGHT.
           04  LINE YPOS COL XPOS.
           04  BACKGROUND-COLOR 1 FOREGROUND-COLOR 7.
               08  PIC X(15)BB
                   USING LSTRECIPE(LISTRPOS).
               08  PIC Z9 USING LSTRUNIT(LISTRPOS).
               08  VALUE "/" HIGHLIGHT.
               08  PIC Z9 USING LSTRFLOAT(LISTRPOS).
               08  PIC BBX(5) USING LSTRSOLIDUNIT(LISTRPOS).
               08  PIC BZ9 USING LSTRALCPER(LISTRPOS).

Using uses a variable in the W-S or file section.

You use them as follows : display s-recipe-list.
You can use accept to : accept s-recipe-list.  To accept, instead of using
"using" you put "to", as in

     *//ACCEPTS INPUT TO EDIT,CREATE A NEW COCKTAIL
       01  S-COCKTAIL-ACCEPTNEW HIGHLIGHT.
           04  LINE  5 COL 46 PIC X(25)
               TO C-NAME.
           04  S-COCKTAIL-ACCEPTEDIT.
               08  LINE  6 COL 46 PIC X(25)
                   TO C-DESCR.
               08  LINE 7 COL 46 VALUE "         ".
               08  LINE  7 COL 46 PIC 9.999,99
                   TO C-PRICE.

Here you can type "accept s-cocktail-acceptnew
or "accept s-cocktail-acceptedit"

Hope this is any help.
