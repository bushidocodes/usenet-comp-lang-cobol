We have been using Micro Focus Cobol since 1985 developing financial
systems. Just last year we converted all our systems to Windows using
the same Micro Focus Cobol and Cobol Sp2 from Flexus for the GUI
interface.

Converting to Cobol Windows is very easy. All the screen sections
will be taken out and replaced by the screen definition generated from
Cobol Sp2 (if using Sp2). The rest of reading and writing the screen
is handled by Cobol Sp2 leaving your program doing only the business
logic.

Should you need more info, please do not hesitate to e-mail me.

Tony M. Mina
