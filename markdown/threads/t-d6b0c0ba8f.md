275-WORDING-MOD-RTN.
IF ONE-TEN
MOVE NUM TO ONES
MOVE ZEROS TO NUM

ELSE
IF ELEVEN-NINETEEN
SUBTRACT 10 FROM NUM GIVING TEENS
MOVE ZEROS TO NUM
ELSE
IF TWENTY-NINTYNINE
COMPUTE TENS =num / 10
COMPUTE NUM = num - (10 * TENS)
ELSE
IF HUNDRED-NINE
COMPUTE HUNDREDS =num / 100
COMPUTE NUM =num - (100 * HUNDREDS)
ELSE
IF THOUSAND-TEN
COMPUTE THOUSANDS =NUM / 1000
COMPUTE NUM =NUM - (1000 * THOUSANDS)
ELSE
IF THOUSAND-NINETEEN
SUBTRACT 10000 FROM NUM GIVING SUB1
COMPUTE THOUSANDS-TEENS =SUB1 / 1000
COMPUTE NUM =NUM - (10000 + (THOUSANDS-TEENS * 1000))
ELSE
IF THOUSAND-TWENTY-NINETYNINE
COMPUTE THOUSAND-TENS =NUM / 10000
COMPUTE NUM =NUM - (10000 * THOUSAND-TENS)
ELSE
IF THOUSAND-HUNDRED-NINE
COMPUTE THOUSAND-HUNDREDS =NUM / 100000
COMPUTE NUM =NUM - (100000 * THOUSAND-HUNDREDS).
