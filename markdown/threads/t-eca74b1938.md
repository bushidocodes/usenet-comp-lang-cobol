Its documented in Toolset Reference Vol 2, Appendix A. Summary:

call x"91" using result, fn-code, param
result is a group field:
error pic x comp-x
handle pic xx comp-x
attr pic x comp-x
time pic xx comp-x
date pic xx comp-x
size pic x(4) comp-x
fname pic x(n)

fn-code pic x comp-x value 69

param is a group:
action pic x comp-x
attrin pic x comp-x
filein pic x(n)

set action 0=find first, 1=next, 2=terminate search
error set to 0=ok, 1=no more files, 2=error
