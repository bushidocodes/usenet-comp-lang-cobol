They allow you to accept from the console????

Anyway, what you are talking about would require a dynamic allocation
routine.  The DD would be assigned at run time through an SVC99 call (thus,
not coded in the JCL).  Your SELECT/ASSIGN and FD would like this:

SELECT  DYNAMIC-FILE-IN    ASSIGN TO UT-S-FILEIN
                                                      FILE STATUS IS
FILEIN-STATUS.

FD  DYNAMIC-FILE-IN
       RECORDING MODE IS F
       BLOCK CONTAIN 0 RECORDS
       LABEL RECORDS ARE STANDARD.

01  FILEIN-RECORD                  PIC X(100).

If the file can be different sizes, or even different formats (such as
variable), then it becomes a little more difficult (although still
possible).  You may even want to consider an Assembler I/O module.

If this is work, I suggest that you talk to your senior programmers or
system programmers.  If this is school, perhaps you should take this to your
instructor....

Dave
