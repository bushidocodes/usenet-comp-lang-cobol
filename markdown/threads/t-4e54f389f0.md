```
I have received several email responses since Friday 3/8/96 in response
to my plea for help in solving how can I pull a file from an IBM system
back to an HP3000 system with the end result leaving packed fields in
tact?

The solution in our case follows:

We were able to convert the file by retranlating the file back to EBCDIC
excluding the packed fields and then performing

FCOPY FROM=hpEBCFIL;TO=hpASCFIL;
EBCDICIN;...;EXCLUDE=


I hope this can help someone else in the future....
```
