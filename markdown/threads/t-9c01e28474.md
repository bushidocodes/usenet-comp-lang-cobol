```
No, neither are part of the 200x Standard - but are in common usage for
several workstation COBOL compilers.

78 levels are "constants"  - The draft Standard includes a new attribute
called "CONSTANT" which is similar - but not identical.

The draft Standard introduces new usages BINARY-CHAR BINARY-SHORT
BINARY-LONG - which take *no* PICTURE clause (as COMP-X does) - but defines
a "true binary" field.
```
