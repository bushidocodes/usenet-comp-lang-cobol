write it as a file with fixed lenth records.  Be warned though, any calculated
numbers will have many decimal points, not just the nice format you see.  Maybe
Access could help with that.  I just wrote a COBOL routine to round the number.

point into the array.  Then I calculated the value based on the position of the
character.  Or something like that.

You need to save as a "formatted text, space delimited" file type which will
generate a ".prn" file.

But first make sure you 'fix' the fields first.
Go to each column and highlight it, then go to Format, Column and Width and
make the column a fixed width
next, go to Format, Cells, and alignment to make the field right adjusted or
left adjusted as needed (alpha or numeric data)

do this for each column then go to File, Save-as, choose 'formatted text,
space delimited' and you have a decent sequential file

any problems, email me directly at    indreboe at   ATT dot Net
