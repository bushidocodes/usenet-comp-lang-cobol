If you select Line Sequential as your organization you can us the CR
at the end of each record.


>What are the differences between sequential/line sequential ?

Line sequential is made up of lines of varying length, terminated by a
record terminator - in this case CR LF.  Normal Sequential files are
fixed length and have no terminators - they are contigous in the data
file.  I can't explain the 6's you are seeing - I don't see things
like that when I write a program to create a sequential file.
