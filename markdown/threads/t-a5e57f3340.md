In a project, below each "Form" appear "(Script)[COBOL Script]" lines.

On each (Script) line, right-click. Pick Properties.

Pick the "Copy Folders" tab.

Type the path to some copy book members, select "Append"
Type the path to some more copy book member, select "Append"
etc.

If you can find a way to do this with an environmental command, I'd
certainly want to hear about it.
