```
Hi all,

We are using Micro Focus OCDS 4.0 to develop some Unix applications on SCO
OpenServer 5.05.

One of these applications is installed on a 5.05 server and there are about
50 users that make use of it. Lately we keep getting some unexpected runtime
errors, where the only message we get is

"fatal error : buffer not empty"

and the application stops. We then restart it and it works correctly until
the error appears again, in a completely unexpected manner. We are getting 5
to 10 of these errors per day, and we have never faced a data corruption
problem, although the application stops abnormally.

Does anybody have seen this problem before ?

Thanks in advance for your reply.

Panos Zotos
Athens, Greece
```
