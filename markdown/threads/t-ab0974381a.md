The AS/400 is an IBM system which has been around for about a dozen years and
has a well-
deserved reputation for being a 'bullet-proof' system.  It runs a proprietary
Operating
System (OS/400) which includes a version of DB/2 built in.  The OpSys and DB
run in a 'separate' processor from problem programs.

It supports a lot of languages, including COBOL, and can be obtained in a
config
strong enough to support anythere from 1 - 4,000 users with terra-byte DBs.
