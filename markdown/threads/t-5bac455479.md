Put the -a argument in quotes, and separate your sub-arguments with
commas or some other delimiter that doesn't appear in the argument. Then,
use UNSTRING to recover the original arguments:

runcobol file.cob -a "arg1,arg2,arg3"

will let you pass three arguments in this way. The double quotes allow
spaces to be passed.

The way I have always done it has been to use a single argument free form
separated by semicolons or commas (whichever is convenient). Within the
COBOL program use UNSTRING to separate the various arguments.

You can refer to the RM/COBOL User's Guide Appendix G for some examples.
The rmmapinx utility, which is written in RM/COBOL, uses the following
command line syntax:
runcobol rmmapinx A='file-name,DETAIL,PRINTER'
