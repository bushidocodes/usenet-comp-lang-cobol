```
--Some 78's...
78 Suffix value "A".
78 Api-LoadImage value "LoadImage" & Suffix.
78 IMAGE-BITMAP value 0.
78 IMAGE-ICON value 1.
78 IMAGE-CURSOR value 2.
78 IMAGE-ENHMETAFILE value 3.
78 LR-DEFAULTCOLOR value h"0000".
78 LR-MONOCHROME value h"0001".
78 LR-COLOR value h"0002".
78 LR-COPYRETURNORG value h"0004".
78 LR-COPYDELETEORG value h"0008".
78 LR-LOADFROMFILE value h"0010".
78 LR-LOADTRANSPARENT value h"0020".
78 LR-DEFAULTSIZE value h"0040".
78 LR-VGACOLOR value h"0080".
78 LR-LOADMAP3DCOLORS value h"1000".
78 LR-CREATEDIBSECTION value h"2000".
78 LR-COPYFROMRESOURCE value h"4000".
78 LR-SHARED value h"8000".
78 No-Of-Bits VALUE 32.
78 Byte-Len VALUE 4.
78 Word-Len VALUE 4.

--here is what I define the hbmp as...

01 hbmp pic x(Word-Len) comp-5.

--... some code...

****************************************************************
* Note: LoadImage with LR-LOADFROMFILE is ONLY support on
* Windows 95/NT4.0 or later!
****************************************************************
call WINAPI Api-LoadImage
using by value 0 size 4,
by reference z"MF2.bmp",
by value IMAGE-BITMAP,
by value 0 size 4,
by value 0 size 4,
by value LR-LOADFROMFILE
returning hbmp
```
