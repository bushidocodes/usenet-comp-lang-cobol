The first question is implementor dependent. The second question is an
implementor extension. So in both cases you must state the platform and which
implementation of COBOL you are using. For example, on IBM mainframes, there is
no difference between "binary" and "text" modes, and system commands usually
involve writing an Assembler routine and using a CALL statement in the COBOL
program to call the assembler routine.
Mark A. Young
