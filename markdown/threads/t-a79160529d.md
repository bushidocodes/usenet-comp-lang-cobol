```
>How many bytes does a disk-dataset (device = 3900) with LRECL = 133;
>BLKSZ = 27930 and TRACKS = 9897 have?

A 3390 device will accommodate 2 blocks per track when the block size
is 27930, so:

   2 * 27,930 * 9,897 = 552,846,420 data bytes
```
