Well, with IBM's mainframe compiler they give you a utility to translate the
reserved words into other languages (as synonyms) and "recompile" the table
to a new "static load module".  (Check out the WORD compiler option.)  There
are 2 interesting problems with this:

A) They don't have the utility for their Workstation products (so if you use
it on the mainframe, you must develop/maintain your code there too)
B) (Historically) they have been AWFUL about providing "migration" tools for
taking the "old" source (much less object) code and upgrading it - when new
reserved words ARE added.

    ***
Micro Focus (OTOH) does allow you to create synonyms (ADDRSRV directive) "on
the fly" (not in a file - unless you COPY it in) or to use "vocabularies" -
which also mean that YOU have to maintain them, instead of them.
