```
ooh!  i know! i know!  finally a question i can handle!
in your ENVIRONMENT DIVISION, INPUT-OUTPUT SECTION, FILE-CONTROL:
SELECT MY-FILE
    ASSIGN TO "c:\my documents\data.001".
you can also name files from your working-storage section should your
program have to create and use files based on user input.
```
