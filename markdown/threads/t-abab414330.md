Windows environment using the Microfocus Dialog System product with some
success. While this does not eliminate the need for some program
restructuring, the amount of work required is quite reasonable.

Another option might be to use the Microfocus Visual COBOL product. I have
not tried it but the product description seems as if it might meet your
requirement.
