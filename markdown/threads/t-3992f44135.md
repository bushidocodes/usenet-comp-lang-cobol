Following on from Jimmy's post, I'm assuming that you have changed the build 
type from Debug to Release. Having done so, I'm also assuming that you still 
get the link errors?

If so, then under Project -> Build Settings, you should be able to invoke 
the Link Wizard, which will help ensure that the correct Micro Focus 
libraries are pulled in at link-time, which should resolve the errors you're 
seeing.

If you're still hitting problems -- and also to get further information on 
your Dialog System query -- I would suggest posting to the CobolPortal forum 
for UE support, which is accessible from under http://www.cobolportal.com .
