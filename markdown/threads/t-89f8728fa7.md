Let me re-state this to see if I have it right... are you asking how to
determine if a record has been modified when there is no record-modified
flag (timestamp or any other) in the record? To the best of my
knowledge this is impossible... how does one check what is not there? I
would say that either the format must be changed or two versions of the
file must be made available so that records with the same key can be
compared, one to the other.

Try this approach:

Create a new file, which holds the modified records. Search by key in
the new file first, then in the old file. Shut down the system
periodically to merge the modified records into the original file.

This approach can be extended to N generations, but my advice is to make
it work for one generation first.

Hope this helps
