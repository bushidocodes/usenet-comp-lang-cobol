I'd first like to say I'm not too familiar with the Cobol language. What I'd
like to know is if it's possible to send a command to a printer to change
fonts. We have a AS/400 program that we would like to use a barcode in. Our
software programmers tell us we can't do it without an IPDS compatible
system. The printer manual (a 4840 model) says to change the font of the
printer, but gives example only in BASIC code. Is it right to assume that
it's not possible to convert that BASIC code into something that can be
understood by Cobol? Thanks for any info.
