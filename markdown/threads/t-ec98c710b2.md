Our DMS Software Reengineering Toolkit is capable
of parsing and building compiler data structures for
COBOL programs,  carrying out arbitrary
transforms on the COBOL program source,
and then prettyprinting the revised source text.
Transforms are specified using patterns:
          LHS -> RHS
where LHS and RHS are arbitrary COBOL fragments with parameters.
Finding and extracting particular bits of code
would be a straightforward application.
See http://www.semdesigns.com/Products/DMS/DMSToolkit.html.
