Frederico,

We're using Parkways ODBC-Driver for a few years now. It works great.
They even offer a Unix and/or NT based Server, so that you're always
using the right filehandler (Unix and NT/DOS Filehandlers are
different so that index-files will crash) and the server is really
fast...but only if you're working on Unix/NT :-)

On the other hand their ODBC-Driver is able to deal with MF's
FileShare.

Check out their website: http://www.parkway-software.com

HTH - Stefan Meyer
