Your approach of putting the file names in the config.ini file with Notepad
(as test data) should work.  Basically, you open that file and read it as a
sequential file, placing the filenames into working storage names you used
in your select clauses. The other selects are written so that the files are
assigned to the working storage names, exactly as you specified.  IE:
open input config-file.
read config-file into WS-FILE-1 (at end is error)
read config-file into WS-FILE-2 (at end is error)
etc.
close config-file.
open other files.
