Dear Mario.

Activelog it is not a utility that is supplied by Acucorp, so I am unable to
provide you with any information concerning this program.

To find out how many users are using AcuServer, you just need to use
'acuserve -info'.   This will display the users that are currently connected
to AcuServer, along with a list of files that they are using, so that you
can keep watch of what files are currently be accessed.

I hope this helps.
