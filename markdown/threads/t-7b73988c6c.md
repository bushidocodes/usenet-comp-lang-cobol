fixed-point numeric literal or a single fixed-point numeric data

words, it might be more precise.

digits. The expression 1/101 would evaluate to 0.009900990 whereas the literal
0.009900990099 would have exactly that value. If I ROUNDED them into a pic v9(10), the
former would give .0099009900 whereas the latter would give .0099009901.
