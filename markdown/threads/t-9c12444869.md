Pretty simple...

Create a variable that contains the command and parameters...

WORKING STORAGE SECTION.

01 W-PRINTER-COMMAND.

05 FILLER PIC X(03) VALUE " lp"
05 W-PRINTER-COMMAND-DEVICE PIC X(02) VALUE "-d".
05 W-PRINTER-COMMAND-PRINTER-ID PIC X(03) VALUE SPACES.
05 FILLER PIC X(01) VALUE SPACES.
05 W-PRINTER-COMMAND-FILE-TO-PRINT PIC X(08) VALUE SPACES.
05 FILLER PIC X(01) VALUE LOW-VALUES.


Then in an appropriate routine...

PROCEDURE DIVISION...

...

9999-FOO SECTION.

...

CALL "SYSTEM" USING W-PRINTER-COMMAND.
CANCEL "SYSTEM".

...

9999-FOO-EXIT.
EXIT.



Hope this helps...

...Steve
