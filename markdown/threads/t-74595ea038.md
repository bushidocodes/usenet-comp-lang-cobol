```
Assuming a modern compiler, the following should work:

01  Unstrung-Field Pic X(13).
01  Numeric-Field Pic S9(7)V99.

Compute Numeric-Field = Function Numval (Unstrung-Field)

If there might be $ symbols in the data, use NUMVAL-C instead of
NUMVAL.
```

```
Try

 01  source-data                 VALUE '1.23'        PIC X(12).
 01  work-field                  JUST RIGHT          PIC X(12).
 01  work-number REDEFINES work-field                PIC Z(9).99.
 01  destin-field                VALUE ZERO   COMP-3 PIC S9(9)V99.
 01  sub1                                     COMP   PIC S9(9).


* Find the last character used in the source field
     PERFORM
       VARYING sub1 FROM LENGTH OF source-data BY -1
       UNTIL sub1 <= 1 OR
             source-data(sub1:1) NOT = SPACE
       CONTINUE
     END-PERFORM
* right justify the data into the destination field
     MOVE source-data(1:sub1)        TO work-field
* convert the formatted number
     MOVE work-number                TO destin-field.
```
