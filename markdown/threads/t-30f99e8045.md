It appears that you want to have 12 values (percentages), one for each
month.  Each of these percentages is to two digits.  You seem to be
trying to predefine the percentages as '1', '2', '1', etc but these are
only _single_ digits.

      WORKING-STORAGE SECTION.
      01  Monthly-Percentages          VALUE '010201000203090501010100'.
          05  Monthly-Percentage       PIC 99 OCCURS 12.

This has 24 bytes of digits divided up into 12 groups of two digits each
such that Monthly-Percentage(1) has a value of 01 and
Monthly-Percentage(7) has a value of 09.

Either that or you only filled out the first 6 in the value statement
above.

What you _probably_ want to do is to have a month number in range 1 ..
12 and to calcualate the percentage and then store this result in the
table.  As in:

        COMPUTE Monthly-Percentage(Month-Number) ROUNDED =
                         ( Month-Sales - Month-Cost ) / Month-Sales *
100

(or however you think the percentage should be).
