Up until about 3 years ago I would have been able to help you with this,

heard was that it continued to be included with the basic OS/390 and 
z/OS install "tapes" (even though support was dropped in 1994), because 
we were still licensed to use it.  I used to be able to run OS/VS batch 
COBOL compiles as long as I steplib'ed or joblib'ed to a special library 
where that compiler lived.  Not any more.

any evidence that they are still running.
