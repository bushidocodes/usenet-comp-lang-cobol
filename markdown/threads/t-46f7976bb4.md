```
I take it you want to access the files FCD before the open, and set/unset
various fields etc? If so, use the directive fcdreg, which will allow you to
access a particular files fcd. ie

$set fcdreg
select testfile assign to "test"
organization is indexed
access mode is dynamic
record key is key1
.

fd testfile.
01 testrec.
03 key1 pic x.
03 rest pic x(10).

linkage section.
01 fcd.
copy "XFHFCD.CPY".

procedure division.
main.
set address of fcd to address of fh--fcd of testfile
*>
*> you can now manipulate the fcd as you wish (definitions in
*> XFHFCD.CPY).
*>
open output testfile
move all "a" to testrec
write testrec
close testfile
stop run
.

Hope this is what you were asking for?
```
