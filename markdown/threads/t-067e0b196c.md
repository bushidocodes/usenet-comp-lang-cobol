```
Link it in to the run time.

I'm thinking of the syntax off the top of my head here so here goes:

$ cc -c foo.c

$ cob -Xe "" myruntime foo.o

Then your cobol can do (bar.cbl):

procedure division.

	call 'foo'

and it will execute the C program. You can pass by reference or value.

You need to (of course) execute your cobol program with

$ myruntime bar

Hope it helps!
```
