Graham,
Sorry I missed the boat...if you're using TSO/ISPF or VM/CMS you can write an
Edit or XEdit macro to do it. A TSO/ISPF Edit macro would look something like:

/* REXX */
address ISREDIT
"MACRO PROCESS"
do h=0 to 256
if datatype(d2c(h),"ALPHANUMERIC") /* exclude a-z, A-Z, 0-9 */
then iterate h
"CHANGE ALL ""'"d2c(h)"'"" ""X'"d2x(h)"'"" .ZF .ZL"
end

PS. I tried it and it does work, but only if your value clause is one-character.
If it's more you'd need some more advanced parseing logic.
