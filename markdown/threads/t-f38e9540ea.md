```
If you are in the Seattle Washington area and are involved in a
migration project or just want to add one more RDBMS to your skill set
... consider the high-level classes now available at the University of
Washington.

http://www.extension.washington.edu/extinfo/certprog/oad/oad_crs.asp

These classes are not OCP exam prep. They are not PowerPoint slide
presentations. They are hands on Oracle training from the basics through
procedures, functions, packages, and triggers to DBA skills essential
for distinguishing yourself as a senior developer.

Extensive work is also done with many of Oracle's built-in packages such
as UTL_FILE, DBMS_JOB, DBMS_PROFILER, DBMS_SQL, etc. REF CURSORS, Native
Dynamic SQL, the Oracle security model ....

Any questions ... just ask.

Daniel Morgan
Instructor
```
