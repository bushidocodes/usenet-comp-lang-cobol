If you are under DOS or UNIX, there is a generic solution.
Create a batch accepting arguments. For example :
FOO.BAT
ECHO %1 %2 %3 >FILECMD
PROG.EXE

and now, in PROG.EXE, read the pure ascii file FILECMD to get the
options "-l testing".
