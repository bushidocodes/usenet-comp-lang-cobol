Dear:

Please, use the following codes, that works with all COBOLs in the world.
Some codes work fine with some specific COBOL compiler, not all.

WORKING-STORAGE SECTION.
01  BLANK-LINE       PIC X(80) VALUE ALL SPACES.
01  GRID-LINE        PIC X(80) VALUE ALL '- '.
 
870-PRINT-HEADINGS. 
        MOVE REPORT-TITLE-LINE    TO  OUT-LINE-RECORD.
        WRITE OUT-LINE-RECORD  AFTER  ADVANCING 1 LINES.
        MOVE COLUMN-HEADING-LINE1 TO  OUT-LINE-RECORD.
        WRITE OUT-LINE-RECORD  AFTER  ADVANCING 3 LINES.

* Move spaces to clear the variables from previous data.
* In some COBOL compilers there is no need to do that.
* This practice works all the time with all COBOL compilers
* although it look tedious to most of programmers, and to me.

        MOVE SPACES               TO  OUT-LINE-RECORD.
        WRITE OUT-LINE-RECORD   FROM  COLUMN-HEADING-LINE2 
                               AFTER  ADVANCING 1 LINE.
        MOVE SPACES               TO  OUT-LINE-RECORD.
        WRITE OUT-LINE-RECORD   FROM  GRID-LINE 
                               AFTER  ADVANCING 1 LINE.
        MOVE SPACES               TO  OUT-LINE-RECORD.
        WRITE OUT-LINE-RECORD   FROM  BLANK-LINE 
                               AFTER  ADVANCING 1 LINE.
The 2 approaches can be used interchangeable.

When you have report, please, use MfCobol Personal(v2.0.02) editor
to view and print it as you do with the source codes.

I have never seen MfCobol Personal(v2.0.02) but I believe it does.
Why don't you try it?
