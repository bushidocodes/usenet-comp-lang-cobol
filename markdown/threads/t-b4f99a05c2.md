Duh- I figured it out again. use figurative constants in the
special-names paragraph.  If anyone else is interested, here a short
little guy that acts liek a CGI program in most COBOLs. (A CGI program
was eaierted to test in.)

       identification division.
       program-id. test1.
       environment division.
       configuration section.
       special-names.
           symbolic characters CR is 14, LF is 11.
       data division.
       
       procedure division.
       program-start.
           display 'Content-type: text/html' CR LF CR LF
           display '<html>This is a test</html>'
           goback.
