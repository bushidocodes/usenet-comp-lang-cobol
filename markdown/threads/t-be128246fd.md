I think the above select should name the columns that are
being acted on the the fetch statement.

Something like

    exec sql
      declare curs1 cursor for
        select col-category, col-breed from breed-table
    end-exec

Where col-category is the column name for "category" and
col-breed is the column name for "breed" and the breed-table
is the name of the table containing the "category" and "breed"
columns
