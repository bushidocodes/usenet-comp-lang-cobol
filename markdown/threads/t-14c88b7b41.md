Try this:

working-storage section.
77 num pic 99.

01 byte.
03 high-nibble pic 99 comp.
03 high-nibble-char pic x redefines high-nibble.
03 low-nibble pic 99 comp.
03 low-nibble pic x redefines low-nibble.

procedure....
divide num by 16 giving high-nibble.
compute low-nibble = num - high-nibble * 16.
if high-nibble < 10
add 48 to high-nibble
else
add 65 to high-nibble.

... The same for Low-Nibble

high-nibble-char and low-nibble-char contain now the Hex-Code for 1 Byte.
